 <h3 align="CENTER">Book 3<br>Chapter 62
 </h3><table width="100%">
 <td width="50%">
  <font color="GREEN" size="-1">1</font>
  taṃ tathā śokasaṃtaptaṃ vilapantam anāthavat<br>
   mohena mahatāviṣṭaṃ paridyūnam acetanam<br>
  <font color="GREEN" size="-1">2</font>
  tataḥ saumitrir āśvāsya muhūrtād iva lakṣmaṇaḥ<br>
   rāmaṃ saṃbodhayām āsa caraṇau cābhipīḍayan<br>
  <font color="GREEN" size="-1">3</font>
  mahatā tapasā rāma mahatā cāpi karmaṇā<br>
   rājñā daśarathenāsīl labdho 'mṛtam ivāmaraiḥ<br>
  <font color="GREEN" size="-1">4</font>
  tava caiva guṇair baddhas tvadviyogān mahīpatiḥ<br>
   rājā devatvam āpanno bharatasya yathā śrutam<br>
  <font color="GREEN" size="-1">5</font>
  yadi duḥkham idaṃ prāptaṃ kākutstha na sahiṣyase<br>
   prākṛtaś cālpasattvaś ca itaraḥ kaḥ sahiṣyati<br>
  <font color="GREEN" size="-1">6</font>
  duḥkhito hi bhavāṁl lokāṃs tejasā yadi dhakṣyate<br>
   ārtāḥ prajā naravyāghra kva nu yāsyanti nirvṛtim<br>
  <font color="GREEN" size="-1">7</font>
  lokasvabhāva evaiṣa yayātir nahuṣātmajaḥ<br>
   gataḥ śakreṇa sālokyam anayas taṃ samaspṛśat<br>
  <font color="GREEN" size="-1">8</font>
  maharṣayo vasiṣṭhas tu yaḥ pitur naḥ purohitaḥ<br>
   ahnā putraśataṃ jajñe tathaivāsya punar hatam<br>
  <font color="GREEN" size="-1">9</font>
  yā ceyaṃ jagato mātā devī lokanamaskṛtā<br>
   asyāś ca calanaṃ bhūmer dṛśyate satyasaṃśrava<br>
  <font color="GREEN" size="-1">10</font>
  yau cemau jagatāṃ netre yatra sarvaṃ pratiṣṭhitam<br>
    ādityacandrau grahaṇam abhyupetau mahābalau<br>
 <font color="GREEN" size="-1">11</font>
  sumahānty api bhūtāni devāś ca puruṣarṣabha<br>
    na daivasya pramuñcanti sarvabhūtāni dehinaḥ<br>
 <font color="GREEN" size="-1">12</font>
  śakrādiṣv api deveṣu vartamānau nayānayau<br>
    śrūyete naraśārdūla na tvaṃ vyathitum arhasi<br>
 <font color="GREEN" size="-1">13</font>
  naṣṭāyām api vaidehyāṃ hṛtāyām api cānagha<br>
    śocituṃ nārhase vīra yathānyaḥ prākṛtas tathā<br>
 <font color="GREEN" size="-1">14</font>
  tvadvidhā hi na śocanti satataṃ satyadarśinaḥ<br>
    sumahatsv api kṛcchreṣu rāmānirviṇṇadarśaṇāḥ<br>
 <font color="GREEN" size="-1">15</font>
  tattvato hi naraśreṣṭha buddhyā samanucintaya<br>
    buddhyā yuktā mahāprājñā vijānanti śubhāśubhe<br>
 <font color="GREEN" size="-1">16</font>
  adṛṣṭaguṇadoṣāṇām adhṛtānāṃ ca karmaṇām<br>
    nāntareṇa kriyāṃ teṣāṃ phalam iṣṭaṃ pravartate<br>
 <font color="GREEN" size="-1">17</font>
  mām eva hi purā vīra tvam eva bahuṣo 'nvaśāḥ<br>
    anuśiṣyād dhi ko nu tvām api sākṣād bṛhaspatiḥ<br>
 <font color="GREEN" size="-1">18</font>
  buddhiś ca te mahāprājña devair api duranvayā<br>
    śokenābhiprasuptaṃ te jñānaṃ saṃbodhayāmy aham<br>
 <font color="GREEN" size="-1">19</font>
  divyaṃ ca mānuṣaṃ caivam ātmanaś ca parākramam<br>
    ikṣvākuvṛṣabhāvekṣya yatasva dviṣatāṃ badhe<br>
 <font color="GREEN" size="-1">20</font>
  kiṃ te sarvavināśena kṛtena puruṣarṣabha<br>
    tam eva tu ripuṃ pāpaṃ vijñāyoddhartum arhasi<br>
 </td>
 <td width="50%">
  <font color="GREEN" size="-1">1</font>
  तं तथा शॊकसंतप्तं विलपन्तम अनाथवत<br>
   मॊहेन महताविष्टं परिद्यूनम अचेतनम<br>
  <font color="GREEN" size="-1">2</font>
  ततः सौमित्रिर आश्वास्य मुहूर्ताद इव लक्ष्मणः<br>
   रामं संबॊधयाम आस चरणौ चाभिपीडयन<br>
  <font color="GREEN" size="-1">3</font>
  महता तपसा राम महता चापि कर्मणा<br>
   राज्ञा दशरथेनासील लब्धॊ ऽमृतम इवामरैः<br>
  <font color="GREEN" size="-1">4</font>
  तव चैव गुणैर बद्धस तवद्वियॊगान महीपतिः<br>
   राजा देवत्वम आपन्नॊ भरतस्य यथा शरुतम<br>
  <font color="GREEN" size="-1">5</font>
  यदि दुःखम इदं पराप्तं काकुत्स्थ न सहिष्यसे<br>
   पराकृतश चाल्पसत्त्वश च इतरः कः सहिष्यति<br>
  <font color="GREEN" size="-1">6</font>
  दुःखितॊ हि भवाँल लॊकांस तेजसा यदि धक्ष्यते<br>
   आर्ताः परजा नरव्याघ्र कव नु यास्यन्ति निर्वृतिम<br>
  <font color="GREEN" size="-1">7</font>
  लॊकस्वभाव एवैष ययातिर नहुषात्मजः<br>
   गतः शक्रेण सालॊक्यम अनयस तं समस्पृशत<br>
  <font color="GREEN" size="-1">8</font>
  महर्षयॊ वसिष्ठस तु यः पितुर नः पुरॊहितः<br>
   अह्ना पुत्रशतं जज्ञे तथैवास्य पुनर हतम<br>
  <font color="GREEN" size="-1">9</font>
  या चेयं जगतॊ माता देवी लॊकनमस्कृता<br>
   अस्याश च चलनं भूमेर दृश्यते सत्यसंश्रव<br>
  <font color="GREEN" size="-1">10</font>
  यौ चेमौ जगतां नेत्रे यत्र सर्वं परतिष्ठितम<br>
    आदित्यचन्द्रौ गरहणम अभ्युपेतौ महाबलौ<br>
 <font color="GREEN" size="-1">11</font>
  सुमहान्त्य अपि भूतानि देवाश च पुरुषर्षभ<br>
    न दैवस्य परमुञ्चन्ति सर्वभूतानि देहिनः<br>
 <font color="GREEN" size="-1">12</font>
  शक्रादिष्व अपि देवेषु वर्तमानौ नयानयौ<br>
    शरूयेते नरशार्दूल न तवं वयथितुम अर्हसि<br>
 <font color="GREEN" size="-1">13</font>
  नष्टायाम अपि वैदेह्यां हृतायाम अपि चानघ<br>
    शॊचितुं नार्हसे वीर यथान्यः पराकृतस तथा<br>
 <font color="GREEN" size="-1">14</font>
  तवद्विधा हि न शॊचन्ति सततं सत्यदर्शिनः<br>
    सुमहत्स्व अपि कृच्छ्रेषु रामानिर्विण्णदर्शणाः<br>
 <font color="GREEN" size="-1">15</font>
  तत्त्वतॊ हि नरश्रेष्ठ बुद्ध्या समनुचिन्तय<br>
    बुद्ध्या युक्ता महाप्राज्ञा विजानन्ति शुभाशुभे<br>
 <font color="GREEN" size="-1">16</font>
  अदृष्टगुणदॊषाणाम अधृतानां च कर्मणाम<br>
    नान्तरेण करियां तेषां फलम इष्टं परवर्तते<br>
 <font color="GREEN" size="-1">17</font>
  माम एव हि पुरा वीर तवम एव बहुषॊ ऽनवशाः<br>
    अनुशिष्याद धि कॊ नु तवाम अपि साक्षाद बृहस्पतिः<br>
 <font color="GREEN" size="-1">18</font>
  बुद्धिश च ते महाप्राज्ञ देवैर अपि दुरन्वया<br>
    शॊकेनाभिप्रसुप्तं ते जञानं संबॊधयाम्य अहम<br>
 <font color="GREEN" size="-1">19</font>
  दिव्यं च मानुषं चैवम आत्मनश च पराक्रमम<br>
    इक्ष्वाकुवृषभावेक्ष्य यतस्व दविषतां बधे<br>
 <font color="GREEN" size="-1">20</font>
  किं ते सर्वविनाशेन कृतेन पुरुषर्षभ<br>
    तम एव तु रिपुं पापं विज्ञायॊद्धर्तुम अर्हसि<br>
