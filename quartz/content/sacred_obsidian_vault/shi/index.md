 <p>
 This section has <a href="#shinto">scriptures of the Shinto religion</a>,
 as well as <a href="#gen">other texts</a>
 that relate to the spirituality of Japan, which combines Shinto, 
 <a href="../bud/index.htm">Buddhism</a>,
 <a href="../cfu/index.htm">Confucianism</a>,
 <a href="../sha/index.htm">Animism</a> and other themes.
 There is also a collection of <a href="ainu/index.htm">Ainu Texts</a>.
