 <h1 align="CENTER">86. at-Tariq: The Night-Comer</h1>
 <a name="an_086_001">1</a> By the heaven and the Morning Star 
 <a name="an_086_002">2</a>  - Ah, what will tell thee what the Morning Star is! 
 <a name="an_086_003">3</a> - The piercing Star! 
 <a name="an_086_004">4</a> No human soul but hath a guardian over it. 
 <a name="an_086_005">5</a> So let man consider from what he is created. 
 <a name="an_086_006">6</a> He is created from a gushing fluid 
 <a name="an_086_007">7</a> That issued from between the loins and ribs. 
 <a name="an_086_008">8</a> Lo! He verily is Able to return him (unto life) 
 <a name="an_086_009">9</a> On the day when hidden thoughts shall be searched out. 
 <a name="an_086_010">10</a> Then will he have no might nor any helper. 
 <a name="an_086_011">11</a> By the heaven which giveth the returning rain, 
 <a name="an_086_012">12</a> And the earth which splitteth (with the growth of trees and plants) 
 <a name="an_086_013">13</a> Lo! this (Qur'an) is a conclusive word, 
 <a name="an_086_014">14</a> It is no pleasantry. 
 <a name="an_086_015">15</a> Lo! they plot a plot (against thee, O Muhammad) 
 <a name="an_086_016">16</a> And I plot a plot (against them). 
 <a name="an_086_017">17</a> So give a respite to the disbelievers. Deal thou gently with them for a while.
