 <p align="CENTER"><font size="-1" color="GREEN"><a href="index.htm"><img src="img/tease.jpg"></a><br><i>The Pictorial Key to the Tarot</i>, by A.E. Waite, ill. by Pamela Colman Smith [1911], at sacred-texts.com</font></p><hr> <h3 align="CENTER">§ 5</h3> <h3 align="CENTER">THE RECURRENCE OF CARDS IN DEALING</h3> <h4 align="CENTER">In the Natural Position</h4> 
 <p>4 Kings = great honour; 3 Kings = consultation; 2 Kings = minor counsel.</p> 
 <p>4 Queens = great debate; 3 Queens = deception by women; 2 Queens = sincere friends.</p> 
 <p>4 Knights = serious matters; 3 Knights = lively debate; 2 Knights = intimacy.</p> 
 <p>4 Pages = dangerous illness; 3 Pages = dispute; 2 Pages = disquiet.</p> 
 <p>4 Tens = condemnation; 3 Tens = new condition; 2 Tens = change.</p> 
 <p>4 Nines = a good friend; 3 Nines = success; 2 Nines = receipt.</p> 
 <p>4 Eights = reverse; 3 Eights = marriage 2 Eights = new knowledge.</p> 
 <p>4 Sevens = intrigue; 3 Sevens = infirmity; 2 Sevens = news.</p> 
 <p>4 Sixes = abundance; 3 Sixes = success; 2 Sixes = irritability.</p> 
 <p>4 Fives = regularity; 3 Fives = determination; 2 Fives = vigils.</p> 
 <p>4 Fours = journey near at hand; 3 Fours = a subject of reflection; 2 Fours = insomnia.</p> 
 <p>4 Threes = progress; 3 Threes = unity 2 Threes = calm.</p> 
 <p>4 Twos = contention; 3 Twos = security; 2 Twos = accord.</p> 
 <p>4 Aces = favourable chance; 3 Aces = small success; 2 Aces = trickery.</p> <h4 align="CENTER">Reversed</h4> 
 <p>4 Kings = celerity; 3 Kings = commerce 2 Kings = projects.</p> 
 <p>4 Queens = bad company; 3 Queens = gluttony; 2 Queens = work.</p> 
 <p>4 Knights = alliance 3 Knights = a duel, or personal encounter; 2 Knights = susceptibility.</p> 
 <p>4 Pages = privation 3 Pages = idleness 2 Pages = society.</p> 
 <p>4 Tens = event, happening; 3 Tens disappointment; 2 Tens = expectation justified.</p> 
 <p>4 Nines = usury; 3 Nines imprudence; 2 Nines = a small profit.</p> 
 <p>4 Eights = error; 3 Eights a spectacle; 2 Eights = misfortune.</p> 
 <p>4 Sevens = quarrellers; 3 Sevens = joy; 2 Sevens = women of no repute.</p> 
 <p>4 Sixes = care; 3 Sixes = satisfaction 2 Sixes = downfall.</p> 
 <p>4 Fives = order; 3 Fives = hesitation; 2 Fives = reverse.</p> 
 <p>4 Fours = walks abroad; 3 Fours = disquiet; 2 Fours = dispute.</p> 
 <p>4 Threes = great success; 3 Threes = serenity; 2 Threes = safety.</p> 
 <p>4 Twos = reconciliation; 3 Twos apprehension; 2 Twos = mistrust.</p> 
 <p>4 Aces = dishonour; 3 Aces debauchery; 2 Aces = enemies.</p>
