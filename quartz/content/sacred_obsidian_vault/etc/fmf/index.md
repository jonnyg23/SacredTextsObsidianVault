 <h1 align="CENTER">Forty Modern Fables</h1>
 <h2 align="CENTER">by George Ade</h2>
 <h4 align="CENTER">[1901]</h4>
 <center>
 <a href="#contents">Contents</a>   
 <a href="fmf00.htm">Start Reading</a>
 </center>
 <hr>
 <p>
 This is a set of 'fractured fables' written on the cusp of the 20th century.
 Written in breezy vernacular style, these tales are timeless,
 even if some of the slang and cultural references have passed into
 obscurity in the intervening century.
 </p>
