<span class="margnote"><font color="GREEN" size="-1"><a name="an_10">10</a></font></span> Ther dwelte a kyng that werreyed Russye,<br> <a name="an_11"></a> Thurgh which ther dyde many a doughty man.<br> <a name="an_12"></a> This noble kyng was cleped Cambyuskan,<br> <a name="an_13"></a> Which in his tyme was of so greet renoun<br> <a name="an_14"></a> That ther was nowher in no regioun<br> <a name="an_15"></a> So excellent a lord in alle thyng:<br> <a name="an_16"></a> Hym lakked noght that longeth to a kyng.<br> <a name="an_17"></a> As of the secte of which that he was born<br> <a name="an_18"></a> He kepte his lay, to which that he was sworn;<br> <a name="an_19"></a> And therto he was hardy, wys, and riche,<br> <span class="margnote"><font color="GREEN" size="-1"><a <a name="an_26"></a> And kept alwey so wel roial estat<br> <a name="an_27"></a> That ther was nowher swich another man.<br> <a name="an_28"></a> This noble kyng, this Tartre Cambyuskan,<br> <a
