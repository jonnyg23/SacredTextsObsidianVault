 <p>
 This is a Potawatomi translation of the books of Matthew and Acts from
 the Bible.
 Potawatomi is a Native American language, a member of the Algonquian family.
 The Potawatomi originally lived in what is now southwestern Michigan.
 The documentation which accompanied the data for this Bible identifies
 the translator as Lykins, 1844.
 This text is in the public domain because it was published prior to 1923.
 </p>
