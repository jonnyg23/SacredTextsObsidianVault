  <h3 align="center">Aramaic Bible: 2 Peter</h3> 
 <p><a name="an_1:1">1:1</a> ܫܡܥܘܢ ܦܛܪܘܤ ܥܒܕܐ ܘܫܠܝܚܐ ܕܝܫܘܥ ܡܫܝܚܐ ܠܐܝܠܝܢ ܕܠܗܝܡܢܘܬܐ ܫܘܝܬ ܒܐܝܩܪܐ ܥܡܢ ܐܫܬܘܝܘ ܒܙܕܝܩܘܬܐ ܕܡܪܢ ܘܦܪܘܩܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:2">1:2</a> ܛܝܒܘܬܐ ܘܫܠܡܐ ܢܤܓܐ ܠܟܘܢ ܒܫܘܘܕܥܐ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:3">1:3</a> ܐܝܟ ܡܢ ܕܟܠܗܝܢ ܐܝܠܝܢ ܕܐܝܬܝܗܝܢ ܕܚܝܠܐ ܐܠܗܝܐ ܠܘܬ ܚܝܐ ܘܕܚܠܬ ܐܠܗܐ ܝܗܒ ܒܝܕ ܫܘܘܕܥܐ ܕܗܘ ܕܩܪܐ ܠܢ ܒܬܫܒܘܚܬܐ ܕܝܠܗ ܘܡܝܬܪܘܬܐ ܀</p> 
 <p><a name="an_1:4">1:4</a> ܕܒܐܝܕܝܗܝܢ ܫܘܘܕܝܐ ܪܘܪܒܐ ܘܝܩܝܪܐ ܠܟܘܢ ܝܗܒ ܕܒܝܕ ܗܠܝܢ ܬܗܘܘܢ ܫܘܬܦܐ ܕܟܝܢܐ ܐܠܗܝܐ ܟܕ ܥܪܩܝܢ ܐܢܬܘܢ ܡܢ ܚܒܠܐ ܕܪܓܝܓܬܐ ܕܒܥܠܡܐ ܀</p> 
 <p><a name="an_1:5">1:5</a> ܘܗܝ ܕܝܢ ܗܕܐ ܟܕ ܫܩܠܛܥܢܐ ܟܠܗ ܡܥܠܝܢ ܐܢܬܘܢ ܐܘܤܦܘ ܥܠ ܗܝܡܢܘܬܟܘܢ ܡܝܬܪܘܬܐ ܥܠ ܕܝܢ ܡܝܬܪܘܬܐ ܝܕܥܬܐ ܀</p> 
 <p><a name="an_1:6">1:6</a> ܥܠ ܕܝܢ ܝܕܥܬܐ ܡܚܡܤܢܢܘܬܐ ܥܠ ܕܝܢ ܡܚܡܤܢܢܘܬܐ ܡܤܝܒܪܢܘܬܐ ܥܠ ܕܝܢ ܡܤܝܒܪܢܘܬܐ ܕܚܠܬ ܐܠܗܐ ܀</p> 
 <p><a name="an_1:7">1:7</a> ܥܠ ܕܝܢ ܕܚܠܬ ܐܠܗܐ ܪܚܡܬ ܐܚܘܬܐ ܥܠ ܕܝܢ ܪܚܡܬ ܐܚܘܬܐ ܚܘܒܐ ܀</p> 
 <p><a name="an_1:8">1:8</a> ܗܠܝܢ ܓܝܪ ܟܕ ܫܟܝܚܢ ܠܟܘܢ ܘܝܬܝܪܢ ܠܐ ܗܘܐ ܒܛܝܠܐ ܐܦܠܐ ܕܠܐ ܦܐܪܐ ܡܩܝܡܢ ܠܟܘܢ ܒܫܘܘܕܥܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:9">1:9</a> ܗܘ ܓܝܪ ܕܠܐ ܫܟܝܚܢ ܠܗ ܗܠܝܢ ܤܡܝܐ ܐܝܬܘܗܝ ܕܠܐ ܚܙܐ ܕܛܥܐ ܕܘܟܝܐ ܕܚܛܗܘܗܝ ܩܕܡܝܐ ܀</p> 
 <p><a name="an_1:10">1:10</a> ܘܥܠ ܗܝ ܝܬܝܪܐܝܬ ܐܚܝ ܝܨܦܘ ܕܒܝܕ ܥܒܕܝܟܘܢ ܛܒܐ ܩܪܝܬܟܘܢ ܘܓܒܝܬܟܘܢ ܡܫܪܪܬܐ ܬܥܒܕܘܢ ܟܕ ܓܝܪ ܗܠܝܢ ܥܒܕܝܢ ܐܢܬܘܢ ܠܐ ܡܬܘܡ ܡܫܬܪܥܝܢ ܐܢܬܘܢ ܀</p> 
 <p><a name="an_1:11">1:11</a> ܗܟܢܐ ܓܝܪ ܥܬܝܪܐܝܬ ܡܬܝܗܒܐ ܠܟܘܢ ܡܥܠܬܐ ܕܡܠܟܘܬܐ ܕܠܥܠܡ ܕܡܪܢ ܘܦܪܘܩܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:12">1:12</a> ܘܥܠ ܗܕܐ ܠܐ ܡܡܐܢ ܐܢܐ ܡܢ ܕܡܥܗܕ ܐܢܐ ܠܟܘܢ ܐܡܝܢܐܝܬ ܥܠ ܗܠܝܢ ܟܕ ܛܒ ܐܦ ܝܕܥܝܢ ܐܢܬܘܢ ܘܤܡܝܟܝܢ ܐܢܬܘܢ ܥܠ ܫܪܪܐ ܗܢܐ ܀</p> 
 <p><a name="an_1:13">1:13</a> ܕܟܐܢܐ ܕܝܢ ܐܤܬܒܪܬ ܠܝ ܕܟܡܐ ܕܐܝܬܝ ܒܦܓܪܐ ܗܢܐ ܐܥܝܪܟܘܢ ܒܥܘܗܕܢܐ ܀</p> 
 <p><a name="an_1:14">1:14</a> ܟܕ ܝܕܥ ܐܢܐ ܕܥܘܢܕܢܐ ܕܦܓܪܝ ܒܥܓܠ ܗܘܐ ܐܝܟܢܐ ܕܐܦ ܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܐܘܕܥܢܝ ܀</p> 
 <p><a name="an_1:15">1:15</a> ܝܨܦ ܐܢܐ ܕܝܢ ܕܐܦ ܐܡܝܢܐܝܬ ܗܘܐ ܐܝܬ ܠܟܘܢ ܕܐܦ ܡܢ ܒܬܪ ܡܦܩܢܐ ܕܝܠܝ ܥܘܗܕܢܐ ܕܗܠܝܢ ܬܗܘܘܢ ܥܒܕܝܢ ܀</p> 
 <p><a name="an_1:16">1:16</a> ܠܐ ܗܘܐ ܓܝܪ ܟܕ ܒܬܪ ܡܬܠܐ ܕܥܒܝܕܝܢ ܒܐܘܡܢܘܬܐ ܐܙܠܝܢܢ ܐܘܕܥܢܟܘܢ ܚܝܠܗ ܘܡܐܬܝܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܐܠܐ ܟܕ ܚܙܝܐ ܗܘܝܢ ܕܪܒܘܬܐ ܕܝܠܗ ܀</p> 
 <p><a name="an_1:17">1:17</a> ܟܕ ܓܝܪ ܢܤܒ ܡܢ ܐܠܗܐ ܐܒܐ ܐܝܩܪܐ ܘܬܫܒܘܚܬܐ ܟܕ ܩܠܐ ܐܬܐ ܠܗ ܕܐܝܟ ܗܢܐ ܡܢ ܬܫܒܘܚܬܐ ܦܐܝܬ ܒܪܒܘܬܗ ܕܗܢܘ ܒܪܝ ܗܘ ܚܒܝܒܐ ܗܘ ܕܒܗ ܐܨܛܒܝܬ ܀</p> 
 <p><a name="an_1:18">1:18</a> ܐܦ ܚܢܢ ܠܗ ܠܗܢܐ ܩܠܐ ܫܡܥܢܢ ܡܢ ܫܡܝܐ ܕܐܬܐ ܠܗ ܟܕ ܐܝܬܝܢ ܗܘܝܢ ܥܡܗ ܒܛܘܪܐ ܩܕܝܫܐ ܀</p> 
 <p><a name="an_1:19">1:19</a> ܘܐܝܬ ܠܢ ܕܫܪܝܪܐ ܐܦ ܡܠܬܐ ܕܢܒܝܘܬܐ ܗܝ ܕܫܦܝܪ ܥܒܕܝܢ ܐܢܬܘܢ ܟܕ ܒܗ ܚܝܪܝܢ ܐܢܬܘܢ ܐܝܟ ܕܒܫܪܓܐ ܕܡܢܗܪ ܒܐܬܪܐ ܥܡܘܛܐ ܥܕܡܐ ܕܐܝܡܡܐ ܢܢܗܪ ܘܫܡܫܐ ܢܕܢܚ ܒܠܒܘܬܟܘܢ ܀</p> 
 <p><a name="an_1:20">1:20</a> ܟܕ ܗܕܐ ܠܘܩܕܡ ܝܕܥܝܢ ܐܢܬܘܢ ܕܟܠ ܢܒܝܘܬܐ ܫܪܝܐ ܕܟܬܒܐ ܕܝܠܗ ܠܐ ܗܘܝܐ ܀</p> 
 <p><a name="an_1:21">1:21</a> ܠܐ ܗܘܐ ܓܝܪ ܒܨܒܝܢܐ ܕܒܪܢܫܐ ܐܬܬ ܡܢ ܡܬܘܡ ܢܒܝܘܬܐ ܐܠܐ ܟܕ ܡܢ ܪܘܚܐ ܩܕܝܫܐ ܡܬܢܓܕܝܢ ܡܠܠܘ ܩܕܝܫܐ ܒܢܝܢܫܐ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_2:1">2:1</a> ܗܘܘ ܕܝܢ ܐܦ ܢܒܝܐ ܕܓܠܐ ܒܥܡܐ ܐܝܟ ܕܐܦ ܒܟܘܢ ܢܗܘܘܢ ܡܠܦܢܐ ܕܓܠܐ ܐܝܠܝܢ ܕܡܥܠܝܢ ܗܪܤܝܤ ܕܐܒܕܢܐ ܘܒܡܪܐ ܕܙܒܢ ܐܢܘܢ ܟܦܪܝܢ ܟܕ ܡܝܬܝܢ ܥܠ ܢܦܫܗܘܢ ܐܒܕܢܐ ܡܤܪܗܒܐ ܀</p> 
 <p><a name="an_2:2">2:2</a> ܘܤܓܝܐܐ ܢܐܙܠܘܢ ܒܬܪ ܛܢܦܘܬܐ ܕܝܠܗܘܢ ܗܠܝܢ ܕܡܛܠܬܗܘܢ ܐܘܪܚܐ ܕܫܪܪܐ ܬܬܓܕܦ ܀</p> 
 <p><a name="an_2:3">2:3</a> ܘܒܥܠܘܒܘܬܐ ܘܒܡܠܐ ܕܒܕܝܐ ܢܬܬܓܪܘܢ ܒܟܘܢ ܗܠܝܢ ܕܕܝܢܗܘܢ ܡܢ ܩܕܝܡ ܠܐ ܒܛܠ ܘܐܒܕܢܗܘܢ ܠܐ ܢܐܡ ܀</p> 
 <p><a name="an_2:4">2:4</a> ܐܢ ܓܝܪ ܐܠܗܐ ܥܠ ܡܠܐܟܐ ܕܚܛܘ ܠܐ ܚܤ ܐܠܐ ܒܫܫܠܬܐ ܕܥܡܛܢܐ ܥܓܢ ܐܢܘܢ ܒܬܚܬܝܬܐ ܘܐܫܠܡ ܐܢܘܢ ܕܢܬܢܛܪܘܢ ܠܕܝܢܐ ܕܫܘܢܩܐ ܀</p> 
 <p><a name="an_2:5">2:5</a> ܘܥܠ ܥܠܡܐ ܩܕܡܝܐ ܠܐ ܚܤ ܐܠܐ ܠܢܘܚ ܕܬܡܢܝܐ ܟܪܘܙܐ ܕܟܐܢܘܬܐ ܢܛܪ ܟܕ ܛܘܦܢܐ ܥܠ ܥܠܡܐ ܕܪܫܝܥܐ ܐܝܬܝ ܀</p> 
 <p><a name="an_2:6">2:6</a> ܘܟܕ ܠܡܕܝܢܬܐ ܕܤܕܘܡ ܘܕܥܡܘܪܐ ܐܘܩܕ ܘܒܗܦܘܟܝܐ ܚܝܒ ܐܢܝܢ ܟܕ ܬܚܘܝܬܐ ܠܪܫܝܥܐ ܕܥܬܝܕܝܢ ܕܢܗܘܘܢ ܤܡ ܀</p> 
 <p><a name="an_2:7">2:7</a> ܐܦ ܠܠܘܛ ܙܕܝܩܐ ܕܡܬܩܦܚ ܗܘܐ ܡܢ ܗܘܦܟܐ ܕܒܛܢܦܘܬܐ ܕܗܠܝܢ ܕܕܠܐ ܢܡܘܤܐ ܐܝܬܝܗܘܢ ܗܘܘ ܦܨܝ ܀</p> 
 <p><a name="an_2:8">2:8</a> ܒܚܙܬܐ ܓܝܪ ܘܒܫܡܥܐ ܟܕ ܥܡܪ ܗܘܐ ܟܐܢܐ ܗܘ ܒܝܢܬܗܘܢ ܝܘܡ ܡܢ ܝܘܡ ܠܢܦܫܗ ܙܕܝܩܬܐ ܒܥܒܕܐ ܕܠܐ ܢܡܘܤ ܡܫܢܩ ܗܘܐ ܀</p> 
 <p><a name="an_2:9">2:9</a> ܝܕܥ ܗܘ ܡܪܝܐ ܕܢܦܪܘܩ ܡܢ ܐܘܠܨܢܐ ܠܐܝܠܝܢ ܕܕܚܠܝܢ ܠܗ ܠܥܘܠܐ ܕܝܢ ܠܝܘܡܐ ܕܕܝܢܐ ܟܕ ܡܫܬܢܩܝܢ ܢܛܪ ܀</p> 
 <p><a name="an_2:10">2:10</a> ܝܬܝܪܐܝܬ ܕܝܢ ܠܐܝܠܝܢ ܕܒܬܪ ܒܤܪܐ ܒܪܓܬܐ ܕܛܡܐܘܬܐ ܐܙܠܝܢ ܘܥܠ ܡܪܘܬܐ ܡܒܤܪܝܢ ܡܪܚܐ ܘܡܫܩܠܐ ܕܡܢ ܬܫܒܘܚܬܐ ܠܐ ܙܝܥܝܢ ܟܕ ܡܓܕܦܝܢ ܀</p> 
 <p><a name="an_2:11">2:11</a> ܐܝܟܐ ܕܡܠܐܟܐ ܕܒܚܝܠܐ ܘܒܥܘܫܢܐ ܪܘܪܒܝܢ ܡܢܗܘܢ ܠܐ ܡܝܬܝܢ ܥܠܝܗܘܢ ܡܢ ܡܪܝܐ ܕܝܢܐ ܕܓܘܕܦܐ ܀</p> 
 <p><a name="an_2:12">2:12</a> ܗܠܝܢ ܕܝܢ ܕܐܝܟ ܚܝܘܬܐ ܚܪܫܬܐ ܗܘܘ ܒܟܝܢܐ ܠܚܪܒܐ ܘܠܚܒܠܐ ܟܕ ܒܐܝܠܝܢ ܕܠܐ ܝܕܥܝܢ ܡܓܕܦܝܢ ܒܚܒܠܐ ܕܝܠܗܘܢ ܢܬܚܒܠܘܢ ܀</p> 
 <p><a name="an_2:13">2:13</a> ܟܕ ܐܝܠܝܢ ܕܗܘܐ ܒܗܘܢ ܥܘܠܐ ܐܓܪܐ ܕܥܘܠܐ ܗܢܝܘܬܐ ܚܫܝܒ ܠܗܘܢ ܒܘܤܡܐ ܕܗܘܐ ܒܐܝܡܡܐ ܡܟܬܡܐ ܘܡܠܝܝ ܡܘܡܐ ܕܟܕ ܡܬܒܤܡܝܢ ܒܢܝܚܬܗܘܢ ܡܬܦܢܩܝܢ ܀</p> 
 <p><a name="an_2:14">2:14</a> ܟܕ ܥܝܢܐ ܐܝܬ ܠܗܘܢ ܕܡܠܝܢ ܓܘܪܐ ܘܚܛܗܐ ܕܠܐ ܡܘܦܝܢ ܟܕ ܡܫܕܠܝܢ ܠܢܦܫܬܐ ܕܠܐ ܤܡܝܟܢ ܘܠܒܐ ܕܡܕܪܫ ܒܥܠܘܒܘܬܐ ܐܝܬ ܠܗܘܢ ܒܢܝܐ ܕܠܘܛܬܐ ܀</p> 
 <p><a name="an_2:15">2:15</a> ܕܟܕ ܫܒܩܘ ܐܘܪܚܐ ܬܪܝܨܬܐ ܫܓܘ ܘܐܙܠܘ ܒܐܘܪܚܐ ܕܒܠܥܡ ܒܪ ܒܥܘܪ ܗܘ ܕܐܓܪܐ ܕܥܘܠܐ ܐܚܒ ܀</p> 
 <p><a name="an_2:16">2:16</a> ܡܟܤܢܘܬܐ ܕܝܢ ܕܗܘܬ ܠܗ ܠܡܬܥܒܪܢܘܬܗ ܐܬܢܐ ܕܠܐ ܩܠܐ ܕܒܩܠܐ ܕܒܢܝܢܫܐ ܡܠܠܬ ܟܠܬ ܠܫܛܝܘܬܗ ܕܢܒܝܐ ܀</p> 
 <p><a name="an_2:17">2:17</a> ܗܠܝܢ ܐܢܘܢ ܡܥܝܢܐ ܕܠܐ ܡܝܐ ܥܢܢܐ ܕܡܢ ܥܠܥܠܐ ܡܬܪܕܦܢ ܗܠܝܢ ܕܥܡܛܢܐ ܕܚܫܘܟܐ ܢܛܝܪ ܠܗܘܢ ܀</p> 
 <p><a name="an_2:18">2:18</a> ܟܕ ܓܝܪ ܓܘܢܚܐ ܕܤܪܝܩܘܬܐ ܡܡܠܠܝܢ ܡܫܕܠܝܢ ܒܪܓܝܓܬܐ ܛܢܦܬܐ ܕܒܤܪܐ ܠܐܝܠܝܢ ܕܒܡܠܐ ܩܠܝܠ ܥܪܩܝܢ ܡܢ ܐܝܠܝܢ ܕܒܛܥܝܘܬܐ ܡܬܗܦܟܝܢ ܀</p> 
 <p><a name="an_2:19">2:19</a> ܘܚܐܪܘܬܐ ܠܗܘܢ ܡܫܬܘܕܝܢ ܟܕ ܗܢܘܢ ܥܒܕܐ ܐܝܬܝܗܘܢ ܕܚܒܠܐ ܠܗܘ ܓܝܪ ܡܕܡ ܕܐܢܫ ܙܟܐ ܠܗ ܠܗܢܐ ܐܦ ܡܫܥܒܕ ܀</p> 
 <p><a name="an_2:20">2:20</a> ܐܢ ܓܝܪ ܟܕ ܥܪܩܘ ܡܢ ܛܢܦܘܬܗ ܕܥܠܡܐ ܒܫܘܘܕܥܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܘܦܪܘܩܢ ܒܗܝܢ ܒܗܠܝܢ ܟܕ ܡܬܥܪܙܠܝܢ ܬܘܒ ܡܙܕܟܝܢ ܗܘܬ ܠܗ ܚܪܬܗܘܢ ܕܒܝܫܐ ܡܢ ܩܕܡܝܬܐ ܀</p> 
 <p><a name="an_2:21">2:21</a> ܦܩܚ ܗܘܐ ܠܗܘܢ ܓܝܪ ܕܠܐ ܢܫܬܘܕܥܘܢ ܠܐܘܪܚܐ ܕܙܕܝܩܘܬܐ ܐܘ ܕܟܕ ܐܫܬܘܕܥܘ ܠܒܬܪܐ ܢܗܦܟܘܢ ܡܢ ܦܘܩܕܢܐ ܩܕܝܫܐ ܕܐܫܬܠܡ ܠܗܘܢ ܀</p> 
 <p><a name="an_2:22">2:22</a> ܓܕܫ ܠܗܘܢ ܕܝܢ ܗܠܝܢ ܕܡܬܠܐ ܫܪܝܪܐ ܕܟܠܒܐ ܕܗܦܟ ܥܠ ܬܝܘܒܗ ܘܚܙܝܪܬܐ ܕܤܚܬ ܒܥܘܪܓܠܐ ܕܤܝܢܐ ܀</p> 
 <p><a name="an_3:1">3:1</a> ܗܕܐ ܡܢ ܟܕܘ ܚܒܝܒܝ ܐܓܪܬܐ ܕܬܪܬܝܢ ܟܬܒ ܐܢܐ ܠܟܘܢ ܗܠܝܢ ܕܒܗܝܢ ܡܥܝܪ ܐܢܐ ܒܥܘܗܕܢܐ ܠܪܥܝܢܟܘܢ ܫܦܝܐ ܀</p> 
 <p><a name="an_3:2">3:2</a> ܕܬܬܥܗܕܘܢ ܠܡܠܐ ܕܩܕܡ ܐܬܐܡܪ ܡܢ ܢܒܝܐ ܩܕܝܫܐ ܘܠܦܘܩܕܢܗ ܕܡܪܢ ܘܦܪܘܩܢ ܕܒܝܕ ܫܠܝܚܐ ܀</p> 
 <p><a name="an_3:3">3:3</a> ܟܕ ܗܕܐ ܠܘܩܕܡ ܝܕܥܝܢ ܐܢܬܘܢ ܕܢܐܬܘܢ ܒܚܪܬܐ ܕܝܘܡܬܐ ܡܒܙܚܢܐ ܕܡܒܙܚܝܢ ܟܕ ܐܝܟ ܪܓܝܓܬܐ ܕܝܠܗܘܢ ܡܗܠܟܝܢ ܀</p> 
 <p><a name="an_3:4">3:4</a> ܘܐܡܪܝܢ ܕܐܝܟܐ ܐܝܬܘܗܝ ܡܘܠܟܢܐ ܕܡܐܬܝܬܗ ܡܢ ܟܕ ܓܝܪ ܐܒܗܬܢ ܫܟܒܘ ܟܠܡܕܡ ܗܟܢܐ ܡܟܬܪ ܡܢ ܫܘܪܝܗ ܕܒܪܝܬܐ ܀</p> 
 <p><a name="an_3:5">3:5</a> ܛܥܝܐ ܠܗܘܢ ܓܝܪ ܗܕܐ ܟܕ ܨܒܝܢ ܕܫܡܝܐ ܐܝܬܝܗܘܢ ܗܘܘ ܡܢ ܩܕܝܡ ܘܐܪܥܐ ܡܢ ܡܝܐ ܘܒܝܕ ܡܝܐ ܩܡܬ ܒܡܠܬܐ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_3:6">3:6</a> ܗܠܝܢ ܕܒܐܝܕܝܗܘܢ ܥܠܡܐ ܕܗܝܕܝܢ ܛܦ ܒܡܝܐ ܘܐܒܕ ܀</p> 
 <p><a name="an_3:7">3:7</a> ܫܡܝܐ ܕܝܢ ܕܗܫܐ ܘܐܪܥܐ ܒܡܠܬܐ ܕܝܠܗ ܐܤܝܢܝܢ ܟܕ ܠܢܘܪܐ ܡܬܢܛܪܝܢ ܠܝܘܡܐ ܕܕܝܢܐ ܘܕܐܒܕܢܐ ܕܒܢܝܢܫܐ ܪܫܝܥܐ ܀</p> 
 <p><a name="an_3:8">3:8</a> ܗܕܐ ܕܝܢ ܚܕܐ ܠܐ ܬܛܥܝܟܘܢ ܚܒܝܒܝ ܕܚܕ ܝܘܡܐ ܠܡܪܝܐ ܐܝܟ ܐܠܦ ܫܢܝܢ ܐܝܬܘܗܝ ܘܐܠܦ ܫܢܝܢ ܐܝܟ ܝܘܡܐ ܚܕ ܀</p> 
 <p><a name="an_3:9">3:9</a> ܠܐ ܡܫܬܘܚܪ ܡܪܝܐ ܒܡܘܠܟܢܘܗܝ ܐܝܟ ܕܐܢܫܝܢ ܫܘܘܚܪܐ ܡܤܒܪܝܢ ܐܠܐ ܡܓܪ ܪܘܚܗ ܡܛܠܬܟܘܢ ܒܕܠܐ ܨܒܐ ܕܐܢܫ ܢܐܒܕ ܐܠܐ ܟܠܢܫ ܠܬܝܒܘܬܐ ܢܐܬܐ ܀</p> 
 <p><a name="an_3:10">3:10</a> ܐܬܐ ܕܝܢ ܝܘܡܗ ܕܡܪܝܐ ܐܝܟ ܓܢܒܐ ܗܘ ܕܒܗ ܫܡܝܐ ܡܢ ܫܠܝ ܥܒܪܝܢ ܐܤܛܘܟܤܐ ܕܝܢ ܟܕ ܝܩܕܝܢ ܢܫܬܪܘܢ ܘܐܪܥܐ ܘܥܒܕܐ ܕܒܗ ܬܫܬܟܚ ܀</p> 
 <p><a name="an_3:11">3:11</a> ܟܕ ܗܟܝܠ ܗܠܝܢ ܟܠܗܘܢ ܡܫܬܪܝܢ ܕܐܝܟ ܐܝܠܝܢ ܙܕܩ ܠܟܘܢ ܕܬܗܘܘܢ ܒܗܘܦܟܝܟܘܢ ܩܕܝܫܐ ܘܒܕܚܠܬ ܐܠܗܐ ܀</p> 
 <p><a name="an_3:12">3:12</a> ܟܕ ܡܤܟܝܢ ܐܢܬܘܢ ܘܤܘܚܝܢ ܐܢܬܘܢ ܠܡܐܬܝܬܐ ܕܝܘܡܗ ܕܐܠܗܐ ܗܘ ܕܒܗ ܫܡܝܐ ܟܕ ܡܬܒܚܪܝܢ ܒܢܘܪܐ ܢܫܬܪܘܢ ܘܐܤܛܘܟܤܐ ܟܕ ܝܩܕܝܢ ܢܫܘܚܘܢ ܀</p> 
 <p><a name="an_3:13">3:13</a> ܠܫܡܝܐ ܕܝܢ ܚܕܬܐ ܘܐܪܥܐ ܚܕܬܐ ܐܝܟ ܡܘܠܟܢܐ ܕܝܠܗ ܡܤܟܝܢܢ ܗܠܝܢ ܕܒܗܘܢ ܙܕܝܩܘܬܐ ܥܡܪܐ ܀</p> 
 <p><a name="an_3:14">3:14</a> ܡܛܠ ܗܢܐ ܚܒܝܒܝ ܟܕ ܠܗܠܝܢ ܡܤܟܝܢ ܐܢܬܘܢ ܝܨܦܘ ܕܕܠܐ ܟܘܬܡܐ ܘܕܠܐ ܡܘܡܐ ܠܗ ܬܫܬܟܚܘܢ ܒܫܠܡܐ ܀</p> 
 <p><a name="an_3:15">3:15</a> ܘܠܡܓܪܬ ܪܘܚܗ ܕܡܪܝܐ ܦܘܪܩܢܐ ܬܚܫܒܘܢ ܐܝܟܢܐ ܕܐܦ ܐܚܘܢ ܚܒܝܒܐ ܦܘܠܘܤ ܐܝܟ ܚܟܡܬܐ ܕܐܬܝܗܒܬ ܠܗ ܟܬܒ ܠܟܘܢ ܀</p> 
 <p><a name="an_3:16">3:16</a> ܐܝܟܢܐ ܕܒܟܠܗܝܢ ܐܓܪܬܗ ܡܠܠ ܒܗܝܢ ܥܠ ܗܠܝܢ ܐܝܠܝܢ ܕܐܝܬ ܒܗܝܢ ܡܕܡ ܕܥܤܝܩ ܠܤܘܟܠܐ ܗܠܝܢ ܕܐܝܠܝܢ ܕܕܠܐ ܝܘܠܦܢܐ ܐܢܘܢ ܘܠܐ ܤܡܝܟܝܢ ܡܥܩܡܝܢ ܠܗܝܢ ܐܝܟ ܕܐܦ ܠܗܠܝܢ ܟܬܒܐ ܕܫܪܟܐ ܠܘܬ ܐܒܕܢܐ ܕܝܠܗܘܢ ܀</p> 
 <p><a name="an_3:17">3:17</a> ܐܢܬܘܢ ܗܟܝܠ ܚܒܝܒܝ ܟܕ ܩܕܡܝܢ ܐܢܬܘܢ ܝܕܥܝܢ ܛܪܘ ܢܦܫܟܘܢ ܕܕܠܡܐ ܟܕ ܐܙܠܝܢ ܐܢܬܘܢ ܒܬܪ ܛܥܝܘܬܐ ܕܐܝܠܝܢ ܕܕܠܐ ܢܡܘܤ ܐܢܘܢ ܬܦܠܘܢ ܡܢ ܤܡܟܐ ܕܝܠܟܘܢ ܀</p> 
 <p><a name="an_3:18">3:18</a> ܗܘܝܬܘܢ ܕܝܢ ܡܬܪܒܝܢ ܒܛܝܒܘܬܐ ܘܒܝܕܥܬܐ ܕܡܪܢ ܘܦܪܘܩܢ ܝܫܘܥ ܡܫܝܚܐ ܘܕܐܠܗܐ ܐܒܐ ܕܠܗ ܬܫܒܘܚܬܐ ܐܦ ܗܫܐ ܘܒܟܠܙܒܢ ܘܠܝܘܡܝ ܥܠܡܐ ܐܡܝܢ ܀</p>
