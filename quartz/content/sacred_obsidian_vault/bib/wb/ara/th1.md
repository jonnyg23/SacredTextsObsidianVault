  <h3 align="center">Aramaic Bible: 1 Thessalonians</h3> 
 <p><a name="an_1:1">1:1</a> ܦܘܠܘܤ ܘܤܠܘܢܘܤ ܘܛܝܡܬܐܤ ܠܥܕܬܐ ܕܬܤܠܘܢܝܩܝܐ ܕܒܐܠܗܐ ܐܒܐ ܘܒܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܛܝܒܘܬܐ ܥܡܟܘܢ ܘܫܠܡܐ ܀</p> 
 <p><a name="an_1:2">1:2</a> ܡܘܕܝܢܢ ܠܐܠܗܐ ܒܟܠܙܒܢ ܥܠ ܟܠܟܘܢ ܘܡܬܕܟܪܝܢܢ ܠܟܘܢ ܒܨܠܘܬܢ ܐܡܝܢܐܝܬ ܀</p> 
 <p><a name="an_1:3">1:3</a> ܘܥܗܕܝܢܢ ܩܕܡ ܐܠܗܐ ܐܒܐ ܥܒܕܐ ܕܗܝܡܢܘܬܟܘܢ ܘܥܡܠܐ ܕܚܘܒܟܘܢ ܘܡܤܝܒܪܢܘܬܐ ܕܤܒܪܟܘܢ ܕܒܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:4">1:4</a> ܝܕܥܝܢܢ ܓܝܪ ܓܒܝܘܬܟܘܢ ܐܚܝ ܚܒܝܒܘܗܝ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_1:5">1:5</a> ܡܛܠ ܕܡܤܒܪܢܘܬܢ ܠܐ ܗܘܐ ܒܡܠܐ ܒܠܚܘܕ ܗܘܬ ܠܘܬܟܘܢ ܐܠܐ ܐܦ ܒܚܝܠܐ ܘܒܪܘܚܐ ܕܩܘܕܫܐ ܘܒܦܝܤܐ ܫܪܝܪܐ ܐܦ ܐܢܬܘܢ ܝܕܥܝܢ ܐܢܬܘܢ ܐܝܟܢܐ ܗܘܝܢ ܒܝܢܬܟܘܢ ܡܛܠܬܟܘܢ ܀</p> 
 <p><a name="an_1:6">1:6</a> ܘܐܢܬܘܢ ܒܢ ܐܬܕܡܝܬܘܢ ܘܒܡܪܢ ܕܩܒܠܬܘܢ ܡܠܬܐ ܒܐܘܠܨܢܐ ܪܒܐ ܘܒܚܕܘܬܐ ܕܪܘܚܐ ܕܩܘܕܫܐ ܀</p> 
 <p><a name="an_1:7">1:7</a> ܘܗܘܝܬܘܢ ܕܡܘܬܐ ܠܟܠܗܘܢ ܡܗܝܡܢܐ ܕܐܝܬ ܒܡܩܕܘܢܝܐ ܘܒܐܟܐܝܐ ܀</p> 
 <p><a name="an_1:8">1:8</a> ܡܢܟܘܢ ܓܝܪ ܐܫܬܡܥܬ ܡܠܬܗ ܕܡܪܢ ܠܐ ܒܠܚܘܕ ܒܡܩܕܘܢܝܐ ܘܒܐܟܐܝܐ ܐܠܐ ܒܟܠ ܐܬܪ ܗܝܡܢܘܬܟܘܢ ܕܒܐܠܗܐ ܐܫܬܡܥܬ ܐܝܟܢܐ ܕܠܐ ܢܤܬܢܩ ܕܢܐܡܪ ܥܠܝܟܘܢ ܡܕܡ ܀</p> 
 <p><a name="an_1:9">1:9</a> ܗܢܘܢ ܓܝܪ ܡܫܬܥܝܢ ܐܝܢܐ ܡܥܠܢܐ ܗܘܐ ܠܢ ܠܘܬܟܘܢ ܘܐܝܟܢܐ ܐܬܦܢܝܬܘܢ ܠܘܬ ܐܠܗܐ ܡܢ ܕܚܠܬ ܦܬܟܪܐ ܕܬܦܠܚܘܢ ܠܐܠܗܐ ܚܝܐ ܘܫܪܝܪܐ ܀</p> 
 <p><a name="an_1:10">1:10</a> ܟܕ ܡܤܟܝܬܘܢ ܠܒܪܗ ܡܢ ܫܡܝܐ ܠܝܫܘܥ ܗܘ ܕܐܩܝܡ ܡܢ ܒܝܬ ܡܝܬܐ ܕܗܘ ܡܦܨܐ ܠܢ ܡܢ ܪܘܓܙܐ ܕܐܬܐ ܀</p> 
 <p><a name="an_2:1">2:1</a> ܘܐܢܬܘܢ ܝܕܥܝܢ ܐܢܬܘܢ ܐܚܝ ܕܡܥܠܢܢ ܕܠܘܬܟܘܢ ܠܐ ܗܘܐ ܤܪܝܩܐܝܬ ܗܘܐ ܀</p> 
 <p><a name="an_2:2">2:2</a> ܐܠܐ ܠܘܩܕܡ ܚܫܢܢ ܘܐܨܛܥܪܢܢ ܐܝܟ ܕܝܕܥܝܬܘܢ ܒܦܝܠܝܦܘܤ ܘܗܝܕܝܢ ܒܐܓܘܢܐ ܪܒܐ ܡܠܠܢ ܥܡܟܘܢ ܒܦܪܗܤܝܐ ܕܐܠܗܢ ܤܒܪܬܗ ܕܡܫܝܚܐ ܀</p> 
 <p><a name="an_2:3">2:3</a> ܒܘܝܐܢ ܓܝܪ ܠܐ ܗܘܐ ܡܢ ܛܥܝܘܬܐ ܐܦܠܐ ܡܢ ܛܢܦܘܬܐ ܐܦܠܐ ܒܢܟܠܐ ܀</p> 
 <p><a name="an_2:4">2:4</a> ܐܠܐ ܐܝܟ ܡܐ ܕܒܩܝܢܢ ܠܐܠܗܐ ܕܬܬܗܝܡܢ ܤܒܪܬܗ ܗܟܢܐ ܡܡܠܠܝܢܢ ܠܐ ܐܝܟ ܕܠܒܢܝ ܐܢܫܐ ܢܫܦܪ ܐܠܐ ܠܐܠܗܐ ܗܘ ܕܒܩܐ ܠܒܘܬܢ ܀</p> 
 <p><a name="an_2:5">2:5</a> ܐܦܠܐ ܓܝܪ ܡܡܬܘܡ ܐܬܚܫܚܢ ܒܡܡܠܠܐ ܫܕܠܐ ܐܝܟ ܕܝܕܥܝܢ ܐܢܬܘܢ ܘܠܐ ܒܥܠܬܐ ܕܝܥܢܘܬܐ ܐܠܗܐ ܤܗܕ ܀</p> 
 <p><a name="an_2:6">2:6</a> ܘܠܐ ܒܥܝܢ ܫܘܒܚܐ ܡܢ ܒܢܝܢܫܐ ܠܐ ܡܢܟܘܢ ܘܠܐ ܡܢ ܐܚܪܢܐ ܟܕ ܡܫܟܚܝܢ ܗܘܝܢ ܝܩܝܪܐ ܠܡܗܘܐ ܐܝܟ ܫܠܝܚܐ ܕܡܫܝܚܐ ܀</p> 
 <p><a name="an_2:7">2:7</a> ܐܠܐ ܗܘܝܢ ܒܝܢܬܟܘܢ ܡܟܝܟܐ ܘܐܝܟ ܡܪܒܝܢܝܬܐ ܕܡܚܒܒܐ ܒܢܝܗ ܀</p> 
 <p><a name="an_2:8">2:8</a> ܗܟܢܐ ܐܦ ܚܢܢ ܡܚܒܒܝܢܢ ܘܡܤܘܚܝܢܢ ܠܡܬܠ ܠܟܘܢ ܠܐ ܒܠܚܘܕ ܤܒܪܬܗ ܕܐܠܗܐ ܐܠܐ ܐܦ ܢܦܫܢ ܡܛܠ ܕܚܒܝܒܝܢ ܐܢܬܘܢ ܀</p> 
 <p><a name="an_2:9">2:9</a> ܥܗܕܝܢ ܐܢܬܘܢ ܓܝܪ ܐܚܝܢ ܕܠܐܝܢ ܗܘܝܢ ܘܥܡܠܝܢ ܒܥܒܕ ܐܝܕܝܢ ܒܠܠܝܐ ܘܒܐܝܡܡܐ ܕܥܠ ܚܕ ܡܢܟܘܢ ܠܐ ܢܐܩܪ ܀</p> 
 <p><a name="an_2:10">2:10</a> ܐܢܬܘܢ ܤܗܕܝܢ ܘܐܠܗܐ ܐܝܟܢܐ ܐܟܪܙܢ ܠܟܘܢ ܤܒܪܬܗ ܕܐܠܗܐ ܕܟܝܐܝܬ ܘܟܐܢܐܝܬ ܘܕܠܐ ܪܫܝܢ ܗܘܝܢ ܠܘܬ ܟܠܗܘܢ ܡܗܝܡܢܐ ܀</p> 
 <p><a name="an_2:11">2:11</a> ܐܝܟܢܐ ܕܝܕܥܝܢ ܐܢܬܘܢ ܕܡܢ ܚܕ ܚܕ ܡܢܟܘܢ ܐܝܟ ܐܒܐ ܡܢ ܒܢܘܗܝ ܒܥܝܢ ܗܘܝܢ ܘܡܠܝܢ ܗܘܝܢ ܒܠܒܟܘܢ ܘܡܤܗܕܝܢܢ ܠܟܘܢ ܀</p> 
 <p><a name="an_2:12">2:12</a> ܕܬܗܠܟܘܢ ܐܝܟ ܕܝܐܐ ܠܐܠܗܐ ܗܘ ܕܩܪܟܘܢ ܠܡܠܟܘܬܗ ܘܠܫܘܒܚܗ ܀</p> 
 <p><a name="an_2:13">2:13</a> ܡܛܠ ܗܢܐ ܐܦ ܚܢܢ ܡܘܕܝܢܢ ܐܡܝܢܐܝܬ ܠܐܠܗܐ ܕܡܠܬܐ ܕܐܠܗܐ ܕܩܒܠܬܘܢ ܡܢܢ ܠܐ ܗܘܐ ܐܝܟ ܡܠܬܐ ܕܒܢܝܢܫܐ ܩܒܠܬܘܢܗ ܐܠܐ ܐܝܟܢܐ ܕܐܝܬܝܗ ܫܪܝܪܐܝܬ ܡܠܬܐ ܕܐܠܗܐ ܘܗܝ ܒܥܒܕܐ ܡܤܬܥܪܐ ܒܟܘܢ ܒܐܝܠܝܢ ܕܡܗܝܡܢܝܢ ܀</p> 
 <p><a name="an_2:14">2:14</a> ܐܢܬܘܢ ܕܝܢ ܐܚܝ ܐܬܕܡܝܬܘܢ ܠܥܕܬܐ ܕܐܠܗܐ ܕܒܝܗܘܕ ܗܠܝܢ ܕܒܝܫܘܥ ܡܫܝܚܐ ܕܗܟܢ ܤܝܒܪܬܘܢ ܐܦ ܐܢܬܘܢ ܡܢ ܒܢܝ ܫܪܒܬܟܘܢ ܐܝܟܢܐ ܕܐܦ ܗܢܘܢ ܡܢ ܝܗܘܕܝܐ ܀</p> 
 <p><a name="an_2:15">2:15</a> ܗܢܘܢ ܕܠܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܩܛܠܘ ܘܠܢܒܝܐ ܕܡܢܗܘܢ ܘܠܢ ܪܕܦܘ ܘܠܐܠܗܐ ܠܐ ܫܦܪܝܢ ܘܥܒܝܕܝܢ ܤܩܘܒܠܐ ܠܟܠܗܘܢ ܒܢܝ ܐܢܫܐ ܀</p> 
 <p><a name="an_2:16">2:16</a> ܕܟܠܝܢ ܠܢ ܕܢܡܠܠ ܥܡ ܥܡܡܐ ܕܢܚܘܢ ܠܡܫܠܡܘ ܚܛܗܝܗܘܢ ܒܟܠܙܒܢ ܡܛܝ ܕܝܢ ܥܠܝܗܘܢ ܪܘܓܙܐ ܥܕܡܐ ܠܚܪܬܐ ܀</p> 
 <p><a name="an_2:17">2:17</a> ܚܢܢ ܕܝܢ ܐܚܝܢ ܗܘܝܢ ܝܬܡܐ ܡܢܟܘܢ ܙܒܢܐ ܕܫܥܬܐ ܒܐܦܝܢ ܘܠܐ ܒܠܒܢ ܘܝܬܝܪܐܝܬ ܐܬܚܦܛܢ ܠܡܚܙܐ ܐܦܝܟܘܢ ܒܚܘܒܐ ܤܓܝܐܐ ܀</p> 
 <p><a name="an_2:18">2:18</a> ܘܨܒܝܢ ܕܢܐܬܐ ܠܘܬܟܘܢ ܐܢܐ ܦܘܠܘܤ ܚܕܐ ܙܒܢ ܘܬܪܬܝܢ ܘܥܘܟܢܝ ܤܛܢܐ ܀</p> 
 <p><a name="an_2:19">2:19</a> ܐܝܢܘ ܓܝܪ ܤܒܪܢ ܘܚܕܘܬܢ ܘܟܠܝܠܐ ܕܫܘܒܗܪܢ ܐܠܐ ܐܢ ܐܢܬܘܢ ܩܕܡ ܡܪܢ ܝܫܘܥ ܒܡܐܬܝܬܗ ܀</p> 
 <p><a name="an_2:20">2:20</a> ܐܢܬܘܢ ܐܢܘܢ ܓܝܪ ܬܫܒܘܚܬܢ ܘܚܕܘܬܢ ܀</p> 
 <p><a name="an_3:1">3:1</a> ܘܡܛܠ ܕܠܐ ܚܡܤܢܢ ܨܒܝܢ ܠܡܦܫ ܒܐܬܢܘܤ ܒܠܚܘܕܝܢ ܀</p> 
 <p><a name="an_3:2">3:2</a> ܘܢܫܕܪ ܠܘܬܟܘܢ ܠܛܝܡܬܐܘܤ ܐܚܘܢ ܘܡܫܡܫܢܐ ܕܐܠܗܐ ܘܡܥܕܪܢܢ ܒܤܒܪܬܗ ܕܡܫܝܚܐ ܕܢܫܪܪܟܘܢ ܘܢܒܥܐ ܡܢܟܘܢ ܥܠ ܗܝܡܢܘܬܟܘܢ ܀</p> 
 <p><a name="an_3:3">3:3</a> ܕܠܐ ܐܢܫ ܡܢܟܘܢ ܬܬܩܛܥ ܠܗ ܒܐܘܠܨܢܐ ܗܠܝܢ ܐܢܬܘܢ ܓܝܪ ܝܕܥܝܢ ܐܢܬܘܢ ܕܠܗܕܐ ܗܘ ܤܝܡܝܢܢ ܀</p> 
 <p><a name="an_3:4">3:4</a> ܐܦ ܟܕ ܠܘܬܟܘܢ ܗܘܝܢ ܓܝܪ ܩܕܡܢ ܐܡܪܢ ܠܟܘܢ ܕܥܬܝܕܝܢܢ ܠܡܬܐܠܨܘ ܐܝܟܢܐ ܕܝܕܥܝܢ ܐܢܬܘܢ ܕܗܘܐ ܀</p> 
 <p><a name="an_3:5">3:5</a> ܡܛܠ ܗܢܐ ܐܦ ܐܢܐ ܠܐ ܚܡܤܢܬ ܥܕܡܐ ܕܫܕܪܬ ܕܐܕܥ ܗܝܡܢܘܬܟܘܢ ܕܕܠܡܐ ܢܢܤܝܟܘܢ ܡܢܤܝܢܐ ܘܢܗܘܐ ܥܡܠܢ ܤܪܝܩܐ ܀</p> 
 <p><a name="an_3:6">3:6</a> ܗܫܐ ܕܝܢ ܡܢ ܕܐܬܐ ܠܘܬܢ ܛܝܡܬܐܘܤ ܡܢ ܨܐܕܝܟܘܢ ܘܤܒܪܢ ܥܠ ܗܝܡܢܘܬܟܘܢ ܘܥܠ ܚܘܒܟܘܢ ܘܕܐܝܬ ܠܘܬܟܘܢ ܥܘܗܕܢܢ ܛܒܐ ܒܟܠ ܥܕܢ ܘܤܘܝܢ ܐܢܬܘܢ ܠܡܚܙܝܢ ܐܝܟܢܐ ܕܐܦ ܚܢܢ ܠܟܘܢ ܀</p> 
 <p><a name="an_3:7">3:7</a> ܡܛܠ ܗܢܐ ܐܬܒܝܐܢ ܒܟܘܢ ܐܚܝܢ ܥܠ ܟܠܗܝܢ ܥܩܬܢ ܘܐܘܠܨܢܝܢ ܡܛܠ ܗܝܡܢܘܬܟܘܢ ܀</p> 
 <p><a name="an_3:8">3:8</a> ܘܗܫܐ ܗܘ ܚܝܝܢܢ ܐܢ ܐܢܬܘܢ ܬܬܩܝܡܘܢ ܒܡܪܢ ܀</p> 
 <p><a name="an_3:9">3:9</a> ܐܝܕܐ ܓܝܪ ܬܘܕܝܬܐ ܡܫܟܚܝܢܢ ܠܡܦܪܥ ܚܠܦܝܟܘܢ ܠܐܠܗܐ ܥܠ ܟܠ ܚܕܘܬܐ ܕܚܕܝܢܢ ܡܛܠܬܟܘܢ ܀</p> 
 <p><a name="an_3:10">3:10</a> ܐܠܐ ܕܩܕܡ ܐܠܗܐ ܝܬܝܪܐܝܬ ܢܬܟܫܦ ܒܠܠܝܐ ܘܒܐܝܡܡܐ ܕܢܚܙܐ ܐܦܝܟܘܢ ܘܢܓܡܘܪ ܡܐ ܕܚܤܝܪܐ ܗܝܡܢܘܬܟܘܢ ܀</p> 
 <p><a name="an_3:11">3:11</a> ܗܘ ܕܝܢ ܐܠܗܐ ܐܒܘܢ ܘܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܢܬܪܘܨ ܐܘܪܚܢ ܠܘܬܟܘܢ ܀</p> 
 <p><a name="an_3:12">3:12</a> ܘܢܤܓܐ ܘܢܝܬܪ ܚܘܒܟܘܢ ܕܚܕ ܠܘܬ ܚܕ ܘܕܠܘܬ ܟܠܢܫ ܐܝܟܢܐ ܕܚܢܢ ܡܚܒܝܢܢ ܠܟܘܢ ܀</p> 
 <p><a name="an_3:13">3:13</a> ܘܢܩܝܡ ܠܒܘܬܟܘܢ ܕܠܐ ܪܫܝܢ ܒܩܕܝܫܘܬܐ ܩܕܡ ܐܠܗܐ ܐܒܘܢ ܒܡܐܬܝܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܕܥܡ ܟܠܗܘܢ ܩܕܝܫܘܗܝ ܀</p> 
 <p><a name="an_4:1">4:1</a> ܡܟܝܠ ܐܚܝ ܒܥܝܢܢ ܡܢܟܘܢ ܘܡܬܟܫܦܝܢܢ ܠܟܘܢ ܒܡܪܢ ܝܫܘܥ ܕܐܝܟ ܕܩܒܠܬܘܢ ܡܢܢ ܐܝܟܢ ܘܠܐ ܠܟܘܢ ܕܬܗܠܟܘܢ ܘܬܫܦܪܘܢ ܠܐܠܗܐ ܕܝܬܝܪܐܝܬ ܬܘܤܦܘܢ ܀</p> 
 <p><a name="an_4:2">4:2</a> ܝܕܥܝܢ ܐܢܬܘܢ ܓܝܪ ܐܝܠܝܢ ܦܘܩܕܢܐ ܝܗܒܢ ܠܟܘܢ ܒܡܪܢ ܝܫܘܥ ܀</p> 
 <p><a name="an_4:3">4:3</a> ܗܢܘ ܓܝܪ ܨܒܝܢܗ ܕܐܠܗܐ ܩܕܝܫܘܬܟܘܢ ܘܕܬܗܘܘܢ ܦܪܝܩܝܢ ܡܢ ܟܠ ܙܢܝܘܬܐ ܀</p> 
 <p><a name="an_4:4">4:4</a> ܘܢܗܘܐ ܝܕܥ ܐܢܫ ܐܢܫ ܡܢܟܘܢ ܠܡܩܢܐ ܡܐܢܗ ܒܩܕܝܫܘܬܐ ܘܒܐܝܩܪܐ ܀</p> 
 <p><a name="an_4:5">4:5</a> ܘܠܐ ܒܚܫܐ ܕܪܓܬܐ ܐܝܟ ܫܪܟܐ ܕܥܡܡܐ ܗܢܘܢ ܕܠܐ ܝܕܥܝܢ ܠܐܠܗܐ ܀</p> 
 <p><a name="an_4:6">4:6</a> ܘܠܐ ܬܗܘܘܢ ܡܫܝܚܝܢ ܠܡܥܒܪ ܘܠܡܥܠܒ ܐܢܫ ܠܐܚܘܗܝ ܒܗܕܐ ܨܒܘܬܐ ܡܛܠ ܕܡܪܢ ܗܘ ܬܒܘܥܐ ܥܠ ܗܠܝܢ ܟܠܗܝܢ ܐܝܟܢܐ ܕܐܦ ܡܢ ܩܕܡ ܙܒܢܐ ܐܡܪܢ ܠܟܘܢ ܘܤܗܕܢ ܀</p> 
 <p><a name="an_4:7">4:7</a> ܠܐ ܓܝܪ ܩܪܟܘܢ ܐܠܗܐ ܠܛܢܦܘܬܐ ܐܠܐ ܠܩܕܝܫܘܬܐ ܀</p> 
 <p><a name="an_4:8">4:8</a> ܡܟܝܠ ܡܢ ܕܛܠܡ ܠܐ ܗܘܐ ܠܒܪܢܫܐ ܛܠܡ ܐܠܐ ܠܐܠܗܐ ܗܘ ܕܝܗܒ ܒܟܘܢ ܪܘܚܗ ܩܕܝܫܬܐ ܀</p> 
 <p><a name="an_4:9">4:9</a> ܥܠ ܚܘܒܐ ܕܝܢ ܕܐܚܐ ܠܐ ܤܢܝܩܝܬܘܢ ܠܡܟܬܒ ܠܟܘܢ ܐܢܬܘܢ ܓܝܪ ܩܢܘܡܟܘܢ ܡܠܦܐ ܐܢܬܘܢ ܕܐܠܗܐ ܕܬܚܒܘܢ ܚܕ ܠܚܕ ܀</p> 
 <p><a name="an_4:10">4:10</a> ܐܦ ܥܒܕܝܢ ܐܢܬܘܢ ܗܟܢܐ ܠܟܠܗܘܢ ܐܚܐ ܕܒܟܠܗ ܡܩܕܘܢܝܐ ܒܥܐ ܐܢܐ ܕܝܢ ܡܢܟܘܢ ܐܚܝ ܕܬܬܝܬܪܘܢ ܀</p> 
 <p><a name="an_4:11">4:11</a> ܘܬܬܚܦܛܘܢ ܕܬܗܘܘܢ ܫܠܝܢ ܘܥܢܝܢ ܒܤܘܥܪܢܝܟܘܢ ܘܗܘܝܬܘܢ ܦܠܚܝܢ ܒܐܝܕܝܟܘܢ ܐܝܟܢܐ ܕܦܩܕܢܟܘܢ ܀</p> 
 <p><a name="an_4:12">4:12</a> ܕܬܗܘܘܢ ܡܗܠܟܝܢ ܒܐܤܟܡܐ ܠܘܬ ܒܪܝܐ ܘܥܠ ܐܢܫ ܠܐ ܬܤܬܢܩܘܢ ܀</p> 
 <p><a name="an_4:13">4:13</a> ܨܒܐ ܐܢܐ ܕܝܢ ܕܬܕܥܘܢ ܐܚܝ ܕܥܠ ܐܝܠܝܢ ܕܕܡܟܝܢ ܠܐ ܬܗܘܐ ܟܪܝܐ ܠܟܘܢ ܐܝܟ ܫܪܟܐ ܕܐܢܫܐ ܕܤܒܪܐ ܠܝܬ ܠܗܘܢ ܀</p> 
 <p><a name="an_4:14">4:14</a> ܐܢ ܓܝܪ ܡܗܝܡܢܝܢܢ ܕܝܫܘܥ ܡܝܬ ܘܩܡ ܗܟܢܐ ܐܦ ܐܠܗܐ ܠܐܝܠܝܢ ܕܕܡܟܘ ܒܝܫܘܥ ܡܝܬܐ ܥܡܗ ܀</p> 
 <p><a name="an_4:15">4:15</a> ܗܕܐ ܕܝܢ ܐܡܪܝܢܢ ܠܟܘܢ ܒܡܠܬܗ ܕܡܪܢ ܕܚܢܢ ܐܝܠܝܢ ܕܡܫܬܚܪܝܢܢ ܒܡܐܬܝܬܗ ܕܡܪܢ ܕܚܝܝܢܢ ܠܐ ܢܕܪܟ ܠܐܝܠܝܢ ܕܕܡܟܘ ܀</p> 
 <p><a name="an_4:16">4:16</a> ܡܛܠ ܕܗܘ ܡܪܢ ܒܦܘܩܕܢܐ ܘܒܩܠܐ ܕܪܝܫ ܡܠܐܟܐ ܘܒܩܪܢܐ ܕܐܠܗܐ ܢܚܬ ܡܢ ܫܡܝܐ ܘܡܝܬܐ ܕܒܡܫܝܚܐ ܢܩܘܡܘܢ ܠܘܩܕܡ ܀</p> 
 <p><a name="an_4:17">4:17</a> ܘܗܝܕܝܢ ܚܢܢ ܐܝܠܝܢ ܕܡܫܬܚܪܝܢܢ ܕܚܝܝܢܢ ܢܬܚܛܦ ܥܡܗܘܢ ܐܟܚܕܐ ܒܥܢܢܐ ܠܐܘܪܥܗ ܕܡܪܢ ܒܐܐܪ ܘܗܟܢܐ ܒܟܠܙܒܢ ܥܡ ܡܪܢ ܢܗܘܐ ܀</p> 
 <p><a name="an_4:18">4:18</a> ܗܘܘ ܗܟܝܠ ܡܒܝܐܝܢ ܚܕ ܠܚܕ ܒܡܠܐ ܗܠܝܢ ܀</p> 
 <p><a name="an_5:1">5:1</a> ܥܠ ܙܒܢܐ ܕܝܢ ܘܥܕܢܐ ܐܚܝ ܠܐ ܤܢܝܩܝܬܘܢ ܠܡܟܬܒ ܠܟܘܢ ܀</p> 
 <p><a name="an_5:2">5:2</a> ܐܢܬܘܢ ܓܝܪ ܫܪܝܪܐܝܬ ܝܕܥܝܢ ܐܢܬܘܢ ܕܝܘܡܗ ܕܡܪܢ ܐܝܟ ܓܢܒܐ ܒܠܠܝܐ ܗܟܢܐ ܐܬܐ ܀</p> 
 <p><a name="an_5:3">5:3</a> ܟܕ ܢܐܡܪܘܢ ܕܫܠܡܐ ܗܘ ܘܫܝܢܐ ܘܗܝܕܝܢ ܡܢܫܠܝܐ ܢܩܘܡ ܥܠܝܗܘܢ ܐܒܕܢܐ ܐܝܟ ܚܒܠܐ ܥܠ ܒܛܢܬܐ ܘܠܐ ܢܡܕܘܢ ܀</p> 
 <p><a name="an_5:4">5:4</a> ܐܢܬܘܢ ܕܝܢ ܐܚܝ ܠܐ ܗܘܝܬܘܢ ܒܚܫܘܟܐ ܕܗܘ ܝܘܡܐ ܐܝܟ ܓܢܒܐ ܢܕܪܟܟܘܢ ܀</p> 
 <p><a name="an_5:5">5:5</a> ܟܠܟܘܢ ܓܝܪ ܒܢܝ ܢܘܗܪܐ ܐܢܬܘܢ ܘܒܢܝ ܐܝܡܡܐ ܘܠܐ ܗܘܝܬܘܢ ܒܢܝ ܠܠܝܐ ܘܠܐ ܒܢܝ ܚܫܘܟܐ ܀</p> 
 <p><a name="an_5:6">5:6</a> ܠܐ ܢܕܡܟ ܗܟܝܠ ܐܝܟ ܫܪܟܐ ܐܠܐ ܗܘܝܢ ܥܝܪܝܢ ܘܡܗܘܢܝܢ ܀</p> 
 <p><a name="an_5:7">5:7</a> ܐܝܠܝܢ ܓܝܪ ܕܕܡܟܝܢ ܒܠܠܝܐ ܗܘ ܕܡܟܝܢ ܘܐܝܠܝܢ ܕܪܘܝܢ ܒܠܠܝܐ ܗܘ ܪܘܝܢ ܀</p> 
 <p><a name="an_5:8">5:8</a> ܚܢܢ ܕܝܢ ܕܒܢܝ ܐܝܡܡܐ ܚܢܢ ܗܘܝܢ ܥܝܪܝܢ ܒܪܥܝܢܢ ܘܠܒܝܫܝܢ ܫܪܝܢܐ ܕܗܝܡܢܘܬܐ ܘܕܚܘܒܐ ܘܢܤܝܡ ܤܢܘܪܬܐ ܕܤܒܪܐ ܕܚܝܐ ܀</p> 
 <p><a name="an_5:9">5:9</a> ܡܛܠ ܕܠܐ ܤܡܢ ܐܠܗܐ ܠܪܘܓܙܐ ܐܠܐ ܠܩܢܝܢܐ ܕܚܝܐ ܒܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_5:10">5:10</a> ܗܘ ܕܡܝܬ ܥܠ ܐܦܝܢ ܕܐܢ ܥܝܪܝܢܢ ܘܐܢ ܕܡܟܝܢܢ ܐܟܚܕܐ ܥܡܗ ܢܚܐ ܀</p> 
 <p><a name="an_5:11">5:11</a> ܡܛܠ ܗܢܐ ܒܝܐܘ ܚܕ ܠܚܕ ܘܒܢܘ ܚܕ ܠܚܕ ܐܝܟܢܐ ܕܐܦ ܥܒܕܝܬܘܢ ܀</p> 
 <p><a name="an_5:12">5:12</a> ܒܥܝܢܢ ܕܝܢ ܡܢܟܘܢ ܐܚܝ ܕܗܘܝܬܘܢ ܝܕܥܝܢ ܠܐܝܠܝܢ ܕܠܐܝܢ ܒܟܘܢ ܘܩܝܡܝܢ ܒܐܦܝܟܘܢ ܒܡܪܢ ܘܡܠܦܝܢ ܠܟܘܢ ܀</p> 
 <p><a name="an_5:13">5:13</a> ܕܗܘܘ ܚܫܝܒܝܢ ܠܟܘܢ ܒܚܘܒܐ ܝܬܝܪܐ ܘܡܛܠ ܥܒܕܗܘܢ ܐܫܬܝܢܘ ܥܡܗܘܢ ܀</p> 
 <p><a name="an_5:14">5:14</a> ܒܥܝܢܢ ܕܝܢ ܡܢܟܘܢ ܐܚܝ ܪܕܘ ܠܡܤܟܠܢܐ ܘܠܒܒܘ ܠܙܥܘܪܝ ܢܦܫܐ ܘܤܒܘ ܛܥܢܐ ܕܡܚܝܠܐ ܘܐܓܪܘ ܪܘܚܟܘܢ ܠܘܬ ܟܠܢܫ ܀</p> 
 <p><a name="an_5:15">5:15</a> ܘܐܙܕܗܪܘ ܕܠܡܐ ܐܢܫ ܡܢܟܘܢ ܒܝܫܬܐ ܚܠܦ ܒܝܫܬܐ ܢܦܪܘܥ ܐܠܐ ܒܟܠܙܒܢ ܗܪܛܘ ܒܬܪ ܛܒܬܐ ܠܘܬ ܚܕܕܐ ܘܠܘܬ ܟܠܢܫ ܀</p> 
 <p><a name="an_5:16">5:16</a> ܗܘܘ ܚܕܝܢ ܒܟܠܙܒܢ ܀</p> 
 <p><a name="an_5:17">5:17</a> ܘܡܨܠܝܢ ܕܠܐ ܫܠܘܐ ܀</p> 
 <p><a name="an_5:18">5:18</a> ܘܒܟܠ ܡܕܡ ܗܘܘ ܡܘܕܝܢ ܗܢܘ ܓܝܪ ܨܒܝܢܗ ܕܐܠܗܐ ܒܝܫܘܥ ܡܫܝܚܐ ܒܟܘܢ ܀</p> 
 <p><a name="an_5:19">5:19</a> ܪܘܚܐ ܠܐ ܬܕܥܟܘܢ ܀</p> 
 <p><a name="an_5:20">5:20</a> ܢܒܝܘܬܐ ܠܐ ܬܤܠܘܢ ܀</p> 
 <p><a name="an_5:21">5:21</a> ܟܠܡܕܡ ܒܩܘ ܘܕܫܦܝܪ ܐܚܘܕܘ ܀</p> 
 <p><a name="an_5:22">5:22</a> ܘܡܢ ܟܠ ܨܒܘ ܒܝܫܐ ܥܪܘܩܘ ܀</p> 
 <p><a name="an_5:23">5:23</a> ܗܘ ܕܝܢ ܐܠܗܐ ܕܫܠܡܐ ܢܩܕܫܟܘܢ ܓܡܝܪܐܝܬ ܠܟܠܟܘܢ ܘܟܠܗ ܪܘܚܟܘܢ ܘܢܦܫܟܘܢ ܘܦܓܪܟܘܢ ܢܢܛܪ ܕܠܐ ܪܫܝܢ ܠܡܐܬܝܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_5:24">5:24</a> ܡܗܝܡܢ ܗܘ ܡܢ ܕܩܪܟܘܢ ܕܗܘ ܢܥܒܕ ܀</p> 
 <p><a name="an_5:25">5:25</a> ܐܚܝ ܨܠܘ ܥܠܝܢ ܀</p> 
 <p><a name="an_5:26">5:26</a> ܫܐܠܘ ܒܫܠܡܐ ܕܐܚܝܢ ܟܠܗܘܢ ܒܢܘܫܩܬܐ ܩܕܝܫܬܐ ܀</p> 
 <p><a name="an_5:27">5:27</a> ܡܘܡܐ ܐܢܐ ܠܟܘܢ ܒܡܪܢ ܕܬܬܩܪܐ ܐܓܪܬܐ ܗܕܐ ܠܟܠܗܘܢ ܐܚܐ ܩܕܝܫܐ ܀</p> 
 <p><a name="an_5:28">5:28</a> ܛܝܒܘܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܥܡܟܘܢ ܐܡܝܢ ܀</p>
