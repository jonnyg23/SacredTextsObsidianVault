 <p><span class="margnote"><font color="GREEN" size="-1"><a name="an_000">lev 2:0</a></font></span></p>
 <p>Overview
 </p><p><a href="../../../bib/kjv/lev002.htm#001">Lev 2:1</a>, The meat offering of flour with oil and incense, <a href="../../../bib/kjv/lev002.htm#004">Lev 2:4</a>, either baked in the oven, <a href="../../../bib/kjv/lev002.htm#005">Lev 2:5</a>, or on a plate, <a href="../../../bib/kjv/lev002.htm#007">Lev 2:7</a>, or in a frying-pan; <a href="../../../bib/kjv/lev002.htm#012">Lev 2:12</a>, The first fruits not to be burnt on the altar; <a href="../../../bib/kjv/lev002.htm#013">Lev 2:13</a>, Salt to be used with every offering; <a href="../../../bib/kjv/lev002.htm#014">Lev 2:14</a>, The offering of first fruits in the ear. 
 </p><p align="center"><a href="../../../bib/kjv/lev002.htm#001">Leviticus 2:1</a></p> 
 <p><span class="margnote"><font color="GREEN" size="-1"><a name="an_001">lev 2:1</a></font></span></p>
 <p>meat offering:  Minchah, from the Arabic manacha, to give, especially as a reciprocal gift, a gift, oblation, or eucharistical or gratitude offering, for the bounties of providence displayed in the fruits of the earth.  It is termed a meat offering by our translators, because the term meat in their time was the general name for food. <a href="../../../bib/kjv/lev006.htm#014">Lev 6:14-18</a>, <a href="../../../bib/kjv/lev006.htm#020">Lev 6:20-23</a>, <a href="../../../bib/kjv/lev009.htm#017">Lev 9:17</a>; Num. 15:4-21; <a href="../../../bib/kjv/isa066.htm#020">Isa 66:20</a>; <a href="../../../bib/kjv/joh006.htm#035">Joh 6:35</a> 
 </p><p>fine flour:  <a href="../../../bib/kjv/exo029.htm#002">Exo 29:2</a>; <a href="../../../bib/kjv/num007.htm#013">Num 7:13</a>, <a href="../../../bib/kjv/num007.htm#019">Num 7:19</a>; <a href="../../../bib/kjv/joe001.htm#009">Joe 1:9</a>, <a href="../../../bib/kjv/joe002.htm#014">Joe 2:14</a> 
 </p><p>pour oil:  <a href="../../../bib/kjv/lev002.htm#004">Lev 2:4-8</a>, <a href="../../../bib/kjv/lev002.htm#015">Lev 2:15</a>, <a href="../../../bib/kjv/lev002.htm#016">Lev 2:16</a>, <a href="../../../bib/kjv/lev007.htm#010">Lev 7:10-12</a>; <a href="../../../bib/kjv/jo1002.htm#020">Jo1 2:20</a>, <a href="../../../bib/kjv/jo1002.htm#027">Jo1 2:27</a>; <a href="../../../bib/kjv/jde001.htm#020">Jde 1:20</a> 
 </p><p>frankincense:  <a href="../../../bib/kjv/mal001.htm#011">Mal 1:11</a>; <a href="../../../bib/kjv/luk001.htm#009">Luk 1:9</a>, <a href="../../../bib/kjv/luk001.htm#010">Luk 1:10</a>; <a href="../../../bib/kjv/rev008.htm#003">Rev 8:3</a>

 </p><p align="center"><a href="../../../bib/kjv/lev002.htm#004">Leviticus 2:4</a></p> 
 <p><span class="margnote"><font color="GREEN" size="-1"><a name="an_004">lev 2:4</a></font></span></p>
 <p>meat offering:  <a href="../../../bib/kjv/ch1023.htm#028">Ch1 23:28</a>, <a href="../../../bib/kjv/ch1023.htm#029">Ch1 23:29</a>; <a href="../../../bib/kjv/psa022.htm#014">Psa 22:14</a>; <a href="../../../bib/kjv/eze046.htm#020">Eze 46:20</a>; <a href="../../../bib/kjv/mat026.htm#038">Mat 26:38</a>; <a href="../../../bib/kjv/joh012.htm#027">Joh 12:27</a> 
 </p><p>the oven:  Tannur, probably such an oven as that described by D'Arvieux, as used by the Arabs.  He states that they make a fire in a great stone pitcher, and when heated, mix meal and water, which they apply with the hollow of their hands to the outside, and this soft paste spreading itself upon it, is baked in an instant, and the bread comes of as thin as our wafers. <a href="../../../bib/kjv/lev001.htm#011">Lev 1:11</a>, <a href="../../../bib/kjv/lev006.htm#017">Lev 6:17</a>, <a href="../../../bib/kjv/lev007.htm#012">Lev 7:12</a>, <a href="../../../bib/kjv/lev010.htm#012">Lev 10:12</a>; <a href="../../../bib/kjv/exo012.htm#008">Exo 12:8</a>; <a href="../../../bib/kjv/co1005.htm#007">Co1 5:7</a>, <a href="../../../bib/kjv/co1005.htm#008">Co1 5:8</a>; <a href="../../../bib/kjv/heb007.htm#026">Heb 7:26</a>; <a href="../../../bib/kjv/pe1002.htm#001">Pe1 2:1</a>, <a href="../../../bib/kjv/pe1002.htm#022">Pe1 2:22</a> 
 </p><p>wafers:  <a href="../../../bib/kjv/exo016.htm#031">Exo 16:31</a>, <a href="../../../bib/kjv/exo029.htm#002">Exo 29:2</a>; <a href="../../../bib/kjv/isa042.htm#001">Isa 42:1</a>, <a href="../../../bib/kjv/isa044.htm#003">Isa 44:3-5</a>, <a href="../../../bib/kjv/isa061.htm#001">Isa 61:1</a>; <a href="../../../bib/kjv/joh003.htm#034">Joh 3:34</a> 
 </p><p align="center"><a href="../../../bib/kjv/lev002.htm#005">Leviticus 2:5</a></p> 
 <p><span class="margnote"><font color="GREEN" size="-1"><a name="an_005">lev 2:5</a></font></span></p>
 <p>in a pan:  or, on a flat plate, or slice, Machavath, a flat iron plate, such as the Arabs still use to bake their cakes on, and which is called a griddle in some of our counties.
