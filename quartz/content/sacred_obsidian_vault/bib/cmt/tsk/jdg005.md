 <p>Sang Deborah:  This verse briefly recites the subject of this inspired song, which consists of eight stanzas:  The first opens with a devout thanksgiving.  The second describes the magnificent scenes at Mount Sinai, etc.  The third states the apostasy and consequent punishment of the Israelites.  The fourth contrasts their present happy state.  The fifth censures the recreant tribes of Reuben, Gad, etc.  The sixth records the defeat of the confederate kings of Canaan.  The seventh contains a panegyric on Jael.  And the eight describes the fond anticipations and disappointment of the mother of Sisera. <a href="../../../bib/kjv/exo015.htm#001">Exo 15:1</a>, <a href="../../../bib/kjv/exo015.htm#021">Exo 15:21</a>; <a href="../../../bib/kjv/num021.htm#017">Num 21:17</a>; <a href="../../../bib/kjv/sa1002.htm#001">Sa1 2:1</a>; <a href="../../../bib/kjv/ch2020.htm#021">Ch2 20:21</a>, <a href="../../../bib/kjv/ch2020.htm#027">Ch2 20:27</a>; <a href="../../../bib/kjv/job038.htm#007">Job 38:7</a>; <a href="../../../bib/kjv/psa018.htm#001">Psa 18:1</a> *title <a href="../../../bib/kjv/isa012.htm#001">Isa 12:1-6</a>, <a href="../../../bib/kjv/isa025.htm#001">Isa 25:1</a>, <a href="../../../bib/kjv/isa026.htm#001">Isa 26:1</a>; <a href="../../../bib/kjv/luk001.htm#046">Luk 1:46</a>, <a href="../../../bib/kjv/luk001.htm#067">Luk 1:67</a>, <a href="../../../bib/kjv/luk001.htm#068">Luk 1:68</a>; <a href="../../../bib/kjv/rev015.htm#003">Rev 15:3</a>, <a href="../../../bib/kjv/rev015.htm#004">Rev 15:4</a>, <a href="../../../bib/kjv/rev019.htm#001">Rev 19:1-3</a>
