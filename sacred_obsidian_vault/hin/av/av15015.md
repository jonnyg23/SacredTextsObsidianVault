 <p><a name="page_160"><font size="1" color="green">p. 160</font></a></p>  <h3 align="center">HYMN XV</h3>  
 <p><span class="margnote"><font color="GREEN" size="-1"><a name="an_1501501">1</a></font></span>Of that Vrātya.<br> <span class="margnote"><font color="GREEN" size="-1"><a name="an_1501502">2</a></font></span>There are seven vital airs, seven downward breaths, seven diff-<br>    used breaths.<br> <span class="margnote"><font color="GREEN" size="-1"><a name="an_1501503">3</a></font></span>His first vital breath, called Upward, is this Agni.<br> <span class="margnote"><font color="GREEN" size="-1"><a name="an_1501504">4</a></font></span>His second vital breath, called Mature, is that Āditya.<br> <span class="margnote"><font color="GREEN" size="-1"><a name="an_1501505">5</a></font></span>His third vital breath, called Approached, is that Moon.<br> <span class="margnote"><font color="GREEN" size="-1"><a name="an_1501506">6</a></font></span>His fourth vital breath, called Pervading is this Pavamāna.<br> <span class="margnote"><font color="GREEN" size="-1"><a name="an_1501507">7</a></font></span>His fifth vital breath, called Source, are these Waters.<br> <span class="margnote"><font color="GREEN" size="-1"><a name="an_1501508">8</a></font></span>His sixth vital breath, called Dear, are these domestic animals.<br> <span class="margnote"><font color="GREEN" size="-1"><a name="an_1501509">9</a></font></span>His seventh vital breath, called Unlimited, are these creatures.</p>
