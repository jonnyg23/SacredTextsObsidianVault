 <p align="center"><font size="-1" color="green">All the errata Eggeling which specifies here have been fixed in the etext using the linked errata file. These are marked *(Eggeling)* in the errata file.--JBH</font></p>  <h3 align="center">CORRECTIONS.</h3>  
 <p>Introduction, page <a href="sbe4102.htm#page_xii">xii</a>, line 33. Read,--the day preceding the Soma-day, the animal offering to Agni-Soma being indeed a constant feature of that day's proceedings at every Soma-sacrifice; whilst the slaughter of the special victim, or victims, of the respective sacrifice takes place during the morning service, &amp;c.</p>  
 <p>P. <a href="sbe4103.htm#page_5">5</a>, last line of text. Read,--therefor.</p>  
 <p>P. <a href="sbe4104.htm#page_6">6</a>, note <a href="sbe4104.htm#fn_37">2</a>, l. 3. P<i>ri</i>sh<i>th</i>ya sha<i>d</i>aha, see Introduction, <a href="sbe4102.htm#page_xxi">p. xxi</a>.</p>  
 <p>P. <a href="sbe4104.htm#page_8">8</a>, last line of notes <font size="-1" color="green">*(</font><a href="sbe4104.htm#fn_39">1</a><font size="-1" color="green">)*</font>. For 'II, 665' read 'II, 663, in a different tune again.'</p>  
 <p>P. <a href="sbe4104.htm#page_9">9</a>, l. 5 of notes <font size="-1" color="green">*(</font><a href="sbe4104.htm#fn_40">1</a><font size="-1" color="green">)*</font>. Read,--II, 720-22.</p>  
 <p>P. <a href="sbe4108.htm#page_34">34</a>, l. 21. Read,--B<i>ri</i>haspati consecration.</p>  
 <p>P. <a href="sbe4109.htm#page_41">41</a>, l. 14. For 'offering' read 'offspring.'</p>  
 <p>P. <a href="sbe4120.htm#page_104">104</a>, l. 2. For 'truth' read 'law;' (cf. <a href="sbe4151.htm#6:7:3:11">VI, 7, 3, 11</a>.)</p>  
 <p>P. <a href="sbe4128.htm#page_146">146</a>, l. 23. For 'become' read 'became.'</p>
