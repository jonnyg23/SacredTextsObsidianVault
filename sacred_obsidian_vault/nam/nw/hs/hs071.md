 <p>There used to be plenty, there used to be plenty, chief,<br>
 There used to be plenty in your house, chief;<br>
 There used to be plenty, there used to be plenty, chief;<br>
 There used to be plenty, there used to be plenty, chief.</p>

 There used to be plenty, there used to be plenty, chief;<br>
 there used to be plenty, there used to be plenty, chief.</p>
