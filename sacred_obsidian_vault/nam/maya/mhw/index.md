 <p>
 J. Eris S. Thompson was one of the leading 20th century Mayan scholars.
 This is a set of excerpts from Thompson's massive opus on Mayan writing.
 Mayan dates were the first elements of this writing system which were decoded,
 so there is extensive discussion of the Mayan calendar in this work.
 New portions of this book will be added in the near future, but it
 is not feasible to post the entire book on the web at this point.
