 <p> <a name="xiii.vii"></a>   
 </p><p class="c46"><a name="xiii.vii-p1"></a> <span class="c2"><a name="xiii.vii-p1.1"></a>(3.) The Third Breviate.</span></p>   
 <p><a name="xiii.vii-p2"></a>III. “Again we must ask,” he says, “what sin is,—natural? or accidental? If natural, it is not sin; if accidental, it is separable; <a name="fr_1368"></a><a href="#fn_1368"><font size="1">1369</font></a>
  and if it is separable, it can be avoided; and because it can be avoided, man can be without that which can be avoided.” The answer to this is, that sin is not natural; but nature (especially in that corrupt state from which we have become by nature “children of wrath” <a name="fr_1369"></a><a href="#fn_1369"><font size="1">1370</font></a>
 ) has too little determination of will to avoid sin, unless assisted and healed by Gods grace through Jesus Christ our Lord.</p>
