 <hr>
 <p>This is a version of the
 ISTA <a href="../sappho/index.htm">Sappho poetry collection</a>
 with the Greek text displayed in <a href="../../unicode.htm">Unicode</a>.
 Note that final sigmas are written as medials (all sigmas were lunate sigmas
 in the source text, but I decided to 'modernize' the orthography slightly).
 This version also corrects some transcription errors in the original etext,
 which has not been altered at this point.
 The introduction has also been updated slightly.</p>
