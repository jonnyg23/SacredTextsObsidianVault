 <p>Overview
 </p><p><a href="../../../bib/kjv/psa133.htm#001">Psa 133:1</a>, The blessedness of unity among brethren.
 </p><p>This Psalm was probably composed when David was made king over all Israel. <a href="../../../bib/kjv/psa122.htm#001">Psa 122:1</a>, <a href="../../../bib/kjv/psa124.htm#001">Psa 124:1</a>, <a href="../../../bib/kjv/psa131.htm#001">Psa 131:1</a> *titles 
 </p><p align="center"><a href="../../../bib/kjv/psa133.htm#001">Psalms 133:1</a></p> 
 <p><span class="margnote"><font color="GREEN" size="-1"><a name="an_001">psa 133:1</a></font></span></p>
 <p>how good:  <a href="../../../bib/kjv/psa122.htm#006">Psa 122:6-8</a>; <a href="../../../bib/kjv/gen013.htm#008">Gen 13:8</a>, <a href="../../../bib/kjv/gen045.htm#024">Gen 45:24</a>; <a href="../../../bib/kjv/sa2002.htm#026">Sa2 2:26</a>, <a href="../../../bib/kjv/sa2002.htm#027">Sa2 2:27</a>; <a href="../../../bib/kjv/isa011.htm#006">Isa 11:6</a>, <a href="../../../bib/kjv/isa011.htm#009">Isa 11:9</a>, <a href="../../../bib/kjv/isa011.htm#013">Isa 11:13</a>; <a href="../../../bib/kjv/jer032.htm#039">Jer 32:39</a>; <a href="../../../bib/kjv/joh013.htm#035">Joh 13:35</a>, <a href="../../../bib/kjv/joh017.htm#021">Joh 17:21</a>; <a href="../../../bib/kjv/co1001.htm#010">Co1 1:10</a>; <a href="../../../bib/kjv/eph004.htm#003">Eph 4:3-6</a>; <a href="../../../bib/kjv/phi002.htm#002">Phi 2:2-5</a>; <a href="../../../bib/kjv/heb013.htm#001">Heb 13:1</a>; <a href="../../../bib/kjv/pe1003.htm#008">Pe1 3:8</a>; <a href="../../../bib/kjv/jo1003.htm#014">Jo1 3:14-19</a> 
 </p><p>together:  Heb. even together 
 </p><p align="center"><a href="../../../bib/kjv/psa133.htm#002">Psalms 133:2</a></p>

 </p><p>that ran down:  <a href="../../../bib/kjv/exo030.htm#025">Exo 30:25-30</a>; <a href="../../../bib/kjv/lev008.htm#012">Lev 8:12</a> 
 </p><p align="center"><a href="../../../bib/kjv/psa133.htm#003">Psalms 133:3</a></p> 
 <p><span class="margnote"><font color="GREEN" size="-1"><a name="an_003">psa 133:3</a></font></span></p>
 <p>As the dew of Hermon:  Mr. Maundrell says, "We were sufficiently instructed by experience what the holy Psalmist means by 'the dew of Hermon,' our tents being as wet with it as if it had rained all night."  Some suppose that Zion here means a part of Mount Hermon (<a href="../../../bib/kjv/deu004.htm#048">Deu 4:48</a>); but it is not written Sion here, but Zion, which is at Jerusalem. <a href="../../../bib/kjv/deu003.htm#008">Deu 3:8</a>, <a href="../../../bib/kjv/deu003.htm#009">Deu 3:9</a>, <a href="../../../bib/kjv/deu004.htm#048">Deu 4:48</a>; <a href="../../../bib/kjv/jos013.htm#011">Jos 13:11</a>
