 <p>Overview
 </p><p><a href="../../../bib/kjv/psa124.htm#001">Psa 124:1</a>, The church blesses God for a miraculous deliverance.
 </p><p>It is uncertain what the particular deliverance was which is celebrated in this Psalm.  Some refer it to the deliverance of Hezekiah from Sennacherib; and others to the return from the Babylonian captivity; while Dr. A. Clarke refers it to that of the Jews from the massacre intended by Haman. <a href="../../../bib/kjv/psa120.htm#001">Psa 120:1</a>, <a href="../../../bib/kjv/psa121.htm#001">Psa 121:1</a>, <a href="../../../bib/kjv/psa122.htm#001">Psa 122:1</a>, <a href="../../../bib/kjv/psa123.htm#001">Psa 123:1</a>, <a href="../../../bib/kjv/psa124.htm#001">Psa 124:1</a>, <a href="../../../bib/kjv/psa125.htm#001">Psa 125:1</a>, <a href="../../../bib/kjv/psa126.htm#001">Psa 126:1</a>, <a href="../../../bib/kjv/psa127.htm#001">Psa 127:1</a>, <a href="../../../bib/kjv/psa128.htm#001">Psa 128:1</a> *titles <a href="../../../bib/kjv/psa129.htm#001">Psa 129:1</a>, <a href="../../../bib/kjv/psa130.htm#001">Psa 130:1</a>, <a href="../../../bib/kjv/psa131.htm#001">Psa 131:1</a>, <a href="../../../bib/kjv/psa132.htm#001">Psa 132:1</a>, <a href="../../../bib/kjv/psa133.htm#001">Psa 133:1</a>, <a href="../../../bib/kjv/psa134.htm#001">Psa 134:1</a> *titles
