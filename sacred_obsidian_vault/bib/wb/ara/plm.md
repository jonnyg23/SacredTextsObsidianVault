  <h3 align="center">Aramaic Bible: Philemon</h3> 
 <p><a name="an_1:1">1:1</a> ܦܘܠܘܤ ܐܤܝܪܗ ܕܝܫܘܥ ܡܫܝܚܐ ܘܛܝܡܬܐܘܤ ܐܚܐ ܠܦܝܠܡܘܢ ܚܒܝܒܐ ܘܦܠܚܐ ܕܥܡܢ ܀</p> 
 <p><a name="an_1:2">1:2</a> ܘܠܐܦܝܐ ܚܒܝܒܬܢ ܘܠܐܪܟܝܦܘܤ ܦܠܚܐ ܕܥܡܢ ܘܠܥܕܬܐ ܕܒܒܝܬܟ ܀</p> 
 <p><a name="an_1:3">1:3</a> ܛܝܒܘܬܐ ܥܡܟܘܢ ܘܫܠܡܐ ܡܢ ܐܠܗܐ ܐܒܘܢ ܘܡܢ ܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:4">1:4</a> ܡܘܕܐ ܐܢܐ ܠܐܠܗܝ ܒܟܠܙܒܢ ܘܡܬܕܟܪ ܐܢܐ ܠܟ ܒܨܠܘܬܝ ܀</p> 
 <p><a name="an_1:5">1:5</a> ܗܐ ܡܢ ܕܫܡܥܬ ܗܝܡܢܘܬܟ ܘܚܘܒܐ ܕܐܝܬ ܠܟ ܠܘܬ ܡܪܢ ܝܫܘܥ ܘܠܘܬ ܟܠܗܘܢ ܩܕܝܫܐ ܀</p> 
 <p><a name="an_1:6">1:6</a> ܕܬܗܘܐ ܫܘܬܦܘܬܐ ܕܗܝܡܢܘܬܟ ܝܗܒܐ ܦܐܪܐ ܒܥܒܕܐ ܘܒܝܕܥܬܐ ܕܟܠ ܛܒܢ ܕܐܝܬ ܠܟܘܢ ܒܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:7">1:7</a> ܚܕܘܬܐ ܓܝܪ ܤܓܝܐܬܐ ܐܝܬ ܠܢ ܘܒܘܝܐܐ ܕܒܝܕ ܚܘܒܟ ܐܬܬܢܝܚܘ ܪܚܡܐ ܕܩܕܝܫܐ ܀</p> 
 <p><a name="an_1:8">1:8</a> ܡܛܠ ܗܕܐ ܦܪܗܤܝܐ ܤܓܝܐܬܐ ܐܝܬ ܠܝ ܒܡܫܝܚܐ ܕܐܦܩܘܕ ܠܟ ܐܝܠܝܢ ܕܙܕܩܢ ܀</p> 
 <p><a name="an_1:9">1:9</a> ܡܛܠ ܚܘܒܐ ܕܝܢ ܡܒܥܐ ܗܘ ܒܥܐ ܐܢܐ ܡܢܟ ܐܢܐ ܦܘܠܘܤ ܕܐܝܬܝ ܤܒܐ ܐܝܟ ܕܝܕܥ ܐܢܬ ܗܫܐ ܕܝܢ ܐܦ ܐܤܝܪܐ ܕܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:10">1:10</a> ܘܒܥܐ ܐܢܐ ܡܢܟ ܥܠ ܒܪܝ ܐܝܢܐ ܕܝܠܕܬ ܒܐܤܘܪܝ ܐܢܤܝܡܘܤ ܀</p> 
 <p><a name="an_1:11">1:11</a> ܗܘ ܕܒܙܒܢ ܠܝܬ ܗܘܐ ܠܟ ܒܗ ܚܫܚܘ ܗܫܐ ܕܝܢ ܐܦ ܠܟ ܐܦ ܠܝ ܛܒ ܚܫܚ ܀</p> 
 <p><a name="an_1:12">1:12</a> ܘܫܕܪܬܗ ܠܟ ܐܢܬ ܕܝܢ ܐܝܟ ܕܠܝܠܕܐ ܕܝܠܝ ܗܟܢܐ ܩܒܠܝܗܝ ܀</p> 
 <p><a name="an_1:13">1:13</a> ܨܒܐ ܗܘܝܬ ܓܝܪ ܕܠܘܬܝ ܐܚܕܝܘܗܝ ܕܢܗܘܐ ܡܫܡܫ ܠܝ ܚܠܦܝܟ ܒܐܤܘܪܐ ܕܐܘܢܓܠܝܘܢ ܀</p> 
 <p><a name="an_1:14">1:14</a> ܒܠܥܕ ܡܠܟܟ ܕܝܢ ܠܐ ܨܒܝܬ ܡܕܡ ܠܡܥܒܕ ܕܠܐ ܐܝܟ ܕܒܩܛܝܪܐ ܬܗܘܐ ܛܒܬܟ ܐܠܐ ܒܨܒܝܢܟ ܀</p> 
 <p><a name="an_1:15">1:15</a> ܟܒܪ ܕܝܢ ܐܦ ܡܛܠ ܗܕܐ ܫܢܝ ܕܫܥܬܐ ܕܠܥܠܡ ܬܐܚܕܝܘܗܝ ܀</p> 
 <p><a name="an_1:16">1:16</a> ܠܐ ܡܟܝܠ ܐܝܟ ܥܒܕܐ ܐܠܐ ܝܬܝܪ ܡܢ ܥܒܕܐ ܐܚܐ ܚܒܝܒܐ ܕܝܠܝ ܚܕ ܟܡܐ ܕܝܠܟ ܘܒܒܤܪ ܘܒܡܪܢ ܀</p> 
 <p><a name="an_1:17">1:17</a> ܐܢ ܗܟܝܠ ܐܝܬܝܟ ܠܝ ܫܘܬܦܐ ܩܒܠܝܗܝ ܐܝܟ ܕܠܝ ܀</p> 
 <p><a name="an_1:18">1:18</a> ܘܐܢ ܡܕܡ ܚܤܪܟ ܐܘ ܚܝܒ ܗܕܐ ܥܠܝ ܚܫܘܒ ܀</p> 
 <p><a name="an_1:19">1:19</a> ܐܢܐ ܦܘܠܘܤ ܟܬܒܬ ܒܐܝܕܝ ܐܢܐ ܦܪܥ ܐܢܐ ܕܠܐ ܐܡܪ ܠܟ ܕܐܦ ܢܦܫܟ ܚܝܒ ܐܢܬ ܠܝ ܀</p> 
 <p><a name="an_1:20">1:20</a> ܐܝܢ ܐܚܝ ܐܢܐ ܐܬܬܢܝܚ ܒܟ ܒܡܪܢ ܐܢܝܚ ܪܚܡܝ ܒܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:21">1:21</a> ܡܛܠ ܕܬܟܝܠ ܐܢܐ ܕܫܡܥ ܐܢܬ ܠܝ ܟܬܒܬ ܠܟ ܘܝܕܥ ܐܢܐ ܕܝܬܝܪ ܡܢ ܡܐ ܕܐܡܪ ܐܢܐ ܥܒܕ ܐܢܬ ܀</p> 
 <p><a name="an_1:22">1:22</a> ܒܚܕܐ ܕܝܢ ܐܦ ܛܝܒ ܠܝ ܒܝܬ ܡܫܪܝܐ ܡܤܒܪ ܐܢܐ ܓܝܪ ܕܒܨܠܘܬܟܘܢ ܡܬܝܗܒ ܐܢܐ ܠܟܘܢ ܀</p> 
 <p><a name="an_1:23">1:23</a> ܫܐܠ ܒܫܠܡܟ ܐܦܦܪܐ ܫܒܝܐ ܕܥܡܝ ܒܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:24">1:24</a> ܘܡܪܩܘܤ ܘܐܪܤܛܪܟܘܤ ܘܕܡܐ ܘܠܘܩܐ ܡܥܕܪܢܝ ܀</p> 
 <p><a name="an_1:25">1:25</a> ܛܝܒܘܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܥܡ ܪܘܚܟܘܢ ܐܡܝܢ ܀</p>
