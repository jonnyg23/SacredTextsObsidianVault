  <h3 align="center">Aramaic Bible: 1 Timothy</h3> 
 <p><a name="an_1:1">1:1</a> ܦܘܠܘܤ ܫܠܝܚܐ ܕܝܫܘܥ ܡܫܝܚܐ ܒܦܘܩܕܢܐ ܕܐܠܗܐ ܡܚܝܢܢ ܘܕܡܫܝܚܐ ܝܫܘܥ ܤܒܪܢ ܀</p> 
 <p><a name="an_1:2">1:2</a> ܠܛܝܡܬܐܘܤ ܒܪܐ ܫܪܝܪܐ ܒܗܝܡܢܘܬܐ ܛܝܒܘܬܐ ܘܪܚܡܐ ܘܫܠܡܐ ܡܢ ܐܠܗܐ ܐܒܘܢ ܘܡܫܝܚܐ ܝܫܘܥ ܡܪܢ ܀</p> 
 <p><a name="an_1:3">1:3</a> ܒܥܝܬ ܗܘܝܬ ܡܢܟ ܟܕ ܐܙܠ ܐܢܐ ܠܡܩܕܘܢܝܐ ܕܬܩܘܐ ܒܐܦܤܘܤ ܘܬܦܩܕ ܠܐܢܫ ܐܢܫ ܕܠܐ ܢܠܦܘܢ ܝܘܠܦܢܐ ܡܫܚܠܦܐ ܀</p> 
 <p><a name="an_1:4">1:4</a> ܘܠܐ ܢܬܪܡܘܢ ܠܫܘܥܝܬܐ ܘܠܬܫܥܝܬܐ ܕܫܪܒܬܐ ܕܤܟܐ ܠܝܬ ܠܗܘܢ ܗܠܝܢ ܕܚܪܝܢܐ ܗܘ ܥܒܕܢ ܝܬܝܪܐܝܬ ܘܠܐ ܒܢܝܢܐ ܒܗܝܡܢܘܬܐ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_1:5">1:5</a> ܤܟܗ ܕܝܢ ܕܦܘܩܕܢܐ ܐܝܬܘܗܝ ܚܘܒܐ ܕܡܢ ܠܒܐ ܕܟܝܐ ܘܡܢ ܬܐܪܬܐ ܛܒܬܐ ܘܡܢ ܗܝܡܢܘܬܐ ܫܪܝܪܬܐ ܀</p> 
 <p><a name="an_1:6">1:6</a> ܘܡܢܗܝܢ ܡܢ ܗܠܝܢ ܛܥܘ ܐܢܫܐ ܘܤܛܘ ܠܡܠܐ ܤܪܝܩܬܐ ܀</p> 
 <p><a name="an_1:7">1:7</a> ܒܕܒܥܘ ܠܡܗܘܐ ܡܠܦܢܐ ܕܢܡܘܤܐ ܟܕ ܠܐ ܡܤܬܟܠܝܢ ܡܕܡ ܕܡܡܠܠܝܢ ܘܠܐ ܡܕܡ ܕܥܠܘܗܝ ܡܬܚܪܝܢ ܀</p> 
 <p><a name="an_1:8">1:8</a> ܝܕܥܝܢܢ ܕܝܢ ܕܢܡܘܤܐ ܫܦܝܪ ܗܘ ܐܢ ܐܢܫ ܐܟܘܬܗ ܕܢܡܘܤܐ ܢܬܕܒܪ ܒܗ ܀</p> 
 <p><a name="an_1:9">1:9</a> ܟܕ ܝܕܥ ܕܠܟܐܢܐ ܢܡܘܤܐ ܠܐ ܤܝܡ ܐܠܐ ܠܥܘܠܐ ܘܠܡܪܘܕܐ ܘܠܪܫܝܥܐ ܘܠܚܛܝܐ ܘܠܥܢܬܐ ܘܠܐܝܠܝܢ ܕܠܐ ܕܟܝܢ ܘܠܐܝܠܝܢ ܕܡܚܝܢ ܠܐܒܗܝܗܘܢ ܘܠܐܝܠܝܢ ܕܡܚܝܢ ܠܐܡܗܬܗܘܢ ܘܠܩܛܘܠܐ ܀</p> 
 <p><a name="an_1:10">1:10</a> ܘܠܙܢܝܐ ܘܠܫܟܒܝ ܥܡ ܕܟܪܐ ܘܠܓܢܒܝ ܒܢܝ ܚܐܪܐ ܘܠܕܓܠܐ ܘܠܥܒܪܝ ܥܠ ܡܘܡܬܐ ܘܠܟܠܡܕܡ ܕܐܝܬܘܗܝ ܤܩܘܒܠܐ ܠܝܘܠܦܢܐ ܚܠܝܡܐ ܀</p> 
 <p><a name="an_1:11">1:11</a> ܕܐܘܢܓܠܝܘܢ ܕܬܫܒܘܚܬܗ ܕܐܠܗܐ ܡܒܪܟܐ ܗܘ ܕܐܢܐ ܐܬܗܝܡܢܬ ܀</p> 
 <p><a name="an_1:12">1:12</a> ܘܡܘܕܐ ܐܢܐ ܠܗ ܕܚܝܠܢܝ ܠܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܕܚܫܒܢܝ ܡܗܝܡܢܐ ܘܤܡܢܝ ܠܬܫܡܫܬܗ ܀</p> 
 <p><a name="an_1:13">1:13</a> ܠܝ ܕܡܢ ܩܕܝܡ ܡܓܕܦܢ ܗܘܝܬ ܘܪܕܘܦ ܘܡܨܥܪܢ ܐܠܐ ܐܬܚܢܢܬ ܡܛܠ ܕܟܕ ܠܐ ܝܕܥ ܗܘܝܬ ܥܒܕܬ ܕܠܐ ܗܝܡܢܘܬܐ ܀</p> 
 <p><a name="an_1:14">1:14</a> ܤܓܝܬ ܒܝ ܕܝܢ ܛܝܒܘܬܗ ܕܡܪܢ ܘܗܝܡܢܘܬܐ ܘܚܘܒܐ ܕܒܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:15">1:15</a> ܡܗܝܡܢܐ ܗܝ ܡܠܬܐ ܘܫܘܝܐ ܗܝ ܠܡܩܒܠܘܬܗ ܕܝܫܘܥ ܡܫܝܚܐ ܐܬܐ ܠܥܠܡܐ ܠܡܚܝܘ ܠܚܛܝܐ ܕܩܕܡܝܗܘܢ ܐܢܐ ܐܢܐ ܀</p> 
 <p><a name="an_1:16">1:16</a> ܐܠܐ ܡܛܠ ܗܢܐ ܪܚܡ ܥܠܝ ܕܒܝ ܩܕܡܐ ܢܚܘܐ ܝܫܘܥ ܡܫܝܚܐ ܟܠܗ ܡܓܪܬ ܪܘܚܗ ܠܬܚܘܝܬܐ ܕܐܝܠܝܢ ܕܥܬܝܕܝܢ ܕܢܗܝܡܢܘܢ ܒܗ ܠܚܝܐ ܕܠܥܠܡ ܀</p> 
 <p><a name="an_1:17">1:17</a> ܠܡܠܟܐ ܕܝܢ ܕܥܠܡܐ ܠܗܘ ܕܠܐ ܡܬܚܒܠ ܘܠܐ ܡܬܚܙܐ ܕܗܘܝܘ ܚܕ ܐܠܗܐ ܐܝܩܪܐ ܘܬܫܒܘܚܬܐ ܠܥܠܡ ܥܠܡܝܢ ܐܡܝܢ ܀</p> 
 <p><a name="an_1:18">1:18</a> ܗܢܐ ܦܘܩܕܢܐ ܡܓܥܠ ܐܢܐ ܠܟ ܒܪܝ ܛܝܡܬܐܘܤ ܐܝܟ ܢܒܝܘܬܐ ܩܕܡܝܬܐ ܕܗܘܝ ܥܠܝܟ ܕܬܦܠܘܚ ܒܗܝܢ ܦܠܚܘܬܐ ܗܕܐ ܫܦܝܪܬܐ ܀</p> 
 <p><a name="an_1:19">1:19</a> ܒܗܝܡܢܘܬܐ ܘܒܬܐܪܬܐ ܛܒܬܐ ܐܝܠܝܢ ܓܝܪ ܕܠܗܕܐ ܕܚܩܘ ܡܢܗܘܢ ܡܢ ܗܝܡܢܘܬܐ ܐܤܬܪܩܘ ܀</p> 
 <p><a name="an_1:20">1:20</a> ܐܝܟ ܗܘܡܢܐܘܤ ܘܐܠܟܤܢܕܪܘܤ ܗܠܝܢ ܕܐܫܠܡܬ ܠܤܛܢܐ ܕܢܬܪܕܘܢ ܕܠܐ ܢܗܘܘܢ ܡܓܕܦܝܢ ܀</p> 
 <p><a name="an_2:1">2:1</a> ܒܥܐ ܐܢܐ ܗܟܝܠ ܡܢܟ ܕܡܢ ܩܕܡ ܟܠܡܕܡ ܒܥܘܬܐ ܬܗܘܐ ܡܩܪܒ ܠܐܠܗܐ ܘܨܠܘܬܐ ܘܬܚܢܢܬܐ ܘܬܘܕܝܬܐ ܚܠܦ ܟܠܗܘܢ ܒܢܝܢܫܐ ܀</p> 
 <p><a name="an_2:2">2:2</a> ܚܠܦ ܡܠܟܐ ܘܪܘܪܒܢܐ ܕܥܘܡܪܐ ܫܠܝܐ ܘܢܝܚܐ ܢܥܡܪ ܒܟܠܗ ܕܚܠܬ ܐܠܗܐ ܘܕܟܝܘܬܐ ܀</p> 
 <p><a name="an_2:3">2:3</a> ܗܕܐ ܓܝܪ ܫܦܝܪܐ ܘܡܩܒܠܐ ܩܕܡ ܐܠܗܐ ܡܚܝܢܢ ܀</p> 
 <p><a name="an_2:4">2:4</a> ܗܘ ܕܨܒܐ ܕܟܠܗܘܢ ܒܢܝܢܫܐ ܢܚܘܢ ܘܢܬܦܢܘܢ ܠܝܕܥܬܐ ܕܫܪܪܐ ܀</p> 
 <p><a name="an_2:5">2:5</a> ܚܕ ܗܘ ܓܝܪ ܐܠܗܐ ܘܚܕ ܗܘ ܡܨܥܝܐ ܕܐܠܗܐ ܘܕܒܢܝܢܫܐ ܒܪܢܫܐ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_2:6">2:6</a> ܗܘ ܕܝܗܒ ܢܦܫܗ ܦܘܪܩܢܐ ܚܠܦ ܟܠ ܐܢܫ ܤܗܕܘܬܐ ܕܐܬܬ ܒܙܒܢܗ ܀</p> 
 <p><a name="an_2:7">2:7</a> ܗܝ ܕܐܢܐ ܐܬܤܝܡܬ ܟܪܘܙܗ ܘܫܠܝܚܗ ܩܘܫܬܐ ܐܡܪ ܐܢܐ ܘܠܐ ܡܕܓܠ ܐܢܐ ܕܗܘܝܬ ܡܠܦܢܐ ܕܥܡܡܐ ܒܗܝܡܢܘܬܐ ܕܫܪܪܐ ܀</p> 
 <p><a name="an_2:8">2:8</a> ܨܒܐ ܐܢܐ ܗܟܝܠ ܕܗܘܘ ܡܨܠܝܢ ܓܒܪܐ ܒܟܠ ܕܘܟܐ ܟܕ ܡܪܝܡܝܢ ܐܝܕܝܗܘܢ ܕܟܝܐܝܬ ܕܠܐ ܪܘܓܙܐ ܘܕܠܐ ܡܚܫܒܬܐ ܀</p> 
 <p><a name="an_2:9">2:9</a> ܗܟܢܐ ܐܦ ܢܫܐ ܒܐܤܟܡܐ ܢܟܦܐ ܕܠܒܘܫܐ ܒܬܚܡܨܬܐ ܘܒܢܟܦܘܬܐ ܢܗܘܐ ܬܨܒܝܬܗܝܢ ܠܐ ܒܓܕܘܠܐ ܘܒܕܗܒܐ ܐܘ ܒܡܪܓܢܝܬܐ ܐܘ ܒܢܚܬܐ ܫܦܝܪܐ ܀</p> 
 <p><a name="an_2:10">2:10</a> ܐܠܐ ܒܥܒܕܐ ܛܒܐ ܐܝܟܢܐ ܕܝܐܐ ܠܢܫܐ ܕܡܫܬܘܕܝܢ ܕܚܠܬ ܐܠܗܐ ܀</p> 
 <p><a name="an_2:11">2:11</a> ܐܢܬܬܐ ܒܫܠܝܐ ܗܘܬ ܝܠܦܐ ܒܟܠ ܫܘܥܒܕ ܀</p> 
 <p><a name="an_2:12">2:12</a> ܠܐܢܬܬܐ ܓܝܪ ܠܡܠܦܘ ܠܐ ܡܦܤ ܐܢܐ ܘܠܐ ܠܡܡܪܚܘ ܥܠ ܓܒܪܐ ܐܠܐ ܬܗܘܐ ܒܫܠܝܐ ܀</p> 
 <p><a name="an_2:13">2:13</a> ܐܕܡ ܓܝܪ ܐܬܓܒܠ ܠܘܩܕܡ ܘܗܝܕܝܢ ܚܘܐ ܀</p> 
 <p><a name="an_2:14">2:14</a> ܘܐܕܡ ܠܐ ܛܥܐ ܐܢܬܬܐ ܕܝܢ ܛܥܬ ܘܥܒܪܬ ܥܠ ܦܘܩܕܢܐ ܀</p> 
 <p><a name="an_2:15">2:15</a> ܚܝܐ ܕܝܢ ܒܝܕ ܝܠܕܝܗ ܐܢ ܢܩܘܘܢ ܒܗܝܡܢܘܬܐ ܘܒܚܘܒܐ ܘܒܩܕܝܫܘܬܐ ܘܒܢܟܦܘܬܐ ܀</p> 
 <p><a name="an_3:1">3:1</a> ܡܗܝܡܢܐ ܗܝ ܡܠܬܐ ܕܐܢ ܐܢܫ ܪܐܓ ܩܫܝܫܘܬܐ ܥܒܕܐ ܛܒܐ ܪܐܓ ܀</p> 
 <p><a name="an_3:2">3:2</a> ܘܠܐ ܕܝܢ ܕܢܗܘܐ ܩܫܝܫܐ ܐܝܢܐ ܕܡܘܡܐ ܠܐ ܡܫܬܟܚ ܒܗ ܘܗܘܐ ܒܥܠܐ ܕܚܕܐ ܐܢܬܬܐ ܐܝܢܐ ܕܥܝܪ ܪܥܝܢܗ ܘܢܟܦ ܘܡܛܟܤ ܘܪܚܡ ܐܟܤܢܝܐ ܘܡܠܦܢ ܀</p> 
 <p><a name="an_3:3">3:3</a> ܘܠܐ ܥܒܪ ܥܠ ܚܡܪܐ ܘܠܐ ܪܗܛܐ ܐܝܕܗ ܠܡܡܚܐ ܐܠܐ ܢܗܘܐ ܡܟܝܟ ܘܠܐ ܢܨܝ ܘܠܐ ܪܚܡ ܟܤܦܐ ܀</p> 
 <p><a name="an_3:4">3:4</a> ܘܡܕܒܪ ܒܝܬܗ ܫܦܝܪ ܘܐܚܝܕ ܒܢܘܗܝ ܒܫܘܥܒܕܐ ܒܟܠܗ ܕܟܝܘܬܐ ܀</p> 
 <p><a name="an_3:5">3:5</a> ܐܢ ܓܝܪ ܒܝܬܐ ܕܢܦܫܗ ܠܐ ܝܕܥ ܕܢܕܒܪ ܫܦܝܪ ܐܝܟܢܐ ܡܫܟܚ ܥܕܬܗ ܕܐܠܗܐ ܕܢܕܒܪ ܀</p> 
 <p><a name="an_3:6">3:6</a> ܘܠܐ ܢܗܘܐ ܛܠܐ ܬܘܠܡܕܗ ܕܠܐ ܢܬܪܝܡ ܘܢܦܠ ܒܕܝܢܗ ܕܤܛܢܐ ܀</p> 
 <p><a name="an_3:7">3:7</a> ܘܠܐ ܕܝܢ ܕܐܦ ܤܗܕܘܬܐ ܫܦܝܪܬܐ ܐܝܬ ܠܗ ܡܢ ܒܪܝܐ ܕܠܐ ܢܦܠ ܒܚܤܕܐ ܘܒܦܚܐ ܕܤܛܢܐ ܀</p> 
 <p><a name="an_3:8">3:8</a> ܘܐܦ ܡܫܡܫܢܐ ܗܟܢܐ ܢܗܘܘܢ ܕܟܝܢ ܘܠܐ ܢܡܠܠܘܢ ܬܪܬܝܢ ܘܠܐ ܢܗܘܘܢ ܨܠܝܢ ܠܚܡܪܐ ܤܓܝܐܐ ܘܠܐ ܢܪܚܡܘܢ ܝܘܬܪܢܐ ܛܢܦܐ ܀</p> 
 <p><a name="an_3:9">3:9</a> ܐܠܐ ܢܐܚܕܘܢ ܐܪܙܐ ܕܗܝܡܢܘܬܐ ܒܬܐܪܬܐ ܕܟܝܬܐ ܀</p> 
 <p><a name="an_3:10">3:10</a> ܘܗܢܘܢ ܗܠܝܢ ܢܬܒܩܘܢ ܠܘܩܕܡ ܘܗܝܕܝܢ ܢܫܡܫܘܢ ܟܕ ܐܝܬܝܗܘܢ ܕܠܐ ܪܫܝܢ ܀</p> 
 <p><a name="an_3:11">3:11</a> ܗܟܢܐ ܐܦ ܢܫܐ ܢܗܘܝܢ ܢܟܦܢ ܘܢܗܘܐ ܥܝܪ ܪܥܝܢܗܝܢ ܘܢܗܘܝܢ ܡܗܝܡܢܢ ܒܟܠ ܡܕܡ ܘܠܐ ܢܗܘܝܢ ܐܟܠܢ ܩܪܨܐ ܀</p> 
 <p><a name="an_3:12">3:12</a> ܡܫܡܫܢܐ ܢܗܘܘܢ ܐܝܢܐ ܕܚܕܐ ܐܢܬܬܐ ܗܘܬ ܠܗ ܘܕܒܪ ܒܢܘܗܝ ܘܒܝܬܗ ܫܦܝܪ ܀</p> 
 <p><a name="an_3:13">3:13</a> ܐܝܠܝܢ ܓܝܪ ܕܫܡܫܘ ܫܦܝܪ ܕܪܓܐ ܛܒܐ ܩܢܝܢ ܠܢܦܫܗܘܢ ܘܡܓܠܐ ܐܦܐ ܤܓܝܐܐ ܒܗܝܡܢܘܬܐ ܕܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_3:14">3:14</a> ܗܠܝܢ ܟܬܒ ܐܢܐ ܠܟ ܟܕ ܡܤܒܪ ܐܢܐ ܕܒܥܓܠ ܐܬܐ ܐܢܐ ܠܘܬܟ ܀</p> 
 <p><a name="an_3:15">3:15</a> ܘܐܢ ܗܘ ܕܝܢ ܕܡܘܚܪ ܐܢܐ ܕܬܕܥ ܐܝܟܢܐ ܘܠܐ ܠܡܬܗܦܟܘ ܒܒܝܬܗ ܕܐܠܗܐ ܕܐܝܬܝܗ ܥܕܬܐ ܕܐܠܗܐ ܚܝܐ ܥܡܘܕܐ ܘܫܬܐܤܬܐ ܕܫܪܪܐ ܀</p> 
 <p><a name="an_3:16">3:16</a> ܘܫܪܝܪܐܝܬ ܪܒ ܗܘ ܐܪܙܐ ܗܢܐ ܕܟܐܢܘܬܐ ܕܐܬܓܠܝ ܒܒܤܪ ܘܐܙܕܕܩ ܒܪܘܚ ܘܐܬܚܙܝ ܠܡܠܐܟܐ ܘܐܬܟܪܙ ܒܝܬ ܥܡܡܐ ܘܐܬܗܝܡܢ ܒܥܠܡܐ ܘܐܤܬܠܩ ܒܫܘܒܚܐ ܀</p> 
 <p><a name="an_4:1">4:1</a> ܪܘܚܐ ܕܝܢ ܦܫܝܩܐܝܬ ܐܡܪܐ ܕܒܙܒܢܐ ܐܚܪܝܐ ܢܦܪܩܘܢ ܐܢܫ ܐܢܫ ܡܢ ܗܝܡܢܘܬܐ ܘܢܐܙܠܘܢ ܒܬܪ ܪܘܚܐ ܛܥܝܬܐ ܘܒܬܪ ܝܘܠܦܢܐ ܕܫܐܕܐ ܀</p> 
 <p><a name="an_4:2">4:2</a> ܗܠܝܢ ܕܒܐܤܟܡܐ ܕܓܠܐ ܡܛܥܝܢ ܘܡܡܠܠܝܢ ܟܕܒܘܬܐ ܘܟܘܝܢ ܒܬܐܪܬܗܘܢ ܀</p> 
 <p><a name="an_4:3">4:3</a> ܘܟܠܝܢ ܠܡܙܕܘܓܘ ܘܡܦܪܩܝܢ ܡܢ ܡܐܟܠܬܐ ܐܝܠܝܢ ܕܐܠܗܐ ܒܪܐ ܐܢܝܢ ܠܚܘܫܚܐ ܘܠܬܘܕܝܬܐ ܠܐܝܠܝܢ ܕܡܗܝܡܢܝܢ ܘܝܕܥܝܢ ܫܪܪܐ ܀</p> 
 <p><a name="an_4:4">4:4</a> ܡܛܠ ܕܟܠ ܕܒܪܐ ܠܐܠܗܐ ܫܦܝܪ ܗܘ ܘܠܝܬ ܡܕܡ ܕܡܤܠܝ ܐܢ ܒܬܘܕܝܬܐ ܢܬܢܤܒ ܀</p> 
 <p><a name="an_4:5">4:5</a> ܡܬܩܕܫ ܓܝܪ ܒܡܠܬܐ ܕܐܠܗܐ ܘܒܨܠܘܬܐ ܀</p> 
 <p><a name="an_4:6">4:6</a> ܗܠܝܢ ܐܢ ܬܗܘܐ ܡܠܦ ܠܐܚܝܟ ܡܫܡܫܢܐ ܛܒܐ ܬܗܘܐ ܕܝܫܘܥ ܡܫܝܚܐ ܟܕ ܡܬܪܒܐ ܐܢܬ ܒܡܠܐ ܕܗܝܡܢܘܬܐ ܘܒܝܘܠܦܢܐ ܛܒܐ ܕܝܠܝܦ ܐܢܬ ܀</p> 
 <p><a name="an_4:7">4:7</a> ܡܢ ܫܘܥܝܬܐ ܕܝܢ ܦܟܝܗܬܐ ܕܤܒܬܐ ܐܫܬܐܠ ܘܕܪܫ ܢܦܫܟ ܒܟܐܢܘܬܐ ܀</p> 
 <p><a name="an_4:8">4:8</a> ܕܘܪܫܐ ܓܝܪ ܕܦܓܪܐ ܙܒܢܐ ܗܘ ܙܥܘܪܐ ܡܘܬܪ ܟܐܢܘܬܐ ܕܝܢ ܒܟܠ ܡܕܡ ܡܘܬܪܐ ܘܐܝܬ ܠܗ ܫܘܘܕܝܐ ܕܚܝܐ ܕܙܒܢܐ ܗܢܐ ܘܕܥܬܝܕ ܀</p> 
 <p><a name="an_4:9">4:9</a> ܡܗܝܡܢܐ ܗܝ ܡܠܬܐ ܘܫܘܝܐ ܗܝ ܠܡܩܒܠܘܬܗ ܀</p> 
 <p><a name="an_4:10">4:10</a> ܡܛܠ ܗܕܐ ܗܘ ܓܝܪ ܠܐܝܢܢ ܘܡܬܚܤܕܝܢܢ ܕܡܤܒܪܝܢܢ ܒܐܠܗܐ ܚܝܐ ܕܐܝܬܘܗܝ ܡܚܝܢܐ ܕܒܢܝܢܫܐ ܟܠܗܘܢ ܝܬܝܪܐܝܬ ܕܡܗܝܡܢܐ ܀</p> 
 <p><a name="an_4:11">4:11</a> ܗܠܝܢ ܐܠܦ ܘܦܩܕ ܀</p> 
 <p><a name="an_4:12">4:12</a> ܘܠܐ ܐܢܫ ܢܒܤܐ ܥܠ ܛܠܝܘܬܟ ܐܠܐ ܗܘܝ ܕܡܘܬܐ ܠܡܗܝܡܢܐ ܒܡܠܬܐ ܘܒܗܘܦܟܐ ܘܒܚܘܒܐ ܘܒܗܝܡܢܘܬܐ ܘܒܕܟܝܘܬܐ ܀</p> 
 <p><a name="an_4:13">4:13</a> ܥܕ ܐܬܐ ܐܢܐ ܐܬܚܦܛ ܒܩܪܝܢܐ ܘܒܒܥܘܬܐ ܘܒܝܘܠܦܢܐ ܀</p> 
 <p><a name="an_4:14">4:14</a> ܘܠܐ ܬܒܤܐ ܒܡܘܗܒܬܐ ܕܐܝܬ ܒܟ ܗܝ ܕܐܬܝܗܒܬ ܠܟ ܒܢܒܝܘܬܐ ܘܒܤܝܡ ܐܝܕܐ ܕܩܫܝܫܘܬܐ ܀</p> 
 <p><a name="an_4:15">4:15</a> ܒܗܠܝܢ ܐܬܗܓܐ ܘܒܗܝܢ ܗܘܝ ܕܬܬܝܕܥ ܠܟܠܢܫ ܕܠܩܕܡܝܟ ܐܬܐ ܐܢܬ ܀</p> 
 <p><a name="an_4:16">4:16</a> ܘܐܙܕܗܪ ܒܢܦܫܟ ܘܒܝܘܠܦܢܟ ܘܚܡܤܢ ܒܗܘܢ ܟܕ ܗܠܝܢ ܓܝܪ ܬܥܒܕ ܢܦܫܟ ܬܚܐ ܘܠܐܝܠܝܢ ܕܫܡܥܝܢ ܠܟ ܀</p> 
 <p><a name="an_5:1">5:1</a> ܒܩܫܝܫܐ ܠܐ ܬܓܥܘܪ ܐܠܐ ܐܦܝܤܝܗܝ ܐܝܟ ܕܠܐܒܐ ܘܠܐܝܠܝܢ ܕܛܠܝܢ ܐܝܟ ܕܠܐܚܝܟ ܀</p> 
 <p><a name="an_5:2">5:2</a> ܘܠܩܫܝܫܬܐ ܐܝܟ ܐܡܗܬܐ ܘܠܐܝܠܝܢ ܕܛܠܝܢ ܐܝܟ ܕܠܐܚܘܬܟ ܒܟܠܗ ܕܟܝܘܬܐ ܀</p> 
 <p><a name="an_5:3">5:3</a> ܠܐܪܡܠܬܐ ܝܩܪ ܠܐܝܠܝܢ ܕܐܪܡܠܬܐ ܐܢܝܢ ܒܫܪܪܐ ܀</p> 
 <p><a name="an_5:4">5:4</a> ܘܐܢ ܐܝܬ ܐܪܡܠܬܐ ܕܐܝܬ ܠܗ ܒܢܝܐ ܐܘ ܒܢܝ ܒܢܝܐ ܢܐܠܦܘܢ ܠܘܩܕܡ ܕܒܒܢܝ ܒܝܬܗܘܢ ܢܙܕܕܩܘܢ ܘܢܦܪܥܘܢ ܚܘܒܠܐ ܠܐܒܗܝܗܘܢ ܗܕܐ ܓܝܪ ܡܩܒܠܐ ܩܕܡ ܐܠܗܐ ܀</p> 
 <p><a name="an_5:5">5:5</a> ܐܝܕܐ ܕܝܢ ܕܫܪܝܪܐܝܬ ܐܪܡܠܬܐ ܗܝ ܘܡܫܘܚܕܬܐ ܗܕܐ ܤܒܪܗ ܥܠ ܐܠܗܐ ܗܘ ܘܐܡܝܢܐ ܒܨܠܘܬܐ ܘܒܒܥܘܬܐ ܒܠܠܝܐ ܘܒܐܝܡܡܐ ܀</p> 
 <p><a name="an_5:6">5:6</a> ܐܝܕܐ ܕܝܢ ܕܦܠܚܐ ܐܤܛܪܢܝܐ ܡܝܬܐ ܗܝ ܟܕ ܚܝܐ ܀</p> 
 <p><a name="an_5:7">5:7</a> ܗܠܝܢ ܗܘܝܬ ܡܦܩܕ ܠܗܝܢ ܕܕܠܐ ܪܫܝܢ ܢܗܘܝܢ ܀</p> 
 <p><a name="an_5:8">5:8</a> ܐܢ ܐܢܫ ܓܝܪ ܕܐܝܠܝܢ ܕܕܝܠܗ ܐܢܘܢ ܘܝܬܝܪܐܝܬ ܕܐܝܠܝܢ ܕܐܝܬܝܗܘܢ ܒܢܝ ܒܝܬܐ ܕܗܝܡܢܘܬܐ ܠܐ ܝܨܦ ܗܢܐ ܟܦܪ ܠܗ ܒܗܝܡܢܘܬܐ ܘܒܝܫ ܗܘ ܡܢ ܐܝܠܝܢ ܕܠܐ ܡܗܝܡܢܝܢ ܀</p> 
 <p><a name="an_5:9">5:9</a> ܗܘܝܬ ܗܟܝܠ ܓܒܐ ܐܪܡܠܬܐ ܐܝܕܐ ܕܠܐ ܒܨܝܪܐ ܡܢ ܫܬܝܢ ܫܢܝܢ ܐܝܕܐ ܕܠܚܕ ܗܘ ܓܒܪܐ ܗܘܬ ܀</p> 
 <p><a name="an_5:10">5:10</a> ܘܐܝܬ ܠܗ ܤܗܕܘܬܐ ܕܥܒܕܐ ܫܦܝܪܐ ܐܢ ܪܒܝܬ ܒܢܝܐ ܐܢ ܩܒܠܬ ܐܟܤܢܝܐ ܐܢ ܐܫܝܓܬ ܪܓܠܝܗܘܢ ܕܩܕܝܫܐ ܐܢ ܐܪܘܚܬ ܠܐܠܝܨܐ ܐܢ ܗܠܟܬ ܒܟܠ ܥܒܕ ܛܒ ܀</p> 
 <p><a name="an_5:11">5:11</a> ܡܢ ܐܪܡܠܬܐ ܕܝܢ ܐܝܠܝܢ ܕܛܠܝܢ ܐܫܬܐܠ ܡܨܛܪܝܢ ܠܗܝܢ ܓܝܪ ܥܠ ܡܫܝܚܐ ܘܒܥܝܢ ܠܡܗܘܐ ܠܓܒܪܐ ܀</p> 
 <p><a name="an_5:12">5:12</a> ܘܕܝܢܗܝܢ ܩܝܡ ܗܘ ܕܛܠܡ ܗܝܡܢܘܬܗܝܢ ܩܕܡܝܬܐ ܀</p> 
 <p><a name="an_5:13">5:13</a> ܝܠܦܢ ܕܝܢ ܐܦ ܚܒܢܢܘܬܐ ܟܕ ܡܬܟܪܟܢ ܒܝܬ ܒܬܐ ܘܠܐ ܒܠܚܘܕ ܚܒܢܢܘܬܐ ܐܠܐ ܐܦ ܕܢܤܓܝܢ ܡܡܠܠܐ ܘܢܦܪܩܢ ܤܪܝܩܬܐ ܘܢܡܠܠܢ ܡܕܡ ܕܠܐ ܘܠܐ ܀</p> 
 <p><a name="an_5:14">5:14</a> ܨܒܐ ܐܢܐ ܗܟܝܠ ܕܐܝܠܝܢ ܕܛܠܝܢ ܢܙܕܘܓܢ ܘܢܐܠܕܢ ܒܢܝܐ ܘܢܕܒܪܢ ܒܬܝܗܝܢ ܘܠܐ ܢܬܠܢ ܠܒܥܠܕܒܒܐ ܐܦܠܐ ܚܕܐ ܥܠܬܐ ܕܨܘܚܝܬܐ ܀</p> 
 <p><a name="an_5:15">5:15</a> ܥܠ ܕܗܫܐ ܓܝܪ ܫܪܝܘ ܐܢܫ ܐܢܫ ܠܡܤܛܐ ܒܬܪ ܤܛܢܐ ܀</p> 
 <p><a name="an_5:16">5:16</a> ܐܢ ܐܢܫ ܡܗܝܡܢܐ ܐܘ ܡܗܝܡܢܬܐ ܐܝܬ ܠܗܘܢ ܐܪܡܠܬܐ ܢܬܪܤܘܢ ܐܢܝܢ ܘܠܐ ܢܐܩܪܢ ܥܠ ܥܕܬܐ ܕܠܐܝܠܝܢ ܕܐܪܡܠܬܐ ܐܢܝܢ ܕܫܪܪܐ ܬܤܦܩ ܀</p> 
 <p><a name="an_5:17">5:17</a> ܩܫܝܫܐ ܐܝܠܝܢ ܕܫܦܝܪ ܡܬܕܒܪܝܢ ܠܐܝܩܪܐ ܥܦܝܦܐ ܢܫܘܘܢ ܝܬܝܪܐܝܬ ܐܝܠܝܢ ܕܠܐܝܢ ܒܡܠܬܐ ܘܒܝܘܠܦܢܐ ܀</p> 
 <p><a name="an_5:18">5:18</a> ܐܡܪ ܓܝܪ ܟܬܒܐ ܕܠܐ ܬܒܠܘܡ ܬܘܪܐ ܒܕܪܟܬܐ ܘܫܘܐ ܗܘ ܦܥܠܐ ܐܓܪܗ ܀</p> 
 <p><a name="an_5:19">5:19</a> ܥܠ ܩܫܝܫܐ ܩܛܓܪܢܘܬܐ ܠܐ ܬܩܒܠ ܐܠܐ ܥܠ ܦܘܡ ܬܪܝܢ ܘܬܠܬܐ ܤܗܕܝܢ ܀</p> 
 <p><a name="an_5:20">5:20</a> ܠܐܝܠܝܢ ܕܚܛܝܢ ܩܕܡ ܟܠܢܫ ܟܘܢ ܕܐܦ ܫܪܟܐ ܕܐܢܫܐ ܢܕܚܠܘܢ ܀</p> 
 <p><a name="an_5:21">5:21</a> ܡܤܗܕ ܐܢܐ ܠܟ ܩܕܡ ܐܠܗܐ ܘܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܘܡܠܐܟܘܗܝ ܓܒܝܐ ܕܗܠܝܢ ܬܛܪ ܘܠܐ ܢܬܩܕܡ ܪܥܝܢܟ ܠܡܕܡ ܘܠܐ ܬܤܥܘܪ ܡܕܡ ܒܡܤܒ ܒܐܦܐ ܀</p> 
 <p><a name="an_5:22">5:22</a> ܐܝܕܐ ܒܥܓܠ ܥܠ ܐܢܫ ܠܐ ܬܤܝܡ ܘܠܐ ܬܫܬܘܬܦ ܒܚܛܗܐ ܢܘܟܪܝܐ ܢܦܫܟ ܛܪ ܒܕܟܝܘܬܐ ܀</p> 
 <p><a name="an_5:23">5:23</a> ܘܡܟܝܠ ܡܝܐ ܠܐ ܬܫܬܐ ܐܠܐ ܚܡܪܐ ܩܠܝܠ ܗܘܝܬ ܫܬܐ ܡܛܠ ܐܤܛܘܡܟܟ ܘܡܛܠ ܟܘܪܗܢܝܟ ܐܡܝܢܐ ܀</p> 
 <p><a name="an_5:24">5:24</a> ܐܝܬ ܒܢܝܢܫܐ ܕܚܛܗܝܗܘܢ ܝܕܝܥܝܢ ܐܢܘܢ ܘܡܩܕܡܝܢ ܠܗܘܢ ܠܒܝܬ ܕܝܢܐ ܘܐܝܬ ܕܒܬܪܗܘܢ ܐܙܠܝܢ ܀</p> 
 <p><a name="an_5:25">5:25</a> ܗܟܢܐ ܐܦ ܥܒܕܐ ܛܒܐ ܝܕܝܥܝܢ ܐܢܘܢ ܘܐܝܠܝܢ ܕܐܚܪܢܝܐܝܬ ܐܢܘܢ ܠܡܛܫܐ ܠܐ ܡܫܟܚܝܢ ܀</p> 
 <p><a name="an_6:1">6:1</a> ܐܝܠܝܢ ܕܐܝܬܝܗܘܢ ܬܚܝܬ ܢܝܪܐ ܕܥܒܕܘܬܐ ܠܡܪܝܗܘܢ ܒܟܠ ܐܝܩܪ ܢܐܚܕܘܢ ܕܠܐ ܢܗܘܐ ܫܡܗ ܕܐܠܗܐ ܘܝܘܠܦܢܗ ܡܬܓܕܦ ܀</p> 
 <p><a name="an_6:2">6:2</a> ܐܝܠܝܢ ܕܝܢ ܕܐܝܬ ܠܗܘܢ ܡܪܝܐ ܡܗܝܡܢܐ ܠܐ ܢܒܤܘܢ ܒܗܘܢ ܥܠ ܕܐܚܝܗܘܢ ܐܢܘܢ ܐܠܐ ܝܬܝܪܐܝܬ ܢܫܡܫܘܢ ܐܢܘܢ ܥܠ ܕܡܗܝܡܢܐ ܐܢܘܢ ܘܚܒܝܒܐ ܗܠܝܢ ܕܡܬܬܢܝܚܝܢ ܒܬܫܡܫܬܗܘܢ ܗܠܝܢ ܐܠܦ ܘܒܥܝ ܡܢܗܘܢ ܀</p> 
 <p><a name="an_6:3">6:3</a> ܐܢ ܕܝܢ ܐܝܬ ܐܢܫ ܕܡܠܦ ܝܘܠܦܢܐ ܐܚܪܢܐ ܘܠܐ ܡܬܩܪܒ ܠܡܠܐ ܚܠܝܡܬܐ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܘܠܝܘܠܦܢܐ ܕܕܚܠܬ ܐܠܗܐ ܀</p> 
 <p><a name="an_6:4">6:4</a> ܗܢܐ ܡܬܪܝܡ ܟܕ ܡܕܡ ܠܐ ܝܕܥ ܐܠܐ ܟܪܝܗ ܒܕܪܫܐ ܘܒܒܥܬܐ ܕܡܠܐ ܕܡܢܗܝܢ ܗܘܐ ܚܤܡܐ ܘܚܪܝܢܐ ܘܓܘܕܦܐ ܘܡܤܡ ܒܪܥܝܢܐ ܒܝܫܐ ܀</p> 
 <p><a name="an_6:5">6:5</a> ܘܫܚܩܐ ܕܒܢܝܢܫܐ ܐܝܠܝܢ ܕܡܚܒܠ ܪܥܝܢܗܘܢ ܘܓܠܝܙܝܢ ܡܢ ܩܘܫܬܐ ܘܤܒܪܝܢ ܕܬܓܘܪܬܐ ܗܝ ܕܚܠܬ ܐܠܗܐ ܐܢܬ ܕܝܢ ܐܬܪܚܩ ܡܢ ܗܠܝܢ ܀</p> 
 <p><a name="an_6:6">6:6</a> ܬܐܓܘܪܬܢ ܓܝܪ ܕܝܠܢ ܪܒܐ ܗܝ ܕܐܝܬܝܗ ܕܚܠܬ ܐܠܗܐ ܒܚܫܚܬܐ ܕܡܤܬܢ ܀</p> 
 <p><a name="an_6:7">6:7</a> ܡܕܡ ܓܝܪ ܠܐ ܐܥܠܢ ܠܥܠܡܐ ܘܝܕܝܥܐ ܕܐܦܠܐ ܕܢܦܩ ܡܢܗ ܡܫܟܚܝܢܢ ܀</p> 
 <p><a name="an_6:8">6:8</a> ܡܛܠ ܗܢܐ ܤܦܩܐ ܠܢ ܡܐܟܘܠܬܐ ܘܬܟܤܝܬܐ ܀</p> 
 <p><a name="an_6:9">6:9</a> ܐܝܠܝܢ ܕܝܢ ܕܨܒܝܢ ܠܡܥܬܪ ܢܦܠܝܢ ܒܢܤܝܘܢܐ ܘܒܦܚܐ ܘܒܪܓܝܓܬܐ ܤܓܝܐܬܐ ܕܤܟܠܢ ܘܡܤܓܦܢ ܘܡܛܒܥܢ ܠܒܢܝܢܫܐ ܒܚܒܠܐ ܘܒܐܒܕܢܐ ܀</p> 
 <p><a name="an_6:10">6:10</a> ܥܩܪܐ ܕܝܢ ܕܟܘܠܗܝܢ ܒܝܫܬܐ ܐܝܬܝܗ ܪܚܡܬ ܟܤܦܐ ܘܐܝܬ ܐܢܫܐ ܕܐܬܪܓܪܓܘ ܠܗ ܘܡܢ ܗܝܡܢܘܬܐ ܛܥܘ ܘܢܦܫܗܘܢ ܐܥܠܘ ܠܕܐܘܢܐ ܤܓܝܐܐ ܀</p> 
 <p><a name="an_6:11">6:11</a> ܐܢܬ ܕܝܢ ܐܘ ܒܪܢܫܐ ܕܐܠܗܐ ܡܢ ܗܠܝܢ ܥܪܘܩ ܘܗܪܛ ܒܬܪ ܙܕܝܩܘܬܐ ܘܒܬܪ ܟܐܢܘܬܐ ܘܒܬܪ ܗܝܡܢܘܬܐ ܘܒܬܪ ܚܘܒܐ ܘܒܬܪ ܡܤܝܒܪܢܘܬܐ ܘܒܬܪ ܡܟܝܟܘܬܐ ܀</p> 
 <p><a name="an_6:12">6:12</a> ܘܐܬܟܬܫ ܒܐܓܘܢܐ ܛܒܐ ܕܗܝܡܢܘܬܐ ܘܐܕܪܟ ܚܝܐ ܕܠܥܠܡ ܕܠܗܘܢ ܐܬܩܪܝܬ ܘܐܘܕܝܬ ܬܘܕܝܬܐ ܛܒܬܐ ܩܕܡ ܤܗܕܐ ܤܓܝܐܐ ܀</p> 
 <p><a name="an_6:13">6:13</a> ܡܤܗܕ ܐܢܐ ܠܟ ܩܕܡ ܐܠܗܐ ܗܘ ܕܡܚܐ ܟܠ ܘܝܫܘܥ ܡܫܝܚܐ ܗܘ ܕܐܤܗܕ ܩܕܡ ܦܢܛܝܘܤ ܦܝܠܛܘܤ ܤܗܕܘܬܐ ܫܦܝܪܬܐ ܀</p> 
 <p><a name="an_6:14">6:14</a> ܕܬܛܪܝܘܗܝ ܦܘܩܕܢܐ ܕܠܐ ܛܘܠܫܐ ܘܕܠܐ ܡܘܡܐ ܥܕܡܐ ܠܓܠܝܢܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_6:15">6:15</a> ܗܘ ܕܥܬܝܕ ܒܙܒܢܗ ܢܚܘܝܘܗܝ ܐܠܗܐ ܡܒܪܟܐ ܘܚܝܠܬܢܐ ܒܠܚܘܕܘܗܝ ܡܠܟܐ ܕܡܠܟܐ ܘܡܪܐ ܕܡܪܘܬܐ ܀</p> 
 <p><a name="an_6:16">6:16</a> ܗܘ ܕܗܘ ܒܠܚܘܕܘܗܝ ܠܐ ܡܬܚܒܠ ܘܥܡܪ ܒܢܘܗܪܐ ܕܐܢܫ ܠܐ ܡܫܟܚ ܕܢܬܩܪܒ ܠܗ ܘܐܢܫ ܡܢ ܒܢܝܢܫܐ ܠܐ ܚܙܝܗܝ ܘܐܦܠܐ ܡܨܐ ܠܡܚܙܝܗ ܗܘ ܕܠܗ ܐܝܩܪܐ ܘܫܘܠܛܢܐ ܠܥܠܡ ܥܠܡܝܢ ܐܡܝܢ ܀</p> 
 <p><a name="an_6:17">6:17</a> ܠܥܬܝܪܐ ܕܥܠܡܐ ܗܢܐ ܦܩܕ ܕܠܐ ܢܬܪܝܡܘܢ ܒܪܥܝܢܝܗܘܢ ܘܠܐ ܢܬܬܟܠܘܢ ܥܠ ܥܘܬܪܐ ܕܠܝܬ ܥܠܘܗܝ ܬܘܟܠܢܐ ܐܠܐ ܥܠ ܐܠܗܐ ܚܝܐ ܗܘ ܕܝܗܒ ܠܢ ܟܠ ܥܬܝܪܐܝܬ ܠܢܝܚܢ ܀</p> 
 <p><a name="an_6:18">6:18</a> ܘܢܥܒܕܘܢ ܥܒܕܐ ܛܒܐ ܘܢܥܬܪܘܢ ܒܤܘܥܪܢܐ ܫܦܝܪܐ ܘܢܗܘܘܢ ܕܠܝܠܝܢ ܠܡܬܠ ܘܠܡܫܬܘܬܦܘ ܀</p> 
 <p><a name="an_6:19">6:19</a> ܘܢܤܝܡܘܢ ܠܢܦܫܗܘܢ ܫܬܐܤܬܐ ܛܒܬܐ ܠܡܕܡ ܕܥܬܝܕ ܕܢܕܪܟܘܢ ܚܝܐ ܫܪܝܪܐ ܀</p> 
 <p><a name="an_6:20">6:20</a> ܐܘ ܛܝܡܬܐܐ ܐܙܕܗܪ ܒܡܕܡ ܕܐܬܓܥܠ ܠܟ ܘܥܪܘܩ ܡܢ ܒܢܬ ܩܠܐ ܤܪܝܩܬܐ ܘܡܢ ܗܦܟܬܐ ܕܝܕܥܬܐ ܕܓܠܬܐ ܀</p> 
 <p><a name="an_6:21">6:21</a> ܐܝܠܝܢ ܓܝܪ ܕܡܫܬܘܕܝܢ ܠܗ ܛܥܘ ܠܗܘܢ ܡܢ ܗܝܡܢܘܬܐ ܛܝܒܘܬܐ ܥܡܟ ܐܡܝܢ ܀</p>
