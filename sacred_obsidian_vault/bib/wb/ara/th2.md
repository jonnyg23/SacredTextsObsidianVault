  <h3 align="center">Aramaic Bible: 2 Thessalonians</h3> 
 <p><a name="an_1:1">1:1</a> ܦܘܠܘܤ ܘܤܠܘܢܘܤ ܘܛܝܡܬܐܘܤ ܠܥܕܬܐ ܕܬܤܠܘܢܝܩܝܐ ܕܒܐܠܗܐ ܐܒܘܢ ܘܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:2">1:2</a> ܛܝܒܘܬܐ ܥܡܟܘܢ ܘܫܠܡܐ ܡܢ ܐܠܗܐ ܐܒܘܢ ܘܡܢ ܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:3">1:3</a> ܠܡܘܕܝܘ ܠܐܠܗܐ ܒܟܠܙܒܢ ܚܝܒܝܢܢ ܥܠܝܟܘܢ ܐܚܝ ܐܝܟ ܡܐ ܕܘܠܐ ܕܝܬܝܪܐܝܬ ܪܒܝܐ ܗܝܡܢܘܬܟܘܢ ܘܤܓܐ ܚܘܒܐ ܕܟܠܟܘܢ ܕܟܠܢܫ ܠܘܬ ܚܒܪܗ ܀</p> 
 <p><a name="an_1:4">1:4</a> ܐܝܟ ܕܐܦ ܚܢܢ ܒܟܘܢ ܢܗܘܐ ܡܫܬܒܗܪܝܢ ܚܢܢ ܒܥܕܬܐ ܕܐܠܗܐ ܥܠ ܗܝܡܢܘܬܟܘܢ ܘܥܠ ܡܤܝܒܪܢܘܬܟܘܢ ܕܒܟܠܗ ܪܕܝܦܘܬܟܘܢ ܘܐܘܠܨܢܝܟܘܢ ܕܡܤܝܒܪܝܢ ܐܢܬܘܢ ܀</p> 
 <p><a name="an_1:5">1:5</a> ܠܬܚܘܝܬܐ ܕܕܝܢܐ ܟܐܢܐ ܕܐܠܗܐ ܕܬܫܘܘܢ ܠܡܠܟܘܬܗ ܗܝ ܕܥܠ ܐܦܝܗ ܚܫܝܬܘܢ ܀</p> 
 <p><a name="an_1:6">1:6</a> ܘܐܢ ܟܐܢܐ ܩܕܡ ܐܠܗܐ ܕܢܦܪܘܥ ܐܘܠܨܢܐ ܠܐܠܘܨܝܟܘܢ ܀</p> 
 <p><a name="an_1:7">1:7</a> ܘܠܟܘܢ ܕܡܬܐܠܨܝܬܘܢ ܢܚܐ ܥܡܢ ܒܓܠܝܢܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܕܡܢ ܫܡܝܐ ܥܡ ܚܝܠܐ ܕܡܠܐܟܘܗܝ ܀</p> 
 <p><a name="an_1:8">1:8</a> ܡܐ ܕܥܒܕ ܬܒܥܬܐ ܒܓܘܙܠܐ ܕܢܘܪܐ ܡܢ ܗܢܘܢ ܕܠܐ ܝܕܥܘ ܠܐܠܗܐ ܘܡܢ ܗܢܘܢ ܕܠܐ ܐܫܬܘܕܥܘ ܠܤܒܪܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:9">1:9</a> ܕܗܢܘܢ ܒܕܝܢܐ ܢܬܦܪܥܘܢ ܐܒܕܢܐ ܕܠܥܠܡ ܡܢ ܦܪܨܘܦܗ ܕܡܪܢ ܘܡܢ ܫܘܒܚܐ ܕܚܝܠܗ ܀</p> 
 <p><a name="an_1:10">1:10</a> ܡܐ ܕܐܬܐ ܕܢܫܬܒܚ ܒܩܕܝܫܘܗܝ ܘܢܚܘܐ ܬܕܡܪܬܗ ܒܡܗܝܡܢܘܗܝ ܕܬܬܗܝܡܢ ܤܗܕܘܬܢ ܕܥܠܝܟܘܢ ܒܗܘ ܝܘܡܐ ܀</p> 
 <p><a name="an_1:11">1:11</a> ܡܛܠܗܢܐ ܒܟܠܙܒܢ ܡܨܠܝܢܢ ܥܠܝܟܘܢ ܕܢܫܘܝܟܘܢ ܐܠܗܐ ܠܩܪܝܢܟܘܢ ܘܢܡܠܝܟܘܢ ܟܠܗ ܨܒܝܢܐ ܕܛܒܬܐ ܘܥܒܕܐ ܕܗܝܡܢܘܬܐ ܒܚܝܠܐ ܀</p> 
 <p><a name="an_1:12">1:12</a> ܕܢܫܬܒܚ ܒܟܘܢ ܫܡܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܐܦ ܐܢܬܘܢ ܒܗ ܐܝܟ ܛܝܒܘܬܗ ܕܐܠܗܢ ܘܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_2:1">2:1</a> ܒܥܝܢܢ ܕܝܢ ܡܢܟܘܢ ܐܚܝ ܥܠ ܡܐܬܝܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܘܥܠ ܟܢܘܫܝܐ ܕܝܠܢ ܕܠܘܬܗ ܀</p> 
 <p><a name="an_2:2">2:2</a> ܕܠܐ ܥܓܠ ܬܬܙܝܥܘܢ ܒܪܥܝܢܝܟܘܢ ܘܠܐ ܬܬܘܗܘܢ ܠܐ ܡܢ ܡܠܬܐ ܘܠܐ ܡܢ ܪܘܚܐ ܘܠܐ ܡܢ ܐܓܪܬܐ ܕܐܝܟ ܕܡܢ ܠܘܬܢ ܗܝ ܕܗܐ ܠܡ ܡܛܝ ܝܘܡܗ ܕܡܪܢ ܀</p> 
 <p><a name="an_2:3">2:3</a> ܠܡܐ ܐܢܫ ܢܛܥܝܟܘܢ ܒܚܕ ܡܢ ܐܤܟܡܝܢ ܡܛܠ ܕܐܢ ܠܐ ܬܐܬܐ ܠܘܩܕܡ ܡܪܘܕܘܬܐ ܘܢܬܓܠܐ ܒܪܢܫܐ ܕܚܛܝܬܐ ܒܪܗ ܕܐܒܕܢܐ ܀</p> 
 <p><a name="an_2:4">2:4</a> ܗܘ ܕܠܩܘܒܠܐ ܗܘ ܘܡܫܬܥܠܐ ܥܠ ܟܠ ܡܢ ܕܡܬܩܪܐ ܐܠܗ ܘܕܚܠܐ ܐܝܟܢܐ ܕܐܦ ܒܗܝܟܠܐ ܕܐܠܗܐ ܐܝܟ ܐܠܗܐ ܢܬܒ ܘܢܚܘܐ ܥܠ ܢܦܫܗ ܐܝܟ ܕܐܠܗܐ ܗܘ ܀</p> 
 <p><a name="an_2:5">2:5</a> ܠܐ ܥܗܕܝܬܘܢ ܕܟܕ ܠܘܬܟܘܢ ܗܘܝܬ ܗܠܝܢ ܐܡܪܬ ܗܘܝܬ ܠܟܘܢ ܀</p> 
 <p><a name="an_2:6">2:6</a> ܘܗܫܐ ܝܕܥܝܢ ܐܢܬܘܢ ܡܐ ܕܐܚܝܕ ܕܢܬܓܠܐ ܗܘ ܒܙܒܢܗ ܀</p> 
 <p><a name="an_2:7">2:7</a> ܐܪܙܐ ܓܝܪ ܕܥܘܠܐ ܡܢ ܟܕܘ ܫܪܝ ܠܡܬܚܦܛܘ ܒܠܚܘܕ ܐܢ ܗܘ ܡܐ ܕܗܫܐ ܐܚܝܕ ܢܫܬܩܠ ܡܢ ܡܨܥܬܐ ܀</p> 
 <p><a name="an_2:8">2:8</a> ܘܗܝܕܝܢ ܢܬܓܠܐ ܥܘܠܐ ܗܘ ܕܡܪܢ ܝܫܘܥ ܢܤܝܦܝܘܗܝ ܒܪܘܚ ܦܘܡܗ ܘܢܒܛܠܝܘܗܝ ܒܓܠܝܢܐ ܕܡܐܬܝܬܗ ܀</p> 
 <p><a name="an_2:9">2:9</a> ܡܐܬܝܬܗ ܓܝܪ ܕܗܘ ܒܡܥܒܕܢܘܬܐ ܗܝ ܕܤܛܢܐ ܒܟܠ ܚܝܠ ܘܐܬܘܬܐ ܘܬܕܡܪܬܐ ܕܓܠܬܐ ܀</p> 
 <p><a name="an_2:10">2:10</a> ܘܒܟܠܗ ܛܘܥܝܝ ܕܥܘܠܐ ܕܗܘܝܐ ܒܐܒܝܕܐ ܥܠ ܕܠܐ ܩܒܠܘ ܚܘܒܐ ܕܩܘܫܬܐ ܕܒܗ ܢܚܘܢ ܀</p> 
 <p><a name="an_2:11">2:11</a> ܡܛܠܗܢܐ ܢܫܕܪ ܠܗܘܢ ܐܠܗܐ ܡܥܒܕܢܘܬܐ ܕܛܘܥܝܝ ܕܢܗܝܡܢܘܢ ܠܫܘܩܪܐ ܀</p> 
 <p><a name="an_2:12">2:12</a> ܘܢܬܕܝܢܘܢ ܟܠܗܘܢ ܐܝܠܝܢ ܕܠܐ ܗܝܡܢܘ ܒܩܘܫܬܐ ܐܠܐ ܐܨܛܒܝܘ ܒܥܘܠܐ ܀</p> 
 <p><a name="an_2:13">2:13</a> ܚܢܢ ܕܝܢ ܚܝܒܝܢܢ ܠܡܘܕܝܘ ܠܐܠܗܐ ܒܟܠܙܒܢ ܥܠ ܐܦܝܟܘܢ ܐܚܝܢ ܚܒܝܒܘܗܝ ܕܡܪܢ ܕܓܒܟܘܢ ܐܠܗܐ ܡܢ ܪܫܝܬܐ ܠܚܝܐ ܒܩܕܝܫܘܬܐ ܕܪܘܚܐ ܘܒܗܝܡܢܘܬܐ ܕܫܪܪܐ ܀</p> 
 <p><a name="an_2:14">2:14</a> ܠܗܠܝܢ ܗܘ ܓܝܪ ܩܪܟܘܢ ܒܡܤܒܪܢܘܬܐ ܕܝܠܢ ܕܬܗܘܘܢ ܬܫܒܘܚܬܐ ܠܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_2:15">2:15</a> ܡܟܝܠ ܐܚܝ ܐܬܩܝܡܘ ܘܚܡܤܢܘ ܒܦܘܩܕܢܐ ܕܝܠܦܬܘܢ ܐܢ ܒܡܠܬܐ ܘܐܢ ܒܐܓܪܬܐ ܕܝܠܢ ܀</p> 
 <p><a name="an_2:16">2:16</a> ܗܘ ܕܝܢ ܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܘܐܠܗܐ ܐܒܘܢ ܗܘ ܕܐܚܒܢ ܘܝܗܒ ܠܢ ܒܘܝܐܐ ܕܠܥܠܡ ܘܤܒܪܐ ܛܒܐ ܒܛܝܒܘܬܗ ܀</p> 
 <p><a name="an_2:17">2:17</a> ܢܒܝܐ ܠܒܘܬܟܘܢ ܘܢܩܝܡ ܒܟܠ ܡܠܐ ܘܒܟܠ ܥܒܕ ܛܒ ܀</p> 
 <p><a name="an_3:1">3:1</a> ܡܢ ܗܫܐ ܐܚܝܢ ܨܠܘ ܥܠܝܢ ܕܡܠܬܗ ܕܡܪܢ ܬܗܘܐ ܪܗܛܐ ܘܡܫܬܒܚܐ ܒܟܠ ܕܘܟܐ ܐܝܟ ܕܠܘܬܟܘܢ ܀</p> 
 <p><a name="an_3:2">3:2</a> ܘܕܢܬܦܨܐ ܡܢ ܒܢܝܢܫܐ ܒܝܫܐ ܘܥܢܬܐ ܠܘ ܓܝܪ ܕܟܠܢܫ ܗܝ ܗܝܡܢܘܬܐ ܀</p> 
 <p><a name="an_3:3">3:3</a> ܡܗܝܡܢ ܗܘ ܕܝܢ ܡܪܝܐ ܕܗܘ ܢܢܛܪܟܘܢ ܘܢܫܘܙܒܟܘܢ ܡܢ ܒܝܫܐ ܀</p> 
 <p><a name="an_3:4">3:4</a> ܬܟܝܠܝܢܢ ܕܝܢ ܥܠܝܟܘܢ ܒܡܪܢ ܕܡܕܡ ܕܡܦܩܕܝܢܢ ܠܟܘܢ ܥܒܕܬܘܢ ܐܦ ܥܒܕܝܬܘܢ ܀</p> 
 <p><a name="an_3:5">3:5</a> ܘܡܪܢ ܢܬܪܘܨ ܠܒܘܬܟܘܢ ܠܚܘܒܗ ܕܐܠܗܐ ܘܠܡܤܒܪܢܘܬܗ ܕܡܫܝܚܐ ܀</p> 
 <p><a name="an_3:6">3:6</a> ܡܦܩܕܝܢܢ ܠܟܘܢ ܕܝܢ ܐܚܝ ܒܫܡܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܕܗܘܝܬܘܢ ܦܪܝܩܝܢ ܡܢ ܟܠ ܐܚܐ ܐܝܢܐ ܕܒܝܫ ܒܝܫ ܡܗܠܟ ܘܠܐ ܐܝܟ ܦܘܩܕܢܐ ܕܩܒܠܘ ܡܢܢ ܀</p> 
 <p><a name="an_3:7">3:7</a> ܐܢܬܘܢ ܓܝܪ ܝܕܥܝܢ ܐܢܬܘܢ ܐܝܟܢܐ ܘܠܐ ܠܡܬܕܡܝܘ ܒܢ ܕܠܐ ܗܠܟܢ ܒܝܫ ܒܝܫ ܒܝܢܬܟܘܢ ܀</p> 
 <p><a name="an_3:8">3:8</a> ܐܦܠܐ ܠܚܡܐ ܕܡܓܢ ܐܟܠܢ ܡܢ ܐܢܫ ܡܢܟܘܢ ܐܠܐ ܒܥܡܠܐ ܘܒܠܐܘܬܐ ܒܠܠܝܐ ܘܒܐܝܡܡܐ ܦܠܚܝܢ ܗܘܝܢ ܕܥܠ ܐܢܫ ܡܢܟܘܢ ܠܐ ܢܐܩܪ ܀</p> 
 <p><a name="an_3:9">3:9</a> ܠܐ ܗܘܐ ܡܛܠ ܕܠܐ ܫܠܝܛ ܠܢ ܐܠܐ ܕܒܢܦܫܢ ܢܬܠ ܠܟܘܢ ܛܘܦܤܐ ܕܒܢ ܬܬܕܡܘܢ ܀</p> 
 <p><a name="an_3:10">3:10</a> ܐܦ ܟܕ ܠܘܬܟܘܢ ܗܘܝܢ ܓܝܪ ܗܕܐ ܡܦܩܕܝܢ ܗܘܝܢ ܠܟܘܢ ܕܟܠ ܕܠܐ ܨܒܐ ܕܢܦܠܘܚ ܐܦܠܐ ܢܠܥܤ ܀</p> 
 <p><a name="an_3:11">3:11</a> ܫܡܥܝܢܢ ܓܝܪ ܕܐܝܬ ܒܟܘܢ ܐܢܫܐ ܕܒܝܫ ܒܝܫ ܡܗܠܟܝܢ ܘܡܕܡ ܠܐ ܦܠܚܝܢ ܐܠܐ ܐܢ ܤܪܝܩܬܐ ܀</p> 
 <p><a name="an_3:12">3:12</a> ܠܗܘܢ ܕܝܢ ܠܗܠܝܢ ܡܦܩܕܝܢܢ ܘܒܥܝܢܢ ܡܢܗܘܢ ܒܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܕܒܫܠܝܐ ܢܗܘܘܢ ܦܠܚܝܢ ܘܐܟܠܝܢ ܠܚܡܗܘܢ ܀</p> 
 <p><a name="an_3:13">3:13</a> ܐܢܬܘܢ ܕܝܢ ܐܚܝ ܠܐ ܬܡܐܢ ܠܟܘܢ ܠܡܥܒܕ ܕܫܦܝܪ ܀</p> 
 <p><a name="an_3:14">3:14</a> ܘܐܢ ܐܢܫ ܠܐ ܡܫܬܡܥ ܠܡܠܝܢ ܗܠܝܢ ܕܒܐܓܪܬܐ ܢܬܦܪܫ ܠܟܘܢ ܗܢܐ ܘܠܐ ܗܘܝܬܘܢ ܡܬܚܠܛܝܢ ܥܡܗ ܕܢܒܗܬ ܀</p> 
 <p><a name="an_3:15">3:15</a> ܘܠܐ ܐܝܟ ܒܥܠܕܒܒܐ ܬܐܚܕܘܢܗ ܐܠܐ ܗܘܝܬܘܢ ܡܪܬܝܢ ܠܗ ܐܝܟ ܕܠܐܚܐ ܀</p> 
 <p><a name="an_3:16">3:16</a> ܗܘ ܕܝܢ ܡܪܗ ܕܫܠܡܐ ܢܬܠ ܠܟܘܢ ܫܠܡܐ ܒܟܠܙܒܢ ܒܟܠܡܕܡ ܡܪܢ ܥܡ ܟܠܟܘܢ ܀</p> 
 <p><a name="an_3:17">3:17</a> ܫܠܡܐ ܒܟܬܒܬ ܐܝܕܝ ܐܢܐ ܦܘܠܘܤ ܟܬܒܬ ܕܐܝܬܘܗܝ ܐܬܐ ܕܒܟܠܗܝܢ ܐܓܪܬܝ ܗܟܢܐ ܟܬܒ ܐܢܐ ܀</p> 
 <p><a name="an_3:18">3:18</a> ܛܝܒܘܬܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܥܡ ܟܠܟܘܢ ܐܡܝܢ ܀</p>
