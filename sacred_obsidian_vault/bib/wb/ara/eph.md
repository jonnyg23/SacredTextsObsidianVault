  <h3 align="center">Aramaic Bible: Ephesians</h3> 
 <p><a name="an_1:1">1:1</a> ܦܘܠܘܤ ܫܠܝܚܐ ܕܝܫܘܥ ܡܫܝܚܐ ܒܨܒܝܢܗ ܕܐܠܗܐ ܠܐܝܠܝܢ ܕܐܝܬܝܗܘܢ ܒܐܦܤܘܤ ܩܕܝܫܐ ܘܡܗܝܡܢܐ ܒܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:2">1:2</a> ܫܠܡܐ ܥܡܟܘܢ ܘܛܝܒܘܬܐ ܡܢ ܐܠܗܐ ܐܒܘܢ ܘܡܢ ܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:3">1:3</a> ܡܒܪܟ ܗܘ ܐܠܗܐ ܐܒܘܗܝ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܗܘ ܕܒܪܟܢ ܒܟܠ ܒܘܪܟܢ ܕܪܘܚ ܒܫܡܝܐ ܒܡܫܝܚܐ ܀</p> 
 <p><a name="an_1:4">1:4</a> ܐܝܟܢܐ ܕܩܕܡ ܓܒܢ ܒܗ ܡܢ ܩܕܡ ܬܪܡܝܬܗ ܕܥܠܡܐ ܕܢܗܘܐ ܩܕܝܫܐ ܘܕܠܐ ܡܘܡ ܩܕܡܘܗܝ ܘܒܚܘܒܐ ܩܕܡ ܪܫܡܢ ܠܗ ܀</p> 
 <p><a name="an_1:5">1:5</a> ܘܤܡܢ ܠܒܢܝܐ ܒܝܫܘܥ ܡܫܝܚܐ ܐܝܟ ܕܫܦܪ ܠܨܒܝܢܗ ܀</p> 
 <p><a name="an_1:6">1:6</a> ܕܢܫܬܒܚ ܫܘܒܚܐ ܕܛܝܒܘܬܗ ܗܘ ܕܐܫܦܥ ܥܠܝܢ ܒܝܕ ܚܒܝܒܗ ܀</p> 
 <p><a name="an_1:7">1:7</a> ܕܒܗ ܐܝܬ ܠܢ ܦܘܪܩܢܐ ܘܒܕܡܗ ܫܘܒܩܢܐ ܕܚܛܗܐ ܐܝܟ ܥܘܬܪܐ ܕܛܝܒܘܬܗ ܀</p> 
 <p><a name="an_1:8">1:8</a> ܗܝ ܕܐܬܝܬܪܬ ܒܢ ܒܟܠ ܚܟܡܐ ܘܒܟܠ ܤܘܟܠ ܀</p> 
 <p><a name="an_1:9">1:9</a> ܘܐܘܕܥܢ ܐܪܙܐ ܕܨܒܝܢܗ ܗܘ ܕܩܕܡ ܗܘܐ ܤܡ ܕܢܤܥܘܪ ܒܗ ܀</p> 
 <p><a name="an_1:10">1:10</a> ܠܡܕܒܪܢܘܬܐ ܕܡܘܠܝܗܘܢ ܕܙܒܢܐ ܕܟܠܡܕܡ ܡܢ ܕܪܝܫ ܢܬܚܕܬ ܒܡܫܝܚܐ ܕܒܫܡܝܐ ܘܒܐܪܥܐ ܀</p> 
 <p><a name="an_1:11">1:11</a> ܘܒܗ ܚܢܢ ܐܬܓܒܝܢ ܐܝܟ ܕܩܕܡ ܪܫܡܢ ܘܨܒܐ ܗܘ ܕܟܠ ܤܥܪ ܐܝܟ ܬܪܥܝܬܐ ܕܨܒܝܢܗ ܀</p> 
 <p><a name="an_1:12">1:12</a> ܕܢܗܘܐ ܚܢܢ ܐܝܠܝܢ ܕܩܕܡܢ ܤܒܪܢ ܒܡܫܝܚܐ ܠܗܕܪܐ ܕܬܫܒܘܚܬܗ ܀</p> 
 <p><a name="an_1:13">1:13</a> ܕܐܦ ܐܢܬܘܢ ܒܗ ܫܡܥܬܘܢ ܡܠܬܐ ܕܩܘܫܬܐ ܕܐܝܬܝܗ ܤܒܪܬܐ ܕܚܝܝܟܘܢ ܘܒܗ ܗܝܡܢܬܘܢ ܘܐܬܚܬܡܬܘܢ ܒܪܘܚܐ ܕܩܘܕܫܐ ܕܡܠܝܟܐ ܗܘܬ ܀</p> 
 <p><a name="an_1:14">1:14</a> ܗܝ ܕܐܝܬܝܗ ܪܗܒܘܢܐ ܕܝܪܬܘܬܢ ܠܦܘܪܩܢܐ ܕܐܝܠܝܢ ܕܚܐܝܢ ܘܠܫܘܒܚܐ ܕܐܝܩܪܗ ܀</p> 
 <p><a name="an_1:15">1:15</a> ܡܛܠ ܗܢܐ ܐܦ ܐܢܐ ܗܐ ܡܢ ܕܫܡܥܬ ܗܝܡܢܘܬܟܘܢ ܕܒܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܘܚܘܒܟܘܢ ܕܠܘܬ ܩܕܝܫܐ ܀</p> 
 <p><a name="an_1:16">1:16</a> ܠܐ ܡܫܬܠܐ ܐܢܐ ܠܡܘܕܝܘ ܥܠ ܐܦܝܟܘܢ ܘܠܡܥܗܕܟܘܢ ܒܨܠܘܬܝ ܀</p> 
 <p><a name="an_1:17">1:17</a> ܕܐܠܗܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܐܒܐ ܕܬܫܒܘܚܬܐ ܢܬܠ ܠܟܘܢ ܪܘܚܐ ܕܚܟܡܬܐ ܘܕܓܠܝܢܐ ܒܝܕܥܬܗ ܀</p> 
 <p><a name="an_1:18">1:18</a> ܘܢܢܗܪܢ ܥܝܢܐ ܕܠܒܘܬܟܘܢ ܕܬܕܥܘܢ ܡܢܘ ܤܒܪܐ ܕܩܪܝܢܗ ܘܡܢܘ ܥܘܬܪܐ ܕܫܘܒܚܐ ܕܝܪܬܘܬܗ ܒܩܕܝܫܐ ܀</p> 
 <p><a name="an_1:19">1:19</a> ܘܡܢܐ ܗܝ ܝܬܝܪܘܬܐ ܕܪܒܘܬܐ ܕܚܝܠܗ ܒܢ ܒܐܝܠܝܢ ܕܡܗܝܡܢܝܢܢ ܐܝܟ ܡܥܒܕܢܘܬܐ ܕܬܘܩܦܐ ܕܚܝܠܗ ܀</p> 
 <p><a name="an_1:20">1:20</a> ܕܥܒܕ ܒܡܫܝܚܐ ܘܐܩܝܡܗ ܡܢ ܒܝܬ ܡܝܬܐ ܘܐܘܬܒܗ ܡܢ ܝܡܝܢܗ ܒܫܡܝܐ ܀</p> 
 <p><a name="an_1:21">1:21</a> ܠܥܠ ܡܢ ܟܠܗܝܢ ܐܪܟܘܤ ܘܫܘܠܛܢܐ ܘܚܝܠܐ ܘܡܪܘܬܐ ܘܠܥܠ ܡܢ ܟܠ ܫܡ ܕܡܫܬܡܗ ܠܐ ܒܠܚܘܕ ܒܥܠܡܐ ܗܢܐ ܐܠܐ ܐܦ ܒܕܥܬܝܕ ܀</p> 
 <p><a name="an_1:22">1:22</a> ܘܫܥܒܕ ܟܠܡܕܡ ܬܚܝܬ ܪܓܠܘܗܝ ܘܠܗ ܕܐܝܬܘܗܝ ܠܥܠ ܡܢ ܟܠ ܝܗܒܗ ܪܫܐ ܠܥܕܬܐ ܀</p> 
 <p><a name="an_1:23">1:23</a> ܕܐܝܬܝܗ ܓܘܫܡܗ ܘܫܘܡܠܝܐ ܕܗܘ ܕܟܠ ܒܟܠ ܡܫܡܠܐ ܀</p> 
 <p><a name="an_2:1">2:1</a> ܘܐܦ ܠܟܘܢ ܕܡܝܬܝܢ ܗܘܝܬܘܢ ܒܚܛܗܝܟܘܢ ܘܒܤܟܠܘܬܟܘܢ ܀</p> 
 <p><a name="an_2:2">2:2</a> ܒܗܠܝܢ ܕܡܢ ܩܕܝܡ ܗܠܟܬܘܢ ܗܘܝܬܘܢ ܒܗܝܢ ܐܝܟ ܥܠܡܝܘܬܗ ܕܥܠܡܐ ܗܢܐ ܘܐܝܟ ܨܒܝܢܗ ܕܪܝܫ ܫܘܠܛܢܐ ܕܐܐܪ ܘܕܪܘܚܐ ܗܕܐ ܕܡܬܚܦܛܐ ܒܒܢܝܐ ܕܠܐ ܡܬܛܦܝܤܢܘܬܐ ܀</p> 
 <p><a name="an_2:3">2:3</a> ܒܗܢܘܢ ܥܒܕܐ ܕܐܦ ܚܢܢ ܐܬܗܦܟܢ ܒܗܘܢ ܡܢ ܩܕܝܡ ܒܪܓܝܓܬܐ ܕܒܤܪܢ ܘܥܒܕܝܢ ܗܘܝܢ ܨܒܝܢܐ ܕܒܤܪܢ ܘܕܬܪܥܝܬܢ ܘܒܢܝܐ ܗܘܝܢ ܕܪܘܓܙܐ ܡܠܝܐܝܬ ܐܝܟ ܫܪܟܐ ܀</p> 
 <p><a name="an_2:4">2:4</a> ܐܠܗܐ ܕܝܢ ܕܥܬܝܪ ܒܪܚܡܘܗܝ ܡܛܠ ܚܘܒܗ ܤܓܝܐܐ ܕܐܚܒܢ ܀</p> 
 <p><a name="an_2:5">2:5</a> ܟܕ ܡܝܬܝܢ ܗܘܝܢ ܒܚܛܗܝܢ ܐܚܝܢ ܥܡ ܡܫܝܚܐ ܘܒܛܝܒܘܬܗ ܦܪܩܢ ܀</p> 
 <p><a name="an_2:6">2:6</a> ܘܐܩܝܡܢ ܥܡܗ ܘܐܘܬܒܢ ܥܡܗ ܒܫܡܝܐ ܒܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_2:7">2:7</a> ܕܢܚܘܐ ܠܥܠܡܐ ܕܐܬܝܢ ܪܒܘܬܐ ܕܥܘܬܪܐ ܕܛܝܒܘܬܗ ܘܒܤܝܡܘܬܗ ܕܗܘܬ ܥܠܝܢ ܒܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_2:8">2:8</a> ܒܛܝܒܘܬܗ ܗܘ ܓܝܪ ܐܬܦܪܩܢ ܒܗܝܡܢܘܬܐ ܘܗܕܐ ܠܐ ܗܘܬ ܡܢܟܘܢ ܐܠܐ ܡܘܗܒܬܐ ܗܝ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_2:9">2:9</a> ܠܐ ܡܢ ܥܒܕܐ ܕܠܐ ܐܢܫ ܢܫܬܒܗܪ ܀</p> 
 <p><a name="an_2:10">2:10</a> ܒܪܝܬܐ ܚܢܢ ܓܝܪ ܕܝܠܗ ܕܐܬܒܪܝܢ ܒܝܫܘܥ ܡܫܝܚܐ ܠܥܒܕܐ ܛܒܐ ܗܢܘܢ ܕܡܢ ܩܕܝܡ ܛܝܒ ܐܠܗܐ ܕܒܗܘܢ ܢܗܠܟ ܀</p> 
 <p><a name="an_2:11">2:11</a> ܡܛܠ ܗܕܐ ܗܘܝܬܘܢ ܥܗܕܝܢ ܕܐܢܬܘܢ ܥܡܡܐ ܡܢ ܩܕܝܡ ܕܒܤܪ ܗܘܝܬܘܢ ܘܡܬܩܪܝܢ ܗܘܝܬܘܢ ܥܘܪܠܘܬܐ ܡܢ ܗܝ ܕܡܬܩܪܝܐ ܓܙܘܪܬܐ ܘܐܝܬܝܗ ܥܒܕ ܐܝܕܝܐ ܒܒܤܪܐ ܀</p> 
 <p><a name="an_2:12">2:12</a> ܘܐܝܬܝܟܘܢ ܗܘܝܬܘܢ ܒܗܘ ܙܒܢܐ ܕܠܐ ܡܫܝܚܐ ܘܢܘܟܪܝܝܢ ܗܘܝܬܘܢ ܡܢ ܕܘܒܪܐ ܕܐܝܤܪܝܠ ܘܐܟܤܢܝܐ ܗܘ ܠܕܝܬܩܐ ܕܡܘܠܟܢܐ ܘܕܠܐ ܤܒܪ ܗܘܝܬܘܢ ܘܕܠܐ ܐܠܗ ܒܥܠܡܐ ܀</p> 
 <p><a name="an_2:13">2:13</a> ܗܫܐ ܕܝܢ ܒܝܫܘܥ ܡܫܝܚܐ ܐܢܬܘܢ ܕܡܢ ܩܕܝܡ ܪܚܝܩܝܢ ܗܘܝܬܘܢ ܩܪܝܒܐ ܗܘܝܬܘܢ ܒܕܡܗ ܕܡܫܝܚܐ ܀</p> 
 <p><a name="an_2:14">2:14</a> ܗܘܝܘ ܓܝܪ ܫܝܢܢ ܗܘ ܕܥܒܕ ܬܪܬܝܗܝܢ ܚܕܐ ܘܫܪܐ ܤܝܓܐ ܕܩܐܡ ܗܘܐ ܒܡܨܥܬܐ ܀</p> 
 <p><a name="an_2:15">2:15</a> ܘܒܥܠܕܒܒܘܬܐ ܒܒܤܪܗ ܘܢܡܘܤܐ ܕܦܘܩܕܐ ܒܦܘܩܕܢܘܗܝ ܒܛܠ ܕܠܬܪܝܗܘܢ ܢܒܪܐ ܒܩܢܘܡܗ ܠܚܕ ܒܪܢܫܐ ܚܕܬܐ ܘܥܒܕ ܫܝܢܐ ܀</p> 
 <p><a name="an_2:16">2:16</a> ܘܪܥܝ ܠܬܪܝܗܘܢ ܒܚܕ ܦܓܪ ܥܡ ܐܠܗܐ ܘܒܙܩܝܦܗ ܩܛܠ ܒܥܠܕܒܒܘܬܐ ܀</p> 
 <p><a name="an_2:17">2:17</a> ܘܐܬܐ ܤܒܪ ܫܠܡܐ ܠܟܘܢ ܠܪܚܝܩܐ ܘܠܩܪܝܒܐ ܀</p> 
 <p><a name="an_2:18">2:18</a> ܡܛܠ ܕܒܗ ܗܘ ܗܘܐ ܠܢ ܩܘܪܒܐ ܠܬܪܝܢ ܒܚܕܐ ܪܘܚ ܠܘܬ ܐܒܐ ܀</p> 
 <p><a name="an_2:19">2:19</a> ܡܟܝܠ ܠܐ ܗܘܝܬܘܢ ܐܟܤܢܝܐ ܘܠܐ ܬܘܬܒܐ ܐܠܐ ܒܢܝ ܡܕܝܢܬܐ ܕܩܕܝܫܐ ܘܒܢܝ ܒܝܬܗ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_2:20">2:20</a> ܘܐܬܒܢܝܬܘܢ ܥܠ ܫܬܐܤܬܐ ܕܫܠܝܚܐ ܘܕܢܒܝܐ ܘܗܘ ܗܘܐ ܪܝܫ ܩܪܢܐ ܕܒܢܝܢܐ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_2:21">2:21</a> ܘܒܗ ܡܬܪܟܒ ܟܠܗ ܒܢܝܢܐ ܘܪܒܐ ܠܗܝܟܠܐ ܩܕܝܫܐ ܒܡܪܝܐ ܀</p> 
 <p><a name="an_2:22">2:22</a> ܟܕ ܐܦ ܐܢܬܘܢ ܒܗ ܡܬܒܢܝܢ ܐܢܬܘܢ ܠܡܥܡܪܐ ܕܐܠܗܐ ܒܪܘܚ ܀</p> 
 <p><a name="an_3:1">3:1</a> ܡܛܠ ܗܢܐ ܐܢܐ ܦܘܠܘܤ ܐܤܝܪܐ ܐܢܐ ܕܝܫܘܥ ܡܫܝܚܐ ܥܠ ܐܦܝܟܘܢ ܥܡܡܐ ܀</p> 
 <p><a name="an_3:2">3:2</a> ܐܢ ܫܡܥܬܘܢ ܡܕܒܪܢܘܬܐ ܕܛܝܒܘܬܗ ܕܐܠܗܐ ܕܐܬܝܗܒܬ ܠܝ ܒܟܘܢ ܀</p> 
 <p><a name="an_3:3">3:3</a> ܕܒܓܠܝܢܐ ܐܬܝܕܥ ܠܝ ܐܪܙܐ ܐܝܟܢܐ ܕܟܬܒܬ ܠܟܘܢ ܒܙܥܘܪܝܬܐ ܀</p> 
 <p><a name="an_3:4">3:4</a> ܐܝܟ ܡܐ ܕܡܫܟܚܝܬܘܢ ܟܕ ܩܪܝܢ ܐܢܬܘܢ ܠܡܤܬܟܠܘ ܝܕܥܬܝ ܕܒܐܪܙܗ ܕܡܫܝܚܐ ܀</p> 
 <p><a name="an_3:5">3:5</a> ܗܘ ܕܒܕܪܐ ܐܚܪܢܐ ܠܐ ܐܬܝܕܥ ܠܒܢܝܢܫܐ ܐܝܟ ܕܗܫܐ ܐܬܓܠܝ ܠܫܠܝܚܘܗܝ ܩܕܝܫܐ ܘܠܢܒܝܘܗܝ ܒܪܘܚ ܀</p> 
 <p><a name="an_3:6">3:6</a> ܕܢܗܘܘܢ ܥܡܡܐ ܒܢܝ ܝܪܬܘܬܗ ܘܫܘܬܦܐ ܕܦܓܪܗ ܘܕܡܘܠܟܢܐ ܕܐܬܝܗܒ ܒܗ ܒܝܕ ܐܘܢܓܠܝܘܢ ܀</p> 
 <p><a name="an_3:7">3:7</a> ܗܘ ܕܐܢܐ ܗܘܝܬ ܡܫܡܫܢܗ ܐܝܟ ܡܘܗܒܬܐ ܕܛܝܒܘܬܗ ܕܐܠܗܐ ܕܐܬܝܗܒܬ ܠܝ ܡܢ ܡܥܒܕܢܘܬܐ ܕܚܝܠܗ ܀</p> 
 <p><a name="an_3:8">3:8</a> ܠܝ ܕܙܥܘܪܐ ܐܢܐ ܕܟܠܗܘܢ ܩܕܝܫܐ ܐܬܝܗܒܬ ܛܝܒܘܬܐ ܗܕܐ ܕܐܤܒܪ ܒܥܡܡܐ ܥܘܬܪܗ ܕܡܫܝܚܐ ܗܘ ܕܠܐ ܡܬܥܩܒ ܀</p> 
 <p><a name="an_3:9">3:9</a> ܘܐܢܗܪ ܠܟܠܢܫ ܐܝܕܐ ܗܝ ܡܕܒܪܢܘܬܐ ܕܐܪܙܐ ܗܘ ܕܟܤܐ ܗܘܐ ܡܢ ܥܠܡܐ ܒܐܠܗܐ ܕܟܠ ܒܪܐ ܀</p> 
 <p><a name="an_3:10">3:10</a> ܕܒܝܕ ܥܕܬܐ ܬܬܝܕܥ ܚܟܡܬܗ ܕܐܠܗܐ ܡܠܝܬ ܦܘܪܫܢܐ ܠܐܪܟܘܤ ܘܠܫܘܠܛܢܐ ܕܒܫܡܝܐ ܀</p> 
 <p><a name="an_3:11">3:11</a> ܗܝ ܕܥܬܕ ܗܘܐ ܡܢ ܥܠܡܐ ܘܥܒܕܗ ܒܝܫܘܥ ܡܫܝܚܐ ܡܪܢ ܀</p> 
 <p><a name="an_3:12">3:12</a> ܗܘ ܕܒܗ ܐܝܬ ܠܢ ܦܪܗܤܝܐ ܘܩܪܝܒܘܬܐ ܒܬܘܟܠܢܐ ܕܗܝܡܢܘܬܗ ܀</p> 
 <p><a name="an_3:13">3:13</a> ܡܛܠ ܗܢܐ ܫܐܠܢܐ ܕܠܐ ܬܡܐܢ ܠܝ ܒܐܘܠܨܢܝ ܕܥܠ ܐܦܝܟܘܢ ܕܗܕܐ ܗܝ ܬܫܒܘܚܬܟܘܢ ܀</p> 
 <p><a name="an_3:14">3:14</a> ܘܟܐܦܢܐ ܒܘܪܟܝ ܠܘܬ ܐܒܘܗܝ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_3:15">3:15</a> ܗܘ ܕܡܢܗ ܡܫܬܡܗܐ ܟܠ ܐܒܗܘܬܐ ܕܒܫܡܝܐ ܘܒܐܪܥܐ ܀</p> 
 <p><a name="an_3:16">3:16</a> ܕܢܬܠ ܠܟܘܢ ܐܝܟ ܥܘܬܪܐ ܕܬܫܒܘܚܬܗ ܕܒܚܝܠܐ ܬܫܬܪܪܘܢ ܒܪܘܚܗ ܕܒܒܪܢܫܟܘܢ ܕܠܓܘ ܀</p> 
 <p><a name="an_3:17">3:17</a> ܢܥܡܪ ܡܫܝܚܐ ܒܗܝܡܢܘܬܐ ܘܒܠܒܘܬܟܘܢ ܒܚܘܒܐ ܟܕ ܢܗܘܐ ܫܪܝܪ ܥܩܪܟܘܢ ܘܫܬܐܤܬܟܘܢ ܀</p> 
 <p><a name="an_3:18">3:18</a> ܕܬܫܟܚܘܢ ܠܡܕܪܟܘ ܥܡ ܟܠܗܘܢ ܩܕܝܫܐ ܡܢܘ ܪܘܡܐ ܘܥܘܡܩܐ ܘܐܘܪܟܐ ܘܦܬܝܐ ܀</p> 
 <p><a name="an_3:19">3:19</a> ܘܬܕܥܘܢ ܪܒܘܬܐ ܕܝܕܥܬܐ ܕܚܘܒܗ ܕܡܫܝܚܐ ܘܬܬܡܠܘܢ ܒܟܠܗ ܡܘܠܝܐ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_3:20">3:20</a> ܠܗܘ ܕܝܢ ܕܡܨܐ ܒܚܝܠܐ ܝܬܝܪ ܡܢ ܟܠ ܠܡܥܒܕ ܠܢ ܘܝܬܝܪ ܡܢ ܡܐ ܕܫܐܠܝܢܢ ܘܪܢܝܢܢ ܐܝܟ ܚܝܠܗ ܕܡܤܬܥܪ ܒܢ ܀</p> 
 <p><a name="an_3:21">3:21</a> ܠܗ ܬܫܒܘܚܬܐ ܒܥܕܬܗ ܒܝܫܘܥ ܡܫܝܚܐ ܒܟܠܗܘܢ ܕܪܐ ܕܥܠܡܝ ܥܠܡܝܢ ܐܡܝܢ ܀</p> 
 <p><a name="an_4:1">4:1</a> ܒܥܐ ܐܢܐ ܗܟܝܠ ܡܢܟܘܢ ܐܢܐ ܐܤܝܪܐ ܒܡܪܢ ܕܬܗܠܟܘܢ ܐܝܟܢܐ ܕܝܐܐ ܠܩܪܝܢܐ ܕܐܬܩܪܝܬܘܢ ܀</p> 
 <p><a name="an_4:2">4:2</a> ܒܟܠܗ ܡܟܝܟܘܬ ܪܥܝܢܐ ܘܢܝܚܘܬܐ ܘܢܓܝܪܘܬ ܪܘܚܐ ܘܗܘܝܬܘܢ ܡܤܝܒܪܝܢ ܚܕ ܠܚܕ ܒܚܘܒܐ ܀</p> 
 <p><a name="an_4:3">4:3</a> ܘܗܘܝܬܘܢ ܚܦܝܛܝܢ ܠܡܛܪ ܐܘܝܘܬܐ ܕܪܘܚܐ ܒܚܙܩܐ ܕܫܠܡܐ ܀</p> 
 <p><a name="an_4:4">4:4</a> ܕܬܗܘܘܢ ܒܚܕ ܦܓܪܐ ܘܒܚܕܐ ܪܘܚܐ ܐܝܟ ܡܐ ܕܐܬܩܪܝܬܘܢ ܒܚܕ ܤܒܪܐ ܕܩܪܝܢܟܘܢ ܀</p> 
 <p><a name="an_4:5">4:5</a> ܚܕ ܗܘ ܓܝܪ ܡܪܝܐ ܘܚܕܐ ܗܝܡܢܘܬܐ ܘܚܕܐ ܡܥܡܘܕܝܬܐ ܀</p> 
 <p><a name="an_4:6">4:6</a> ܘܚܕ ܐܠܗܐ ܐܒܐ ܕܟܠ ܘܥܠ ܟܠ ܘܒܝܕ ܟܠ ܘܒܟܠܢ ܀</p> 
 <p><a name="an_4:7">4:7</a> ܠܚܕ ܚܕ ܡܢܢ ܕܝܢ ܐܬܝܗܒܬ ܛܝܒܘܬܐ ܐܝܟ ܡܫܘܚܬܐ ܕܡܘܗܒܬܗ ܕܡܫܝܚܐ ܀</p> 
 <p><a name="an_4:8">4:8</a> ܡܛܠ ܗܢܐ ܐܡܝܪ ܕܤܠܩ ܠܡܪܘܡܐ ܘܫܒܐ ܫܒܝܬܐ ܘܝܗܒ ܡܘܗܒܬܐ ܠܒܢܝܢܫܐ ܀</p> 
 <p><a name="an_4:9">4:9</a> ܕܤܠܩ ܕܝܢ ܡܢܐ ܗܝ ܐܠܐ ܐܢ ܕܐܦ ܢܚܬ ܠܘܩܕܡ ܠܬܚܬܝܬܗ ܕܐܪܥܐ ܀</p> 
 <p><a name="an_4:10">4:10</a> ܗܘ ܕܢܚܬ ܗܘܝܘ ܗܘ ܕܐܦ ܤܠܩ ܠܥܠ ܡܢ ܟܠܗܘܢ ܫܡܝܐ ܕܢܫܠܡ ܟܠ ܀</p> 
 <p><a name="an_4:11">4:11</a> ܘܗܘ ܝܗܒ ܐܝܬ ܕܫܠܝܚܐ ܘܐܝܬ ܕܢܒܝܐ ܘܐܝܬ ܕܡܤܒܪܢܐ ܘܐܝܬ ܕܪܥܘܬܐ ܘܐܝܬ ܕܡܠܦܢܐ ܀</p> 
 <p><a name="an_4:12">4:12</a> ܠܓܡܪܐ ܕܩܕܝܫܐ ܠܥܒܕܐ ܕܬܫܡܫܬܐ ܠܒܢܝܢܐ ܕܦܓܪܐ ܕܡܫܝܚܐ ܀</p> 
 <p><a name="an_4:13">4:13</a> ܥܕܡܐ ܕܟܠܢ ܢܗܘܐ ܚܕ ܡܕܡ ܒܗܝܡܢܘܬܐ ܘܒܝܕܥܬܐ ܕܒܪܗ ܕܐܠܗܐ ܘܚܕ ܓܒܪܐ ܓܡܝܪܐ ܒܡܫܘܚܬܐ ܕܩܘܡܬܐ ܕܫܘܡܠܝܗ ܕܡܫܝܚܐ ܀</p> 
 <p><a name="an_4:14">4:14</a> ܘܠܐ ܢܗܘܐ ܝܠܘܕܐ ܕܡܙܕܥܙܥܝܢ ܘܡܫܬܢܝܢ ܠܟܠ ܪܘܚ ܕܝܘܠܦܢܐ ܢܟܝܠܐ ܕܒܢܝܢܫܐ ܗܢܘܢ ܕܒܚܪܥܘܬܗܘܢ ܡܨܛܢܥܝܢ ܕܢܛܥܘܢ ܀</p> 
 <p><a name="an_4:15">4:15</a> ܐܠܐ ܗܘܝܢ ܫܪܝܪܝܢ ܒܚܘܒܢ ܕܟܘܠ ܡܕܡ ܕܝܠܢ ܢܪܒܐ ܒܡܫܝܚܐ ܕܗܘܝܘ ܪܫܐ ܀</p> 
 <p><a name="an_4:16">4:16</a> ܘܡܢܗ ܟܠܗ ܦܓܪܐ ܡܬܪܟܒ ܘܡܬܩܛܪ ܒܟܠ ܫܪܝܢ ܐܝܟ ܡܘܗܒܬܐ ܕܡܬܝܗܒܐ ܒܡܫܘܚܬܐ ܠܟܠ ܗܕܡ ܠܬܪܒܝܬܐ ܕܝܠܗ ܕܦܓܪܐ ܕܒܚܘܒܐ ܢܫܬܠܡ ܒܢܝܢܗ ܀</p> 
 <p><a name="an_4:17">4:17</a> ܗܕܐ ܕܝܢ ܐܡܪ ܐܢܐ ܘܡܤܗܕ ܐܢܐ ܒܡܪܝܐ ܕܡܢ ܗܫܐ ܠܐ ܬܗܘܘܢ ܡܗܠܟܝܢ ܐܝܟ ܫܪܟܐ ܕܥܡܡܐ ܕܡܗܠܟܝܢ ܒܤܪܝܩܘܬ ܪܥܝܢܗܘܢ ܀</p> 
 <p><a name="an_4:18">4:18</a> ܘܚܫܘܟܝܢ ܒܡܕܥܝܗܘܢ ܘܢܘܟܪܝܝܢ ܐܢܘܢ ܡܢ ܚܝܘܗܝ ܕܐܠܗܐ ܡܛܠ ܕܠܝܬ ܒܗܘܢ ܝܕܥܬܐ ܘܡܛܠ ܥܘܝܪܘܬ ܠܒܗܘܢ ܀</p> 
 <p><a name="an_4:19">4:19</a> ܗܢܘܢ ܕܦܤܩܘ ܤܒܪܗܘܢ ܘܐܫܠܡܘ ܢܦܫܗܘܢ ܠܦܚܙܘܬܐ ܘܠܦܘܠܚܢܐ ܕܟܠܗ ܛܢܦܘܬܐ ܒܝܥܢܘܬܗܘܢ ܀</p> 
 <p><a name="an_4:20">4:20</a> ܐܢܬܘܢ ܕܝܢ ܠܐ ܗܘܐ ܗܟܢܐ ܝܠܦܬܘܢܝܗܝ ܠܡܫܝܚܐ ܀</p> 
 <p><a name="an_4:21">4:21</a> ܐܢ ܫܪܝܪܐܝܬ ܫܡܥܬܘܢܝܗܝ ܘܒܗ ܝܠܦܬܘܢ ܐܝܟ ܡܐ ܕܐܝܬܘܗܝ ܩܘܫܬܐ ܒܝܫܘܥ ܀</p> 
 <p><a name="an_4:22">4:22</a> ܐܠܐ ܕܬܢܝܚܘܢ ܡܢܟܘܢ ܗܘܦܟܝܟܘܢ ܩܕܡܝܐ ܠܒܪܢܫܐ ܗܘ ܥܬܝܩܐ ܕܡܬܚܒܠ ܒܪܓܝܓܬܐ ܕܛܘܥܝܝ ܀</p> 
 <p><a name="an_4:23">4:23</a> ܘܬܬܚܕܬܘܢ ܒܪܘܚܐ ܕܡܕܥܝܟܘܢ ܀</p> 
 <p><a name="an_4:24">4:24</a> ܘܬܠܒܫܘܢ ܠܒܪܢܫܐ ܚܕܬܐ ܗܘ ܕܒܐܠܗܐ ܐܬܒܪܝ ܒܙܕܝܩܘܬܐ ܘܒܚܤܝܘܬܐ ܕܩܘܫܬܐ ܀</p> 
 <p><a name="an_4:25">4:25</a> ܡܛܠ ܗܢܐ ܐܢܝܚܘ ܡܢܟܘܢ ܟܕܒܘܬܐ ܘܡܠܠܘ ܩܘܫܬܐ ܐܢܫ ܥܡ ܩܪܝܒܗ ܗܕܡܐ ܚܢܢ ܓܝܪ ܚܕ ܕܚܕ ܀</p> 
 <p><a name="an_4:26">4:26</a> ܪܓܙܘ ܘܠܐ ܬܚܛܘܢ ܘܫܡܫܐ ܥܠ ܪܘܓܙܟܘܢ ܠܐ ܢܥܪܒ ܀</p> 
 <p><a name="an_4:27">4:27</a> ܘܠܐ ܬܬܠܘܢ ܐܬܪܐ ܠܐܟܠ ܩܪܨܐ ܀</p> 
 <p><a name="an_4:28">4:28</a> ܘܐܝܢܐ ܕܓܢܒ ܗܘܐ ܡܟܝܠ ܠܐ ܢܓܢܘܒ ܐܠܐ ܢܠܐܐ ܒܐܝܕܘܗܝ ܘܢܦܠܘܚ ܛܒܬܐ ܕܢܗܘܐ ܠܗ ܠܡܬܠ ܠܡܢ ܕܤܢܝܩ ܀</p> 
 <p><a name="an_4:29">4:29</a> ܟܠ ܡܠܐ ܤܢܝܐ ܡܢ ܦܘܡܟܘܢ ܠܐ ܬܦܘܩ ܐܠܐ ܐܝܕܐ ܕܫܦܝܪܐ ܘܚܫܚܐ ܠܒܢܝܢܐ ܕܬܬܠ ܛܝܒܘܬܐ ܠܐܝܠܝܢ ܕܫܡܥܝܢ ܀</p> 
 <p><a name="an_4:30">4:30</a> ܘܠܐ ܬܗܘܘܢ ܡܥܝܩܝܢ ܠܪܘܚܐ ܩܕܝܫܬܐ ܕܐܠܗܐ ܕܐܬܚܬܡܬܘܢ ܒܗ ܠܝܘܡܐ ܕܦܘܪܩܢܐ ܀</p> 
 <p><a name="an_4:31">4:31</a> ܟܠܗ ܡܪܝܪܘܬܐ ܘܚܡܬܐ ܘܪܘܓܙܐ ܘܪܘܒܐ ܘܓܘܕܦܐ ܢܫܬܩܠܢ ܡܢܟܘܢ ܥܡ ܟܠܗ ܒܝܫܘܬܐ ܀</p> 
 <p><a name="an_4:32">4:32</a> ܘܗܘܝܬܘܢ ܒܤܝܡܝܢ ܚܕ ܠܘܬ ܚܕ ܘܡܪܚܡܢܝܢ ܘܗܘܝܬܘܢ ܫܒܩܝܢ ܚܕ ܠܚܕ ܐܝܟܢܐ ܕܐܠܗܐ ܒܡܫܝܚܐ ܫܒܩ ܠܢ ܀</p> 
 <p><a name="an_5:1">5:1</a> ܗܘܝܬܘܢ ܗܟܝܠ ܡܬܕܡܝܢ ܒܐܠܗܐ ܐܝܟ ܒܢܝܐ ܚܒܝܒܐ ܀</p> 
 <p><a name="an_5:2">5:2</a> ܘܗܠܟܘ ܒܚܘܒܐ ܐܝܟ ܕܐܦ ܡܫܝܚܐ ܐܚܒܢ ܘܐܫܠܡ ܢܦܫܗ ܥܠ ܐܦܝܢ ܩܘܪܒܢܐ ܘܕܒܚܐ ܠܐܠܗܐ ܠܪܝܚܐ ܒܤܝܡܐ ܀</p> 
 <p><a name="an_5:3">5:3</a> ܙܢܝܘܬܐ ܕܝܢ ܘܟܠܗ ܛܢܦܘܬܐ ܘܥܠܘܒܘܬܐ ܐܦܠܐ ܡܫܬܡܗܘ ܬܫܬܡܗ ܒܝܢܬܟܘܢ ܐܝܟ ܕܝܐܐ ܠܩܕܝܫܐ ܀</p> 
 <p><a name="an_5:4">5:4</a> ܘܠܐ ܨܘܚܝܬܐ ܘܠܐ ܡܠܐ ܕܫܛܝܘܬܐ ܐܘ ܕܒܙܚܐ ܐܘ ܕܫܥܝܐ ܗܠܝܢ ܕܠܐ ܡܬܒܥܝܢ ܐܠܐ ܚܠܦ ܗܠܝܢ ܬܘܕܝܬܐ ܀</p> 
 <p><a name="an_5:5">5:5</a> ܗܕܐ ܕܝܢ ܗܘܝܬܘܢ ܝܕܥܝܢ ܕܟܠܢܫ ܕܐܝܬܘܗܝ ܙܢܝܐ ܐܘ ܛܢܦܐ ܐܘ ܥܠܘܒܐ ܕܐܝܬܘܗܝ ܦܠܚ ܦܬܟܪܐ ܠܝܬ ܠܗ ܝܪܬܘܬܐ ܒܡܠܟܘܬܗ ܕܡܫܝܚܐ ܘܕܐܠܗܐ ܀</p> 
 <p><a name="an_5:6">5:6</a> ܕܠܡܐ ܐܢܫ ܢܛܥܝܟܘܢ ܒܡܠܐ ܤܪܝܩܬܐ ܡܛܠ ܗܠܝܢ ܗܘ ܓܝܪ ܐܬܐ ܪܘܓܙܗ ܕܐܠܗܐ ܥܠ ܒܢܝܐ ܕܠܐ ܡܬܛܦܝܤܢܘܬܐ ܀</p> 
 <p><a name="an_5:7">5:7</a> ܠܐ ܗܟܝܠ ܬܗܘܘܢ ܠܗܘܢ ܫܘܬܦܐ ܀</p> 
 <p><a name="an_5:8">5:8</a> ܐܝܬܝܟܘܢ ܗܘܝܬܘܢ ܓܝܪ ܡܢ ܠܘܩܕܡ ܚܫܘܟܐ ܗܫܐ ܕܝܢ ܢܘܗܪܐ ܐܢܬܘܢ ܒܡܪܢ ܐܝܟ ܒܢܝ ܢܘܗܪܐ ܗܟܝܠ ܗܟܢܐ ܗܠܟܘ ܀</p> 
 <p><a name="an_5:9">5:9</a> ܦܐܪܘܗܝ ܓܝܪ ܕܢܘܗܪܐ ܒܟܠܗ ܐܢܘܢ ܛܒܘܬܐ ܘܙܕܝܩܘܬܐ ܘܩܘܫܬܐ ܀</p> 
 <p><a name="an_5:10">5:10</a> ܘܗܘܝܬܘܢ ܦܪܫܝܢ ܡܢܐ ܫܦܝܪ ܩܕܡ ܡܪܢ ܀</p> 
 <p><a name="an_5:11">5:11</a> ܘܠܐ ܗܘܝܬܘܢ ܡܫܬܘܬܦܝܢ ܠܘܬ ܥܒܕܐ ܕܚܫܘܟܐ ܕܦܐܪܐ ܠܝܬ ܒܗܘܢ ܐܠܐ ܗܘܝܬܘܢ ܡܟܘܢܝܢ ܠܗܘܢ ܀</p> 
 <p><a name="an_5:12">5:12</a> ܡܕܡ ܓܝܪ ܕܒܛܘܫܝܝ ܥܒܕܝܢ ܢܕܝܕ ܗܘ ܐܦ ܠܡܡܠܠܘܬܗ ܀</p> 
 <p><a name="an_5:13">5:13</a> ܟܠ ܡܕܡ ܓܝܪ ܡܬܟܘܢ ܡܢ ܢܘܗܪܐ ܘܡܬܓܠܐ ܘܟܠܡܕܡ ܕܓܠܐ ܢܘܗܪܐ ܗܘ ܀</p> 
 <p><a name="an_5:14">5:14</a> ܡܛܠ ܗܢܐ ܐܡܝܪ ܕܐܬܬܥܝܪ ܕܡܟܐ ܘܩܘܡ ܡܢ ܒܝܬ ܡܝܬܐ ܘܢܢܗܪ ܠܟ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_5:15">5:15</a> ܚܙܘ ܗܟܝܠ ܐܝܟܢܐ ܬܗܠܟܘܢ ܙܗܝܐܝܬ ܠܐ ܐܝܟ ܤܟܠܐ ܐܠܐ ܐܝܟ ܚܟܝܡܐ ܀</p> 
 <p><a name="an_5:16">5:16</a> ܕܙܒܢܝܢ ܩܐܪܤܗܘܢ ܡܛܠ ܕܝܘܡܬܐ ܒܝܫܐ ܐܢܘܢ ܀</p> 
 <p><a name="an_5:17">5:17</a> ܡܛܠ ܗܢܐ ܠܐ ܬܗܘܘܢ ܚܤܝܪܝ ܪܥܝܢܐ ܐܠܐ ܐܤܬܟܠܘ ܡܢܘ ܨܒܝܢܗ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_5:18">5:18</a> ܘܠܐ ܗܘܝܬܘܢ ܪܘܝܢ ܒܚܡܪܐ ܕܒܗ ܐܝܬ ܐܤܘܛܘܬܐ ܐܠܐ ܐܬܡܠܘ ܒܪܘܚܐ ܀</p> 
 <p><a name="an_5:19">5:19</a> ܘܡܠܠܘ ܥܡ ܢܦܫܟܘܢ ܒܡܙܡܘܪܐ ܘܒܬܫܒܚܬܐ ܘܒܙܡܝܪܬܐ ܕܪܘܚܐ ܗܘܝܬܘܢ ܙܡܪܝܢ ܒܠܒܘܬܟܘܢ ܠܡܪܝܐ ܀</p> 
 <p><a name="an_5:20">5:20</a> ܘܗܘܝܬܘܢ ܡܘܕܝܢ ܒܟܠܙܒܢ ܥܠ ܐܦܝ ܟܠܢܫ ܒܫܡܗ ܕܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܠܐܠܗܐ ܐܒܐ ܀</p> 
 <p><a name="an_5:21">5:21</a> ܘܗܘܝܬܘܢ ܡܫܬܥܒܕܝܢ ܚܕ ܠܚܕ ܒܚܘܒܗ ܕܡܫܝܚܐ ܀</p> 
 <p><a name="an_5:22">5:22</a> ܢܫܐ ܗܘܝܬܝܢ ܡܫܬܥܒܕܢ ܠܒܥܠܝܟܝܢ ܐܝܟ ܕܠܡܪܢ ܀</p> 
 <p><a name="an_5:23">5:23</a> ܡܛܠ ܕܓܒܪܐ ܪܫܗ ܗܘ ܕܐܢܬܬܐ ܐܝܟܢܐ ܕܐܦ ܡܫܝܚܐ ܪܫܐ ܗܘ ܕܥܕܬܐ ܘܗܘܝܘ ܡܚܝܢܐ ܕܦܓܪܐ ܀</p> 
 <p><a name="an_5:24">5:24</a> ܐܠܐ ܐܝܟܢܐ ܕܥܕܬܐ ܡܫܬܥܒܕܐ ܠܡܫܝܚܐ ܗܟܢܐ ܐܦ ܢܫܐ ܠܒܥܠܝܗܝܢ ܒܟܠܡܕܡ ܀</p> 
 <p><a name="an_5:25">5:25</a> ܓܒܪܐ ܐܚܒܘ ܢܫܝܟܘܢ ܐܝܟ ܕܐܦ ܡܫܝܚܐ ܐܚܒ ܠܥܕܬܗ ܘܢܦܫܗ ܐܫܠܡ ܥܠ ܐܦܝܗ ܀</p> 
 <p><a name="an_5:26">5:26</a> ܕܢܩܕܫܝܗ ܘܢܕܟܝܗ ܒܤܚܝܐ ܕܡܝܐ ܘܒܡܠܬܐ ܀</p> 
 <p><a name="an_5:27">5:27</a> ܘܢܩܝܡܝܗ ܥܕܬܐ ܠܢܦܫܗ ܟܕ ܡܫܒܚܐ ܘܠܝܬ ܒܗ ܛܘܠܫܐ ܘܠܐ ܩܡܛܐ ܘܠܐ ܡܕܡ ܕܠܗܠܝܢ ܕܡܐ ܐܠܐ ܬܗܘܐ ܩܕܝܫܐ ܕܠܐ ܡܘܡ ܀</p> 
 <p><a name="an_5:28">5:28</a> ܗܟܢܐ ܘܠܐ ܠܓܒܪܐ ܕܢܚܒܘܢ ܢܫܝܗܘܢ ܐܝܟ ܕܠܦܓܪܝܗܘܢ ܡܢ ܕܠܐܢܬܬܗ ܓܝܪ ܡܚܒ ܢܦܫܗ ܗܘ ܡܚܒ ܀</p> 
 <p><a name="an_5:29">5:29</a> ܠܐ ܓܝܪ ܐܢܫ ܡܡܬܘܡ ܤܢܐ ܦܓܪܗ ܐܠܐ ܡܬܪܤܐ ܠܗ ܘܝܨܦ ܕܝܠܗ ܐܝܟܢܐ ܕܐܦ ܡܫܝܚܐ ܕܥܕܬܗ ܀</p> 
 <p><a name="an_5:30">5:30</a> ܡܛܠ ܕܗܕܡܐ ܚܢܢ ܕܦܓܪܗ ܘܡܢ ܒܤܪܗ ܚܢܢ ܘܡܢ ܓܪܡܘܗܝ ܀</p> 
 <p><a name="an_5:31">5:31</a> ܡܛܠ ܗܢܐ ܢܫܒܘܩ ܓܒܪܐ ܠܐܒܘܗܝ ܘܠܐܡܗ ܘܢܩܦ ܠܐܢܬܬܗ ܘܢܗܘܘܢ ܬܪܝܗܘܢ ܚܕ ܒܤܪ ܀</p> 
 <p><a name="an_5:32">5:32</a> ܗܢܐ ܐܪܙܐ ܪܒ ܗܘ ܐܢܐ ܕܝܢ ܐܡܪ ܐܢܐ ܥܠ ܡܫܝܚܐ ܘܥܠ ܥܕܬܗ ܀</p> 
 <p><a name="an_5:33">5:33</a> ܒܪܡ ܐܦ ܐܢܬܘܢ ܟܠ ܚܕ ܚܕ ܡܢܟܘܢ ܗܟܢܐ ܢܪܚܡ ܐܢܬܬܗ ܐܝܟ ܕܠܢܦܫܗ ܐܢܬܬܐ ܕܝܢ ܬܗܘܐ ܕܚܠܐ ܡܢ ܒܥܠܗ ܀</p> 
 <p><a name="an_6:1">6:1</a> ܒܢܝܐ ܐܫܬܡܥܘ ܠܐܒܗܝܟܘܢ ܒܡܪܢ ܗܕܐ ܓܝܪ ܟܐܢܐ ܀</p> 
 <p><a name="an_6:2">6:2</a> ܘܗܢܘ ܦܘܩܕܢܐ ܩܕܡܝܐ ܕܡܠܝܟ ܝܩܪ ܠܐܒܘܟ ܘܠܐܡܟ ܀</p> 
 <p><a name="an_6:3">6:3</a> ܕܢܗܘܐ ܠܟ ܕܫܦܝܪ ܘܢܐܪܟܘܢ ܚܝܝܟ ܥܠ ܐܪܥܐ ܀</p> 
 <p><a name="an_6:4">6:4</a> ܐܒܗܐ ܠܐ ܬܪܓܙܘܢ ܒܢܝܟܘܢ ܐܠܐ ܪܒܘ ܐܢܘܢ ܒܡܪܕܘܬܐ ܘܒܝܘܠܦܢܐ ܕܡܪܢ ܀</p> 
 <p><a name="an_6:5">6:5</a> ܥܒܕܐ ܐܫܬܡܥܘ ܠܡܪܝܟܘܢ ܕܒܒܤܪ ܒܕܚܠܬܐ ܘܒܪܬܝܬܐ ܘܒܦܫܝܛܘܬ ܠܒܐ ܐܝܟ ܕܠܡܫܝܚܐ ܀</p> 
 <p><a name="an_6:6">6:6</a> ܠܐ ܒܡܚܙܐ ܥܝܢܐ ܐܝܟ ܗܘ ܕܠܒܢܝܢܫܐ ܫܦܪܝܢ ܐܢܬܘܢ ܐܠܐ ܐܝܟ ܥܒܕܐ ܕܡܫܝܚܐ ܕܥܒܕܝܢ ܨܒܝܢܐ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_6:7">6:7</a> ܘܫܡܫܘ ܐܢܘܢ ܡܢ ܟܠܗ ܢܦܫܟܘܢ ܒܚܘܒܐ ܐܝܟ ܕܠܡܪܢ ܘܠܐ ܐܝܟ ܕܠܒܢܝܢܫܐ ܀</p> 
 <p><a name="an_6:8">6:8</a> ܟܕ ܝܕܥܝܬܘܢ ܕܗܘ ܡܕܡ ܕܥܒܕ ܐܢܫ ܕܫܦܝܪ ܗܘ ܡܬܦܪܥ ܡܢ ܡܪܢ ܐܢ ܥܒܕܐ ܗܘ ܘܐܢ ܒܪ ܚܐܪܐ ܀</p> 
 <p><a name="an_6:9">6:9</a> ܐܦ ܐܢܬܘܢ ܡܪܝܐ ܗܟܢܐ ܗܘܝܬܘܢ ܥܒܕܝܢ ܠܥܒܕܝܟܘܢ ܗܘܝܬܘܢ ܫܒܩܝܢ ܠܗܘܢ ܤܟܠܘܬܐ ܡܛܠ ܕܝܕܥܝܢ ܐܢܬܘܢ ܕܐܦ ܡܪܟܘܢ ܕܝܠܟܘܢ ܐܝܬܘܗܝ ܒܫܡܝܐ ܘܡܤܒ ܒܐܦܐ ܠܝܬ ܠܘܬܗ ܀</p> 
 <p><a name="an_6:10">6:10</a> ܡܟܝܠ ܐܚܝ ܐܬܚܝܠܘ ܒܡܪܢ ܘܒܬܘܩܦܐ ܕܚܝܠܗ ܀</p> 
 <p><a name="an_6:11">6:11</a> ܘܠܒܫܘ ܟܠܗ ܙܝܢܐ ܕܐܠܗܐ ܐܝܟܢܐ ܕܬܫܟܚܘܢ ܠܡܩܡ ܠܘܩܒܠ ܨܢܥܬܗ ܕܐܟܠ ܩܪܨܐ ܀</p> 
 <p><a name="an_6:12">6:12</a> ܡܛܠ ܕܬܟܬܘܫܟܘܢ ܠܐ ܗܘܐ ܥܡ ܒܤܪܐ ܘܕܡܐ ܐܠܐ ܥܡ ܐܪܟܘܤ ܘܥܡ ܫܠܝܛܢܐ ܘܥܡ ܐܚܝܕܝ ܥܠܡܐ ܕܗܢܐ ܚܫܘܟܐ ܘܥܡ ܪܘܚܐ ܒܝܫܬܐ ܕܬܚܝܬ ܫܡܝܐ ܀</p> 
 <p><a name="an_6:13">6:13</a> ܡܛܠ ܗܢܐ ܠܒܫܘ ܟܠܗ ܙܝܢܐ ܕܐܠܗܐ ܕܬܫܟܚܘܢ ܬܐܪܥܘܢ ܒܝܫܐ ܘܟܕ ܡܥܬܕܝܢ ܐܢܬܘܢ ܒܟܠܡܕܡ ܬܩܘܡܘܢ ܀</p> 
 <p><a name="an_6:14">6:14</a> ܩܘܡܘ ܗܟܝܠ ܘܚܙܘܩܘ ܚܨܝܟܘܢ ܒܩܘܫܬܐ ܘܠܒܫܘ ܫܪܝܢܐ ܕܙܕܝܩܘܬܐ ܀</p> 
 <p><a name="an_6:15">6:15</a> ܘܤܐܢܘ ܒܪܓܠܝܟܘܢ ܛܘܝܒܗ ܕܐܘܢܓܠܝܘܢ ܕܫܠܡܐ ܀</p> 
 <p><a name="an_6:16">6:16</a> ܘܥܡ ܗܠܝܢ ܤܒܘ ܠܟܘܢ ܤܟܪܐ ܕܗܝܡܢܘܬܐ ܕܒܗ ܬܬܡܨܘܢ ܚܝܠܐ ܠܡܕܥܟܘ ܟܠܗܘܢ ܓܐܪܘܗܝ ܝܩܕܐ ܕܒܝܫܐ ܀</p> 
 <p><a name="an_6:17">6:17</a> ܘܤܝܡܘ ܤܢܘܪܬܐ ܕܦܘܪܩܢܐ ܘܐܚܘܕܘ ܤܝܦܐ ܕܪܘܚܐ ܕܐܝܬܘܗܝ ܡܠܬܐ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_6:18">6:18</a> ܘܒܟܠ ܨܠܘܢ ܘܒܟܠ ܒܥܘܢ ܨܠܘ ܒܟܠܙܒܢ ܒܪܘܚ ܘܒܗ ܒܨܠܘܬܐ ܗܘܝܬܘܢ ܫܗܪܝܢ ܒܟܠܥܕܢ ܟܕ ܡܨܠܝܬܘܢ ܐܡܝܢܐܝܬ ܘܡܬܟܫܦܝܢ ܥܠ ܐܦܝ ܟܠܗܘܢ ܩܕܝܫܐ ܀</p> 
 <p><a name="an_6:19">6:19</a> ܐܦ ܥܠܝ ܕܬܬܝܗܒ ܠܝ ܡܠܬܐ ܒܡܦܬܚ ܦܘܡܝ ܕܥܝܢ ܒܓܠܐ ܐܟܪܙ ܐܪܙܐ ܕܤܒܪܬܐ ܀</p> 
 <p><a name="an_6:20">6:20</a> ܗܘ ܕܐܢܐ ܐܝܙܓܕܗ ܒܫܫܠܬܐ ܕܒܦܪܗܤܝܐ ܐܡܠܠܝܘܗܝ ܐܝܟ ܕܘܠܐ ܠܝ ܠܡܡܠܠܘܬܗ ܀</p> 
 <p><a name="an_6:21">6:21</a> ܕܬܕܥܘܢ ܕܝܢ ܐܦ ܐܢܬܘܢ ܡܕܡ ܕܠܘܬܝ ܘܡܕܡ ܕܤܥܪ ܐܢܐ ܗܐ ܡܘܕܥ ܠܟܘܢ ܛܘܟܝܩܘܤ ܐܚܐ ܚܒܝܒܐ ܘܡܫܡܫܢܐ ܡܗܝܡܢܐ ܒܡܪܢ ܀</p> 
 <p><a name="an_6:22">6:22</a> ܕܠܗ ܫܕܪܬ ܠܘܬܟܘܢ ܥܠܝܗ ܥܠ ܗܕܐ ܕܬܕܥܘܢ ܡܐ ܕܠܘܬܝ ܘܢܒܝܐ ܠܒܘܬܟܘܢ ܀</p> 
 <p><a name="an_6:23">6:23</a> ܫܠܡܐ ܥܡ ܐܚܝܢ ܘܚܘܒܐ ܥܡ ܗܝܡܢܘܬܐ ܡܢ ܐܠܗܐ ܐܒܐ ܘܡܢ ܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_6:24">6:24</a> ܛܝܒܘܬܐ ܥܡ ܟܠܗܘܢ ܐܝܠܝܢ ܕܡܚܒܝܢ ܠܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܕܠܐ ܚܒܠ ܐܡܝܢ ܀</p>
