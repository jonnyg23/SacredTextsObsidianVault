  <h3 align="center">Aramaic Bible: Titus</h3> 
 <p><a name="an_1:1">1:1</a> ܦܘܠܘܤ ܥܒܕܗ ܕܐܠܗܐ ܘܫܠܝܚܗ ܕܝܫܘܥ ܡܫܝܚܐ ܒܗܝܡܢܘܬܐ ܕܓܒܝܐ ܕܐܠܗܐ ܘܝܕܥܬܐ ܕܫܪܪܐ ܕܒܕܚܠܬ ܐܠܗܐ ܀</p> 
 <p><a name="an_1:2">1:2</a> ܥܠ ܤܒܪܐ ܕܚܝܐ ܕܠܥܠܡ ܕܡܠܟ ܐܠܗܐ ܫܪܝܪܐ ܩܕܡ ܙܒܢܘܗܝ ܕܥܠܡܐ ܀</p> 
 <p><a name="an_1:3">1:3</a> ܘܓܠܐ ܡܠܬܗ ܒܙܒܢܗ ܒܝܕ ܟܪܘܙܘܬܢ ܗܝ ܕܐܬܗܝܡܢܬ ܐܢܐ ܒܦܘܩܕܢܐ ܕܐܠܗܐ ܡܚܝܢܢ ܀</p> 
 <p><a name="an_1:4">1:4</a> ܠܛܛܘܤ ܒܪܐ ܫܪܝܪܐ ܒܗܝܡܢܘܬܐ ܕܓܘܐ ܛܝܒܘܬܐ ܘܫܠܡܐ ܡܢ ܐܠܗܐ ܐܒܐ ܘܡܢ ܡܪܢ ܝܫܘܥ ܡܫܝܚܐ ܡܚܝܢܢ ܀</p> 
 <p><a name="an_1:5">1:5</a> ܡܛܠܗܢܐ ܗܘ ܫܒܩܬܟ ܗܘܝܬ ܒܩܪܛܐ ܕܐܝܠܝܢ ܕܚܤܝܪܢ ܬܬܩܢ ܘܬܩܝܡ ܩܫܝܫܐ ܒܡܕܝܢܐ ܡܕܝܢܐ ܐܝܟܢܐ ܕܦܩܕܬܟ ܀</p> 
 <p><a name="an_1:6">1:6</a> ܐܝܢܐ ܕܕܠܐ ܪܫܝܢ ܗܘ ܘܗܘܐ ܒܥܠܐ ܕܚܕܐ ܐܢܬܬܐ ܘܐܝܬ ܠܗ ܒܢܝܐ ܡܗܝܡܢܐ ܕܠܐ ܡܨܚܝܢ ܘܠܐ ܐܝܬܝܗܘܢ ܕܠܐ ܫܘܥܒܕ ܒܐܤܘܛܘܬܐ ܀</p> 
 <p><a name="an_1:7">1:7</a> ܚܝܒ ܗܘ ܓܝܪ ܩܫܝܫܐ ܕܢܗܘܐ ܕܠܐ ܪܫܝܢ ܐܝܟ ܪܒܝܬܐ ܕܐܠܗܐ ܘܠܐ ܢܗܘܐ ܡܬܕܒܪ ܒܪܥܝܢ ܢܦܫܗ ܘܠܐ ܢܗܘܐ ܚܡܬܢ ܘܠܐ ܢܗܘܐ ܥܒܪ ܥܠ ܚܡܪܐ ܘܠܐ ܬܗܘܐ ܪܗܛܐ ܐܝܕܗ ܠܡܡܚܐ ܘܠܐ ܢܗܘܐ ܪܚܡ ܝܘܬܪܢܐ ܛܢܦܐ ܀</p> 
 <p><a name="an_1:8">1:8</a> ܐܠܐ ܢܗܘܐ ܪܚܡ ܐܟܤܢܝܐ ܘܢܗܘܐ ܪܚܡ ܛܒܬܐ ܘܢܗܘܐ ܢܟܦ ܘܢܗܘܐ ܟܐܝܢ ܘܢܗܘܐ ܚܤܝܐ ܘܠܒܝܟ ܢܦܫܗ ܡܢ ܪܓܝܓܬܐ ܀</p> 
 <p><a name="an_1:9">1:9</a> ܘܒܛܝܠ ܠܗ ܥܠ ܝܘܠܦܢܐ ܕܡܠܬܐ ܕܗܝܡܢܘܬܐ ܕܢܫܟܚ ܐܦ ܠܡܒܝܐܘ ܒܝܘܠܦܢܗ ܚܠܝܡܐ ܘܠܡܟܤܘ ܠܐܝܠܝܢ ܕܡܬܚܪܝܢ ܀</p> 
 <p><a name="an_1:10">1:10</a> ܐܝܬ ܓܝܪ ܤܓܝܐܐ ܕܠܐ ܡܫܬܥܒܕܝܢ ܘܤܪܝܩܢ ܡܠܝܗܘܢ ܘܡܛܥܝܢ ܪܥܝܢܐ ܕܒܢܝܢܫܐ ܝܬܝܪܐܝܬ ܐܝܠܝܢ ܕܡܢ ܓܙܘܪܬܐ ܐܢܘܢ ܀</p> 
 <p><a name="an_1:11">1:11</a> ܗܢܘܢ ܕܘܠܐ ܠܡܤܟܪܘ ܦܘܡܗܘܢ ܒܬܐ ܓܝܪ ܤܓܝܐܐ ܡܚܒܠܝܢ ܘܡܠܦܝܢ ܡܕܡ ܕܠܐ ܘܠܐ ܡܛܠ ܝܘܬܪܢܐ ܛܢܦܐ ܀</p> 
 <p><a name="an_1:12">1:12</a> ܐܡܪ ܐܢܫ ܡܢܗܘܢ ܢܒܝܐ ܕܝܠܗܘܢ ܕܒܢܝ ܩܪܛܐ ܒܟܠܙܒܢ ܕܓܠܝܢ ܐܢܘܢ ܚܝܘܬܐ ܒܝܫܬܐ ܘܟܪܤܬܐ ܒܛܝܠܬܐ ܀</p> 
 <p><a name="an_1:13">1:13</a> ܘܫܪܝܪܐܝܬ ܐܝܬܝܗ ܤܗܕܘܬܐ ܗܕܐ ܡܛܠ ܗܢܐ ܩܫܝܐܝܬ ܗܘܝܬ ܡܟܤ ܠܗܘܢ ܕܢܗܘܘܢ ܚܠܝܡܝܢ ܒܗܝܡܢܘܬܐ ܀</p> 
 <p><a name="an_1:14">1:14</a> ܘܠܐ ܢܬܪܡܘܢ ܠܫܘܥܝܬܐ ܕܝܗܘܕܝܐ ܘܠܦܘܩܕܐ ܕܒܢܝܢܫܐ ܕܤܢܝܢ ܫܪܪܐ ܀</p> 
 <p><a name="an_1:15">1:15</a> ܟܠܡܕܡ ܓܝܪ ܕܟܐ ܗܘ ܠܕܟܝܐ ܠܐܝܠܝܢ ܕܝܢ ܕܡܤܝܒܝܢ ܘܠܐ ܡܗܝܡܢܝܢ ܡܕܡ ܕܕܟܐ ܠܝܬ ܠܗܘܢ ܐܠܐ ܡܤܝܒ ܗܘ ܪܥܝܢܗܘܢ ܘܬܐܪܬܗܘܢ ܀</p> 
 <p><a name="an_1:16">1:16</a> ܘܡܘܕܝܢ ܕܝܕܥܝܢ ܠܐܠܗܐ ܒܥܒܕܝܗܘܢ ܕܝܢ ܟܦܪܝܢ ܒܗ ܘܐܝܬܝܗܘܢ ܤܢܝܐܐ ܘܕܠܐ ܦܝܤ ܘܡܤܠܝܐ ܠܟܠ ܥܒܕ ܛܒ ܀</p> 
 <p><a name="an_2:1">2:1</a> ܐܢܬ ܕܝܢ ܡܠܠ ܡܕܡ ܕܝܐܐ ܠܝܘܠܦܢܐ ܚܠܝܡܐ ܀</p> 
 <p><a name="an_2:2">2:2</a> ܘܐܠܦ ܕܢܗܘܘܢ ܩܫܝܫܐ ܥܝܪܝܢ ܒܪܥܝܢܝܗܘܢ ܘܢܗܘܘܢ ܢܟܦܝܢ ܘܢܗܘܘܢ ܕܟܝܢ ܘܢܗܘܘܢ ܚܠܝܡܝܢ ܒܗܝܡܢܘܬܐ ܘܒܚܘܒܐ ܘܒܡܤܝܒܪܢܘܬܐ ܀</p> 
 <p><a name="an_2:3">2:3</a> ܘܐܦ ܩܫܝܫܬܐ ܗܟܢܐ ܕܢܗܘܝܢ ܒܐܤܟܡܐ ܕܝܐܐ ܠܕܚܠܬ ܐܠܗܐ ܘܠܐ ܢܗܘܝܢ ܐܟܠܢ ܩܪܨܐ ܘܠܐ ܢܗܘܝܢ ܡܫܥܒܕܢ ܠܚܡܪܐ ܤܓܝܐܐ ܘܢܗܘܝܢ ܡܠܦܢ ܫܦܝܪܬܐ ܀</p> 
 <p><a name="an_2:4">2:4</a> ܘܡܢܟܦܢ ܠܐܝܠܝܢ ܕܛܠܝܢ ܕܢܪܚܡܢ ܒܥܠܝܗܝܢ ܘܒܢܝܗܝܢ ܀</p> 
 <p><a name="an_2:5">2:5</a> ܘܢܗܘܝܢ ܢܟܦܢ ܘܩܕܝܫܢ ܘܢܗܘܝܢ ܝܨܦܢ ܫܦܝܪ ܕܒܬܝܗܝܢ ܘܡܫܬܥܒܕܢ ܠܒܥܠܝܗܝܢ ܕܠܐ ܐܢܫ ܢܓܕܦ ܥܠ ܡܠܬܗ ܕܐܠܗܐ ܀</p> 
 <p><a name="an_2:6">2:6</a> ܘܡܢ ܐܝܠܝܢ ܕܛܠܝܢ ܗܟܢܐ ܒܥܝ ܕܢܗܘܘܢ ܢܟܦܝܢ ܀</p> 
 <p><a name="an_2:7">2:7</a> ܒܟܠܡܕܡ ܕܝܢ ܕܡܘܬܐ ܒܢܦܫܟ ܚܘܐ ܒܟܠ ܥܒܕܐ ܛܒܐ ܘܒܡܠܦܢܘܬܟ ܬܗܘܐ ܠܟ ܡܠܬܐ ܚܠܝܡܬܐ ܀</p> 
 <p><a name="an_2:8">2:8</a> ܕܢܟܦܐ ܘܠܐ ܡܚܒܠܐ ܘܠܐ ܐܢܫ ܒܤܪ ܠܗ ܕܐܝܢܐ ܕܩܐܡ ܠܩܘܒܠܢ ܢܒܗܬ ܟܕ ܠܐ ܢܫܟܚ ܕܢܐܡܪ ܥܠܝܢ ܡܕܡ ܕܤܢܐ ܀</p> 
 <p><a name="an_2:9">2:9</a> ܥܒܕܐ ܠܡܪܝܗܘܢ ܢܫܬܥܒܕܘܢ ܒܟܠܡܕܡ ܘܢܫܬܦܪܘܢ ܘܠܐ ܗܘܘ ܡܥܨܝܢ ܀</p> 
 <p><a name="an_2:10">2:10</a> ܘܠܐ ܗܘܘ ܓܢܒܝܢ ܐܠܐ ܢܚܘܘܢ ܫܪܪܗܘܢ ܛܒܐ ܒܟܠܡܕܡ ܕܢܨܒܬܘܢ ܒܟܠܡܕܡ ܝܘܠܦܢܗ ܕܐܠܗܐ ܡܚܝܢܢ ܀</p> 
 <p><a name="an_2:11">2:11</a> ܐܬܓܠܝܬ ܓܝܪ ܛܝܒܘܬܗ ܕܐܠܗܐ ܡܚܝܬ ܟܠ ܠܒܢܝܢܫܐ ܟܠܗܘܢ ܀</p> 
 <p><a name="an_2:12">2:12</a> ܘܪܕܝܐ ܠܢ ܕܢܟܦܘܪ ܒܪܘܫܥܐ ܘܒܪܓܝܓܬܐ ܕܥܠܡܐ ܘܢܚܐ ܒܥܠܡܐ ܗܢܐ ܒܢܟܦܘܬܐ ܘܒܟܐܢܘܬܐ ܘܒܕܚܠܬ ܐܠܗܐ ܀</p> 
 <p><a name="an_2:13">2:13</a> ܟܕ ܡܤܟܝܢܢ ܠܤܒܪܐ ܒܪܝܟܐ ܘܠܓܠܝܢܐ ܕܬܫܒܘܚܬܗ ܕܐܠܗܐ ܪܒܐ ܘܡܚܝܢܢ ܝܫܘܥ ܡܫܝܚܐ ܀</p> 
 <p><a name="an_2:14">2:14</a> ܕܗܘ ܝܗܒ ܢܦܫܗ ܚܠܦܝܢ ܕܢܦܪܩܢ ܡܢ ܟܠ ܥܘܠܐ ܘܢܕܟܝܢ ܠܢܦܫܗ ܥܡܐ ܚܕܬܐ ܕܚܤܡ ܒܥܒܕܐ ܛܒܐ ܀</p> 
 <p><a name="an_2:15">2:15</a> ܗܠܝܢ ܡܠܠ ܘܒܥܝ ܘܟܘܢ ܒܟܠ ܦܘܩܕܢ ܘܠܐ ܐܢܫ ܢܒܤܐ ܒܟ ܀</p> 
 <p><a name="an_3:1">3:1</a> ܘܗܘܝܬ ܡܥܗܕ ܠܗܘܢ ܕܠܪܫܐ ܘܠܫܠܝܛܢܐ ܢܫܬܡܥܘܢ ܘܢܫܬܥܒܕܘܢ ܘܕܢܗܘܘܢ ܥܬܝܕܝܢ ܠܟܠ ܥܒܕ ܛܒ ܀</p> 
 <p><a name="an_3:2">3:2</a> ܘܥܠ ܐܢܫ ܠܐ ܢܓܕܦܘܢ ܘܠܐ ܢܗܘܘܢ ܡܬܟܬܫܝܢ ܐܠܐ ܢܗܘܘܢ ܡܟܝܟܝܢ ܘܒܟܠܡܕܡ ܢܚܘܘܢ ܒܤܝܡܘܬܗܘܢ ܠܘܬ ܟܠܗܘܢ ܒܢܝܢܫܐ ܀</p> 
 <p><a name="an_3:3">3:3</a> ܐܦ ܚܢܢ ܓܝܪ ܡܢ ܩܕܝܡ ܕܠܐ ܪܥܝܢܐ ܗܘܝܢ ܘܕܠܐ ܦܝܤܐ ܘܛܥܝܢ ܗܘܝܢ ܘܡܫܥܒܕܝܢ ܗܘܝܢ ܠܐܪܓܝܓܬܐ ܡܫܚܠܦܬܐ ܘܒܒܝܫܘܬܐ ܘܒܚܤܡܐ ܡܬܗܦܟܝܢ ܗܘܝܢ ܘܤܢܝܐܝܝܢ ܗܘܝܢ ܐܦ ܚܕ ܠܚܕ ܤܢܝܢ ܗܘܝܢ ܀</p> 
 <p><a name="an_3:4">3:4</a> ܟܕ ܕܝܢ ܐܬܓܠܝܬ ܒܤܝܡܘܬܗ ܘܡܪܚܡܢܘܬܗ ܕܐܠܗܐ ܡܚܝܢܢ ܀</p> 
 <p><a name="an_3:5">3:5</a> ܠܐ ܒܥܒܕܐ ܕܙܕܝܩܘܬܐ ܕܥܒܕܢ ܐܠܐ ܒܪܚܡܘܗܝ ܕܝܠܗ ܐܚܝܢ ܒܤܚܬܐ ܕܡܘܠܕܐ ܕܡܢ ܕܪܝܫ ܘܒܚܘܕܬܐ ܕܪܘܚܐ ܕܩܘܕܫܐ ܀</p> 
 <p><a name="an_3:6">3:6</a> ܗܝ ܕܐܫܕ ܥܠܝܢ ܥܬܝܪܐܝܬ ܒܝܕ ܝܫܘܥ ܡܫܝܚܐ ܡܚܝܢܢ ܀</p> 
 <p><a name="an_3:7">3:7</a> ܕܒܛܝܒܘܬܗ ܢܙܕܕܩ ܘܢܗܘܐ ܝܪܬܐ ܒܤܒܪܐ ܒܚܝܐ ܕܠܥܠܡ ܀</p> 
 <p><a name="an_3:8">3:8</a> ܡܗܝܡܢܐ ܗܝ ܡܠܬܐ ܘܒܗܠܝܢ ܨܒܐ ܐܢܐ ܕܐܦ ܐܢܬ ܗܘܝܬ ܡܫܪܪ ܠܗܘܢ ܕܢܬܒܛܠ ܠܗܘܢ ܠܡܦܠܚ ܥܒܕܐ ܛܒܐ ܐܝܠܝܢ ܕܗܝܡܢܘ ܒܐܠܗܐ ܗܠܝܢ ܐܢܝܢ ܛܒܢ ܘܡܘܬܪܢ ܠܒܢܝܢܫܐ ܀</p> 
 <p><a name="an_3:9">3:9</a> ܡܢ ܒܥܬܐ ܕܝܢ ܤܟܠܬܐ ܘܡܢ ܬܫܥܝܬܐ ܕܫܪܒܬܐ ܘܡܢ ܚܪܝܢܐ ܘܡܢ ܬܟܬܘܫܐ ܕܤܦܪܐ ܐܫܬܐܠ ܝܘܬܪܢ ܓܝܪ ܠܝܬ ܒܗܝܢ ܘܤܪܝܩܢ ܐܢܝܢ ܀</p> 
 <p><a name="an_3:10">3:10</a> ܡܢ ܓܒܪܐ ܗܪܤܝܘܛܐ ܡܢ ܒܬܪ ܚܕܐ ܙܒܢ ܘܬܪܬܝܢ ܕܡܪܬܐ ܐܢܬ ܠܗ ܐܫܬܐܠ ܀</p> 
 <p><a name="an_3:11">3:11</a> ܘܗܘܝܬ ܝܕܥ ܕܡܢ ܕܗܟܢܐ ܗܘ ܡܥܩܡ ܗܘ ܘܚܛܐ ܘܗܘ ܚܝܒ ܢܦܫܗ ܀</p> 
 <p><a name="an_3:12">3:12</a> ܡܐ ܕܫܕܪܬ ܠܘܬܟ ܠܐܪܛܡܐ ܐܘ ܠܛܘܟܝܩܘܤ ܢܬܒܛܠ ܠܟ ܕܬܐܬܐ ܠܘܬܝ ܠܢܝܩܦܘܠܝܤ ܬܡܢ ܓܝܪ ܤܡܬ ܒܪܥܝܢܝ ܕܐܤܬܐ ܀</p> 
 <p><a name="an_3:13">3:13</a> ܥܠ ܙܢܐ ܕܝܢ ܤܦܪܐ ܘܥܠ ܐܦܠܘ ܢܬܒܛܠ ܠܟ ܕܬܠܘܐ ܐܢܘܢ ܫܦܝܪ ܕܡܕܡ ܠܐ ܢܚܤܪ ܠܗܘܢ ܀</p> 
 <p><a name="an_3:14">3:14</a> ܘܢܐܠܦܘܢ ܐܦ ܐܝܠܝܢ ܕܕܝܠܢ ܐܢܘܢ ܕܢܗܘܘܢ ܦܠܚܝܢ ܥܒܕܐ ܛܒܐ ܒܨܒܘܬܐ ܕܐܠܨܢ ܕܠܐ ܢܗܘܘܢ ܕܠܐ ܦܐܪܐ ܀</p> 
 <p><a name="an_3:15">3:15</a> ܫܐܠܝܢ ܒܫܠܡܟ ܟܠܗܘܢ ܐܝܠܝܢ ܕܥܡܝ ܐܢܘܢ ܫܐܠ ܒܫܠܡܐ ܕܟܠܗܘܢ ܐܝܠܝܢ ܕܪܚܡܝܢ ܠܢ ܒܗܝܡܢܘܬܐ ܛܝܒܘܬܐ ܥܡ ܟܠܟܘܢ ܐܡܝܢ ܀</p>
