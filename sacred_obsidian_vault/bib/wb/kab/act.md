  <h3 align="center">Kabyle Bible: Acts</h3> 
 <p><a name="an_1:1">1:1</a> A Tawfilus, di tektabt-inu tamezwarut, ḥkiɣ-ed ayen ixdem Sidna Ɛisa d wayen yesselmed seg wass amezwaru armi d ass i deg yețwarfed ɣer igenni. </p> 
 <p><a name="an_1:2">1:2</a> Uqbel ad ițwarfed, yefka lumuṛat-is s Ṛṛuḥ iqedsen i wid i gextaṛ ad ilin d ṛṛusul.  </p> 
 <p><a name="an_1:3">1:3</a> Mi d-iḥya si ger lmegtin, i nutni daɣen iwumi i d-isban iman-is belli yedder ; ṛebɛin wussan nețța ițban asen-d yerna ițmeslay-asen-d ɣef wayen yeɛnan tagelda n Sidi Ṛebbi.  </p> 
 <p><a name="an_1:4">1:4</a> Yiwen wass, mi gella itețț yid-sen, yefka-yasen lameṛ-agi : Ur țeffɣet ara si temdint n Lquds, lameɛna eṛǧut lemɛahda n Baba Ṛebbi, tin akken i wen-d nniɣ ;  </p> 
 <p><a name="an_1:5">1:5</a> « Yeḥya isseɣḍas imdanen deg aman, ma d kunwi, di kra n wussan aț-țețwaɣeḍsem s Ṛṛuḥ iqedsen.»  </p> 
 <p><a name="an_1:6">1:6</a> Mi llan ṛṛusul nnejmaɛen yid-es, steqsan-t : A Sidi, deg ussan-agi ara d terreḍ tageldit i wegdud n wat Isṛail ?  </p> 
 <p><a name="an_1:7">1:7</a> Yerra-yasen : Ur țnadit ara aț-țeẓrem ayen yeɛnan lewqat d wussan i ghegga Baba Ṛebbi s lḥekma-ines si zik, ayagi mačči d ccɣel-nwen.  </p> 
 <p><a name="an_1:8">1:8</a> Kunwi a wen-d-tețțunefk tezmert n Ṛṛuḥ iqedsen ara d-yersen fell-awen, imiren aț-țilim d inigan-iw di temdint n Lquds, di tmurt n Yahuda ț-țmurt n Samarya, aț-țawḍem alamma d ixfawen n ddunit.  </p> 
 <p><a name="an_1:9">1:9</a> Akken i d-yenna imeslayen-agi, Sidna Ɛisa yețwarfed ɣer igenni nutni țmuqulen deg-s armi i t-tɣumm tagut. </p> 
 <p><a name="an_1:10">1:10</a> Mazal ṛeṣṣan allen-nsen deg-s ɣer tegnaw, mi sen-d-dehṛen sin yergazen s llebsa tamellalt, nnan-asen :  </p> 
 <p><a name="an_1:11">1:11</a> Ay imezdaɣ n tmurt n Jlili, acuɣeṛ mazal-ikkun tețmuqqulem ɣer tegnaw ? Akken i twalam Sidna Ɛisa yuli ɣer igenni, akken daɣen ara t-twalim asmi ara d-yuɣal.  </p> 
 <p><a name="an_1:12">1:12</a> Seg iɣil uzemmur, uɣalen ɣer temdint n Lquds ibeɛden azal n tikli n wass n westeɛfu. </p> 
 <p><a name="an_1:13">1:13</a> Mi wwḍen ɣer temdint n Lquds, ulin ɣer texxamt anda nnumen țnejmaɛen. Yella gar-asen Buṭrus, Yuḥenna, Yeɛqub, Andriyus, Filibas, Suma, Bartelmay, Matta, Yeɛqub mmi-s n Ḥalfi, Semɛun awaṭani akk-d Yahuda mmi-s n Yeɛqub. </p> 
 <p><a name="an_1:14">1:14</a> Țnejmaɛen s yiwen n ṛṛay iwakken ad deɛɛun ɣer Sidi Ṛebbi nutni d kra n tlawin, llan yid-sen daɣen watmaten n Sidna Ɛisa akk-d yemma-s, Meryem.  </p> 
 <p><a name="an_1:15">1:15</a> Deg ussan-nni, llan nnejmaɛen azal n meyya uɛecrin n wayetmaten, yekker-ed Buṭrus gar-asen, yenna-d :  </p> 
 <p><a name="an_1:16">1:16</a> Ay atmaten, ilaq a d-yedṛu wayen i d-yenna Ṛṛuḥ iqedsen si zik seg yimi n Sidna Dawed, ɣef wayen yeɛnan Yudas i d-yezwaren zdat n wid i geṭṭfen Sidna Ɛisa.  </p> 
 <p><a name="an_1:17">1:17</a> Yudas-agi yella d yiwen seg neɣ, yerna ițekki di leqdic-nneɣ ɣef Sidi Ṛebbi. </p> 
 <p><a name="an_1:18">1:18</a> Argaz-agi yuɣ akal s yedrimen-nni n lexdeɛ, yuɣal yeɣli ɣef wudem, ifelleq uɛebbuḍ-is, mmaren-d akk ijuɣdan-is.  </p> 
 <p><a name="an_1:19">1:19</a> Imezdaɣ n temdint n Lquds slan akk s wannect-agi, daymi i semman i wemkan-nni s tmeslayt-nsen : Ḥaqel damma , yeɛni iger n idammen. </p> 
 <p><a name="an_1:20">1:20</a> Am akken yura di tektabt n Sabur : Axxam-is ad yexlu, ur t-izeddeɣ yiwen. Yura daɣen :  Wayeḍ ad yeṭṭef amkan-is.  </p> 
 <p><a name="an_1:21">1:21</a> Ilaq-aɣ ihi a d-nextiṛ yiwen seg wid yellan yid-nneɣ mi nțeddu d Sidna Ɛisa di mkul amkan, </p> 
 <p><a name="an_1:22">1:22</a> seg wasmi i t-isseɣḍes Yeḥya aɣeṭṭas armi d ass i deg yețwarfed ɣer igenni ; iwakken ad yili am nukkni d inigi n ḥeggu-ines.  </p> 
 <p><a name="an_1:23">1:23</a> Sbedden-d sin yergazen : Yusef yețțusemman Barsaba iwumi qqaṛen daɣen « Aḥeqqi » akk-d Matyas.  </p> 
 <p><a name="an_1:24">1:24</a> Dɛan ɣer Sidi Ṛebbi s imeslayen agi : A Sidi Ṛebbi, kečč yesnen ulawen n yemdanen meṛṛa, sbeggen aɣ-ed anwa deg-sen i gwulmen  </p> 
 <p><a name="an_1:25">1:25</a> iwakken ad iṭṭef amkan di lecɣal n ṛṛusul i ɣef yeṭṭaxeṛ Yudas, imi nețța iṛuḥ ɣer wemkan i s-ilaqen. </p> 
 <p><a name="an_1:26">1:26</a> Gren tasɣaṛt, tṣaḥ-ed i Matyas ; dɣa rnan-t-id ɣer ḥdac-nni n ṛṛusul.  </p> 
 <p><a name="an_2:1">2:1</a> Ass n Lɛid n wass wis xemsin, inelmaden llan nnejmaɛen akk deg yiwen n wemkan. </p> 
 <p><a name="an_2:2">2:2</a> Taswiɛt kan, yekka-d yiwen n lḥess ameqqran seg igenni am waḍu iǧehden, yeččuṛ akk axxam-nni anda qqimen.  </p> 
 <p><a name="an_2:3">2:3</a> Sehṛen-asen-d am akken d ilsawen n tmes ițemfaṛaqen wway gar-asen, rsen-d ɣef yal yiwen deg sen. </p> 
 <p><a name="an_2:4">2:4</a> Sčuṛen akk s Ṛṛuḥ iqedsen, bdan țmeslayen s tutlayin nniḍen, yal yiwen seg-sen iheddeṛ s tutlayt i s-d-yefka Ṛṛuḥ iqedsen. </p> 
 <p><a name="an_2:5">2:5</a> Deg ussan-nni, di temdint n Lquds, llan yemdanen n wat Isṛail yețḍuɛun Sidi Ṛebbi, i d-yusan si mkul tamurt n ddunit. </p> 
 <p><a name="an_2:6">2:6</a> Mi slan i lḥess-nni, lɣaci uzzlen d meṛṛa ; dehcen, axaṭer sellen-asen țmeslayen-d s tutlayt n tmurt i deg țɛicin.  </p> 
 <p><a name="an_2:7">2:7</a> Wehmen, țɛeǧben, qqaṛen wway gar-asen : Imdanen-agi yețmeslayen mačči n tmurt n Jlili ? </p> 
 <p><a name="an_2:8">2:8</a> Amek armi yal yiwen deg-nneɣ isell-asen țmeslayen s tmeslayt-is ?  </p> 
 <p><a name="an_2:9">2:9</a> Llan gar-aneɣ yemdanen i d yusan si tmurt n Bartas, si tmurt n Midyas, si tmurt n Ilam, kra nniḍen si tmurt n Mizubuṭamya, si tmurt n Yahuda akk-d tmurt n Kafadusya, wiyaḍ si tmurt n Tqenṭert, si tmurt n Asya, </p> 
 <p><a name="an_2:10">2:10</a> si tmurt n Frijya akk-d Bamfilya. Usan-d daɣen si tmurt n Maṣer, si leǧwahi n Qiṛwan di tmurt n Libya, widak i d-yusan si temdint n Ṛuma ; </p> 
 <p><a name="an_2:11">2:11</a> llan daɣen wat Isṛail akk-d wid itabaɛen di ddin nsen, llan wid n tegzirt n Kritus akk-d waɛṛaben. Amek armi i sen-nesla țmeslayen s tutlayin-nneɣ meṛṛa ɣef leɛǧayeb n Sidi Ṛebbi ? </p> 
 <p><a name="an_2:12">2:12</a> Dehcen akk, ur ẓrin d acu ara xemmemen, qqaṛen wway gar-asen : D acu-t wayagi ?  </p> 
 <p><a name="an_2:13">2:13</a> Ma d wiyaḍ stehzayen fell-asen qqaṛen : D askaṛ i sekkṛen !  </p> 
 <p><a name="an_2:14">2:14</a> Buṭrus yellan akk-d ḥdac n ṛṛusul-nni, issaɛli taɣect-is yenna i lɣaci : Ay imezdaɣ n temdint n Lquds akk-d kunwi meṛṛa isɛeddayen ussan di temdint-agi, slet-ed mliḥ i wayen ara wen-d-iniɣ, ilaq aț-țeẓrem ayagi : </p> 
 <p><a name="an_2:15">2:15</a> irgazen-agi ur skiṛen ara am akken i ɣilen kra seg-wen, axaṭer ațan d țesɛa n ṣṣbeḥ kan !  </p> 
 <p><a name="an_2:16">2:16</a> Lameɛna ayagi yedṛan d ayen i d-icar nnbi Yuwil :  </p> 
 <p><a name="an_2:17">2:17</a> Sidi Ṛebbi yenna-d : Deg ussan ineggura a d-sserseɣ Ṛṛuḥ-iw ɣef yemdanen meṛṛa ; arrac-nwen d yessi-twen a d-țcirin ayen ara sen-d-țxebbiṛeɣ, ilmeẓyen-nwen a sen-d-țweḥḥiɣ, imɣaṛen-nwen a sen-d-mmeslayeɣ di tirga.  </p> 
 <p><a name="an_2:18">2:18</a> Anɛam, deg ussan-nni a d-smireɣ Ṛṛuḥ-iw ɣef yiqeddacen-iw ț-țqeddacin-iw, a d-ciren s wayen ara sen-d-țxebbiṛeɣ.  </p> 
 <p><a name="an_2:19">2:19</a> Deg igenni ad xedmeɣ leɛǧayeb, di lqaɛa a d-sekneɣ licaṛat yessewhamen : idammen, times d usigna n ddexxan.  </p> 
 <p><a name="an_2:20">2:20</a> Iṭij ad yuɣal d ṭṭlam, aggur ad yuɣal am idammen uqbel a d-yas wass n Sidi Ṛebbi ; ass-nni d ass ameqqran, yeččuṛen d lɛaḍima.  </p> 
 <p><a name="an_2:21">2:21</a> Imiren kra n win ineddhen s yisem n Sidi Ṛebbi ad ițțusellek.  </p> 
 <p><a name="an_2:22">2:22</a> Ay imdanen n wat Isṛail, ḥesset-ed mliḥ i wayen ara wen-d iniɣ : ɣef wayen yeɛnan Ɛisa anaṣari, teẓram akk belli Sidi Ṛebbi isbeggen ed tazmert-is mi gexdem zdat-wen lbeṛhanat, licaṛat d leɛǧubat, am akken i teẓram s yiman-nwen.  </p> 
 <p><a name="an_2:23">2:23</a> Akken yella si zik di leqsed n Sidi Ṛebbi ; argaz-agi tezzenzem-t, d kunwi i t-yenɣan mi t-tsellmem ger ifassen n ikafriwen i t-isemmṛen ɣef lluḥ. </p> 
 <p><a name="an_2:24">2:24</a> Lameɛna Sidi Ṛebbi yesseḥya t-id, isellek-it-id si tesraft n lmut axaṭer d lmuḥal a t-teṭṭef ɣuṛ-es. </p> 
 <p><a name="an_2:25">2:25</a> Sidna Dawed yenna-d fell-as : Țwaliɣ kull ass Sidi Ṛebbi zdat-i axaṭer yețțili ɣer uyeffus-iw iwakken ur ɣelliɣ ara ;  </p> 
 <p><a name="an_2:26">2:26</a> daymi i gefṛeḥ wul-iw, i tcennu taɣect-iw ; ula d iɣsan-iw ad ṛtiḥen di laman ;  </p> 
 <p><a name="an_2:27">2:27</a> axaṭer ur iyi-tețțaǧǧaḍ ara di laxeṛt,ur tețțaǧaḍ ara win i k-iḍuɛen ad yerku deg uẓekka.  </p> 
 <p><a name="an_2:28">2:28</a> Teldiḍ-iyi iberdan n tudert ; ul-iw yeččuṛ d lfeṛḥ imi tețțiliḍ yid-i.  </p> 
 <p><a name="an_2:29">2:29</a> Ay atmaten ilaq-iyi a wen d-mmeslayeɣ ɛinani ɣef tideț yeɛnan jeddi-tneɣ Sidna Dawed. Teẓram belli yemmut, yețwamḍel, yerna aẓekka ines mazal-it ar ass-a gar-aneɣ. </p> 
 <p><a name="an_2:30">2:30</a> Lameɛna imi d nnbi i gella, yeẓra belli Sidi Ṛebbi ad iṭṭef di lemɛahda i s-d-yefka mi s-d-yenna : D yiwen si tarwa-k ara sbeddeɣ d agellid deg wemkan-ik.  </p> 
 <p><a name="an_2:31">2:31</a> Ɣef ḥeggu n Lmasiḥ i d-icar mi d-yenna lehduṛ-agi ; axaṭer ț-țideț, ur t-yeǧǧi di laxeṛt, ur t-yeǧǧi ad yerku.+  </p> 
 <p><a name="an_2:32">2:32</a> Win ɣef i wen-d-țmeslayeɣ, d Ɛisa-agi i d-isseḥya Sidi Ṛebbi si ger lmegtin, aql-aɣ akk d inagan. </p> 
 <p><a name="an_2:33">2:33</a> Mi gețwarfed ɣer uyeffus n Sidi Ṛebbi, yețțunefk-as Ṛṛuḥ iqedsen i s-yewɛed Baba Ṛebbi, nețța ismar-it-id fell-aɣ ; d ayen i teslam, i twalam. </p> 
 <p><a name="an_2:34">2:34</a> Mačči d Sidna Dawed i gulin ɣer igenni, axaṭer d nețța i d-yennan : Sidi Ṛebbi yenna-d i Ssid-iw :   </p> 
 <p><a name="an_2:35">2:35</a> Sṭef amkan n lḥekma ɣer uyeffus-iw alamma rriɣ iɛdawen-ik seddaw idaṛṛen-ik.+ +  </p> 
 <p><a name="an_2:36">2:36</a> Ilaq ihi at Isṛail meṛṛa ad ẓren s tideț belli Ɛisa-nni i tsemmṛem ɣef lluḥ, d nețța i gerra Sidi Ṛebbi d Agellid ameqqran, d Lmasiḥ.  </p> 
 <p><a name="an_2:37">2:37</a> Mi slan i wayen i d-yenna Buṭrus, lɣaci meṛṛa saxen wulawen-nsen, nnan i Buṭrus d ṛṛusul nniḍen : Ay atmaten, d acu ara nexdem ?  </p> 
 <p><a name="an_2:38">2:38</a> Buṭrus yerra-yasen : Tubet, beddlet tikli ! Mkul yiwen deg-wen ad yețwaɣḍes s yisem n Sidna Ɛisa Lmasiḥ iwakken a wen-țwasemmḥen ddnubat-nwen, yerna a wen-d yețțunefk Ṛṛuḥ iqedsen.  </p> 
 <p><a name="an_2:39">2:39</a> Axaṭer lweɛd-agi n Sidi Ṛebbi, yețțunefk-ed i kunwi d warraw-nwen akk-d wid meṛṛa n leǧnas nniḍen iwumi ara d-issiwel Sidi Ṛebbi Illu-nneɣ. </p> 
 <p><a name="an_2:40">2:40</a> Buṭrus inehhu-ten s waṭas n yimeslayen nniḍen, iwakken ad amnen, yeqqaṛ-asen : Xḍut i lǧil-agi iɛewjen iwakken aț-țețțusellkem !  </p> 
 <p><a name="an_2:41">2:41</a> Wid iqeblen ayen i d-yenna Buṭrus, qeblen ad țwaɣedṣen. Ass-nni rnan-d wazal n tlata alaf n yemdanen i gumnen.  </p> 
 <p><a name="an_2:42">2:42</a> Seg imiren, zgan țnejmaɛen am atmaten iwakken ad slen i wselmed n ṛṛusul ; beṭṭun aɣṛum n usmekti n Lmasiḥ, yerna deɛɛun ɣer Sidi Ṛebbi s yiwen n wul.  </p> 
 <p><a name="an_2:43">2:43</a> Tɛeǧǧben, tekcem-iten akk tugdi mi țwalin ṛṛusul xeddmen aṭas n lbeṛhanat d leɛǧayeb.  </p> 
 <p><a name="an_2:44">2:44</a> Wid akk yumnen, țɛicin s tdukli yerna beṭṭun wway gar-asen ayen akk sɛan. </p> 
 <p><a name="an_2:45">2:45</a> Znuzun ayla-nsen meṛṛa, ayen akken i d-jemɛen, feṛqen-t wway gar-asen mkul yiwen ɣef leḥsab n wakken yeḥwaǧ. </p> 
 <p><a name="an_2:46">2:46</a> Kull ass, s yiwen ṛṛay, țnejmaɛen di lǧameɛ iqedsen, țnejmaɛen daɣen deg ixxamen-nsen, beṭṭun aɣṛum iwakken ad mektin Lmasiḥ, tețțen lqut-nsen s nneya d lfeṛḥ ;  </p> 
 <p><a name="an_2:47">2:47</a> țḥemmiden Sidi Ṛebbi, imdanen meṛṛa țqadaṛen-ten ; Sidi Ṛebbi irennu-d kull ass ɣer tejmaɛt-nsen wid ițțusellken.  </p> 
 <p><a name="an_3:1">3:1</a> Aț-țili d leǧwahi n tlata n tmeddit, Buṭrus d Yuḥenna ddukklen, ulin ɣer lǧameɛ iqedsen iwakken ad dɛun ɣer Sidi Ṛebbi.  </p> 
 <p><a name="an_3:2">3:2</a> Țțawin-d mkul ass yiwen n wergaz i d-ilulen d aquḍar ɣer zdat n tewwurt n lǧameɛ, tin iwumi qqaṛen « tawwurt tucbiḥt »; yețɣimi dinna iwakken ad issuter tin n Ṛebbi i wid i gkeččmen ɣer lǧameɛ-nni.  </p> 
 <p><a name="an_3:3">3:3</a> Mi gwala Buṭrus d Yuḥenna țeddun ad kecmen, issuter-asen tin n Ṛebbi. </p> 
 <p><a name="an_3:4">3:4</a> Buṭrus d Yuḥenna, ṛeṣṣan allen-nsen fell-as, nnan-as : Muqel-ed ɣuṛ-nneɣ !  </p> 
 <p><a name="an_3:5">3:5</a> Argaz-nni imuqel-iten, iḍmeɛ deg-sen a s-d-fken kra. </p> 
 <p><a name="an_3:6">3:6</a> Dɣa Buṭrus yenna-yas : Ur sɛiɣ idrimen n lfeṭṭa, ur sɛiɣ wid n ddheb, lameɛna ayen sɛiɣ a k-t-fkeɣ : s yisem n Sidna Ɛisa Lmasiḥ anaṣari, ekker telḥuḍ !  </p> 
 <p><a name="an_3:7">3:7</a> Iṭṭef-it-id seg ufus ayeffus, issekker-it-id. Imiren kan iḍaṛṛen-is ț-țkeɛburin n idaṛṛen-is ḥlan. </p> 
 <p><a name="an_3:8">3:8</a> Ibedd ɣef yidaṛṛen-is, ineggez, yebda ileḥḥu. Ikcem yid-sen ɣer lǧameɛ, ileḥḥu ițjellib, ițḥemmid Sidi Ṛebbi. </p> 
 <p><a name="an_3:9">3:9</a> Lɣaci akk walan-t mi gleḥḥu ițḥemmid Sidi Ṛebbi. </p> 
 <p><a name="an_3:10">3:10</a> Mi t-ɛeqlen qqaṛen : Mačči d win akken yețɣimin ɣer tewwurt tucbiḥt n lǧameɛ iqedsen win akken issuturen tin n Ṛebbi ? Dehcen meṛṛa, wehmen deg wayen yedṛan yid-es. </p> 
 <p><a name="an_3:11">3:11</a> Argaz-nni iteddu d Buṭrus akk-d Yuḥenna, ur yeṭṭixiṛ ara fell-asen, lɣaci-nni meṛṛa wehmen, uzzlen-d ɣuṛ-sen ɣer usqif iwumi qqaṛen « Asqif n Sidna Sliman ».  </p> 
 <p><a name="an_3:12">3:12</a> Buṭrus mi gwala ayagi, yenṭeq ɣer lɣaci yenna : Ay at Isṛail, acuɣeṛ akka twehmem deg wayen yedṛan ? Acuɣeṛ i d-țmuqulem ɣuṛ-nneɣ, am akken d nukkni i gesselḥan argaz-agi s tezmert-nneɣ neɣ s ṭṭaɛa-nneɣ ?  </p> 
 <p><a name="an_3:13">3:13</a> D Sidi Ṛebbi Illu n Sidna Ibṛahim, n Isḥaq d Yeɛqub, Ṛebbi n lejdud-nneɣ, d nețța i d-isbeggnen lɛaḍima n Sidna Ɛisa aqeddac-is, win akken i tnekṛem, i tsellmem i lḥakem Bilaṭus i gebɣan a s-d-iserreḥ. </p> 
 <p><a name="an_3:14">3:14</a> Kunwi tnekṛem win akken yellan d azedgan, d aḥeqqi ; tessutrem a d-serrḥen i bu tmegṛaḍ deg umkan-is. </p> 
 <p><a name="an_3:15">3:15</a> Tenɣam Mmi-s n ugellid n tudert, meɛna Sidi Ṛebbi yesseḥya-t-id si ger lmegtin , aql-aɣ d inigan. </p> 
 <p><a name="an_3:16">3:16</a> Ț-țazmert yellan deg isem n Sidna Ɛisa i d-yerran lǧehd i wergaz agi i tețwalim, yerna tessnem-t. D Sidna Ɛisa s wayes numen i t-isseḥlan zdat-wen meṛṛa.  </p> 
 <p><a name="an_3:17">3:17</a> Ay atmaten, ẓriɣ s lqella n lefhama i txedmem annect-nni, am kunwi am imeqranen-nwen. </p> 
 <p><a name="an_3:18">3:18</a> Meɛna s wakka i gesnekmal Sidi Ṛebbi ayen i d-yenna si zik seg imawen n lenbiya meṛṛa : yenna-d belli Lmasiḥ-ines ad inɛețțab.  </p> 
 <p><a name="an_3:19">3:19</a> Ndemmet ihi, beddlet tikli iwakken ad mḥun ddnubat-nwen, </p> 
 <p><a name="an_3:20">3:20</a> iwakken a kkun-id-asen wussan n ṛṛaḥa s ɣuṛ Sidi Ṛebbi, a wen-d iceggeɛ Ɛisa Lmasiḥ, win akken i wen-ihegga.  </p> 
 <p><a name="an_3:21">3:21</a> Tura atan deg igenni alamma yusa-d lweqt i deg ara tuɣal txelqit meṛṛa ț-țajḍiṭ, akken i t-id-yenna Sidi Ṛebbi si zik seg imawen n lenbiya.  </p> 
 <p><a name="an_3:22">3:22</a> Sidna Musa yenna i wat Isṛail : Sidi Ṛebbi Illu-nwen a wen-d iceggeɛ si ger watmaten-nwen,+ yiwen n nnbi am nekk, a s-tḥessem deg wayen akk ara wen-d-yini.+  </p> 
 <p><a name="an_3:23">3:23</a> Kra n win ur nețḥessis ara i nnbi-yagi, ad yețwakkes si ddunit ger watmaten-is.  </p> 
 <p><a name="an_3:24">3:24</a> Lenbiya meṛṛa, si zzman n nnbi Samwil d wid meṛṛa i d-yernan deffir-es, caren-d daɣen ɣef wussan agi i deg nețɛici.  </p> 
 <p><a name="an_3:25">3:25</a> D kunwi i d arraw n lenbiya, i d arraw n leɛqed-nni  i gexdem Sidi Ṛebbi d lejdud-nneɣ, mi s-yenna i Sidna Ibṛahim : S dderya n dderya-k ara barkeɣ tiwaculin meṛṛa n ddunit+ .  </p> 
 <p><a name="an_3:26">3:26</a> I kunwi d imezwura iwumi i d iceggeɛ Sidi Ṛebbi aqeddac-is, isseḥya t-id iwakken a kkun-ibarek yerna a d-issebɛed mkul yiwen deg-wen seg yir n lecɣal-is.  </p> 
 <p><a name="an_4:1">4:1</a> Buṭrus d Yuḥenna mazal-iten heddṛen i lɣaci mi d-wwḍen kra n lmuqedmin, yedda-d yid-sen umeqqran n iɛessasen n lǧameɛ iqedsen akk-d kra n yergazen si tejmaɛt n isaduqiyen  </p> 
 <p><a name="an_4:2">4:2</a> Zeɛfen mi walan ṛṛusul sselmaden lɣaci yerna ssefhamen-ten ɣef ḥeggu n lmegtin qqaṛen-asen : -- Akken i d-yeḥya Sidna Ɛisa si ger lmegtin, akken daɣen ara d-ḥyun lmegtin.  </p> 
 <p><a name="an_4:3">4:3</a> Sṭfen-ten, rran-ten ɣer lḥebs armi d azekka-nni axaṭer imiren yeɣli-d yiḍ. </p> 
 <p><a name="an_4:4">4:4</a> Lameɛna aṭas seg wid yeslan i wawal, umnen. Ad ilin azal n xemsa alaf n yemdanen i gumnen.  </p> 
 <p><a name="an_4:5">4:5</a> Azekka-nni, lḥukkam, lecyux akk-d lɛulama n ccariɛa yellan di temdint n Lquds, nnejmaɛen.  </p> 
 <p><a name="an_4:6">4:6</a> Yella lmuqeddem ameqqran ițusemman Hannan akk-d wid i s-yețțilin, ḥedṛen daɣen : Kayef, Yuḥenna d Alixandru.  </p> 
 <p><a name="an_4:7">4:7</a> Wwin-d Buṭrus d Yuḥenna, sbedden-ten di tlemmast-nsen akken a ten-beḥten, qqaṛen-asen : Ansi i wen-d-tekka tezmert-agi, anwa isem s wayes i tneddhem armi txedmem ayagi ?  </p> 
 <p><a name="an_4:8">4:8</a> Dɣa Buṭrus yeččuṛen d Ṛṛuḥ iqedsen, yenna-yasen :  </p> 
 <p><a name="an_4:9">4:9</a> A lḥukkam d lecyux n wat Isṛail ! Imi ɣef lxiṛ i nexdem i wergaz-agi aɛiban d wamek i geḥla i ɣ-d-testeqsayem ass-agi, </p> 
 <p><a name="an_4:10">4:10</a> ilaq aț-țeẓrem kunwi d wat Isṛail meṛṛa, belli s yisem n Sidna Ɛisa anaṣari, win akken i tsemmṛem ɣef wumidag, win akken i d-yesseḥya Sidi Ṛebbi si ger lmegtin, s tezmert-is i gbedd wergaz agi tura zdat-wen yeḥla. </p> 
 <p><a name="an_4:11">4:11</a> D nețța i ɣef d-nnant tira iqedsen : D azṛu-nni i tḍeggṛem kunwi ay ibennayen, i guɣalen d azṛu i geṭṭfen lebni. +  </p> 
 <p><a name="an_4:12">4:12</a> Deg-s i gella leslak ; ulac di ddunit meṛṛa isem nniḍen s wayes i nezmer a nețțusellek.  </p> 
 <p><a name="an_4:13">4:13</a> Mi walan belli Buṭrus d Yuḥenna heddṛen s laman mbla akukru, wehmen axaṭer ẓran d imdanen ur neɣṛi ara ; ɛeqlen-ten seg wid yeddan d Sidna Ɛisa.  </p> 
 <p><a name="an_4:14">4:14</a> Lameɛna imi walan argaz-nni yeḥlan ibedd zdat-sen, ur ufin d acu ara d-inin. </p> 
 <p><a name="an_4:15">4:15</a> Umṛen-ten ad ffɣen seg unejmaɛ n ccṛeɛ iwakken ad mcawaṛen wway gar-asen, </p> 
 <p><a name="an_4:16">4:16</a> nnan : D acu ara nexdem s yemdanen agi ? Imezdaɣ n temdint n Lquds akk wwin-d s lexbaṛ belli xedmen lbeṛhan yerna ț-țideț ur nezmir ara a nenkeṛ ayagi.  </p> 
 <p><a name="an_4:17">4:17</a> Lameɛna iwakken lexbaṛ-agi ur iteffeɣ ara ger lɣaci, a neggal deg-sen ur țțuɣalen ara ad mmeslayen ula i yiwen s yisem-agi.  </p> 
 <p><a name="an_4:18">4:18</a> Uɣalen ssawlen-asen, gullen deg-sen iwakken ur țțuɣalen ara ad mmeslayen neɣ ad slemden s yisem-agi n Ɛisa. </p> 
 <p><a name="an_4:19">4:19</a> Lameɛna Buṭrus d Yuḥenna nnan-asen : D acu i d lḥeqq zdat Ṛebbi, a wen-naɣ awal i kunwi neɣ a naɣ awal i Sidi Ṛebbi ? </p> 
 <p><a name="an_4:20">4:20</a> Walit s yiman nwen ! Nukni ur nezmir ara a nessusem ɣef wayen neẓra d wayen nesla.  </p> 
 <p><a name="an_4:21">4:21</a> Rnan gullen deg-sen, lameɛna serrḥen-asen imi ur d-ufin ara sebba s wayes ara ten-ɛaqben, yerna lɣaci akk țḥemmiden Sidi Ṛebbi ɣef wayen yedṛan, </p> 
 <p><a name="an_4:22">4:22</a> axaṭer argaz-agi i geḥlan s lbeṛhan di leɛmeṛ-is akteṛ n ṛebɛin iseggasen.  </p> 
 <p><a name="an_4:23">4:23</a> Mi sen-serrḥen, ṛuḥen ɣer watmaten, ḥkan-asen akk ayen i sen-d-nnan lmuqedmin imeqqranen d lecyux.  </p> 
 <p><a name="an_4:24">4:24</a> Mi slan watmaten-nni i wayen i d-yenna Buṭrus, akken ma llan serrḥen i taɣect-nsen, nedhen ɣer Sidi Ṛebbi nnan : A Sidi Ṛebbi, kečč i gxelqen igenni, lqaɛa, lebḥeṛ d wayen akk yellan. </p> 
 <p><a name="an_4:25">4:25</a> D kečč i d-ineṭqen s Ṛṛuḥ-ik iqedsen seg yimi n jeddi-tneɣ Dawed aqeddac-ik, mi d-yenna : Acuɣeṛ ccwal-agi ger leǧnas,  acuɣeṛ igduden ggaren-d anagar yir ixemmimen ?  </p> 
 <p><a name="an_4:26">4:26</a> Igelliden n ddunit heggan iman nsen, imeqqranen ddukklen akk, ffɣen-d d iɛdawen n Sidi Ṛebbi akk-d Lmasiḥ-ines.  </p> 
 <p><a name="an_4:27">4:27</a> Axaṭer ț-țideț, di temdint-agi, Hiṛudus d Bilaṭus mcawaṛen, ddukklen nutni d ibeṛṛaniyen akk-d wat Isṛail, uɣalen d iɛdawen n Sidna Ɛisa imqeddes i textaṛeḍ d Lmasiḥ.   </p> 
 <p><a name="an_4:28">4:28</a> Ayen akka xedmen, d ayen akken i tqesdeḍ keččini di lebɣi-inek, s tezmert-ik, si zik. </p> 
 <p><a name="an_4:29">4:29</a> I tura, a Sidi Ṛebbi, wali-d amek akka i ɣ ssaggaden ; efk-aɣ-d a Sidi Ṛebbi tazmert s wayes ara nbecceṛ awal-ik s ṭṭmana mbla akukru. </p> 
 <p><a name="an_4:30">4:30</a> Ssers-ed afus-ik iwakken ad ḥlun imuḍan, iwakken a d-ilint licaṛat d lbeṛhanat s yisem n Sidna Ɛisa aqeddac-ik imqeddes.  </p> 
 <p><a name="an_4:31">4:31</a> Mi fukken taẓallit, amkan-nni anda nnejmaɛen yergagi ; ččuṛen akk s Ṛṛuḥ iqedsen, bdan țbecciṛen awal n Sidi Ṛebbi s ṭṭmana mbla tugdi.  </p> 
 <p><a name="an_4:32">4:32</a> Wid akk yumnen s Lmasiḥ, ddukklen akk s yiwen n wul, s yiwen n ṛṛuḥ, ula d yiwen deg-sen ur yeḥsib ayen yesɛa d ayla-s i yiman-is kan, meɛna cerken kullec wway gar-asen.  </p> 
 <p><a name="an_4:33">4:33</a> Ṛṛusul țcehhiden-d ɣef ḥeggu n Sidna Ɛisa s tezmert tameqqrant yerna ṛṛeḥma n Sidi Ṛebbi txeddem deg-sen s lǧehd.  </p> 
 <p><a name="an_4:34">4:34</a> Ulac gar-asen win ixuṣṣen ; wid akk yesɛan igran neɣ ixxamen zzenzen-ten, </p> 
 <p><a name="an_4:35">4:35</a> idrimen-nni i d-țțawin deg-sen, țțawin-ten ɣer ifassen n ṛṛusul i ten-ifeṛṛqen i mkul yiwen ɣef leḥsab n wakken yeḥwaǧ.  </p> 
 <p><a name="an_4:36">4:36</a> Yusef illan n at Lewwi iwumi semman ṛṛusul Barnabas, ( yeɛni argaz inehhun ) i glulen di tegzirt n Qubṛus, </p> 
 <p><a name="an_4:37">4:37</a> yesɛa yiwen n yiger, izzenz-it, yewwi-d idrimen-nni, yefka-ten i ṛṛusul.  </p> 
 <p><a name="an_5:1">5:1</a> Yiwen wergaz isem-is Ananyas, nețța ț-țmeṭṭut-is Safira zzenzen yiwen n weḥric seg wakal-nsen ; msefhamen ad kksen yiwen n umur si ssuma n lbiɛ-nni ; </p> 
 <p><a name="an_5:2">5:2</a> amur nniḍen yewwi-t-id ɣer ifassen n ṛṛusul.  </p> 
 <p><a name="an_5:3">5:3</a> Buṭrus inṭeq ɣuṛ-es yenna : Ay Ananyas, acuɣeṛ i teǧǧiḍ Cciṭan yeččuṛ ul-ik ? Acuɣeṛ i teskadbeḍ i Ṛṛuḥ iqedsen mi tekkseḍ yiwen umur n yedrimen n wakal i tezzenzeḍ ? Uqbel a t- tezzenzeḍ, akal-nni yella d ayla-k !  </p> 
 <p><a name="an_5:4">5:4</a> Mi t-tezzenzeḍ, idrimen-nni daɣen d ayla-k ! Amek armi i teṣṣawḍed aț-țxeḍmeḍ yiwet lḥaǧa am tagi ? Ihi mačči i yemdanen iwumi teskadbeḍ, lameɛna i Sidi Ṛebbi !  </p> 
 <p><a name="an_5:5">5:5</a> Mi gesla i yimeslayen-nni, Ananyas yeɣli, yeffeɣ-it ṛṛuḥ, yemmut. Wid akk yellan dinna, ikcem-iten akk lxuf, țergigin.  </p> 
 <p><a name="an_5:6">5:6</a> Kkren ilmeẓyen kefnen-t, wwin-t a t-meḍlen. </p> 
 <p><a name="an_5:7">5:7</a> Ɛeddant wazal n tlata sswayeɛ, mi d-tekcem tmeṭṭut n Ananyas, nețțat ur tesli ara s wayen yedṛan. </p> 
 <p><a name="an_5:8">5:8</a> Buṭrus yenṭeq ɣuṛ-es yenna-yas : Ini-yi-d, ma s ssuma-yagi i tezzenzem akal-nwen ? Terra-yas-d : Ih, s ssuma-yagi i t-nezzenz.  </p> 
 <p><a name="an_5:9">5:9</a> Dɣa Buṭrus yenna-yas : Amek armi i temsefhamem iwakken aț-țjeṛṛbem Ṛṛuḥ n Sidi Ṛebbi ? Wid imeḍlen argaz-im atnan ɣer tewwurt, a kkem-awin ula d kemm !  </p> 
 <p><a name="an_5:10">5:10</a> Imiren kan teɣli ɣer idaṛṛen-is yeffeɣ-iț ṛṛuḥ, temmut. Ilmeẓyen-nni mi d-kecmen ufan-ț-id temmut, wwin-ț meḍlen-ț ɣer tama n wergaz-is. </p> 
 <p><a name="an_5:11">5:11</a> Yeɣli-d ṛṛɛac ( lxuf ) d ameqqran ɣef tejmaɛt n watmaten d wid akk yeslan s wayen yedṛan.  </p> 
 <p><a name="an_5:12">5:12</a> Aṭas n lbeṛhanat d lmuɛǧizat i xeddmen ṛṛusul ger lɣaci. Wid yumnen țnejmaɛen akk s yiwen ṛṛay deg wesqif n Sidna Sliman.  </p> 
 <p><a name="an_5:13">5:13</a> Lɣaci meṛṛa țcekkiṛen-ten meɛna kukran akk a d-rnun ɣuṛ-sen.  </p> 
 <p><a name="an_5:14">5:14</a> Leḥsab n yergazen ț-țilawin yumnen s Sidna Ɛisa yețzad irennu.  </p> 
 <p><a name="an_5:15">5:15</a> Țțawin-d imuḍan ɣer iberdan, srusun-ten ɣef tgertyal iwakken m'ara d-iɛeddi Buṭrus, ad innal yiwen seg-sen ulamma s tili-ines.  </p> 
 <p><a name="an_5:16">5:16</a> Lɣaci țțazzalen-d daɣen ula si temdinin nniḍen iqeṛben tamdint n Lquds ; țțawin-d imuḍan akk-d wid yețwazedɣen, ḥellun akken ma llan.  </p> 
 <p><a name="an_5:17">5:17</a> Ɣef ddemma n wayen ideṛṛun kkrent-asen-d tismin i lmuqeddem ameqqran akk-d isaduqiyen-nni yellan yid-es, </p> 
 <p><a name="an_5:18">5:18</a> ṭṭfen ṛṛusul, rran-ten ɣer lḥebs n baylek.  </p> 
 <p><a name="an_5:19">5:19</a> Lameɛna deg yiḍ-nni, yiwen lmelk n Sidi Ṛebbi yeldi tiwwura n lḥebs, issufeɣ-iten-id, yenna-yasen :  </p> 
 <p><a name="an_5:20">5:20</a> Ṛuḥet beccṛet di lǧameɛ iqedsen, slemdet i lɣaci awal n tudert.  </p> 
 <p><a name="an_5:21">5:21</a> Mi slan i wayen i sen-d-yenna lmelk-nni, ṛuḥen ṣṣbeḥ zik ɣer lǧameɛ iqedsen, bdan sselmaden. Ma d lmuqeddem ameqqran d wid akk yellan yid-es, snejmaɛen-d lecyux akk-d imeqqranen n wat Isṛail ɣer unejmaɛ n ccṛeɛ ; imiren ceggɛen a d-awin ṛṛusul-nni si lḥebs.  </p> 
 <p><a name="an_5:22">5:22</a> Mi wwḍen iɛessasen ɣer lḥebs, ur ten-ufin ara dinna ; uɣalen-d ɣer unejmaɛ, nnan-asen :  </p> 
 <p><a name="an_5:23">5:23</a> Nufa lḥebs isekkeṛ, iɛessasen bedden zdat n tewwurt, lameɛna mi nekcem, ur nufi ula d yiwen zdaxel.  </p> 
 <p><a name="an_5:24">5:24</a> Mi ten-yewweḍ lexbaṛ-agi, lqebṭan n lǧameɛ akk-d lmuqedmin imeqqranen ɛewqen ! Ur ẓrin ara ɣer wanda ara taweḍ temsalt-agi.  </p> 
 <p><a name="an_5:25">5:25</a> Iṛuḥ-ed ɣuṛ-sen yiwen yenna yasen : Irgazen-nni i terram ɣer lḥebs, atnan deg ufrag n lǧameɛ iqedsen yerna sselmaden lɣaci.  </p> 
 <p><a name="an_5:26">5:26</a> Lqebṭan n lǧameɛ d iɛessasen ṛuḥen wwin-ten-id lameɛna ḥudren ten axaṭer uggaden a ten-ṛejmen lɣaci.  </p> 
 <p><a name="an_5:27">5:27</a> Mi ten-id-wwin ɣer zdat usqamu n ccṛeɛ, lmuqeddem ameqqran isteqsa ten yenna-yasen :  </p> 
 <p><a name="an_5:28">5:28</a> Neggul deg-wen ur tețțuɣalem aț-țeslemdem s yisem-agi ; lameɛna kunwi tessufɣem aselmed nwen di temdint n Lquds meṛṛa ! Tura tebɣam a d-terrem ɣer yirawen nneɣ tamgeṛṭ n wergaz-agi !  </p> 
 <p><a name="an_5:29">5:29</a> Buṭrus d ṛṛusul nniḍen erran asen : Ilaq a naɣ awal i Sidi Ṛebbi, mačči i yemdanen ! </p> 
 <p><a name="an_5:30">5:30</a> Ṛebbi, Illu n lejdud-nneɣ yesseḥya-d Sidna Ɛisa, win akken i tenɣam, i tsemmṛem ɣef lluḥ.  </p> 
 <p><a name="an_5:31">5:31</a> D nețța i gessuli Sidi Ṛebbi ɣer uyeffus-is d Agellid, d Amsellek, iwakken a d-yefk i wat Isṛail leɛfu n ddnubat m'ara beddlen tikli.  </p> 
 <p><a name="an_5:32">5:32</a> Aql-aɣ d inagan ɣef wayagi, nukkni akk-d Ṛṛuḥ iqedsen i d-yefka Sidi Ṛebbi i wid i t-iḍuɛen.  </p> 
 <p><a name="an_5:33">5:33</a> Mi sen-slan, ikcem-iten wurrif d ameqqran armi bɣan a ten-nɣen. </p> 
 <p><a name="an_5:34">5:34</a> Meɛna Gamalyel, yiwen n lɛalem n ccariɛa yellan d afarizi, ḥemmlen-t akk lɣaci ; ikker-ed deg unejmaɛ-nni, yefka lameṛ ad ssufɣen ṛṛusul-nni kra n lweqt. </p> 
 <p><a name="an_5:35">5:35</a> Dɣa yenṭeq yenna : Ay at Isṛail, ḥadret iman-nwen ɣef wayen akka tebɣam a t-txedmem i yemdanen-agi !  </p> 
 <p><a name="an_5:36">5:36</a> Mmektit-ed, mačči aṭas aya ikker-ed yiwen yețțuseman Tudas yerran iman-is d ameqqran. Azal n ṛebɛa meyya n yergazen i ddan yid-es; asmi i t-nɣan, wid akk i geddan yid-es mfaṛaqen, ur d-yeqqim lateṛ-nsen. </p> 
 <p><a name="an_5:37">5:37</a> Deffir Tudas, ikker-ed daɣen Yahuda ajlili di lweqt-nni n ujerred n wegdud ; aṭas daɣen n wid i geddan yid-es, meɛna ula d nețța mi gemmut, wid akk yeddan yid-es mfaṛaqen.  </p> 
 <p><a name="an_5:38">5:38</a> Ihi tura a wen-d-iniɣ : ur ceɣlet ara d yemdanen-agi, anfet-asen ad ṛuḥen. Ma yella aselmed-agi-nsen yekka-d s ɣuṛ-sen, ur yețdumu ara ;  </p> 
 <p><a name="an_5:39">5:39</a> ma yella s ɣuṛ Sidi Ṛebbi i d-ikka, ur tețțizmirem ara a t-tekksem ! Ḥadret iman-nwen ihi, neɣ m'ulac aț-țuɣalem d iɛdawen n Sidi Ṛebbi !  </p> 
 <p><a name="an_5:40">5:40</a> Mi mcawaṛen, uɣen ṛṛay-is. Uɣalen skecmen ṛṛusul-nni, fkan lameṛ a ten-wwten s iɛekkzan, rnan gullen deg-sen ur țțuɣalen ara ad mmeslayen s yisem-agi n Ɛisa, dɣa serrḥen-asen.  </p> 
 <p><a name="an_5:41">5:41</a> Mi d-ffɣen ṛṛusul seg unejmaɛ n ccṛeɛ, feṛḥen imi țwaḥesben d wid yuklalen ad țewten, ad țwaregmen ɣef ddemma n yisem n Sidna Ɛisa. </p> 
 <p><a name="an_5:42">5:42</a> Dɣa kull ass sselmaden țbecciṛen di lǧameɛ iqedsen akk-d ixxamen, lexbaṛ n lxiṛ  yeɛnan Ɛisa Lmasiḥ.  </p> 
 <p><a name="an_6:1">6:1</a> Deg ussan-nni, leḥsab n widak yumnen s Sidna Ɛisa yețzid irennu. At Isṛail i d-yusan si tmura nniḍen, yețmeslayen s tyunanit, ccetkan ɣef wat Isṛail yellan di tmurt, axaṭer m'ara feṛqen lmakla, ur d-țelhayen ara ț-țuǧǧal-nsen. </p> 
 <p><a name="an_6:2">6:2</a> Tnac ṛṛusul-nni snejmaɛen-d inelmaden meṛṛa, nnan-asen : Mačči d lḥeqq a neǧǧ Awal n Sidi Ṛebbi di rrif iwakken a nelhi d wefṛaq n lqut i mkul yiwen. </p> 
 <p><a name="an_6:3">6:3</a> Ihi ay atmaten, xtiṛet-ed si gar-awen sebɛa yergazen i deg sɛan laman yemdanen n tmurt meṛṛa, yeččuṛen d Ṛṛuḥ iqedsen akk-d leɛqel, d nutni ara nwekkel ɣef ccɣel-agi. </p> 
 <p><a name="an_6:4">6:4</a> Ma d nukkni, lweqt-nneɣ a t-nesɛeddi di tẓallit akk-d uselmed n wawal n Sidi Ṛebbi.  </p> 
 <p><a name="an_6:5">6:5</a> Tajmaɛt akk iɛeǧǧeb-as ṛṛay-agi. Kkren xtaṛen-d Stifan yellan d argaz yeččuṛen d liman akk-d Ṛṛuḥ iqedsen. Xtaṛen daɣen Filbas, Brukurus, Nikanur, Timun, Barmenas, akk-d Nikulus n temdint n Antyuc, i gkecmen si zik di ddin n wat Isṛail. </p> 
 <p><a name="an_6:6">6:6</a> Wwin-ten-id, sbedden-ten zdat n ṛṛusul. Ṛṛusul ssersen ifassen-nsen fell-asen, dɛan-asen s lxiṛ ɣer Sidi Ṛebbi. </p> 
 <p><a name="an_6:7">6:7</a> Yuɣal, awal n Sidi Ṛebbi iffeɣ yețwassen, leḥsab n wid yumnen s Lmasiḥ yețzid irennu di temdint n Lquds yerna aṭas n lmuqedmin i gumnen, qeblen abrid n Lmasiḥ.  </p> 
 <p><a name="an_6:8">6:8</a> Stifan yeččuṛen d ṛṛeḥma ț-țezmert n Sidi Ṛebbi, ixeddem lbeṛhanat d leɛǧayeb timeqqranin ger lɣaci. </p> 
 <p><a name="an_6:9">6:9</a> Kra n yemdanen n tejmaɛt n iḥeṛṛiyen i d-yusan si tmurt n Qiṛwan, si tmurt n Skandriya, si tmurt n Silisya akk-d wid i d-yusan si tmurt n Asya, usan-d ad mjadalen nutni d Stifan ; </p> 
 <p><a name="an_6:10">6:10</a> lameɛna ur zmiren ara a t-ɣelben deg awal axaṭer ițmeslay s ṣṣwab d leɛqel i s-d-yefka Ṛṛuḥ iqedsen. </p> 
 <p><a name="an_6:11">6:11</a> Imiren fkan tajɛelt i kra n yergazen iwakken a d-inin : « Nesla yas yekfeṛ ɣef Sidna Musa akk-d Sidi Ṛebbi ! » </p> 
 <p><a name="an_6:12">6:12</a> Sekkren ccwal ger lɣaci d lecyux akk-d lɛulama n ccariɛa , dɣa ṭṭfen Stifan ewwtent, wwin-t ɣer unejmaɛ n usqamu n ccṛeɛ.  </p> 
 <p><a name="an_6:13">6:13</a> Sbedden-d inigan i d-icehden fell-as s ẓẓur, nnan-d : Argaz-agi, yezga yekkat deg wemkan-agi iqedsen akk-d ccariɛa n Sidna Musa ! </p> 
 <p><a name="an_6:14">6:14</a> Nesla-yas mi geqqaṛ : « Ɛisa-nni Anaṣari, ad ihudd amkan agi yerna ad ibeddel lɛaddat i ɣ-d yeǧǧa Sidna Musa. »  </p> 
 <p><a name="an_6:15">6:15</a> Wid akk yellan deg unejmaɛ n ccṛeɛ, ṛeṣṣan allen-nsen ɣef Stifan ; țwalin udem-is am akken d udem n lmalayekkat.  </p> 
 <p><a name="an_7:1">7:1</a> Lmuqeddem ameqqran isteqsa Stifan yenna-yas : Ț-țideț wayen akka i d-nnan ? </p> 
 <p><a name="an_7:2">7:2</a> Stifan yerra-yas : Ay atmaten, ay imɣaṛen n wat Isṛail, ḥesset-iyi-d ! Sidi Ṛebbi di lɛaḍima-s tameqqrant, yesbeggen-ed iman-is i jeddi-tneɣ Sidna Ibṛahim, mi gella di tmurt n Mizubuṭamya uqbel ad yezdeɣ di temdint n Haran.  </p> 
 <p><a name="an_7:3">7:3</a> Yenna-yas : Ffeɣ si tmurt-ik, si tmurt n imawlan-ik, tṛuḥeḍ ɣer tmurt ara k-d-ssekneɣ.  </p> 
 <p><a name="an_7:4">7:4</a> Iffeɣ ihi si tmurt n iqaldiyen, iṛuḥ ad izdeɣ di temdint n Haran. Mi gemmut baba-s, Sidi Ṛebbi yewwi-t-id ɣer tmurt-agi i deg tzedɣem tura.  </p> 
 <p><a name="an_7:5">7:5</a> Sidi Ṛebbi ur s-yefki ara di tmurt-agi ula d azal n terdast i deg ara yesres aḍar-is ; meɛna iwɛed-as a s-yefk tamurt-agi i nețța d dderya n dderya-s ɣas akken imiren mazal ur yesɛi ara dderya, Sidi Ṛebbi yenna-yas :  </p> 
 <p><a name="an_7:6">7:6</a> Dderya n dderya-k aț-țezdeɣ deg yiwet n tmurt tabeṛṛanit, dinna a ten-erren d aklan, ad sɛeddin fell-asen lḥif azal n ṛebɛa meyya iseggasen.  </p> 
 <p><a name="an_7:7">7:7</a> Lameɛna Sidi Ṛebbi yenna-d : Ad uɣalen a d-ffɣen si tmurt-nni, ad iyi-ɛebden deg wemkan-agi ; ma d lǧens-nni i ten-yerran d aklan, d nekk ara ten-iḥasben.  </p> 
 <p><a name="an_7:8">7:8</a> Dɣa Sidi Ṛebbi yefka-d i Sidna Ibṛahim leɛqed yeɛnan ṭṭhaṛa. S wakka, mi d-yesɛa mmi-s Isḥaq, Sidna Ibṛahim iṭeḥheṛ-as deg wass wis tmanya n tlalit-is. Isḥaq daɣen yexdem akken i mmi-s yeɛqub ; ula d Yeɛqub iṭeḥheṛ i tnac-nni warraw-is yellan d lejdud-nneɣ.  </p> 
 <p><a name="an_7:9">7:9</a> Ihi arraw-agi n yeɛqub usmen ɣef gma-tsen Yusef, kkren zenzen-t ɣer tmurt n Maṣer anda yuɣal d akli.  </p> 
 <p><a name="an_7:10">7:10</a> Lameɛna Sidi Ṛebbi yella yid-es, ițsellik-it si tegnatin n ccedda ; issers-ed fell-as ṛṛeḥma yerna-yas-d leɛqel s wayes ara yeɛziz ɣer Ferɛun agellid n tmurt n Maṣeṛ. Ferɛun isbedd-it d lḥakem ɣef tmurt-is akk-d wexxam-is meṛṛa.  </p> 
 <p><a name="an_7:11">7:11</a> Yeɣli-d laẓ ɣef tmurt n Maṣeṛ akk ț-țmurt n Kenɛan, d lweqt n cedda tameqqrant ; lejdud-nneɣ ur ufin ara ayen ara ččen.  </p> 
 <p><a name="an_7:12">7:12</a> Mi gesla Yeɛqub belli llan yirden di tmurt n Maṣer, iceggeɛ ɣer dinna tikkelt tamezwarut, arraw-is yellan d lejdud-nneɣ.  </p> 
 <p><a name="an_7:13">7:13</a> Mi uɣalen tikkelt tis snat, Yusef isbeggen-ed iman-is ɣer watmaten-is, ɛeqlen-t d nețța. Yuɣal Ferɛun yeẓra laṣel n Yusef ansi-t.  </p> 
 <p><a name="an_7:14">7:14</a> Yusef iceggeɛ a d-awin baba-s Yeɛqub akk-d wat wexxam-is meṛṛa i gellan di xemsa usebɛin yid-sen.  </p> 
 <p><a name="an_7:15">7:15</a> Yeɛqub iṣubb ɣer tmurt n Maṣer ikemmel dinna ussan-is armi yemmut, am nețța am lejdud-nneɣ.  </p> 
 <p><a name="an_7:16">7:16</a> Wwin-d iɣsan-nsen ɣer taddart n Sicem, meḍlen-ten deg uẓekka i guɣ Sidna Ibṛahim s yedrimen ɣef warraw n Hamur n taddart n Sicem.  </p> 
 <p><a name="an_7:17">7:17</a> Mi d-yewweḍ lweqt i deg ara yedṛu wayen i gewɛed Sidi Ṛebbi i Sidna Ibṛahim s limin, agdud-nneɣ yufa-t-id lḥal innerna s waṭas di tmurt n Maṣer ;  </p> 
 <p><a name="an_7:18">7:18</a> imiren ikker-ed ugellid nniḍen ur nessin ara Yusef.  </p> 
 <p><a name="an_7:19">7:19</a> Agellid-nni iḥqeṛ lɣaci-nneɣ, yețqehhiṛ lejdud nneɣ, yeṣṣaweḍ armi i gefka lameṛ ad ǧǧen arraw-nsen i cceṛ akken kan ara d-lalen, iwakken ur țidiren ara.  </p> 
 <p><a name="an_7:20">7:20</a> Di lweqt-nni i d-ilul Sidna Musa ; icbeḥ yerna eɛziz ɣer Sidi Ṛebbi. Yețțuṛebba deg wexxam n imawlan-is tlata wagguren s tuffra ; </p> 
 <p><a name="an_7:21">7:21</a> uɣalen rran-t zdaxel n tqecwalt, serrḥen-as deg wasif ; d yelli-s n Ferɛun i t-id-ijemɛen, tṛebba-t-id am mmi-s.  </p> 
 <p><a name="an_7:22">7:22</a> Ikker-ed ger imaṣriyen, yelmed akk tamusni-nsen ifazen. Yuɣal d argaz amusnaw, izewṛen ama deg yimeslayen ama di lecɣal.  </p> 
 <p><a name="an_7:23">7:23</a> Mi geṣṣaweḍ ṛebɛin iseggasen di leɛmeṛ-is, yemmekta-d atmaten-is at Isṛail, iṛuḥ ad iẓer acu n lḥala i deg țɛicin. </p> 
 <p><a name="an_7:24">7:24</a> Yiwen wass iwala s wallen-is yiwen umaṣri ikkat yiwen seg wat Isṛail ; ikker a t-iḥudd, yerra-as-d țțar ; yewwet amaṣri-nni, yenɣa-t. </p> 
 <p><a name="an_7:25">7:25</a> Yenwa ad fehmen watmaten-is belli s ufus-is ara ten-isellek Sidi Ṛebbi ; lameɛna nutni ur fhimen ara.  </p> 
 <p><a name="an_7:26">7:26</a> Azekka-nni iwala sin yergazen n wat Isṛail țnaɣen wway gar-asen, iger iman-is, yeɛṛed a ten-ifru, yenna-yasen : Eɛni mačči d atmaten i tellam ? Acimi i tețțnaɣem wway gar-awen ?  </p> 
 <p><a name="an_7:27">7:27</a> Lameɛna win akken iḍelmen wayeḍ idemmer Musa yenna-yas : Anwa i k-yerran d aqeṛṛuy neɣ d lḥakem fell-aɣ ?  </p> 
 <p><a name="an_7:28">7:28</a> Eɛni tebɣiḍ a yi-tenɣeḍ akken tenɣiḍ iḍelli amaṣri-nni ?  </p> 
 <p><a name="an_7:29">7:29</a> Ɣef ddemma n yimeslayen-agi Sidna Musa yerwel, iṛuḥ ad izdeɣ di tmurt n Madyan. Dinna i gesɛa sin n warrac.  </p> 
 <p><a name="an_7:30">7:30</a> Mi ɛeddan ṛebɛin n iseggasen, idheṛ-as-ed yiwen n lmelk deg yiwen uḥeǧǧaǧu n tmes iceɛlen deg inijel, mi gella deg unezṛuf n wedrar n Sinay.  </p> 
 <p><a name="an_7:31">7:31</a> Sidna Musa yewhem deg wayen yețwali ; akken kan iqeṛṛeb ɣuṛ-es, Sidi Ṛebbi yessawel-as-d seg uḥeǧǧaǧu-nni n tmes, yenna-yas-d :  </p> 
 <p><a name="an_7:32">7:32</a> D nekk i d Illu n lejdud-ik, d Illu n Ibṛahim, n Isḥaq akk-d Yeɛqub. Sidna Musa yeqqim ițergigi, yuggad ad yerfed ula d allen-is.  </p> 
 <p><a name="an_7:33">7:33</a> Sidi Ṛebbi yenna-yas : Kkes arkasen seg iḍaṛṛen-ik axaṭer amkan-agi anda tbeddeḍ d amkan iqedsen.  </p> 
 <p><a name="an_7:34">7:34</a> Walaɣ lbaṭel ideṛṛun d wegdud-iw di tmurt n Maṣer ; sliɣ-ed i unazeɛ-nsen ; ṣubbeɣ-d iwakken a ten-sellkeɣ. Ṛuḥ tura a k-ceggɛeɣ ɣer tmurt n Maṣeṛ.  </p> 
 <p><a name="an_7:35">7:35</a> Musa-nni i nekṛen m'akken i s-nnan : Anwa i k-yerran d amdebbar neɣ d lḥakem ? D nețța i d-iceggeɛ Sidi Ṛebbi d aqeṛṛuy, d amcafeɛ n wegdud ; s lemɛawna n lmelk i s-d-idehṛen deg udarnu ireqqen.  </p> 
 <p><a name="an_7:36">7:36</a> D nețța i ten-id-issufɣen si tmurt n Maṣer. Yexdem aṭas n lbeṛhanat d leɛǧayeb di tmurt-nni, di lebḥeṛ azeggaɣ, deg unezṛuf n Sinay, anda akken qqimen ṛebɛin iseggasen.  </p> 
 <p><a name="an_7:37">7:37</a> D nețța daɣen i gennan i wat Isṛail : Sidi Ṛebbi a wen-d-yefk si ger watmaten-nwen yiwen n nnbi am nekk.  </p> 
 <p><a name="an_7:38">7:38</a> Asm'akken i gennejmaɛ wegdud deg unezṛuf, d nețța iwumi d yemmeslay lmelk-nni deg wedrar n Sinay, d nețța iwumi d-yefka Sidi Ṛebbi awal n tudert iwakken a ɣ-t-id issiweḍ.  </p> 
 <p><a name="an_7:39">7:39</a> Lameɛna lejdud-nneɣ ugin a s-aɣen awal ; ur t-qbilen ara axaṭer deg ulawen-nsen bɣan ad uɣalen ɣer tmurt n Maṣer.  </p> 
 <p><a name="an_7:40">7:40</a> Nnan i Haṛun gma-s n Sidna Musa : Xdem-aɣ-d iṛebbiten am wid n imaṣriyen, ara nezzwer zdat-nneɣ, axaṭer Musa-agi i ɣ-d-yessufɣen si tmurt n Maṣeṛ, ur neẓri ara d acu i gedṛan yid-es.  </p> 
 <p><a name="an_7:41">7:41</a> Dɣa deg ussan-nni, jemɛen-d ddheb, ssefsin-t xedmen yis yiwen ugenduz. Imiren zlan-as asfel, xedmen-as tameɣṛa, feṛḥen s wayen xedmen yifassen-nsen.  </p> 
 <p><a name="an_7:42">7:42</a> Sidi Ṛebbi yeṭṭaxeṛ fell-asen, yeǧǧa-ten ad ɛebden itran n yigenni , d ayen i guran di tektabt n lenbiya : Ay agdud n wat Isṛail, eɛni i nekk iwumi d-tefkam lewɛadi d iseflawen, di ṛebɛin iseggasen-nni i tesɛeddam deg unezṛuf ?  </p> 
 <p><a name="an_7:43">7:43</a> Xaṭi meɛna tbubbem aqiḍun n Mulux akk-d itri n Rufan i tḥesbem d iṛebbiten-nwen ;+ lmeṣnuɛat nni i txedmem iwakken a ten-tɛebdem ! Daymi ara kkun-nfuɣ ɣer lebɛid, akkin i temdint n Babilun !  </p> 
 <p><a name="an_7:44">7:44</a> Deg unezṛuf, lejdud-nneɣ sɛan aqiḍun i deg yella leɛqed n Sidi Ṛebbi ; aqiḍun-nni ițwaxedmen akken i s-t-id-yemla Sidi Ṛebbi i Sidna Musa mi s-d-yenna : Xdem-it am akken i t-twalaḍ di lemtel i k-d-ssekneɣ.  </p> 
 <p><a name="an_7:45">7:45</a> Mi d-yewweḍ uqiḍun-nni ɣer ifassen n lejdud-nneɣ , wwin-t-id yid-sen mi i ten-issekcem Yacuɛa ɣer tmurt-nni i sen-yewɛed Sidi Ṛebbi , mi ssufɣen imezdaɣ-is s tezmert n Sidi Ṛebbi. Aqiḍun-nni yeqqim dinna armi d zzman n Sidna Dawed. </p> 
 <p><a name="an_7:46">7:46</a> Sidna Dawed eɛzizen ɣer Sidi Ṛebbi, yedɛa ɣuṛ-es yenna : A Sidi Ṛebbi ! A Yillu n jeddi-tneɣ Yeɛqub ! Eǧǧ-iyi ad a k-bnuɣ lǧameɛ iqedsen ara yilin ț-țanezduɣt-ik !  </p> 
 <p><a name="an_7:47">7:47</a> Lameɛna d Sidna Sliman i s yebnan lǧameɛ-nni ! </p> 
 <p><a name="an_7:48">7:48</a> Ɣas akken Sidi Ṛebbi ɛlayen ur izeddeɣ ara deg ixxamen ițwabnan s ufus n wemdan, akken i d-yenna nnbi Iceɛya :  </p> 
 <p><a name="an_7:49">7:49</a> Igenni d amkan n lḥekma-w, ma d lqaɛa ț-țaḥsiṛt i ɣef srusuɣ idaṛṛen-iw. Sidi Ṛebbi yenna-d : Acu n wexxam i tzemrem a yi-tebnum, neɣ anda-t wemkan i deg zemreɣ ad steɛfuɣ ?  </p> 
 <p><a name="an_7:50">7:50</a> Eɛni mačči d afus-iw i d-ixelqen kullec ?  </p> 
 <p><a name="an_7:51">7:51</a> A wid isbeddayen iɛenqiqen nsen ! Tesɣaṛayem ulawen-nwen, tqeflem imeẓẓuɣen-nwen, tețțeddum di nneqma i Ṛṛuḥ iqedsen am kunwi am lejdud-nwen ! </p> 
 <p><a name="an_7:52">7:52</a> Anwa nnbi ur qhiṛen ara lejdud-nwen ? Nɣan wid akk ițbecciṛen si zik ɣef Uḥeqqi ara d-yasen ; tura d kunwi i t-ixedɛen yerna tenɣam-t. </p> 
 <p><a name="an_7:53">7:53</a> Tewweḍ-ikkun-id ccariɛa n Sidi Ṛebbi s ufus n lmalayekkat, lameɛna kunwi tugim a s-taɣem awal !  </p> 
 <p><a name="an_7:54">7:54</a> Mi slan i yemslayen-agi, lmuqedmin iffeɣ-iten leɛqel, qeṛcen tuɣmas-nsen fell-as. </p> 
 <p><a name="an_7:55">7:55</a> Lameɛna Stifan yeččuṛen d Ṛṛuḥ iqedsen, yerfed allen-is ɣer igenni, iwala tamanegt n Sidi Ṛebbi akk-d Sidna Ɛisa ibedden ɣer uyeffus-is. </p> 
 <p><a name="an_7:56">7:56</a> Imiren iɛeggeḍ, yenna : Atan țwaliɣ igenwan ldin, țwaliɣ Mmi-s n bunadem  ibedd ɣer tama tayeffust n Ṛebbi !  </p> 
 <p><a name="an_7:57">7:57</a> Qeflen imeẓẓuɣen-nsen, bdan țɛeggiḍen, ẓedmen fell-as akken ma llan. </p> 
 <p><a name="an_7:58">7:58</a> Zzuɣṛen-t beṛṛa n temdint, ṛejmen-t s iblaḍen iwakken a t-nɣen. Inigan ǧǧan leḥwayeǧ-nsen ɣer yiwen ilemẓi isem-is Caɛul, iwakken a ten-iɛas. </p> 
 <p><a name="an_7:59">7:59</a> M'akken i t-ṛeǧmen, Stifan ideɛɛu ɣer Sidi Ṛebbi yeqqaṛ : A Sidna Ɛisa, atan ṛṛuḥ-iw ger ifassen-ik !  </p> 
 <p><a name="an_7:60">7:60</a> Imiren yeɣli ɣef tgecrar, iɛeggeḍ s lǧehd-is : A baba Ṛebbi ur ten-țḥasab ara ɣef ddnub-agi ! Akken kan i d-yenna imeslayen agi, iffeɣ-it ṛṛuḥ.  </p> 
 <p><a name="an_8:1">8:1</a> Caɛul, d yiwen seg wid iqeblen lmut n Stifan. Ass-nni kkren-d wid ibdan țqehhiṛen tajmaɛt n wid yețțamnen s Lmasiḥ yellan di temdint n Lquds. Wid akk yumnen s Lmasiḥ rewlen ɣer tudrin n tmurt n Yahuda armi ț-țamurt n Samarya ; anagar ṛṛusul i d-yeqqimen di temdint n Lquds. </p> 
 <p><a name="an_8:2">8:2</a> Kra yergazen iḍuɛen Sidi Ṛebbi, wwin-d Stifan meḍlen-t, ḥeznen aṭas, țrun fell-as. </p> 
 <p><a name="an_8:3">8:3</a> Ma d Caɛul ixeddem akk lmejhud-is iwakken ad ihudd tajmaɛt n wid yumnen s Sidna Ɛisa ; issufuɣ ed seg ixxamen-nsen, irgazen ț-țilawin yumnen s Lmasiḥ, issekcam iten ɣer lḥebs.  </p> 
 <p><a name="an_8:4">8:4</a> Ihi wid akk i grewlen, mfaṛaqen, țṛuḥun seg wemkan ɣer wayeḍ, țbecciṛen lexbaṛ n lxiṛ n wawal n Sidi Ṛebbi. </p> 
 <p><a name="an_8:5">8:5</a> Filbas iṣubb ɣer temdint n Samarya, ibecceṛ dinna lexbaṛ n lxiṛ yeɛnan Lmasiḥ.  </p> 
 <p><a name="an_8:6">8:6</a> Lɣaci meṛṛa rran ddehn-nsen ɣer Filbas, mi slan i wayen i d-yeqqaṛ yerna walan lbeṛhanat i gxeddem ; </p> 
 <p><a name="an_8:7">8:7</a> axaṭer leǧnun țeffɣen s leɛyaḍ seg waṭas n yemdanen ițțumelken, aṭas daɣen n iquḍaṛen akk-d wid ikerfen i geḥlan.  </p> 
 <p><a name="an_8:8">8:8</a> Dɣa imezdaɣ n temdint-nni meṛṛa feṛḥen mačči d kra. </p> 
 <p><a name="an_8:9">8:9</a> Yuɣ lḥal di temdint-nni yella yiwen wergaz isem-is Semɛun, issewham imezdaɣ n tmurt n Samarya s ssḥur i gxeddem ; iḥețțeb iman-is d win yesɛan tazmert.  </p> 
 <p><a name="an_8:10">8:10</a> Imdanen akk am umeẓyan am umeqqran, țțaran ddehn-nsen ɣuṛ-es qqaṛen : « Argaz-agi ț-țazmert n Ṛebbi s yiman-is ; ț-țazmert-nni iwumi neqqaṛ tazmert yessewhamen ! »  </p> 
 <p><a name="an_8:11">8:11</a> Țțamnen yis axaṭer si zik i ten-issewham s ssḥur i gxeddem ; </p> 
 <p><a name="an_8:12">8:12</a> lameɛna, asmi umnen s wayen i sen-ibecceṛ Filbas ɣef wayen yeɛnan tagelda n Sidi Ṛebbi d yisem n Sidna Ɛisa Lmasiḥ, usan-d akk ad țwaɣeḍsen deg waman am yergazen am tlawin.  </p> 
 <p><a name="an_8:13">8:13</a> Semɛun s yiman-is yumen ; ula d nețța yețwaɣḍes deg waman, seg wass-nni ur yettixiṛ ara ɣef Filbas ; ițțawham deg-s mi gețwali leɛǧayeb d lbeṛhanat timeqqranin i gxeddem.  </p> 
 <p><a name="an_8:14">8:14</a> Mi slan belli lɣaci n tmurt n Samarya qeblen awal n Sidi Ṛebbi, ṛṛusul yeqqimen di temdint n Lquds ceggɛen-asen Buṭrus d Yuḥenna.  </p> 
 <p><a name="an_8:15">8:15</a> Mi wwḍen ɣuṛ-sen, dɛan fell asen ɣer Sidi Ṛebbi iwakken a sen-d-yefk Ṛṛuḥ iqedsen ; </p> 
 <p><a name="an_8:16">8:16</a> axaṭer mazal ur d-iris ula ɣef yiwen deg-sen, țwaɣeḍsen kan deg aman s yisem n Sidna Ɛisa.  </p> 
 <p><a name="an_8:17">8:17</a> Buṭrus akk-d Yuḥenna ssersen ifassen-nsen fell-asen, dɣa yețțunefk asen-d Ṛṛuḥ iqedsen. </p> 
 <p><a name="an_8:18">8:18</a> Mi gwala Semɛun yețțunefk-ed Ṛṛuḥ iqedsen i lɣaci imi ssersen ṛṛusul ifassen-nsen fell-asen, yewwi-yasen-d idrimen yenna-yasen :  </p> 
 <p><a name="an_8:19">8:19</a> Ffket-iyi ula d nekk tazmert agi iwakken kra n win i ɣef ara sserseɣ ifassen-iw, a t-ikcem Ṛṛuḥ iqedsen.  </p> 
 <p><a name="an_8:20">8:20</a> Lameɛna Buṭrus yenna-yas : Ad negren yedrimen-ik ad glun yis-ek imi tɣileḍ Ṛṛuḥ iqedsen n Sidi Ṛebbi yețnuzu s idrimen !  </p> 
 <p><a name="an_8:21">8:21</a> Ul-ik ur yeṣfi ara ɣer Sidi Ṛebbi ! Ur k-id-iṣaḥ umur, ur tesɛiḍ lweṛt di ccɣel-agi. </p> 
 <p><a name="an_8:22">8:22</a> Endem di yir ṛṛay-ik, tedɛuḍ ɣer Sidi Ṛebbi iwakken ma yella wamek, a k-yeɛfu ɣef yir axemmem-agi i d-yeffɣen seg ul-ik ! </p> 
 <p><a name="an_8:23">8:23</a> Atan walaɣ teččuṛeḍ d ddɣel, imlek-ik lbaṭel !  </p> 
 <p><a name="an_8:24">8:24</a> Semɛun yerra-yasen : Dɛut kunwi s yiman-nwen fell-i ɣer Sidi Ṛebbi iwakken ur d-ideṛṛu ara yid-i wayen akka i d-tennam.  </p> 
 <p><a name="an_8:25">8:25</a> Buṭrus d Yuḥenna kemmlen cehden ɣef Sidna Ɛisa, beccṛen awal n Sidi Ṛebbi, mbeɛd uɣalen ɣer temdint n Lquds. Deg webrid-nsen, țbecciṛen lexbaṛ n lxiṛ  deg waṭas n tudrin n tmurt n Samarya.  </p> 
 <p><a name="an_8:26">8:26</a> Yiwen lmelk n Sidi Ṛebbi inṭeq-ed ɣer Filbas yenna-yas : Kker, ṛuḥ ɣer lǧiha n usammer, ɛeddi seg webrid-nni yexlan i gețṣubbun si temdint n Lquds ɣer tmurt n Ɣaza.  </p> 
 <p><a name="an_8:27">8:27</a> Ikker, iṛuḥ. Immuger-ed deg webrid yiwen n wergaz, d aneɣlaf n Kendyas tagellit n tmurt n Ityubya ; d nețța i d lewkil ɣef lmelk-is meṛṛa , lameɛna d argaz ur nesɛi ddunit ; yusa-d ɣer temdint n Lquds iwakken ad iɛbed Sidi Ṛebbi. </p> 
 <p><a name="an_8:28">8:28</a> Mi gțeddu ɣef tkeṛṛust-is iwakken ad yuɣal ɣer wexxam-is, yufa-t-id lḥal yeqqaṛ di tektabt n nnbi Iceɛya. </p> 
 <p><a name="an_8:29">8:29</a> Ṛṛuḥ iqedsen yenna i Filbas : Ṛuḥ qeṛṛeb ɣer tkeṛṛust-ihina.  </p> 
 <p><a name="an_8:30">8:30</a> Filbas yuzzel ɣuṛ-es, yesla i uneɣlaf-nni n tmurt n Ityubya yeqqaṛ di tektabt n nnbi Iceɛya. Yenna-yas : Tfehmeḍ ayen akka i teqqaṛeḍ ?  </p> 
 <p><a name="an_8:31">8:31</a> Yerra-yas : Amek ara fehmeɣ ma ulac win ara yi-d-isfehmen ? Dɣa yesserkeb-ed Filbas ɣer tama-s.  </p> 
 <p><a name="an_8:32">8:32</a> Atah wayen akken yella yeqqaṛ di tektabt-nni : Am izimer m'ara t-awin a t-zlun, am tixsi iggugmen zdat win i ț-ițellsen, ur d-yeldi imi-is.  </p> 
 <p><a name="an_8:33">8:33</a> Yețwaḥqeṛ, ițwačč lḥeqq-is ! Anwa ara d-yeḥkun ɣef wayen yedṛan di lǧil-ines ? Anwa ara d-yeḥkun ɣef tudert-is yețwakksen si ddunit ?  </p> 
 <p><a name="an_8:34">8:34</a> Aneɣlaf-nni yenna-yas i Filbas : Di leɛnaya-k ssefhem-iyi-d ɣef wanwa i d-ițmeslay nnbi-yagi, ɣef yiman-is neɣ ɣef wayeḍ ?  </p> 
 <p><a name="an_8:35">8:35</a> Dɣa Filbas s wawal-agi i gebda a s-ițbecciṛ lexbaṛ n lxiṛ yeɛnan Sidna Ɛisa. </p> 
 <p><a name="an_8:36">8:36</a> Mi teddun deg webrid-nsen, wwḍen ɣer yiwen wemkan i deg llan waman. Aneɣlaf-nni yenna-yas : Atnan waman ! D acu ara yi-d iḥebsen iwakken ad țwaɣedṣeɣ ?  </p> 
 <p><a name="an_8:37">8:37</a> Filbas yenna-yas : Ma tumneḍ seg ul-ik, tzemreḍ aț-tețwaɣeḍseḍ. Aneɣlaf-nni yerra-yas : Umneɣ belli Sidna Ɛisa Lmasiḥ d Mmi-s n Ṛebbi.  </p> 
 <p><a name="an_8:38">8:38</a> Imiren yumeṛ ad ḥebsen takeṛṛust-is. Filbas d uneɣlaf-nni rsen-d, kecmen i sin ɣer waman, dɣa Filbas isseɣḍes-it. </p> 
 <p><a name="an_8:39">8:39</a> Mi d-ffɣen seg waman, Ṛṛuḥ n Sidi Ṛebbi yerfed Filbas, yexfa ger wallen uneɣlaf-nni. Aneɣlaf-nni ikemmel abrid-is s lfeṛḥ ;  </p> 
 <p><a name="an_8:40">8:40</a> ma d Filbas yufa-d iman-is di temdint n Acdud ; syenna ikemmel abrid-is, ițeddu ițbecciṛ awal n tudert di tudrin meṛṛa i ɣef iɛedda, armi i gewweḍ ɣer temdint n Qiṣarya.  </p> 
 <p><a name="an_9:1">9:1</a> Caɛul mazal ițqehhiṛ inelmaden n Sidna Ɛisa, ițgalla deg-sen a ten-ineɣ. Iṛuḥ ɣer lmuqeddem ameqqran, </p> 
 <p><a name="an_9:2">9:2</a> issuter-as a s-d-ixdem lkaɣeḍ ara yawi ɣer leǧwameɛ meṛṛa n temdint n Dimecq, iwakken ma yufa deg-sen wid ittabaɛen abrid n Ɛisa Lmasiḥ, ama d argaz ama ț-țameṭṭut, a ten-id-yawi țwarzen ɣer temdint n Lquds. </p> 
 <p><a name="an_9:3">9:3</a> Yella iteddu deg webrid, qṛib ad yaweḍ ɣer temdint n Dimecq. Taswiɛt kan, tekka-d yiwet n tafat seg igenni tfeǧǧeǧ-ed fell-as, tezzi yas-d. </p> 
 <p><a name="an_9:4">9:4</a> Yeɣli ɣer lqaɛa, yesla i yiwen n ṣṣut i s-d-yeqqaṛen : Caɛul ! A Caɛul ! Acuɣeṛ i yi tețqehhiṛeḍ ?  </p> 
 <p><a name="an_9:5">9:5</a> Yenna-yas : Anwa-k a Sidi ? Sṣut-nni yerra-yas-d : D nekk i d Ɛisa, win akken i tețqehhiṛeḍ ! </p> 
 <p><a name="an_9:6">9:6</a> Kker fell-ak ! Ṛuḥ ɣer temdint n Dimecq, dinna ara k-d-inin acu ara txedmeḍ.  </p> 
 <p><a name="an_9:7">9:7</a> Irgazen i geddan yid-es bedden, dehcen, ggugmen ; slan i taɣect-nni lameɛna ur walan ula d yiwen. </p> 
 <p><a name="an_9:8">9:8</a> Caɛul ikker-ed si lqaɛa, ɣas akken ldint wallen-is, ur yețwali ara. Sṭfen-as afus, wwin-t ɣer temdint n Dimecq. </p> 
 <p><a name="an_9:9">9:9</a> Yeqqim tlata wussan ur iwala, ur ičči, ur yeswi. Caɛul yumen s Ɛisa Lmasiḥ </p> 
 <p><a name="an_9:10">9:10</a> Di temdint n Dimecq, yella yiwen unelmad isem-is Ananyas ; Sidi Ṛebbi yessawel-as-d deg uweḥḥi, Ananyas yerra-yas : Aql-iyi a Sidi.  </p> 
 <p><a name="an_9:11">9:11</a> Sidi Ṛebbi yenna-yas : Kker tṛuḥeḍ ɣer webrid ițusemman « Abrid ayeffus », ɣer wexxam n Yuda ; dinna steqsi ɣef yiwen wergaz isem-is Caɛul n temdint n Sars. Atan ideɛɛu ɣer Sidi Ṛebbi, </p> 
 <p><a name="an_9:12">9:12</a> yețwali deg uweḥḥi ikcem ɣuṛ-es yiwen wergaz isem-is Ananyas, yessers ifassen-is fell-as iwakken a t-id-yuɣal yeẓri.  </p> 
 <p><a name="an_9:13">9:13</a> Ananyas yerra-yas : A Sidi, acḥal n yemdanen i d yeḥkan fell-as ițqehhiṛ inelmaden-ik imqedsen  di temdint n Lquds !  </p> 
 <p><a name="an_9:14">9:14</a> Yusa-d ɣer dagi daɣen s țțesriḥ n lmuqedmin imeqqranen iwakken ad yerr ɣer lḥebs wid akk ineddhen s yisem-ik.  </p> 
 <p><a name="an_9:15">9:15</a> Lameɛna Sidna Ɛisa yenna-yas : Ṛuḥ, axaṭer argaz-agi d nekk i t-yextaṛen iwakken ad ibecceṛ isem-iw i wat Isṛail, i leǧnas nniḍen akk-d igelliden-nsen. </p> 
 <p><a name="an_9:16">9:16</a> Yerna a s ssekneɣ acḥal ara yeɛteb ɣef ddemma n yisem-iw.  </p> 
 <p><a name="an_9:17">9:17</a> Ananyas iṛuḥ ; mi gewweḍ ɣer wexxam-nni, yessers ifassen-is ɣef Caɛul yenna-yas : A Caɛul a gma, d Sidna Ɛisa-nni i k-d-iḍehṛen deg-webrid ansi i d-tusiḍ, i yi-d-iceggɛen iwakken a k-id-yuɣal yeẓri yerna aț-țeččaṛeḍ d Ṛṛuḥ iqedsen.  </p> 
 <p><a name="an_9:18">9:18</a> Imiren kan ɣlin-d seg wallen-is am akken d iqecṛan, dɣa yuɣal-it-id yeẓri ; yekker, yețwaɣḍes. </p> 
 <p><a name="an_9:19">9:19</a> Mi gečča, yuɣal-it-id lǧehd ; isɛedda kra wussan ɣer inelmaden i gellan di temdint n Dimecq.  </p> 
 <p><a name="an_9:20">9:20</a> Ur iɛeṭṭel ara, yebda yețbecciṛ di leǧwameɛ n wat Isṛail belli Sidna Ɛisa d Mmi-s n Ṛebbi. </p> 
 <p><a name="an_9:21">9:21</a> Wid akk i s isellen, wehmen qqaṛen : Eɛni mačči d win akken i gețqehhiṛen di temdint n Lquds wid ineddhen s yisem-agi n Ɛisa ? Eɛni ur d-yusi ara ɣer dagi iwakken a ten-yawi țwarzen ɣer lmuqedmin imeqqranen ?  </p> 
 <p><a name="an_9:22">9:22</a> Lameɛna Caɛul simmal yețnerni di liman ; isseɛweq at Isṛail izedɣen di temdint n Dimecq mi sen-d-ițbeggin belli d Ɛisa i d Lmasiḥ. </p> 
 <p><a name="an_9:23">9:23</a> Mi Ɛeddan kra wussan, at Isṛail mcawaṛen akken a t-nɣen ; </p> 
 <p><a name="an_9:24">9:24</a> lameɛna Caɛul yesla s txazabit i bɣan a s xedmen, am yiḍ am ass țɛassan ula ț-țiwura n temdint iwakken a t-nɣen. </p> 
 <p><a name="an_9:25">9:25</a> Yiwen yiḍ, inelmaden gren-t zdaxel n uqecwal, ṣubben-t-id si ṣṣuṛ i d-izzin i temdint.  </p> 
 <p><a name="an_9:26">9:26</a> Mi i d-yusa ɣer temdint n Lquds, Caɛul inuda ɣef yinelmaden, lameɛna kukran-t akk axaṭer ur uminen ara belli yuɣal d anelmad n Sidna Ɛisa.  </p> 
 <p><a name="an_9:27">9:27</a> Dɣa Barnabas yewwi-t ɣer ṛṛusul, yeḥka-yasen amek i s-d-iban Sidna Ɛisa deg webrid, amek i s-d-ihḍeṛ d wamek i gbecceṛ s yisem n Sidna Ɛisa ɛinani mbla akukru di temdint n Dimecq.  </p> 
 <p><a name="an_9:28">9:28</a> Seg imiren yețṛuḥu yețțuɣal yid-sen di temdint n Lquds, yețbecciṛ ɛinani s yisem n Sidna Ɛisa. </p> 
 <p><a name="an_9:29">9:29</a> Iheddeṛ daɣen nețța d wat Isṛail i d-yusan si tmura i gețmeslayen tayunanit, yețțemjadal yid-sen, meɛna ula d nutni țqelliben a t-nɣen. </p> 
 <p><a name="an_9:30">9:30</a> Mi slan watmaten s wannect-agi, srewlen-t ɣer temdint n Qiṣarya iwakken ad iṛuḥ syenna ɣer temdint n Sars.  </p> 
 <p><a name="an_9:31">9:31</a> Tajmaɛt n watmaten yellan di tmura n Yahuda d Jlili akk-d Samarya tethenna, tețțimɣuṛ di liman, tețțeddu s tugdi n Sidi Ṛebbi, tețnerni s lemɛawna n Ṛṛuḥ iqedsen. </p> 
 <p><a name="an_9:32">9:32</a> Buṭrus i d-inudan tijmaɛin meṛṛa, iṛuḥ daɣen ɣer watmaten izedɣen di temdint n Lud. </p> 
 <p><a name="an_9:33">9:33</a> Yufa dinna yiwen wergaz isem-is Inyas ; argaz-agi tmanya iseggasen nețța deg usu axaṭer ikref.  </p> 
 <p><a name="an_9:34">9:34</a> Yenṭeq ɣuṛ-es Buṭrus yenna-yas : Ay Inyas ! Ɛisa Lmasiḥ yesseḥla- k ! Kker fell-ak tekkseḍ usu-inek ! Imiren kan yekker. </p> 
 <p><a name="an_9:35">9:35</a> Mi t-walan yeḥla, imezdaɣ akk n temdint n Lud d wid n uzaɣaṛ n tmurt n Sarun beddlen tikli, qeblen abrid n Sidna Ɛisa.  </p> 
 <p><a name="an_9:36">9:36</a> Di temdint n Jafa, tella yiwet n tmeṭṭut isem-is Tabita ; tețseddiq, tețɛawan igellilen.  </p> 
 <p><a name="an_9:37">9:37</a> Di lweqt-nni, yeɣli-d fell-as lehlak dɣa temmut. Ssarden-ț, ssersen-ț di tɣuṛfeț.  </p> 
 <p><a name="an_9:38">9:38</a> Mi slan inelmaden belli Buṭrus di temdint n Jafa i gella, yerna ț-țamdint ur nebɛid ara ɣef temdint n Lud, ceggɛen ɣuṛ-es sin yergazen iwakken a t-ḥellelen a d-yas ɣuṛ-sen s lemɣawla.  </p> 
 <p><a name="an_9:39">9:39</a> Buṭrus ikker yedda yid-sen imiren kan. Mi gewweḍ, ssulin-t ɣer tɣuṛfeț-nni. Tuǧǧal akk țrunt, zzint-as-d i Buṭrus, sseknent-as-d tiqendyaṛ d ibeṛnyas i tzeṭṭ Durka uqbel aț-țemmet.  </p> 
 <p><a name="an_9:40">9:40</a> Buṭrus issufeɣ ɣer beṛṛa wid akk yellan dinna, yeqqim ɣef tgecrar, yedɛa ɣer Sidi Ṛebbi ; izzi ɣer lmegget-nni yenna-yas : Tabita, ekker ! Dɣa teldi-d allen-is, twala-d Buṭrus, imiren teqqim-ed.  </p> 
 <p><a name="an_9:41">9:41</a> Buṭrus yefka-yas afus-is iskker-iț-id, dɣa issawel i inelmaden nni akk-d tuǧǧal, ufan-ț-id tedder.  </p> 
 <p><a name="an_9:42">9:42</a> Imezdaɣ meṛṛa n temdint n Jafa slan s wayen yedṛan, daymi aṭas i gumnen s Sidna Ɛisa. </p> 
 <p><a name="an_9:43">9:43</a> Buṭrus yeqqim kra n wussan di temdint n Jafa, ɣer yiwen wergaz isem-is Semɛun, d axeddam deg uɣerruz ( aglim ).  </p> 
 <p><a name="an_10:1">10:1</a> Di temdint n Qiṣarya, yella yiwen wergaz iwumi qqaṛen Kurnilyus, d lqebṭan n yiwet n terbaɛt isem-is « tarbaɛt n Selyan ». </p> 
 <p><a name="an_10:2">10:2</a> Am nețța am wexxam-is ḍuɛen Sidi Ṛebbi ; ițseddiq i igellilen, yerna am iḍ am ass yezga ideɛɛu ɣer Sidi Ṛebbi.  </p> 
 <p><a name="an_10:3">10:3</a> Yiwen wass, leǧwahi n tlata n tmeddit, iweḥḥa-yas-ed Sidi Ṛebbi, iwala yiwen lmelk ikcem-ed ɣuṛ-es yenna-yas-d : A Kurnilyus !  </p> 
 <p><a name="an_10:4">10:4</a> Kurnilyus yerfed allen-is ɣuṛ-es, tekcem-it tugdi, ițergigi, yerra-yas : D acu, a Sidi ? Lmelk yenna-yas-d : Sidi Ṛebbi yeqbel taẓallit-ik akk-d lewɛadi i tețțakeḍ i imeɣban ; </p> 
 <p><a name="an_10:5">10:5</a> Tura ceggeɛ irgazen ɣer temdint n Jafa, ɣer Semɛun, win iwumi qqaṛen Butṛus, iwakken a d-yas. </p> 
 <p><a name="an_10:6">10:6</a> A t-tafeḍ ɣer yiwen wergaz i gxeddmen deg uɣerruz, ula d nețța isem-is Semɛun ; axxam-is yezga-d ɣef rrif n lebḥeṛ. </p> 
 <p><a name="an_10:7">10:7</a> Mi gṛuḥ lmelk-nni i s-d-ihedṛen, Kurnilyus yessawel i sin seg iqeddacen-is akk-d yiwen uɛeskṛi n terbɛat-is i gellan d argaz iḍuɛen Sidi Ṛebbi. </p> 
 <p><a name="an_10:8">10:8</a> iḥka-yasen-d ayen akk yedṛan; dɣa iceggeɛ-iten ɣer temdint n Jafa.  </p> 
 <p><a name="an_10:9">10:9</a> Azekka-nni, mi qṛib ad awḍen wid i gceggeɛ Kurnilyus ɣer temdint n Jafa, Buṭrus yuɣ-it lḥal yuli ɣef ssḍeḥ iwakken ad idɛu ɣer Sidi Ṛebbi, aț-țili d leǧwahi n tnac n wass, </p> 
 <p><a name="an_10:10">10:10</a> yeṭṭef-it laẓ yebɣa ad yečč. Llan țheggin-as-ed lqut, taswiɛt kan iweḥḥa-yas-ed Sidi Ṛebbi ; </p> 
 <p><a name="an_10:11">10:11</a> iwala igenni yeldi, yiwet lḥaǧa icuban ɣer tẓeṛbit tameqqrant icudden si ṛebɛa tɣemmaṛ, tețṣubbu-d ɣer lqaɛa. </p> 
 <p><a name="an_10:12">10:12</a> Llan deg-s lewḥuc at ṛebɛa idaṛṛen n mkul ṣṣifa, wid iteddun ɣef wuɛebbuḍ, akk-d yefṛax n igenni s mkul ṣṣifa. </p> 
 <p><a name="an_10:13">10:13</a> Yesla i yiwen n ṣṣut yenna yas-d : A Buṭrus, ekker ! Ezlu teččeḍ !  </p> 
 <p><a name="an_10:14">10:14</a> Lameɛna Butṛus yenna-yas : Ala a Sidi, di leɛmeṛ-iw ur ččiɣ ayen iḥeṛmen neɣ ayen ur neṣfi.  </p> 
 <p><a name="an_10:15">10:15</a> Sṣut-nni inṭeq-ed tikkelt tis snat yenna-yas-d : Ayen iḥseb Sidi Ṛebbi d leḥlal, ur k-id-iṣaḥ ara a t-tḥesbeḍ d leḥṛam !  </p> 
 <p><a name="an_10:16">10:16</a> Ayagi, iweḥḥa-yas-t-id Sidi Ṛebbi tlata n tikkal ; imiren kan, lḥaǧa-nni tețwarfed ɣer igenni.  </p> 
 <p><a name="an_10:17">10:17</a> Akken mazal Buṭrus yețxemmim ɣef lmeɛna n uweḥḥi-nni, wwḍen-d yergazen-nni i d-iceggeɛ Kurnilyus steqsayen ɣef wexxam n Semɛun. </p> 
 <p><a name="an_10:18">10:18</a> Mi d-wwḍen ɣer tewwurt, ssawlen, steqsan ma deg wexxam-nni i gețțili Semɛun iwumi qqaṛen Butṛus.  </p> 
 <p><a name="an_10:19">10:19</a> Mazal Buṭrus ițxemmim ɣef wuweḥḥi-nni, mi s-d-yenna Ṛṛuḥ iqedsen : Atnaya tlata yergazen țnadin fell-ak. </p> 
 <p><a name="an_10:20">10:20</a> Ur țkukru ara, kker, ddu yid-sen axaṭer d nekk i ten-id iceggɛen.  </p> 
 <p><a name="an_10:21">10:21</a> Butṛus iṣubb-ed ɣuṛ-sen, yenna yasen : Aql-i ! D nekk i d Buṭrus ! D acu akka i kkun-id-yewwin ɣuṛ-i ?  </p> 
 <p><a name="an_10:22">10:22</a> Nutni rran-as : D lqebṭan-nneɣ Kurnilyus, i ɣ-d iceggɛen ɣuṛ-ek ! D argaz țcekkiṛen akk wat Isṛail yerna ḥemmlen-t axaṭer d amdan aḥeqqi iḍuɛen Sidi Ṛebbi. Ibedd-ed ɣuṛ-es lmelk n Sidi Ṛebbi yenna-yas-d a d-iceggeɛ ɣuṛ-ek, a d-taseḍ ɣer wexxam-is iwakken ad isel i wayen i tesɛiḍ a s-t-id-tiniḍ.  </p> 
 <p><a name="an_10:23">10:23</a> Buṭrus issekcem-iten ad nsen ɣuṛ-es. Azekka-nni ikker yedda yid-sen nețța d kra n watmaten n Jafa.  </p> 
 <p><a name="an_10:24">10:24</a> Sellazekka-nni, wwḍen ɣer temdint n Qiṣarya. Kurnilyus yețṛaǧu ten deg wexxam-is nețța d leḥbab-is akk-d imawlan-is i d-yeɛṛeḍ.  </p> 
 <p><a name="an_10:25">10:25</a> Akken iteddu Butṛus ad yekcem, immuger-it-id Kurnilyus ɣer tewwurt n wefrag, yeɣli ɣer idaṛṛen-is, yekna zdat-es iwakken ad isṭerḥeb yis s leqdeṛ ameqqran. </p> 
 <p><a name="an_10:26">10:26</a> Meɛna Butṛus issekker-it-id yenna-yas : Kker fell-ak ! D amdan am kečč i lliɣ.  </p> 
 <p><a name="an_10:27">10:27</a> Leḥḥun, țmeslayen armi kecmen ɣer wexxam, yufa dinna aṭas n yemdanen i gennejmaɛen. </p> 
 <p><a name="an_10:28">10:28</a> Buṭrus yenna-yasen : Yak teẓram d leḥṛam i wemdan n wat Isṛail ad ixaleḍ abeṛṛani neɣ ad ikcem ɣer wexxam-is. Lameɛna Sidi Ṛebbi yessefhem-iyi-d belli ur ilaq ara a d-nini ɣef walebɛaḍ belli ur zeddig ara neɣ ur iṣfi ara ; </p> 
 <p><a name="an_10:29">10:29</a> daymi i qebleɣ a d-aseɣ mi yi-d-tessawlem. Init-iyi-d tura sebba ɣef wacu i d-tceggɛem ɣuṛ-i.  </p> 
 <p><a name="an_10:30">10:30</a> Kurnilyus yerra-yas : Ṛebɛa wussan aya, leǧwahi n tlata n tmeddit, lliɣ deɛɛuɣ ɣer Sidi Ṛebbi deg wexxam-iw ; ataya ibedd-ed zdat-i yiwen wergaz s llebsa yețfeǧǧiǧen,  </p> 
 <p><a name="an_10:31">10:31</a> yenna-yi-d : « A Kurnilyus ! Sidi Ṛebbi yeqbel taẓallit-ik akk-d lewɛadi i tețțakeḍ i igellilen. </p> 
 <p><a name="an_10:32">10:32</a> Ceggeɛ ɣer temdint n Jafa ɣerSemɛun ițusemman Butṛus iwakken a d-yas ; a t-tafeḍ ɣer yiwen i gxeddmen igelman, ula d nețța isem-is Semɛun ; axxam-is yezga-d ɣef rrif n lebḥeṛ.  </p> 
 <p><a name="an_10:33">10:33</a> Imiren ceggɛeɣ-en ɣuṛ-ek ; tanemmirt-ik imi d-tusiḍ. Tura ihi aql-aɣ akk dagi zdat Sidi Ṛebbi iwakken a nsel s ɣuṛ-ek, ayen akk i k-d-yumeṛ Sidi Ṛebbi a ɣ-t-id-tiniḍ.  </p> 
 <p><a name="an_10:34">10:34</a> Butṛus yebda ameslay yenna : Armi ț-țura i fehmeɣ belli ț-țideț, Sidi Ṛebbi ur ixeddem ara lxilaf ger yemdanen.  </p> 
 <p><a name="an_10:35">10:35</a>  Lameɛna di mkul lǧens, yal amdan i t-yețḍuɛun, ixeddmen ayen yellan d lḥeqq, iɛǧeb-as i Ṛebbi i gellan d Illu n yemdanen meṛṛa ; teslam belli iceggeɛ-ed awal-is i wat Isṛail, ibecceṛ-asen-d lehna s Ɛisa Lmasiḥ. </p> 
 <p><a name="an_10:37">10:37</a> Ahat teslam s wayen yedṛan di tmurt n Yahuda meṛṛa ; yebda-d si tmurt n Jlili seg wasmi ițbecciṛ Yeḥya aɣeṭṭas, aɣḍas n waman.  </p> 
 <p><a name="an_10:38">10:38</a> Teslam s Ɛisa Anaṣari amek i t-iččuṛ Sidi Ṛebbi s tezmert akk-d Ṛṛuḥ iqedsen, amek i gteddu seg umkan ɣer wayeḍ, ixeddem lxiṛ, amek isseḥlay wid akk i gemlek Cciṭan, axaṭer Sidi Ṛebbi ițțili yid-es.  </p> 
 <p><a name="an_10:39">10:39</a> Aql-aɣ d inigan n wayen akk i gexdem di tmurt n wat Isṛail ț-țemdint n Lquds anda i t-nɣan mi t-semmṛen ɣef wumidag,  </p> 
 <p><a name="an_10:40">10:40</a> meɛna ass wis tlata Sidi Ṛebbi isseḥya-t-id si ger lmegtin ; yefka-yas tazmert s wayes i d-isbeggen iman-is, </p> 
 <p><a name="an_10:41">10:41</a> mačči i yemdanen meṛṛa meɛna i nukkni kan i gextaṛ Sidi Ṛebbi a nili d inagan-is ; nukkni yeččan, yeswan yid-es mi d-iḥya si ger lmegtin. </p> 
 <p><a name="an_10:42">10:42</a> Sidna Ɛisa iweṣṣa-yaɣ a nbecceṛ i yemdanen meṛṛa, belli d nețța i gextaṛ Sidi Ṛebbi iwakken ad iḥaseb wid yeddren akk-d wid yemmuten. </p> 
 <p><a name="an_10:43">10:43</a> Lenbiya  akk cehden-d fell-as belli kra n win ara yamnen yis, a s-țwaɛeffun ddnubat-is s yisem-is.  </p> 
 <p><a name="an_10:44">10:44</a> Urɛad ifuk Buṭrus ameslay mi d-yers Ṛṛuḥ iqedsen ɣef wid akk yellan dinna, i gesmeḥsisen i wawal n Sidi Ṛebbi.  </p> 
 <p><a name="an_10:45">10:45</a> Wid i geddan d Butṛus, yellan d at Isṛail yumnen s Lmasiḥ, dehcen, wehmen imi d-yețțunefk Ṛṛuḥ iqedsen ula i wid ur nelli ara n wat Isṛail.  </p> 
 <p><a name="an_10:46">10:46</a> Axaṭer sellen-asen heddṛen yerna țḥemmiden Sidi Ṛebbi s tutlayin ur nețwassen ara. Dɣa Butṛus yenna :  </p> 
 <p><a name="an_10:47">10:47</a> Eɛni nezmer a nḥeṛṛem seg weɣḍas deg waman wid iwumi i d-yețțunefk Ṛṛuḥ iqedsen am nukkni ?  </p> 
 <p><a name="an_10:48">10:48</a> Dɣa Yefka lameṛ ad țwaɣeḍsen s yisem n Sidna Ɛisa Lmasiḥ ; imiren ḥellelen-t ad yeqqim yid-sen kra n wussan.  </p> 
 <p><a name="an_11:1">11:1</a> Ṛṛusul d watmaten yellan di tmurt n Yahuda slan belli ula d leǧnas  ur nelli ara n wat Isṛail qeblen awal n Ṛebbi.  </p> 
 <p><a name="an_11:2">11:2</a> Mi guɣal Butṛus ɣer temdint n Lquds, at Isṛail yumnen s Ɛisa Lmasiḥ ur ten-iɛǧib ara lḥal ; lummen fell-as, nnan-as :  </p> 
 <p><a name="an_11:3">11:3</a> Amek armi tkecmeḍ ɣer wexxam n wid ur neḍhiṛ ara yerna teččiḍ yid-sen ?  </p> 
 <p><a name="an_11:4">11:4</a> Dɣa Butṛus yebda a sen-d-iḥekku ayen akk yedṛan, yenna-yasen :   </p> 
 <p><a name="an_11:5">11:5</a> Mi lliɣ di temdint n Jafa, lliɣ deɛɛuɣ ɣer Sidi Ṛebbi ; taswiɛt kan, iṛuḥ leɛqel-iw, iweḥḥa-yi-d Sidi Ṛebbi , walaɣ taẓeṛbit tameqqrant tcudd si ṛebɛa n tɣemmaṛ, tețṣubbu-d seg igenni, tewweḍ-ed armi d ɣuṛ-i.  </p> 
 <p><a name="an_11:6">11:6</a> Reṣṣaɣ allen-iw fell-as, walaɣ deg-s mkul ṣṣifa n lewḥuc at ṛebɛa idaṛṛen d wid iteddun ɣef wuɛebbuḍ akk-d yefṛax n igenni n mkul ṣṣenf.  </p> 
 <p><a name="an_11:7">11:7</a> Dɣa sliɣ i yiwen n ṣṣut i yi-d iqqaṛen : a Buṭrus ! Ekker, ezlu teččeḍ ! </p> 
 <p><a name="an_11:8">11:8</a> Lameɛna nniɣ-as : ala a Sidi ! Leɛmeṛ yekcim ɣer yimi-w wayen iḥeṛmen neɣ wayen ur neṣfi.  </p> 
 <p><a name="an_11:9">11:9</a> Tikkelt tis snat, ṣṣut-nni i d-yusan seg igenni yenna-yi-d : ayen iḥseb Sidi Ṛebbi d leḥlal, ur k-id iṣaḥ ara a t-tḥesbeḍ d leḥṛam !  </p> 
 <p><a name="an_11:10">11:10</a> Ayagi yedṛa-d tlata n tikkal, dɣa kullec ițwarfed ɣer igenni. </p> 
 <p><a name="an_11:11">11:11</a> Imiren kan wwḍen-d tlata yergazen ɣer wexxam i deg lliɣ, țwaceggɛen-d ɣuṛ-i si temdint n Qiṣarya. </p> 
 <p><a name="an_11:12">11:12</a> Ṛṛuḥ iqedsen yenna-yi-d : ddu kan yid-sen ur țțaggad ara. Wwiɣ yid-i sețța watmaten-agi i tețwalim, dɣa nṛuḥ.  </p> 
 <p><a name="an_11:13">11:13</a> Nekcem ɣer wexxam n Kurnilyus, yeḥka-yaɣ-d ayen i s yenna lmelk n Sidi Ṛebbi i d ibedden ɣuṛ-es, mi s-yenna : ceggeɛ ɣer temdint n Jafa a d-awin Semɛun ițusemman Butṛus, </p> 
 <p><a name="an_11:14">11:14</a> a k-d-yini amek ara tețwasellkeḍ kečč d wat wexxam-ik.  </p> 
 <p><a name="an_11:15">11:15</a> Mi bdiɣ ameslay, Ṛṛuḥ iqedsen yers-ed fell-asen, akken i d-yers fell-aɣ ass amezwaru.  </p> 
 <p><a name="an_11:16">11:16</a> Imiren mmektaɣ-ed awal-nni i d-yenna Sidna Ɛisa : « Yeḥya isseɣḍas deg aman, lameɛna kunwi aț-țețwaɣeḍsem di Ṛṛuḥ iqedsen. »  </p> 
 <p><a name="an_11:17">11:17</a> Ihi imi sen-d yefka Sidi Ṛebbi tikci am nukkni yumnen s Sidna Ɛisa Lmasiḥ, d acu-yi nekk iwakken ad xalfeɣ lebɣi n Sidi Ṛebbi ? </p> 
 <p><a name="an_11:18">11:18</a> Mi slan i wannect-nni, yethedden wul-nsen, ḥemden Ṛebbi nnan : A neḥmed Sidi Ṛebbi i geldin abrid n țțuba ula i wat leǧnas ur nelli ara n wat Isṛail, iwakken ad sɛun tudert !  </p> 
 <p><a name="an_11:19">11:19</a> Asmi gemmut Stifan, yekker uqehheṛ i gessemfaṛaqen inelmaden ɣer mkul amkan ; ṛuḥen armi ț-țamurt n Finisya, armi ț-țigzirt n Qubṛus ț-țemdint n Antyuc, țbecciṛen awal n Ṛebbi i wat Isṛail kan ; </p> 
 <p><a name="an_11:20">11:20</a> lameɛna llan gar-asen kra n yergazen n tegzirt n Qubṛus akk-d kra nniḍen n temdint n Qiṛwan, usan-d ɣer temdint n Antyuc, țmeslayen i iyunaniyen, țbecciṛen-asen lexbaṛ n lxiṛ  yeɛnan Sidna Ɛisa. </p> 
 <p><a name="an_11:21">11:21</a> Tazmert n Sidi Ṛebbi tețțili yid-sen ; aṭas n yemdanen i gumnen beddlen tikli, tebɛen abrid n Sidna Ɛisa. </p> 
 <p><a name="an_11:22">11:22</a> Mi gewweḍ lexbaṛ-agi ɣer tejmaɛt n watmaten n temdint n Lquds, ceggɛen Barnabas armi d Antyuc.  </p> 
 <p><a name="an_11:23">11:23</a> Mi gewweḍ, iwala ayen i texdem dinna ṛṛeḥma n Sidi Ṛebbi dɣa yeččuṛ d lfeṛḥ. Yenha-ten akk iwakken ad ṭṭfen seg wul di liman sɛan di Lmasiḥ. </p> 
 <p><a name="an_11:24">11:24</a> Axaṭer Barnabas d argaz ixeddmen lxiṛ, iččuṛ d Ṛṛuḥ iqedsen akk-d liman. S wakka, aṭas n yemdanen i d-yernan ɣer Sidna Ɛisa.  </p> 
 <p><a name="an_11:25">11:25</a> Barnabas iṛuḥ ɣer temdint n Sars iwakken ad inadi ɣef Caɛul. </p> 
 <p><a name="an_11:26">11:26</a> Mi t-yufa, yewwi-t-id yid-es ɣer temdint n Antyuc. Di tejmaɛt-nni, sɛeddan aseggas kamel, xeddmen s tdukli, sselmaden aṭas n yemdanen. Di temdint n Antyuc ițțusemman inelmaden n Sidna Ɛisa tikkelt tamezwarut : « imasiḥiyen »  </p> 
 <p><a name="an_11:27">11:27</a> Deg ussan-nni, ṣubben-d kra n lenbiya si temdint n Lquds ɣer temdint n Antyuc. </p> 
 <p><a name="an_11:28">11:28</a> Yiwen seg-sen isem-is Agabus, ikker icar-ed s Ṛṛuḥ iqedsen belli a d-yeɣli laẓ d ameqqran di ddunit meṛṛa ; ț-țideț, laẓ-agi yeɣli-d di zzman n ugellid ameqqran : Kludyus </p> 
 <p><a name="an_11:29">11:29</a> Inelmaden n temdint n Antyuc, msefhamen belli mkul yiwen deg-sen a d-yefk ayen iwumi yezmer, iwakken ad ɛiwnen atmaten izedɣen di tmurt n Yahuda. </p> 
 <p><a name="an_11:30">11:30</a> D ayen i xedmen ; ceggɛen Barnabas d Caɛul ad awin i imeqqranen n tejmaɛin n tmurt n Yahuda, ayen akk i d-jemɛen.  </p> 
 <p><a name="an_12:1">12:1</a> Di lweqt-nni, agellid Hiṛudus yebda ițqehhiṛ inelmaden n tejmaɛt yellan di temdint n Lquds. </p> 
 <p><a name="an_12:2">12:2</a> Yefka lameṛ, ad nɣen s usekkin Yeɛqub gma-s n Yuḥenna. </p> 
 <p><a name="an_12:3">12:3</a> Mi gwala d ayen isfeṛḥen at Isṛail, yerna daɣen isseḥbes Butṛus, deg wussan n lɛid n leslak.  </p> 
 <p><a name="an_12:4">12:4</a> Mi t-id-ṭṭfen, rran-t ɣer lḥebs. Isbedd-as taɛessast n ṛebɛa trebbaɛ n lɛeskeṛ, mkul tarbaɛt deg-s ṛebɛa iɛeskṛiwen ; yebɣa a t-icaṛeɛ zdat lɣaci meṛṛa, m'ara tɛeddi Tfaska n yizimer.  </p> 
 <p><a name="an_12:5">12:5</a> Butṛus ihi yella di lḥebs ; ma ț-țajmaɛt n watmaten tezga di tẓallit, tdeɛɛu fell-as ɣer Sidi Ṛebbi.  </p> 
 <p><a name="an_12:6">12:6</a> Iḍ-nni uqbel a t-icaṛeɛ Hiṛudus, Butṛus ițwarzen s snat snasel, yella iṭṭes ger sin iɛeskṛiwen , wiyaḍ ɛussen tawwurt n lḥebs.  </p> 
 <p><a name="an_12:7">12:7</a> Taswiɛt kan, ataya lmelk n Sidi Ṛebbi iḍheṛ-as-d ; yiwet n tafat tameqqrant tceɛceɛ-ed di lḥebs-nni. Lmelk-nni yennul idis n Buṭrus, issendekwal-it-id yenna-yas : Ɣiwel Kker ! Imiren kan, qqeṛsent snasel-nni ɣlint-ed seg ifassen-is. </p> 
 <p><a name="an_12:8">12:8</a> Lmelk-nni yenna-yas : Els-ed arkasen-ik tbeggseḍ ɣef yiman-ik ! Buṭrus ixdem akken i s-d-yenna, lmelk yenna-yas-d daɣen : Sburr-ed abeṛnus-ik tettebɛeḍ-iyi-d ! </p> 
 <p><a name="an_12:9">12:9</a> Butṛus ikker itbeɛ-it ; ur ifaq ara belli lmelk nni ț-țideț iḍheṛ-as-ed, inwa d aweḥḥi n Sidi Ṛebbi. </p> 
 <p><a name="an_12:10">12:10</a> Ɛeddan zdat tɛessast tamezwarut, rnan tis snat, wwḍen ɣer tewwurt n wuzzal yessufuɣen ɣer temdint, teldi weḥd-es ; ffɣen ɛeddan deg yiwet n tezniqt, imiren lmelk yexfa ger wallen n Buṭrus.  </p> 
 <p><a name="an_12:11">12:11</a> Mi d-yufa iman-is di beṛṛa, ifaq belli ț-țideț, Sidi Ṛebbi iceggeɛ-d lmelk-is, isellek-it-id seg-ufus n Hiṛudus akk-d lbaṭel i bɣan a s-t xedmen wat Isṛail.  </p> 
 <p><a name="an_12:12">12:12</a> Mi gefhem lḥala i deg yella, ikker iṛuḥ ɣer wexxam n Meryem yemma-s n Yuḥenna iwumi qqaṛen daɣen Maṛqus, nnejmaɛen dinna waṭas n watmaten, deɛɛun ɣer Sidi Ṛebbi.  </p> 
 <p><a name="an_12:13">12:13</a> Mi gesqeṛbeb di tewwurt n beṛṛa, tusa-d yiwet n tqeddact isem-is Rudya, tqeṛṛeb ɣer tewwurt aț-țesmeḥses ; </p> 
 <p><a name="an_12:14">12:14</a> mi teɛqel taɣect n Buṭrus, di lfeṛḥ-nni i deg tella ur teldi ara tawwurt, tuzzel aț-țissiweḍ lexbaṛ belli Butṛus atan ibedd zdat tewwurt. </p> 
 <p><a name="an_12:15">12:15</a> Nnan-as : Waqila iffeɣ-ikem leɛqel ! Lameɛna teṭṭef deg wayen i d tenna. </p> 
 <p><a name="an_12:16">12:16</a> Nnan-as : Ahat d lexyal-is ihi ! Llan țemsteqsayen, ma d Butṛus mazal-it isqeṛbub-ed ; mi d-ldin tawwurt wehmen mi t-walan. </p> 
 <p><a name="an_12:17">12:17</a> Buṭrus iwehha-yasen s ufus-is ad ssusmen, yeḥka-yasen amek i t-id issufeɣ Sidi Ṛebbi si lḥebs, yenna yasen : Sṣiwḍet lexbaṛ-agi i Yeɛqub d watmaten. Dɣa iṛuḥ ibeddel amkan. </p> 
 <p><a name="an_12:18">12:18</a> Mi guli wass, lɛeskeṛ iɛussen Butṛus tqelqen, țmesteqsayen amek i geffeɣ si lḥebs, anwa i t-yessufɣen ? </p> 
 <p><a name="an_12:19">12:19</a> Hiṛudus iceggeɛ a t-id-qellben, meɛna ur t-ufin ara. Yebḥet iɛessasen-nni, dɣa yefka lameṛ a ten-nɣen ; imiren iṛuḥ si tmurt n Yahuda ɣer tmurt n Qiṣarya, anda yesɛedda kra n wussan.  </p> 
 <p><a name="an_12:20">12:20</a> Hiṛudus yerfa ɣef yimezdaɣ n temdinin n Sur akk d Sidun, lameɛna nutni mcawaṛen, msefhamen akk-d Blastus yellan d lewkil-is ; dɣa ceggɛen wid ara s-issutren lehna, axaṭer tamurt-nsen tețɛic seg wayen i d-itekken si tmurt n ugellid Hiṛudus.  </p> 
 <p><a name="an_12:21">12:21</a> Yewweḍ-ed wass i deg ara d-yini Hiṛudus lxeṭba i lɣaci. Mi d-yewweḍ wass n temlilit, Hiṛudus yelsa llebsa n ugellid, yuli ɣer wemkan n lxeṭba ad immeslay i lɣaci.  </p> 
 <p><a name="an_12:22">12:22</a> Mi gebda ameslay, lɣaci meṛṛa qqaṛen : D Ṛebbi i d-ițmeslayen, mačči d amdan !  </p> 
 <p><a name="an_12:23">12:23</a> Imiren kan lmelk n Sidi Ṛebbi yesseɣli-d fell-as aṭan, axaṭer yeqbel a t-ḥesben yemdanen am Ṛebbi, iceggeɛ-as-d abeɛɛuc i t-yeččan dɣa yemmut ; </p> 
 <p><a name="an_12:24">12:24</a> ma d awal n Sidi Ṛebbi țbecciṛen-t, yewweḍ ɣer mkul amkan.  </p> 
 <p><a name="an_12:25">12:25</a> Barnabas akk-d Caɛul mi fukken lxedma-nsen di temdint n Lquds uɣalen, wwin yid-sen Yuḥenna ițusemman Maṛqus.  </p> 
 <p><a name="an_13:1">13:1</a> Di tejmaɛt n temdint n Antyuc llan lenbiya d lɛulama : Barnabas, Semɛun ițusemman Aberkan, Lusyus n temdint n Qiṛwan, Manahen i d-ițțuṛebban d Hiṛudus Antibas, akk-d Caɛul.  </p> 
 <p><a name="an_13:2">13:2</a> Yiwen wass atmaten nnejmaɛen iwakken ad uẓummen yerna ad dɛun ɣer Sidi Ṛebbi. Ṛṛuḥ iqedsen yenna-yasen-d : Swejdet-iyi-d Barnabas akk-d Caɛul i ccɣel-nni iwumi i ten-heggaɣ.  </p> 
 <p><a name="an_13:3">13:3</a> Mi ẓummen, ssersen ifassen-nsen fell-asen, rnan dɛan ɣer Sidi Ṛebbi, dɣa ǧǧan-ten ad ṛuḥen.  </p> 
 <p><a name="an_13:4">13:4</a> Barnabas akk-d Caɛul i gceggeɛ Ṛṛuḥ iqedsen, ṣubben ɣer lmeṛṣa n Slukya ; syenna rekben di lbabuṛ ɣer tegzirt n Qubṛus. </p> 
 <p><a name="an_13:5">13:5</a> Mi wwḍen ɣer lmeṛṣa n Salamin, beccṛen awal n Ṛebbi di leǧwameɛ n wat Isṛail ; Yuḥenna yella yid-sen, yețɛawan-iten.  </p> 
 <p><a name="an_13:6">13:6</a> Mi zegren akk tigzirt-nni, wwḍen ɣer temdint n Bafus, mlalen-d d yiwen useḥḥar n wat Isṛail i gețțarran iman-is d nnbi, isem-is Barɛisa. </p> 
 <p><a name="an_13:7">13:7</a> Ițțili ɣer lḥakem Serjyus Bulus, yellan d argaz ifehmen. Serjyus iceggeɛ ɣer Barnabas d Caɛul, axaṭer yebɣa ad isel i wawal n Ṛebbi. </p> 
 <p><a name="an_13:8">13:8</a> Meɛna Ilimas aseḥḥar-nni, ( akkagi i gețwaṭerjem yisem-is s tyunanit ) yețxaṣam-iten, yețnadi ad isseɛṛeq abrid n liman i lḥakem-nni.  </p> 
 <p><a name="an_13:9">13:9</a> Caɛul, ițusemman daɣen Bulus, yeččuṛen d Ṛṛuḥ iqedsen, issers allen-is fell-as, </p> 
 <p><a name="an_13:10">13:10</a> yenna-yas : A mmi-s n Cciṭan yeččuṛen ț-țiḥila d lexdeɛ ! Ay aɛdaw n lḥeqq ! Ar melmi ara tesseɛwajeḍ akka iberdan n lewqam n Sidi Ṛebbi ? </p> 
 <p><a name="an_13:11">13:11</a> Tura atan ufus n Sidi Ṛebbi fell-ak : aț-țedderɣleḍ, aț-țeqqimeḍ kra n wussan ur tețwaliḍ ara tafat n yiṭij. Imiren kan, Ilimas idderɣel, yeɣli-d fell-as ṭṭlam ; yeqqim isferfud, ițnadi anwa ara s-yeṭṭfen afus.  </p> 
 <p><a name="an_13:12">13:12</a> Lḥakem-nni mi gwala ayen yedṛan, itɛeǧǧeb deg wawal n Sidi Ṛebbi dɣa yumen.  </p> 
 <p><a name="an_13:13">13:13</a> Bulus d yeṛfiqen-is zegren tigzirt n Qubṛus, wwḍen ɣer temdint n Bafus, syenna rekben di lbabuṛ iwakken ad zegren ɣer temdint n Barja yellan di tmurt n Bamfilya.  </p> 
 <p><a name="an_13:14">13:14</a> Mi wwḍen ɣer dinna, Yuḥenna yeǧǧa-ten yuɣal ɣer temdint n Lquds ; ma d nutni kemmlen abrid-nsen si temdint n Barja, wwḍen ɣer temdint n Antyuc yellan di tmurt n Bisidya. Deg wass n westeɛfu, i deg țnejmaɛen wat Isṛail di leǧwameɛ, kecmen ɣer lǧameɛ, qqimen.  </p> 
 <p><a name="an_13:15">13:15</a> Mi d-ɣṛan tiktabin n Sidna Musa akk-d Lenbiya, imeqqranen n lǧameɛ nni nnan-asen : Ay atmaten, ma yella tesɛam kra n wawal s wayes ara tenhum lɣaci-agi, ɛeddit-ed aț-țhedṛem !  </p> 
 <p><a name="an_13:16">13:16</a> Bulus ikker-ed, iwehha-yasen s ufus-is, yenna : Ay at Isṛail ! A kunwi iḍuɛen Ṛebbi, ḥesset-iyi-d !  </p> 
 <p><a name="an_13:17">13:17</a> Sidi Ṛebbi Illu n wat Isṛail, yextaṛ lejdud-nneɣ, yessefti-ten mi llan di tmurt n Maṣeṛ, yessufeɣ iten id s tezmert-is tameqqrant. </p> 
 <p><a name="an_13:18">13:18</a> Iḥuder-iten azal n ṛebɛin iseggasen deg unezṛuf n Sinay. </p> 
 <p><a name="an_13:19">13:19</a> Issenger sebɛa leǧnas di tmurt n Kenɛan, iwakken a sen-yefk tamurt-nni i lejdud-nneɣ.  </p> 
 <p><a name="an_13:21">13:21</a> Dɣa ssutren i Sidi Ṛebbi a sen-d-yefk agellid ; yefka-yasen-d Caɛul, mmi-s n Qic n wedrum n Benyamin, yuɣal d agellid fell-asen azal n ṛebɛin iseggasen.  </p> 
 <p><a name="an_13:22">13:22</a> Lameɛna Sidi Ṛebbi yekkes leḥkum i Caɛul, yerra Sidna Dawed d agellid deg umkan-is. Atan wayen i d-yenna fell-as :  Ufiɣ Dawed, mmi-s n Yassa, d argaz yellan akken i t-yebɣa wul-iw, ara ixedmen lebɣi-w.  </p> 
 <p><a name="an_13:23">13:23</a> Tura, si dderya n Sidna Dawed i d-ifka Sidi Ṛebbi amsellek i wat Isṛail akken i t-id-yewɛed ; amsellek-agi d Ɛisa. </p> 
 <p><a name="an_13:24">13:24</a> Uqbel a d-yas Sidna Ɛisa, Yeḥya ibecceṛ țțuba s weɣḍas n waman i wegdud n wat Isṛail. </p> 
 <p><a name="an_13:25">13:25</a> Uqbel a t-ṭṭfen ɣer lḥebs iwakken a t-nɣen, Yeḥya yenna i lɣaci : Nekk ur lliɣ ara d win akken i tɣilem ; lameɛna a d-yas deffir-i yiwen, ur uklaleɣ ara ad fsiɣ ula d lexyuḍ n warkasen-is.  </p> 
 <p><a name="an_13:26">13:26</a> Ay atmaten ! Ay arraw n Sidna Ibṛahim d wid akk ițțaggaden Sidi Ṛebbi ! I nukkni iwumi i d-ițțuceggeɛ wawal-agi n leslak. </p> 
 <p><a name="an_13:27">13:27</a> Axaṭer imezdaɣ n temdint n Lquds d imeqqranen-nsen, ur fhimen ara d acu-t Ɛisa-agi, ur fhimen ara daɣen imeslayen i d qqaṛen di tektabt n lenbiya mkul ass n westeɛfu ; lameɛna xedmen akken i gura di tira iqedsen, mi ḥekmen ɣef Sidna Ɛisa s lmut.  </p> 
 <p><a name="an_13:28">13:28</a> Ɣas akken ur s-d-ufin ara sebba s wayes ara ḥekmen fell-as s lmut, ssutren i Bilaṭus ad yefk lameṛ iwakken a t-semmṛen ɣef lluḥ. </p> 
 <p><a name="an_13:29">13:29</a> Mi gedṛa wayen akk ițwaketben fell-as, ṣubben-t-id seg umidag, rran-t ɣer uẓekka.  </p> 
 <p><a name="an_13:30">13:30</a> Lameɛna Sidi Ṛebbi isseḥya-t-id si ger lmegtin.  </p> 
 <p><a name="an_13:31">13:31</a> Acḥal wussan nețța ițbeggin-ed iman-is i widak-nni i d-yeddan yid-es si tmurt n Jlili armi ț-țamdint n Lquds ; tura d nutni i d inigan-is zdat n yemdanen meṛṛa.  </p> 
 <p><a name="an_13:32">13:32</a> Daymi, nukkni tura newwi yawen-d lexbaṛ-agi n lxiṛ.  Ayen akken i gewɛed Sidi Ṛebbi i lejdud nneɣ, yedṛa-d tura gar-aneɣ, nukkni yellan d dderya-nsen.  </p> 
 <p><a name="an_13:33">13:33</a> Mi d-yesseḥya Sidi Ṛebbi Sidna Ɛisa, akken yura di Sabur wis-sin : Kečč d mmi, d nekk i k-id-yefkan ɣer ddunit ass-a !  </p> 
 <p><a name="an_13:34">13:34</a> Ihi Sidi Ṛebbi isseḥya-t-id si ger lmegtin iwakken ur irekku ara, akken i t-id-yenna di tira iqedsen : Si Dawed ara wen-d-fkeɣ lbaṛakat yeṣfan, lbaṛakat n tideț i wen weɛdeɣ. +  </p> 
 <p><a name="an_13:35">13:35</a> Yura daɣen : Ur tețțaǧǧaḍ ara ameɛzuz-ik ad yerku zdaxel n uẓekka .  </p> 
 <p><a name="an_13:36">13:36</a> Di lweqt-is, Sidna Dawed yexdem lebɣi n Sidi Ṛebbi ; mi gemmut, yețwamḍel ger lejdud-is, tura lǧețța-s terka deg uẓekka. </p> 
 <p><a name="an_13:37">13:37</a> Lameɛna win i d-yesseḥya Sidi Ṛebbi, ur t-yeǧǧi ara ad yerku deg uẓekka.  </p> 
 <p><a name="an_13:38">13:38</a> Ilaq ihi aț-țeẓrem ay atmaten, belli s yisem-is i wen-d-ițțubecceṛ leɛfu n ddnubat. </p> 
 <p><a name="an_13:39">13:39</a> Kra n win yumnen yis, a s-țwasemmḥen ddnubat-is, ddnubat-nni i ɣef ur tezmirem ara aț-țețwasemmḥem s ccariɛa n Musa.  </p> 
 <p><a name="an_13:40">13:40</a> Ihi ɣuṛ-wat a d-yedṛu yid-wen wayen i d-nnan lenbiya :  </p> 
 <p><a name="an_13:41">13:41</a> Ay imdanen yeččuṛen d zzux, iḥeqqṛen wiyaḍ, dehcet, tenfum syagi ! Axaṭer atan ad xedmeɣ di zzman-nwen yiwet lḥaǧa s wayes ur tețțamnem ara ma yella ḥkan awen-ț-id.  </p> 
 <p><a name="an_13:42">13:42</a> Mi kkren ad ffɣen si lǧameɛ, ḥellelen Bulus d Barnabas iwakken a d-uɣalen ass n ssebt i d-iteddun a d-rnun a d-mmeslayen ɣef wannect nni. </p> 
 <p><a name="an_13:43">13:43</a> Mi mfaṛaqen, aṭas n wat Isṛail akk-d wid i gkecmen ddin-nsen, tebɛen-d Bulus akk-d Barnabas i sen-ițmeslayen, i ten-inehhun ad ṭṭfen di ṛṛeḥma n Sidi Ṛebbi.  </p> 
 <p><a name="an_13:44">13:44</a> Ass n westeɛfu nniḍen, qṛib tamdint meṛṛa i d-innejmaɛen iwakken ad slen i wawal n Ṛebbi. </p> 
 <p><a name="an_13:45">13:45</a> Mi walan annect-nni n lɣaci, at Isṛail usmen, sseɛwajen s rregmat ayen i d-yeqqaṛ Bulus. </p> 
 <p><a name="an_13:46">13:46</a> Lameɛna Bulus d Barnabas nnan-asen ɛinani : I kunwi d imezwura iwumi ilaq ad ițțubecceṛ wawal n Sidi Ṛebbi, meɛna imi ur t-teqbilem ara, tḥusbem iman-nwen ur tuklalem ara tudert n dayem, ihi a nezzi ɣer wid ur nelli ara n wat Isṛail.  </p> 
 <p><a name="an_13:47">13:47</a> Axaṭer atah wayen i d-yenna Sidi Ṛebbi : Sbeddeɣ-k aț-țiliḍ ț-țafat i leǧnas, aț-țessiwḍeḍ leslak alamma d ixfawen n ddunit.  </p> 
 <p><a name="an_13:48">13:48</a> Leǧnas ur nelli ara n wat Isṛail feṛḥen mi slan s wannect-agi, țḥemmiden awal n Ṛebbi ; wid akk yețțuheggan i tudert n dayem umnen.  </p> 
 <p><a name="an_13:49">13:49</a> Awal n Sidi Ṛebbi yețțubecceṛ di mkul amkan n tmurt. </p> 
 <p><a name="an_13:50">13:50</a> Lameɛna at Isṛail sḥeṛcen imeqqranen n temdint akk-d d kra n tilawin timeṛkantiyin i gettabaɛen ddin n wat Isṛail ; sekkren-d aqehheṛ ameqqran ɣef Bulus d Barnabas, ssufɣen-ten si tmurt-nsen. </p> 
 <p><a name="an_13:51">13:51</a> Bulus d Barnabas zwin aɣebbaṛ n iḍaṛṛen-nsen , ṛuḥen ɣer temdint n Ikunyum.  </p> 
 <p><a name="an_13:52">13:52</a> Ma d inelmaden ijdiden, ččuṛen d lfeṛḥ akk-d Ṛṛuḥ iqedsen.  </p> 
 <p><a name="an_14:1">14:1</a> Di temdint n Ikunyum, Bulus d Barnabas kecmen daɣen ɣer lǧameɛ n wat Isṛail, beccṛen-asen lexbaṛ n lxiṛ ; aṭas n Iyunaniyen akk-d wat Isṛail i gumnen.  </p> 
 <p><a name="an_14:2">14:2</a> Lameɛna kra seg wat Isṛail i gugin ad amnen, ɛemṛen-asen aqeṛṛuy i wat n leǧnas niḍen sḥeṛcen-ten iwakken ad xedmen lbaṭel i watmaten ; </p> 
 <p><a name="an_14:3">14:3</a> ɣas akken, Bulus d Barnabas ɛeṭṭlen di temdint n Ikunyum , țeklen ɣef Sidi Ṛebbi, heddṛen s țțbut mbla akukru ɣef ṛṛeḥma-ines. Sidi Ṛebbi ițbeggin-ed s lbeṛhanat d licaṛat belli ayen țbecciṛen ț-țideț. </p> 
 <p><a name="an_14:4">14:4</a> Imezdaɣ n temdint mfaṛaqen : kra deg-sen tebɛen at Isṛail, ma d wiyaḍ ddan d ṛṛusul. </p> 
 <p><a name="an_14:5">14:5</a> At Isṛail d iyunaniyen akk-d imeqqranen-nsen ɛewlen ad xedmen lbaṭel i ṛṛusul, bɣan a ten nɣen s weṛjam.  </p> 
 <p><a name="an_14:6">14:6</a> Bulus d Barnabas mi ten-yewweḍ lexbaṛ, beddlen amkan ṛuḥen ɣer tmurt n Likawnya, ɣer temdinin n Listra, d Derba akk-d tudrin i sent-id yezzin ; </p> 
 <p><a name="an_14:7">14:7</a> țbecciṛen dinna lexbaṛ n lxiṛ n Sidi Ṛebbi.  </p> 
 <p><a name="an_14:8">14:8</a> Di temdint n Listra, yella yiwen wergaz ikerfen seg-wasmi i d-ilul, di leɛmeṛ-is ur yelḥi. </p> 
 <p><a name="an_14:9">14:9</a> Yesmeḥsis i wayen i d-yeqqaṛ Bulus ; Bulus iṛeṣṣa allen-is fell-as, iwala belli yesɛa liman iwakken ad yeḥlu, </p> 
 <p><a name="an_14:10">14:10</a> dɣa yenṭeq ɣur-es s ṣṣut ɛlayen yenna-yas : Ekker, bedd ɣef yidaṛṛen-ik ! Imiren kan yekker, yebda ițeddu. </p> 
 <p><a name="an_14:11">14:11</a> Mi walan ayen yexdem Bulus, lɣaci n tmurt n Likawnya țɛeggiḍen s tutlayt nsen : Iṛebbiten ṣubben-d ɣuṛ-nneɣ s ṣṣifa n yemdanen !  </p> 
 <p><a name="an_14:12">14:12</a> Semman-as i Barnabas « SSus », ma d Bulus « Hermas » axaṭer d nețța i d-ițmeslayen. </p> 
 <p><a name="an_14:13">14:13</a> Lmuqeddem n lǧameɛ n SSus yellan ɣer tewwurt n temdint, yewwi-d izgaren akk-d ijeǧǧigen zdat tewwura n lǧameɛ ; yebɣa a ten-yezlu d asfel nețța d lɣaci i Bulus akk-d Barnabas. </p> 
 <p><a name="an_14:14">14:14</a> Ṛṛusul mi slan s wannect nni cerrgen llebsa-nsen, uzzlen ɣer tlemmast n lɣaci, țɛeggiḍen :  </p> 
 <p><a name="an_14:15">14:15</a> Ay irgazen, acuɣeṛ i txeddmem akka ? Nukni d imdanen kan i nella am kunwi ! D lexbaṛ n lxiṛ  i wen-d newwi iwakken aț-țeṭṭixṛem i leɛbada n wayen ur nesɛi lmeɛna ; aț-tebɛem abrid n Sidi Ṛebbi, Bab n tudert i d-ixelqen igenni d lqaɛa, lebḥeṛ d wayen akk yellan deg-s.  </p> 
 <p><a name="an_14:16">14:16</a> Ɣas akken di zzmanat iɛeddan yeǧǧa imdanen n leǧnas meṛṛa ad tebɛen abrid i sen-yehwan ; </p> 
 <p><a name="an_14:17">14:17</a> nețța yețbeggin-ed iman-is s lxiṛat i gxeddem : d nețța i wen-d-ițțaken ageffur ( lehwa ), d nețța i wen-d ițțaken lɣellat di lweqt-nsent, d nețța daɣen i wen-d-ițakken tamɛict-nwen, ula d ulawen-nwen d nețța i ten issefṛaḥen. </p> 
 <p><a name="an_14:18">14:18</a> Ɣas akken ṛṛusul nnan asen imeslayen-agi, s lḥif ameqqran i ḥebsen lɣaci-nni iwakken ur sen zellun ara asfel.  </p> 
 <p><a name="an_14:19">14:19</a> Dɣa usan-d kra seg wat Isṛail si temdinin n Antyuc akk-d Ikunyum, sekkren ccwal ger lɣaci, sḥeṛcen-ten ad ṛejmen Bulus iwakken a t-nɣen. Mi t-ṛejmen, zzuɣṛen-t ɣer beṛṛa n temdint, axaṭer ɣilen yemmut.  </p> 
 <p><a name="an_14:20">14:20</a> Lameɛna nnejmaɛen-d ɣuṛ-es inelmaden, ṭṭfen-as afus, yuɣal yekker yekcem ɣer temdint. Azekka-nni, nețța d Barnabas ṛuḥen ɣer temdint n Derba.  </p> 
 <p><a name="an_14:21">14:21</a> Mi beccṛen lexbaṛ n lxiṛ  di temdint-nni n Derba, aṭas inelmaden i d-yernan ; dɣa uɣalen ɣer temdinin n Listra, Ikunyum d Antyuc  </p> 
 <p><a name="an_14:22">14:22</a> iwakken ad sǧehden ulawen n inelmaden. Nehhun-ten ad ṭṭfen di liman, qqaṛen-asen : S waṭas n leɛtab ara nekcem ɣer ddewla n Sidi Ṛebbi.  </p> 
 <p><a name="an_14:23">14:23</a> Sbedden imḍebbṛen di mkul tajmaɛt ; mi uẓamen, rnan dɛan ɣer Sidi Ṛebbi, ǧǧan-ten seddaw leɛnaya n Sidi Ṛebbi s wayes umnen.  </p> 
 <p><a name="an_14:24">14:24</a> Zegren si tmurt n Bisidya, wwḍen-d ɣer tmurt n Bamfilya, </p> 
 <p><a name="an_14:25">14:25</a> beccṛen awal n Ṛebbi di temdint n Barja, imiren ṣubben-d ɣer temdint n Aṭalya.  </p> 
 <p><a name="an_14:26">14:26</a> Syenna rekben lbabuṛ, uɣalen ɣer temdint n Antyuc ansi i ten wekklen i Ṛṛeḥma n Sidi Ṛebbi ɣef cceɣl-nni xedmen. </p> 
 <p><a name="an_14:27">14:27</a> Mi wwḍen, snejmaɛen-d atmaten, ḥkan-asen ayen akk i gexdem Sidi Ṛebbi yis-sen, d wamek i d-wwin ɣer webrid n Sidi Ṛebbi at leǧnas ur nelli ara n wat Isṛail.  </p> 
 <p><a name="an_14:28">14:28</a> Sɛeddan aṭas n wussan akk-d inelmaden.  </p> 
 <p><a name="an_15:1">15:1</a> Kra n yemdanen i d-yusan si tmurt n Yahuda ɣer temdint n Antyuc, bɣan ad slemden atmaten, qqaṛen-asen : Ma yella ur teḍhiṛem ara am akken i d-tenna ccariɛa n Musa, ur tezmirem ara aț-țețwasellkem  </p> 
 <p><a name="an_15:2">15:2</a> Bulus d Barnabas ur qbilen ara ayen i sselmaden yergazen-agi i d-yusan si tmurt n Yahuda dɣa yekker lxilaf ameqqran gar-asen. Ceggɛen Bulus d Barnabas akk-d kra seg-sen, ad alin ɣer ṛṛusul akk-d imeqqranen yellan di temdint n Lquds iwakken ad msefhamen ɣef temsalt-agi n ṭṭhaṛa. </p> 
 <p><a name="an_15:3">15:3</a> Tajmaɛt-nni n watmaten n temdint n Antyuc tɛawen-iten deg wayen ara ḥwiǧen deg webrid-nsen ; dɣa ṛuḥen, ɛeddan si tmura n Finisya akk-d Samarya, ḥekkun di mkul amkan amek umnen s Sidna Ɛisa leǧnas ur nelli ara n wat Isṛail. Mi slan i wannect-agi, feṛḥen aṭas watmaten.  </p> 
 <p><a name="an_15:4">15:4</a> Mi wwḍen ɣer temdint n Lquds, tesṭerḥeb yis-sen tejmaɛt n watmaten d ṛṛusul akk-d imeqqranen n tejmaɛt ; nutni ḥkan-asen lecɣal imeqqranen i gexdem Sidi Ṛebbi gar-asen. </p> 
 <p><a name="an_15:5">15:5</a> Kra si terbaɛt n ifariziyen i gumnen s Sidna Ɛisa kkren-d nnan : Ilaq aț-ḍehhṛem i watmaten ur nelli ara n wat Isṛail yerna a sen tamṛem ad tebɛen ccariɛa n Sidna Musa.  </p> 
 <p><a name="an_15:6">15:6</a> Dɣa ṛṛusul akk-d imeqqranen n tejmaɛt nnejmaɛen iwakken ad frun taluft-agi. </p> 
 <p><a name="an_15:7">15:7</a> Mi ketṛen deg wawal, ikker-ed Butṛus yenna-yasen : Ay atmaten, teẓram belli Sidi Ṛebbi yextaṛ-iyi-d gar-awen seg wass amezwaru iwakken ad beccṛeɣ lexbaṛ n lxiṛ i leǧnas ur nelli ara n wat Isṛail, akken ad amnen.  </p> 
 <p><a name="an_15:8">15:8</a> Sidi Ṛebbi yessnen ulawen, ibeggen-ed belli iqbel-iten imi i sen-yefka Ṛṛuḥ iqedsen am akken i ɣ-t-id-yefka i nukkni ; </p> 
 <p><a name="an_15:9">15:9</a> ur yexdim ara lxilaf gar-aɣ d yid-sen, imi i gṣeffa ulawen-nsen s liman ula d nutni.  </p> 
 <p><a name="an_15:10">15:10</a> Ihi tura acuɣeṛ i tețjeṛṛibem Sidi Ṛebbi, mi tebɣam aț-țessersem ɣef tuyat n wid yumnen azaglu ur nerfid nukkni, ur rfiden lejdud-nneɣ ;  </p> 
 <p><a name="an_15:11">15:11</a> lameɛna s ṛṛeḥma n Sidna Ɛisa swayes i numen, i nețțusellek am akken țțusellken ula d nutni.  </p> 
 <p><a name="an_15:12">15:12</a> Akken ma llan di tejmaɛt ssusmen, dɣa smeḥsisen i Bulus d Barnabas i d-iḥekkun licaṛat d leɛǧayeb i gexdem Sidi Ṛebbi yis-sen ger leǧnas ur nelli ara n wat Isṛail.  </p> 
 <p><a name="an_15:13">15:13</a> Mi kfan ameslay, inṭeq Yeɛqub yenna-d : Ay atmaten, ḥesset-iyi-d : Tura </p> 
 <p><a name="an_15:14">15:14</a> Semɛun yeḥka-d amek i d-ixtaṛ Sidi Ṛebbi si ger leǧnas si tazwara, lumma ara yeddun s yisem-is.  </p> 
 <p><a name="an_15:15">15:15</a> S wakka, yedṛa-d wayen i d nnan lenbiya am akken yura di tektabt iqedsen :  </p> 
 <p><a name="an_15:16">15:16</a> Deg ussan-nni, a d-uɣaleɣ a s-ɛiwdeɣ lebni i wexxam n Dawed ; axxam-nni yeɣlin, ad sbeddeɣ leḥyuḍ-is ihudden,  </p> 
 <p><a name="an_15:17">15:17</a> iwakken ayen i d-iqqimen seg imdanen akk-d leǧnas meṛṛa i geslan s yisem-iw, ad nadin ɣef Sidi Ṛebbi.  </p> 
 <p><a name="an_15:18">15:18</a> Akka i d-yenna Sidi Ṛebbi i gxeddmen lecɣal-agi i d-ihegga si zik.  </p> 
 <p><a name="an_15:19">15:19</a> Ihi a wen-d-iniɣ : lemmer ufiɣ, ur nrennu ara aɣilif i wid ur nelli ara n wat Isṛail, i gumnen s Sidna Ɛisa.  </p> 
 <p><a name="an_15:20">15:20</a> Lameɛna a sen-naru tabṛaț iwakken ad ṭṭixṛen ɣef wučči n weksum immezlen i lmeṣnuɛat d ssadat. Ad ttixṛen daɣen i yir tikli, i wučči n lmal immurḍsen akk-d tissit n idammen, </p> 
 <p><a name="an_15:21">15:21</a> axaṭer si zzman aqdim llan yemdanen ițbecciṛen ccariɛa n Musa di mkul tamdint, yerna mkul ass n westeɛfu qqaṛen taktabt-is di leǧwameɛ n wat Isṛail.  </p> 
 <p><a name="an_15:22">15:22</a> Ihi ṛṛusul, imeqqranen n tejmaɛt akk-d watmaten meṛṛa walan belli d ṛṛay yelhan ma yella xtaṛen-d kra n watmaten si gar-asen, iwakken a ten-ceggɛen ɣer temdint n Antyuc nutni d Bulus akk-d Barnabas. Xtaṛen Yuda, ițusemman Barsaba akk-d Silas ; d irgazen i țqadaṛen watmaten. </p> 
 <p><a name="an_15:23">15:23</a> Fkan asen tabṛaț i deg uran :  Ay atmaten ur nelli ara n wat Isṛail, i gzedɣen di temdinin n Antyuc, n Surya akk-d Silisya, țsellimen-d fell-awen ṛṛusul akk-d imeqqranen n tejmaɛt akk-d watmaten n temdint n Lquds.  </p> 
 <p><a name="an_15:24">15:24</a> Nesla belli kra n watmaten-nneɣ ṛuḥen-d ɣuṛ-wen mbla ma nefka-yasen lameṛ, rwin-kkun s yimeslayen-nsen, skecmen-awen ccekk.  </p> 
 <p><a name="an_15:25">15:25</a> Tura mi nemcawaṛ, neqsed s yiwen ṛṛay a d-nextiṛ kra n watmaten, ara wen-nceggeɛ nutni d watmaten nneɣ eɛzizen Bulus d Barnabas ;  </p> 
 <p><a name="an_15:26">15:26</a> nutni i gsebblen tudert nsen ɣef yisem n Sidna Ɛisa Lmasiḥ. + </p> 
 <p><a name="an_15:27">15:27</a> Nceggeɛ-awen ihi Yuda akk-d Silas ara wen-d-yeṣṣiwḍen s yiman nsen imeslayen-agi :  </p> 
 <p><a name="an_15:28">15:28</a> Ṛṛuḥ iqedsen akk-d nukkni, nwala d ayen yelhan m'ur nḥettem ara fell-awen ayen nniḍen sennig wayen ilaqen,   </p> 
 <p><a name="an_15:29">15:29</a> yeɛni aț-țettixṛem i wučči n iseflawen yemmezlen i lmeṣnuɛat, i tissit n idammen, i lmal yemmuṛdsen d zzna. D ayen yelhan ma tḥudrem iman-nwen ɣef wannect-agi. Qqimet di lehna.  </p> 
 <p><a name="an_15:30">15:30</a> Imiren wid-nni yețwaxtaṛen ǧǧan atmaten di lehna, ṛuḥen ɣer temdint n Antyuc. Dinna jemɛen-d atmaten, fkan-asen tabṛaț-nni.  </p> 
 <p><a name="an_15:31">15:31</a> Mi ț-id-ɣṛan, feṛḥen aṭas s lewṣayat yellan deg-s d yimeslayen i ten-isǧehden. </p> 
 <p><a name="an_15:32">15:32</a> Yuda d Silas iwumi yețțunefk a d-țxebbiṛen s ɣuṛ Sidi Ṛebbi, nhan atmaten-nni, sǧehden-ten s waṭas n yimeslayen.  </p> 
 <p><a name="an_15:33">15:33</a> Mi sɛeddan dinna kra n wussan, serrḥen-asen watmaten ad uɣalen di lehna ɣer wid i ten-id-iceggɛen ; </p> 
 <p><a name="an_15:34">15:34</a> ma d Silas iɛǧeb-as lḥal yebɣa ad yernu kra n wussan dinna. </p> 
 <p><a name="an_15:35">15:35</a> Bulus d Barnabas qqimen di temdint-agi n Antyuc , sselmaden, țbecciṛen awal n Sidi Ṛebbi nutni d waṭas n watmaten nniḍen.  </p> 
 <p><a name="an_15:36">15:36</a> Mi ɛeddan kra n wussan, Bulus yenna-yas i Barnabas : Eyya a nuɣal a nesteqsi ɣef watmaten yellan di temdinin anda nbecceṛ awal n Sidi Ṛebbi, iwakken a nẓer amek i țilin. </p> 
 <p><a name="an_15:37">15:37</a> Barnabas yebɣa ad yawi yid-sen Yuḥenna ițusemman Maṛqus, </p> 
 <p><a name="an_15:38">15:38</a> lameɛna Bulus ur yeqbil ara ad awin yid-sen win akken i ten yeǧǧan di tmurt n Bamfilya, yerna ur ten-iɛawen ara di ccɣel-nsen.  </p> 
 <p><a name="an_15:39">15:39</a> Ikker yiwen lxilaf gar-asen armi mfaṛaqen. Barnabas yewwi yid-es Maṛqus, rekben di lbabuṛ ɣer tegzirt n Qubṛus, </p> 
 <p><a name="an_15:40">15:40</a> ma d Bulus yextaṛ Silas ad iddu yid-es. Atmaten n tejmaɛt wekklen-t i ṛṛeḥma n Sidi Ṛebbi, dɣa iṛuḥ. </p> 
 <p><a name="an_15:41">15:41</a> Yekka-d nețța d Silas si tmura n Surya akk-d Silisya, isseǧhad tijmaɛin n watmaten.  </p> 
 <p><a name="an_16:1">16:1</a> Bulus d Silas wwḍen ɣer temdinin n Derba d Listra ; ufan dinna yiwen unelmad isem-is Timuti, baba-s d ayunani, ma d yemma-s n wat Isṛail, nețțat daɣen tumen s Lmasiḥ. </p> 
 <p><a name="an_16:2">16:2</a> Atmaten yellan di temdint n Listra ț-țemdint n Ikunyum ḥemmlen-t yerna țcekkiṛen-t. </p> 
 <p><a name="an_16:3">16:3</a> Bulus yebɣa ad yawi yid-es Timuti, daymi i s-iḍehheṛ ɣef ddemma n wat Isṛail yellan dinna, axaṭer ẓran akk belli baba-s d ayunani.  </p> 
 <p><a name="an_16:4">16:4</a> Di yal tamdint i deg ɛeddan, țxebbiṛen atmaten s lewṣayat yellan di tebṛaț i d-fkan ṛṛusul d imeqqranen n temdint n Lquds, nehhun-ten ad tebɛen lewṣayat-agi.  </p> 
 <p><a name="an_16:5">16:5</a> Tijmaɛin n watmaten țimɣuṛent di liman, leḥsab n widak yumnen s Lmasiḥ yețnerni kull ass.  </p> 
 <p><a name="an_16:6">16:6</a> Ṛṛuḥ iqedsen ur sen-iserreḥ ara ad ṛuḥen ad beccṛen awal n Sidi Ṛebbi di tmurt n Asya, dɣa zegren timura n Frijya d Galasya. </p> 
 <p><a name="an_16:7">16:7</a> Mi wwḍen ɣer tmurt n Misya, ɛeṛden ad ṛuḥen ɣer tmurt n Bitinya ; lameɛna Ṛṛuḥ n Sidna Ɛisa ur sen-iserreḥ ara daɣen ad ṛuḥen ɣer dinna.  </p> 
 <p><a name="an_16:8">16:8</a> Ɛeddan si tmurt n Misya, ṣubben ɣer temdint n Truwas. </p> 
 <p><a name="an_16:9">16:9</a> Deg iḍ, iweḥḥa-yas-ed Ṛebbi i Bulus, iwala yiwen umezdaɣ n Masidunya ibedd-ed ɣuṛ-es, ițḥellil-it ad iɛeddi ɣuṛ-sen ɣer tmurt n Masidunya a ten-iɛiwen.  </p> 
 <p><a name="an_16:10">16:10</a> Mbeɛd ayen i s-d-iweḥḥa Sidi Ṛebbi i Bulus, nennuda amek ara nṛuḥ ɣer tmurt n Masidunya, axaṭer nefhem belli d Sidi Ṛebbi i ɣ-d yessawlen iwakken a nbecceṛ Lexbeṛ n lxiṛ  di tmurt-nni.  </p> 
 <p><a name="an_16:11">16:11</a> Mi nerkeb di lbabuṛ si temdint n Truwas, nṛuḥ qbala ɣer tegzirt n Samutras. Azekka-nni nkemmel ɣer temdint n Nabulis. </p> 
 <p><a name="an_16:12">16:12</a> Syenna nṛuḥ ɣer temdint n Filibus, illan țamdint tamezwarut n Masidunya i gesteɛmeṛ Ṛuman ; nesɛedda kra n wussan dinna.  </p> 
 <p><a name="an_16:13">16:13</a> Ass n westeɛfu, neffeɣ si temdint nṛuḥ ɣer rrif n wasif, nɣil a naf amkan anda țnejmaɛen i tẓallit. Nufa dinna kra n tilawin nnejmaɛent, neqqim iwakken a nemmeslay yid sent. </p> 
 <p><a name="an_16:14">16:14</a> Tella dinna yiwet n tmeṭṭut isem-is Lidya n temdint n Tyatir, teznuzu lkețțan azeggaɣ ɣlayen ; ț-țameṭṭut iḍuɛen Sidi Ṛebbi. M'akken i d-tesmeḥsis, Sidi Ṛebbi yeldi-yas ul-is iwakken aț-țerr ddehn is ɣer wayen i d-yeqqaṛ Bulus.  </p> 
 <p><a name="an_16:15">16:15</a> Mi tețwaɣḍes deg waman nețțat d wat wexxam-is, teɛṛeḍ-aɣ, tenna-d : ma twalam umneɣ seg ul-iw s Sidi Ṛebbi, eyyaw aț-țesɛeddim kra n wussan deg wexxam-iw. Dɣa teḥreṣ-aɣ iwakken a nekcem.  </p> 
 <p><a name="an_16:16">16:16</a> Yiwen wass mi nteddu ɣer wemkan-nni n tẓallit, temmuger-aɣ-d yiwet n tqeddact ițwamelken, ț- țaderwict, tețțawi-d aṭas n lfayda i yemɛellmen i ɣef txeddem.  </p> 
 <p><a name="an_16:17">16:17</a> Tettabaɛ-aɣ-d nekk d Bulus, tețɛeggiḍ teqqaṛ : Imdanen-agi d iqeddacen n Ṛebbi ɛlayen, țbecciṛen-awen abrid n leslak.  </p> 
 <p><a name="an_16:18">16:18</a> Acḥal n wussan nețțat tettabaɛ deg-nneɣ, yuɣal Bulus yeɛya deg-s, yezzi ɣuṛ-es, yenna i uṛuḥani i ț-imelken : Umṛeɣ-k s yisem n Ɛisa Lmasiḥ, effeɣ si tmeṭṭut-agi ! Imiren kan, yeffeɣ-it uṛuḥani-nni.  </p> 
 <p><a name="an_16:19">16:19</a> Imɛellmen-is mi walan dayen ur sen-d-tețțawi ara lfayda, ṭṭfen Bulus d Silas, zzuɣṛen-ten ɣer wezniq n temdint anda țnejmaɛen lɣaci zdat lḥukkam. </p> 
 <p><a name="an_16:20">16:20</a> Wwin-ten ɣer lḥakem n tmurt n Ṛuman nnan : Imdanen-agi-nni n wat Isṛail cewwlen tamdint-nneɣ. </p> 
 <p><a name="an_16:21">16:21</a> Sselmaden lɛaddat ur nezmir a tent-neqbel neɣ a tent-netbeɛ nukkni yellan d iṛumaniyen.  </p> 
 <p><a name="an_16:22">16:22</a> Lɣaci kkren-d ɣuṛ-sen s leɛyaḍ, dɣa lḥukkam n Ṛuman umṛen a sen-cerrgen llebsa-nsen, a ten wwten s ujelkkaḍ ( s ucelliṭ ). </p> 
 <p><a name="an_16:23">16:23</a> Mi sen-sseṛwan tiyitiwin, rran-ten ɣer lḥebs, umṛen i wɛessas n lḥebs a ten-iɛas akken ilaq. </p> 
 <p><a name="an_16:24">16:24</a> Akken i t-yewweḍ lameṛ-agi, aɛessas-nni yerra-ten ɣer lḥebs yellan seddaw tmurt, yerna yurez-asen idaṛṛen s snasel. </p> 
 <p><a name="an_16:25">16:25</a> Di țnaṣfa n yiḍ, Bulus d Silas deɛɛun ɣer Sidi Ṛebbi, cennun, țḥemmiden-t, imeḥbas nniḍen smeḥsisen-asen.  </p> 
 <p><a name="an_16:26">16:26</a> Taswiɛt kan, tezlez tmurt armi yenhezz ula d lsas n lḥebs ; imiren kan ldint tewwura n lḥebs, snasel icudden imeḥbas meṛṛa qelɛent-ed.  </p> 
 <p><a name="an_16:27">16:27</a> Aɛessas-nni ikker-ed ; mi gwala tiwwura n lḥebs ldint, ijbed-ed asekkin is iwakken ad ineɣ iman-is ; yenwa rewlen imeḥbas-nni.  </p> 
 <p><a name="an_16:28">16:28</a> Lameɛna Bulus iɛeggeḍ s lǧehd is, yenna-yas : Eḥbes ! Ḥader aț-țenɣeḍ iman ik ! Aql-aɣ akk dagi.  </p> 
 <p><a name="an_16:29">16:29</a> Dɣa aɛessas yumeṛ a s-d-ceɛlen tafat, yuzzel ɣer daxel n lḥebs ; yeɣli ɣer idaṛṛen n Bulus d Silas, ițergigi. </p> 
 <p><a name="an_16:30">16:30</a> Yessufeɣ-iten-id si lḥebs, yenna yasen : A ssyad-i, d acu i glaqen a t xedmeɣ iwakken ad țțusellkeɣ ?  </p> 
 <p><a name="an_16:31">16:31</a> Rran-as : Amen s Sidna Ɛisa, aț-țețusellkeḍ s kečč s wat wexxam-ik.  </p> 
 <p><a name="an_16:32">16:32</a> Dɣa beccṛen-as awal n Sidi Ṛebbi i nețța d wat wexxam-is meṛṛa. </p> 
 <p><a name="an_16:33">16:33</a> Deg yiḍ-nni kan, aɛessas-nni yewwi-ten anda ara sen-yessired lejruḥ-nsen ; imiren-nni sɣeḍsen-t deg aman s nețța s twacult-is. </p> 
 <p><a name="an_16:34">16:34</a> Yuɣal nețța d Bulus d Silas ɣer wexxam-is, issers-asen-d ad ččen. Yefṛeḥ aṭas nețța d wat wexxam-is imi umnen s Sidi Ṛebbi. </p> 
 <p><a name="an_16:35">16:35</a> Akken yuli wass, lḥukkam n Ṛuman ceggɛen wid ara yinin i uɛessas n lḥebs : Serreḥ-asen i imeḥbas-nni ! </p> 
 <p><a name="an_16:36">16:36</a> Aɛessas ixebbeṛ Bulus yenna yas : Lḥukkam umṛen-iyi-d a wen serrḥeɣ ; tura tzemrem aț-țefɣem, ṛuḥet di lehna !  </p> 
 <p><a name="an_16:37">16:37</a> Lameɛna Bulus yenna-yas : Amek ! Ewten-aɣ zdat medden s ujelkaḍ ( s ucelliḍ ) mbla ma nɛedda di ccṛeɛ nukni yellan d iṛumaniyen yerna skecmen-aɣ ɣer lḥebs tura bɣan a ɣ-ssufɣen s tuffra ? Awah ! A d-asen nutni s yiman-nsen a ɣ-serrḥen.  </p> 
 <p><a name="an_16:38">16:38</a> Wid-nni i d-ițțuceggɛen ssawḍen imeslayen-nni i lḥukkam n Ṛuman. Lḥukkam ikcem-iten lxuf mi slan belli Bulus d Silas d iṛumaniyen i llan. Usan-d ḍelben deg-sen ssmaḥ, serrḥen-asen si lḥebs, </p> 
 <p><a name="an_16:39">16:39</a> ḥellelen-ten ad ffɣen si temdint. </p> 
 <p><a name="an_16:40">16:40</a> Mi ffɣen si lḥebs, Bulus d Silas ṛuḥen ɣer wexxam n Lidya, anda nnejmaɛen d watmaten, nhan-ten, imiren ṛuḥen.  </p> 
 <p><a name="an_17:1">17:1</a> Bulus d Silas ɛeddan si temdinin n Anfibulis akk-d Abulunis, wwḍen ɣer temdint n Tiṣalunik anda sɛan wat Isṛail lǧameɛ.  </p> 
 <p><a name="an_17:2">17:2</a> Akken yennum, Bulus yekcem ɣer lǧameɛ ; tlata smanat, mkul ass n westeɛfu, ițmeslay yid-sen ɣef wawal n Ṛebbi. </p> 
 <p><a name="an_17:3">17:3</a> Issefham-asen, ițbeggin-asen-d di Tira belli Lmasiḥ ilaq ad yeɛteb yerna a d-yeḥyu si ger lmegtin, iqqaṛ-asen : Ɛisa-yagi ɣef i wen-d-hedṛeɣ, d nețța i d Lmasiḥ.  </p> 
 <p><a name="an_17:4">17:4</a> Kra deg-sen umnen ; aṭas n iyunaniyen iḍuɛen Sidi Ṛebbi d waṭas n tilawin n laṣel i geddan daɣen d Bulus akk-d Silas.  </p> 
 <p><a name="an_17:5">17:5</a> Lameɛna at Isṛail usmen, dɣa wwin yid-sen kra n yergazen n diri ițmenṭaṛen deg iberdan, sekkren ccwal ger lɣaci, cewwlen tamdint meṛṛa, zeḍmen ɣer wexxam n Yazun, qellben ɣef Bulus d Silas iwakken a ten-caṛɛen zdat lɣaci. </p> 
 <p><a name="an_17:6">17:6</a> Imi ur ten-ufin ara, zuɣṛen-d Yazun d kra n watmaten ɣer lḥukkam n temdint, bdan țɛeggiḍen : D irgazen-agi i gerwin ddunit, atan usan-d armi d ɣuṛ-nneɣ, </p> 
 <p><a name="an_17:7">17:7</a> Yazun isṭerḥeb yis-sen. Xulfen lumuṛ n Qayṣer, qqaṛen belli yella ugellid nniḍen ițusemman Ɛisa.  </p> 
 <p><a name="an_17:8">17:8</a> S yimeslayen-agi, sḥeṛcen lɣaci akk-d lḥakem n temdint. </p> 
 <p><a name="an_17:9">17:9</a> Ugin a sen-serrḥen i Yazun d watmaten nniḍen armi xellṣen lexṭiya.  </p> 
 <p><a name="an_17:10">17:10</a> Mi d-yewweḍ yiḍ, atmaten srewlen Bulus d Silas ɣer temdint n Biri. Mi wwḍen, kecmen ɣer lǧameɛ n wat Isṛail. </p> 
 <p><a name="an_17:11">17:11</a> Lɣaci n dinna sṭerḥben yis-sen axiṛ n wid n temdint n Tiṣalunik , qeblen awal n Ṛebbi s wul yeṣfan. Kull ass țnadin di Tira iqedsen iwakken ad walin ma yella ț-țideț, yella di tira wayen i d-qqaṛen. </p> 
 <p><a name="an_17:12">17:12</a> Aṭas seg-sen i gumnen ; ger iyunaniyen, aṭas n tilawin n laṣel d yergazen nniḍen i gumnen daɣen.  </p> 
 <p><a name="an_17:13">17:13</a> Lameɛna mi slan wat Isṛail n Tiṣalunik belli Bulus ițbecciṛ awal n Ṛebbi di temdint n Biri, uzzlen-d ad rwin lɣaci, a ten-cewwlen. </p> 
 <p><a name="an_17:14">17:14</a> Dɣa atmaten srewlen imiren Bulus ɣer lǧiha n lebḥeṛ, ma d Silas d Timuti qqimen di temdint n Biri. </p> 
 <p><a name="an_17:15">17:15</a> Widak yewwin Bulus, ssawḍen-t armi ț-țamdint n Atinya, syenna uɣalen-d ɣer temdint n Biri. Iweṣṣa-ten Bulus ad inin i Silas d Timuti, ur țɛeṭṭilen ara a t-leḥqen.  </p> 
 <p><a name="an_17:16">17:16</a> M'akken i gețṛaǧu Silas akk-d Timuti di temdint n Atinya, Bulus iɣaḍ-it lḥal mi gwala tamdint-nni teččuṛ d lmeṣnuɛat i țɛebbiden imezdaɣ-is.  </p> 
 <p><a name="an_17:17">17:17</a> Di lǧameɛ, yebda ițmeslay d wat Isṛail akk-d iyunaniyen iḍuɛen Sidi Ṛebbi. Ițmeslay daɣen akk-d wid i d-ițemlili kull ass di tejmaɛt n temdint.  </p> 
 <p><a name="an_17:18">17:18</a> Kra n lɛulama yețfelsifen, yeɛni wid ițekkin di tejmaɛt n Ibikur akk-d Stuwis, bdan țmeslayen yid-es. Kra qqaṛen : D acu i gebɣa ad yini bu lehduṛ agi ? Wiyaḍ qqaṛen : Waqila yețbecciṛ iṛebbiten ibeṛṛaniyen ! Axaṭer Bulus yella ițbecciṛ-asen ɣef Sidna Ɛisa akk-d ḥeggu n lmegtin.  </p> 
 <p><a name="an_17:19">17:19</a> Dɣa wwin-t yid-sen ɣeryiwen wemkan n ccṛeɛ n temdint-nni iwumi qqaṛen Laryufaj, nnan-as : Nezmer a nẓer d acu-t uselmed agi ajdid i ɣ-d-tewwiḍ ?  </p> 
 <p><a name="an_17:20">17:20</a> Axaṭer nesla-yak tețmeslayeḍ-ed ɣef temsalin yessewhamen. Nebɣa a nissin lmeɛna-nsent.  </p> 
 <p><a name="an_17:21">17:21</a> Imezdaɣ n temdint n Atinya akk-d ibeṛṛaniyen i gzedɣen dinna, ḥemmlen ad sɛeddin lweqt-nsen anagar deg umeslay d usmeḥses i lexbaṛat ijdiden i d-ițeffɣen.  </p> 
 <p><a name="an_17:22">17:22</a> Bulus ibedd di tlemmast n Laryufaj yenna : Ay irgazen n temdint n Atinya, ẓriɣ-kkun tẓewṛem yerna tettabaɛem ddin di mkul ḥaǧa; </p> 
 <p><a name="an_17:23">17:23</a> axaṭer ḥewwseɣ akk tamdint-nwen, walaɣ lmeṣnuɛat akk i tɛebdem ; ufiɣ daɣen yiwen n idebder anda yura : « I Ṛebbi-nni ur nețwassen ara. » Win akken i tɛebbdem mbla ma tessnem-t, d nețța i wen-țbecciṛeɣ. </p> 
 <p><a name="an_17:24">17:24</a> D nețța i d-ixelqen ddunit d wayen yellan deg-s, d Bab igenwan d lqaɛa, ur yezdiɣ ara di leǧwameɛ i bnan ifassen n yemdanen. </p> 
 <p><a name="an_17:25">17:25</a> Ur yeḥwaǧ ara ad xedmen fell-as yemdanen am akken yella wayen i t-ixuṣṣen, nețța i d-ițțaken i yemdanen meṛṛa tudert, ṛṛuḥ d wayen akk yellan.  </p> 
 <p><a name="an_17:26">17:26</a> Seg yiwen wemdan i d-ixleq akk leǧnas, yeɛmeṛ yis-sen ddunit ; yefka-yasen lewqat i tudert-nsen akk-d tilisa i tmura-nsen.  </p> 
 <p><a name="an_17:27">17:27</a> Sidi Ṛebbi ixdem annect-agi meṛṛa iwakken ad nadin fell-as ; ahat a t-afen s usferfed, nețța ur nebɛid ara ɣef mkul yiwen deg-nneɣ.  </p> 
 <p><a name="an_17:28">17:28</a> Yis i nesɛa tudert, i nteddu, i nețțili ; d ayen i d-nnan daɣen kra seg imedyazen-nwen : « ula d nukkni, darraw-is ! »  </p> 
 <p><a name="an_17:29">17:29</a> Ihi imi nella d arraw n Sidi Ṛebbi, ur ilaq ara a nɣil belli Ṛebbi yețțemcabi ɣer lmeṣnuɛ n ddheb, neɣ n lfeṭṭa, neɣ n wezṛu yețwanejṛen s ufus ț-țmusni n wemdan.  </p> 
 <p><a name="an_17:30">17:30</a> Sidi Ṛebbi isemmeḥ-asen i yemdanen asmi akken ur t-ssinen ara; tura issawel-asen-d akk di mkul amkan iwakken ad beddlen tikli, a d-uɣalen ɣuṛ-es. </p> 
 <p><a name="an_17:31">17:31</a> Axaṭer yextaṛ ass i deg ara iḥaseb imdanen s lḥeqq, s wemdan i gextaṛ yerna yefka-yaɣ-d akk țțbut mi t-id-isseḥya si ger lmegtin.  </p> 
 <p><a name="an_17:32">17:32</a> Mi slan iheddeṛ-ed ɣef ḥeggu n lmegtin, kra țmesxiṛen, wiyaḍ qqaṛen-as : A k-nḥess ɣef wannect-agi tikkelt nniḍen...  </p> 
 <p><a name="an_17:33">17:33</a> Imiren, Bulus iṛuḥ yeǧǧa-ten. </p> 
 <p><a name="an_17:34">17:34</a> Ɣas akken aṭas i gețmesxiṛen, llan kra deg-sen umnen, ddan yid-es . Yella gar-asen Dunisus i gțekkin di tejmaɛt n Laryufaj, yiwet n tmeṭṭut isem-is Damaris akk-d wiyaḍ.  </p> 
 <p><a name="an_18:1">18:1</a> Bulus iṛuḥ si temdint n Atinya ɣer temdint n Kurintus. </p> 
 <p><a name="an_18:2">18:2</a> Yufa dinna yiwen wergaz n wat Isṛail, isem-is Akilas, laṣel-is n tmurt n Tqenṭert ; yewweḍ-ed deg ussan-nni kan si tmurt n Selyan nețța ț-țmeṭṭut-is Brisila, axaṭer Qayṣer Kludyus yumeṛ i wat Isṛail meṛṛa ad ffɣen si temdint n Ṛuma.  </p> 
 <p><a name="an_18:3">18:3</a> Imi Brisila d Akilas xeddmen iqiḍunen am Bulus, yeqqim ɣuṛ-sen xeddmen s tdukli. </p> 
 <p><a name="an_18:4">18:4</a> Mkul ass n westeɛfu, Bulus ițmeslay di lǧameɛ, ițbecciṛ i wat Isṛail akk-d iyunaniyen ttabaɛen ddin n wat Isṛail, iwakken ad amnen.  </p> 
 <p><a name="an_18:5">18:5</a> Mi d-usan Silas d Timuti si tmurt n Masidunya, Bulus yuɣal ițțelhay kan d ubecceṛ n wawal n Ṛebbi, ițbeggin ed i wat Isṛail belli d Ɛisa i d Lmasiḥ. </p> 
 <p><a name="an_18:6">18:6</a> Imi i t-xulfen yerna regmen-t, Bulus yezwi icuḍaḍ-is, yenna-yasen : Cfut ur teqbilem ara leslak-agi, ddnub i yirawen-nwen ! Nekk ayen i yi-d-yewwi lḥal xedmeɣ-t. Ihi sya d asawen, ad ṛuḥeɣ ɣer wat leǧnas ur nelli ara seg warraw n wat Isṛail. </p> 
 <p><a name="an_18:7">18:7</a> Mi d-iffeɣ syenna, ikcem ɣer wexxam n Tit Justus, i gellan d amdan iḍuɛen Sidi Ṛebbi ; axxam-is yezga-d tama n lǧameɛ n wat Isṛail. </p> 
 <p><a name="an_18:8">18:8</a> Krisbus, ameqqran n lǧameɛ yumen s Ɛisa Lmasiḥ nețța d wat wexxam-is meṛṛa ; aṭas n wat kurintus daɣen i gumnen mi slan s wayen i d-yenna Bulus, dɣa țwaɣeḍsen deg waman.  </p> 
 <p><a name="an_18:9">18:9</a> Yiwen n yiḍ, Sidi Ṛebbi iweḥḥa yas-ed di targit i Bulus yenna-yas-d : Ur țțaggad ara, kemmel ehdeṛ ur țsusum ara ! </p> 
 <p><a name="an_18:10">18:10</a> Axaṭer aqli-n yid-ek ! Yiwen ur izmir ad issers afus-is fell-ak iwakken a k-yexdem cceṛ, axaṭer aṭas i gellan d ayla-w di temdint-agi.  </p> 
 <p><a name="an_18:11">18:11</a> Bulus yeqqim aseggas d sețța wagguren ger wat kurintus, isselmad asen awal n Sidi Ṛebbi.  </p> 
 <p><a name="an_18:12">18:12</a> Di lweqt-nni i deg yella Galyun d lḥakem ɣef tmurt n Lakaya, at Isṛail msefhamen, ddukklen ɣef Bulus, wwin-t ɣer wexxam n ccṛeɛ, nnan :  </p> 
 <p><a name="an_18:13">18:13</a> Argaz-agi issamaṛ lɣaci ad ɛebden Ṛebbi s webrid ixulfen ccariɛa.  </p> 
 <p><a name="an_18:14">18:14</a> Mi gekker Bulus ad ihdeṛ, inṭeq Galyun yenna i wat Isṛail : Lemmer d lbaṭel neɣ d ayen n diri i gexdem, zemreɣ a wen ḥesseɣ am akken yella di lqanun ay at Isṛail ; </p> 
 <p><a name="an_18:15">18:15</a> meɛna imi d amjadel ɣef yimeslayen, ɣef yismawen akk-d ccariɛa n Musa, ayagi d ccɣel-nwen ! Ur bɣiɣ ara ad ḥekmeɣ ɣef wannect-a. </p> 
 <p><a name="an_18:16">18:16</a> Imiren issufeɣ-iten-id syenna. </p> 
 <p><a name="an_18:17">18:17</a> Dɣa lɣaci mmɣen ɣef Sustin, ameqqran n lǧameɛ n wat Isṛail, ewten-t zdat wexxam n ccṛeɛ , Galyun ur s-tewqiɛ ara lmeɛna.  </p> 
 <p><a name="an_18:18">18:18</a> Bulus yerna aṭas n wussan di temdint n Kurintus ; syenna yeǧǧa atmaten di lehna, yerkeb lbabuṛ ɣer tmurt n Surya ; Brisila d Akilas ddan yid-es. Uqbel ad iṛuḥ si lmeṛṣa n temdint n Senkriya, iseṭṭel aqeṛṛuy-is axaṭer yefka lemɛahda i Sidi Ṛebbi.  </p> 
 <p><a name="an_18:19">18:19</a> Mi wwḍen ɣer temdint n Ifasus, mfaṛaqen : Bulus yeǧǧa iṛfiqen-is Akilas d Brisila, ma d nețța ikcem ɣer lǧameɛ n wat Isṛail, ițmeslay yid-sen.  </p> 
 <p><a name="an_18:20">18:20</a> Ḥellelen-t ad yernu kra wussan yid-sen, lameɛna ur yeqbil ara. </p> 
 <p><a name="an_18:21">18:21</a> Yenna-yasen : A d-uɣaleɣ ɣuṛ-wen ma yebɣa Ṛebbi.  </p> 
 <p><a name="an_18:22">18:22</a> Yerkeb di lbabuṛ si temdint n Ifasus ɣer temdint n Qiṣarya ; syenna iṛuḥ a d-iẓer tajmaɛt n watmaten yellan di temdint n Lquds ; imiren iṣubb ɣer temdint n Antyuc. </p> 
 <p><a name="an_18:23">18:23</a> Mi gesɛedda kra n wussan di temdint n Antyuc, yuɣal yeṭṭef abrid ɣer tmura n Galasya d Frijya, di mkul amkan yesseǧhad inelmaden di Liman.  </p> 
 <p><a name="an_18:24">18:24</a> Di lweqt-nni, yiwen wergaz n wat Isṛail yețțusemman Abulus, ilulen di temdint n Skandriya yellan di tmurt n Maṣeṛ, yusa-d ɣer temdint n Ifasus ; yezweṛ deg wawal yerna yessen mliḥ tira iqedsen. </p> 
 <p><a name="an_18:25">18:25</a> Argaz-agi yelmed abrid n Sidi Ṛebbi, iǧhed di liman, ițbecciṛ, yesselmad s teḥqiq ayen i d-qqaṛent tira iqedsen ɣef Ɛisa Lmasiḥ , ɣas akken d aɣḍas n Yeḥya kan i gessen.  </p> 
 <p><a name="an_18:26">18:26</a> Yebda ițmeslay ɛinani di lǧameɛ n wat Isṛail. Akilas d Brisila mi s-slan wwin-t yid-sen, sfehmen-as tideț n webrid n Sidi Ṛebbi akken tella. </p> 
 <p><a name="an_18:27">18:27</a> Imi i gebɣa Abulus ad iɛeddi ɣer tmurt n Akaya, atmaten kksen-as akukru, uran tabṛaț i inelmaden n temdint n Kurintus iwakken ad stṛeḥben yis. Mi gewweḍ ɣer dinna, s ṛṛeḥma n Sidi Ṛebbi iɛawen aṭas widak yumnen ; </p> 
 <p><a name="an_18:28">18:28</a> axaṭer yețṛuẓu ɛinani awal n lecyux n wat Isṛail zdat lɣaci meṛṛa ; yerna ițbeggin-ed s tira iqedsen belli d Ɛisa i d Lmasiḥ.  </p> 
 <p><a name="an_19:2">19:2</a> Yers-ed fell-awen Ṛṛuḥ iqedsen asmi i tumnem ? Rran-as : Leɛmeṛ nesli s Ṛṛuḥ-agi iqedsen.  </p> 
 <p><a name="an_19:3">19:3</a> Yenna-yasen : Anwa aɣḍas i-s-tețwaɣeḍsem ihi ? Rran-as : S weɣḍas n Yeḥya.  </p> 
 <p><a name="an_19:4">19:4</a> Bulus yenna-yasen : Yeḥya yesseɣḍes wid akk iqeblen ad beddlen tikli, yeqqaṛ i wegdud ad amnen s win ara d-yasen deffir-es, yeɛni s Sidna Ɛisa.  </p> 
 <p><a name="an_19:5">19:5</a> Mi slan i yimeslayen-agi, țwaɣeḍsen s yisem n Sidna Ɛisa. </p> 
 <p><a name="an_19:6">19:6</a> Akken yessers Bulus ifassen-is fell-asen, ikcem-iten Ṛṛuḥ iqedsen, bdan heddṛen s tutlayin ur nețwassen ara, yerna țcaren-d ayen i sen-d-yusan s ɣuṛ Sidi Ṛebbi. </p> 
 <p><a name="an_19:7">19:7</a> Imdanen-agi ad ilin di tnac yid-sen.  </p> 
 <p><a name="an_19:8">19:8</a> Dɣa Bulus ikcem ɣer lǧameɛ n wat Isṛail. Azal n tlata wagguren ițmeslay yid-sen, ițqenniɛ-iten iwakken ad amnen s wayen yeɛnan tagelda n Sidi Ṛebbi. </p> 
 <p><a name="an_19:9">19:9</a> Lameɛna kra deg-sen yeqquṛ wul-nsen ugin ad amnen, rnan kkaten deg webrid n Sidna Ɛisa zdat lɣaci. Mi gwala annect-agi, Bulus iṭṭaxeṛ-asen, yewwi yid-es inelmaden, isselmad-iten kull ass deg wexxam n yiwen wergaz isem-is Tiranus.  </p> 
 <p><a name="an_19:10">19:10</a> Yeqqim dinna sin iseggasen armi imezdaɣ meṛṛa n tmurt n Asya, ama d at Isṛail ama d iyunaniyen, slan i wawal n Sidi Ṛebbi.  </p> 
 <p><a name="an_19:11">19:11</a> Sidi Ṛebbi ixeddem lbeṛhanat imeqqranen s ifassen n Bulus ; </p> 
 <p><a name="an_19:12">19:12</a> lɣaci wwḍen armi srusun ɣef yimuḍan ula ț-țimucwaṛin neɣ iceṭṭiḍen yennulen Bulus, dɣa imuḍan ḥellun, wid akk yețțumelken țeffɣen ten leǧnun.  </p> 
 <p><a name="an_19:13">19:13</a> Kra iseḥḥaren n wat Isṛail, ileḥḥun seg wemkan ɣer wayeḍ, ɛerden ad ssuffɣen leǧnun seg wid yețwamelken, qqaṛen-asen : S yisem n Sidna Ɛisa, win akken i gețbecciṛ Bulus, effɣet seg-es !  </p> 
 <p><a name="an_19:14">19:14</a> Widak ixeddmen annect-agi llan di sebɛa yid-sen, d arraw n Ciba yellan d yiwen si lmuqedmin imeqqranen n at Isṛail. </p> 
 <p><a name="an_19:15">19:15</a> Lǧen-nni yerra-yasen-d : Ɛisa ssneɣ-t, ssneɣ daɣen anwa i d Bulus, ma d kunwi d acu-kkun ?  </p> 
 <p><a name="an_19:16">19:16</a> Argaz-nni i gezdeɣ lǧen, izḍem fell-asen yekkat-iten armi rewlen seg wexxam d iɛeryanen, yerna jerḥen. </p> 
 <p><a name="an_19:17">19:17</a> At Isṛail akk-d iyunaniyen meṛṛa izedɣen tamdint n Ifasus slan s wayagi, ikcem-iten meṛṛa lxuf, ḥemmden isem n Sidna Ɛisa.  </p> 
 <p><a name="an_19:18">19:18</a> Aṭas seg wid i gumnen, usan-d ad qiṛṛen zdat wiyaḍ, ḥkan-d ayen xedmen.  </p> 
 <p><a name="an_19:19">19:19</a> Aṭas seg wid yellan xeddmen ssḥur, wwin-d tiktabin-nsen s wayes xeddmen ssḥur, sseṛɣen-tent zdat medden. Ufan belli azal n xemsin alef n twiztin i d ssuma-nsent. </p> 
 <p><a name="an_19:20">19:20</a> S wakka i gewweḍ wawal n Sidi Ṛebbi ɣer mkul amkan, ițnerni s tezmert n Sidi Ṛebbi. </p> 
 <p><a name="an_19:21">19:21</a> Mi gɛedda wayagi akk, Bulus yeqsed ad iṛuḥ ɣer temdint n Lquds, ad iɛeddi si tmura n Masidunya akk-d Akaya ; ixemmem deg iman-is yenna : « M'ara awḍeɣ ɣer dinna, ilaq-iyi daɣen ad ṛuḥeɣ ɣer temdint n Ṛuma.»  </p> 
 <p><a name="an_19:22">19:22</a> Iceggeɛ ɣer tmurt n Masidunya sin seg imɛawnen-is, Timuti d Irastus ; ma d nețța yerna kra n wussan di tmurt n Asya.  </p> 
 <p><a name="an_19:23">19:23</a> Di lweqt-nni, ɣef ddemma n webrid n Sidi Ṛebbi ikker ccwal d ameqqran di temdint n Ifasus. </p> 
 <p><a name="an_19:24">19:24</a> Yella yiwen uḥeddad isem-is Dimitriyus, iṣenneɛ s lfeṭṭa leǧwameɛ imecṭaḥ i d-yețmettilen lǧameɛ n Artimis, yesserbeḥ aṭas ixeddamen-is. </p> 
 <p><a name="an_19:25">19:25</a> Isnejmaɛ-iten nutni d wid i gxeddmen lfeṭṭa am nețța, yenna yasen : Ay atmaten ! Teẓram belli rrbeḥ-nneɣ itekk-ed si lxedma-agi ; </p> 
 <p><a name="an_19:26">19:26</a> teẓram yerna teslam daɣen ɣef Bulus-agi issamnen aṭas n lɣaci di temdint n Ifasus d waṭas n tmura n Asya. Yessamen lɣaci, yewwi-ten d webrid-is mi sen-iqqaṛ : « Iṛebbiten agi ițwaxedmen s ifassen n wemdan mačči d iṛebbiten n tideț » ; </p> 
 <p><a name="an_19:27">19:27</a> ur nuggad ara kan aț-țeɣli ṣṣenɛa-nneɣ, lameɛna nuggad ad sseɣlin lqima i lǧameɛ n Artimis tameqqrant, taṛebbiț-nneɣ, qṛib a s-kksen lɛaḍima s wayes tețwaɛbed di tmurt n Asya d ddunit meṛṛa.  </p> 
 <p><a name="an_19:28">19:28</a> Mi slan i lehduṛ-agi, ikcem-iten wurrif, bdan țɛeggiḍen : Ț-țameqqrant Artimis, taṛebbiț n at Ifasus !  </p> 
 <p><a name="an_19:29">19:29</a> Tețțucewwel akk temdint, lɣaci uzzlen ɣer wexxam n umezgun, seddan yid-sen Gayus akk-d Aristark yellan d imasiduniyen, d iṛfiqen n Bulus, ițṣafaṛen yid-es. </p> 
 <p><a name="an_19:30">19:30</a> Bulus yebɣa ad ikcem di tlemmast n lɣaci-nni, lameɛna ugin-as inelmaden. </p> 
 <p><a name="an_19:31">19:31</a> Ula d kra n lḥukkam imeqqranen n tmurt n Asya yellan d imdukkal-is, ceggɛen ɣuṛ-es iwakken a t-ḥellelen ur ițṛuḥ ara ɣer wexxam n umezgun. </p> 
 <p><a name="an_19:32">19:32</a> Anejmaɛ-nni yexṛeb, kra țɛeggiḍen akka, wiyaḍ akken nniḍen yerna aṭas deg-sen ur ẓrin ara acu i d sebba n unejmaɛ-nni.  </p> 
 <p><a name="an_19:33">19:33</a> At Isṛail sbedden-d Alixandru ad immeslay i lɣaci. Iwehha-yasen ufus-is ad ssusmen, iwakken a d-ihdeṛ, </p> 
 <p><a name="an_19:34">19:34</a> lameɛna mi t-ɛeqlen n wat Isṛail i gella, bdan țɛeggiḍen meṛṛa ɣef tikkelt azal n snat sswayeɛ : Ț-țameqqrant Artimis n at Ifasus !  </p> 
 <p><a name="an_19:35">19:35</a> Lameɛna lamin n temdint ihedden lɣaci, yenna-yasen : Ay at Ifasus, anwa ur neẓri ara belli tamdint n Ifasus ț-țaɛessast n lǧameɛ Artimis tameqqrant akk-d lmeṣnuɛ-ines i d-iɣlin seg igenni ?  </p> 
 <p><a name="an_19:36">19:36</a> Imi ayagi yiwen ur yezmir a t-yenkeṛ, ilaq ihi aț-țhennim iman-nwen, ur xeddmet acemma s tuffɣa n leɛqel ! </p> 
 <p><a name="an_19:37">19:37</a> Axaṭer irgazen-agi i d-tewwim ur ukiren lǧameɛ, ur rgimen Artimis taṛebbiț-nneɣ.  </p> 
 <p><a name="an_19:38">19:38</a> Ihi ma yella Dimitriyus d imdukkal-is sɛan sebba n uccetki ɣef yiwen, llan wussan n ccṛeɛ am akken llan lḥukkam ara ten-icaṛɛen , ad ṛuḥen ad ccetkin fell-asen. </p> 
 <p><a name="an_19:39">19:39</a> Ma yella wayen nniḍen i tessuturem deg-sen, ad ḥekmen fell-asen deg unejmaɛ zdat lḥukkam. </p> 
 <p><a name="an_19:40">19:40</a> Atan nuggad ad ccetkin fell-aɣ ɣef wayen yedṛan ass-agi imi ulac sebba ara nefk ɣef wunejmaɛ-agi. Mi gfukk ameslay, iserreḥ i lɣaci, yefra unejmaɛ.  </p> 
 <p><a name="an_20:1">20:1</a> Mi tethedden lḥala ifuk ccwal, imiren Bulus isnejmaɛ-ed inelmaden, yenha-ten, dɣa yeǧǧa-ten di lehna, yeṭṭef abrid ɣer tmurt n Masidunya. </p> 
 <p><a name="an_20:2">20:2</a> Mi gezger tamurt n Masidunya, inehhu inelmaden, isseǧhad-iten s waṭas yimeslayen ; syenna ikemmel abrid ɣer tmurt n iyunaniyen anda yeqqim tlata wagguren. </p> 
 <p><a name="an_20:3">20:3</a> Mi gțeddu ad yerkeb di lbabuṛ ɣer tmurt n Surya, yesla belli at Isṛail mcawaṛen iwakken a t-xedɛen ; dɣa ibeddel ṛṛay, iṭṭef abrid ad yuɣal ɣer tmurt n Masidunya.  </p> 
 <p><a name="an_20:4">20:4</a> Ddan yid-es : Subatir, mmi-s n Birus si temdint n Biri ; Arisṭark akk-d Skundus n temdint n Tiṣalunik, Gayus n temdint n Derba ; yedda daɣen Timuti, Tucik akk-d Trufim n tmurt n Asya. </p> 
 <p><a name="an_20:5">20:5</a> Wigi zwaren ɣer zdat, eṛǧan-aɣ di temdint n Truwas, </p> 
 <p><a name="an_20:6">20:6</a> ma d nukni mi gɛedda Lɛid n weɣṛum mbla iɣes ( tamtunt ), nerkeb lbabuṛ si temdint n Filibus ; xemsa wussan mbeɛd, nelḥeq-iten ɣer temdint n Truwas, anda nesɛedda sebɛa wussan.  </p> 
 <p><a name="an_20:7">20:7</a> Ass amezwaru n dduṛt nennejmaɛ a nebḍu aɣṛum iwakken ad nemekti Lmasiḥ, Bulus ițmeslay nețța d inelmaden. Imi ilaq ad iṛuḥ azekka-nni, ikemmel yid-sen awal armi i gneṣṣef yiḍ ; </p> 
 <p><a name="an_20:8">20:8</a> taɣuṛfeț anda i nennejmaɛ, ceɛlent deg-s aṭas n teftilin. </p> 
 <p><a name="an_20:9">20:9</a> Yiwen ilemẓi, isem-is Utikus, yella yeqqim ɣef yiri n ṭṭaq. Imi i gɛeṭṭel Bulus deg umeslay, Utikus yewwi-t nuddam ; yeɣli-d si leɛli wis tlata armi d lqaɛa, mi t-id-refden, ufan-t yemmut. </p> 
 <p><a name="an_20:10">20:10</a> Bulus iṣubb-ed, yekna ɣuṛ-es, iddem-it ger iɣallen-is yenna : Ur țțaggadet ara, mazal deg-s taṛwiḥt, idder.  </p> 
 <p><a name="an_20:11">20:11</a> Syenna yuɣal yuli ar tɣuṛfeț, yebḍa aɣṛum ; mi ččan, ikemmel yid-sen ameslay armi d lefjer, dɣa iṛuḥ. </p> 
 <p><a name="an_20:12">20:12</a> Ilemẓi-nni, wwin-t yedder ; dɣa yers-ed lfeṛḥ deg ulawen-nsen.  </p> 
 <p><a name="an_20:13">20:13</a> Nukni nerkeb lbabuṛ, nezwar Bulus ɣer temdint n Assus anda ara nemyagar yid-es axaṭer yebɣa ad iṛuḥ ɣef wuḍaṛ.  </p> 
 <p><a name="an_20:14">20:14</a> Mi i ɣ-d-yelḥeq di temdint n Assus, yerkeb yid-nneɣ di lbabuṛ ɣer temdint n Mitilan ; </p> 
 <p><a name="an_20:15">20:15</a> syenna nkemmel abrid di lebḥeṛ, newweḍ azekka-nni zdat temdint n Ciyu, sellazekka-nni newweḍ ɣer temdint n Samus ; ass wis ṛebɛa newweḍ ɣer temdint n Mili.  </p> 
 <p><a name="an_20:16">20:16</a> Bulus yebɣa ad iɛeddi rrif n temdint n Ifasus mbla ma yeḥbes, iwakken ur isṛuḥuy ara lweqt di tmurt n Asya. Ițɣawal iwakken ma yella wamek, ad yaweḍ ɣer temdint n Lquds ass n lɛid n wass wis xemsin.  </p> 
 <p><a name="an_20:17">20:17</a> Si temdint n Mili, Bulus iceggeɛ ɣer imeqqranen n tejmaɛt n temdint n Ifasus iwakken a d-asen. </p> 
 <p><a name="an_20:18">20:18</a> Mi d-wwḍen ɣuṛ-es, yenna-yasen : Tesnem si zik tikli-inu yid-wen seg wass amezwaru mi d-kecmeɣ tamurt n Asya. </p> 
 <p><a name="an_20:19">20:19</a> Qedceɣ ɣef Sidi Ṛebbi s wannuz d imeṭṭawen ger iɣeblan d lemḥayen i yi-sseṛwan wat Isṛail.  </p> 
 <p><a name="an_20:20">20:20</a> Țbecciṛeɣ-awen rniɣ slemdeɣ kkun ama zdat lɣaci ama deg ixxamen-nwen, mbla ma ffreɣ acemma seg wayen i kkun-inefɛen.  </p> 
 <p><a name="an_20:21">20:21</a> Țbecciṛeɣ i wat Isṛail d iyunaniyen a d-uɣalen ɣer webrid n Sidi Ṛebbi yerna ad amnen s Sidna Ɛisa. </p> 
 <p><a name="an_20:22">20:22</a> Tura, Ṛṛuḥ iqedsen iḥṛes-iyi ad ṭṭfeɣ abrid ɣer temdint n Lquds, ur ẓriɣ ara d acu ara yedṛun yid-i.  </p> 
 <p><a name="an_20:23">20:23</a> Si temdint ɣer tayeḍ, Ṛṛuḥ iqedsen ițxebbiṛ-iyi-d belli țṛaǧun-iyi leḥbus d wussan n ddiq.  </p> 
 <p><a name="an_20:24">20:24</a> Lameɛna tudert-iw ur s-fkiɣ ara lqima, ur ț-ḥsibeɣ ara eɛzizet fell-i ; a wi yufan kan ad kemmleɣ tikli-inu akk-d Ṛebbi, ad xedmeɣ s lfeṛḥ ayen i yi-d-ifka Sidna Ɛisa, iwakken ad beccṛeɣ lexbaṛ n lxiṛ  n ṛṛeḥma n Sidi Ṛebbi.  </p> 
 <p><a name="an_20:25">20:25</a> Tura ẓriɣ belli ur tețțuɣalem ara aț-țeẓrem udem-iw, kunwi iwumi beccṛeɣ tageldit n Sidi Ṛebbi.  </p> 
 <p><a name="an_20:26">20:26</a> Daymi i wen-d-qqaṛeɣ ass-agi, ma yella yiwen yeffeɣ i webrid n Sidi Ṛebbi, ddnub i yiri-s, </p> 
 <p><a name="an_20:27">20:27</a> imi i wen-beccṛeɣ ayen akk yellan di leqsed n Sidi Ṛebbi, ur ffireɣ ula d acemma fell-awen.  </p> 
 <p><a name="an_20:28">20:28</a> Ɛasset ɣef yiman-nwen, ɛasset ɣef tqeḍɛit ɣef i kkun-iwekkel Ṛṛuḥ iqedsen. Ilit d iɛessasen ɣef tejmaɛt n Ṛebbi i d-iḥerr s idammen n Mmi-s. </p> 
 <p><a name="an_20:29">20:29</a> Sriɣ belli m'ara ṛuḥeɣ, a d-kecmen gar-awen wuccanen iweɛṛen, wid ur nețsamaḥ ara i tqeḍɛit ; </p> 
 <p><a name="an_20:30">20:30</a> a d-kkren daɣen gar-awen yergazen ara islemden leḥwayeǧ iɛewjen iwakken ad jebden inelmaden ɣuṛ-sen.  </p> 
 <p><a name="an_20:31">20:31</a> Ɣuṛ-wat ihi, ɛasset iman-nwen ! Mmektit-ed belli azal n tlata iseggasen, ur ḥbiseɣ am yiḍ am ass, nehhuɣ mkul yiwen seg-wen s imeṭṭawen.  </p> 
 <p><a name="an_20:32">20:32</a> Tura a kkun-ǧǧeɣ ger ifassen n Sidi Ṛebbi d ṛṛeḥma n wawal-is; D nețța i gzemren a kkun-isseǧhed, a wen-d-yefk lweṛt i kunwi akk yellan d ayla-s. </p> 
 <p><a name="an_20:33">20:33</a> Ur ḍmiɛeɣ, ama d ddheb ama d idrimen neɣ leḥwayeǧ n yiwen deg-wen.  </p> 
 <p><a name="an_20:34">20:34</a> Teẓram s yiman-nwen belli s ifassen-iw i xeddmeɣ iwakken ad ɛeyyceɣ iman-iw akk-d wid yellan yid-i. </p> 
 <p><a name="an_20:35">20:35</a> Di mkul lḥaǧa, ssekneɣ-awen amek ara txedmem iwakken aț-țɛiwnem wid ur nezmir ara. Mmektit-ed imeslayen n Sidna Ɛisa mi d-yenna : « Win ițseddiqen ițțubarek akteṛ n win iteṭṭfen ssadaqa ».  </p> 
 <p><a name="an_20:36">20:36</a> Mbeɛd imeslayen-agi, yuɣal ɣef tgecrar, yedɛa ɣer Sidi Ṛebbi yid-sen. </p> 
 <p><a name="an_20:37">20:37</a> Sṭeṛḍqen s imeṭṭawen, ṭṭfen-t ger iɣallen-nsen, selmen akk fell-as ;  </p> 
 <p><a name="an_20:38">20:38</a> ḥeznen aṭas imi i sen-d-yenna : « Ur tețțuɣalem ara aț-țeẓrem udem- iw. » Syenna ddan yid-es armi d lbabuṛ.  </p> 
 <p><a name="an_21:1">21:1</a> Mi nemfaṛaq yid-sen, nerkeb lbabuṛ nṛuḥ qbala ɣer temdint n Kus, azekka-nni ɣer temdint n Ṛudus ; syenna nkemmel abrid ɣer temdint n Batara.  </p> 
 <p><a name="an_21:2">21:2</a> Mi nufa lbabuṛ ara izegren ɣer tmurt n Finisya, nerkeb deg-s nṛuḥ. </p> 
 <p><a name="an_21:3">21:3</a> Mi nqubel tigzirt n Qubṛus, neǧǧa-ț ɣer tama tazelmaṭ, nkemmel abrid nneɣ ɣer tmurt n Surya, newweḍ ɣer temdint n Sur imi dinna ara yessers lbabuṛ sselɛa-s.  </p> 
 <p><a name="an_21:4">21:4</a> Mi nufa dinna inelmaden, nesɛedda sebɛa wussan yid-sen ; s Ṛṛuḥ iqedsen, inelmaden-nni nnan-as i Bulus : Ur țțali ara ɣer temdint n Lquds.  </p> 
 <p><a name="an_21:5">21:5</a> Mi ɛeddan wussan-nni, nekker a nṛuḥ, ddan yid-nneɣ armi neffeɣ i temdint, nutni ț-țilawin-nsen d warraw-nsen. Mi newweḍ ɣer rrif n lebḥeṛ nuɣal ɣef tgecrar, nedɛa ɣer Sidi Ṛebbi.  </p> 
 <p><a name="an_21:6">21:6</a> Mi nemsalam yid-sen, nerkeb di lbabuṛ nṛuḥ ; nutni uɣalen ɣer yexxamen-nsen, </p> 
 <p><a name="an_21:7">21:7</a> nukkni nkemmel abrid s lbabuṛ si temdint n Sur armi ț-țamdint n Bṭulimays, neẓra dinna atmaten, neqqim yid-sen yiwen wass.  </p> 
 <p><a name="an_21:8">21:8</a> Azekka-nni nṛuḥ, newweḍ ɣer Qiṣarya ; nekcem ɣer wexxam n Filbas, win ițbecciṛen lexbaṛ n lxiṛ, nețța yellan d yiwen si sebɛa-nni ițwaxtaṛen di temdint n Lquds ; neqqim ɣuṛ-es.  </p> 
 <p><a name="an_21:9">21:9</a> Filbas-nni, ɣuṛ-es ṛebɛa yessi-s urɛad zwiǧent, țxebbiṛent-ed s wayen i tent-id-ițțasen s ɣuṛ Sidi Ṛebbi.  </p> 
 <p><a name="an_21:10">21:10</a> Ɛeddan kra n wussan, mi d yewweḍ yiwen seg watmaten isem-is Agabus, ula d nețța ițxebbiṛ-ed s ɣuṛ Sidi Ṛebbi, iṣubb-ed si tmurt n Yahuda, yusa-d ɣuṛ-nneɣ.  </p> 
 <p><a name="an_21:11">21:11</a> Iddem tabagust n Bulus, icudd idaṛṛen-is d ifassen-is, yenna :  Atan wayen i d-iqqaṛ Ṛṛuḥ iqedsen : akka ara cidden wat Isṛail di temdint n Lquds bab n tbagust-agi, a t-sellmen ger ifassen n ikafriwen.  </p> 
 <p><a name="an_21:12">21:12</a> Mi nesla annect-agi, nukkni d watmaten n Sizari nḥellel Bulus iwakken ur ițțali ara ɣer temdint n Lquds.  </p> 
 <p><a name="an_21:13">21:13</a> Dɣa Bulus yerra-yasen : Acu i kkun-yuɣen mi tețrum akka armi tgezmem tasa-w ? Nekk qebleɣ mačči kan ad țwacuddeɣ, lameɛna qebleɣ ula d lmut di temdint n Lquds ɣef ddemma n yisem n Sidna Ɛisa.  </p> 
 <p><a name="an_21:14">21:14</a> Imi yugi ad yaɣ awal-nneɣ, ur nketteṛ ara fell-as lehduṛ nenna : A neǧǧ lebɣi n Sidi Ṛebbi ad yedṛu !  </p> 
 <p><a name="an_21:15">21:15</a> Akken ɛeddan wussan-nni, nhegga iman-nneɣ a nali ɣer temdint n Lquds. </p> 
 <p><a name="an_21:16">21:16</a> Kra inelmaden n Qiṣarya ddan yid-nneɣ, wwin-aɣ ɣer yiwen wergaz wuɣuṛ ara nili, isem-is Mansun, n tegzirt n Qubṛus, d yiwen seg inelmaden imezwura.  </p> 
 <p><a name="an_21:17">21:17</a> Mi newweḍ ɣer temdint n Lquds, sṭreḥben yis-nneɣ watmaten s lfeṛḥ. </p> 
 <p><a name="an_21:18">21:18</a> Azekka-nni, yedda yid-nneɣ Bulus ɣer wexxam n Yeɛqub anda nnejmaɛen imeqqranen n tejmaɛt.  </p> 
 <p><a name="an_21:19">21:19</a> Mi gsellem fell-asen, Bulus yeḥka-yasen ayen akk ixdem yis Sidi Ṛebbi ger leǧnas ur nelli ara n wat Isṛail. </p> 
 <p><a name="an_21:20">21:20</a> Mi slan ayagi, ḥemmden Sidi Ṛebbi nnan-as : Twalaḍ a gma-tneɣ acḥal n luluf n wat Isṛail i gumnen s Sidna Ɛisa yerna ṭṭfen akk di ccariɛa n Musa.  </p> 
 <p><a name="an_21:21">21:21</a> Atan slan wat Isṛail n temdint n Lquds belli tesselmadeḍ i wat Isṛail yellan ger leǧnas ad ǧǧen ccariɛa n Musa ; slan daɣen teqqaṛeḍ asen : « ur seḍhaṛet ara i warrac nwen, ur ttabaɛet ara leɛwayed i ɣ-d-ǧǧan lejdud ».  </p> 
 <p><a name="an_21:22">21:22</a> Acu ara nexdem tura ? Mbla ccekk ad slen belli tusiḍ-ed ; </p> 
 <p><a name="an_21:23">21:23</a> ihi exdem ayen ara k-d-nini : llan gar-aneɣ ṛebɛa yergazen i gefkan lemɛahda i Sidi Ṛebbi,  </p> 
 <p><a name="an_21:24">21:24</a> awi-ten yid-ek, ssizdeg iman-ik yid-sen, seṛṛef fell-asen iwakken ad seṭṭlen iqeṛṛay-nsen ; s wakkagi ad ẓren belli ayen akk slan fell-ak d lekdeb yerna ad walin mazal teṭṭfeḍ di ccariɛa n Musa. </p> 
 <p><a name="an_21:25">21:25</a> Ma d at leǧnas i gumnen s Sidna Ɛisa, nura-yasen : Ad ṭṭixṛen i wučči n weksum ițțunefken d asfel i lmeṣnuɛat d ssadat, i tissit n idammen, i lmal yemmuṛḍsen akk-d yir tikli.  </p> 
 <p><a name="an_21:26">21:26</a> Azekka-nni, Bulus yewwi yid-es ṛebɛa yergazen-nni, issazdeg iman-is yid-sen, ikcem yid-sen ɣer wefrag n lǧameɛ iqedsen iwakken ad ixebbeṛ lmuqeddem ɣef lweqt i deg ara fakken wussan n uzizdeg-nsen, yeɛni ass i deg ara yefk mkul yiwen seg-sen lweɛda-ines.  </p> 
 <p><a name="an_21:27">21:27</a> Mi qṛib ad fakken sebɛa wussan-nni n usizdeg, at Isṛail n tmurt n Asya walan Bulus di lǧameɛ iqedsen. Sekkren ccwal ger lɣaci dɣa ṭṭfen-t. </p> 
 <p><a name="an_21:28">21:28</a> Bdan țɛeggiḍen : Ay at Isṛail, ɛiwnet-aɣ ! Atah wergaz-nni ițbecciṛen di mkul amkan i yemdanen meṛṛa. Iheddeṛ ayen n diri ɣef wegdud-nneɣ, ɣef wemkan-agi iqedsen, isselmad ayen ur teqbil ara ccariɛa-nneɣ, yewwi-d iyunaniyen ɣer lǧameɛ, issenǧes amkan-agi iqedsen !  </p> 
 <p><a name="an_21:29">21:29</a> Nnan-d ayagi axaṭer walan yid-es Trufim di temdint, ɣilen issekcem-it ɣer daxel n lǧameɛ iqedsen.  </p> 
 <p><a name="an_21:30">21:30</a> Ikker ccwal di temdint meṛṛa ; uzzlen-d yemdanen si mkul tama. Sṭfen Bulus, zzuɣṛen-t beṛṛa n wefrag n lǧameɛ iqedsen, sekkṛen-d tiwwura. </p> 
 <p><a name="an_21:31">21:31</a> Mi țeddun lɣaci a t-nɣen, yewweḍ lexbaṛ ɣer lqebṭan n lɛeskeṛ n Ṛuman tamdint n Lquds meṛṛa tenhewwal.  </p> 
 <p><a name="an_21:32">21:32</a> Imiren kan yessawel i ifesyanen d lɛeskeṛ-nsen, uzzlen ɣer lɣaci. Mi walan lqebṭan akk-d lɛeskeṛ-is, ttaxṛen-as i Bulus.  </p> 
 <p><a name="an_21:33">21:33</a> Dɣa lqebṭan iqeṛṛeb ɣer Bulus, yefka lameṛ a t-ṭṭfen, a t-cudden s snat snasel ; dɣa isteqsa anwa-t, d acu i gexdem. </p> 
 <p><a name="an_21:34">21:34</a> Meɛna lɣaci-nni, kra țɛeggiḍen akka, wiyaḍ akken nniḍen. Imi ur ifhim ara wayen i s-d-nnan si leɛyaḍ-nni, yumeṛ a t-awin ɣer lbeṛj.  </p> 
 <p><a name="an_21:35">21:35</a> Mi gewweḍ Bulus ɣer tseddaṛin, refden-t lɛeskeṛ iwakken a t-sellken si ger ifassen n lɣaci. </p> 
 <p><a name="an_21:36">21:36</a> Axaṭer lɣaci-nni ttabaɛen-t țɛeggiḍen : « Enɣ-it ! Enɣ-it !... »  </p> 
 <p><a name="an_21:37">21:37</a> Mi qṛib a t-skecmen ɣer zdaxel n lbeṛj, Bulus yenna-yas i lqebṭan : Semmeḥ-iyi, zemreɣ a k-d-iniɣ yiwen n wawal ? Yerra-yas : Tessneḍ aț-țhedṛeḍ tayunanit ? </p> 
 <p><a name="an_21:38">21:38</a> Mačči d kečč i d amaṣri-nni i gsekkren ccwal ussan-agi iɛeddan, i gessufɣen yid-es ṛebɛa alaf n iqettalen ɣer lxali ?  </p> 
 <p><a name="an_21:39">21:39</a> Bulus yenna-yas : Xaṭi ! Nekk n wat Isṛail, luleɣ di temdint n Sars yellan di tmurt n Silisya ; d amezdaɣ n temdint mechuṛen. Di leɛnaya-k serreḥ-iyi ad hedṛeɣ i lɣaci-yagi.  </p> 
 <p><a name="an_21:40">21:40</a> Lqebṭan iɛemmed-as, dɣa Bulus ibedd ɣef tseddaṛin, iwehha i lɣaci s ufus-is, dɣa ssusmen. Yenṭeq s tɛibṛanit, yenna-yasen :  </p> 
 <p><a name="an_22:1">22:1</a> Ay atmaten akken ma tellam dagi meẓẓi meqqeṛ, semḥesset ed tura a wen-d-iniɣ ayen s wayes ara dafɛeɣ ɣef yiman-iw !   </p> 
 <p><a name="an_22:2">22:2</a> Mi slan iheddeṛ-asen-d s tɛibṛanit, ssusmen meṛṛa iwakken a s-slen. Dɣa yenna :  </p> 
 <p><a name="an_22:3">22:3</a> Nekk n wat Isṛail, ț-țideț luleɣ di temdint n Sars n tmurt n Silisya, lameɛna țțuṛebbaɣ di temdint-agi n Lquds. Lliɣ d anelmad n ccix Gamalyel s yiman-is ; yesselmed-iyi akken ilaq amek ara tebɛeɣ ccariɛa i d-ǧǧan lejdud-nneɣ. Ttabaɛeɣ seg ul-iw abrid n Sidi Ṛebbi, am kunwi ass-agi.  </p> 
 <p><a name="an_22:4">22:4</a> Qehṛeɣ armi d lmut wid ittabaɛen abrid-agi ajdid n Sidna Ɛisa, am yergazen am tlawin, țțawiɣ-ten-id ɣer leḥbus țwarzen.  </p> 
 <p><a name="an_22:5">22:5</a> Lmuqeddem ameqqran ț-țejmaɛt n lɛuqal atnan d inigan ma bɣan a d-cehden ; d nutni s yiman-nsen i yi-d-yefkan tibṛatin i watmaten n Dimecq, syenna a d-awiɣ țwarzen ɣer temdint n Lquds, wid akk yumnen s webrid-agi, iwakken a ten-ɛaqben.  </p> 
 <p><a name="an_22:6">22:6</a> Lliɣ tedduɣ deg webrid, qṛib ad awḍeɣ ɣer temdint n Dimecq. Aț-țili d țnaṣfa n wass, taswiɛt kan, yiwet n tafat tameqqrant tewwet-ed seg igenni, tețfeǧǧiǧ, tezzi-yi-d.  </p> 
 <p><a name="an_22:7">22:7</a> Ɣliɣ ɣer lqaɛa, sliɣ i yiwen n ṣṣut i yi-d-iqqaṛen : Caɛul, a Caɛul, acuɣeṛ i yi tețqehhiṛeḍ ? </p> 
 <p><a name="an_22:8">22:8</a> Nniɣ-as : Anwa-k, a Sidi ? Yenna-yi-d : Nekk d Ɛisa Anaṣari, win akken i tețqehhiṛeḍ.  </p> 
 <p><a name="an_22:9">22:9</a> Wid yellan yid-i walan tafat-nni, lameɛna ur slin ara i ṣṣut i yi-d iheddṛen, dɣa nniɣ : Acu i tebɣiḍ a t-xedmeɣ a Sidi ?  </p> 
 <p><a name="an_22:10">22:10</a> Sidna Ɛisa yenna-yi-d : Ekker, ṛuḥ ɣer Dimecq, dinna ara k-d-inin ayen akk ilaqen a t-txedmeḍ.  </p> 
 <p><a name="an_22:11">22:11</a> Imi iyi-tesderɣel tafat-nni s ufeǧǧeǧ-ines, d imdukkal-iw i yi ṭṭfen afus armi wwḍeɣ ɣer Dimecq. </p> 
 <p><a name="an_22:12">22:12</a> Yella dinna yiwen wergaz isem-is Ananyas, d argaz iḍuɛen Sidi Ṛebbi, yerna yettabaɛ ccariɛa ; at Isṛail meṛṛa izedɣen di temdint n Dimecq țcehhiden fell-as s lxiṛ. </p> 
 <p><a name="an_22:13">22:13</a> Yusa-d ɣuṛ-i yenna-yi-d : A Caɛul a gma, Sidi Ṛebbi yerra-yak-d iẓri ! Di teswiɛt-nni, yuɣal-iyi-d yezṛi, dɣa walaɣ-t. </p> 
 <p><a name="an_22:14">22:14</a> Ananyas yenna-yi-d : Sidi Ṛebbi n lejdud-nneɣ yextaṛ ik aț-țissineḍ lebɣi-ines, aț-țwaliḍ s wallen-ik win yellan d Aḥeqqi, aț-țesleḍ imeslayen seg yimi-is . </p> 
 <p><a name="an_22:15">22:15</a> axaṭer aț-țuɣaleḍ d inigi-ines zdat yemdanen meṛṛa ɣef wayen akk i teẓriḍ d wayen iwumi tesliḍ.  </p> 
 <p><a name="an_22:16">22:16</a> Tura d acu i tețṛaǧuḍ ? Kker aț-tețwaɣeḍseḍ, aț-țizdigeḍ seg ddnubat-ik m'ara tedɛuḍ s yisem-is.  </p> 
 <p><a name="an_22:17">22:17</a> Asmi uɣaleɣ ɣer temdint n Lquds, yiwen n wass lliɣ țẓallaɣ deg wefrag n lǧameɛ iqedsen, iweḥḥa-yi-d Sidi Ṛebbi, </p> 
 <p><a name="an_22:18">22:18</a> walaɣ Sidna Ɛisa yenna-yi-d : Ffeɣ s lemɣawla si temdint n Lquds, axaṭer ur qebblen ara ayen s wayes ara tcehdeḍ fell-i. </p> 
 <p><a name="an_22:19">22:19</a> Nniɣ : A Sidi, ssnen-iyi ! Sran s yiman nsen belli ttabaɛeɣ si lǧameɛ ɣer wayeḍ wid yumnen yis-ek iwakken a ten-erreɣ ɣer lḥebs ad țewten. </p> 
 <p><a name="an_22:20">22:20</a> Ula d Stifan inigi-inek, mi ssuzlen idammen-is lliɣ dinna, qebleɣ at nɣen yerna ɛusseɣ leḥwayeǧ n wid i t-yenɣan.  </p> 
 <p><a name="an_22:21">22:21</a> Dɣa Sidi Ṛebbi yenna-yi-d : Ṛuḥ a k-ceggɛeɣ ɣer lebɛid ɣer wat leǧnas ur nelli ara n wat Isṛail.  </p> 
 <p><a name="an_22:22">22:22</a> Smeḥsisen-as armi d awal-agi imiren ɛeggḍen nnan : Ekkset si ddunit argaz am wagi ! Ur t-țaǧǧat ara ad yidir !  </p> 
 <p><a name="an_22:23">22:23</a> Țɛeggiḍen, ṭeggiṛen leḥwayeǧ nsen, ssafagen aɣebbaṛ ɣer igenni.  </p> 
 <p><a name="an_22:24">22:24</a> Lqebṭan n Ṛuman yumeṛ ad skecmen Bulus ɣer lbeṛj, a t-ewten s ujelkkaḍ ( s ucelliṭ ) iwakken ad issinen sebba ɣef wacu țɛeggiḍen fell-as. </p> 
 <p><a name="an_22:25">22:25</a> Mi t-cudden s tɣeggadin, Bulus yenna i lqebṭan yellan dinna : Eɛni iserreḥ-awen lqanun aț țewtem aṛumani s ujelkkaḍ uqbel ad iɛeddi di ccṛeɛ ?  </p> 
 <p><a name="an_22:26">22:26</a> Ɣef yimeslayen-agi, lqebṭan-nni iṛuḥ ɣer wemdebbar ameqqran yenna-yas : Ḥader aț-țewteḍ argaz-agi ! D aṛumani !  </p> 
 <p><a name="an_22:27">22:27</a> Amdebbar ameqqran mi d-yusa ɣer Bulus, yenna-yas : Ț-țideț kečč d aṛumani ? Ih, d aṛumani i lliɣ !  </p> 
 <p><a name="an_22:28">22:28</a> Amdebbar ameqqran yenna : Nekk aṭas n yedrimen i fkiɣ iwakken ad uɣaleɣ d aṛumani. Bulus yerra-yas : Nekk, seg wasmi i d-luleɣ i rseɣ di lekwaɣeḍ d aṛumani.  </p> 
 <p><a name="an_22:29">22:29</a> Imiren widak i d-ikkren a t wten, ṭṭaxṛen-as. Lqebṭan ameqqran s yiman-is yuggad mi geẓra Bulus d aṛumani yerna icudd-it.  </p> 
 <p><a name="an_22:30">22:30</a> Azekka-nni, imi i gebɣa ad yissin wemdebbar ameqqran ɣef wacu i d-ccetkan fell-as wat Isṛail, iserreḥ i Bulus, yumeṛ i lmuqedmin akk-d usqamu n ccṛeɛ ad nnejmaɛen. Dɣa yewwi-d Bulus isbedd-it-id zdat-nsen.  </p> 
 <p><a name="an_23:2">23:2</a> Lmuqeddem ameqqran Ananyas yumeṛ i wid yellan zdat-es, a t-wten ɣer yimi. </p> 
 <p><a name="an_23:3">23:3</a> Dɣa Bulus yenna : A k-id-iwet Ṛebbi a bu sin wudmawen ! Teqqimeḍ aț-țḥekmeḍ fell-i s ccariɛa, lameɛna kečč s yiman-ik txulfeḍ-ț imi tumreḍ ad iyi wten.  </p> 
 <p><a name="an_23:4">23:4</a> Widak yellan zdat-es nnan-as : Tregmeḍ lmuqeddem ameqqran n Ṛebbi !  </p> 
 <p><a name="an_23:5">23:5</a> Bulus yenna-yasen : Ur ẓriɣ ara ay atmaten belli d nețța i d lmuqeddem ameqqran, axaṭer yura di ccariɛa : Ur țekkes ara leqdeṛ ɣef lḥakem n wegdud-ik.  </p> 
 <p><a name="an_23:6">23:6</a> Bulus, mi gwala deg unejmaɛ-nni azgen deg-sen d isaduqiyen, azgen nniḍen d ifariziyen, iɛeggeḍ gar-asen, yenna : Ay atmaten, nekk d afarizi, d mmi-s n ufarizi ; ɣef ddemma n usirem i nesɛa di ḥeggu n lmegtin i d-beddeɣ ɣer ccṛeɛ ass-agi !  </p> 
 <p><a name="an_23:7">23:7</a> Mi d-yenna ayagi, ikker umennuɣ ger ifariziyen d isaduqiyen ; dɣa yebḍa unejmaɛ-nni. </p> 
 <p><a name="an_23:8">23:8</a> Axaṭer isaduqiyen qqaṛen ulac ḥeggu, ulac lmalayekkat, ulac leṛwaḥ ; ma d ifariziyen, nutni umnen s wannect agi.  </p> 
 <p><a name="an_23:9">23:9</a> Ikker umennuɣ gar-asen ; kra n lɛulama n ccariɛa n tejmaɛt n ifariziyen kkren iwakken ad ḥudden Bulus, nnan : Ur nufi ara ayen n diri deg wemdan-agi. Yezmer lḥal ihdeṛ-as-d Ṛebbi neɣ lmelk !  </p> 
 <p><a name="an_23:10">23:10</a> Imi ițțimɣuṛ umenɣi, lqebṭan ameqqran yugad ad feṛqen Bulus d iftaten. Yumeṛ i lɛeskeṛ ad ṛuḥen a t-id-kksen si tlemmast-nsen, a t awin ɣer lbeṛj.  </p> 
 <p><a name="an_23:11">23:11</a> Azekka-nni deg iḍ, Sidna Ɛisa iḍheṛ-as-d i Bulus yenna-yas : Ur țțaggad ara, axaṭer am akken tcehdeḍ fell-i di temdint n Lquds, ilaq daɣen aț-țcehdeḍ fell-i di temdint n Ṛuma.  </p> 
 <p><a name="an_23:12">23:12</a> Mi guli wass, at Isṛail xedmen tiḥileț, msefhamen, gullen ur ččin ur swin alamma nɣan Bulus.  </p> 
 <p><a name="an_23:13">23:13</a> Wid akk i s-ixedmen taxazabit agi, llan akteṛ n ṛebɛin yid-sen.  </p> 
 <p><a name="an_23:14">23:14</a> Ṛuḥen ad ẓren imeqqranen d lmuqedmin akk-d lɛuqal nnan asen : Neggul s limin ameqqran, ur nečči acemma alamma nenɣa Bulus.  </p> 
 <p><a name="an_23:15">23:15</a> Tura ihi, init-as i lqebṭan ameqqran a t-id-yawi ɣer unejmaɛ n ccṛeɛ zdat-wen, amzun tebɣam a s-tɛiwdem abeḥḥet ; ma d nukkni, a nheggi iman-nneɣ a t-nenɣ uqbel a d-yaweḍ ɣer dagi.  </p> 
 <p><a name="an_23:16">23:16</a> Lameɛna mmi-s n weltma-s n Bulus imi gesla s tḥileț-nni i s heggan, iṛuḥ ɣer lbeṛj anda yețwaḥbes Bulus, a s-yessiweḍ lexbaṛ.  </p> 
 <p><a name="an_23:17">23:17</a> Bulus yessawel-ed i yiwen si lfesyanat, yenna-yas : Awi aqcic-agi ɣer lqebṭan ameqqran, axaṭer yesɛa awal ara s-yini.  </p> 
 <p><a name="an_23:18">23:18</a> Lfesyan yewwi yid-es aqcic-nni ɣer lqebṭan ameqqran, yenna-yas : Bulus ameḥbus-nni yessawel iyi-d iḥellel-iyi iwakken a k-d-awiɣ aqcic-agi, yesɛa awal ara k-d-yini.  </p> 
 <p><a name="an_23:19">23:19</a> Lqebṭan ameqqran iddem aqcic-nni seg ufus, ɛezlen iman-nsen ɣer ṭṭeṛf ; yenna-yas : D acu i tebɣiḍ a yi-t-id-tiniḍ ?  </p> 
 <p><a name="an_23:20">23:20</a> Yerra-yas : At Isṛail msefhamen a k-d ḍelben aț-țawiḍ Bulus azekka ɣer zdat unejmaɛ n ccṛeɛ, ufan-d sebba a s-ɛiwden abeḥḥet ɣef temsalt-is. </p> 
 <p><a name="an_23:21">23:21</a> Ur țțaɣ ara awal-nsen, axaṭer akteṛ n ṛebɛin deg-sen ḥeyylen-as, gullen s limin ameqqran ur tețțen ur tessen alamma nɣan-t. Tura heggan iman-nsen, țṛaǧun anagar lebɣi-k.  </p> 
 <p><a name="an_23:22">23:22</a> Amdebbar ameqqran iserreḥ-as i weqcic-nni ad iṛuḥ, yenna-yas : Ur qqaṛ i yiwen belli tessaweḍed iyi-d lexbaṛ-agi.  </p> 
 <p><a name="an_23:23">23:23</a> Yuɣal issawel i sin lfesyanat, yenna-yasen : Heggit mitin iɛeskṛiwen, sebɛin yemnayen akk-d mitin iɛeskṛiwen i gteddun s imezṛagen, ad ṛuḥen ɣef tesɛa n yiḍ ɣer Qiṣarya. </p> 
 <p><a name="an_23:24">23:24</a> Heggit daɣen iɛewdiwen iwakken ad ssiwḍen Bulus di laman ɣer lḥakem Filiks.  </p> 
 <p><a name="an_23:25">23:25</a> Yura tabṛaț-agi i lḥakem Filiks :  </p> 
 <p><a name="an_23:26">23:26</a> « Nekk Kludyus Lizyas i lḥakem ameqqran Filiks, sslam fell-ak !  </p> 
 <p><a name="an_23:27">23:27</a> At Isṛail ṭṭfen argaz-agi teddun a t-nɣen, mi wwḍeɣ nekk d lɛeskeṛ-iw, sellkeɣ-t-id si ger ifassen-nsen. Imi sliɣ laṣel-is d aṛumani,  </p> 
 <p><a name="an_23:28">23:28</a> bɣiɣ ad ẓreɣ sebba ɣef wacu ccetkan fell-as ; wwiɣ-t ɣer zdat unejmaɛ nsen.  </p> 
 <p><a name="an_23:29">23:29</a> Ufiɣ belli sebba ɣef wacu i ccetkan fell-as, d ayen yeɛnan ccariɛa-nsen, lameɛna ur ixdim acemma yuklalen lmut neɣ lḥebs.  </p> 
 <p><a name="an_23:30">23:30</a> Mi sliɣ ḥeyylen-as wat Isṛail, ceggɛeɣ-t-in ɣuṛ-ek. Ma d ixṣimen-is nniɣ-asen a n-ṛuḥen ɣuṛ-ek ad ccetkin fell-as. Sǧiɣ-k di lehna !  </p> 
 <p><a name="an_23:31">23:31</a> Akken i ten-d-yewweḍ lameṛ, iɛeskṛiwen wwin Bulus deg yiḍ ɣer temdint n Antibatris.  </p> 
 <p><a name="an_23:32">23:32</a> Azekka-nni uɣalen ɣer lbeṛj, ǧǧan imnayen ad kemmlen abrid yid-es. </p> 
 <p><a name="an_23:33">23:33</a> Mi wwḍen ɣer temdint n Qiṣarya, wwin Bulus ɣer lḥakem Filiks, rnan fkan-as tabṛaț-nni.  </p> 
 <p><a name="an_23:34">23:34</a> Lḥakem yeɣṛa tabṛaț-nni, isteqsa anta i ț-țamurt n Bulus. Mi gesla si tmurt n Silisya i gella, </p> 
 <p><a name="an_23:35">23:35</a> yenna-yas : M'ara d-awḍen yexṣimen iccetkan fell-ak, a k-sleɣ . Dɣa yumeṛ a t-ɛassen di lbeṛj n Hiṛudus. At Isṛail cccetkan ɣef Bulus  </p> 
 <p><a name="an_24:1">24:1</a> Mi ɛeddan xemsa wussan, lmuqeddem ameqqran Ananyas iṣubb-ed ɣer Qiṣarya nețța d kra n lɛuqal n wat Isṛail akk-d yiwen ubugaṭu isem-is Tertulus ; usan-d ɣer lḥakem ad ccetkin ɣef Bulus.  </p> 
 <p><a name="an_24:2">24:2</a> Mi d-ssawlen i Bulus, Tertulus yebda accetki fell-as s yimeslayen agi :  </p> 
 <p><a name="an_24:3">24:3</a> A lḥakem ameqqran ! S uḍebbaṛ-ik yelhan i d-yers lxiṛ d lehna ɣef tmurt-agi ! Ɣef wayagi, a Filiks ameqqran, ncekkeṛ-ik meṛṛa di mkul amkan ; </p> 
 <p><a name="an_24:4">24:4</a> lameɛna iwakken ur k-nețɛeṭṭil ara aṭas, di leɛnaya-k semḥess-ed i kra n yimeslayen-agi ara k-d-nini :  </p> 
 <p><a name="an_24:5">24:5</a> Argaz-agi d țerka, iskker ccwal ger wat Isṛail n ddunit meṛṛa, d nețța i d ameqqran n tejmaɛt n inaṣariyen.  </p> 
 <p><a name="an_24:6">24:6</a> Yeɛṛed ad issenǧes ula d lǧameɛ iqedsen. Mi t-neṭṭef, nebɣa a neḥkem fell-as s ccariɛa-nneɣ,  </p> 
 <p><a name="an_24:7">24:7</a> lameɛna yewweḍ-ed lqebṭan ameqqran Lizyas, ikkes-aɣ-t s ddreɛ si ger ifassen-nneɣ, </p> 
 <p><a name="an_24:8">24:8</a> yerna yumeṛ i wid iccetkan fell-as a d-asen ad ccetkin ɣuṛ-ek. Tura tzemreḍ s yiman-ik a t-tbeḥteḍ, iwakken aț-țesleḍ seg yimi-s wayen akk i ɣef neccetka fell-as.  </p> 
 <p><a name="an_24:9">24:9</a> At Isṛail yellan dinna, cehden akk belli akka i gella lḥal.  </p> 
 <p><a name="an_24:10">24:10</a> Mi i s-iwehha lḥakem i Bulus iwakken a d-ihdeṛ, Bulus yenna : Sriɣ aṭas iseggasen aya i telliḍ d lḥakem ɣef wegdud-agi ; ihi s laman ara d-qeddmeɣ tamsalt-iw zdat-ek. </p> 
 <p><a name="an_24:11">24:11</a> Ur wwiḍen ara tnac wussan segmi uliɣ ɣer temdint n Lquds iwakken ad ɛebdeɣ Ṛebbi, tzemreḍ aț-țesteqsiḍ s yiman-ik. </p> 
 <p><a name="an_24:12">24:12</a> Ulac win i yi-d-yufan deg wefrag n lǧameɛ iqedsen țemjadaleɣ nekk d walebɛaḍ, ulac daɣen win i yi-d-yufan di leǧwameɛ n wat Isṛail neɣ di temdint sekkreɣ ccwal ger lɣaci. </p> 
 <p><a name="an_24:13">24:13</a> Ur zmiren ara a d-awin țțbut n wayen akka i ɣef d-ccetkan fell-i.  </p> 
 <p><a name="an_24:14">24:14</a> Zemreɣ a d-iniɣ zdat-ek a lḥakem Filiks : Ṛebbi i ɛebbdeɣ, d win akken ɛebden lejdud-iw ; s tideț umneɣ s webrid-agi ajdid i ɣef d-qqaṛen : « ixulef abrid-nsen » ; lameɛna umneɣ s wayen akk yuran di ccariɛa n Musa akk-d lenbiya.  </p> 
 <p><a name="an_24:15">24:15</a> Sɛiɣ asirem ɣer Sidi Ṛebbi -- am akken i t-sɛan ula d nutni -- belli a d-ḥyun yemdanen meṛṛa ama d iḥeqqiyen ama d wid ixedmen cceṛ.  </p> 
 <p><a name="an_24:16">24:16</a> Daymi țɛassaɣ iman-iw kull ass, iwakken ad sɛuɣ ul yeṣfan zdat Ṛebbi akk-d yemdanen.  </p> 
 <p><a name="an_24:17">24:17</a> Acḥal n iseggasen, nekk ɛaceɣ di lɣeṛba ; usiɣ-ed iwakken a d-awiɣ kra n yedrimen i d-jemɛeɣ d lemɛawna i igellilen n tmurt-iw yerna wwiɣ-ed daɣen lweɛdat i Sidi Ṛebbi.  </p> 
 <p><a name="an_24:18">24:18</a> Atan wamek i yi-d-ufan deg wefrag n lǧameɛ iqedsen : ufan-iyi-d akken kan i ssezdegeɣ iman-iw akken i d-tenna ccariɛa; ur iyi-d zzin imdanen ur yelli ccwal </p> 
 <p><a name="an_24:19">24:19</a> yerna d at Isṛail n tmurt n Asya i gellan dinna, d nutni i glaq ad bedden zdat-ek ma sɛan kra ɣef wacu ara ccetkin fell-i.  </p> 
 <p><a name="an_24:20">24:20</a> Neɣ m'ulac wid yellan dagi, a d-inin d acu ufan deg-i n diri mi beddeɣ zdat unejmaɛ-nsen n ccṛeɛ ; </p> 
 <p><a name="an_24:21">24:21</a> anagar ma yella ɣef wawal-nni i d-nniɣ s leɛyaḍ gar-asen : « imi umneɣ s ḥeggu n lmegtin, i d-beddeɣ ass-agi ɣer ccṛeɛ zdat-wen ! »  </p> 
 <p><a name="an_24:22">24:22</a> Filiks yesnen akken ilaq abrid n Lmasiḥ, yenna-yasen : Uɣalet-ed m'ara d-yaweḍ lqebṭan Lizyas, imiren ad ẓreɣ tamsalt-nwen.  </p> 
 <p><a name="an_24:23">24:23</a> Yefka lameṛ i yiwen n lfesyan ad iɛass Bulus, lameɛna a s-yeǧǧ kra n țesriḥ, yerna ad yeǧǧ imdukkal-is a d-reSSun fell-ass.  </p> 
 <p><a name="an_24:24">24:24</a> Mi ɛeddan kra wussan, yusa-d Filiks akk-d țmeṭṭut-is Drusila, nețțat yellan n wat Isṛail, iceggeɛ a d-awin Bulus, iwakken a sen-d-ihdeṛ ɣef wayen yeɛnan liman di Ɛisa Lmasiḥ. </p> 
 <p><a name="an_24:25">24:25</a> Mi gebda Bulus iheddeṛ ɣef lḥeqq, ɣef wamek ilaq ad iḥkem wemdan deg iman-is, ɣef wass n lḥisab i d-iteddun, Filiks ikcem-it lxuf, yenna-yas : Dayen, tzemreḍ aț-țṛuḥeḍ tura. M'ara d-stifeɣ, a k-d-ssiwleɣ.  </p> 
 <p><a name="an_24:26">24:26</a> Imi i geḍmeɛ di Bulus a s-yefk idrimen, si teswiɛt ɣer tayeḍ issawal-as-ed iwakken ad ihdeṛ yid-es. </p> 
 <p><a name="an_24:27">24:27</a> Ɛeddan sin n iseggasen ; dɣa Bursyus Fistus yuɣal d lḥakem deg wemkan n Filiks. Filiks-nni imi i gebɣa ad iɛǧeb i wat Isṛail, yeǧǧa Bulus di lḥebs.  </p> 
 <p><a name="an_25:1">25:1</a> Mi ɛeddan tlata wussan segmi yuɣal d lḥakem, Fistus yuli si temdint n Qiṣarya ɣer temdint n Lquds.  </p> 
 <p><a name="an_25:2">25:2</a> Dɣa lmuqedmin imeqqranen akk-d imeqqranen n wat Isṛail, ccetkan ɣuṛ-es ɣef Bulus.  </p> 
 <p><a name="an_25:3">25:3</a> Ssutren i Fistus a sen-ixdem lemziya, a sen-d-yawi Bulus ɣer temdint n Lquds, nutni heggan-as-d taxazabit iwakken a t-nɣen deg webrid. </p> 
 <p><a name="an_25:4">25:4</a> Lameɛna Fistus yerra-yasen : Bulus ițwaḥbes di Qiṣarya, nekk s yiman-iw qṛib ad uɣaleɣ ɣer dinna. </p> 
 <p><a name="an_25:5">25:5</a> Xtiṛet-ed si gar-awen wid ara yeddun yid-i ; ma yella kra n wayen n diri i gexdem wergaz-agi, ad ccetkin fell-as.  </p> 
 <p><a name="an_25:6">25:6</a> Fistus isɛedda azal n tmanya neɣ ɛecṛa wussan gar-asen, taggara yuɣal ɣer Qiṣarya. Azekka-nni, iṭṭef amkan ɣef wukersi n ccṛeɛ, iceggeɛ a d-awin Bulus.  </p> 
 <p><a name="an_25:7">25:7</a> Mi d-yewweḍ Bulus, at Isṛail i d-yusan si temdint n Lquds zzin-as, gren-d fell-as lbaṭel, lameɛna ur zmiren ara a d-tebten wayen i d qqaṛen.  </p> 
 <p><a name="an_25:8">25:8</a> Bulus iḥudd iman-is yenna : Ur ɛuṣaɣ ccariɛa n wat Isṛail, ur skecmeɣ leḥṛam ɣer lǧameɛ iqedsen, ur xulfeɣ Qayṣer.  </p> 
 <p><a name="an_25:9">25:9</a> Imi gebɣa ad issefṛeḥ at Isṛail, Fistus yenna i Bulus : Tebɣiḍ aț-țaliḍ ɣer temdint n Lquds iwakken aț-țɛeddiḍ di ccṛeɛ dinna zdat-i ?  </p> 
 <p><a name="an_25:10">25:10</a> Lameɛna Bulus yenna : Ɣer ccṛeɛ n Qayṣer i beddeɣ, ɣuṛ-es ara țțucaṛɛeɣ. Ur xdimeɣ ula d acemma i gḍuṛṛen at Isṛail, akken teẓriḍ kečč s yiman-ik. </p> 
 <p><a name="an_25:11">25:11</a> Ma yella xedmeɣ lbaṭel neɣ ayen yuklalen lmut, ur țțaggadeɣ ara ad mmteɣ. Lameɛna ma yella lecɣal-agi ɣef i d-ccetkan fell-i ur seḥḥan ara, yiwen ur izmir a yi-isellem ger ifassen-nsen. Ad rẓeɣ ccṛeɛ ɣer Qayṣer !  </p> 
 <p><a name="an_25:12">25:12</a> Mi gemcawaṛ Fistus nețța ț-țejmaɛt-nni n ccṛeɛ, yenna i Bulus : Tessutreḍ-ed ccṛeɛ n Qayṣer, aț-țețțucarɛeḍ zdat Qayṣer !  </p> 
 <p><a name="an_25:13">25:13</a> Mi ɛeddan kra wussan, agellid Aɣribas akk-d weltma-s Birinis, usan-d ɣer Qiṣarya ad sellmen ɣef lḥakem Fistus. </p> 
 <p><a name="an_25:14">25:14</a> Imi qqimen dinna kra wussan, Fistus iḥka-yas i ugellid-nni taqsiṭ n Bulus, yenna-yas : Yella yiwen wergaz i geǧǧa Filiks dagi d ameḥbus. </p> 
 <p><a name="an_25:15">25:15</a> Mi lliɣ di temdint n Lquds, ccetkan fell-as imeqqranen n lmuqedmin d lɛuqal n wat Isṛail ; ssutren-iyi-d ad ḥekmeɣ fell-as.  </p> 
 <p><a name="an_25:16">25:16</a> Nniɣ-asen mačči di lɛadda n iṛumaniyen ad ḥekmen ɣef wemdan uqbel a t-semqabalen nețța d wid i d-iccetkan fell-as, iwakken ad yaf abrid amek ara idafeɛ ɣef yiman-is.  </p> 
 <p><a name="an_25:17">25:17</a> Ihi usan-d ɣer dagi. Ur ɛeṭṭleɣ ara, azekka-nni fkiɣ lameṛ a d-awin argaz-nni iwakken a t-sɛeddiɣ di ccṛeɛ ; </p> 
 <p><a name="an_25:18">25:18</a> wid iccetkan fell-as ur d-nnin acemma seg wayen țrajuɣ a t-id-inin.  </p> 
 <p><a name="an_25:19">25:19</a> Țemjadalen anagar ɣef ddin akk-d yiwen ițusemman Ɛisa. Ɛisa-agi yemmut, meɛna Bulus yeqqaṛ-ed yedder.  </p> 
 <p><a name="an_25:20">25:20</a> Imi ur ẓriɣ ara acu ara xedmeɣ di temsalt-agi, nniɣ-as i Bulus ma yebɣa ad iṛuḥ ɣer temdint n Lquds iwakken ad ițțucaṛeɛ dinna, </p> 
 <p><a name="an_25:21">25:21</a> lameɛna yerẓa ccṛeɛ iwakken ad ițțucaṛeɛ zdat Qayṣer ; dɣa fkiɣ lameṛ a t-ɛassen alamma d ass i deg ara t-ceggɛeɣ ɣer Qayṣer.S  </p> 
 <p><a name="an_25:22">25:22</a> Agellid Aɣribas yenna i Fistus : Bɣiɣ ula d nekk ad sleɣ i wergaz-agi. Fistus yenna-yas : Azekka a s-tesleḍ !  </p> 
 <p><a name="an_25:23">25:23</a> Azekka-nni agellid Aɣribas d weltma-s Birinis usan-d, cebbḥen-d iman-nsen, ṭṭfen amkan deg wexxam n ccṛeɛ nutni d lqebṭanat akk-d imeqqranen n temdint ; lḥakem Fistus yefka lameṛ a d-skecmen Bulus, </p> 
 <p><a name="an_25:24">25:24</a> dɣa yenna : Ay agellid Aɣribas akk-d kunwi i gḥedṛen dagi ! Argaz-agi, d win akken i ɣef d-ccetkan wat Isṛail ama di temdint n Lquds ama dagi ; țɛeggiḍen qqaṛen : « ilaq ad immet »!  </p> 
 <p><a name="an_25:25">25:25</a> Ma d nekk, idheṛ-iyi-d lḥal ur ixdim ula d acemma yuklalen lmut ; imi nețța s yiman-is yerẓa ccṛeɛ ɣer Qayṣer, qebleɣ a s-t-ceggɛeɣ.  </p> 
 <p><a name="an_25:26">25:26</a> Ur ufiɣ ula d yiwet n sebba iseḥḥan i ɣef ara ketbeɣ fell-as i Qayṣer ; daymi i t-id-sbeddeɣ zdat wen, ula zdat-ek ay agellid Aɣribas, iwakken m'ara t-beḥten ad yili wayen ara ketbeɣ fell-as. </p> 
 <p><a name="an_25:27">25:27</a> Axaṭer walaɣ mačči d ṣṣwab ma ceggɛeɣ yiwen umeḥbus ɣer temdint n Ṛuma mbla ma fkiɣ-asen sebba s wayes i d ccetkan fell-as.  </p> 
 <p><a name="an_26:1">26:1</a> Aɣribas yenna i Bulus : -- Atan ɣuṛ-ek wawal, tzemreḍ aț-țdafɛeḍ ɣef yiman-ik. Bulus yerfed afus-is iwakken ad idafeɛ ɣef yiman-is, yenna :  </p> 
 <p><a name="an_26:2">26:2</a> Feṛḥeɣ aṭas ay agellid Aɣribas imi zdat-ek ara ɛeddiɣ di ccṛeɛ ass-agi ɣef wayen akk i ccetkan fell-i wat Isṛail, </p> 
 <p><a name="an_26:3">26:3</a> axaṭer tesneḍ akken ilaq leɛwayed akk-d yimeslayen-nsen ; di leɛnaya-k ihi semḥess-iyi-d s leɛqel.  </p> 
 <p><a name="an_26:4">26:4</a> At Isṛail ssnen tikli-inu si temẓi-w, axaṭer țțuṛebbaɣ di temdint n Lquds ger warraw n tmurt-iw.  </p> 
 <p><a name="an_26:5">26:5</a> Ssnen-iyi si zik ma bɣan ad cehden fell-i. Lliɣ d afarizi, tebɛeɣ leqwanen iweɛṛen yeɛnan ddin-nneɣ di tejmaɛt n ifariziyen.  </p> 
 <p><a name="an_26:6">26:6</a> Beddeɣ ass-agi di ccṛeɛ axaṭer ssarmeɣ aț-țwakmel lemɛahda-nni i gefka Sidi Ṛebbi i lejdud-nneɣ.  </p> 
 <p><a name="an_26:7">26:7</a> Tnac leɛṛac-nneɣ i gɛebden Ṛebbi am yiḍ am ass, țṛaǧun a d-taweḍ lemɛahda-agi i ssaramen. Ɣef wusirem-agi i ccetkan wat Isṛail fell-i.  </p> 
 <p><a name="an_26:8">26:8</a> Ay agellid Aɣribas, acuɣeṛ i tɣilem d lmuḥal a d-isseḥyu Sidi Ṛebbi lmegtin ? </p> 
 <p><a name="an_26:9">26:9</a> Ma d nekkini, di tazwara ɛeṛdeɣ ad xaṣmeɣ s wayen yellan di tezmert-iw, isem-agi n Ɛisa Anaṣari ;  </p> 
 <p><a name="an_26:10">26:10</a> d ayen i xedmeɣ di temdint n Lquds, rriɣ aṭas imasiḥiyen ɣer lḥebs, yerna d lmuqedmin imeqqranen i yi-d-yefkan țesriḥ-agi. M'ara ḥekkmen fell-asen s lmut, nekkini qebbleɣ ; </p> 
 <p><a name="an_26:11">26:11</a> qehṛeɣ-ten acḥal d abrid si lǧameɛ ɣer wayeḍ, ḥeṛseɣ-ten ad ṭṭixṛen i webrid-agi n Ɛisa ; deg urrif-iw ameqqran, țṛuḥeɣ armi ț-țimdinin tibeṛṛaniyin iwakken a ten-qehhṛeɣ.  </p> 
 <p><a name="an_26:12">26:12</a> Ɣef wannect-nni lmuqedmin imeqqranen fkan-iyi tazmert s wayes ṛuḥeɣ ɣer temdint n Dimecq.  </p> 
 <p><a name="an_26:13">26:13</a> Taswiɛt kan di țnaṣfa wass ay agellid, walaɣ deg webrid yiwet n tafat i d-yekkan seg igenni, tețfeǧǧiǧ akteṛ n yiṭij, tezzi-yi-d i nekk akk-d wid yellan yid-i, </p> 
 <p><a name="an_26:14">26:14</a> neɣli akk ɣer lqaɛa. Sliɣ i ṣṣut i d-iqqaṛen s tmeslayt taɛibṛanit : A Caɛul, a Caɛul ! Acuɣeṛ i yi tețqehhiṛeḍ ? Yewɛeṛ fell-ak aț-țɛaṣiḍ inezlan. </p> 
 <p><a name="an_26:15">26:15</a> Rriɣ-as : Anwa-k a Sidi ? Yenna-yi-d : Nekkini d Ɛisa-nni i tețqehhiṛeḍ ; </p> 
 <p><a name="an_26:16">26:16</a> kker fell-ak, bedd ɣef yidaṛṛen-ik, ḍehṛeɣ-ak-d iwakken a k-rreɣ d aqeddac-iw, d inigi ɣef wayen twalaḍ ass-agi d wayen ara k-d-sekneɣ sya d asawen. </p> 
 <p><a name="an_26:17">26:17</a> Xtaṛeɣ-k-id si ger wat Isṛail akk-d leǧnas nniḍen uɣuṛ ara k ceggɛeɣ. A sen-teldiḍ allen </p> 
 <p><a name="an_26:18">26:18</a> iwakken a d ffɣen si ṭṭlam ɣer tafat, si tezmert n Cciṭan ɣer tezmert n Ṛebbi, ad amnen yis-i, iwakken a sen-neɛfuɣ ddnubat-nsen, ad sɛun amkan ger wid i gextaṛ Sidi Ṛebbi. </p> 
 <p><a name="an_26:19">26:19</a> Daymi, ay agellid Aɣribas, ur ɛuṣaɣ ara aweḥḥi i yi-d-yusan seg igenni. </p> 
 <p><a name="an_26:20">26:20</a> Bdiɣ abecceṛ di temdint n Dimecq akk-d ț-țemdint n Lquds, beccṛeɣ daɣen di tmurt n Yahuda meṛṛa, di taggara, beccṛeɣ i wat leǧnas ur nelli ara seg wat Isṛail, nhiɣ-ten ad tuben, a d-uɣalen ɣer webrid n Ṛebbi, ad xedmen lecɣal i d-ițbegginen belli tuben s wul-nsen.  </p> 
 <p><a name="an_26:21">26:21</a> Ɣef wannect-agi i yi-ṭṭfen wat Isṛail deg wefrag n lǧameɛ iqedsen, ɛeṛden ad iyi-nɣen.  </p> 
 <p><a name="an_26:22">26:22</a> Lameɛna s lemɛawna n Sidi Ṛebbi, ddreɣ armi d ass-a, țbecciṛeɣ i yemdanen meṛṛa, ama d ameɣbun ama d ameṛkanti ; yerna ayen i d-qqaṛeɣ, d ayen akken i ɣef d-caren lenbiya akk-d Sidna Musa : </p> 
 <p><a name="an_26:23">26:23</a> yeɛni Lmasiḥ ad iɛteb, d nețța i d amezwaru ara d-iḥyun si ger lmegtin, ad ibecceṛ tafat i wegdud n wat Isṛail akk-d leǧnas nniḍen.  </p> 
 <p><a name="an_26:24">26:24</a> Akken yella Bulus yețdafaɛ ɣef yiman-is, Fistus yenṭeq-ed s leɛyaḍ, yenna : Iffeɣ-ik leɛqel a Bulus ! Lketṛa n tmusni-inek tessufeɣ-ak leɛqel !  </p> 
 <p><a name="an_26:25">26:25</a> Bulus yerra-yas : Ur iyi-iffiɣ ara leɛqel a Fistus ameqqran ! Ayen akka i d-qqaṛeɣ d imeslayen n tideț yesɛan ṣṣwab. </p> 
 <p><a name="an_26:26">26:26</a> Ula d agellid Aɣribas, yeɛlem s wayagi ; daymi i d-mmeslayeɣ zdat-es ɛinani, imi mačči s tuffra i gedṛa wannect-a. </p> 
 <p><a name="an_26:27">26:27</a> Ay agellid, tumneḍ s wayen i d-nnan lenbiya ? Sriɣ tumneḍ yis-sen !  </p> 
 <p><a name="an_26:28">26:28</a> Aɣribas yenna i Bulus : Lemmer a yi-d ternuḍ kra yimeslayen ahat ad iyi-tqenɛeḍ ad uɣaleɣ d amasiḥi !  </p> 
 <p><a name="an_26:29">26:29</a> Bulus yenna-yas : A wi-yufan a Ṛebbi, mačči d keččini kan, lameɛna ula d wid akk i yi-d-ismeḥsisen ass-agi, aț-țuɣalem am nekk tura neɣ deg ussan i d-iteddun, ḥaca snasel-agi i yi cudden !  </p> 
 <p><a name="an_26:30">26:30</a> Dɣa agellid Aɣribas, lḥakem Fistus, Birinis d wid akk yeqqimen yid-sen, kkren-d. </p> 
 <p><a name="an_26:31">26:31</a> Mi kkren ad ṛuḥen, nnan wway gar-asen : Argaz-agi ur ixdim ula d acemma yuklalen lmut neɣ lḥebs.  </p> 
 <p><a name="an_26:32">26:32</a> Aɣribas yenna i Fistus : Argaz-agi izmer ad ițțuserreḥ lemmer ur yerẓi ara ccṛeɛ ɣer Qayṣer.  </p> 
 <p><a name="an_27:1">27:1</a> Mi fkan lameṛ a nerkeb lbabuṛ ɣer tmurt n Selyan, ǧǧan Bulus akk-d kra n imeḥbas nniḍen i yiwen ufesyan n terbaɛt n Qayṣer, isem-is Xulyus. </p> 
 <p><a name="an_27:2">27:2</a> Nerkeb deg yiwen lbabuṛ n temdint n Adramit, ara iṛuḥen rrif n tmura n Asya ; nṛuḥ, yedda yid-nneɣ Aristaṛk yellan d amasiduni n temdint n Tiṣalunik. </p> 
 <p><a name="an_27:3">27:3</a> Azekka-nni, newweḍ ɣer temdint n Sidun , Xulyus yețqadaṛen Bulus, ixdem-as lemziya, iserreḥ-as ad iṛuḥ ɣer imdukkal-is iwakken a s-d-fken ayen yeḥwaǧ.  </p> 
 <p><a name="an_27:4">27:4</a> Mi neqleɛ syenna, nɛedda rrif rrif n tegzirt n Qubṛus axaṭer iṣbeḥ-aɣ-d waḍu iǧehden si zdat.  </p> 
 <p><a name="an_27:5">27:5</a> Nezger lebḥeṛ n Silisya akk-d Bamfilya, newweḍ ɣer lmeṛṣa n Mira yellan di tmurt n Lizya.  </p> 
 <p><a name="an_27:6">27:6</a> Afesyan-nni yufa dinna lbabuṛ n Skandriya ara iṛuḥen ɣer tmurt n Selyan, isserkeb-aɣ deg-s. </p> 
 <p><a name="an_27:7">27:7</a> Lbabuṛ nni yewwi-yaɣ kra n wussan s țțawil ; armi neṛwa leɛtab i newweḍ ɣer tama n temdint n Knidus, imi ur aɣ-yeǧǧi ara waḍu a nqeṛṛeb, nɛedda rrif n tegzirt n Kritus, ɣer tama n Salmuni. </p> 
 <p><a name="an_27:8">27:8</a> Nkemmel rrif rrif s leɛtab, armi newweḍ ɣer yiwen wemkan isem-is « Lemraṣi icebḥen », zdat temdint n Lazaya.  </p> 
 <p><a name="an_27:9">27:9</a> Nesṛuḥ aṭas n lweqt, lweqt n usafer di lebḥeṛ yuɣal yewɛeṛ aṭas, imi ass i deg țțuẓummen wat Isṛail di taggara n lexṛif iɛedda.  </p> 
 <p><a name="an_27:10">27:10</a> Daymi i sen-iɛeggen Bulus yenna-yasen : Ay atmaten, ẓriɣ belli tikkelt-agi ma nkemmel aṣafeṛ-nneɣ, a d-tili lexsaṛa mačči kan i lbabuṛ d sselɛa yellan deg-s, lameɛna tezmer aț-țeglu ula yis-nneɣ.  </p> 
 <p><a name="an_27:11">27:11</a> Afesyan-nni aṛumani yuɣ awal i bab n lbabuṛ akk-d win i t-inehṛen wala ad yaɣ awal i Bulus. </p> 
 <p><a name="an_27:12">27:12</a> Imi lmeṛṣa-nni ur telhi ara iwakken ad sɛeddin deg-s ccetwa, azgen ameqqran n ibeḥriyen bɣan ad ṛuḥen, iwakken ma yella wamek, ad awḍen ɣer lmeṛṣa n Finikus yellan di tegzirt n Kritus ; lmeṛṣa-yagi tqubel lǧiha taɣeṛbit, bɣan ad sɛeddin deg-s ccetwa.  </p> 
 <p><a name="an_27:13">27:13</a> Akken i d-yekker yiwen ubeḥri xfifen, ɣilen ad awḍen ɣer leqsed nsen. Refden amextaf n lbabuṛ, ṛuḥen rrif rrif n tegzirt n Kritus.  </p> 
 <p><a name="an_27:14">27:14</a> Lameɛna ur tɛeṭṭel ara tekker-ed yiwet n tbuciḍant iwumi qqaṛen Erakilun i d-ikkan seg idurar n tegzirt-nni.  </p> 
 <p><a name="an_27:15">27:15</a> Imi lbabuṛ ur s-izmir ara i tbuciḍant-nni, neǧǧa-t a t-yawi waḍu ɣer wanda i s-yehwa,  </p> 
 <p><a name="an_27:16">27:16</a> nɛedda seddaw yiwet n tegzirt tamecṭuḥt yeddurin ɣef waḍu, isem-is Kluda. Mi ɣ-teɣli teflukt-nni n leslak, s leɛtab ameqqran i ț-id-nessuli ɣer lbabuṛ.  </p> 
 <p><a name="an_27:17">27:17</a> Imi nuggad a ɣ-yawi waḍu ɣer leryuf iweɛṛen n tmurt n Libya, ncudd lbabuṛ s imurar ; nerna nṣubb-ed lbac-nni s wayes i t-yețțawi waḍu, neǧǧa iman-nneɣ a ɣ-yawi ɣer wanda yebɣa.  </p> 
 <p><a name="an_27:18">27:18</a> Azekka-nni, imi mazal lebḥeṛ yenhewwal, nessenqes kra n sselɛa yellan deg-s nḍeggeṛ-iț ɣer lebḥeṛ.  </p> 
 <p><a name="an_27:19">27:19</a> Ass wis tlata, ixeddamen n lbabuṛ rnan ḍeggṛen sselɛa-nni i d-iqqimen. </p> 
 <p><a name="an_27:20">27:20</a> Acḥal n wussan ur d-iban yiṭij ur d-banen itran, lebḥeṛ mazal-it yerwi, ur numin ara a nțwasellek.  </p> 
 <p><a name="an_27:21">27:21</a> Aṭas wussan i neqqim mbla lmakla ; dɣa Bulus ibedd-ed gar-asen yenna : Ay irgazen, lemmer i yi-tuɣem awal m'akken i wen-d-nniɣ : a neqqim axiṛ di lmeṛṣa n Kritus, tili ur d-tdeṛṛu ara lexsaṛa akk-d lxuf-agi !  </p> 
 <p><a name="an_27:22">27:22</a> Tura ur țțagadet ara, sǧehdet iman-nwen axaṭer ula d yiwen deg-wen ur ițmețțat ; ur d-tețțili lexsaṛa anagar lbabuṛ-agi ara iɣeṛqen.  </p> 
 <p><a name="an_27:23">27:23</a> Iḍ iɛeddan, Sidi Ṛebbi i ɛebbdeɣ iceggeɛ-ed yiwen lmelk ibedd-ed ɣuṛ-i </p> 
 <p><a name="an_27:24">27:24</a> yenna-yi-d : « a Bulus, ur țțaggad ara ! Ilaq aț-țbeddeḍ zdat Qayṣer, atan Sidi Ṛebbi a kkun isellek s kečč s wid yellan yid-ek ». </p> 
 <p><a name="an_27:25">27:25</a> Sewṛet ihi ay atmaten ! Țekleɣ ɣef Ṛebbi ad idṛu am akken i d-yenna ; </p> 
 <p><a name="an_27:26">27:26</a> mbla ccekk ihi a ɣ-ḍeggṛent lemwaji ɣer yiwet n tegzirt.  </p> 
 <p><a name="an_27:27">27:27</a> Iḍ wis ṛbeɛṭac, mazal aḍu yețhuccu-yaɣ ițțawi-yaɣ akkin akka ɣef lebḥeṛ Agrakal ; di țnaṣfa n yiḍ, ibeḥṛiyen ḥussen belli ur bɛiden ara ɣef lberr. </p> 
 <p><a name="an_27:28">27:28</a> Ktalen lqaɛ n lebḥeṛ s wemrar, ufan llant  </p> 
 <p><a name="an_27:29">27:29</a> Imi uggaden lbabuṛ a t-id-iḥaz kra seddaw-as, ḍeggṛen ṛebɛa imextafen ɣer deffir n lbabuṛ, tqelqen melmi ara yali wass. </p> 
 <p><a name="an_27:30">27:30</a> Lameɛna ixeddamen n lbabuṛ țqelliben ad rewlen ; ṣubben taflukt n leslak ɣer lebḥeṛ, steɛmilen am akken d amextaf ara ḍeggṛen ɣer zdat lbabuṛ.  </p> 
 <p><a name="an_27:31">27:31</a> Dɣa Bulus yenna i lqebṭan d lɛeskeṛ-is : M'ur qqimen ara yergazen-agi di lbabuṛ, ur tezmirem ara aț-țmenɛem ! S </p> 
 <p><a name="an_27:32">27:32</a> Imiren lɛeskeṛ gezmen imurar n teflukt, ǧǧan-ț aț-țeɣli ɣer lebḥeṛ.  </p> 
 <p><a name="an_27:33">27:33</a> Weqbel ad yali wass, Bulus inha-ten akk ad ččen cwiṭ n lqut, yenna : Ass-agi d ass wis ṛbeɛṭac ur teɛṛiḍem lqut. </p> 
 <p><a name="an_27:34">27:34</a> Tura nhiɣ-kkun ihi aț-țeččem iwakken aț-țețțusellkem, ula d yiwen deg-wen ur s-ițṛuḥ wenẓad seg uqeṛṛuy.  </p> 
 <p><a name="an_27:35">27:35</a> Mi d-yenna ayagi, iddem-ed aɣṛum, iḥmed Ṛebbi zdat-sen meṛṛa, yebḍa-t, yebda itețț.  </p> 
 <p><a name="an_27:36">27:36</a> Yekkes-asen akk lxuf dɣa ččan. </p> 
 <p><a name="an_27:38">27:38</a> Mi ččan armi ṛwan, ḍeggṛen ɣer lebḥeṛ ticekkaṛin n yirden, iwakken ad yifsus lbabuṛ. </p> 
 <p><a name="an_27:39">27:39</a> Mi guli wass, ur eɛqilen ara maḍi tamurt-nni anda wwḍen. Lameɛna walan yiwen wemkan iqeṛben, iban-ed deg-s ṛṛmel, bɣan ad rren lbabuṛ ɣer dinna ma yella wamek.  </p> 
 <p><a name="an_27:40">27:40</a> Fsin imurar i imextafen bran asen ɣer lebḥeṛ, fsin daɣen imurar i imeqdafen s wacu nehṛen lbabuṛ ; dɣa ssulin yiwen ubeḥnuq yellan ɣer zdat n lbabuṛ iwakken a ten-yawi waḍu ɣer rrif n lebḥeṛ.  </p> 
 <p><a name="an_27:41">27:41</a> Lameɛna wwḍen ɣer yiwen wemkan n ṛṛmel iwumi d-izzi lebḥeṛ ; rran ɣer dinna lbabuṛ-nni, lǧiha n zdat teḥṣel di ṛṛmel teḥbes ma d lǧiha n deffir tebda tețṛuẓu si lqewwa n lemwaji.  </p> 
 <p><a name="an_27:42">27:42</a> Lɛeskeṛ bɣan ad nɣen imeḥbas-nni axaṭer uggaden a sen-rewlen s lɛum.  </p> 
 <p><a name="an_27:43">27:43</a> Lqebṭan-nni i gebɣan ad isellek Bulus, ur ten-yeǧǧi ara ad xedmen lebɣi-nsen. Yefka lameṛ i wid yessnen ad ɛummen, ad ḍeggṛen iman-nsen d imezwura ɣer waman iwakken ad awḍen ɣer rrif.  </p> 
 <p><a name="an_27:44">27:44</a> Ma d wiyaḍ a ten-tebɛen, ad ṭṭfen di telwiḥin neɣ deg iceqfan n lbabuṛ. S wakka, wwḍen meṛṛa ɣer rrif n lebḥeṛ di laman.  </p> 
 <p><a name="an_28:1">28:1</a> Mi d-nemneɛ si lebḥeṛ, nesla belli tigzirt-nni isem-is Malṭa. </p> 
 <p><a name="an_28:2">28:2</a> Imezdaɣ-is setṛeḥben yis-nneɣ, xedmen-aɣ lxiṛ d ameqqran ; ceɛlen times axaṭer yekkat ugeffur ( lehwa ), yerna d asemmiḍ.  </p> 
 <p><a name="an_28:3">28:3</a> Bulus ijmeɛ-ed kra n yesɣaṛen a ten-iger ɣer tmes, si leḥmu-nni n tmes yeffeɣ-ed yiwen n wezrem yenṭed deg ufus-is.  </p> 
 <p><a name="an_28:4">28:4</a> Lɣaci mi walan azrem-nni yeckenṭeḍ deg ufus-is, qqaṛen wway gar-asen : « Argaz-agi iban d bu tmegṛaḍ, axaṭer ɣas akken yemneɛ-ed si lebḥeṛ, Sidi Ṛebbi yellan d Bab n lḥeqq ur t-yețțaǧa ara ad yidir. »  </p> 
 <p><a name="an_28:5">28:5</a> Lameɛna Bulus yezwi azrem-nni ɣer tmes, ur t-yuɣ ula d acemma ; </p> 
 <p><a name="an_28:6">28:6</a> lɣaci-nni țṛaǧun a t-walin ibzeg neɣ ad yemmet imiren kan. Ɛussen-t, mi walan ur t-yuɣ wacemma, beddlen ṛṛay, ḥesben-t d yiwen seg iṛebbiten s wayes țțamnen.  </p> 
 <p><a name="an_28:7">28:7</a> Di leǧwahi-nni i gezdeɣ Bubliyus lḥakem ameqqran n tegzirt-nni ; yesɛa aṭas n wakal, yesṭerḥeb yis-nneɣ, yerna yessens-aɣ ɣuṛ-es tlata wussan.  </p> 
 <p><a name="an_28:8">28:8</a> Deg ussan-nni, baba-s n Bubliyus yella deg usu yehlek, tuɣ-it tawla yerna tuzzel tɛebbuṭ-is. Bulus iṛuḥ ɣuṛ-es, issers ifassen-is fell-as, yedɛa ɣer Sidi Ṛebbi dɣa yeḥla.  </p> 
 <p><a name="an_28:9">28:9</a> Mi slan yis imuḍan n tegzirt-nni usan-d ɣuṛ-es isseḥla-ten ula d nutni. </p> 
 <p><a name="an_28:10">28:10</a> Ɣef wayagi, qudṛen-aɣ aṭas ; mi nekker a nṛuḥ, fkan-aɣ-d ayen akk neḥwaǧ i webrid.  </p> 
 <p><a name="an_28:11">28:11</a> Mi ɛeddan tlata waguren, nerkeb yiwen lbabuṛ n temdint n Skandriya iwumi qqaṛen Dyuskuṛ, isɛeddan ccetwa di tegzirt-nni ; </p> 
 <p><a name="an_28:12">28:12</a> newweḍ ɣer temdint n Sirakuz, anda neqqim tlata wussan.  </p> 
 <p><a name="an_28:13">28:13</a> Syenna nkemmel rrif rrif n lebḥeṛ armi ț-țamdint n Ṛejyu, azekka-nni yekker-ed yiwen n waḍu i d-yekkan seg usamer, sin wussan mbeɛd, newweḍ ɣer temdint n Buzul.  </p> 
 <p><a name="an_28:14">28:14</a> Nufa dinna atmaten i ɣ iḥellelen a neqqim yid-sen sebɛa wussan, syenna nkemmel abrid ɣer temdint n Ṛuma. </p> 
 <p><a name="an_28:15">28:15</a> Mi slan yis-nneɣ watmaten n temdint n Ṛuma, usan-d armi d ssuq n Abyus akk-d wemkan yețțusemman « Tlata Țbernat » iwakken a ɣ-d-mmagren. Mi ten iwala Bulus, yefṛeḥ yeḥmed Ṛebbi yerna yuɣal-it-id lǧehd.  </p> 
 <p><a name="an_28:16">28:16</a> Mi newweḍ ɣer temdint n Ṛuma, serrḥen-as i Bulus ad izdeɣ deg yiwen wexxam, nețța d yiwen uɛeskṛi i t yețɛassan.  </p> 
 <p><a name="an_28:17">28:17</a> Mi ɛeddan tlata wussan, Bulus yessawel i imeqqranen n wat Isṛail. Mi d-nnejmaɛen, Bulus yenna yasen : Ay atmaten, ɣas akken ur xdimeɣ acemma i gxulfen agdud d leɛwayed n lejdud-nneɣ, ṭṭfen-iyi ɣer lḥebs di temdint n Lquds, sellmen-iyi ger ifassen n lɛeskeṛ n Ṛuman.  </p> 
 <p><a name="an_28:18">28:18</a> Mi yi-beḥten, bɣan ad iyi serrḥen axaṭer ur ufin ara deg-i sebba s wayes ara ḥekmen fell-i s lmut.  </p> 
 <p><a name="an_28:19">28:19</a> Imi ur qbilen ara wat Isṛail ad iyi-d-serrḥen iṛumaniyen, iḥettem-iyi lḥal ad rẓeɣ ccṛeɛ ɣer Qayṣer ; lameɛna mačči d acetki i bɣiɣ ad ccetkiɣ ɣef wegdud-iw.  </p> 
 <p><a name="an_28:20">28:20</a> Daymi i bɣiɣ a kkun-ẓreɣ, ad mmeslayeɣ yid-wen axaṭer ɣef ddemma n usirem n wat Isṛail i țwarzeɣ s snasel-agi.  </p> 
 <p><a name="an_28:21">28:21</a> Imeqqranen n wat Isṛail nnan-as : Ur d-neṭṭif ula d yiwet n tebṛaț si tmurt n Yahuda ɣef wayen yeɛnan tamsalt-ik, ula d yiwen seg watmaten ur d-yusi ad iccetki neɣ a ɣ-d-yini ayen n diri fell-ak.  </p> 
 <p><a name="an_28:22">28:22</a> Lameɛna nebɣa a nsel s ɣuṛ-ek ayen akka s wacu i tumneḍ, ma d ayen yeɛnan tajmaɛt-agi seg i d-tekkiḍ, neẓra belli di yal amkan tețmagar-ed uguren. </p> 
 <p><a name="an_28:23">28:23</a> Dɣa msefhamen ɣef wass nniḍen i deg ara mlilen. Mi d yewweḍ wass-nni, usan-d deg waṭas yid-sen ɣer wexxam n Bulus. Si ṣṣbeḥ armi ț-țameddit, Bulus yețbecciṛ-asen ɣef tgeldit n Ṛebbi s ccariɛa n Musa ț-țektabin n lenbiya, yețqellib a ten-iqenneɛ iwakken ad amnen s Sidna Ɛisa. </p> 
 <p><a name="an_28:24">28:24</a> Kra umnen s wayen i d-yenna, ma d wiyaḍ ugin ad amnen. </p> 
 <p><a name="an_28:25">28:25</a> Mi țeddun ad ṛuḥen mazal lxilaf gar-asen, dɣa Bulus yerna-yasen imeslayen-agi : Ț-țideț ayen i d-yenna Ṛṛuḥ iqedsen i lejdud-nwen s yimi n nnbi Iceɛya </p> 
 <p><a name="an_28:26">28:26</a> mi d-yenna : Ṛuḥ ɣer wegdud-agi tiniḍ-asen : ɣas aț-țeslem s imeẓẓuɣen-nwen ur tfehmem ara ; aț-țemmuqlem s wallen-nwen ur tețwalim ara,  </p> 
 <p><a name="an_28:27">28:27</a> imi ul n wegdud-agi yuɣal d azṛu ; qeflen imeẓẓuɣen-nsen, qemcen allen-nsen ; ur bɣin ara ad walin s wallen-nsen, d slen s imeẓẓuɣen-nsen, ad fehmen s wul-nsen, iwakken a d-uɣalen ɣuṛ-i ad beddlen tikli akken a ten-sseḥluɣ !  </p> 
 <p><a name="an_28:28">28:28</a> Atan ihi tura leslak n Sidi Ṛebbi ad ițțunefk i leǧnas nniḍenur nelli ara n wat Isṛail axaṭer nutni a s-semḥessen yerna a t-qeblen. </p> 
 <p><a name="an_28:29">28:29</a> Mi d-yenna Bulus ayagi, ffɣen-d imeqqranen n wat Isṛail, leḥḥun țemjadalen wway gar-asen.  </p> 
 <p><a name="an_28:30">28:30</a> Bulus yeqqim sin iseggasen deg wexxam-nni i gekra. Isṭerḥib s kra n win i d-yețțasen a t-iẓer.  </p> 
 <p><a name="an_28:31">28:31</a> Yețbecciṛ tageldit n Ṛebbi s tlelli, isselmad-asen ayen akk yeɛnan Sidna Ɛisa Lmasiḥ, ulac win i t-iḥebsen.  </p>
