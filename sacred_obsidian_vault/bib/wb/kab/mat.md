  <h3 align="center">Kabyle Bible: Matthew</h3> 
 <p><a name="an_1:1">1:1</a> A ten-an izuṛan n Ɛisa Lmasiḥ yellan si dderya n Sidna Dawed akk-d Sidna Ibṛahim :  </p> 
 <p><a name="an_1:2">1:2</a> Sidna Ibṛahim yeǧǧa-d Isḥaq, Isḥaq yeǧǧa-d Yeɛqub, Yeɛqub yeǧǧa-d Yahuda d watmaten-is.  </p> 
 <p><a name="an_1:3">1:3</a> Yahuda akk-d Tamaṛ ǧǧan-d Fares, akk-d Ziraḥ ; Fares yeǧǧa-d Ḥesrun, Ḥesrun yeǧǧa-d Aram.  </p> 
 <p><a name="an_1:4">1:4</a> Aram yeǧǧa-d Ɛaminadab, Ɛaminadab yeǧǧa-d Naḥsun, Naḥsun yeǧǧa-d Salmun.  </p> 
 <p><a name="an_1:5">1:5</a> Salmun d Raḥab ǧǧan-d Buɛaz, Buɛaz akk-d Rut sɛan-d Ɛubed, Ɛubed yeǧǧa-d Yassa.  </p> 
 <p><a name="an_1:6">1:6</a> Yassa yeǧǧa-d Sidna Dawed, Sidna Dawed yuɣal d agellid, yeǧǧa-d ɣer tmeṭṭut n Uryaḥ Sidna Sliman.  </p> 
 <p><a name="an_1:7">1:7</a> Sidna Sliman yeǧǧa-d Raḥabɛam, Raḥabɛam yeǧǧa-d Abiya, Abiya yeǧǧa-d Asaf,  </p> 
 <p><a name="an_1:8">1:8</a> Asaf yeǧǧa-d Yehucafat ; Yehucafat yeǧǧa-d Yihuram ; Yihuram yeǧǧa-d Ɛuzya ;  </p> 
 <p><a name="an_1:9">1:9</a> Ɛuzya yeǧǧa-d Yuṭam ; Yuṭam yeǧǧa-d Aḥaz ; Aḥaz yeǧǧa-d Ḥizeqya ;  </p> 
 <p><a name="an_1:10">1:10</a> Ḥizeqya yeǧǧa-d Mennac ; Mennac yeǧǧa-d Ɛamun ; Ɛamun yeǧǧa-d Yuciya ;  </p> 
 <p><a name="an_1:11">1:11</a> Yuciya yeǧǧa-d Yixunya akk-d watmaten-is, di lweqt i deg iqaldiyen wwin at Isṛail d imeḥbas ɣer tmurt n Babilun.  </p> 
 <p><a name="an_1:12">1:12</a> Mbeɛd lweqt n usgiǧǧi ɣer tmurtn Babilun, Yixunya yeǧǧa-d Calatyel ; Calatyel yeǧǧa-d Zurubabil .  </p> 
 <p><a name="an_1:13">1:13</a> Zurubabil yeǧǧa-d Abihud. Abihud yeǧǧa-d Ilyaqim ; Ilyaqim yeǧǧa-d Ɛazuṛ ; </p> 
 <p><a name="an_1:14">1:14</a> Ɛazuṛ yeǧǧa-d Saduq ; Saduq yeǧǧa-d Yaxin ; Yaxin yeǧǧa-d Ilihud. </p> 
 <p><a name="an_1:15">1:15</a> Ilhud yeǧǧa-d Ilɛazaṛ ; Ilɛazaṛ yeǧǧa-d Mattan ; Mattan yeǧǧa-d Yeɛqub.  </p> 
 <p><a name="an_1:16">1:16</a> Yeɛqub yeǧǧa-d Yusef, d Yusef agi i d argaz n Meryem i d-yeǧǧan Sidna Ɛisa i gețțusemman Lmasiḥ.  </p> 
 <p><a name="an_1:17">1:17</a> Si lweqt n Sidna Ibṛahim  armi d lweqt n Sidna Dawed,  ɛeddan ṛbeɛṭac n leǧyal. Ɛeddan daɣen ṛbeɛṭac n leǧyal si lweqt n Sidna Dawed armi d lweqt n usgiǧǧi ɣer tmurt n Babilun. Si lweqt-agi n usgiǧǧi armi d lweqt n Lmasiḥ  ɛeddan daɣen ṛbeɛṭac n leǧyal.  </p> 
 <p><a name="an_1:18">1:18</a> Atan wamek i d-ilul Sidna Ɛisa : yemma-s Meryem tella tețwaxḍeb i yiwen wergaz ițțusemman Yusef. S tezmert n Ṛṛuḥ iqedsen,  terfed s tadist uqbel aț-țeddu ț-țislit.  </p> 
 <p><a name="an_1:19">1:19</a> Yusef axḍib-is, yellan d argaz n ṣṣwab, ur yebɣi ara a ț-țicemmet, dɣa yeqsed ad imsefṛaq yid-es s tuffra.  </p> 
 <p><a name="an_1:20">1:20</a> Akken yețxemmim i waya, yers-ed ɣuṛ-es lmelk n Ṛebbi di targit yenna-yas : A Yusef ! A mmi-s n Dawed ! Ur țțaggad ara aț-țesɛuḍ Meryem ț-țameṭṭut-ik, axaṭer llufan yellan di tɛebbuṭ-is, yusa-yas-d s Ṛṛuḥ iqedsen. </p> 
 <p><a name="an_1:21">1:21</a> A d-tesɛu aqcic, a s-tsemmiḍ Ɛisa,  axaṭer d nețța ara iselken lumma-s si ddnubat-nsen.  </p> 
 <p><a name="an_1:22">1:22</a> Ayagi akk yedṛa iwakken ad ițwakemmel wayen i d-yenna Sidi Ṛebbi seg imi n nnbi Iceɛya  :  </p> 
 <p><a name="an_1:23">1:23</a> Ațan tɛezrit aț-țerfed tadist, a d-tesɛu aqcic ad ițțusemmi Imanuwil yeɛni :« Ṛebbi yid-nneɣ.»  </p> 
 <p><a name="an_1:24">1:24</a> Mi i d-yuki Yusef, ixdem ayen i s-d-yenna lmelk n Sidi Ṛebbi, yeqbel Meryem aț-țili ț-țameṭṭut-is ; yewwi-ț-id ɣer wexxam-is. </p> 
 <p><a name="an_1:25">1:25</a> Meɛna ur ț-innul ara armi i d-tesɛa mmi-s iwumi isemma Ɛisa.  </p> 
 <p><a name="an_2:1">2:1</a> Sidna Ɛisa ilul di taddart n Bitelḥem, di tmurt n Yahuda di lweqt n ugellid Hiṛudus ameqqran, kra n imusnawen usan-d si cceṛq ɣer temdint n Lquds, iwakken ad steqsin : </p> 
 <p><a name="an_2:2">2:2</a> anda ilul llufan-nni ara yilin d agellid n wat Isṛail ? Axaṭer nwala-d itri-ines si cceṛq, nusa-d a t-neɛbed.  </p> 
 <p><a name="an_2:3">2:3</a> Agellid Hiṛudus  akk-d imezdaɣ n temdint n Lquds meṛṛa dehcen, yerwi lxaṭer-nsen mi slan s lexbaṛ-agi.  </p> 
 <p><a name="an_2:4">2:4</a> Hiṛudus yesnejmaɛ-ed akk lmuqedmin d lɛulama n wegdud, isteqsa-ten ɣef wemkan anda ara d-ilal Lmasiḥ ara d-yasen.  </p> 
 <p><a name="an_2:5">2:5</a> Nnan-as :  Ahat di tmurt n Yahuda di taddart n Bitelḥem ! Axaṭer atan wayen i gura nnbi Mixa :  </p> 
 <p><a name="an_2:6">2:6</a> I kemm a taddart n Bitelḥem, ur telliḍ ara ț-țaneggarut ger temdinin n Yahuda, axaṭer seg-em ara d-iffeɣ ugellid ara yeksen at Isṛail agdud-iw.  </p> 
 <p><a name="an_2:7">2:7</a> Hiṛudus yessawel i imusnawen nni s tuffra, isteqsa-ten si melmi i walan itri-nni.  </p> 
 <p><a name="an_2:8">2:8</a> Dɣa iceggeɛ-iten ɣer Bitelḥem, yenna-yasen : Nadit lexbaṛ n ṣṣeḥ ɣef weqcic-agi, m'ara t-tafem, init-iyi-d iwakken ad ṛuḥeɣ ula d nekk a t-ɛabdeɣ.  </p> 
 <p><a name="an_2:9">2:9</a> Mi slan i ugellid, ṛuḥen. AAkken kan i ffɣen, walan itri-nni i ẓran di cceṛq yezwar, iteddu zdat-sen. MMi gewweḍ sennig wemkan anda yella weqcic-nni, yeḥbes. </p> 
 <p><a name="an_2:10">2:10</a> Feṛḥen aṭas mi walan itri-nni.  </p> 
 <p><a name="an_2:11">2:11</a> Kecmen ɣer wexxam-nni, walan llufan akk-d yemma-s Meryem ; tɛeǧben s weqcic-nni dɣa seǧǧden zdat-es. FFsin tiyemmusin-nsen, fkan-as tirezfin : ddheb, lebxuṛ akk-d leɛṭeṛ ɣlayen.  </p> 
 <p><a name="an_2:12">2:12</a> Sidi Ṛebbi ixebbeṛ-iten-id di targit ur țțuɣalen ara ɣer Hiṛudus, dɣa uɣalen ɣer tmurt-nsen seg ubrid nniḍen.  </p> 
 <p><a name="an_2:13">2:13</a> Mi ṛuḥen, lmelk n Sidi Ṛebbi idheṛ-as-ed i Yusef di targit, yenna-yas-ed :-- Kker, ddem aqcic akk-d yemma-s, rewlet ɣer tmurt n Maṣeṛ, qqimet dinna alamma nniɣ-ak-ed a d-tuɣaleḍ, axaṭer Hiṛudus ad iqelleb ɣef weqcic-agi iwakken a t-ineɣ.  </p> 
 <p><a name="an_2:14">2:14</a> Yusef ikker yewwi aqcic-nni akk-d yemma-s deg iḍ, yerwel ɣer tmurt n Maṣeṛ.  </p> 
 <p><a name="an_2:15">2:15</a> Yeqqim dinna armi yemmut Hiṛudus, iwakken ad idṛu wayen i d-yenna Sidi Ṛebbi seg imi n nnbi Huceɛ : Ssawleɣ-as i Mmi a d-iffeɣ si tmurt n Maṣer.  </p> 
 <p><a name="an_2:16">2:16</a> Mi gwala Hiṛudus belli kelxen-t imusnawen-nni, ikcem-it zzɛaf d ameqqran, iceggeɛ ɣer taddart n Bitelḥem akk-d ț-țmura i s-d-izzin, ad nɣen arrac meṛṛa yesɛan si sin iseggasen d akessar, s wakken yella leḥsab i s-d-fkan imusnawen-nni.  </p> 
 <p><a name="an_2:17">2:17</a> Dɣa yețwakemmel wayen i d yenna nnbi Irmiya :  </p> 
 <p><a name="an_2:18">2:18</a> Yekker leɛyaḍ di taddart n Rama, nesla i imeṭṭawen d umeǧǧed, d Ṛaḥil i gețrun ɣef warraw-is, tegguma aț-țesbeṛ fell-asen, axaṭer mmuten.  </p> 
 <p><a name="an_2:19">2:19</a> Mi gemmut Hiṛudus ameqqran, lmelk n Sidi Ṛebbi iweḥḥa-d i Yusef di targit mi gella di tmurt n Maṣer, </p> 
 <p><a name="an_2:20">2:20</a> yenna-yas : Kker, ddem aqcic d yemma-s tuɣaleḍ ɣer tmurt n wat Isṛail, axaṭer wid yebɣan ad nɣen aqcic-nni, mmuten.  </p> 
 <p><a name="an_2:21">2:21</a> Yusef iddem aqcic-nni d yemma-s, yuɣal ɣer tmurt n wat Isṛail  </p> 
 <p><a name="an_2:22">2:22</a> Meɛna mi gesla belli d Aṛxilyus mmi-s n Hiṛudus, i gellan d agellid ɣef tmurt n Yahuda, yuggad. SSidi Ṛebbi iɛeggen-as-ed di targit, iṛuḥ ɣer tmurt n Jlili, </p> 
 <p><a name="an_2:23">2:23</a> izdeɣ deg yiwet n taddart ițțusemman Naṣaret, iwakken ad yedṛu wayen i d-nnan lenbiya : Ad ițțusemmi Anaṣari.  </p> 
 <p><a name="an_3:1">3:1</a> Deg wussan-nni, iban-ed Yeḥya aɣeṭṭas, yețbecciṛ deg unezṛuf n tmurt n Yahuda, </p> 
 <p><a name="an_3:2">3:2</a> iqqaṛ-ed : Tubet, beddlet tikli, axaṭer tagelda n igenwan tqeṛṛeb-ed.  </p> 
 <p><a name="an_3:3">3:3</a> Fell-as i d-immeslay nnbi Iceɛya mi d-yenna : Ț-țaɣect n win ițɛeggiḍen di lxali : Heggit abrid n Sidi Ṛebbi, ssemsawit iberdan-is.  </p> 
 <p><a name="an_3:4">3:4</a> Yeḥya yelsa llebsa yețwaxedmen s ccɛeṛ n welɣem, tabagust n weglim ɣef wammas-is, tamɛict-is d ibẓaẓ akk-d tament n lexla.  </p> 
 <p><a name="an_3:5">3:5</a> Imezdaɣ meṛṛa n temdint n Lquds, tamurt n Yahuda akk-d tmura i d-yezzin i wasif n Urdun, țțasen-d ɣuṛ-es </p> 
 <p><a name="an_3:6">3:6</a> iwakken a d-qiṛṛen s ddnubat nsen, nețța yesseɣḍas-iten deg wasif nni n Urdun.  </p> 
 <p><a name="an_3:7">3:7</a> Mi gwala aṭas n at ifariziyen d isaduqiyen i d-ițasen ad țwaɣeḍsen, yenna-yasen : A ccetla n izerman, anwa i kkun isfaqen belli tzemrem aț-țrewlem i lḥisab i d-iteddun ?  </p> 
 <p><a name="an_3:8">3:8</a> Sbeggnet-ed s lecɣal-nwen belli tettubem, tbeddlem tikli.  </p> 
 <p><a name="an_3:9">3:9</a> Ur qqaṛet ara i yiman-nwen : « Sidna Ibṛahim d jeddi-tneɣ » ! Axaṭer, a wen-iniɣ : SSidi Ṛebbi yezmer a d-yefk dderya i Ibṛahim seg idɣaɣen-agi.  </p> 
 <p><a name="an_3:10">3:10</a> Ațan ihi tcaquṛt thegga ɣer izuṛan n ttjuṛ, yal ttejṛa ur d-nețțak ara lfakya lɛali, aț-țețwagzem, aț-țețwaḍeggeṛ ɣer tmes.  </p> 
 <p><a name="an_3:11">3:11</a> Nekk sseɣḍaseɣ-kkun deg waman iwakken aț-țetubem, lameɛna a d-yas yiwen yesɛan tazmert akteṛ-iw, ur uklaleɣ ara a s-fsiɣ ula d arkasen is. Nețța a kkun-yesseɣḍes s Ṛṛuḥ iqedsen ț-țmes.  </p> 
 <p><a name="an_3:12">3:12</a> Tazzert deg ufus-is, ad yebrez annar-is, ad ijmeɛ irden-is ɣer ikuffan, ma d alim a t-yesseṛɣ di tmes ur nxețți.  </p> 
 <p><a name="an_3:13">3:13</a> Sidna Ɛisa yusa-d si tmurt n Jlili ɣer wasif n Urdun iwakken a t- yesseɣḍes Yeḥya.  </p> 
 <p><a name="an_3:14">3:14</a> Yeḥya yugi, yenna-yas : D nekk i geḥwaǧen ad iyi- tesɣeḍseḍ, kečč tusiḍ-ed ɣuṛ-i iwakken a k-sɣeḍseɣ !  </p> 
 <p><a name="an_3:15">3:15</a> Sidna Ɛisa yerra-yas : Anef tura ! Akkagi i glaq a nexdem lebɣi n Sidi Ṛebbi ! DDɣa Yeḥya yunef-as. </p> 
 <p><a name="an_3:16">3:16</a> Mi gețwaɣḍes Sidna Ɛisa, yeffeɣ-ed seg waman ; imiren kan ldin igenwan, yers-ed fell-as Ṛṛuḥ n Sidi Ṛebbi s ṣṣifa n tetbirt.  </p> 
 <p><a name="an_3:17">3:17</a> Tekka-d yiwet n taɣect seg igenwan, tenna-d : «` Wagi d Mmi ameɛzuz, deg-s i gella lfeṛḥ-iw ! »  </p> 
 <p><a name="an_4:1">4:1</a> Imiren Ṛṛuḥ iqedsen yewwi Sidna Ɛisa ɣer unezṛuf iwakken a t-ijeṛṛeb Cciṭan. </p> 
 <p><a name="an_4:2">4:2</a> Mi guẓam ṛebɛin wussan d ṛebɛin wuḍan, yuɣal yelluẓ. </p> 
 <p><a name="an_4:3">4:3</a> Iqeṛṛeb ɣuṛ-es Yeblis yenna-yas : Ma d Mmi-s n Ṛebbi i telliḍ, ini-yasen i yedɣaɣen-agi ad uɣalen d aɣṛum.  </p> 
 <p><a name="an_4:4">4:4</a> Sidna Ɛisa yerra-yas : Yura di tira iqedsen : Mačči s weɣṛum kan ara iɛic wemdan, lameɛna s mkul awal i d-ițasen s ɣuṛ Sidi Ṛebbi.   </p> 
 <p><a name="an_4:5">4:5</a> Cciṭan yewwi-t ɣer temdint n Lquds issers-it ɣef yixef n lǧameɛ iqedsen, </p> 
 <p><a name="an_4:6">4:6</a> yenna-yas : Ma d Mmi-s n Ṛebbi i telliḍ, ḍeggeṛ iman-ik d akessar, anaɣ yura :  Sidi Ṛebbi ad yefk lameṛ i lmalayekkat-is fell-ak,  aa k-awint ger ifassen-nsent iwakken ad ḥadrent aḍar-ik ɣef yedɣaɣe n .  </p> 
 <p><a name="an_4:7">4:7</a> Sidna Ɛisa yerra-yas : Yura daɣen : Ur tețjeṛṛibeḍ ara Sidi Ṛebbi IIllu-inek .  </p> 
 <p><a name="an_4:8">4:8</a> Cciṭan yewwi-t daɣen ɣef wedrar ɛlayen, isken-as-ed tigeldiwin meṛṛa n ddunit d lɛaḍima-nsent, yenna-yas :  </p> 
 <p><a name="an_4:9">4:9</a> A k-tent-fkeɣ meṛṛa ma yella tseǧǧdeḍ zdat-i a yi-tɛebdeḍ.  </p> 
 <p><a name="an_4:10">4:10</a> Sidna Ɛisa yenna-yas : Beɛɛed akkin fell-i a Cciṭan, axaṭer yura : Anagar Sidi Ṛebbi-inek ara tɛebdeḍ , i nețța kan iwumi ara tseǧdeḍ .  </p> 
 <p><a name="an_4:11">4:11</a> Dɣa Cciṭan iṭṭaxeṛ fell-as. Imiren usant-ed lmalayekkat ɣuṛ-es, iwakken a s-qedcent.  </p> 
 <p><a name="an_4:12">4:12</a> Mi gesla s Yeḥya yețwaḥbes, Sidna Ɛisa yuɣal ɣer tmurt n Jlili. </p> 
 <p><a name="an_4:13">4:13</a> Iffeɣ si taddart n Naṣaret, iṛuḥ ad izdeɣ di Kafernaḥum ; ț-țamdint yyellan rrif n lebḥeṛ di leǧwahi n tmura n Zabulun d Nefṭali, </p> 
 <p><a name="an_4:14">4:14</a> iwakken ad yedṛu wayen yenna nnbi Iceɛya :   </p> 
 <p><a name="an_4:15">4:15</a>  A tamurt n Zabulun akk-d Nefṭali, a timura iqeṛben lebḥeṛ akkin i wasif n Urdun , a tamurt n Jlili i deg zedɣen leǧnas ur nelli ara n wat Isṛail,  </p> 
 <p><a name="an_4:16">4:16</a> agdud-nni yezgan di ṭṭlam, iwala tafat tameqqrant, wid izedɣen di ṭṭlam n lmut, tceṛq-ed fell-asen tafat !   </p> 
 <p><a name="an_4:17">4:17</a> Seg imiren, Sidna Ɛisa yebda yețbecciṛ yeqqaṛ : Tubet, uɣalet-ed ɣer webrid, axaṭer tagelda n igenwan tqeṛṛeb-ed.  </p> 
 <p><a name="an_4:18">4:18</a> Mi gțeddu Sidna Ɛisa ɣef rrif n lebḥeṛ n Jlili, iwala sin iḥewwaten : SSemɛun ițțusemman Buṭrus akk-d gma-s Andriyus. Llan ṭeggiṛen icebbaken-nsen ɣer lebḥeṛ, țṣeggiḍen. </p> 
 <p><a name="an_4:19">4:19</a> Yenna-yasen : Ddut-ed yid-i, a kkun-rreɣ d iṣeggaḍen n yemdanen.  </p> 
 <p><a name="an_4:20">4:20</a> Dɣa imiren kan, ǧǧan icebbaken nsen, ddan yid-es.  </p> 
 <p><a name="an_4:21">4:21</a> Mi gerna yelḥa kra, iwala sin watmaten nniḍen : Yeɛqub d Yuḥenna yellan d arraw n Zabadi. LLlan akk-d baba-tsen di teflukt, țxiḍin icebbaken-nsen.  </p> 
 <p><a name="an_4:22">4:22</a> Yessawel-asen, imiren kan ǧǧan dinna baba-tsen, taflukt-nni, ṛuḥen ddan yid-es. </p> 
 <p><a name="an_4:23">4:23</a> Syenna, Sidna Ɛisa yekka-d tamurt n Jlili meṛṛa, yesselmad di leǧwameɛ n wat Isṛail, yețbecciṛ lexbaṛ n lxiṛ n tgeldit n Ṛebbi, isseḥlay yal aṭan d yal leɛyubat n lɣaci.  </p> 
 <p><a name="an_4:24">4:24</a> Slan yis ula di tmurt n Surya meṛṛa ; țțawin-as-ed imuḍan i ghelken si mkul aṭan : wid ițwamelken, wid iwumi yețṛuḥu leɛqel akk-d wukrifen. Sidna Ɛisa isseḥla-ten akk.  </p> 
 <p><a name="an_4:25">4:25</a> D izumal n lɣaci i t-id-itebɛen si tmurt n Jlili, si ɛecṛa n temdinin-nni, si temdint n Lquds, si tmurt n Yahuda akk-d leǧwahi yellan agemmaḍ i wasif n Urdun.  </p> 
 <p><a name="an_5:1">5:1</a> Mi gwala annect-nni n lɣaci, Sidna Ɛisa yuli ɣer wedrar iqqim. Inelmaden-is qeṛṛben ɣuṛ-es, </p> 
 <p><a name="an_5:2">5:2</a> dɣa ibda isselmad-iten :  </p> 
 <p><a name="an_5:3">5:3</a> D iseɛdiyen wid ițeddun s neyya, aaxaṭer tagelda n igenwan d ayla-nsen !  </p> 
 <p><a name="an_5:4">5:4</a> D iseɛdiyen wid ițrun, axaṭer ad țwaṣebbṛen !  </p> 
 <p><a name="an_5:5">5:5</a> D iseɛdiyen wid ḥninen, aaxaṭer ad weṛten tamurt i sen-iwɛed Sidi Ṛebbi !  </p> 
 <p><a name="an_5:6">5:6</a> D iseɛdiyen wid illuẓen, iffuden lḥeqq, axaṭer ad ṛwun !  </p> 
 <p><a name="an_5:7">5:7</a> D iseɛdiyen wid yesɛan ṛṛeḥma deg wulawen-nsen, aaxaṭer ad iḥunn fell-asen Sidi Ṛebbi !  </p> 
 <p><a name="an_5:8">5:8</a> D iseɛdiyen wid iwumi yeṣfa wul, axaṭer ad walin Sidi Ṛebbi !  </p> 
 <p><a name="an_5:9">5:9</a> D iseɛdiyen wid i d-isrusun talwit, aad țțusemmin d arraw n Sidi Ṛebbi !  </p> 
 <p><a name="an_5:10">5:10</a> D iseɛdiyen wid ițțuqehṛen ɣef lḥeqq, aaxaṭer ddewla igenwan d ayla-nsen !  </p> 
 <p><a name="an_5:11">5:11</a> D iseɛdiyen ara tilim, m'ara kkun-regmen, mm'ara tețwaqehṛem, m'ara xedmen deg-wen lbaṭel ɣef ddemma-w.  </p> 
 <p><a name="an_5:12">5:12</a> Feṛḥet, ilit di lfeṛḥ, axaṭer ṛṛezq-nwen d ameqqran deg igenwan, aakka i țwaqehṛen lenbiya i kkun-id izwaren.  </p> 
 <p><a name="an_5:13">5:13</a> D kunwi i d lmelḥ n ddunit, lameɛna ma tṛuḥ-as lbenna i lmelḥ, s wacu ara s-ț-id-nerr ? Yelha kan ma nḍeggeṛ-it ɣer beṛṛa a t-rekkḍen yemdanen. </p> 
 <p><a name="an_5:14">5:14</a> D kunwi i ț-țafat n ddunit ; taddart yellan ɣef wudrar ulamek ara teffer !  </p> 
 <p><a name="an_5:15">5:15</a> Akken daɣen, ur nceɛɛel ara taftilt iwakken a ț-nɣumm s kra, meɛna a ț-nessers ɣef lmeṣbeḥ, iwakken aț-țfeǧǧeǧ i wid akk yellan deg wexxam. </p> 
 <p><a name="an_5:16">5:16</a> Akka i glaq aț-țecceɛceɛ tafat-nwen zdat yemdanen, iwakken ad walin lecɣal-nwen ilhan, yerna ad ḥemden Baba-twen yellan deg igenwan.  </p> 
 <p><a name="an_5:17">5:17</a> Ɣuṛ-wat aț-țɣilem usiɣ-ed ad sseɣliɣ ayen i d-tenna ccariɛa d wayen i d-nnan lenbiya ! Ur d-usiɣ ara ad sseɣliɣ, lameɛna usiɣ-ed ad snekmaleɣ.  </p> 
 <p><a name="an_5:18">5:18</a> A wen-iniɣ tideț : skud mazal igenwan d lqaɛa, ula d yiwen wawal neɣ usekkil n ccariɛa ur imeḥḥu, alamma yedṛa wayen akk yuran deg-s.  </p> 
 <p><a name="an_5:19">5:19</a> Win ur nqudeṛ ara ula d yiwen n lameṛ amecṭuḥ n ccariɛa, yerna isselmad i wiyaḍ ad xedmen am nețța, ad ițwaḥseb d amecṭuḥ akk di tgelda n igenwan. MMa d win ixeddmen ayen i d-tenna ccariɛa, yerna isselmad i wiyaḍ ad xedmen akken, ad yili d ameqqran di tgelda n igenwan.  </p> 
 <p><a name="an_5:20">5:20</a> Axaṭer a wen-iniɣ, ma yella ur tuɣem ara awal i Sidi Ṛebbi akteṛ n wakken tețțaɣem awal i yimusnawen n ccariɛa d ifariziyen, ur tkeččmem ara tagelda n igenwan.  </p> 
 <p><a name="an_5:21">5:21</a> Teslam daɣen s leqwanen i d- ițțunefken i lejdud-nneɣ : Ur tneqqeḍ ara tamgeṛt, WWin ara yenɣen tamgeṛṭ, ad iɛeddi di ccṛeɛ, ad ițwaḥkem fell-as.  </p> 
 <p><a name="an_5:22">5:22</a> Ma d nekk a wen-iniɣ : kra n win ara izeɛfen ɣef gma-s, ad iɛeddi di ccṛeɛ. Win ara yinin i gma-s : « ay abuhal, » ad ibedd zdat wesqamu n ccraɛ. Win ara yinin i gma-s : « ay amehbul, » yuklal ad ikcem ɣer ǧahennama.  </p> 
 <p><a name="an_5:23">5:23</a> M'ara tțedduḍ aț-țefkeḍ lweɛda i Sidi Ṛebbi, temmektaḍ-ed zdat n wemkan n iseflawen belli gma-k iṭṭef-ak cceḥna, </p> 
 <p><a name="an_5:24">5:24</a> eǧǧ dinna lweɛda-nni, tṛuḥeḍ uqbel aț-ḍelbeḍ ssmaḥ i gma-k, d wamek ara tefkeḍ lweɛda-k i Sidi Ṛebbi.  </p> 
 <p><a name="an_5:25">5:25</a> Ma yella iḍlem-ik yiwen, tedduklem di sin ɣer wexxam n ccṛeɛ, ɣiwel msefham kečč yid-es deg ubrid, m'ulac axṣim-ik a k-yawi ɣer lḥakem, lḥakem a k-yefk i wɛessas, aɛessas-nni, a k-yerr ɣer lḥebs. </p> 
 <p><a name="an_5:26">5:26</a> A k-iniɣ tideț : ur d-țeffɣeḍ ara syenna alamma txellṣeḍ aṣurdi aneggaru n ṭṭlaba-inek.  </p> 
 <p><a name="an_5:27">5:27</a> Teslam belli qqaṛen-d : ur xeddem ara asekkak yeɛni zzna.  </p> 
 <p><a name="an_5:28">5:28</a> Lameɛna, nekkini a wen-d-iniɣ : win ara imuqlen tameṭṭut, imenna deg ul-is ad iznu yid-es, atan am akken izna yid-es. </p> 
 <p><a name="an_5:29">5:29</a> Daymi ma yella ț-țiṭ-ik tayeffust i k-ițawin ɣer leḥṛam, qleɛ-iț, ḍeggeṛ-iț akkin fell-ak. Axaṭer axiṛ-ak a k-iṛuḥ yiwen si lemfaṣel-ik wala aț-țkecmeḍ s lekmal-ik ɣer ǧahennama. </p> 
 <p><a name="an_5:30">5:30</a> Ma yella s ufus-ik ayeffus i txeddmeḍ leḥṛam, gzem-it, tḍegṛeḍ-t akkin fell-ak, axiṛ-ak a k-ixaṣ ufus wala ma tkecmeḍ s lekmal-ik ɣer ǧahennama.  </p> 
 <p><a name="an_5:31">5:31</a> Qqaṛen-d daɣen : Win ara yebrun i tmeṭṭut-is, a s-yefk lkaɣeḍ n berru .  </p> 
 <p><a name="an_5:32">5:32</a> Lameɛna nekkini a wen-iniɣ : win ara yebrun i tmeṭṭut-is, yili ur tezni ara, d nețța i ț-iwelhen ɣer zzna ma tezweǧ d wayeḍ. Daɣen win yuɣen tameṭṭut innebran, ula d nețța yezna.  </p> 
 <p><a name="an_5:33">5:33</a> Teslam daɣen s wayen i d-nnan i lejdud-nneɣ : Ur ḥennet ara, lameɛna ayen i tɛuhdeḍ s limin zdat Sidi Ṛebbi, ilaq a t-txedmeḍ.  </p> 
 <p><a name="an_5:34">5:34</a> Meɛna nekk a wen-iniɣ : ur țgallat ara maḍi ! Ur țgallat s igenni, axaṭer d amkan n lḥekma n Sidi Ṛebbi, </p> 
 <p><a name="an_5:35">5:35</a> ur țgallat s lqaɛa axaṭer fell-as i gesrusu iḍaṛṛen-is, neɣ s Lquds imi ț-țamdint n ugellid ameqqran.  </p> 
 <p><a name="an_5:36">5:36</a> Ur țgallat ara daɣen : « s yixef n uqeṛṛuy-nwen » axaṭer ur tezmirem ara aț-țerrem ula d yiwen wenẓad d aberkan neɣ d amellal. </p> 
 <p><a name="an_5:37">5:37</a> Init « ih » ma ilaq aț-ținim ih, neɣ « xaṭi » ma ilaq aț-ținim xaṭi ; ayen akk ara ternum d zzyada, s ɣuṛ Cciṭan i d-ițas.  </p> 
 <p><a name="an_5:38">5:38</a> Teslam s wayen i d-qqaṛen : Tiṭ s tiṭ, tuɣmest s tuɣmest.  </p> 
 <p><a name="an_5:39">5:39</a> Meɛna nekk a wen-iniɣ : ma yella win i wen-ixedmen cceṛ ur țțarat ara țțaṛ. Ma iwwet-ik yiwen ɣer lḥenk ayeffus, sken-as lḥenk nniḍen.  </p> 
 <p><a name="an_5:40">5:40</a> Ma yebɣa yiwen a k-isiweḍ ɣer ccṛeɛ iwakken a k-ikkes aqenduṛ-ik, rnu-yas ula d abeṛnus-ik.  </p> 
 <p><a name="an_5:41">5:41</a> Ma yella iḥṛes-ik walebɛaḍ a s-tbibbeḍ taɛkumt azgen webrid-is, kečč awi-yas-ț armi d axxam. </p> 
 <p><a name="an_5:42">5:42</a> Efk-as ayen yeḥwaǧ i win i k-d-issutren, ur reggel ara ɣef win ibɣan ad yerḍel s ɣuṛ-ek.  </p> 
 <p><a name="an_5:43">5:43</a> Teslam i wayen i d-qqaṛen :  Ḥemmel amdakkel-ik, tkeṛheḍ aɛdaw ik.  </p> 
 <p><a name="an_5:44">5:44</a> Nekk a wen-iniɣ : ḥemmlet iɛdawen-nwen i kkun ițqehhiṛen. BBarket wid i kkun-ineɛlen, xedmet lxiṛ i wid i kkun-ikeṛhen, dɛut s lxiṛ i wid i kkun-iḍelmen, akk-d wid i kkun-ițqehhiṛen.  </p> 
 <p><a name="an_5:45">5:45</a> S wakka ara d-tbanem d arraw n Baba-twen yellan deg igenwan, axaṭer nețța icṛeq-ed iṭij-is ama ɣef wid yelhan ama ɣef yemcumen, yețțak-ed ageffur ( lehwa ) ama i wid ixeddmen lxiṛ, ama i wid ixeddmen cceṛ.  </p> 
 <p><a name="an_5:46">5:46</a> Ma yella tḥemmlem kan wid i kkun-iḥemmlen, d acu n lfayda ara tesɛum ? Ula d imekkasen n tebzert ( leɣṛama ) xeddmen akenni.  </p> 
 <p><a name="an_5:47">5:47</a> Ma yella tețțemsalamem d watmaten-nwen kan, d acu n lxiṛ i txedmem ? Ula d imednuben xeddmen akken. </p> 
 <p><a name="an_5:48">5:48</a> Sfut ihi akken yeṣfa Baba-twen n igenwan.  </p> 
 <p><a name="an_6:1">6:1</a> Ayen txeddmem i Sidi Ṛebbi, ur t-id-sbegginet ara iwakken a kkun-walin medden, m'ulac ur tseɛɛum ara lfayda ɣer Baba-twen yellan deg igenwan.  </p> 
 <p><a name="an_6:2">6:2</a> M'ara tseddqeḍ, ur țberriḥ ara am akken xeddmen at sin wudmawen di leǧwameɛ neɣ deg iberdan, iwakken ad sɛun ccan ɣer yemdanen. A wen iniɣ tideț : d ayagi kan i d lfayda-nsen.  </p> 
 <p><a name="an_6:3">6:3</a> Ma d kečč m'ara tseddqeḍ, ilaq afus-ik ayeffus ur ițwali ara wayen ixeddem ufus-ik azelmaḍ ; </p> 
 <p><a name="an_6:4">6:4</a> iwakken ssadaqa-inek aț-țeqqim di sser, akka ara tețwaqebleḍ ɣer Baba Ṛebbi yețwalin ayen txeddmeḍ di sser.  </p> 
 <p><a name="an_6:5">6:5</a> M'ara tețẓallam, ur xeddmet ara am at sin wudmawen iḥemmlen ad ẓallen s ibeddi di leǧwameɛ neɣ deg iberdan iwakken a ten-walin medden. AA wen-iniɣ tideț : d ayagi kan i d lfayda-nsen.  </p> 
 <p><a name="an_6:6">6:6</a> Ma d kečč m'ara tebɣuḍ aț-țẓalleḍ, ṛuḥ ekcem ɣer texxamt-ik, sekkeṛ tawwurt, tedɛuḍ di sser ɣer Baba Ṛebbi, nețța yețwalin ayen yellan di sser, a k-id-iqbel.  </p> 
 <p><a name="an_6:7">6:7</a> M'ara tețẓallam, ur sṭuqqutet ara lehduṛ am akken xeddmen at ddunit, ɣilen s lketṛa n imeslayen ara d țwaqeblen.  </p> 
 <p><a name="an_6:8">6:8</a> Ur ten-țɛanadet ara, axaṭer Baba Ṛebbi yeẓra d acu teḥwaǧem uqbel a s-t-tessutrem.  </p> 
 <p><a name="an_6:9">6:9</a> Atah wamek ilaq aț-țețẓallam :  AA Baba-tneɣ yellan deg igenwan, iisem-ik ad ițwaqeddes,  </p> 
 <p><a name="an_6:10">6:10</a> lḥekma n tgeldit-ik a d-tass, llebɣi-k ad idṛu di lqaɛa aakken yedṛa deg igenwan.  </p> 
 <p><a name="an_6:11">6:11</a> Efk-aɣ-d mkul ass tamɛict-nneɣ.  </p> 
 <p><a name="an_6:12">6:12</a> Semmeḥ-aɣ ddnubat-nneɣ aakken i nețsamaḥ i wid i ɣ-iḍelmen.  </p> 
 <p><a name="an_6:13">6:13</a> Ssebɛed fell-aɣ ajeṛṛeb, ssellek-aɣ si tḥila n Cciṭan. AAxaṭer ɣuṛ-ek i tella tgeldit, ttazmert d lɛaḍima i dayem. Amin !  </p> 
 <p><a name="an_6:14">6:14</a> Ma yella tețsamaḥem i yemdanen ii wen-ixeddmen cceṛ, Baba Ṛebbi yyellan deg igenwan a wen-isameḥ uula i kunwi. </p> 
 <p><a name="an_6:15">6:15</a> Lameɛna ma yella kkunwi ur tețsamaḥem ara i wiyaḍ, aatan Baba Ṛebbi ur a wen-ițsamaḥ aara ula i kunwi ddnubat-nwen.  </p> 
 <p><a name="an_6:16">6:16</a> Asmi ara tuẓummem, ur d tesbegginet ara iman-nwen tḥeznem am at sin wudmawen issexsaṛen udmawen-nsen iwakken a ten-țwalin yemdanen belli uẓamen. A wen-iniɣ tideț : d ayagi kan i d lfayda-nsen.  </p> 
 <p><a name="an_6:17">6:17</a> Ma d kečč, m'ara tuẓumeḍ, ssired udem-ik, tdehneḍ aqeṛṛuy-ik s rriḥa, </p> 
 <p><a name="an_6:18">6:18</a> iwakken yiwen ur k-iɛeqqel belli tuẓameḍ. Ilaq ayagi ad iqqim gar-ak d Baba Ṛebbi ițwalin ayen yellan di sser, nețța a k-id-iqbel.  </p> 
 <p><a name="an_6:19">6:19</a> Ur ssexzanet ara igerrujen di ddunit anda ara ten-yečč ubeɛɛuc d ṣṣdiḍ, anda ara ten-akren wid ifettken leḥyuḍ.  </p> 
 <p><a name="an_6:20">6:20</a> Meɛna sxeznet axiṛ igerrujen deg igenwan anda ur llin ibeɛɛac d ṣṣdiḍ isserkuyen, anda ur zmiren ara imakaren ad fetken neɣ ad akren.  </p> 
 <p><a name="an_6:21">6:21</a> Axaṭer anda yella ugerruj-ik, dinna ara yili wul-ik.  </p> 
 <p><a name="an_6:22">6:22</a> Allen, ț-țiftilin n lǧețța, ma yella seḥḥant wallen-ik, aț-țilliḍ s lekmal-ik di tafat.  </p> 
 <p><a name="an_6:23">6:23</a> Lameɛna, ma yella allen-ik ur seḥḥant ara, aț-țiliḍ di ṭṭlam. Ma yella dɣa allen-ik ur seḥḥant ara, tafat yellan deg-k d ṭṭlam, acḥal ihi berrik ṭṭlam i deg telliḍ.  </p> 
 <p><a name="an_6:24">6:24</a> Ur izmir yiwen ad iqdec ɣef sin imɛellmen deg yiwet n tikkelt. Ma iḥemmel yiwen, ad ikṛeh wayeḍ ; ma yeṭṭef deg yiwen, ad iḥqeṛ wayeḍ. Ur tezmirem ara ihi aț-țɛebdem Ṛebbi akk-d idrimen !  </p> 
 <p><a name="an_6:25">6:25</a> Daymi i wen-d-qqaṛeɣ : ur țḥebbiṛet ara i tudert-nwen, ɣef wayen ara teččem d wayen ara teswem akk-d wayen ara telsem. Acu i gesɛan azal, ț-țudert neɣ d lqut ? D lǧețța neɣ d llebsa ? Tudert tugar lqut, lǧețța tugar llebsa !  </p> 
 <p><a name="an_6:26">6:26</a> Walit igṭaṭ ( ifṛax ) deg igenwan : ur zerrɛen, ur meggren, ur jemmɛen ula d acemma ɣer ikuffan, meɛna Baba-twen yellan deg igenwan, yețțak-asen-d tamɛict-nsen. Eɛni ur teswim ara akteṛ n yefṛax ?  </p> 
 <p><a name="an_6:27">6:27</a> Anwa i gzemren s uḥebbeṛ, ad yernu kra n wussan i leɛmeṛ-is ? </p> 
 <p><a name="an_6:28">6:28</a> Iwacu ara tḥebbṛem ɣef llebsa ? Walit amek i gemmun ijeǧǧigen n lexla : ur zeṭṭen, ur țellmen, </p> 
 <p><a name="an_6:29">6:29</a> lameɛna a wen-iniɣ : ula d agellid Sliman s yiman-is di ccan-is ameqqran, ur yelsi am yiwen seg-sen.  </p> 
 <p><a name="an_6:30">6:30</a> Ma yella Ṛebbi islusuy akka leḥcic yellan di lexla ass-agi, azekka ad ițwadeggeṛ ɣer tmes, amek ur kkun-islusu ara ula d kunwi ay imdanen ixuṣṣen di liman ?  </p> 
 <p><a name="an_6:31">6:31</a> Kkset anezgum i yiman-nwen, ur qqaṛet ara : d acu ara nečč, d acu ara nsew neɣ d acu ara nels ?  </p> 
 <p><a name="an_6:32">6:32</a> Axaṭer ayagi d imejhal i gețḥebbiṛen fell-as. Ma d Baba-twen yellan deg igenwan, yeẓra ayen akk teḥwaǧem !  </p> 
 <p><a name="an_6:33">6:33</a> Nadit uqbel ɣef tgeldit n Sidi Ṛebbi d wayen yellan d lḥeqq, ayen nniḍen meṛṛa d nețța ara wen-t-id yefken.  </p> 
 <p><a name="an_6:34">6:34</a> Ur țḥebbiṛet ara ihi i uzekka, axaṭer azekka s unezgum-is weḥd-es. Yal ass s leɛtab-ines !  </p> 
 <p><a name="an_7:1">7:1</a> Ur țḥasabet ara wiyaḍ iwakken ur kkun-ițḥasab ula d yiwen. </p> 
 <p><a name="an_7:2">7:2</a> Akken tețḥasabem wiyaḍ ara kkun-iḥaseb Sidi Ṛebbi. A wen-d-iktil s lkil s wayes tețțektilim i wiyaḍ.  </p> 
 <p><a name="an_7:3">7:3</a> Acuɣeṛ tețmuqquleḍ axeclaw yellan di tiṭ n gma-k, kečč ur nețwali ara tigejdit yellan di tiṭ-ik ? </p> 
 <p><a name="an_7:4">7:4</a> Amek ara tiniḍ i gma-k : « Eǧǧ-iyi ad kkseɣ axeclaw-nni yellan di tiṭ-ik, » kečč yesɛan tigejdit di tiṭ-ik ?  </p> 
 <p><a name="an_7:5">7:5</a> A bu sin wudmawen ! Ekkes uqbel tigejdit yellan di tiṭ-ik, imiren aț-țwaliḍ amek ara d-tekkseḍ axeclaw yellan di tiṭ n gma-k.  </p> 
 <p><a name="an_7:6">7:6</a> Ur țțaket ara ayen yeṣfan i yeqjan, ur ṭeggiṛet ara tiɛeqcin ɣlayen zdat yilfan, m'ulac a ten ṛekḍen yerna a d-zzin fell-awen a kkun-gezren.  </p> 
 <p><a name="an_7:7">7:7</a> Ssutret, a wen-d-ițunefk ! Nadit, aț-țafem ! Sqerbebbet, a wen-d-ldin ! </p> 
 <p><a name="an_7:8">7:8</a> Axaṭer kra n win ara issutren, a s-d-ițțunefk ! Win ițnadin, ad yaf ! A s-d-teldi tewwurt i win ara isṭebṭben.  </p> 
 <p><a name="an_7:9">7:9</a> Anwa deg-wen ara yefken ablaḍ i mmi-s ma yessuter-as-d aɣṛum ? </p> 
 <p><a name="an_7:10">7:10</a> Neɣ, anwa deg-wen ara yefken azrem i mmi-s ma yella yessuter-as-d aslem ? </p> 
 <p><a name="an_7:11">7:11</a> Ma yella kunwi yellan d imcumen tessnem aț-țefkem ayen yelhan i warraw-nwen, amek Baba twen yellan deg igenwan ur d-ițțak ara ayen yelhan i wid ara s-t-issutren ?  </p> 
 <p><a name="an_7:12">7:12</a> Ihi, xedmet i wiyaḍ ayen i tebɣam a wen-t-xedmen i kunwi, axaṭer akka i ɣ-d-tweṣṣa ccariɛa n Musa akk-d lenbiya.  </p> 
 <p><a name="an_7:13">7:13</a> Kecmet si tewwurt iḍeyqen ! Axaṭer acḥal tewseɛ tewwurt, acḥal yeshel webrid yețțawin ɣer nnger, yerna aṭas i gețɛeddayen syenna. </p> 
 <p><a name="an_7:14">7:14</a> Meɛna acḥal teḍyeq tewwurt, acḥal yewɛeṛ webrid yețțawin ɣer tudert yerna drus i gețɛeddayen syenna.  </p> 
 <p><a name="an_7:15">7:15</a> Ḥadret iman-nwen si lenbiya n lekdeb ! Țțasen-d ɣuṛ-wen am izamaren, nutni ɣer daxel d uccanen. </p> 
 <p><a name="an_7:16">7:16</a> A ten-tɛeqlem s lecɣal-nsen. UUr d-nțekkes ara tiẓurin seg inijel, neɣ lexṛif seg isennanen. </p> 
 <p><a name="an_7:17">7:17</a> Ttejṛa yelhan, tețțak-ed lfakya yelhan, yir ttejṛa tețțak-ed lfakya n diri.  </p> 
 <p><a name="an_7:18">7:18</a> Ttejṛa yelhan ur tezmir ara a d-tefk yir lfakya, akken daɣen yir ttejṛa ur tezmir ara a d-tefk lfakya yelhan.  </p> 
 <p><a name="an_7:19">7:19</a> Yal ttejṛa ur d-nețțak ara lfakya lɛali aț-țețwagzem, aț-țețwaḍeggeṛ ɣer tmes. </p> 
 <p><a name="an_7:20">7:20</a> S wakka, s lecɣal-nsen ara tɛeqlem lenbiya n lekdeb.  </p> 
 <p><a name="an_7:21">7:21</a> Mačči d wid kan i yi-d-iqqaṛen : « A Sidi, a Sidi » ara ikecmen ɣer tgelda n igenwan, meɛna d wid ixeddmen lebɣi n Baba yellan deg igenwan.  </p> 
 <p><a name="an_7:22">7:22</a> Aṭas ara yi-d-yinin ass n lḥisab : « a Sidi a Sidi, s yisem-ik i nbecceṛ imeslayen s ɣuṛ Ṛebbi, s yisem-ik i nessufeɣ leǧnun, s yisem-ik i nexdem aṭas n lbeṛhanat ! » </p> 
 <p><a name="an_7:23">7:23</a> Dɣa a sen-d-rreɣ : beɛdet akkin fell-i a wid ixeddmen cceṛ, ur kkun ssineɣ ara !  </p> 
 <p><a name="an_7:24">7:24</a> Akka, s kra n win yesmeḥsisen i wawal-iw yerna ixeddem-it, ițemcabi ɣer wemdan aɛeqli yebnan lsas n wexxam-is ɣef wezṛu.  </p> 
 <p><a name="an_7:25">7:25</a> Ageffur ( lehwa ) yeɣli-d, isaffen ḥemlen-d, aḍu ihubb-ed ɣef wexxam-nni, meɛna ur yeɣli ara, axaṭer lsas-is yers ɣef wezṛu. </p> 
 <p><a name="an_7:26">7:26</a> Ma d win ismeḥsisen i wawal iw, ur nxeddem ara wayen i d-qqaṛeɣ, icuba ɣer wemdan ur nețxemmim ara, i gebnan axxam-is ɣef ṛṛmel ;  </p> 
 <p><a name="an_7:27">7:27</a> yeɣli-d ugeffur, ḥemlen-d isaffen, ihubb-ed waḍu ɣef wexxam-nni, imiren yeɣli. Lexsaṛa-s ț-țameqqrant !  </p> 
 <p><a name="an_7:28">7:28</a> Mi gfukk Sidna Ɛisa aselmed, lɣaci akk wehmen deg imeslayen-ines, </p> 
 <p><a name="an_7:29">7:29</a> axaṭer isselmad-iten s tissas, mačči am lecyux-nsen.  </p> 
 <p><a name="an_8:1">8:1</a> Mi d-yuder Sidna Ɛisa seg udrar, aṭas n lɣaci i t-id-itebɛen. </p> 
 <p><a name="an_8:2">8:2</a> Ataya yiwen wemdan ihelken lbeṛs yusa-d ɣuṛ-es, iseǧǧed zdat-es, yenna-yas : A Sidi, ma yella tebɣiḍ, tzemreḍ a yi-tseḥluḍ. </p> 
 <p><a name="an_8:3">8:3</a> Sidna Ɛisa yessers afus-is fell-as, yenna-yas : Bɣiɣ ! Ili-k teḥliḍ ! IImiren kan iḥla wergaz-nni si lbeṛs-ines, yeṣfa weglim-is.  </p> 
 <p><a name="an_8:4">8:4</a> Dɣa Sidna Ɛisa yenna-yas : Ḥader aț-țeḥkuḍ ula i yiwen ɣef wayagi, meɛna ṛuḥ ɣer yiwen si lmuqedmin a k-iẓer, ad iwali belli teṣfiḍ, tefkeḍ lweɛda-nni n tezdeg s wayes i wen-d-yumeṛ nnbi Musa, iwakken ayagi a sen-yili d țbut.  </p> 
 <p><a name="an_8:5">8:5</a> Akken i gekcem Sidna Ɛisa ɣer temdint n Kafernaḥum, ataya yiwen umeqqran n lɛeskeṛ n Ṛuman iqeṛṛeb ed ɣuṛ-es, iḥellel-it, yenna yas :  </p> 
 <p><a name="an_8:6">8:6</a> A Sidi, aqeddac-iw yenṭer, atan iḍleq deg wexxam, yekref !  </p> 
 <p><a name="an_8:7">8:7</a> Sidna Ɛisa yenna-yas : Ad ṛuḥeɣ a t-sseḥluɣ.  </p> 
 <p><a name="an_8:8">8:8</a> Ameqqran n lɛeskeṛ yerra-yas : A Sidi, ur uklaleɣ ara aț-țkecmeḍ ɣer wexxam-iw, meɛna ini-d kan yiwen n wawal, aqeddac-iw ad iḥlu.  </p> 
 <p><a name="an_8:9">8:9</a> Axaṭer nekk s yiman-iw akken sɛiɣ wid iḥekmen fell-i, i sɛiɣ wid yellan seddaw n lḥekma-w. Ad iniɣ i yiwen ṛuḥ ad iṛuḥ, ad iniɣ i wayeḍ aṛwaḥ a d-yas, ad iniɣ daɣen i uqeddac-iw : xdem aya, a t-ixdem.  </p> 
 <p><a name="an_8:10">8:10</a> Mi gesla Sidna Ɛisa imeslayen-agi itɛeǧǧeb, yenna i wid yellan dinna : A wen-iniɣ tideț, ger wat Isṛail meṛṛa ur ufiɣ ara liman am wagi. </p> 
 <p><a name="an_8:11">8:11</a> A wen-iniɣ daɣen : aṭas ara d-yasen si cceṛq d lɣeṛb, ad ṭṭfen imukan ɣer tama n Ibṛahim, n Isḥaq akk-d Yeɛqub di tgelda n igenwan.  </p> 
 <p><a name="an_8:12">8:12</a> Meɛna aṭas seg wid i gɣilen ad weṛten tagelda ara yețwaḍeggṛen ɣer beṛṛa, ɣer ṭṭlam anda ara ilin imeṭṭawen, nndama tameqqrant akk-d weqṛac n tuɣmas.  </p> 
 <p><a name="an_8:13">8:13</a> Sidna Ɛisa yenna i umeqqran-nni n lɛeskeṛ : Ṛuḥ, imi tumneḍ atan wayen i tḍelbeḍ a k-id-yaweḍ ! DDi teswiɛt-nni, yeḥla uqeddac-is.  </p> 
 <p><a name="an_8:14">8:14</a> Sidna Ɛisa iṛuḥ ɣer wexxam n Buṭrus, yufa taḍeggalt n Buṭrus deg usu tuɣ-iț tawla. </p> 
 <p><a name="an_8:15">8:15</a> Innul afus-is, dɣa teffeɣ-iț tawla-nni. Imiren kan tekker-ed, tebda tqeddec-asen.  </p> 
 <p><a name="an_8:16">8:16</a> Mi geɣli yiṭij, wwin-d i Sidna Ɛisa aṭas n yemdanen ițwamelken. S wawal-is, issufeɣ seg-sen leǧnun yerna yesseḥla imuḍan meṛṛa.  </p> 
 <p><a name="an_8:17">8:17</a> Ixdem ayagi iwakken ad ițwakemmel wawal i d-yenna nnbi Iceɛya : Yewwi leɛyub-nneɣ , iɛebba lehlakat-nneɣ .  </p> 
 <p><a name="an_8:18">8:18</a> Sidna Ɛisa mi gwala annect-nni n lɣaci yezzi-yas-d, yumeṛ i inelmaden-is ad zegren ɣer ugemmaḍ n lebḥeṛ. </p> 
 <p><a name="an_8:19">8:19</a> Yiwen lɛalem n ccariɛa iqeṛṛeb ɣer Sidna Ɛisa, yenna-yas : A Sidi, anda teddiḍ ad dduɣ.  </p> 
 <p><a name="an_8:20">8:20</a> Sidna Ɛisa yerra-yas : Uccanen sɛan lɣiṛan, ifṛax n igenni sɛan leɛcuc, ma d Mmi-s n bunadem ur yesɛi ara anda ara yessers aqeṛṛuy-is. </p> 
 <p><a name="an_8:21">8:21</a> Yiwen seg inelmaden-is, yenna-yas : A Sidi, semmeḥ-iyi ad ṛuḥeɣ ad meḍleɣ baba. </p> 
 <p><a name="an_8:22">8:22</a> Sidna Ɛisa yerra-yas : Ddu yid-i eǧǧ wid yemmuten ad meḍlen lmegtin-nsen.  </p> 
 <p><a name="an_8:23">8:23</a> Sidna Ɛisa yuli ɣer teflukt, ddan yid-es inelmaden-is. </p> 
 <p><a name="an_8:24">8:24</a> Deg yiwet n teswiɛt, tekker-ed tbuciḍant di lebḥeṛ armi tɣumm teflukt-nni s lemwaji. Sidna Ɛisa yella iṭṭes.  </p> 
 <p><a name="an_8:25">8:25</a> Qeṛṛben-d ɣuṛ-es inelmaden-is, ssakin-t-id, nnan-as : A Sidi, sellek-aɣ m'ulac a nemmet !  </p> 
 <p><a name="an_8:26">8:26</a> Yenna-yasen : Acimi tuggadem, a wid iwumi ixuṣṣ liman ! IImiren ikker, yumeṛ i waḍu d lebḥeṛ ad rsen dɣa ters-ed talwit ț-țameqqrant. </p> 
 <p><a name="an_8:27">8:27</a> Wid iḥedṛen, tɛeǧǧben qqaṛen : Anwa-t wagi ? Ula d aḍu d lebḥeṛ țțaɣen-as awal !  </p> 
 <p><a name="an_8:28">8:28</a> Mi gewweḍ Sidna Ɛisa agemmaḍ i lebḥeṛ, ɣer tmurt n at Gadaṛa, a ten-aya sin yergazen yețwazedɣen, ffɣen-d si tmeqbeṛt, mmugren-t-id. Yiwen ur izmir ad iɛeddi seg webrid-nni axaṭer weɛṛen aṭas. </p> 
 <p><a name="an_8:29">8:29</a> Bdan țɛeggiḍen : D acu i tebɣiḍ ɣuṛ-nneɣ a Mmi-s n Ṛebbi ? Tusiḍ-ed iwakken a ɣ-tɛețbeḍ uqbel lweqt ?  </p> 
 <p><a name="an_8:30">8:30</a> Di leǧwahi-nni, tella yiwet n tqeḍɛit tameqqrant n yilfan i gkessen dinna. </p> 
 <p><a name="an_8:31">8:31</a> Leǧnun-nni țḥellilen Sidna Ɛisa, qqaṛen-as : Ma tessufɣeḍ-aɣ, ceggeɛ-aɣ ɣer tqeḍɛit-ihina n yilfan.  </p> 
 <p><a name="an_8:32">8:32</a> Sidna Ɛisa yenna-yasen : Ṛuḥet ! FFfɣen-d si sin-nni n yemdanen, kecmen deg yilfan-nni. IImiren kan, grarben ɣer daxel n lebḥeṛ, ɣeṛqen, mmuten akk. </p> 
 <p><a name="an_8:33">8:33</a> Imeksawen-nni rewlen, ṛuḥen ɣer temdint ad ḥkun ayen akk yedṛan d yergazen-nni yețwamelken.  </p> 
 <p><a name="an_8:34">8:34</a> Imezdaɣ akk n temdint ffɣen-d ɣer Sidna Ɛisa ; mi d-wwḍen ɣuṛ-es, ḥellelen-t ad iffeɣ si tmurt-nsen.  </p> 
 <p><a name="an_9:1">9:1</a> Sidna Ɛisa yuli ɣer teflukt, yezger lebḥeṛ yuɣal ɣer temdint n Kafernaḥum. </p> 
 <p><a name="an_9:2">9:2</a> Wwin-as-d yiwen wukrif iḍleq ɣef wusu. MMi gwala liman n yemdanen-agi, yenna i wukrif-nni : Ur țțaggad ara a mmi, țwaɛeffan ak ddnubat-ik.  </p> 
 <p><a name="an_9:3">9:3</a> Llan dinna kra si lɛulama țxemmimen deg wulawen-nsen qqaṛen : Argaz-agi yekfeṛ !  </p> 
 <p><a name="an_9:4">9:4</a> Dɣa Sidna Ɛisa yeẓran ayen țxemmimen, yenna-yasen :  Acuɣeṛ yir axemmem-agi deg ulawen-nwen ? </p> 
 <p><a name="an_9:5">9:5</a> D acu i gsehlen a t-id-iniɣ : « Țwaɛfan-ak ddnubat-ik neɣ kker telḥuḍ » ?  </p> 
 <p><a name="an_9:6">9:6</a> Meɛna iwakken aț-țeẓrem belli Mmi-s n bunadem yesɛa tazmert di ddunit-agi ad yeɛfu ddnubat... YYenna i wukrif-nni : Ekker, ddem usu-inek tṛuḥeḍ ɣer wexxam-ik !  </p> 
 <p><a name="an_9:7">9:7</a> Imiren, argaz-nni yekker, iṛuḥ ɣer wexxam-is.  </p> 
 <p><a name="an_9:8">9:8</a> Mi walan ayagi, lɣaci akk ikcem-iten lxuf, țḥemmiden Sidi Ṛebbi, imi i d-ifka tazmert am tagi i yemdanen.  </p> 
 <p><a name="an_9:9">9:9</a> Mi gɛedda zdat wexxam anda țxelliṣen tabzert, ( leɣṛama ) Sidna Ɛisa iwala yiwen umekkas isem-is Matta, yenna-yas : Ddu-d yid-i ! IImiren kan ikker Matta, yedda yid-es. </p> 
 <p><a name="an_9:10">9:10</a> Mi llan tețțen deg wexxam n Matta, aṭas n imekkasen akk-d yir imdanen i d-yusan, qqimen akk-d Sidna Ɛisa d inelmaden-is, tețțen yid-sen. </p> 
 <p><a name="an_9:11">9:11</a> Ifariziyen mi walan ayagi nnan i inelmaden-is : Acuɣeṛ Ssid-nwen iteț akk-d imekkasen d yir imdanen ?  </p> 
 <p><a name="an_9:12">9:12</a> Sidna Ɛisa yesla i wayen i d nnan, yerra-yasen : Mačči d wid iseḥḥan i geḥwaǧen ṭṭbib, meɛna d wid ihelken.  </p> 
 <p><a name="an_9:13">9:13</a> Meyzet ɣef lmeɛna n wawal-agi yellan di tira iqedsen : D ulawen yeṣfan i bɣiɣ mačči d iseflawen n lmal. AAxaṭer ur d-usiɣ ara ad ssiwleɣ i iḥeqqiyen iwakken a d uɣalen ɣer webrid, meɛna i imednuben.  </p> 
 <p><a name="an_9:14">9:14</a> Inelmaden n Yeḥya aɣeṭṭas usan-d ɣer Sidna Ɛisa nnan-as : Acuɣeṛ nukkni akk-d ifariziyen nețțuẓum ma d inelmaden-ik ur țțuẓumen ara ?  </p> 
 <p><a name="an_9:15">9:15</a> Sidna Ɛisa yerra-yasen : Eɛni inebgawen n tmeɣṛa zemren ad ilin di leḥzen skud yella yesli yid-sen ? A d-yas wass i deg ara sen-ițwakkes yesli, imiren ad uẓummen.  </p> 
 <p><a name="an_9:16">9:16</a> Ur nezmir ara a nxiḍ tafaweț tajḍiṭ i uceṭṭiḍ aqdim, axaṭer tafaweț-nni tajḍiṭ aț-țejbed aceṭṭiḍ-nni aqdim, acerrig-nni ad innerni.  </p> 
 <p><a name="an_9:17">9:17</a> Ur nețɛemmiṛ ara daɣen iyeddiden iqdimen s ccṛab ajdid, neɣ m'ulac ad fellqen, ccṛab ad inɣel, iyeddiden ad xesṛen ; meɛna a nerr ccṛab ajdid deg yeddiden ijdiden, s wakkenni ccṛab d uyeddid ad țwaḥerzen.  </p> 
 <p><a name="an_9:18">9:18</a> Mazal Sidna Ɛisa ițmeslay, yusa-d yiwen n ccix n lǧameɛ iseǧǧed zdat-es, yenna-yas : Tura kan i temmut yelli, di leɛnaya-k as-ed aț-țserseḍ afus-ik fell-as, a d-teḥyu !  </p> 
 <p><a name="an_9:19">9:19</a> Sidna Ɛisa ikker yedda yid-es nețța d inelmaden-is.  </p> 
 <p><a name="an_9:20">9:20</a> Ațaya yiwet n tmeṭṭut yuḍnen ssiq tnac iseggasen aya, tekka-yas-d si deffir, tennul tacḍaṭ n ubeṛnus-is, </p> 
 <p><a name="an_9:21">9:21</a> axaṭer teqqaṛ deg ul-is : « lemmer zemreɣ ad nnaleɣ ɣas ula ț-țacḍaṭ n ubeṛnus-is, ad ḥluɣ. »  </p> 
 <p><a name="an_9:22">9:22</a> Sidna Ɛisa yezzi ɣer deffir, mi ț-iwala yenna-yas : Ur țțaggad a yell-i, liman-im isseḥla-kkem. SSeg imiren tameṭṭut-nni teḥla.  </p> 
 <p><a name="an_9:23">9:23</a> Mi gewweḍ Sidna Ɛisa ɣer wexxam n ccix-nni, yufa lɣaci țmeǧǧiden, yerna nhewwalen meṛṛa. Yenna-yasen :  </p> 
 <p><a name="an_9:24">9:24</a> Ffɣet syagi, taqcict ur temmut ara, d iḍes kan i teṭṭes. DDɣa țmesxiṛen fell-as. </p> 
 <p><a name="an_9:25">9:25</a> Mi ssufɣen lɣaci, ikcem, yeṭṭef-ed taqcict-nni seg ufus, dɣa tekker-ed.  </p> 
 <p><a name="an_9:26">9:26</a> Yeffeɣ lexbaṛ-agi di tmurt meṛṛa.  </p> 
 <p><a name="an_9:27">9:27</a> Mi gṛuḥ Sidna Ɛisa syenna, tebɛen-t sin iderɣalen, țɛeggiḍen : A mmi-s n Sidna Dawed, ḥunn fell-aɣ ! </p> 
 <p><a name="an_9:28">9:28</a> Mi gewweḍ ɣer wexxam, iderɣalen-nni kecmen ɣuṛ-es, dɣa Sidna Ɛisa yenna-yasen : Tumnem belli zemreɣ a wen xedmeɣ ayen i yi-d-tessutrem ? RRran-as :  Ih a Sidi numen !  </p> 
 <p><a name="an_9:29">9:29</a> Innul allen-nsen, yenna : A wen-ițwaxdem s wakken yella liman-nwen !  </p> 
 <p><a name="an_9:30">9:30</a> Dɣa yuɣal-asen-d yeẓri. Sidna Ɛisa yeggul deg-sen : Ɣuṛ-wat ad iffeɣ lexbaṛ ɣef wayagi !  </p> 
 <p><a name="an_9:31">9:31</a> Meɛna akken kan ffɣen, bdan aberreḥ, lexbaṛ-nni yekka-d meṛṛa tamurt.  </p> 
 <p><a name="an_9:32">9:32</a> Mi ṛuḥen sin-nni, a ten-aya wiyaḍ wwin-as-d yiwen wergaz i gesgugem uṛuḥani. </p> 
 <p><a name="an_9:33">9:33</a> Akken issufeɣ Sidna Ɛisa aṛuḥani-nni, agugam-nni yebda ițmeslay. Lɣaci akk tɛeǧben, qqaṛen : Leɛmeṛ neẓri ayagi di tmurt n wat Isṛail !  </p> 
 <p><a name="an_9:34">9:34</a> Lameɛna ifariziyen qqaṛen : S tezmert n ugellid n leǧnun i ten-issufuɣ.  </p> 
 <p><a name="an_9:35">9:35</a> Sidna Ɛisa yekka-d ɣef temdinin ț-țudrin meṛṛa, yesselmad di leǧwameɛ nsen, yețbecciṛ lexbaṛ n lxiṛ n tgeldit n Sidi Ṛebbi, isseḥlay mkul aṭan d mkul lɛib. </p> 
 <p><a name="an_9:36">9:36</a> Sidna Ɛisa tegzem tasa-s mi gwala annect-nni n lɣaci ɛyan, feclen, ḥeznen am ulli ur nesɛi ameksa .   </p> 
 <p><a name="an_9:37">9:37</a> Dɣa yenna i inelmaden-is : Tamegra meqqṛet, meɛna drus ifellaḥen i gellan.  </p> 
 <p><a name="an_9:38">9:38</a> Ḥellelet ihi bab n tmegra a d iceggeɛ ixeddamen ɣer tmegra-s.  </p> 
 <p><a name="an_10:1">10:1</a> Sidna Ɛisa yessawel i tnac inelmaden-is, yefka-yasen tazmert s wayes ara ssufuɣen leǧnun, ara seḥluyen mkul aṭan d mkul lɛib.  </p> 
 <p><a name="an_10:2">10:2</a> Atnan yismawen n tnac-nni inelmaden n Sidna Ɛisa :Amezwaru d Semɛun ițțusemman Buṭrus akk-d gma-s Andriyus, Yeɛqub mmi-s n Zabadi akk-d gma-s Yuḥenna, </p> 
 <p><a name="an_10:3">10:3</a> Filibus d Bartelmay, Suma akk-d Matta amekkas n tebzert, Yeɛqub mmi-s n Ḥalfi akk-d Taddi ;  </p> 
 <p><a name="an_10:4">10:4</a> Semɛun awaṭani, Yudas n Qeṛyut, win akken ixedɛen Sidna Ɛisa. </p> 
 <p><a name="an_10:5">10:5</a> Sidna Ɛisa iceggeɛ tnac inelmaden-is, iweṣṣa-ten : Ur țṛuḥut ara ɣer tmura tibeṛṛaniyin, ur keččmet ara ɣer temdinin n at Samarya ; </p> 
 <p><a name="an_10:6">10:6</a> ṛuḥet axiṛ ɣer wat Isṛail yellan am ulli iḍaɛen.  </p> 
 <p><a name="an_10:7">10:7</a> M'ara tețțeddum, țbecciṛet qqaṛet : « Ațaya tgeldit n igenwan tqeṛṛeb-ed » ! </p> 
 <p><a name="an_10:8">10:8</a> Sseḥlut imuḍan, sseḥyut-ed lmegtin, ssizdeget wid ihelken lbeṛs, ssufɣet leǧnun. Akken i wen-d-ițțunefk mbla idrimen, fket ula d kunwi mbla idrimen.  </p> 
 <p><a name="an_10:9">10:9</a> Ur țțawit yid-wen ama d ddheb, ama d lfeṭṭa, ama d idrimen ; </p> 
 <p><a name="an_10:10">10:10</a> ur țțawit yid-wen agrab neɣ sin iqendyaṛ, ur țțawit irkasen neɣ aɛekkaz, axaṭer axeddam yuklal lqut-is.  </p> 
 <p><a name="an_10:11">10:11</a> Di yal tamdint neɣ taddart anda ara tkecmem, steqsit ma yella win ara yesṭerḥben yis-wen, qqimet ɣuṛ-es alamma tekkrem aț-țṛuḥem.  </p> 
 <p><a name="an_10:12">10:12</a> M'ara tkecmem ɣer yiwen wexxam, sellmet fell-asen, </p> 
 <p><a name="an_10:13">10:13</a> ma sṭerḥben yis-wen, lehna-nwen aț-țers fell-asen ; ma yella ur sṭerḥben ara yis-wen, lehna-nwen aț-țeqqim ɣuṛ-wen.  </p> 
 <p><a name="an_10:14">10:14</a> M'ur qbilen ara ad sṭreḥben yis-wen neɣ ad semḥessen i wawal, ffɣet seg wexxam-nni neɣ si temdint nni, zwit ula d aɣebbaṛ seg iḍaṛṛen nwen.  </p> 
 <p><a name="an_10:15">10:15</a> A wen-iniɣ tideț : ass n lḥisab, imezdaɣ n temdint-agi ad țțuɛaqben akteṛ n temdinin n Sudum akk-d Gumuṛ.  </p> 
 <p><a name="an_10:16">10:16</a> Atan, a kkun-ceggɛeɣ am ulli ger wuccanen, ḥeṛcet ihi am izerman, sɛut neyya am yetbiren. </p> 
 <p><a name="an_10:17">10:17</a> Ḥadret ɣef yiman-nwen, axaṭer kra n yemdanen a kkun-sbedden ɣer ccṛeɛ; yerna a kkun-wten s ujelkaḍ di leǧwameɛ-nsen.  </p> 
 <p><a name="an_10:18">10:18</a> Ɣef ddemma n yisem-iw, a kkun-sbedden zdat lḥekkam akk-d igelliden, iwakken am nutni am at leǧnas nniḍen, ad slen s yisem-iw. </p> 
 <p><a name="an_10:19">10:19</a> M'ara kkun-ṭṭfen, ur țḥebbiṛet ara ɣef wamek ara temmeslayem, neɣ ɣef wayen ara d-tinim, imeslayen ara d-tinim, a wen-d-țțunefken di teswiɛt nni kan. </p> 
 <p><a name="an_10:20">10:20</a> Axaṭer mačči d kunwi ara imeslayen, meɛna d Ṛṛuḥ n Baba Ṛebbi ara d-imeslayen seg-wen.  </p> 
 <p><a name="an_10:21">10:21</a> Amdan ad yefk gma-s ɣer lmut, ababat ad yefk mmi-s, dderya a d kkren ɣer imawlan-nsen, a ten ssiwḍen ɣer lmut. </p> 
 <p><a name="an_10:22">10:22</a> A kkun-keṛhen irkul ɣef ddemma n yisem-iw, meɛna win ara yeṭṭfen alamma ț-țaggara, ad ițțusellek. </p> 
 <p><a name="an_10:23">10:23</a> M'ara tețwaqehṛem deg yiwet n temdint, rewlet ɣer tayeḍ. A wen-iniɣ tideț : Mmi-s n bunadem a d-yuɣal uqbel a d-tekkem tudrin meṛṛa n tmurt n wat Isṛail.  </p> 
 <p><a name="an_10:24">10:24</a> Ulac anelmad yugaren ccix-is neɣ aqeddac yellan sennig umɛellem is.  </p> 
 <p><a name="an_10:25">10:25</a> Akken ițwaḥseb ccix ara yețwaḥseb unelmad-is. Daɣen akken yella wemɛellem ara yili uqeddac-is. MMa semman i bab n wexxam Balzabul iḥekkmen ɣef leǧnun, amek ur țsemmin ara akkenni i wat wexxam-is !  </p> 
 <p><a name="an_10:26">10:26</a> Ur ten-țaggadet ara ihi ! Axaṭer kra n wayen yeffren a d-iḍheṛ, kra n lbaḍna yellan a d-tban.  </p> 
 <p><a name="an_10:27">10:27</a> Ayen i wen-qqaṛeɣ deg iḍ, init-eț ɛinani deg wass, ayen slan imeẓẓuɣen-nwen di sser, berrḥet-eț deg iberdan.  </p> 
 <p><a name="an_10:28">10:28</a> Ur țțaggadet ara wid ineqqen lǧețța lameɛna ur zmiren ara ad nnɣen ṛṛuḥ, aggadet win izemren ad iḍeggeṛ ṛṛuḥ akk-d lǧețța ɣer ǧahennama. </p> 
 <p><a name="an_10:29">10:29</a> Eɛni ur znuzun ara sin iẓiwcen s uṣuṛdi ? Meɛna ula d yiwen deg-sen ur d-iɣelli ɣer lqaɛa mbla lebɣi n Baba Ṛebbi.  </p> 
 <p><a name="an_10:30">10:30</a> Ula d anẓaden uqeṛṛuy-nwen țwaḥesben irkulli. </p> 
 <p><a name="an_10:31">10:31</a> Ihi ur țțaggadet ara : tesɛam azal akteṛ n waṭas n iẓiwcen !  </p> 
 <p><a name="an_10:32">10:32</a> Kra win ara yessetḥin s yisem-iw zdat yemdanen, ula d nekk, ad ssetḥiɣ yis zdat Baba yellan deg igenwan.  </p> 
 <p><a name="an_10:33">10:33</a> Kra win ara yi-nekṛen zdat yemdanen, a t-nekkṛeɣ ula d nekk zdat Baba yellan deg igenwan.  </p> 
 <p><a name="an_10:34">10:34</a> Ɣuṛ-wat aț-țɣilem d lehna i d wwiɣ ɣer ddunit ; mačči d lehna i d-wwiɣ, meɛna d ccwal i d-wwiɣ.  </p> 
 <p><a name="an_10:35">10:35</a> Axaṭer wwiɣ-d lfiṛaq ger weqcic d baba-s, ger teqcict d yemma-s, ger teslit ț-țemɣaṛt-is,   </p> 
 <p><a name="an_10:36">10:36</a> a d-kkren yeɛdawen seg at wexxam.  </p> 
 <p><a name="an_10:37">10:37</a> Win iḥemmlen baba-s d yemma-s neɣ mmi-s d yelli-s akteṛ iw ur yuklal ara ad yili d anelmad-iw.  </p> 
 <p><a name="an_10:38">10:38</a> Kra win ur neqbil ara ad inɛețțab ɣef ddemma n yisem-iw iwakken ad iyi-itbeɛ, ur yuklal ara ad yili d anelmad-iw.  </p> 
 <p><a name="an_10:39">10:39</a> Win iḥerzen taṛwiḥt-is, a s-tṛuḥ, ma d win ara isebblen taṛwiḥt-is ɣef ddemma-w ad yuɣal a ț-yaf. </p> 
 <p><a name="an_10:40">10:40</a> Win isṭerḥben yis-wen, yis-i i gesṭerḥeb, win isṭerḥben yis-i, isṭerḥeb s win i yi-d-iceggɛen.  </p> 
 <p><a name="an_10:41">10:41</a> Win isṭerḥben s nnbi imi s ɣuṛ Ṛebbi i d-yusa a s-d-ițțunefk wayen yuklal nnbi, win isṭerḥben s uḥeqqi imi d aḥeqqi i gella, a s-d-ițunefk wayen yuklal uḥeqqi.  </p> 
 <p><a name="an_10:42">10:42</a> Kra win ara yefken lkas n waman isemmaḍen i yiwen seg imecṭuḥen-agi imi d anelmad-iw i gella, a wen-iniɣ tideț : ur a s-ițṛuḥ ara lḥeqq yuklal.  </p> 
 <p><a name="an_11:1">11:1</a> Mi gweṣṣa tnac-nni inelmaden is, Sidna Ɛisa iṛuḥ syenna ad ibecceṛ di tudrin n leǧwahi-nni. </p> 
 <p><a name="an_11:2">11:2</a> Yeḥya yellan di lḥebs, yesla s lecɣal n Lmasiḥ, iceggeɛ ɣuṛ-es inelmaden is iwakken a s-inin :  </p> 
 <p><a name="an_11:3">11:3</a> D kečč i d win akken ara d yasen neɣ ilaq-aɣ a nerǧu wayeḍ ?  </p> 
 <p><a name="an_11:4">11:4</a> Sidna Ɛisa yerra-yasen-d : Ṛuḥet ssiwḍet-as lexbaṛ i Yeḥya ɣef wayen twalam d wayen teslam : </p> 
 <p><a name="an_11:5">11:5</a> Iderɣalen țwalin, iquḍaren leḥḥun, ibeṛsiyen ṣeffun, iɛeẓẓugen sellen, lmegtin ḥeggun-d, lexbaṛ n lxiṛ ițțubecceṛ i yimeɣban.  </p> 
 <p><a name="an_11:6">11:6</a> D aseɛdi win iwumi ur țțiliɣ ara d sebba n tuccḍa ( ugur ) !  </p> 
 <p><a name="an_11:7">11:7</a> Mi ṛuḥen inelmaden-is, Sidna Ɛisa yebda iheddeṛ i lɣaci ɣef Yeḥya : D acu i tṛuḥem a t-id-teẓrem deg unezṛuf ? D aɣanim yețhuzzu waḍu ? </p> 
 <p><a name="an_11:8">11:8</a> Ɣer wacu i tṛuḥem aț-țfeṛǧem ? ƔƔer yiwen wergaz yelsan llebsa ifazen ? Wid yețlusun llebsa ifazen, deg ixxamen n igelliden i țțilin.  </p> 
 <p><a name="an_11:9">11:9</a> Anwa i ɣer tṛuḥem a t-id-teẓrem ihi ? D yiwen n nnbi ? Țideț ih, d nnbi ! A wen-iniɣ : yugar nnbi. </p> 
 <p><a name="an_11:10">11:10</a> Axaṭer d nețța i ɣef yura : Atan zewwreɣ zdat-ek amceggeɛ-inu, iwakken ad iheggi abrid zdat-ek.  </p> 
 <p><a name="an_11:11">11:11</a> A wen-iniɣ tideț : ulac mmi-s n tmeṭṭut yugaren Yeḥya aɣeṭṭas. Lameɛna, amecṭuḥ maḍi di tgelda n igenwan, meqqeṛ akteṛ-is.   </p> 
 <p><a name="an_11:12">11:12</a> Seg wussan-nni i deg ițbecciṛ Yeḥya aɣeṭṭas armi ț-țura, imdanen țnaɣen ɣef tgelda n igenwan, bɣan a ț-țkecmen s ddreɛ. </p> 
 <p><a name="an_11:13">11:13</a> Axaṭer ccariɛa n Musa d lenbiya hedṛen-d ɣef tgelda n igenwan armi d lweqt n Yeḥya aɣeṭṭas.  </p> 
 <p><a name="an_11:14">11:14</a> Ma tebɣam aț-țfehmem, d nețța i d Ilyas-nni ilaqen a d-yas. </p> 
 <p><a name="an_11:15">11:15</a> Win yesɛan imeẓẓuɣen, isel-ed !  </p> 
 <p><a name="an_11:16">11:16</a> Ɣer wuɣuṛ ara metleɣ lǧil-agi ? Icuba ɣer warrac yeqqimen deg uzniq, ssawalen i warrac nniḍen, qqaṛen asen :  </p> 
 <p><a name="an_11:17">11:17</a> Newwet ajewwaq uur tecḍiḥem ara, nnecna cnawi n leḥzen uur tmeǧdem ara.  </p> 
 <p><a name="an_11:18">11:18</a> Axaṭer Yeḥya yusa-d, ur iteț ur itess, tennam : « izdeɣ-it uṛuḥani. »  </p> 
 <p><a name="an_11:19">11:19</a> Mmi-s n bunadem yusa-d, iteț, itess am wiyaḍ, teqqaṛem : «` Ițxemmim kan ɣef wučči ț-țissit, d amdakkul n imekkasen ixeddaɛen akk-d yir imdanen » lameɛna tideț tban-ed s lecɣal-is.  </p> 
 <p><a name="an_11:20">11:20</a> Sidna Ɛisa ibda ițlumu timdinin nni i deg ixdem aṭas n lbeṛhanat, axaṭer imezdaɣ-is ur d-uɣalen ara ɣer webrid n Ṛebbi.  </p> 
 <p><a name="an_11:21">11:21</a> A nnger-im a tamdint n Kurazim ! A nnger-im a tamdint n Bitsayda ! Axaṭer lemmer lbeṛhanat yedṛan deg-kkunt dṛan di temdinin n Sur akk-d Sidun, tili imezdaɣ-nsent aṭas aya i sburren tacekkaṛt n leḥzen, qqimen ɣef yiɣiɣden iwakken a d sbegnen belli ndemmen uɣalen-d ɣer ubrid n Ṛebbi. </p> 
 <p><a name="an_11:22">11:22</a> Daymi, imezdaɣ n temdinin-agi ad țuɛaqben akteṛ n wid n temdinin n Sur akk-d Sidun.  </p> 
 <p><a name="an_11:23">11:23</a> Ula d kemm a tamdint n Kafernaḥum, tɣileḍ aț-țaliḍ ɣer igenni ? Xaṭi ! Aț-țețwaḍeggreḍ ɣer laxeṛt ; axaṭer lemmer di temdint n Sudum i dṛan lbeṛhanat i gedṛan ɣuṛ-em, tili mazal-iț ar ass-a !  </p> 
 <p><a name="an_11:24">11:24</a> Daymi, ass n lḥisab anegaru aț-țețțuɛaqbeḍ akteṛ n tmurt n Sudum.  </p> 
 <p><a name="an_11:25">11:25</a> Di lweqt-nni, Sidna Ɛisa inṭeq yenna : Ḥemmdeɣ-k a Baba Ṛebbi, a Bab igenwan d lqaɛa, imi i d tessekneḍ i imecṭaḥ ayen i teffreḍ i imusnawen d lɛuqal. </p> 
 <p><a name="an_11:26">11:26</a> Ih a Baba Ṛebbi ! Ḥemmdeɣ-k imi d wagi i d lebɣi-k. </p> 
 <p><a name="an_11:27">11:27</a> Baba Ṛebbi yerra-d kullec ger ifassen-iw.  YYiwen ur yessin Mmi-s n Ṛebbi anagar Baba Ṛebbi ! Yiwen ur issin daɣen Baba Ṛebbi anagar Mmi-s akk-d win iwumi yebɣa Mmi-s a s-t-id-isken.  </p> 
 <p><a name="an_11:28">11:28</a> Aset-ed ɣuṛ-i a wid akk yeɛyan, iɛebban taɛekkumt ẓẓayen, a wen-fkeɣ talwit d westeɛfu.  </p> 
 <p><a name="an_11:29">11:29</a> Refdet ɣef tuyat-nwen azaglu inu tlemdem s ɣuṛ-i, axaṭer ul-iw yeččuṛ d ṛṛeḥma yerna d aḥnin, aț-țafem talwit i teṛwiḥin-nwen. </p> 
 <p><a name="an_11:30">11:30</a> Azaglu-inu yeshel, taɛekkumt-iw fsuset.  </p> 
 <p><a name="an_12:1">12:1</a> Yiwet n tikkelt, deg wass n ssebt yellan d ass n westeɛfu, Sidna Ɛisa izger igran n yirden. Inelmaden-is lluẓen, bdan țekksen-d tigedrin, tețțen-tent. </p> 
 <p><a name="an_12:2">12:2</a> Ifariziyen mi ten-walan, nnan i Sidna Ɛisa : Acuɣeṛ inelmaden-ik xeddmen ayen iḥeṛmen deg wass n westeɛfu ?  </p> 
 <p><a name="an_12:3">12:3</a> Meɛna Sidna Ɛisa yerra-yasen : Eɛni ur teɣṛim ara wayen yexdem ugellid Dawed, mi gelluẓ nețța d wid yellan yid-es ? </p> 
 <p><a name="an_12:4">12:4</a> Mi gekcem ɣer wexxam iqedsen, yečča seg weɣṛum-nni yețțunefken d lweɛda i lmuqedmin yerna yefka seg-s i imdukkal-is !  </p> 
 <p><a name="an_12:5">12:5</a> Eɛni ur teɣṛim ara di tektabt n ccariɛa n Musa belli lmuqedmin xeddmen ɣas akken d ass n westeɛfu yerna mačči d leḥṛam fell-asen ?  </p> 
 <p><a name="an_12:6">12:6</a> Ihi a wen-d-iniɣ : yella dagi win yugaren axxam iqedsen.  </p> 
 <p><a name="an_12:7">12:7</a> Lemmer tfehmem d acu i d lmeɛna n wawal-agi : D ulawen yesɛan ṛṛeḥma i bɣiɣ mačči d isflawen n lmal ttili ur tḥekkmem ara s lmut ɣef wid ur neḍlim ara !  </p> 
 <p><a name="an_12:8">12:8</a> Axaṭer Mmi-s n bunadem, d nețța i gḥekmen ɣef wass n westeɛfu.  </p> 
 <p><a name="an_12:9">12:9</a> Syenna, yerra ɣer yiwen n lǧameɛ, </p> 
 <p><a name="an_12:10">12:10</a> yella dinna yiwen wergaz iwumi yekref ufus. Ifariziyen steqsan-t nnan-as : Eɛni iḥlel useḥlu deg wass n westeɛfu ? SSteqsan-t akka, iwakken a s-d-afen sebba s wayes ara t-sḍelmen.  </p> 
 <p><a name="an_12:11">12:11</a> Yerra-yasen : Anwa deg-wen yesɛan yiwet n tixsi kan, ma yella teɣli ɣer tesraft deg wass n westeɛfu ur ț-id-issalay ara ? </p> 
 <p><a name="an_12:12">12:12</a> Eɛni tixsi teswa akteṛ n wemdan ? Ihi, yeḥlel a nexdem lxiṛ deg wass n westeɛfu !  </p> 
 <p><a name="an_12:13">12:13</a> Imiren Sidna Ɛisa yenna-yas i wergaz-nni : Sleq-ed afus-ik ! YYeḍleq afus-is, dɣa yuɣal iṣeḥḥa am ufus nniḍen. </p> 
 <p><a name="an_12:14">12:14</a> Ifariziyen ffɣen, ṛuḥen mcawaṛen wway gar-asen amek ara a s-d-afen sebba s wayes ara t-nɣen.  </p> 
 <p><a name="an_12:15">12:15</a> Mi gesla bɣan a t-nɣen, Sidna Ɛisa iṛuḥ syenna. Aṭas i geddan yid-es, yesseḥlay imuḍan meṛṛa, </p> 
 <p><a name="an_12:16">12:16</a> iqqaṛ-asen : Ɣuṛ-wat aț-țessufɣem lexbaṛ !  </p> 
 <p><a name="an_12:17">12:17</a> Iwakken ad idṛu wayen yenna nnbi Iceɛya :  </p> 
 <p><a name="an_12:18">12:18</a> Ataya uqeddac-iw i xtaṛeɣ, win eɛzizen fell-i, deg-s i gella lfeṛḥ-iw. A d-yers Ṛṛuḥ-iw fell-as, ad ibecceṛ lḥeqq i leǧnas.  </p> 
 <p><a name="an_12:19">12:19</a> Ur iskkaray ara ccwal, ur ițɛeggiḍ ara, ur ițḥettim ara iman-is ɣef wiyaḍ.  </p> 
 <p><a name="an_12:20">12:20</a> Ur ițkemmil ara i uɣanim iceqqeqen, ur isnusuy ara taftilt yețmețțaten, alamma yesbedd lḥeqq.  </p> 
 <p><a name="an_12:21">12:21</a> Imiren leǧnas meṛṛa ad ssirmen ɣuṛ-es, fell-as ara țeklen.  </p> 
 <p><a name="an_12:22">12:22</a> Wwin-d i Sidna Ɛisa yiwen wergaz i gesgugem, i gesderɣel uṛuḥani. Yesseḥla-t, yuɣal wergaz-nni iheddeṛ, yețwali. </p> 
 <p><a name="an_12:23">12:23</a> Lɣaci meṛṛa wehmen di Sidna Ɛisa qqaṛen : Mačči d wagi i d mmi-s n Sidna Dawed, win akken ara d-yasen ?  </p> 
 <p><a name="an_12:24">12:24</a> Mi sen-slan ifariziyen nnan : Argaz-agi yessufuɣ leǧnun s tezmert n Balzabul iḥekkmen ɣef leǧnun. </p> 
 <p><a name="an_12:25">12:25</a> Sidna Ɛisa yeẓran ayen i țxemmimen, yenna-yasen : Yal tagelda anda țnaɣen lɣaci wway gar-asen, aț-țuɣal aț-țenger. YYal taddart neɣ yal axxam i deg țnaɣen imezdaɣ wway gar-asen, ur d-yețɣimi ara. </p> 
 <p><a name="an_12:26">12:26</a> Ma yella Cciṭan issufuɣ Cciṭan, atan yebḍa d yiman-is ! Amek ihi ara d-tdum ddewla-s ?  </p> 
 <p><a name="an_12:27">12:27</a> Ma yella nekkini ssufuɣeɣ leǧnun s tezmert n Balzabul, i warraw nwen s wacu i ten ssufuɣen ? Daymi, d arraw-nwen ara kkun-iḥasben ! </p> 
 <p><a name="an_12:28">12:28</a> Meɛna ma yella nekkini s Ṛṛuḥ n Ṛebbi i ssufuɣeɣ leǧnun, ihi tagelda n Ṛebbi ațan tewweḍ-ed armi d ɣuṛ-wen.  </p> 
 <p><a name="an_12:29">12:29</a> Yiwen ur yezmir ad ikcem ɣer wexxam n wergaz iǧehden iwakken a s-iddem ayla-as ma yella ur t-yuriz ara uqbel, meɛna m'ara t-yarez, imiren ad yizmir ad iddem ayen akk yellan deg wexxam-is.  </p> 
 <p><a name="an_12:30">12:30</a> Win ur nelli ara yid-i, ixulef-iyi ; win ur d-njemmeɛ ara yid-i ițḍeggiɛ ! </p> 
 <p><a name="an_12:31">12:31</a> Daymi i wen-qqaṛeɣ : Mkul ddnub d mkul rregmat zemren ad țwasemmḥen i yemdanen, ma d rregmat ɣef Ṛṛuḥ iqedsen ur țwasemmaḥent ara ! </p> 
 <p><a name="an_12:32">12:32</a> Yal win ara iregmen di Mmi-s n bunadem, a s-ițwasameḥ, ma d win ara iregmen Ṛṛuḥ iqedsen, ur s-ițwasamaḥ ara ama di lweqt-agi, ama di lweqt i d-iteddun.  </p> 
 <p><a name="an_12:33">12:33</a> Ɛeqlet ttejṛa ɣer lfakya-s : ma telha ttejṛa, a d-tefk lfakya yelhan, ma yella diri-ț, a d-tefk lfakya n diri ! </p> 
 <p><a name="an_12:34">12:34</a> A ccetla n izerman, amek ara d-tinim ayen yelhan ma yella kunwi diri-kkun ? Axaṭer d ayen yellan deg ul i d-ițeffɣen seg imi ! </p> 
 <p><a name="an_12:35">12:35</a> Argaz lɛali yessufuɣ-ed ayen yelhan seg ugerruj yellan deg ul-is ma d amcum yessufuɣ-ed ayen n diri si lexzin n diri yellan deg ul-is !  </p> 
 <p><a name="an_12:36">12:36</a> Aql-i nniɣ-awen-t : ass n lḥisab, ad țțuḥasben yemdanen ɣef yal awal ur nesɛi lmeɛna i d-iffɣen seg yimi-nsen ; </p> 
 <p><a name="an_12:37">12:37</a> axaṭer ɣef leḥsab n imeslayen-ik ara tețțuḥasbeḍ, d imeslayen-ik ara d-isbeggnen ma d aḥeqqi i telliḍ neɣ d amcum.  </p> 
 <p><a name="an_12:38">12:38</a> Dɣa kra n lɛulama n ccariɛa akk-d ifariziyen nnan i Sidna Ɛisa : A Sidi, nebɣa a k-nwali txedmeḍ yiwen n lbeṛhan.  </p> 
 <p><a name="an_12:39">12:39</a> Sidna Ɛisa yerra yasen : Lǧil am-agi ijehlen ixedɛen Ṛebbi, yessutur lbeṛhan, ur a sen d-yețțunefkay ara lbeṛhan nniḍen anagar win n nnbi Yunes.  </p> 
 <p><a name="an_12:40">12:40</a> Axaṭer akken yesɛedda nnbi Yunes tlata wussan d tlata wuḍan deg uɛebbuḍ n uḥewtiw ameqqran di lebḥeṛ, akken daɣen ara yesɛeddi Mmi-s n bunadem tlata wussan d tlata wuḍan zdaxel n wakal.  </p> 
 <p><a name="an_12:41">12:41</a> Ass n lḥisab, imezdaɣ n temdint n Ninebi a d-kkren ɣer yemdanen n lǧil-agi ad ccetkin fell-asen iwakken ad țțuḥasben, axaṭer nutni uɣalen-d ɣer webrid mi slan i ubecceṛ n nnbi Yunes. AA wen-d-iniɣ : atan yella dagi win yugaren nnbi Yunes !  </p> 
 <p><a name="an_12:42">12:42</a> Ass n lḥisab, tagellit n tmurt n Ḥabac a d-tekker ɣer yemdanen n lǧil-agi aț-țeccetki fell-asen iwakken ad țțuḥasben, axaṭer tusa-d seg yixef n ddunit iwakken aț-țsel i tmusni n ugellid Sliman. AAtan nniɣ-awen yella dagi win yugaren agellid Sliman !  </p> 
 <p><a name="an_12:43">12:43</a> M'ara yețwaqecceɛ uṛuḥani seg yiwen wemdan, ad ițezzi deg imukan yexlan, ad ițqellib anda ara yaf ṛṛaḥa ur yețțaf ara. </p> 
 <p><a name="an_12:44">12:44</a> Dɣa ad yini : Ad uɣaleɣ ɣer umkan ansi i d-ffɣeɣ ! MMi guɣal ɣer wemdan-nni, ad yaf amkan d ilem, zeddig yerna iseggem. </p> 
 <p><a name="an_12:45">12:45</a> Imiren ad iṛuḥ a d-yawi sebɛa iṛuḥaniyen nniḍen i gweɛṛen akteṛ-is, ad kecmen ɣer dinna, ad zedɣen. Lḥala n wergaz-nni aț-țenṭaṛ akteṛ n tikkelt tamezwarut. Akka ara tedṛu d lǧil-agi ijehlen.  </p> 
 <p><a name="an_12:46">12:46</a> Mazal Sidna Ɛisa yețmeslay i lɣaci, mi d-yusa yiwen wergaz yenna-yas : Yemma-k d watmaten-ik atnan di beṛṛa, bɣan a k-ẓren.  </p> 
 <p><a name="an_12:48">12:48</a> Sidna Ɛisa yerra-yas : Anta i d yemma, anwi i d atmaten-iw ?  </p> 
 <p><a name="an_12:49">12:49</a> Iwehha s ufus-is ɣer inelmaden is yenna : D wigi i d yemma d watmaten-iw ! </p> 
 <p><a name="an_12:50">12:50</a> Axaṭer win ixeddmen lebɣi n Baba yellan deg igenwan, d winna i d gma, i d weltma, i d yemma.  </p> 
 <p><a name="an_13:1">13:1</a> Ass-nni kan, Sidna Ɛisa yeffeɣ-ed seg wexxam, iṛuḥ ad iqqim rrif n lebḥeṛ, </p> 
 <p><a name="an_13:2">13:2</a> d izumal n lɣaci i s-d-izzin, dɣa yuli ɣer teflukt, yeqqim. LLɣaci ibedd ɣef rrif n lebḥeṛ, </p> 
 <p><a name="an_13:3">13:3</a> nețța yebda isselmad-iten s lemtul iqqaṛ-asen : Yiwen ufellaḥ yeffeɣ ad izreɛ. </p> 
 <p><a name="an_13:4">13:4</a> Mi gzerreɛ, ɣlin-as kra n iɛeqqayen rrif n webrid, usan-d igṭaṭ ( ifṛax igenni ) ččan-ten.  </p> 
 <p><a name="an_13:5">13:5</a> Kra iɛeqqayen nniḍen ɣlin deg wakal yeččuṛen d izṛa, mɣin-d s lemɣawla axaṭer ulac aṭas n wakal.  </p> 
 <p><a name="an_13:6">13:6</a> Mi gebda wezɣal, yezlef imɣan nni dɣa qquṛen, axaṭer ur sɛin ara izuṛan lqayen.  </p> 
 <p><a name="an_13:7">13:7</a> Kra iɛeqqayen nniḍen ɣlin ger isennanen. Isennanen-nni gman dɣa ɣummen-ten.  </p> 
 <p><a name="an_13:8">13:8</a> Meɛna kra iɛeqqayen ɣlin deg wakal yelhan : fkan-d tigedrin, ta tefka-d meyya iɛeqqayen, ta sețțin, tayeḍ tlatin.  </p> 
 <p><a name="an_13:9">13:9</a> Win yesɛan imeẓẓuɣen isel-ed !  </p> 
 <p><a name="an_13:10">13:10</a> Inelmaden-is qeṛṛben ɣuṛ-es, nnan-as : Acimi i sen-theddṛeḍ s lemtul ?  </p> 
 <p><a name="an_13:11">13:11</a> Yerra-yasen : Kunwi yețțunefk-awen-d aț țissinem lbaḍna n tgelda igenwan, ma d wiyaḍ, ur sen-d-ițțunefk ara.  </p> 
 <p><a name="an_13:12">13:12</a> Axaṭer win yesɛan a s-d-nernu alamma yella di tawant, ma d win ur nesɛi ara, a s-nekkes ula d ayen yesɛa.  </p> 
 <p><a name="an_13:13">13:13</a> Daymi i sen-țmeslayeɣ s lemtul, axaṭer țmuqulen ur țwalin, țḥessisen ur sellen, ur fehhmen.  </p> 
 <p><a name="an_13:14">13:14</a> S wakka, yedṛa-d yid-sen wayen i d-ixebbeṛ fell-asen nnbi Iceɛya : Aț-țsellem s imeẓẓuɣen-nwen, ur tfehhmem ara, aț-țmuqulem s wallen-nwen, ur tețwalim ara,  </p> 
 <p><a name="an_13:15">13:15</a> Imi agdud-agi qquṛen wulawen-nsen, qeflen imeẓẓuɣen- sen, qqnen allen-nsen, iwakken ur țwalin, ur sellen ; sseɣṛen ulawen nsen, ugin ad fehmen, axaṭer lemmer tuben, uɣalen-d ɣer webrid, tili sseḥlaɣ ten.  </p> 
 <p><a name="an_13:16">13:16</a> Ma d kunwi amarezg-nwen ! Axaṭer allen-nwen țwalint, imeẓẓuɣen nwen sellen.  </p> 
 <p><a name="an_13:17">13:17</a> Aql-i nniɣ-awen tideț, aṭas si lenbiya akk-d imdanen iḥeqqiyen i gebɣan ad walin ayen tețwalim, ur t-walan, ad slen ayen tsellem, ur t-slin.  </p> 
 <p><a name="an_13:18">13:18</a> A wen-d-iniɣ lmeɛna n lemtel n win izerrɛen :  </p> 
 <p><a name="an_13:19">13:19</a> m'ara isel wemdan i wawal yeɛnan tageldit n igenwan, ur t-yefhim ara, ițemcabi ɣer yiri n webrid i ɣer tɣelli zzerriɛa ; d Cciṭan i d-ițasen ad iqleɛ ayen izerɛen deg ul-is.  </p> 
 <p><a name="an_13:20">13:20</a> Wayeḍ ițemcabi ɣer wakal-nni yeččuṛen d izṛa anda teɣli zzerriɛa. M'ara isel i wawal n Ṛebbi, iqebbel-it s lfeṛḥ, </p> 
 <p><a name="an_13:21">13:21</a> meɛna ur t-yețțaǧǧa ara ad iger izuṛan deg-s, ur yețțaṭṭaf ara. MM'ara d-tass lmeḥna neɣ aqehheṛ ɣef ddemma n wawal n Ṛebbi, ad yeǧǧ imiren kan liman-is.  </p> 
 <p><a name="an_13:22">13:22</a> Wayeḍ daɣen ițemcabi ɣer zzerriɛa yeɣlin ger isennanen. M'ara isel i wawal n Ṛebbi, iɣeblan n ddunit d ṭṭmeɛ n rrbeḥ țɣummun awal-nni, dɣa ur d-ițțak ara lfakya.  </p> 
 <p><a name="an_13:23">13:23</a> Wayeḍ icuba daɣen ɣer zzerriɛa yeɣlin deg wakal yelhan, m'ara isel i wawal n Ṛebbi, ifehhem-it, daymi i d-ițțak lfakya ! YYal aɛeqqa ițțak-ed : wa tlatin, wa sețțin, wa meyya n tɛeqqayin.  </p> 
 <p><a name="an_13:24">13:24</a> Sidna Ɛisa yefka-yasen-d lemtel nniḍen : Tagelda n igenwan tcuba ɣer yiwen wergaz i gzerɛen irden deg iger-is. </p> 
 <p><a name="an_13:25">13:25</a> Yiwen yiḍ mi ṭṭsen akk medden, yusa-d weɛdaw-is izreɛ aẓekkun ger yirden, imiren iṛuḥ.  </p> 
 <p><a name="an_13:26">13:26</a> Mi d-gman yirden-nni fkan-d tigedrin, aẓekkun-nni ula d nețța igma-d. </p> 
 <p><a name="an_13:27">13:27</a> Iqeddacen n bab n iger-nni, usan-d ɣuṛ-es nnan-as : A Sidi, eɛni mačči d irden i tzerɛeḍ deg iger-ik ? Ansi i d-ikka ihi uẓekkun-agi ?  </p> 
 <p><a name="an_13:28">13:28</a> Yerra-yasen : D yiwen weɛdaw i gxedmen akka ! IIqeddacen-nni nnan-as : Ihi tebɣiḍ a nṛuḥ a neqleɛ aẓekkun-agi ?  </p> 
 <p><a name="an_13:29">13:29</a> Yerra-yasen : Xaṭi ! Axaṭer m'ara tekksem aẓekkun, tzemrem a d-teglum s yirden.  </p> 
 <p><a name="an_13:30">13:30</a> Anfet-asen ad gmun di sin alamma ț-țamegra. Ass-nni, a sen iniɣ i wid imeggren : qelɛet-ed uqbel aẓekkun, cuddet-eț ț-țadliwin iwakken a tent-nesseṛɣ ma d irden jemɛet țen-id ɣer ikuffan-iw.  </p> 
 <p><a name="an_13:31">13:31</a> Sidna Ɛisa yefka-yasen-d lemtel nniḍen : Tagelda n igenwan tcuba ɣer yiwen uɛeqqa n uxerḍel i d-iddem yiwen wergaz iwakken a t-izreɛ deg iger-is. </p> 
 <p><a name="an_13:32">13:32</a> Aɛeqqa-yagi d nețța i d amecṭuḥ meṛṛa ger zzerriɛat yellan, lameɛna mi gețwazreɛ, igemmu yețțali alamma yekka-d sennig akk yemɣan n tebḥirt, yeggar-ed ifurkawen d imeqqranen, ula d ifṛax n igenni , bennun leɛcuc-nsen s ufella-s.  </p> 
 <p><a name="an_13:33">13:33</a> Sidna Ɛisa yenna-yasen-d lemtel nniḍen : Tagelda n igenwan tețțemcabi ɣer temtunt ( yiɣes n temtunt ) ara d-teddem tmeṭṭut a t-tsexleḍ i tlata lkilat n uwren, s wakka arukti-nni meṛṛa ad yali.  </p> 
 <p><a name="an_13:34">13:34</a> Sidna Ɛisa ițmeslay-ed i lɣaci anagar s lemtul. </p> 
 <p><a name="an_13:35">13:35</a> Ițmeslay-asen-d akka iwakken ad yedṛu wayen i d-nnan lenbiya : A sen-mmeslayeɣ s lemtul, a sen-beccṛeɣ tiɣawsiwin yeffren seg wasmi i d-texleq ddunit. cc  </p> 
 <p><a name="an_13:36">13:36</a> Mi gekfa Sidna Ɛisa ameslay, lɣaci ṛuḥen, nețța ikcem ɣer wexxam. Inelmaden-is qeṛṛben ɣuṛ-es nnan-as : Ssefru-yaɣ-d lemtel n uẓekkun deg iger.  </p> 
 <p><a name="an_13:37">13:37</a> Yerra-yasen-d : Win izerrɛen irden d Mmi-s n bunadem ; </p> 
 <p><a name="an_13:38">13:38</a> iger, d ddunit ; irden, d wid ițekkan di tgelda n igenwan ; ma d aẓekkun, d widak ițekkan d Cciṭan ; </p> 
 <p><a name="an_13:39">13:39</a> aɛdaw i gzerrɛen aẓekkun, d Cciṭan ; tamegra , ț-țaggara nddunit ; ma d wid imeggren, d lmalayekkat.  </p> 
 <p><a name="an_13:40">13:40</a> Akken ara qellɛen aẓekkun ṭeggiṛen-t ɣer tmes, ara tedṛu di taggara n ddunit.  </p> 
 <p><a name="an_13:41">13:41</a> Mmi-s n bunadem a d-iceggeɛ lmalayekkat-is, ad ssufɣen si tgelda-s wid akk yesseɣlayen wiyaḍ di ddnub akk-d wid ixeddmen cceṛ, </p> 
 <p><a name="an_13:42">13:42</a> dɣa a ten-ḍeggṛen ɣer tmes anda ara yilin imeṭṭawen d nndama tameqqrant ; </p> 
 <p><a name="an_13:43">13:43</a> ma d wid iḍuɛen Ṛebbi ad feǧǧeǧen am yiṭij di tgelda n Baba Ṛebbi. Win yesɛan imeẓẓuɣen isel-ed !  </p> 
 <p><a name="an_13:44">13:44</a> Tagelda n igenwan tețțemcabi ɣer ugerruj yeffren deg yiwen n yiger. Yiwen wergaz yufa-t, yuɣal iffer-it. Si lfeṛḥ-nni i gefṛeḥ, iṛuḥ izzenz ayen akk yesɛa iwakken ad yaɣ iger-nni.  </p> 
 <p><a name="an_13:45">13:45</a> Tagelda n igenwan tețțemcabi daɣen ɣer yiwen umetjaṛ ițqelliben tiɛeqcin ifazen. </p> 
 <p><a name="an_13:46">13:46</a> Mi gufa yiwet ifazen ɣef tiyaḍ, iṛuḥ izzenz ayen akk yesɛa, yuɣal-ed yuɣ-iț.  </p> 
 <p><a name="an_13:47">13:47</a> Tagelda n igenwan tețțemcabi daɣen ɣer ucebbak ițwadeggṛen ɣer lebḥeṛ, iwakken a d-iṭṭef iselman ( lḥut ) n mkul ṣṣenf.  </p> 
 <p><a name="an_13:48">13:48</a> M'ara yeččaṛ, iḥewwaten a t-id-jebden ɣer rrif n lebḥeṛ, dɣa ad qqimen iwakken ad fernen iselman, ad jemɛen wid yelhan, ad ḍeggṛen wid n diri.  </p> 
 <p><a name="an_13:49">13:49</a> Akka ara tedṛu di taggara n ddunit : lmalayekkat a d-asent a d ddment imcumen si ger iḥeqqiyen, </p> 
 <p><a name="an_13:50">13:50</a> iwakken a ten-ḍeggṛent ɣer tmes anda llan imeṭṭawen d nndama tameqqrant.  </p> 
 <p><a name="an_13:51">13:51</a> Dɣa Sidna Ɛisa yenna-yasen : Tfehmem ayagi meṛṛa ? RRran-as : Anɛam a Sidi, nefhem !  </p> 
 <p><a name="an_13:52">13:52</a> Yerna-yasen : Ihi, yal amusnaw n ccariɛa ara ifehmen tagelda n igenwan, icuba ɣer bab n wexxam i d-ițekksen seg ugerruj-is tiɣawsiwin tijdidin, akk-d ț-țɣawsiwin tiqdimin.  </p> 
 <p><a name="an_13:53">13:53</a> Mi d-yenna lemtul-agi, Sidna Ɛisa iṛuḥ syenna. </p> 
 <p><a name="an_13:54">13:54</a> Yerra ɣer taddart i deg ițțuṛebba. Isselmad di lǧameɛ nsen, lɣaci akk iḥedṛen wehmen, qqaṛen : Ansi i s-d-tekka tmusni-yagi akk d tezmert s wayes ixeddem lbeṛhanat agi ? </p> 
 <p><a name="an_13:55">13:55</a> Mačči d nețța i d mmi-s n win ineǧṛen ijegwa ? Eɛni yemma-s mačči ț-țin iwumi qqaṛen Meryem ? AAtmaten-is mačči d Yeɛqub, Yusef, Semɛun akk-d Yuda ?  </p> 
 <p><a name="an_13:56">13:56</a> Yessetma-s daɣen, mačči gar aneɣ i llant ? Ansi i s-d-ikka ihi wannect-agi ?  </p> 
 <p><a name="an_13:57">13:57</a> Ɣef wayagi akk i sen-yella d sebba n tuccḍa, ugin ad amnen yis. DDɣa Sidna Ɛisa yenna-yasen : Nnbi yețwaḥqeṛ anagar di tmurt-is akk-d wexxam-is.  </p> 
 <p><a name="an_13:58">13:58</a> Sidna Ɛisa ur yexdim ara aṭas n lbeṛhanat dinna, axaṭer ugin ad amnen.  </p> 
 <p><a name="an_14:2">14:2</a> yenna i iqeddacen-is : D Yeḥya aɣeṭṭas ! D nețța i-d iḥyan si ger lmegtin, daymi i gesɛa tazmert s wayes ixeddem lbeṛhanat.  </p> 
 <p><a name="an_14:3">14:3</a> D agellid Hiṛudus i gefkan yakan lameṛ ad qnen Yeḥya s ssnasel, a t-rren ɣer lḥebs. Ayagi ɣef sebba n Hiṛudyad, tameṭṭut n Filibus yellan d gma-s n Hiṛudus. </p> 
 <p><a name="an_14:4">14:4</a> Axaṭer Yeḥya yella yeqqaṛ i Hiṛudus : D leḥṛam fell-ak aț-țesɛuḍ Hiṛudyad ț-țameṭṭut-ik !  </p> 
 <p><a name="an_14:5">14:5</a> Seg imiren, Hiṛudus ițqellib ad ineɣ Yeḥya, meɛna ikukra agdud n wat Isṛail axaṭer ḥesben-t akk d nnbi.  </p> 
 <p><a name="an_14:6">14:6</a> Ass n lɛid n umulli n Hiṛudus, yelli-s n Hiṛudyad tecḍeḥ zdat inebgawen, teɛǧeb-as aṭas i Hiṛudus, </p> 
 <p><a name="an_14:7">14:7</a> armi i s-iɛuhed, a s-yefk ayen ara s-d tessuter. </p> 
 <p><a name="an_14:8">14:8</a> Tweṣṣa-ț yemma-s a s-tini : Awi-yi-d ɣer dagi aqeṛṛuy n Yeḥya aɣeṭṭas deg uḍebsi.  </p> 
 <p><a name="an_14:9">14:9</a> Agellid Hiṛudus iḥzen aṭas, yemmuɣben, lameɛna ɣef ddemma n limin i geggul zdat inebgawen-is, issendeh a s-t-id-awin. </p> 
 <p><a name="an_14:10">14:10</a> Iceggeɛ a s-d-gezmen aqeṛṛuy i Yeḥya aɣeṭṭas di lḥebs ; </p> 
 <p><a name="an_14:11">14:11</a> wwin-as-t-id i teqcict-nni deg uḍebsi, nețțat tefka-t i yemma-s.  </p> 
 <p><a name="an_14:12">14:12</a> Inelmaden n Yeḥya usan-d ad awin lǧețța-s iwakken a ț-meḍlen ; imiren ṛuḥen ssawḍen-as lexbaṛ i Sidna Ɛisa.  </p> 
 <p><a name="an_14:13">14:13</a> Mi gesla s lexbaṛ n lmut n Yeḥya, Sidna Ɛisa yuli ɣer teflukt, iṛuḥ iwakken ad iḍeṛṛef iman-is. Akken slan lɣaci s wemkan i ɣer iteddu, ffɣen-d si tudrin, tebɛen-t ɣef wuḍar. </p> 
 <p><a name="an_14:14">14:14</a> Sidna Ɛisa yers-ed si teflukt, iwala annect-nni n lɣaci, dɣa gezmen tasa-s ; ibda yesseḥlay imuḍan-nsen. </p> 
 <p><a name="an_14:15">14:15</a> Mi geɣli yiṭij, inelmaden n Sidna Ɛisa qeṛṛben ɣuṛ-es, nnan-as : Amkan-agi yexla yerna iṛuḥ lḥal ; ini-yasen i lɣaci-yagi ad ṛuḥen ɣer tudrin iwakken a d-aɣen ayen ara ččen.  </p> 
 <p><a name="an_14:16">14:16</a> Sidna Ɛisa yerra-yasen : Acuɣeṛ ara ṛuḥen, fket-asen kunwi ad ččen ! </p> 
 <p><a name="an_14:17">14:17</a> Nnan-as : Ɣuṛ-nneɣ anagar xemsa teḥbulin n weɣṛum, d sin iselman ( iḥutiwen ).  </p> 
 <p><a name="an_14:18">14:18</a> Yerra-yasen : Awit-țen-id ɣer dagi !  </p> 
 <p><a name="an_14:19">14:19</a> Yenna i lɣaci ad qqimen ɣef leḥcic, iddem-ed xemsa teḥbulin-nni n weɣṛum d sin iselman-nni, yerfed allen-is ɣer igenni, iḥmed Ṛebbi. YYebḍa aɣṛum-nni, yefka-t i inelmaden-is, feṛqen-t i lɣaci.  </p> 
 <p><a name="an_14:20">14:20</a> Sčan akk armi ṛwan, yerna inelmaden-is ččuṛen-d tnac iqecwalen n wayen i d-yegran. </p> 
 <p><a name="an_14:21">14:21</a> Ad ilin azal n xemsa alaf n yergazen i geččan, mbla tilawin d warrac.  </p> 
 <p><a name="an_14:22">14:22</a> Imiren kan, Sidna Ɛisa iḥṛes inelmaden-is ad alin ɣer teflukt iwakken ad zwiren ɣer rrif nniḍen n lebḥeṛ, ma d nețța ad yerr lɣaci-nni.  </p> 
 <p><a name="an_14:23">14:23</a> Mi ṛuḥen akk lɣaci-nni, yuli weḥd-es ɣer wedrar iwakken ad iẓẓall, dɣa yeɣli-d yiḍ. </p> 
 <p><a name="an_14:24">14:24</a> Taflukt-nni tella di tlemmast n lebḥeṛ, lemwaji kkatent-eț axaṭer aḍu iqubel-iten-id. </p> 
 <p><a name="an_14:25">14:25</a> Mi qṛib ad yali wass, ataya Sidna Ɛisa iteddu-d ɣef wuḍar s ufella n waman. </p> 
 <p><a name="an_14:26">14:26</a> Akken i t-walan ileḥḥu ɣef waman, inelmaden-is xelɛen qqaṛen : D lexyal ! BBdan țɛeggiḍen s lxuf. </p> 
 <p><a name="an_14:27">14:27</a> Meɛna imiren kan Sidna Ɛisa yenna-yasen-d : Ur xellɛet ara, d nekk ! Ur țțaggadet ara !  </p> 
 <p><a name="an_14:28">14:28</a> Dɣa Buṭrus yenna-yas : A Sidi, ma yella d keččini, efk-ed lameṛ a n-aseɣ ɣuṛ-ek ɣef wuḍar s ufella n waman.  </p> 
 <p><a name="an_14:29">14:29</a> Sidna Ɛisa yenna-yas : As-ed ! BBuṭrus yers-ed si teflukt, ileḥḥu ɣef waman, iteddu ɣer Sidna Ɛisa. </p> 
 <p><a name="an_14:30">14:30</a> Meɛna mi gwala yekker-ed waḍu, ikcem-it lxuf, yebda izedder, dɣa iɛeggeḍ : A Sidi, sellek-iyi !  </p> 
 <p><a name="an_14:31">14:31</a> Imiren kan, Sidna Ɛisa yeḍleq afus-is, ijbed-it-id yenna-yas : A win iwumi ixuṣṣ liman ! Acuɣeṛ i k-ikcem ccekk ?  </p> 
 <p><a name="an_14:32">14:32</a> Ulin ɣer teflukt di sin, dɣa yers waḍu-nni. </p> 
 <p><a name="an_14:33">14:33</a> Inelmaden-nni yellan di teflukt seǧǧden zdat Sidna Ɛisa nnan-as : Ț-țideț, d kečč i d Mmi-s n Ṛebbi !  </p> 
 <p><a name="an_14:34">14:34</a> Zegren lebḥeṛ-nni, armi wwḍen ɣer tmurt n Jiniṣaret. </p> 
 <p><a name="an_14:35">14:35</a> Imezdaɣ n lǧiha nni ɛeqlen Sidna Ɛisa, dɣa ceggɛen ɣer yal lǧiha n tmurt, wwin-as-d imuḍan nsen meṛṛa.  </p> 
 <p><a name="an_14:36">14:36</a> Ḥellelen-t iwakken a ten-yeǧǧ ad nalen ɣas ula ț-țacḍaṭ uqenduṛ-is. Wid akk i t-innulen, ḥlan.  </p> 
 <p><a name="an_15:2">15:2</a> Acuɣeṛ inelmaden-ik țṛuẓun lɛadda i d-ǧǧan lejdud-nneɣ ? M'ara qeṛṛben ɣer lqut, ur ssiriden ara ifassen-nsen.  </p> 
 <p><a name="an_15:3">15:3</a> Sidna Ɛisa yerra-yasen : I kunwi, acimi tețṛuẓum lumuṛat n Ṛebbi iwakken aț-țxedmem ṛṛay nwen ? </p> 
 <p><a name="an_15:4">15:4</a> Yak d ṣṣeḥ, Ṛebbi yenna-d : Qadeṛ baba-k d yemma-k, yyenna daɣen : Kra win ara ineɛlen baba-s d yemma-s, ad yețwaḥkem fell-as s lmut.  </p> 
 <p><a name="an_15:5">15:5</a> Meɛna kunwi teqqaṛem : win yefkan d lweɛda i Ṛebbi ayen s wacu i glaq ad iɛiwen imawlan-is, </p> 
 <p><a name="an_15:6">15:6</a> am akken d imawlan-is i gɛawen. SS wakka, tbeddlem awal n Ṛebbi, iwakken aț-țesɛeddim lɛaddat-nwen.  </p> 
 <p><a name="an_15:7">15:7</a> Ay at sin wudmawen ! Ț-țideț i d-yenna nnbi Iceɛya mi d-icar fell awen :  </p> 
 <p><a name="an_15:8">15:8</a> Y enna-d Ṛebbi : agdud-agi, ițɛuzzu-yi s wawal kan, ma d ul-is yebɛed fell-i.  </p> 
 <p><a name="an_15:9">15:9</a> L eɛbada-nsen ur tesɛi ara azal axaṭer sselmaden ayen i d-yekkan seg imdanen.  </p> 
 <p><a name="an_15:10">15:10</a> Dɣa Sidna Ɛisa yessawel i lɣaci yenna-yasen : Semḥesset-d tfehmem ayagi : </p> 
 <p><a name="an_15:11">15:11</a> mačči d ayen i gkeččmen seg imi n wemdan i t-yessamasen, meɛna d ayen i d-ițeffɣen seg-s.  </p> 
 <p><a name="an_15:12">15:12</a> Inelmaden qeṛṛben-d ɣer Sidna Ɛisa nnan-as : Ifariziyen iɣaḍ-iten lḥal ɣef yimeslayen-nni i d-tenniḍ !  </p> 
 <p><a name="an_15:13">15:13</a> Yerra-yasen : Yal imɣi ur yeẓẓi ara Baba yellan deg igenwan, ad yețwaqleɛ.  </p> 
 <p><a name="an_15:14">15:14</a> Anfet-asen, d iderɣalen iteṭṭfen afus i iderɣalen nniḍen ; ma yella aderɣal yeṭṭef afus i wayeḍ, ad ɣlin i sin ɣer tesraft.  </p> 
 <p><a name="an_15:15">15:15</a> Buṭrus yenṭeq-ed yenna-yas :  Ssefhem-aɣ-d lemtel-agi.  </p> 
 <p><a name="an_15:16">15:16</a> Sidna Ɛisa yenna i inelmaden-is : Eɛni ula d kunwi txuṣṣ-ikkun lefhama ? </p> 
 <p><a name="an_15:17">15:17</a> Ur tefhimem ara belli ayen ikeččmen seg imi n wemdan yețțara ɣer uɛebbuḍ-is, mbeɛd ad ițwadeggeṛ ɣer beṛṛa. </p> 
 <p><a name="an_15:18">15:18</a> Lameɛna ayen i d-iteffɣen seg imi itekk-ed seg wul, d annect-agi i gessenǧasen amdan.  </p> 
 <p><a name="an_15:19">15:19</a> Axaṭer seg wul i d-tekken : yir ixemmimen, timegṛaḍ, zzna ( lɛaṛ ), yir tikli, tukerḍa, cchada n ẓẓuṛ akk-d rregmat.  </p> 
 <p><a name="an_15:20">15:20</a> Atan wayen yessenǧasen amdan, mačči d učči mbla tarda n ifassen i t-yessenǧasen.  </p> 
 <p><a name="an_15:21">15:21</a> Sidna Ɛisa iṛuḥ syenna, iteddu ɣer tmura n Sur akk-d Sidun.  </p> 
 <p><a name="an_15:22">15:22</a> Yiwet n tmeṭṭut n at Kenɛan i gzedɣen dinna tusa-d ɣuṛ-es tețɛeggiḍ : A Sidi, a mmi-s n Sidna Dawed, ḥunn fell-i ! Ațan yelli izdeɣ-iț uṛuḥani, iɛețțeb-iț aṭas.  </p> 
 <p><a name="an_15:23">15:23</a> Sidna Ɛisa ur as-d-yerri ula d yiwen wawal. Inelmaden-is qeṛṛben ɣuṛ-es nnan-as : Err-as awal, axaṭer tugi aț-țeḥbes leɛyaḍ deffir-nneɣ.  </p> 
 <p><a name="an_15:24">15:24</a> Sidna Ɛisa yerra-yasen : Țțuceggɛeɣ-ed anagar ɣer wulli iḍaɛen n wat Isṛail !  </p> 
 <p><a name="an_15:25">15:25</a> Lameɛna tameṭṭut-nni teɣli ɣer iḍaṛṛen n Sidna Ɛisa tenna-yas : A Sidi ɛiwen-iyi di leɛnaya-k !  </p> 
 <p><a name="an_15:26">15:26</a> Yenna-yas : Ur ilaq ara ad ițwakkes weɣṛum i warrac, ad ițwadeggeṛ i yeqjan.  </p> 
 <p><a name="an_15:27">15:27</a> Tenna-yas : Ț-țideț a Sidi, lameɛna ula d iqjan tețțen tifețțitin i d-iɣellin si ṭṭabla n imawlan-nsen.  </p> 
 <p><a name="an_15:28">15:28</a> Imiren Sidna Ɛisa yerra-yas : Liman-inem d ameqqran a tameṭṭut ! A m-yețwaxdem wayen tebɣiḍ ! IImiren kan yelli-s teḥla.  </p> 
 <p><a name="an_15:29">15:29</a> Sidna Ɛisa iṛuḥ syenna, yerra ɣer leǧwahi n lebḥeṛ n Jlili, yuli ɣer wedrar yeqqim. </p> 
 <p><a name="an_15:30">15:30</a> Usan-d ɣuṛ-es lɣaci d izumal, wwin-as-d iquḍaren, iderɣalen, ineɛyuba, igugamen d waṭas imuḍan nniḍen. Ssersen-ten ɣer zdat iḍaṛṛen-is dɣa isseḥla-ten.  </p> 
 <p><a name="an_15:31">15:31</a> Imdanen meṛṛa tɛeǧben mi walan igugamen heddṛen, ineɛyuba ḥlan, iquḍaren leḥḥun, iderɣalen țwalin, dɣa bdan țḥemmiden Sidi Ṛebbi Illu n wat Isṛail.  </p> 
 <p><a name="an_15:32">15:32</a> Sidna Ɛisa yessawel-asen i inelmaden-is yenna-yasen : Ɣaḍen-iyi lɣaci-agi axaṭer tlata wussan aya segmi llan yid-i, yerna ur sɛin d acu ara ččen. Ur bɣiɣ ara a ten-rreɣ akka lluẓen neɣ m'ulac ad feclen deg webrid.  </p> 
 <p><a name="an_15:33">15:33</a> Inelmaden nnan-as : Ansi ara ɣ-d-yekk deg umkan-agi yexlan, weɣṛum ara isseṛwun annect agi n lɣaci ?  </p> 
 <p><a name="an_15:34">15:34</a> Sidna Ɛisa yenna-yasen : Acḥal n teḥbulin n weɣṛum i tesɛam ɣuṛ-wen ? RRran-as : Sebɛa n teḥbulin akk-d kra n iselman ( iḥutiwen ).  </p> 
 <p><a name="an_15:35">15:35</a> Imiren yenna i lɣaci ad qqimen ɣer lqaɛa. </p> 
 <p><a name="an_15:36">15:36</a> Yeddem-ed tiḥbulin-nni n weɣṛum d iselman-nni, iḥmed Ṛebbi, yebḍa-ten, yefka-ten i inelmaden-is iwakken a ten-feṛqen i lɣaci.  </p> 
 <p><a name="an_15:37">15:37</a> Sčan akk armi ṛwan, yerna ččuṛen sebɛa n tqecwalin n wayen i d-yegran. </p> 
 <p><a name="an_15:38">15:38</a> Ger wid yeččan, llan azal n ṛebɛa alaf n yergazen mbla tilawin d warrac.  </p> 
 <p><a name="an_15:39">15:39</a> Mi gerra lɣaci, Sidna Ɛisa yerkeb di teflukt, iṛuḥ ɣer tmurt n Magadan.  </p> 
 <p><a name="an_16:1">16:1</a> Ifariziyen d isaduqiyen qeṛṛben ɣer Sidna Ɛisa, ssutren-as lbeṛhan iwakken a t-jeṛben. </p> 
 <p><a name="an_16:2">16:2</a> Sidna Ɛisa yerra-yasen : M'ara yeɣli yiṭij, twalam igenni d azeggaɣ, teqqaṛem ad yelhu lḥal. </p> 
 <p><a name="an_16:3">16:3</a> Akken daɣen taṣebḥit, teqqaṛem a d-iwwet ugeffur ( lehwa ) axaṭer yella usigna. AAmek ! Tesnem aț-țfehmem lḥala n igenni, ur tezmirem ara aț-țfehmem licaṛat yeɛnan lweqt-agi !  </p> 
 <p><a name="an_16:4">16:4</a> Lǧil am agi ijehlen ixedɛen Ṛebbi, yessutur lbeṛhan ! Meɛna ur as-d-ițțunefkay ara lbeṛhan nniḍen anagar win n nnbi Yunes ! DDɣa iṛuḥ, yeǧǧa-ten. </p> 
 <p><a name="an_16:5">16:5</a> Mi zegren agummaḍ i lebḥeṛ, inelmaden-is țțun ur wwin ara yid-sen aɣṛum. </p> 
 <p><a name="an_16:6">16:6</a> Dɣa Sidna Ɛisa yenna-yasen : Ḥadret iman-nwen seg iɣes n temtunt n ifariziyen d isaduqiyen !  </p> 
 <p><a name="an_16:7">16:7</a> Inelmaden xemmemen deg yiman-nsen, nnan : Ahat imi ur d-newwi ara yid-nneɣ aɣṛum i ɣ-d-yenna akka !  </p> 
 <p><a name="an_16:8">16:8</a> Sidna Ɛisa iẓra ɣef wacu i țxemmimen yenna-yasen : A wid i gxuṣṣen di liman ! Tɣilem imi ur d-tewwim ara aɣṛum i wen-d-nniɣ akka ?  </p> 
 <p><a name="an_16:9">16:9</a> Mazal ur tefhimem ara ? Eɛni tețțum yakan xemsa teḥbulin-nni n weɣṛum i bḍiɣ i xemsa alaf n yergazen ? D wacḥal n tqecwalin i d-yegran ?  </p> 
 <p><a name="an_16:10">16:10</a> Ur tecfim ara daɣen ɣef sebɛa teḥbulin-nni n weɣṛum i bḍiɣ i ṛebɛa alaf n yemdanen, d wacḥal n tqecwalin i d-yegran ? </p> 
 <p><a name="an_16:11">16:11</a> Amek akka ur tefhimem ara belli mačči ɣef weɣṛum i wen-d-țmeslayeɣ ? ḤḤadret iman-nwen seg iɣes n temtunt n ifariziyen d isaduqiyen !  </p> 
 <p><a name="an_16:12">16:12</a> Imiren inelmaden-is fehmen belli mačči ɣef yiɣes n temtunt i sen-d-immeslay, lameɛna ad ḥadren iman-nsen ɣef wuselmed n ifariziyen d isaduqiyen.  </p> 
 <p><a name="an_16:13">16:13</a> Mi gewweḍ Sidna Ɛisa ɣer tmurt n Qiṣarya n Filibus, isteqsa inelmaden is, yenna-yasen : D acu i-d-qqaṛen medden ɣef Mmi-s n bunadem ?  </p> 
 <p><a name="an_16:14">16:14</a> Rran-as : Kra qqaṛen d Yeḥya aɣeṭṭas, wiyaḍ qqaṛen d nnbi Ilyas, wiyaḍ daɣen d nnbi Irmiya neɣ d yiwen ger lenbiya.  </p> 
 <p><a name="an_16:15">16:15</a> I kunwi, d acu-yi ɣuṛ-wen ?  </p> 
 <p><a name="an_16:16">16:16</a> Buṭrus yerra-yas : Kečč d Lmasiḥ, Mmi-s n Sidi Ṛebbi yeddren !  </p> 
 <p><a name="an_16:17">16:17</a> Yenṭeq ɣuṛ-es yenna-yas : Amarezg-ik, a Buṭrus a mmi-s n Yunes, axaṭer mačči s tmusni-inek i tfehmeḍ ayagi, meɛna d Baba Ṛebbi yellan deg igenwan i k-t-id-isbegnen.  </p> 
 <p><a name="an_16:18">16:18</a> Ihi nekk a k-iniɣ : kečč a Buṭrus d azṛu, yerna ɣef wezṛu-agi ara bnuɣ tajmaɛt-iw, ula ț-ținezmarin n lmut ur țțaṭṭafent ara zdat-es !  </p> 
 <p><a name="an_16:19">16:19</a> Ad a k-fkeɣ tisura n tgelda n igenwan, ayen ara teqneḍ neɣ ayen iwumi ara tserrḥeḍ di ddunit, ad yețwaqbel deg igenwan.  </p> 
 <p><a name="an_16:20">16:20</a> Imiren yumeṛ i yinelmaden-is ur qqaṛen ula i yiwen belli d nețța i d Lmasiḥ.  </p> 
 <p><a name="an_16:21">16:21</a> Seg imiren, Sidna Ɛisa yebda issefham i inelmaden-is belli ilaq ad yali ɣer temdint n Lquds, anda ara yenneɛtab aṭas ger ifassen n lmuqedmin akk-d lɛulama n ccariɛa ad ḥekmen fell-as s lmut, ass wis tlata a d-iḥyu si ger lmegtin.  </p> 
 <p><a name="an_16:22">16:22</a> Buṭrus ijbed-it weḥd-es, yenna yas : A k-imneɛ Ṛebbi a Sidi, a wer d-yedṛu wayagi yid-ek !  </p> 
 <p><a name="an_16:23">16:23</a> Sidna Ɛisa idewweṛ ɣuṛ-es, yenna-yas : Beɛɛed akkin fell-i a Cciṭan ! Tebɣiḍ aț-țiliḍ d ugur deg webrid-iw ! Ixemmimen-ik ur d-kkan ara s ɣuṛ Ṛebbi, lameɛna d ixemmimen n yemdanen.  </p> 
 <p><a name="an_16:24">16:24</a> Imiren Sidna Ɛisa yenna i inelmaden-is : Kra win yebɣan ad iddu yid-i, ilaq-as ur yețḥebbiṛ ara ɣef yiman-is, ad iqbel ad inɛețțab ɣef ddemma n yisem-iw, a yi-d-itbeɛ. </p> 
 <p><a name="an_16:25">16:25</a> Kra win yebɣan ad isellek taṛwiḥt-is, a s-tṛuḥ ; ma d win iseblen taṛwiḥt-is ɣef ddemma-w ad yuɣal a ț-yaf.  </p> 
 <p><a name="an_16:26">16:26</a> D acu n lfayda ara yesɛu wemdan di rrbeḥ n ddunit meṛṛa, ma yella yesṛuḥ taṛwiḥt-is ? Neɣ d acu i gezmer a t-yefk wemdan iwakken a d-ifdu taṛwiḥt-is ?  </p> 
 <p><a name="an_16:27">16:27</a> Axaṭer Mmi-s n bunadem a d-yas nețța d lmalayekkat-is di lɛaḍima n Baba-s, ad yefk i yal yiwen ayen yuklal, yal yiwen ɣef leḥsab n lefɛayel-is.  </p> 
 <p><a name="an_16:28">16:28</a> A wen-iniɣ tideț : llan kra seg wid yellan dagi, ur țmețțaten ara alamma ẓran Mmi-s n bunadem yusa-d am ugellid.  </p> 
 <p><a name="an_17:1">17:1</a> Mi ɛeddan sețța wussan, Sidna Ɛisa yewwi yid-es Buṭrus, Yeɛqub akk-d gma-s Yuḥenna, ulin ɣer wedrar eɛlayen, anda qqimen ḍeṛfen iman-nsen.  </p> 
 <p><a name="an_17:2">17:2</a> Taswiɛt kan ibeddel ṣṣifa zdat sen, walan udem-is yețfeǧǧiǧ am yiṭij, llebsa-s tuɣal ț-țamellalt, tecceɛceɛ am tafat. </p> 
 <p><a name="an_17:3">17:3</a> A ten-aya dehṛen-asen-d Sidna Musa akk-d Sidna Ilyas țmeslayen d Sidna Ɛisa.  </p> 
 <p><a name="an_17:4">17:4</a> Buṭrus inṭeq yenna i Sidna Ɛisa : A Sidi, yelha ma neqqim dagi, ma tebɣiḍ ad sbeddeɣ tlata iqiḍunen : yiwen i kečč, yiwen i Sidna Musa, wayeḍ i Sidna Ilyas.  </p> 
 <p><a name="an_17:5">17:5</a> Mazal-it ițmeslay mi d-ters yiwet n tagut ițfeǧǧiǧen tɣumm-iten. Si tagut-nni tenṭeq-ed yiwet n taɣect, tenna-d : Wagi d Mmi eɛzizen deg-s i gella lfeṛḥ-iw, smeḥsiset-as !  </p> 
 <p><a name="an_17:6">17:6</a> Mi slan i taɣect-nni, inelmaden ikcem-iten lxuf, ɣlin ɣef wudem. </p> 
 <p><a name="an_17:7">17:7</a> Sidna Ɛisa yessers afus-is fell-asen, yenna-yasen : Ur țțaggadet ara, kkret !  </p> 
 <p><a name="an_17:8">17:8</a> Mi refden allen-nsen, ur walan ula d yiwen nniḍen anagar Sidna Ɛisa. </p> 
 <p><a name="an_17:9">17:9</a> Akken i d-țadren seg wedrar, Sidna Ɛisa yumeṛ-iten yenna-yasen : Ur qqaṛet i yiwen wayen akka twalam alamma yeḥya-d Mmi-s n bunadem si ger lmegtin.  </p> 
 <p><a name="an_17:10">17:10</a> Inelmaden steqsan Sidna Ɛisa nnan-as : Iwacu imusnawen n ccariɛa qqaṛen-d : « Ilaq a d-yas nnbi Ilyas d amezwaru ? »  </p> 
 <p><a name="an_17:11">17:11</a> Yerra-yasen :  Ț-țideț ilaq a d-yas nnbi Ilyas d amezwaru, iwakken ad iseggem kullec. </p> 
 <p><a name="an_17:12">17:12</a> Ma d nekk aql-i nniɣ awen-d : nnbi Ilyas yusa-d yakan, lameɛna imdanen ur t-ɛqilen ara yerna xedmen deg-s akken i sen ihwa. Akka daɣen ara inɛețțab Mmi-s n bunadem ger ifassen-nsen !  </p> 
 <p><a name="an_17:13">17:13</a> Inelmaden fehmen imiren belli ițmeslay-asen-d ɣef Yeḥya aɣeṭṭas.  </p> 
 <p><a name="an_17:14">17:14</a> Mi wwḍen ɣer wanda llan lɣaci, yiwen wergaz iqeṛṛeb-ed ɣer Sidna Ɛisa, yeɣli ɣef tgecrar zdat-es yenna yas :  </p> 
 <p><a name="an_17:15">17:15</a> A Sidi, ḥunn ɣef mmi ! Atan kkaten-t imsigman, yenneɛtab aṭas ! Acḥal d abrid i geɣli ɣer tmes neɣ ɣer waman,  </p> 
 <p><a name="an_17:16">17:16</a> wwiɣ-t-id i inelmaden-ik, meɛna ur zmiren ara a t-sseḥlun.  </p> 
 <p><a name="an_17:17">17:17</a> Inṭeq Sidna Ɛisa yenna-yasen : A lǧil ijehlen, iɛewjen ! Ar melmi ara yiliɣ yid-wen ? Ar melmi ara wen-sebṛeɣ ? Awit-iyi-d aqcic-agi ɣer dagi !  </p> 
 <p><a name="an_17:18">17:18</a> Sidna Ɛisa iɛeggeḍ ɣef lǧen-nni, yumeṛ-as ad iffeɣ seg weqcic-nni. Imiren kan, aqcic-nni yeḥla. </p> 
 <p><a name="an_17:19">17:19</a> Mi llan inelmaden iman-nsen, qeṛṛben ɣer Sidna Ɛisa nnan-as : Acuɣeṛ nukkni ur nezmir ara a t-nessufeɣ ?  </p> 
 <p><a name="an_17:20">17:20</a> Sidna Ɛisa yerra-yasen : Axaṭer txuṣṣem di liman ! Aql-i nniɣ-awen, lemmer tesɛim liman annect n uɛeqqa n uxerḍel, tili a s tinim i wedrar-agi, qleɛ iman-ik sya ɣer dihin ad iqleɛ. Ulac wayen iwumi ur tețțizmirem ara. </p> 
 <p><a name="an_17:21">17:21</a> Sṣenf-agi n leǧnun țeffɣen anagar s tẓallit d uẓummu.  </p> 
 <p><a name="an_17:22">17:22</a> Mi llan di tmurt n Jlili, Sidna Ɛisa yenna-yasen : Mmi-s n bunadem ad ițțusellem ger ifassen n yemdanen ; </p> 
 <p><a name="an_17:23">17:23</a> a t-nɣen, meɛna ass wis tlata a d-iḥyu si ger lmegtin. DDɣa inelmaden ikcem-iten leḥzen d ameqqran.  </p> 
 <p><a name="an_17:24">17:24</a> Mi gewweḍ Sidna Ɛisa akk-d inelmaden-is ɣer Kafernaḥum, imekkasen n lǧameɛ qeṛṛben-d ɣer Buṭrus nnan-as : Eɛni Ssid-nwen ur ițxelliṣ ara tabzert n lǧameɛ ?  </p> 
 <p><a name="an_17:25">17:25</a> Yerra-yasen Buṭrus : Ih, ad ixelleṣ ! Imiren kan mi gekcem Buṭrus ɣer wexxam, Sidna Ɛisa iluɛa-t, yenna-yas : D acu twalaḍ a Buṭrus ? Anwa i glaqen ad ixelleṣ tabzert i igelliden n ddunit-a , d arraw-nsen neɣ d ibeṛṛaniyen ? </p> 
 <p><a name="an_17:26">17:26</a> Buṭrus yerra-yas : D ibeṛṛaniyen ! DDɣa Sidna Ɛisa yerra-yas-d : Ihi arraw n tmurt ur țxelliṣen ara !  </p> 
 <p><a name="an_17:27">17:27</a> Lameɛna iwakken ur ten-yețɣaḍ ara lḥal, ṛuḥ, ḍeggeṛ tasennaṛt ɣer lebḥeṛ, aslem ( aḥewtiw ) amezwaru ara d-teṭṭfeḍ, ldi-yas imi, aț-țafeḍ deg-s aṣurdi n lfeṭṭa, efk-asen-t d lexlaṣ n tebzert-nneɣ nekk yid-ek.  </p> 
 <p><a name="an_18:1">18:1</a> Di lweqt-nni, inelmaden n Sidna Ɛisa qeṛṛben ɣuṛ-es nnan-as : -- Anwa i d ameqqran di tgelda n igenwan ?  </p> 
 <p><a name="an_18:2">18:2</a> Sidna Ɛisa yessawel i yiwen weqcic isbedd-it di tlemmast-nsen, yenna :  </p> 
 <p><a name="an_18:3">18:3</a> A wen-d-iniɣ tideț ; ma yella ur d-tuɣalem ara ɣer webrid, m'ur teqbilem ara aț-țilim .am arrac imecṭuḥen, ur tkeččmem ara ɣer tgelda n igenwan ! </p> 
 <p><a name="an_18:4">18:4</a> Daymi, win ara yeṣṣemẓin iman-is am weqcic-agi amecṭuḥ, d nețța ara yilin d ameqqran di tgelda n igenwan.  </p> 
 <p><a name="an_18:5">18:5</a> Kra win ara yestṛeḥben s weqcic am-agi ɣef ddemma n yisem-iw, yis-i i gesṭerḥeb. </p> 
 <p><a name="an_18:6">18:6</a> Win ara yesseɣlin di ddnub ula d yiwen seg imecṭuḥen-agi yumnen yis-i, axiṛ-as a s-cudden aɣaṛef n tsirt ɣer temgeṛṭ-is, a t-ḍeggṛen ɣer lebḥeṛ ad iɣṛeq.  </p> 
 <p><a name="an_18:7">18:7</a> A tawaɣit n ddunit-a, imi qwant tsebbiwin n tuccḍa ! Axaṭer tisebbiwin n tuccḍa ulamek ur d-țilint ara. Meɛna a nnger n win ara yilin d sebba-nsent.  </p> 
 <p><a name="an_18:8">18:8</a> Ma yella d afus-ik neɣ d aḍaṛ-ik ara k-yesseɣlin di ddnub, gzem-iten, ḍeggeṛ-iten akkin fell-ak, axiṛ-ak aț-țiliḍ di ddunit d aɛiban wala aț-țesɛuḍ sin iḍaṛṛen neɣ sin ifassen, aț-țwaḍeggṛeḍ ɣer tmes ur nxețți.  </p> 
 <p><a name="an_18:9">18:9</a> Ma yella daɣen ț-țiṭ-ik ara k-yawin ɣer ddnub, qleɛ-iț ḍeggeṛ-iț akkin fell-ak ; axiṛ-ik aț-țiliḍ di ddunit s yiwet n tiṭ wala aț-țesɛuḍ snat wallen, aț- țwaḍeggṛeḍ ɣer tmes n ǧahennama.  </p> 
 <p><a name="an_18:10">18:10</a> Ɣuṛ-wat aț-țḥeqṛem yiwen seg imecṭuḥen-agi axaṭer aql-i nniɣ awen-t : lmalayekkat-nsen deg igenwan, zgant daymen zdat Baba Ṛebbi. </p> 
 <p><a name="an_18:11">18:11</a> [ Axaṭer Mmi-s n bunadem yusa-d ad isellek wid i gḍaɛen ! ]  </p> 
 <p><a name="an_18:12">18:12</a> D acu twalam ? Ma yella win yesɛan meyya wulli dɣa teɛṛeq-as yiwet deg-sent, ur ițțaǧa ara țesɛa uțesɛin-nni nniḍen di lexla iwakken a d-inadi ɣef tin i s-iɛeṛqen ?  </p> 
 <p><a name="an_18:13">18:13</a> A wen-iniɣ tideț : ma yufa-ț, ad ifṛeḥ yis akteṛ n țesɛa uțesɛin-nni nniḍen ur neɛṛiq ara. </p> 
 <p><a name="an_18:14">18:14</a> Akken daɣen Baba-twen Ṛebbi yellan deg igenwan, ur yebɣi ara a s-iṛuḥ ula d yiwen seg imecṭuḥen-agi.  </p> 
 <p><a name="an_18:15">18:15</a> Ma yella twalaḍ gma-k idneb, ṛuḥ ṭṭef-it weḥd-es, nhu-t. Ma yuɣ-ak awal, atan trebḥeḍ-ed gma-k ! </p> 
 <p><a name="an_18:16">18:16</a> Meɛna ma igumma a k-d-isel, awi yiwen neɣ sin yemdanen yid-ek, iwakken ad ilin d inigan m'ara tefru temsalt , akken i d-tenna ccariɛa.  </p> 
 <p><a name="an_18:17">18:17</a> Ma yugi a sen-isel, ssiweḍ tamsalt ɣer tejmaɛt n watmaten, m'ur iḥess ara ula i tejmaɛt-nni, ḥseb-it am akken d akafriw neɣ d amekkas axabit ! </p> 
 <p><a name="an_18:18">18:18</a> A wen-iniɣ tideț : ayen akk ara teqnem neɣ iwumi ara tserrḥem di ddunit, ad yețwaqbel deg igenwan.  </p> 
 <p><a name="an_18:19">18:19</a> A wen-iniɣ daɣen s tideț : ma yella sin seg-wen ddukklen iwakken ad ssutren kra di tẓallit, Baba Ṛebbi yellan deg igenwan, a sen-t-id-yefk. </p> 
 <p><a name="an_18:20">18:20</a> Axaṭer anda ara dduklen sin neɣ tlata s yisem-iw, ad ḥedṛeɣ gar-asen !  </p> 
 <p><a name="an_18:21">18:21</a> Imiren Buṭrus iqeṛṛeb ɣuṛ-es yenna-yas : A Sidi, acḥal n tikkal ilaq ad semḥeɣ i gma ma yeḍlem-iyi, armi d sebɛa n tikkal ? SSidna Ɛisa yerra yas :  </p> 
 <p><a name="an_18:22">18:22</a> Xaṭi ! Ur k-d-nniɣ ara alamma d sebɛa tikkal, meɛna alamma d sebɛin iberdan sebɛa tikkal !  </p> 
 <p><a name="an_18:23">18:23</a> Daymi, tagelda n igenwan tețțemcabi ɣer yiwen ugellid yebɣan ad imḥasab nețța d iqeddacen-is. </p> 
 <p><a name="an_18:24">18:24</a> Akken yebda leḥsab, wwin-as-d yiwen uqeddac iwumi yețțalas imelyunen.  </p> 
 <p><a name="an_18:25">18:25</a> Imi ur yesɛi ara s wacu ara ixelleṣ, agellid-nni ifka lameṛ a ten-zzenzen d aklan, s nețța s tmeṭṭut-is, s warraw-is d wayen akk yesɛa, iwakken ad ixelleṣ ṭṭlaba-ines. </p> 
 <p><a name="an_18:26">18:26</a> Aqeddac-nni yeɣli ɣer iḍaṛṛen n ugellid, iḥellel-it yenna-yas : «` A Sidi, ṣṣbeṛ kra wussan, ad uɣaleɣ a k-xellṣeɣ ayen akk i yi tețțalaseḍ. » </p> 
 <p><a name="an_18:27">18:27</a> Imi i t-iɣaḍ aṭas, agellid-nni isumeḥ-as ṭṭlaba-ines iserreḥ-as ad iṛuḥ.  </p> 
 <p><a name="an_18:28">18:28</a> Akken kan i d-iffeɣ uqeddac-nni immuger-ed yiwen seg imdukkal-is ixeddmen yid-es, iwumi yețțalas kra n iṣurdiyen. IIkker ɣuṛ-es yeṭṭef-it si temgeṛṭ yeqqaṛ-as : « Err-iyi-d ayen i k țțalaseɣ ! » </p> 
 <p><a name="an_18:29">18:29</a> Amdakkel-is yeɣli ɣer iḍaṛṛen-is iḥellel-it, yenna-yas : « Sṣbeṛ kra n wussan, ad uɣaleɣ a k-xellṣeɣ ! »  </p> 
 <p><a name="an_18:30">18:30</a> Lameɛna ur yeqbil ara ad isbeṛ, yerra-t ɣer lḥebs, alamma tețwaxelleṣ ṭṭlaba-ines. </p> 
 <p><a name="an_18:31">18:31</a> Iqeddacen nniḍen mi walan ayen yedṛan, iɣaḍ-iten lḥal, dɣa ṛuḥen ṣṣawḍen lexbaṛ i ugellid.  </p> 
 <p><a name="an_18:32">18:32</a> Agellid iceggeɛ ɣer uqeddac-nni akken a d-yas, dɣa yenna-yas : « A yir amdan ! Mi tețḥellileḍ deg-i, eɛni ur k-sumḥeɣ ara akk ṭṭlaba-inek ? </p> 
 <p><a name="an_18:33">18:33</a> Acuɣeṛ ihi ur as-tsumḥeḍ ara i wemdakkel-ik akken i k-sumḥeɣ nekkini ? »  </p> 
 <p><a name="an_18:34">18:34</a> Agellid-nni yezɛef aṭas, yerra aqeddac-nni ɣer lḥebs, iwakken ad inɛețțab alamma tețwaxelleṣ ṭṭlaba ines.  </p> 
 <p><a name="an_18:35">18:35</a> Akken daɣen ara wen-yexdem Baba Ṛebbi yellan deg igenwan ma ur tețsamaḥem ara i watmaten-nwen seg ul.  </p> 
 <p><a name="an_19:1">19:1</a> Mi gekfa lewṣayat-agi, Sidna Ɛisa iṛuḥ si tmurt n Jlili ɣer tmurt n Yahuda ɣer leǧwahi n wasif n Urdun. </p> 
 <p><a name="an_19:2">19:2</a> Aṭas n lɣaci i t-iḍefṛen ɣer dinna, dɣa yesseḥla wid akk ihelken. </p> 
 <p><a name="an_19:3">19:3</a> Kra ifariziyen qeṛṛben-d ɣuṛ-es iwakken a t-jeṛben, nnan-as : Eɛni ccariɛa-nneɣ tsumeḥ-as i wergaz ad yebru i tmeṭṭut-is ɣef yal sebba yellan ?  </p> 
 <p><a name="an_19:4">19:4</a> Yerra-yasen : Ur teɣṛim ara ayen i d-teqqaṛ ccariɛa : di tazwara Sidi Ṛebbi ixleq-ed argaz ț-țmeṭṭut,   </p> 
 <p><a name="an_19:5">19:5</a> yenna-d : AArgaz ad yeǧǧ baba-s d yemma-s, iiwakken ad iɛic nețța ț-țmeṭṭut-is, ddɣa di sin yid-sen, ad uɣalen d yiwen.  </p> 
 <p><a name="an_19:6">19:6</a> S wakka ur llin ara d sin, meɛna uɣalen d yiwen. Ur as-ilaq ara ihi i wemdan ad ifṛeq ayen yesdukkel Sidi Ṛebbi !  </p> 
 <p><a name="an_19:7">19:7</a> Nnan-as : Acuɣeṛ ihi Sidna Musa yeǧǧa yaɣ-d lqanun belli m'ara yebru wergaz i tmeṭṭut-is, a s-yefk lkaɣeḍ n berru ?  </p> 
 <p><a name="an_19:8">19:8</a> Sidna Ɛisa yerra-yasen : Ɣef ddemma n wulawen-nwen yeqquṛen i wen-iserreḥ nnbi Musa aț-țebrum i tilawin-nwen. Meɛna di tazwara mačči akka i gella lḥal. </p> 
 <p><a name="an_19:9">19:9</a> Ma d nekk a wen-d-iniɣ : kra n win ara yebrun i tmeṭṭut-is yili ur tezni ara, iwakken ad yaɣ tayeḍ, d nețța i geznan !  </p> 
 <p><a name="an_19:10">19:10</a> Inelmaden nnan-as : Ma yella akka i d lḥala n wergaz ɣer tmeṭṭut, ulac lfayda di zzwaǧ !  </p> 
 <p><a name="an_19:11">19:11</a> Sidna Ɛisa yerra-yasen : Mačči d imdanen meṛṛa i gzemren ad qeblen aselmed agi lameɛna anagar wid iwumi i d-ifka Sidi Ṛebbi lefhama i gzemren ad qeblen annect-agi. </p> 
 <p><a name="an_19:12">19:12</a> Aṭas n sebbat i gqeḍɛen zzwaǧ i wemdan. Llan wid i d-ilulen si tɛebbuṭ n yemma-tsen ur sɛin ara ddunit, wiyaḍ d imdanen i sen-ț-ikksen. LLlan daɣen wid iḥeṛmen iman-nsen ɣef zwaǧ, ɣef ddemma n tgelda n igenwan. Win i gzemren ad ifhem aselmed agi a t-yefhem !  </p> 
 <p><a name="an_19:13">19:13</a> Kra yemdanen țawin-d i Sidna Ɛisa arrac imecṭuḥen iwakken ad issers afus-is fell-asen a ten-ibarek, meɛna inelmaden-is țțaran-ten.  </p> 
 <p><a name="an_19:14">19:14</a> Sidna Ɛisa yenna-yasen : Anfet-asen i warrac a d-asen ɣuṛ-i, ur sen-qeṭṭɛet ara abrid, axaṭer tagelda n igenwan thegga i nutni akk-d wid yellan am nutni !  </p> 
 <p><a name="an_19:15">19:15</a> Yessers afus-is fell-asen, iburek iten, dɣa iṛuḥ syenna.  </p> 
 <p><a name="an_19:16">19:16</a> Imiren yiwen wergaz iqeṛṛeb-ed ɣer Sidna Ɛisa, yenna-yas : A Sidi, d acu i yi-ilaqen a t xedmeɣ n wayen yelhan iwakken ad sɛuɣ tudert n dayem ?  </p> 
 <p><a name="an_19:17">19:17</a> Sidna Ɛisa yerra-yas : Acuɣeṛ i yi-testeqsayeḍ ɣef wayen yelhan ? AAnagar yiwen i gelhan ! Ma tebɣiḍ aț-țesɛuḍ tudert-agi, xdem lumuṛat yellan di ccariɛa. YYerra-yas-ed weqcic-nni : Anwi i d lumuṛat ara tebɛeɣ ?  </p> 
 <p><a name="an_19:18">19:18</a> Sidna Ɛisa yerra-yas :  Ur tneqqeḍ ara tamgeṛṭ, ur tzennuḍ ara, ur tețțcehhideḍ ara s ẓẓur, ur tețțakkṛreḍ ara,   </p> 
 <p><a name="an_19:19">19:19</a> qadeṛ baba-k d yemma-k, tḥemmleḍ lɣiṛ-ik am yiman-ik !  </p> 
 <p><a name="an_19:20">19:20</a> Aqcic-nni yerra-yas-ed : Ayagi akk xedmeɣ-t, d acu i yi-ixuṣṣen ?  </p> 
 <p><a name="an_19:21">19:21</a> Sidna Ɛisa yenna-yas : Ma tebɣiḍ ihi aț-țkemleḍ deg webrid n Sidi Ṛebbi , ṛuḥ, zzenz akk ayen i tesɛiḍ, tefkeḍ idrimen-nni i yimeɣban, s wakka aț-țesɛuḍ agerruj deg igenwan ! Imiren as-ed, aț-țedduḍ yid-i.  </p> 
 <p><a name="an_19:22">19:22</a> Mi gesla i yimeslayen-agi, ilemẓi-nni yeḥzen aṭas dɣa iṛuḥ, axaṭer d ameṛkanti ameqqran i gella. </p> 
 <p><a name="an_19:23">19:23</a> Sidna Ɛisa yenna i inelmaden-is : A wen-iniɣ s tideț : iwɛeṛ aṭas i umeṛkanti ad ikcem ɣer tgelda n igenwan ! </p> 
 <p><a name="an_19:24">19:24</a> A wen-iniɣ daɣen : ma yella iwɛeṛ i welɣem ad iɛeddi di tiṭ n tissegnit, iwɛeṛ akteṛ i umeṛkanti ad ikcem ɣer tgelda n igenwan ! </p> 
 <p><a name="an_19:25">19:25</a> Mi slan i imeslayen-agi, inelmaden-is dehcen aṭas, nnan-as : Anwa i gzemren ad ițțusellek ihi ?  </p> 
 <p><a name="an_19:26">19:26</a> Dɣa Sidna Ɛisa imuqel-iten, yenna : Imdanen s yiman-nsen ur zmiren ara, meɛna Sidi Ṛebbi yezmer i kullec.  </p> 
 <p><a name="an_19:27">19:27</a> Buṭrus inṭeq, yenna-yas : Aql-aɣ nukkni neǧǧa kullec iwakken a k-netbeɛ, d acu ara yedṛun yid-nneɣ ?  </p> 
 <p><a name="an_19:28">19:28</a> Sidna Ɛisa yerra-yasen : A wen-iniɣ s tideț, asm'ara d-ḥyun lmegtin ɣer ddunit tajḍiṭ, asm'ara yeqqim Mmi-s n bunadem ɣef wukersi n lḥekma di lɛaḍima-s, ula d kunwi yeddan yid-i, aț-țeqqimem di tnac yid-wen ɣef yikersiyen n lḥekma iwakken aț-țḥasbem tnac n leɛṛac n wat Isṛail. </p> 
 <p><a name="an_19:29">19:29</a> Yerna, kra win ara yeǧǧen ɣef ddemma n yisem-iw axxam-is neɣ atmaten-is, yessetma-s neɣ baba-s, yemma-s, arraw-is neɣ tiferkiwin-is, ad as-d-yețțunefk akteṛ n wayen yeǧǧa, yerna a t-id-tṣaḥ tudert n dayem.  </p> 
 <p><a name="an_19:30">19:30</a> Aṭas seg wid yellan d imezwura ad uɣalen d ineggura, aṭas daɣen seg wid yellan d ineggura ad uɣalen d imezwura.  </p> 
 <p><a name="an_20:2">20:2</a> Imsefham yid-sen a ten-ixelleṣ s twizeț n lfeṭṭa i wass, dɣa iceggeɛ-iten ɣer tfeṛṛant-is. </p> 
 <p><a name="an_20:3">20:3</a> Iffeɣ daɣen ɣef țesɛa n ṣṣbeḥ, iwala wiyaḍ qqimen di beṛṛa mbla lxedma. </p> 
 <p><a name="an_20:4">20:4</a> yenna-yasen : Ṛuḥet ula d kunwi aț-țxedmem di tfeṛṛant-iw, a wen-fkeɣ ayen tuklalem.  </p> 
 <p><a name="an_20:5">20:5</a> Dɣa ṛuḥen. Bab n tfeṛṛant-nni yexdem akken daɣen ɣef tnac n wass akk-d ț-țlata n tmeddit.  </p> 
 <p><a name="an_20:6">20:6</a> Iffeɣ daɣen ɣef lxemsa n tmeddit, yufa wiyaḍ qqimen di beṛṛa yenna-yasen : Acuɣeṛ i tețɣimim ass kamel mbla lxedma ?  </p> 
 <p><a name="an_20:7">20:7</a> Nnan-as-ed : Ulac win i ɣ-d-ifkan lxedma ! YYenna-yasen : Ṛuḥet ula d kunwi aț-țxedmem di tfeṛṛant-iw !  </p> 
 <p><a name="an_20:8">20:8</a> Mi i d-tewweḍ tmeddit, bab n tfeṛṛant yenna i lewkil-is : Ssiwel i ixeddamen txellseḍ-ten yiwen yiwen. Zwir seg ineggura tessegriḍ imezwura.  </p> 
 <p><a name="an_20:9">20:9</a> Usan-d wid yebdan lxedma ɣef lxemsa n tmeddit, ixelleṣ-iten s ssuma n yiwen wass, </p> 
 <p><a name="an_20:10">20:10</a> mi i d-tewweḍ nnuba n wid ibdan lxedma ɣef țesɛa n ṣṣbeḥ, ɣilen ad țwaxelṣen akteṛ n wiyaḍ, meɛna ula d nutni țwaxellṣen s ssuma n yiwen wass.  </p> 
 <p><a name="an_20:11">20:11</a> Mi ṭṭfen lexlaṣ-nsen, iɣaḍ-iten lḥal, nnan i bab n tfeṛṛant-nni :  </p> 
 <p><a name="an_20:12">20:12</a> Ixeddamen-agi ineggura xedmen yiwet n ssaɛa kan, txellṣeḍ ten am nukkni yenɛețțaben ass kamel i wezɣal ?  </p> 
 <p><a name="an_20:13">20:13</a> Meɛna yerra-yas i yiwen seg sen : Ay amdakkel, acuɣeṛ i k-iɣaḍ lḥal ? Eɛni ur nemsefham ara ɣef lexlaṣ-nwen ad yili tawizeț n lfeṭṭa d ssuma n yiwen wass ? </p> 
 <p><a name="an_20:14">20:14</a> Awi lḥeqq-ik tṛuḥeḍ. I ma yella bɣiɣ ad fkeɣ i uneggaru annect i k-fkiɣ i kečč ? </p> 
 <p><a name="an_20:15">20:15</a> Eɛni ur zmireɣ ara ad xedmeɣ akken bɣiɣ s idrimen-iw ? Neɣ iɣaḍ-ik lḥal imi ur lliɣ ara d amecḥaḥ ?  </p> 
 <p><a name="an_20:16">20:16</a> S wakka aṭas ineggura ara yuɣalen d imezwura, aṭas n imezwura ara yuɣalen d ineggura.  </p> 
 <p><a name="an_20:17">20:17</a> Mi țeddun ad alin ɣer temdint n Lquds, Sidna Ɛisa iṭṭef tnac-nni inelmaden-is weḥḥed-sen, yenna yasen :  </p> 
 <p><a name="an_20:18">20:18</a> Ḥesset-ed : atan a nali ɣer temdint n Lquds, anda ara ițțusellem Mmi-s n bunadem ger ifassen n lmuqedmin imeqqranen akk-d lɛulama n ccariɛa. Ad ḥekmen fell-as s lmut, </p> 
 <p><a name="an_20:19">20:19</a> yerna a t-sellmen ger ifassen nn ikafriwen iwakken ad stehzin fell-as, a t-jelkḍen ( a t-cellṭen ), a t-semmṛen ɣef wumidag, ass wis tlata a d-iḥyu si ger lmegtin.  </p> 
 <p><a name="an_20:20">20:20</a> Imiren yemma-s n Yeɛqub d Yuḥenna, arraw n Zabadi, tqeṛṛeb-ed ɣer Sidna Ɛisa nețțat d warraw-is, tseǧǧed ɣer iḍaṛṛen-is.  </p> 
 <p><a name="an_20:21">20:21</a> Sidna Ɛisa yenna-yas : D acu i tebɣiḍ ? Tenna-yas : Di leɛnaya-k a Sidi, efk-asen imukan i sin-agi n warraw-iw di tgelda-inek, iwakken ad qqimen wa ɣer uyeffus-ik, wa ɣer uzelmaḍ-ik.  </p> 
 <p><a name="an_20:22">20:22</a> Sidna Ɛisa yerra-yasen : Ur teẓrim ara ayen akka i d tessutrem ! Eɛni tzemrem aț-țeswem si tbuqalt-agi n lemṛaṛ ara sweɣ ? NNnan-as : Ih a Sidi, nezmer !  </p> 
 <p><a name="an_20:23">20:23</a> Yerra-yasen : Ț-țideț, aț-țeswem si tbuqalt-agi, meɛna ɣef wayen yeɛnan imukan ɣer tama-w tayeffust neɣ tazelmaṭ, mačči d nekk ara ten-ifken, ad țțunefken i wid iwumi i ten-ihegga Baba Ṛebbi.  </p> 
 <p><a name="an_20:24">20:24</a> Ɛecṛa n inelmaden-nni mi slan ayagi, ikcem-iten zzɛaf ɣef ddemma n sin-nni watmaten. </p> 
 <p><a name="an_20:25">20:25</a> Sidna Ɛisa yessawel-asen, yenna-yasen : Walit igelliden n yegduden nniḍen akk-d imeqqranen-nsen amek ḥeṛsen yerna ḥeqṛen lɣaci-nsen ! </p> 
 <p><a name="an_20:26">20:26</a> Ur ilaq ara ad yili wannect-agi gar-awen. Ma yella yiwen seg-wen yebɣa ad yuɣal d ameqqran, ilaq ad yili d aqeddac-nwen. </p> 
 <p><a name="an_20:27">20:27</a> Akken daɣen, win yebɣan ad yili d amezwaru, ilaq ad yuɣal d akli-nwen. </p> 
 <p><a name="an_20:28">20:28</a> Daymi Mmi-s n bunadem ur d-yusi ara iwakken ad qedcen fell-as, meɛna yusa-d iwakken ad yili d aqeddac, yerna ad isebbel tudert-is iwakken ad isellek aṭas n yemdanen.  </p> 
 <p><a name="an_20:29">20:29</a> Mi d-ffɣen si temdint n Yiriku, aṭas n lɣaci i d-iḍefṛen Sidna Ɛisa. </p> 
 <p><a name="an_20:30">20:30</a> Llan sin iderɣalen qqimen rrif n webrid, mi slan belli d Sidna Ɛisa i d-iɛeddan bdan țɛeggiḍen :  A Sidi ! A mmi-s n Sidna Dawed ! Ḥunn fell-aɣ !  </p> 
 <p><a name="an_20:31">20:31</a> Lɣaci-nni țlumun-ten, qqaṛen asen ad ssusmen, ma d nutni țkemmilen leɛyaḍ : Ḥunn fell-aɣ a Sidi, a mmi-s n Sidna Dawed !  </p> 
 <p><a name="an_20:32">20:32</a> Sidna Ɛisa iḥbes, issawel-asen, yenna-yasen : D acu i tebɣam a wen-t- xedmeɣ ? </p> 
 <p><a name="an_20:33">20:33</a> Rran-as : Nebɣa a ɣ-d-yuɣal yeẓri a Sidi !  </p> 
 <p><a name="an_20:34">20:34</a> Ɣaḍen aṭas Sidna Ɛisa, yennul allen-nsen, imiren kan yuɣal-iten-id yeẓri, dɣa ddan yid-es.  </p> 
 <p><a name="an_21:1">21:1</a> Mi qṛib ad kecmen ɣer temdint n Lquds, wwḍen ɣer yiɣil n uzemmur yellan tama n taddart n Bitfaji, Sidna Ɛisa iceggeɛ sin inelmaden-is, </p> 
 <p><a name="an_21:2">21:2</a> yenna-yasen : Ṛuḥet ɣer taddart yellan ɣer zdat, dinna aț-țafem taɣyult akk-d mmi-s, brut-asen-d, tawim-iyi-ten-id. </p> 
 <p><a name="an_21:3">21:3</a> Ma yella win i wen-d-yennan kra, init-as : d Ssid-nneɣ i ten-iḥwaǧen, imiren a sen-d-iserreḥ.  </p> 
 <p><a name="an_21:4">21:4</a> Ayagi yedṛa-d am akken i t-id yenna nnbi Zakarya di tira iqedsen :  </p> 
 <p><a name="an_21:5">21:5</a> Init i wat wedrar n Siyun : AA taya ugellid-nwen iteddu-d ɣuṛ-wen s wannuz, irkeb-ed ɣef weɣyul amecṭuḥ mmi-s n teɣyult.  </p> 
 <p><a name="an_21:6">21:6</a> Inelmaden-nni ṛuḥen, xedmen ayen i sen-d-yenna Sidna Ɛisa. </p> 
 <p><a name="an_21:7">21:7</a> Wwin-d taɣyult-nni d mmi-s, sburren-asen llebsa-nsen, dɣa srekben Sidna Ɛisa. </p> 
 <p><a name="an_21:8">21:8</a> Aṭas n lɣaci ssan iceṭṭiḍen-nsen deg ubrid, wiyaḍ țṛuẓun-d tiseḍwa si ttjuṛ, țessun-tent ula d nutni deg ubrid. </p> 
 <p><a name="an_21:9">21:9</a> Lɣaci yezwaren zdat Sidna Ɛisa d wid i t-id-iḍefṛen, țɛeggiḍen s lfeṛḥ qqaṛen : Ḥusana i mmi-s n Sidna Dawed ! Ițțubarek win i d-yusan s yisem n Sidi Ṛebbi ! Ḥusana deg imukan eɛlayen.  </p> 
 <p><a name="an_21:10">21:10</a> Mi gekcem Sidna Ɛisa ɣer temdint n Lquds, imezdaɣ meṛṛa nhewwalen, qqaṛen : Anwa-t wagi ?  </p> 
 <p><a name="an_21:11">21:11</a> Lɣaci-nni țțaran-asen : D Sidna Ɛisa, nnbi-nni n taddart n Naṣaret i d-yusan si tmurt n Jlili !  </p> 
 <p><a name="an_21:12">21:12</a> Sidna Ɛisa yekcem ɣer lǧameɛ iqedsen, iqecceɛ meṛṛa wid yețțaɣen znuzun dinna deg ufrag n lǧameɛ. Iqleb-asen ṭṭwabel i wid yețbeddilen idrimen akk-d wid yeznuzun itbiren. </p> 
 <p><a name="an_21:13">21:13</a> dɣa yenna-yasen : Yura : Axxam-iw ad ițțusemmi d axxam n tẓallit, ma d kunwi terram-t d lɣaṛ n imakaren.  </p> 
 <p><a name="an_21:14">21:14</a> Di lǧameɛ-nni, qeṛṛben-d ɣuṛ-es iderɣalen d iquḍaren, dɣa yesseḥla ten.  </p> 
 <p><a name="an_21:15">21:15</a> Lmuqedmin imeqqranen d lɛulama n ccariɛa rfan mi walan leɛǧayeb i gxeddem akk-d warrac ițɛeggiḍen di lǧameɛ : Ḥusana i mmi-s n Sidna Dawed !  </p> 
 <p><a name="an_21:16">21:16</a> Imiren nnan i Sidna Ɛisa : Tesliḍ d acu i d-qqaṛen ? SSidna Ɛisa yerra-yasen : Sliɣ ih ! Eɛni ur teɣṛim ara imeslayen-agi yuran di tira iqedsen : Tessufɣeḍ-ed leḥmadi seg imawen n warrac akk-d wid n llufanat iteṭṭḍen .  </p> 
 <p><a name="an_21:17">21:17</a> Imiren yeǧǧa-ten dinna, yeffeɣ si temdint, iṛuḥ ɣer taddart n Bitanya, dinna i gensa.  </p> 
 <p><a name="an_21:18">21:18</a> Azekka-nni taṣebḥit mi d-yuɣal ɣer temdint, Sidna Ɛisa yelluẓ ; </p> 
 <p><a name="an_21:19">21:19</a> iwala yiwet n tneqleț rrif n webrid, iqeṛṛeb ɣuṛ-es, ur yufi deg-s acemma anagar afriwen ; dɣa yenna-yas : Seg wass-a, ur tețțuɣaleḍ ara a d-tefkeḍ lfakya ! IImiren kan, taneqleț-nni teqquṛ. </p> 
 <p><a name="an_21:20">21:20</a> Inelmaden-is mi walan ayagi tɛeǧben, nnan : Amek akka teqquṛ tneqleț-agi deg yiwet n teswiɛt ?  </p> 
 <p><a name="an_21:21">21:21</a> Sidna Ɛisa yerra-yasen : A wen-iniɣ tideț, ma yella tesɛam liman ur tețcukkum ara, aț-țizmirem aț-țxedmem akteṛ n wayen xedmeɣ i tneqleț-agi ! AAț-ținim i wedrar-agi qleɛ iman-ik syagi tḍeggṛeḍ iman-ik ɣer lebḥeṛ, ayagi a d-yedṛu !  </p> 
 <p><a name="an_21:22">21:22</a> Ayen akk ara tessutrem s liman di tẓallit, a wen-d-ițunefk !  </p> 
 <p><a name="an_21:23">21:23</a> Sidna Ɛisa yekcem ɣer lǧameɛ ibda yesselmad. Lmuqedmin akk-d imeqqranen n wegdud qeṛṛben ɣuṛ-es nnan-as : Ansi i k-d-tekka tezmert s wayes i txeddmeḍ ayagi ? Anwa i k-d-yefkan tazmert-agi ?  </p> 
 <p><a name="an_21:24">21:24</a> Sidna Ɛisa yerra-yasen : A wen-fkeɣ asteqsi ula d nekk, yiwen kan ; ma tǧawbem-iyi-d, ula d nekk a wen-iniɣ ansi i yi-d-tekka tezmert-agi. </p> 
 <p><a name="an_21:25">21:25</a> Aɣḍas n Yeḥya, s ɣuṛ Ṛebbi i d-ikka neɣ s ɣuṛ yemdanen ? DDɣa bdan qqaṛen wway gar-asen : Ma nenna-yas seg igenwan, a ɣ-d-yini acuɣeṛ ihi ur tuminem ara yis, </p> 
 <p><a name="an_21:26">21:26</a> ma nenna-yas daɣen seg imdanen, nuggad lɣaci-agi, axaṭer ḥesben akk Yeḥya d nnbi.  </p> 
 <p><a name="an_21:27">21:27</a> Dɣa nnan-as i Sidna Ɛisa : Ur neẓri ara ! YYerra-yasen : Ihi ula d nekk ur wen-d-qqaṛeɣ ara ansi i yi-d-tekka tezmert s wayes i xeddmeɣ ayagi !  </p> 
 <p><a name="an_21:28">21:28</a> D acu twalam ? Yiwen wergaz yesɛa sin warraw-is, yenṭeq ɣer umezwaru, yenna-yas : A mmi, ṛuḥ aț-țxedmeḍ ass-agi di tfeṛṛant-iw.  </p> 
 <p><a name="an_21:29">21:29</a> Yerra-yas : Ur țṛuḥuɣ ara ! MMeɛna yuɣal yendem, iṛuḥ ad ixdem di tfeṛṛant.  </p> 
 <p><a name="an_21:30">21:30</a> Yenna-yas daɣen i mmi-s wi sin ad iṛuḥ ad ixdem, yerra-yas-d :  Ih a baba ad ṛuḥeɣ ! MMeɛna ur iṛuḥ ara. </p> 
 <p><a name="an_21:31">21:31</a> Anwa deg sen i gxedmen lebɣi n baba-s ? RRran-as-ed : D amezwaru a Sidi ! SSidna Ɛisa yenna-yasen : A wen-d-iniɣ tideț, imekkasen imakaren akk-d yir tilawin a kkun zwiren ɣer tgelda n igenwan !  </p> 
 <p><a name="an_21:32">21:32</a> Axaṭer Yeḥya yusa-d ɣuṛ-wen yesken-awen-d abrid n tideț, meɛna ur t-tuminem ara ; ma d imekkasen d yir tilawin umnen yis. Yerna ɣas akken twalam ayagi, ur d-tuɣalem ara ɣer webrid teggumam aț-țamnem yis.  </p> 
 <p><a name="an_21:33">21:33</a> A wen-d-rnuɣ daɣen lemtel nniḍen : yella yiwen umɛellem, yeẓẓa tafeṛṛant, izzi-yas-ed s zzeṛb, yeɣza amkan i tɛeṣṣart n tẓurin, yebna taɛrict i tɛessast, issekra tafeṛṛant-is i ixemmasen, imiren iṛuḥ ɣer lɣeṛba.  </p> 
 <p><a name="an_21:34">21:34</a> Mi d-yewweḍ lweqt n tẓurin, iceggeɛ iqeddacen-is ɣer ixemmasen nni iwakken a s-d-fken amur-is n lɣella.  </p> 
 <p><a name="an_21:35">21:35</a> Meɛna ixemmasen-nni ṭṭfen iqeddacen-nni, wa wten-t, wa nɣan-t, wayeḍ ṛejmen-t. </p> 
 <p><a name="an_21:36">21:36</a> Iceggeɛ-asen daɣen tarbaɛt nniḍen n iqeddacen yugaren tamezwarut. Meɛna tedṛa yid-sen am imezwura-nni.  </p> 
 <p><a name="an_21:37">21:37</a> Taggara, iceggeɛ-asen mmi-s, yenna : Ahat imi d mmi a t-qadṛen !  </p> 
 <p><a name="an_21:38">21:38</a> Lameɛna mi walan mmi-s, ixemmasen-nni nnan wway gar-asen : Ataya win ara iweṛten, kkret a t-nenneɣ iwakken a nawi lweṛt-is.  </p> 
 <p><a name="an_21:39">21:39</a> Sṭfen-t, ḍeggṛen-t ɣer beṛṛa n tfeṛṛant, dɣa nɣan-t.  </p> 
 <p><a name="an_21:40">21:40</a> Sidna Ɛisa yenna i lɣaci : Ihi tura, m'ara d-yas bab n tfeṛṛant-nni, d acu ara sen-yexdem i ixemmasen-agi ?  </p> 
 <p><a name="an_21:41">21:41</a> Rran-as-ed : Ad ineɣ ixemmasen imcumen, ma ț-țafeṛṛant a ț-issekru i ixemmasen nniḍen ara s-d-ifken amur-is di lweqt n lɣella.  </p> 
 <p><a name="an_21:42">21:42</a> Sidna Ɛisa yenna-yasen : Leɛmeṛ teɣṛim imeslayen-agi i d-nnant tira iqedsen : Adɣaɣ-nni i ḍeggṛen wid ibennun, d nețța i guɣalen d azṛu alemmas , WWin yeṭṭfen lebni. AA yagi yekka-d s ɣuṛ Ṛebbi, d ayen i ɣ-istɛeǧben .  </p> 
 <p><a name="an_21:43">21:43</a> Daymi i wen-qqaṛeɣ : Tagelda n igenwan a wen tețwakkes, aț-țețțunefk i leǧnas nniḍen ara d-ifken lɣella.  </p> 
 <p><a name="an_21:44">21:44</a> Win ara yeɣlin ɣef wusalas-agi ad yerreẓ, daɣen win i ɣef ara d-yeɣli usalas-agi ad yețwamḥeq.  </p> 
 <p><a name="an_21:45">21:45</a> Mi slan i imeslayen-agi, ifariziyen d lmuqedmin imeqqranen fehmen belli fell-asen i d-iheddeṛ Sidna Ɛisa. </p> 
 <p><a name="an_21:46">21:46</a> Meɛna ɣas akken bɣan a t-ṭṭfen, ur zmiren ara axaṭer uggaden lɣaci-nni i t-iḥesben d nnbi.  </p> 
 <p><a name="an_22:1">22:1</a> Sidna Ɛisa imeslay-asen-d daɣen s lemtel, yenna-yasen : -- </p> 
 <p><a name="an_22:2">22:2</a> Tagelda n igenwan tcuba ɣer yiwen ugellid i gxedmen tameɣṛa i mmi-s.  </p> 
 <p><a name="an_22:3">22:3</a> Iceggeɛ iqeddacen-is ad ssiwlen i wid yețwaɛeṛden, meɛna ugin a d-asen.  </p> 
 <p><a name="an_22:4">22:4</a> Yerna iceggeɛ-asen iqeddacen nniḍen a sen-inin : Heggaɣ imensi, zliɣ izgaren-iw d wakraren-iw, kullec ihegga, aset-ed ɣer tmeɣṛa !  </p> 
 <p><a name="an_22:5">22:5</a> Meɛna widak-nni ițwaɛeṛḍen ur s-fkin ara azal, yal yiwen yelha d ccɣel-is, wa iṛuḥ ɣer yiger-is, wa ɣer ssuq-is.  </p> 
 <p><a name="an_22:6">22:6</a> Wiyaḍ ṭṭfen iqeddacen-nni, regmen-ten, nɣan-ten. </p> 
 <p><a name="an_22:7">22:7</a> Agellid-nni ikcem-it zzɛaf d ameqqran, dɣa iceggeɛ lɛeskeṛ-is nɣan iqettalen-nni, sseṛɣen tamdint-nsen. </p> 
 <p><a name="an_22:8">22:8</a> Imiren yenna i iqeddacen-is : Imensi n tmeɣṛa ihegga, meɛna widak-nni ițwaɛeṛḍen ur t-uklalen ara ; </p> 
 <p><a name="an_22:9">22:9</a> ṛuḥet ɣer izenqan, tɛeṛḍem-d ɣer tmeɣṛa wid akk ara tafem deg iberdan.  </p> 
 <p><a name="an_22:10">22:10</a> Iqeddacen-nni ṛuḥen ɣer izenqan jemɛen-d wid akk ufan, ama d amcum ama d win yelhan, s wakka axxam n tmeɣṛa yeččuṛ-ed d lɣaci. </p> 
 <p><a name="an_22:11">22:11</a> Mi d-ikcem ugellid ad iwali inebgawen, iwala yiwen wergaz ur yelsi ara llebsa n tmeɣṛa. </p> 
 <p><a name="an_22:12">22:12</a> yenna-yas :  Ay amdakkel, amek i d-tkecmeḍ ɣer dagi mbla llebsa n tmeɣṛa yețwaheggan i yinebgawen ? AArgaz-nni yessusem, ur yufi d acu ara s-yini. </p> 
 <p><a name="an_22:13">22:13</a> Agellid-nni yefka lameṛ i iqeddacen-is yenna-yasen : Cuddet-as ifassen d iḍaṛṛen, tḍeggṛem-t ɣer ṭṭlam n beṛṛa, anda llan imeṭṭawen d nndama tameqqrant.  </p> 
 <p><a name="an_22:14">22:14</a> Axaṭer aṭas i gețwaɛerḍen, meɛna drus ara yețwaqeblen.  </p> 
 <p><a name="an_22:15">22:15</a> Imiren ṛuḥen ifariziyen ad mcawaṛen amek ara d-sseɣlin Sidna Ɛisa deg wawal. </p> 
 <p><a name="an_22:16">22:16</a> Ceggɛen-as kra seg inelmaden-nsen akk-d kra si terbaɛt n Hiṛudus, nnan-as : A Sidi, neẓra-k d bab n lḥeqq i telliḍ, tesselmadeḍ iberdan n Ṛebbi s tideț, ur tuggadeḍ ula d yiwen, ur txeddmeḍ ara lxilaf ger yemdanen.  </p> 
 <p><a name="an_22:17">22:17</a> Ini-yaɣ-d ihi d acu twalaḍ : d leḥlal neɣ d leḥṛam a nxelleṣ tabzert ( leɣṛama ) i ugellid Qayṣar (yestɛemṛen tamurt-nneɣ) ?  </p> 
 <p><a name="an_22:18">22:18</a> Meɛna Sidna Ɛisa yeẓran ayen i țxemmimen, yenna-yasen : Ay at sin wudmawen ! Acuɣeṛ tebɣam a yi-tjeṛbem ? </p> 
 <p><a name="an_22:19">22:19</a> Seknet-iyi-d aṣurdi s wacu țxelliṣen tabzert ! FFkan-as-d aṣurdi n lfeṭṭa.  </p> 
 <p><a name="an_22:20">22:20</a> Dɣa isteqsa-ten : Udem akk-d tira yellan deg uṣuṛdi-agi, wi ten-ilan ?  </p> 
 <p><a name="an_22:21">22:21</a> Rran-as : N Qayṣer ! IImiren yenna-yasen : Erret ihi i Qayṣer ayen yellan n Qayṣer, terrem i Ṛebbi ayen yellan n Ṛebbi ! </p> 
 <p><a name="an_22:22">22:22</a> Mi slan i imeslayen-agi, wehmen, dɣa ṛuḥen ǧǧan-t.  </p> 
 <p><a name="an_22:23">22:23</a> Ass-nni, isaduqiyen i gellan d imdanen ur nețțamen ara s ḥeggu n lmegtin, usan-d ɣer Sidna Ɛisa, steqsan-t, nnan-as :  </p> 
 <p><a name="an_22:24">22:24</a> A Sidi, Sidna Musa yenna-d : ma yella yemmut wergaz d amengur, ilaq gma-s ad yaɣ taǧǧalt-nni iwakken a d-isɛu yid-es dderya ara iweṛten gma-s-nni yemmuten.  </p> 
 <p><a name="an_22:25">22:25</a> Yuɣ lḥal llan ɣuṛ-nneɣ sebɛa watmaten, amezwaru izewǧen yemmut ur d-yeǧǧi dderya, dɣa gma-s yerra ( yuɣa ) tameṭṭut-nni. </p> 
 <p><a name="an_22:26">22:26</a> Akkenni i tedṛa d wis sin, d wis tlata, armi d wis sebɛa.  </p> 
 <p><a name="an_22:27">22:27</a> Mi mmuten irkulli, temmut ula ț-țameṭṭut-nni ; </p> 
 <p><a name="an_22:28">22:28</a> ass n ḥeggu n lmegtin anwa deg-sen ara ț-yesɛun ț-țameṭṭut-is imi i ț-uɣen di sebɛa yid-sen ?  </p> 
 <p><a name="an_22:29">22:29</a> Sidna Ɛisa yerra-yasen : Tɣelṭem ! Axaṭer ur tefhimem tira iqedsen, ur tessinem tazmert n Ṛebbi. </p> 
 <p><a name="an_22:30">22:30</a> Ass n ḥeggu n lmegtin, irgazen ț-țilawin ur zeggjen ara, meɛna ad uɣalen am lmalayekkat deg igenni.  </p> 
 <p><a name="an_22:31">22:31</a> Ɣef wayen yeɛnan ḥeggu n lmegtin, ur teɣṛim ara acu i d-yenna Sidi Ṛebbi ? : </p> 
 <p><a name="an_22:32">22:32</a> D nekk i d Illu n Ibṛahim, n Isḥaq, n Yeɛqub. Sidi Ṛebbi mačči d Ṛebbi n lmegtin, meɛna d Ṛebbi n wid yeddren.  </p> 
 <p><a name="an_22:33">22:33</a> Mi s-slan, lɣaci qqimen țweḥḥiden deg imeslayen-is !  </p> 
 <p><a name="an_22:34">22:34</a> Mi slan ifariziyen belli Sidna Ɛisa yeɣleb isaduqiyen deg wawal, nnejmaɛen wway gar-asen. </p> 
 <p><a name="an_22:35">22:35</a> Yiwen seg-sen d lɛalem n ccariɛa, isteqsa-t iwakken a t-ijeṛṛeb :  </p> 
 <p><a name="an_22:36">22:36</a> A Sidi, anwa i d lameṛ ameqqran meṛṛa di ccariɛa ?  </p> 
 <p><a name="an_22:37">22:37</a> Sidna Ɛisa yerra-yas : -- Aț-țḥemleḍ Illu Sidi Ṛebbi-inek seg wul-ik, s teṛwiḥt-ik, s wayen yellan di tezmert-ik.  </p> 
 <p><a name="an_22:38">22:38</a> D wagi i d lameṛ yellan d amezwaru yerna d ameqqran akk di ccariɛa. </p> 
 <p><a name="an_22:39">22:39</a> Atan wayeḍ yesɛan azal annect umezwaru-agi : Aț-țḥemleḍ lɣiṛ-ik am yiman-ik.  </p> 
 <p><a name="an_22:40">22:40</a> D sin lumuṛat-agi i d lsas n ccariɛa akk-d lenbiya.  </p> 
 <p><a name="an_22:41">22:41</a> Mi llan ifariziyen nnejmaɛen, Sidna Ɛisa isteqsa-ten yenna :  </p> 
 <p><a name="an_22:42">22:42</a> D acu ara d-tinim ɣef Lmasiḥ ? Ansi ara d-yekk, yeɛni anwi i d lejdud-is ? NNnan-as : D mmi-s n Sidna Dawed.  </p> 
 <p><a name="an_22:43">22:43</a> Sidna Ɛisa yenna-yasen : Amek ihi i s-isemma Dawed s tezmert n Ṛṛuḥ iqedsen « Sidi » mi genna :  </p> 
 <p><a name="an_22:44">22:44</a> Sidi Ṛebbi yenna i Ssid-iw : ṭṭef amkan ɣer tama-w tayeffust, alamma rriɣ iɛdawen-ik seddaw iḍaṛṛen-ik.  </p> 
 <p><a name="an_22:45">22:45</a> Ma yella ihi Dawed yessawel-as « Sidi » amek ara yili ihi d mmi-s ?  </p> 
 <p><a name="an_22:46">22:46</a> Yiwen deg-sen ur izmir a s-d yerr awal, seg wass-nni țțaggaden a t-steqsin.  </p> 
 <p><a name="an_23:1">23:1</a> Imiren Sidna Ɛisa yenṭeq ɣer lɣaci d inelmaden-is yenna yasen :  </p> 
 <p><a name="an_23:2">23:2</a> Lɛulama d ifariziyen sselmaden ccariɛa n Musa. </p> 
 <p><a name="an_23:3">23:3</a> Xedmet ayen akk ara wen-d-inin, meɛna ur xeddmet ara wayen xeddmen nutni, axaṭer ayen i d-qqaṛen ur t-xeddmen ara.  </p> 
 <p><a name="an_23:4">23:4</a> Țcuddun tiɛekkmin ẓẓayen țɛebbin-tent ɣef tuyat n yemdanen, ma d nutni țțagin a ten-ɛiwnen ula s yixef n iḍuḍan-nsen.  </p> 
 <p><a name="an_23:5">23:5</a> Ayen akk i xeddmen, xeddmen-t iwakken a ten-walin yemdanen, țɛelliqen tiḥerztin ț-țeɣṛisin ɣef yiman-nsen.  </p> 
 <p><a name="an_23:6">23:6</a> Ḥemmlen ad ṭṭfen imukan imezwura ama di tmeɣṛiwin ama di leǧwameɛ, ḥemmlen m'ara țsellimen fell-asen yemdanen di tejmuyaɛ akk-d leswaq ; </p> 
 <p><a name="an_23:7">23:7</a> ḥemmlen daɣen m'ara sen-qqaṛen yemdanen : « a Sidi ».  </p> 
 <p><a name="an_23:8">23:8</a> Ma d kunwi ɣuṛ-wat a wen qqaṛen : « a Sidi » axaṭer d atmaten i tellam yerna yiwen kan i d Ssid-nwen.  </p> 
 <p><a name="an_23:9">23:9</a> Ur qqaṛet ula i yiwen di ddunit : « Baba » axaṭer anagar yiwen i d baba-wen, d Baba Ṛebbi yellan deg igenwan.  </p> 
 <p><a name="an_23:10">23:10</a> Ur qebblet ara aț-țețțusemmim : « imeqqranen » axaṭer anagar yiwen i d ameqqran : d Lmasiḥ !   </p> 
 <p><a name="an_23:11">23:11</a> Ameqqran deg-wen, d win iqeddcen fell-awen.  </p> 
 <p><a name="an_23:12">23:12</a> Kra n win issimɣuṛen iman-is a d-yers, ma d win issimẓiyen iman is ad ițwarfed !  </p> 
 <p><a name="an_23:13">23:13</a> A nnger-nwen ay ifariziyen d lɛulama n ccariɛa, ay at sin wudmawen ! AAxaṭer tmeddlem tiwwura n igenwan i yemdanen, ur tkeččmem kunwi, ur tețțaǧǧam ad kecmen wid yebɣan ! </p> 
 <p><a name="an_23:14">23:14</a> A nnger-nwen ay ifariziyen d lɛulama n ccariɛa, ay at sin wudmawen ! Axaṭer tettețțem ayla n tuǧǧal, tesṭuqqutem awal di tẓallit-nwen iwakken a d-tesbegnem iman-nwen telham. Ɣef wayagi, lɛiqab i kkun-ițṛaǧun d ameqqran ! </p> 
 <p><a name="an_23:15">23:15</a> A nnger-nwen ay ifariziyen d lɛulama n ccariɛa, ay at sin udmawen ! Tețnadim timura, tzeggrem lebḥuṛ iwakken a d-rebḥem ɣas ula d yiwen ɣer ddin i tettabaɛem; mi t-id-rnam ɣuṛ-wen, tețțaram-t d win yuklalen ǧahennama akteṛ-nwen !  </p> 
 <p><a name="an_23:16">23:16</a> A nnger-nwen ay iderɣalen iteṭṭfen afus i wiyaḍ ! Teqqaṛem : « Ma yella yeggul bunadem s lǧameɛ iqedsen, ɣas yeḥnet ulac fell-as ; ma d win ara yegallen s ddheb yellan di lǧameɛ, ur s-ilaq ara ad iḥnet, ilaq ad iṭṭef di limin-ines ! »  </p> 
 <p><a name="an_23:17">23:17</a> Ay imehbal, ay iderɣalen ! D acu i gesɛan azal, d ddheb neɣ d lǧameɛ i gețțarran ddheb-nni d imqeddes ? </p> 
 <p><a name="an_23:18">23:18</a> Teqqaṛem daɣen : « Win ara yegallen s udekkan n iseflawen, ɣas iḥnet ulac fell-as, ma d win ara yegallen s iseflawen nni, ur s-ilaq ara ad iḥnet, ilaq ad iṭṭef di limin-ines.» </p> 
 <p><a name="an_23:19">23:19</a> Ay iderɣalen ! Anwa i gesɛan azal, d iseflawen neɣ d udekkan-nni i gerran iseflawen d imqedsen ?  </p> 
 <p><a name="an_23:20">23:20</a> Win ara yegallen s udekkan n iseflawen, yeggul s udekkan-nni d wayen akk yersen fell-as. </p> 
 <p><a name="an_23:21">23:21</a> Win ara yegallen daɣen s lǧameɛ, yeggul s lǧameɛ akk-d Win i gellan deg-s. </p> 
 <p><a name="an_23:22">23:22</a> Win ara yegallen s igenwan, yeggul daɣen s igenwan akk-d Sidi Ṛebbi iḥekkmen deg-sen. </p> 
 <p><a name="an_23:23">23:23</a> A nnger-nwen ay ifariziyen d lɛulama n ccariɛa, ay at sin udmawen ! tețțakem leɛcuṛ ɣef nneɛneɛ, abesbas d lkemmun, lameɛna tețțaǧǧam ɣer deffir lḥeqq, ṛṛeḥma d laman. DD annect-agi i gețṛaǧu deg-wen Sidi Ṛebbi aț-țezwirem deg-sen a ten-txedmem, mbla ma terram deg idis ayen nniḍen.  </p> 
 <p><a name="an_23:24">23:24</a> Ay iderɣalen iteṭṭfen afus i wiyaḍ ! Tețṣeffim tissit-nwen ɣef yizi, tesseblaɛem alɣem. </p> 
 <p><a name="an_23:25">23:25</a> A nnger-wen ay ifariziyen d lɛulama n ccariɛa, ay at sin wudmawen ! Axaṭer tessizdigem taqbuct d uḍebsi s ufella kan, meɛna ɣer daxel-nsen ččuṛen ț-țukeṛdiwin d ṭṭmeɛ.  </p> 
 <p><a name="an_23:26">23:26</a> Ay afarizi aderɣal ! Ssizdeg uqbel daxel n teqbuct d uḍebsi, iwakken ula s ufella-nsen ad izdigen !  </p> 
 <p><a name="an_23:27">23:27</a> A nnger-nwen ay ifariziyen d lɛulama n ccariɛa, ay at sin udmawen ! TTețțemcabim ɣer iẓekwan isebɣen s lǧir ; s ufella cebḥen, ɣer daxel ččuṛen d iɣsan n lmegtin d rekku. </p> 
 <p><a name="an_23:28">23:28</a> Akka ula d kunwi, tețbegginem-d iman-nwen telham ɣer yemdanen, lameɛna zdaxel teččuṛem d leɣruṛ d cceṛ. </p> 
 <p><a name="an_23:29">23:29</a> A nnger-nwen ay ifariziyen d lɛulama n ccariɛa, ay at sin wudmawen ! Tbennum iẓekwan n lenbiya, tețcebbiḥem iẓekwan n wid iḍuɛen Sidi Ṛebbi, </p> 
 <p><a name="an_23:30">23:30</a> teqqaṛem lemmer i nɛac di zzman n lejdud-nneɣ, tili ur neqbil ara a nțekki yid-sen deg ussizel n idammen n lenbiya.  </p> 
 <p><a name="an_23:31">23:31</a> S wakka tețcehhidem belli kunwi ț-țarwa n wid ineqqen lenbiya </p> 
 <p><a name="an_23:32">23:32</a> yerna kunwi tețkemmilem ayen xedmen lejdud-nwen !  </p> 
 <p><a name="an_23:33">23:33</a> A ccetla n izerman ! Amek ara tmenɛem si lɛiqab n ǧahennama ? </p> 
 <p><a name="an_23:34">23:34</a> Daymi ara wen-d-ceggɛeɣ lenbiya, imusnawen akk-d lɛulama. KKra seg-sen a ten-tenɣem, kra a ten-tsemmṛem ɣef wumidag, kra nniḍen a ten-tejlekḍem di leǧwameɛ nwen yerna a ten-tețqehhiṛem si temdint ɣer tayeḍ.  </p> 
 <p><a name="an_23:35">23:35</a> Ihi, idammen n iḥeqqiyen yuzzlen di lqaɛa, si Habil aḥeqqi mmi-s n Adem armi d Zakarya mmi-s n Birikya, i tenɣam ger lǧameɛ iqedsen d udekkan iseflawen, a d-uɣalen ɣer yiri-nwen.  </p> 
 <p><a name="an_23:36">23:36</a> A wen-d-iniɣ tideț : ayagi meṛṛa a d-yeɣli ɣef lǧil-agi !  </p> 
 <p><a name="an_23:37">23:37</a> A tamdint n Lquds ! A tamdint n Lquds ineqqen lenbiya, ireǧǧmen wid i m-d-ițuceggɛen s ɣuṛ Ṛebbi, acḥal n tikkal i bɣiɣ a d-jemɛeɣ arraw-im akken tjemmeɛ tyaziṭ ifṛax-is ddaw wafriwen-is, meɛna tegummaḍ ! </p> 
 <p><a name="an_23:38">23:38</a> Ihi atan wexxam-im ad yexlu !  </p> 
 <p><a name="an_23:39">23:39</a> Axaṭer aql-i a wen-iniɣ : ur tețțuɣalem ara a yi-twalim, alamma d ass i deg ara d-tinim : Yețțubarek win i d-yusan s yisem n Sidi Ṛebbi !  </p> 
 <p><a name="an_24:1">24:1</a> Mi d-iffeɣ si lǧameɛ iqedsen, Sidna Ɛisa iteddu ad iṛuḥ. Inelmaden-is qeṛṛben-d ɣuṛ-es, țwehhin-as iwakken ad yerr ddehn-is ɣer cbaḥa akk-d lebni n lǧameɛ iqedsen. </p> 
 <p><a name="an_24:2">24:2</a> Meɛna Sidna Ɛisa yenna-yasen : Twalam annect-agi meṛṛa ? A wen-iniɣ tideț : ur d-yețɣimi wezṛu ɣef wayeḍ, kullec ad ihudd. </p> 
 <p><a name="an_24:3">24:3</a> Yuli ɣer yiɣil n uzemmur iqqim weḥd-es, inelmaden-is usan-d a t-steqsin nnan-as : Melmi ara d-idṛu wayen i d tenniḍ, amek ara neɛqel ass ameqqran n tuɣalin-ik akk-d taggara n ddunit ?  </p> 
 <p><a name="an_24:4">24:4</a> Sidna Ɛisa yenna-yasen : Ɣuṛ-wat win ara wen-ikellxen ! </p> 
 <p><a name="an_24:5">24:5</a> Axaṭer aṭas n yemdanen ara d-yasen s yisem-iw, a d-inin : «` D nekk i d Lmasiḥ ! » Yerna ad kellxen aṭas n lɣaci. </p> 
 <p><a name="an_24:6">24:6</a> Aț-țeslem s ṭradat ț-țegrawliwin, meɛna ɣuṛ-wat a kkun-iffeɣ leɛqel. Ur țțaggadet ara, axaṭer ilaq a d-yedṛu wannect-a, meɛna mazal mačči d nnger n ddunit.  </p> 
 <p><a name="an_24:7">24:7</a> Agdud ad yennaɣ d wayeḍ, tagelda aț-țeẓdem ɣef tayeḍ, a d-yili laẓ d zzelzlat deg waṭas n imukan.  </p> 
 <p><a name="an_24:8">24:8</a> Ayagi akk ara yedṛun, ad yili am lqeṛḥ amezwaru n tmeṭṭut iṭṭfen addud.  </p> 
 <p><a name="an_24:9">24:9</a> Imiren a kkun-fken iwakken aț-țețțuɛetbem, a kkun-nɣen, leǧnas meṛṛa a kkun-keṛhen ɣef ddemma n yisem-iw.  </p> 
 <p><a name="an_24:10">24:10</a> Aṭas ara yeǧǧen liman-nsen, ad msekṛahen, ad țemyexdaɛen wway gar-asen.  </p> 
 <p><a name="an_24:11">24:11</a> A d-nnulfun lenbiya n lekdeb, ad ɣuṛṛen aṭas n lɣaci, </p> 
 <p><a name="an_24:12">24:12</a> simmal ad yețzad lbaṭel, simmal lmaḥibba n waṭas n yemdanen aț-țenqes. </p> 
 <p><a name="an_24:13">24:13</a> Ma d win ara yeṭṭfen di liman-is alamma ț-țaggara, ad ițțusellek.  </p> 
 <p><a name="an_24:14">24:14</a> Lexbaṛ-agi n lxiṛ ɣef tgelda n Sidi Ṛebbi, ad ițțubecceṛ di ddunit meṛṛa iwakken leǧnas ad slen yis. Imiren ara d-taweḍ taggara n ddunit.  </p> 
 <p><a name="an_24:15">24:15</a> M'ara twalim deg wemkan iqedsen win iwumi semman lmuṣiba issengaren i ɣef d-ihḍeṛ nnbi Danyel ; ( win ara yeɣṛen taktabt-agi n nnbi Danyel ad ifhem !)  </p> 
 <p><a name="an_24:16">24:16</a> Imiren, wid ara yilin di tmurt n Yahuda ilaq ad rewlen ɣer idurar, </p> 
 <p><a name="an_24:17">24:17</a> win ara yilin s ufella n ssqef, ur ilaq ara ad ikcem ɣer wexxam iwakken a d-iddem lqecc-is, </p> 
 <p><a name="an_24:18">24:18</a> win daɣen ara yilin di lexla, ur ilaq ara ad yuɣal ɣer deffir iwakken a d-iddem abeṛnus-is !  </p> 
 <p><a name="an_24:19">24:19</a> A tawaɣit n tilawin ara yilin s tadist akk-d țid ara yessuṭuḍen deg ussan-nni ! </p> 
 <p><a name="an_24:20">24:20</a> Dɛut ɣer Ṛebbi iwakken tarewla agi-nwen ur d-tdeṛṛu ara di ccetwa neɣ deg wass n westeɛfu. </p> 
 <p><a name="an_24:21">24:21</a> Axaṭer « ad yili yiwen n leɛtab ur d-nelli seg wasmi d-tebda ddunit ar ass-a yerna ur ițțuɣal ara a d-yili ! »  </p> 
 <p><a name="an_24:22">24:22</a> Lemmer ur issenqes ara Sidi Ṛebbi deg wussan-nni, tili yiwen ur imenneɛ, meɛna ɣef ddemma n wid yextaṛ ara yeṣṣenqes i wussan-agi.  </p> 
 <p><a name="an_24:23">24:23</a> Imiren ma yenna-yawen-d yiwen : Lmasiḥ atan dagi neɣ atan, dihin, ur țțamnet ara ! </p> 
 <p><a name="an_24:24">24:24</a> Axaṭer a d-asen wid ara yerren iman-nsen d Lmasiḥ neɣ d lenbiya, ad xedmen lbeṛhanat d leɛǧayeb akk-d licaṛat iwakken ad kelxen ma zemren ula d wid yextaṛ Sidi Ṛebbi.  </p> 
 <p><a name="an_24:25">24:25</a> Atan nniɣ-awen-t-id ihi : </p> 
 <p><a name="an_24:26">24:26</a> ma nnan-awen Lmasiḥ atan deg unezṛuf, ur țṛuḥut ara, neɣ ma nnan-awen walit, atan deg umkan-ihin yeffren, ur țțamnet ara. </p> 
 <p><a name="an_24:27">24:27</a> Axaṭer asm'ara d-yas Mmi-s n bunadem, ad yili am tafat n lbeṛeq m'ara d-tfeǧǧeǧ si cceṛq alamma d lɣeṛb.  </p> 
 <p><a name="an_24:28">24:28</a> Anda yella umuṛḍus ( lǧifa ), dinna ara nnejmaɛen igudar.  </p> 
 <p><a name="an_24:29">24:29</a> M'ara ɛeddin wussan-nni n leɛtab,  iṭij ad inqes, aggur ur d-ițțak ara tiziri-ines, itran a d-ɣlin seg igenni,  tizmar n igenwan ad rgagint.  </p> 
 <p><a name="an_24:30">24:30</a> Imiren a d-iban Mmi-s n bunadem deg igenni, leɛṛac meṛṛa n ddunit ad meǧden, a d wwten deg idmaren-nsen m'ara walin Mmi-s n bunadem deg igenwan, yusa-d ɣef wusigna s tezmert akk-d lɛaḍima tameqqrant.   </p> 
 <p><a name="an_24:31">24:31</a> A d-iceggeɛ lmalayekkat-is ad berrḥent s ṣṣut n lbuq, a d snejmaɛent wid akk yextaṛ Sidi Ṛebbi, si yal amkan, seg ixef n ddunit ɣer wayeḍ.  </p> 
 <p><a name="an_24:32">24:32</a> Meyzet ɣef lemtel n tneqleț : mi lqaqit isegman-is, fsan wafriwen-is, teḥsam yewweḍ-d unebdu !  </p> 
 <p><a name="an_24:33">24:33</a> Ula d kunwi, m'ara twalim yewweḍ-ed wannect-agi meṛṛa, ḥṣut belli Mmi-s n bunadem ițeddu-d, atan ɣer tewwurt.  </p> 
 <p><a name="an_24:34">24:34</a> A wen-d-iniɣ tideț : lǧil-agi ur ițɛedday ara alamma ideṛṛu-d wannect agi meṛṛa. </p> 
 <p><a name="an_24:35">24:35</a> Igenni d lqaɛa ad fnun, ma d imeslayen-iw ur fennun ara !  </p> 
 <p><a name="an_24:36">24:36</a> Meɛna ɣef wayen yeɛnan ass neɣ ssaɛa n wannect-agi meṛṛa, yiwen ur ten-issin, ama d lmalayekkat n igenwan, ama d Mmi-s n Ṛebbi, anagar Baba Ṛebbi i geẓran.  </p> 
 <p><a name="an_24:37">24:37</a> Am akken i tedṛa di zzman n nnbi Nuḥ, akken daɣen ara tedṛu di lweqt n Mmi-s n bunadem. </p> 
 <p><a name="an_24:38">24:38</a> Axaṭer deg ussan-nni uqbel a d-neflen waman ɣef ddunit, imdanen llan tețțen, tessen, țemyezwaǧen, zeggjen i warraw-nsen, armi d ass i deg yekcem nnbi Nuḥ ɣer lbabuṛ.  </p> 
 <p><a name="an_24:39">24:39</a> Yiwen ur ifaq s wacemma armi d-ḥemlen waman i ten-yewwin, snegren-ten akk. AAkken ara tedṛu di lweqt n tuɣalin n Mmi-s n bunadem ! </p> 
 <p><a name="an_24:40">24:40</a> Imiren, ger sin yergazen ara yilin deg iger, yiwen ad ițwarfed wayeḍ a d-iqqim. </p> 
 <p><a name="an_24:41">24:41</a> Ger snat n tilawin yeẓẓaden s tsirt, yiwet aț-țețwarfed tayeḍ a d-teqqim !  </p> 
 <p><a name="an_24:42">24:42</a> Ɛasset ihi ur ɣefflet ara ! Axaṭer ur teẓrim ara lweqt i deg ara d-yas Ssid-nwen ! </p> 
 <p><a name="an_24:43">24:43</a> Rret ddehn-nwen ɣer wayagi : lemmer yeẓri bab n wexxam lweqt i deg ara d-yas umakar, tili ad iɛiwez ur iggan ara, iwakken ur t-țțakren ara.  </p> 
 <p><a name="an_24:44">24:44</a> Daymi ula d kunwi heggit iman-nwen axaṭer di lweqt i ɣef ur tebnim ara, ara d-yas Mmi-s n bunadem.  </p> 
 <p><a name="an_24:45">24:45</a> Anwa i d aqeddac aɛeqli i ɣef ara yețkel bab n wexxam ? D win ara iwekkel ɣef wat wexxam-is meṛṛa, iwakken a sen-yefk lqut-nsen di lawan. </p> 
 <p><a name="an_24:46">24:46</a> Amarezg n uqeddac-agi ara d-yaf bab n wexxam m'ara d-yaweḍ, yelha d ccɣel-nni.  </p> 
 <p><a name="an_24:47">24:47</a> A wen-iniɣ tideț : a t-yerr d lewkil ɣef wayla-s meṛṛa !  </p> 
 <p><a name="an_24:48">24:48</a> Meɛna ma yella axeddam-agi diri-t, ad yini deg ul-is : bab n wexxam iɛeṭṭel, mazal ur d-ițas ara ;  </p> 
 <p><a name="an_24:49">24:49</a> dɣa ad ibdu ad ikkat iqeddacen n wexxam-nni, ad iteț, ad itess nețța d isekṛanen. </p> 
 <p><a name="an_24:50">24:50</a> Bab n wexxam a d-yas deg wass i ɣef ur ibni ara uqeddac-nni, di lawan ur yeẓri. </p> 
 <p><a name="an_24:51">24:51</a> A t-iqecceɛ, a t-iɛaqeb akken țwaɛaqben at sin udmawen : imiren ara yilin imeṭṭawen d nndama tameqqrant.  </p> 
 <p><a name="an_25:1">25:1</a> Daymi deg wussan-nni, tagelda n igenwan aț-țecbu ɣer ɛecṛa telmeẓyin i geddmen tiftilin-nsent iwakken ad ṛuḥent ad mmagrent isli. </p> 
 <p><a name="an_25:2">25:2</a> Xemsa seg-sent ț-țuḥṛicin, xemsa nniḍen ț-țungifin.  </p> 
 <p><a name="an_25:3">25:3</a> Mi d-ddment tungifin-nni tiftilin nsent, ur d-wwint ara zzit yid-sent. </p> 
 <p><a name="an_25:4">25:4</a> Ma ț-țuḥṛicin-nni, wwint-ed yid-sent tiqbucin n zzit.  </p> 
 <p><a name="an_25:5">25:5</a> Imi i gɛeṭṭel yesli, tilmeẓyin-nni nuddment dɣa ṭsent. </p> 
 <p><a name="an_25:6">25:6</a> Di tlemmast n yiḍ, yiwet n taɣect tɛeggeḍ-ed : Ataya yesli iteddu-d, ffɣemt a t-temmagremt !  </p> 
 <p><a name="an_25:7">25:7</a> Imiren kan ukint-ed akk telmeẓyin-nni, heggant tiftilin-nsent. </p> 
 <p><a name="an_25:8">25:8</a> Tungifin-nni nnant i tuḥṛicin : Fkemt-aɣ-d kra si zzit-nkunt, tiftilin-nneɣ qṛib ad xsint.  </p> 
 <p><a name="an_25:9">25:9</a> Tuḥṛicin rrant-asent : Ulamek, ur aɣ-d-tkeffu ara i nekkunti d kunemti, ṛuḥemt a d taɣemt i yiman-nkunt ɣer wid yeznuzun.  </p> 
 <p><a name="an_25:10">25:10</a> Mi ṛuḥent a d-aɣent, isli yewweḍ-ed, tuḥṛicin-nni iheggan iman-nsent kecment yid-es ɣer wexxam n tmeɣṛa, dɣa rran-d tawwurt.  </p> 
 <p><a name="an_25:11">25:11</a> Mi i d-wwḍent tungifin-nni, bdant țɛeggiḍent : A Sidi, a Sidi ldi-yaɣ-d tawwurt.  </p> 
 <p><a name="an_25:12">25:12</a> Meɛna nețța yerra-yasent-ed : A kunt-iniɣ tideț : ur kkunt ssineɣ ara !  </p> 
 <p><a name="an_25:13">25:13</a> Daymi a wen-iniɣ : Ɛasset, imi ur teẓrim ass, ur teẓrim lweqt.  </p> 
 <p><a name="an_25:14">25:14</a> Ass n tuɣalin n Mmi-s n bunadem ad yili daɣen am yiwen wergaz i gțeddun ad iṛuḥ ɣer lɣeṛba, yesnejmaɛ-ed iqeddacen-is, iwekkel iten ɣef wayen akk yesɛa.  </p> 
 <p><a name="an_25:15">25:15</a> Ifka i umezwaru xemsmeyya twiztin n ddheb, wi sin yefka-yas mitin, wis tlata yefka-yas meyya ; yefka i yal yiwen s wakken tella tezmert-is, dɣa iṛuḥ.  </p> 
 <p><a name="an_25:16">25:16</a> Imiren kan, aqeddac-nni iwumi yefka xemsmeyya twiztin, iṛuḥ yetjeṛ yis-sent, irbeḥ-ed xemsmeyya nniḍen.  </p> 
 <p><a name="an_25:17">25:17</a> Ula d win iwumi yefka mitin, ixdem akken, irbeḥ-ed mitin nniḍen.  </p> 
 <p><a name="an_25:18">25:18</a> Ma d win iwumi yefka meyya, iṛuḥ yeɣza di lqaɛa, yeffer tiwiztin-nni i s-d-ifka umɛellem-is.  </p> 
 <p><a name="an_25:19">25:19</a> Mi gɛedda lweqt, yuɣal-ed umɛellem-nni, isnejmaɛ-iten-id iwakken ad imḥasab yid-sen.  </p> 
 <p><a name="an_25:20">25:20</a> Win akken iwumi yefka xemsmeyya twiztin, yusa-d yewwi-d yid-es xemsmeyya twiztin nniḍen, yenna-yas : A Sidi tefkiḍ-iyi-d xemsmeyya twiztin n ddheb, a tent-ih xemsmeyya nniḍen i d-rebḥeɣ yis-sent.  </p> 
 <p><a name="an_25:21">25:21</a> Amɛellem-is yenna-yas : D ayen yelhan i txedmeḍ ay aqeddac unṣiḥ, imi deg-ek laman ɣef tɣawsiwin timecṭuḥin, sya d asawen a k-wekkleɣ ɣef timeqqranin ! Ili-k di lfeṛḥ yid-i !  </p> 
 <p><a name="an_25:22">25:22</a> Aqeddac-nni iwumi yefka mitin twiztin n ddheb, iqeṛṛeb-ed ula d nețța, yenna-yas : A Sidi tefkiḍ-iyi mitin twiztin n ddheb, a tent-ih mitin nniḍen i d-rebḥeɣ yis-sent.  </p> 
 <p><a name="an_25:23">25:23</a> Amɛellem-is yenna-yas : D ayen yelhan i txedmeḍ, ay aqeddac unṣiḥimi deg-k laman ɣef tɣawsiwin timecṭuḥin, sya d asawen a k-wekkleɣ ɣef timeqqranin ! Ili-k di lfeṛḥ yid-i !  </p> 
 <p><a name="an_25:24">25:24</a> Iwweḍ-ed win akken iwumi yefka meyya twiztin, yenna yas : A Sidi ẓriɣ-k d argaz iweɛṛen, tmeggreḍ anda ur tezriɛeḍ, tjemmɛeḍ seg wennar ayen ur tesrewteḍ , </p> 
 <p><a name="an_25:25">25:25</a> dɣa uggadeɣ-k, daymi i ṛuḥeɣ ffreɣ deg wakal tiwiztin-nni i yi-tefkiḍ. Atan wayla-k, ddem-it !  </p> 
 <p><a name="an_25:26">25:26</a> Amɛellem-is yenna-yas : Ay aqeddac amcum, ameɛdaz ! Imi iyi-teẓriḍ meggreɣ-d anda ur zriɛeɣ, jemmɛeɣ-ed daɣen anda ur srewteɣ, </p> 
 <p><a name="an_25:27">25:27</a> acuɣeṛ ihi ur tessekraḍ ara idrimin-iw iwakken m'ara d-uɣaleɣ a ten-id-erreɣ s lfayda.  </p> 
 <p><a name="an_25:28">25:28</a> Imiren amɛellem-nni yumeṛ i iqeddacen-is yenna-yasen : Kkset-as meyya-nni twiztin ternum-tent i win yesɛan ɛecṛa alef n twiztin ! </p> 
 <p><a name="an_25:29">25:29</a> Axaṭer kra win yesɛan, a s-nernu ad yili di tawant, ma d win ur nesɛi ara, a s-nekkes ula d ayen yesɛa.  </p> 
 <p><a name="an_25:30">25:30</a> Ma d aqeddac-agi ur nemɛin, ḍeggṛet-eț ɣer ṭṭlam n beṛṛa anda ara yilin imeṭṭawen d nndama tameqqrant.  </p> 
 <p><a name="an_25:31">25:31</a> Asm'ara d-yas Mmi-s n bunadem di tmanegt-is nețța d lmalayekkat meṛṛa, ad iḥkem di lɛaḍima-s ;  </p> 
 <p><a name="an_25:32">25:32</a> leǧnas meṛṛa a d-nnejmaɛen zdat-es, a ten-ifṛeq am umeksa iɛeSlen ulli ɣef tɣeṭṭen. </p> 
 <p><a name="an_25:33">25:33</a> Ad yerr ulli ɣer uyeffus-is, tiɣeṭṭen ɣer uzelmaḍ-is.  </p> 
 <p><a name="an_25:34">25:34</a> Imiren agellid ad yini i wid yellan ɣer uyeffus-is : Aset-ed a kunwi iburek Baba, aț-țweṛtem tagelda i wen-ițțuheggan seg wasmi i tebda ddunit.  </p> 
 <p><a name="an_25:35">25:35</a> Axaṭer lliɣ lluẓeɣ tefkam-iyi ččiɣ, ffudeɣ tefkam-iyi swiɣ, usiɣ-ed d abeṛṛani testeṛḥbem yis-i, </p> 
 <p><a name="an_25:36">25:36</a> lliɣ d aɛeryan tesselsem-iyi, d amuḍin telham-d yid-i, lliɣ daɣen d ameḥbus tusam-d teẓram-iyi.  </p> 
 <p><a name="an_25:37">25:37</a> Imiren iḥeqqiyen a s-d-rren : A Sidi melmi i k-neẓra telluẓeḍ, nefka-yak teččiḍ ? Neɣ neẓra-k teffuḍeḍ nefka-yak teswiḍ ?  </p> 
 <p><a name="an_25:38">25:38</a> Melmi daɣen i d-tusiḍ d abeṛṛani nesṭeṛḥeb yis-ek, neɣ melmi i k-nwala d aɛeryan, nessels-ik ? </p> 
 <p><a name="an_25:39">25:39</a> Melmi i k-neẓra d amuḍin neɣ di lḥebs nusa-d ɣuṛ-ek ?  </p> 
 <p><a name="an_25:40">25:40</a> Agellid a sen-d-yerr : A wen-iniɣ tideț, yal tikkelt i deg txedmem ayagi i yiwen seg imecṭuḥen-agi yellan d atmaten-iw, i nekk iwumi t-txedmem.   </p> 
 <p><a name="an_25:41">25:41</a> Imiren ad yini i wid yellan ɣer uẓelmaḍ-is : Beɛdet akkin fell-i a kunwi yețwaneɛlen, ṛuḥet ɣer tmes n dayem yețwaheggan i Cciṭan d wid i t- tebɛen. </p> 
 <p><a name="an_25:42">25:42</a> Axaṭer lliɣ lluẓeɣ ur iyi tesseččem ara, ffudeɣ ur iyi-tefkim ara ad sweɣ,  </p> 
 <p><a name="an_25:43">25:43</a> usiɣ-ed d abeṛṛani ur tesṭerḥbem ara yis-i, lliɣ d aɛeryan ur iyi tesselsem ara, lliɣ daɣen d amuḍin, d ameḥbus, ur d-tusim ara aț-țesteqsim fell-i. </p> 
 <p><a name="an_25:44">25:44</a> Imiren ula d nutni a s-d-rren : A Sidi, melmi i k-neẓra telluẓeḍ neɣ teffudeḍ d abeṛṛani neɣ d aɛeryan, d amuḍin neɣ d ameḥbus, ur d-nelhi ara yid-ek ?  </p> 
 <p><a name="an_25:45">25:45</a> Nețța a sen-d-yerr : A wen-iniɣ tideț : yal tikkelt i deg ur texdimem ara ayagi i yiwen seg imecṭuḥen-agi, i nekk iwumi ur t-texdimem ara.  </p> 
 <p><a name="an_25:46">25:46</a> Imiren widak-agi ad kecmen ɣer lɛiqab ur nfennu, ma d iḥeqqiyen ad kecmen ɣer tudert n dayem !  </p> 
 <p><a name="an_26:1">26:1</a> Mi gfukk Sidna Ɛisa imeslayen-agi, yenna i inelmaden-is : -- </p> 
 <p><a name="an_26:2">26:2</a> Mazal sin wussan i lɛid n Tuffɣa, Mmi-s n bunadem ad ițțuzenz iwakken ad ițuṣemmer ɣef wumidag.  </p> 
 <p><a name="an_26:3">26:3</a> Dɣa lmuqedmin imeqqranen akk-d imeqqranen n wegdud nnejmaɛen deg ufrag n Kayef, i gellan d lmuqeddem ameqqran.  </p> 
 <p><a name="an_26:4">26:4</a> Msefhamen ad ṭṭfen Sidna Ɛisa s tḥila iwakken a t-nɣen. </p> 
 <p><a name="an_26:5">26:5</a> Meɛna nnan : « mačči deg ussan-agi n lɛid n Tuffɣa ara t-neḥbes, neɣ m'ulac ad yekker ccwal ger lɣaci .»  </p> 
 <p><a name="an_26:6">26:6</a> Mi gella Sidna Ɛisa di taddart n Bitanya, deg wexxam n Semɛun abeṛsi, </p> 
 <p><a name="an_26:7">26:7</a> ațaya yiwet n tmeṭṭut tewwi-d tabuqalt yețwaxedmen s wedɣaɣ amellal, teččuṛ d leɛṭeṛ ɣlayen. Mi geqqim Sidna Ɛisa ɣer ṭṭabla, tameṭṭut-nni tusa-d tesmar leɛṭeṛ-nni ɣef wuqeṛṛuy-is.  </p> 
 <p><a name="an_26:8">26:8</a> Mi walan ayagi, inelmaden iɣaḍ-iten lḥal, nnan : Iwumi lexsaṛa-yagi ? </p> 
 <p><a name="an_26:9">26:9</a> Lemmer nezzenz leɛṭeṛ-nni, tili yewwi-d ssuma tameqqrant ara nefṛeq i igellilen !  </p> 
 <p><a name="an_26:10">26:10</a> Sidna Ɛisa yeẓran wayen nnan wway-gar-asen, yenna-yasen :  Acuɣeṛ i tesḥeznem tameṭṭut agi ? D ayen yelhan i yi-texdem !  </p> 
 <p><a name="an_26:11">26:11</a> Axaṭer aț-țesɛum daymen igellilen yid-wen, meɛna nekk ur țțiliɣ ara daymen yid-wen. </p> 
 <p><a name="an_26:12">26:12</a> Mi d-tesmar leɛṭeṛ-agi fell-i, d lǧețța-w i thegga i temḍelt-iw. </p> 
 <p><a name="an_26:13">26:13</a> A wen-iniɣ tideț : di yal amkan n ddunit i deg ara ițțubecceṛ lexbaṛ n lxiṛ, a d țmektayen tameṭṭut-agi d wayen texdem.  </p> 
 <p><a name="an_26:14">26:14</a> Imiren Yudas n Qeṛyut, yiwen si tnac-nni inelmaden n Sidna Ɛisa, iṛuḥ ɣer lmuqedmin imeqqranen, </p> 
 <p><a name="an_26:15">26:15</a> yenna yasen : Ma wwiɣ-kkun ɣuṛ-es a t-id teṭṭfem, d acu ara yi-tefkem ? FFkan-as tlatin iṣurdiyen n lfeṭṭa. </p> 
 <p><a name="an_26:16">26:16</a> Seg imiren, Yudas yețɛassa tagniț i deg ara yizmir ad izzenz Sidna Ɛisa.  </p> 
 <p><a name="an_26:17">26:17</a> Ass amezwaru n lɛid n weɣṛum mbla iɣes n temtunt , inelmaden usan-d ɣer Sidna Ɛisa nnan-as : A Sidi, anda tebɣiḍ a k-nheggi imensi n lɛid n Izimer n leslak ?  </p> 
 <p><a name="an_26:18">26:18</a> Yerra-yasen : Ṛuḥet ɣer temdint, ɣer leflani, init-as : yenna-yak Ssid-nneɣ, lweqt-iw yewweḍ-ed, deg uxxam-ik ara sɛeddiɣ lɛid n Izimer n leslak nekk d inelmaden-iw.  </p> 
 <p><a name="an_26:19">26:19</a> Inelmaden-nni xedmen ayen i sen-yenna Sidna Ɛisa, heggan imensi n lɛid n Izimer n leslak.  </p> 
 <p><a name="an_26:20">26:20</a> Mi d-teɣli tmeddit, Sidna Ɛisa yeqqim ad yečč nețța d inelmaden-is. </p> 
 <p><a name="an_26:21">26:21</a> Mi llan tețțen yenna-yasen : A wen-iniɣ tideț : yiwen seg-wen ad iyi-izzenz.  </p> 
 <p><a name="an_26:22">26:22</a> Mi d-yenna ayagi, ikcem-iten akk leḥzen d ameqqran, bdan qqaṛen as yiwen yiwen : Neɣ mačči d nekk a Sidi ?  </p> 
 <p><a name="an_26:23">26:23</a> Yerra-yasen : Win igren afus-is yid-i deg uḍebsi, d nețța ara yi-izzenzen. </p> 
 <p><a name="an_26:24">26:24</a> Ad yedṛu d Mmi-s n bunadem akken yura fell-as ; mmeɛna a nnger n win ara t-izzenzen, lemmer ur d-ilul ara axiṛ-is.  </p> 
 <p><a name="an_26:25">26:25</a> Yudas ara t-ixedɛen yenṭeq yenna-yas :  Eɛni d nekk a Sidi ? SSidna Ɛisa yerra-yas : Atan tenniḍ-t-id !  </p> 
 <p><a name="an_26:26">26:26</a> Mi llan tețțen, Sidna Ɛisa yeddem-d aɣṛum, iḥmed Ṛebbi, icekkeṛ-it, dɣa yebḍa-t, ifṛeq-it i inelmaden-is, yenna-yasen : Eččet aɣṛum-agi : d lǧețța-w.  </p> 
 <p><a name="an_26:27">26:27</a> Iddem-ed daɣen taqbuct, iḥmed Ṛebbi, icekkeṛ-it, yefka-yasen-ț, yenna-yasen : Swet akk si teqbuct-agi, </p> 
 <p><a name="an_26:28">26:28</a> axaṭer ayagi d idammen-iw, d idammen n lemɛahda ara yazzlen ɣef yizumal n lɣaci ɣef ddemma n leɛfu n ddnubat-nsen.  </p> 
 <p><a name="an_26:29">26:29</a> A wen-d-iniɣ : ur țțuɣaleɣ ara ad sweɣ seg waman-agi n tẓurin alamma d ass i deg ara ten-sweɣ d ijdiden yid-wen, di tgelda n Baba Ṛebbi.  </p> 
 <p><a name="an_26:30">26:30</a> Mi ḥemden Sidi Ṛebbi s ccnawi d isefra, ṛuḥen ulin ɣer yiɣil n uzemmur, </p> 
 <p><a name="an_26:31">26:31</a> dinna Sidna Ɛisa yenna-yasen : Iḍ-agi ad a wen-iliɣ akk d sebba n tuccḍa, axaṭer yura di tira iqedsen  ad wwteɣ ameksa, ulli n tqeḍɛit-is  ad rewlent ɣer yal tama .   </p> 
 <p><a name="an_26:32">26:32</a> Meɛna m'ara d-ḥyuɣ si ger lmegtin, ad iyi-tafem zwareɣ ɣer tmurt n Jlili.  </p> 
 <p><a name="an_26:33">26:33</a> Buṭrus yenṭeq, yenna : Ɣas a sen-tiliḍ d sebba n tuccḍa i wiyaḍ akk, nekk d lmuḥal ad ccḍeɣ.  </p> 
 <p><a name="an_26:34">26:34</a> Sidna Ɛisa yenna-yas : A k-iniɣ tideț : deg iḍ-agi uqbel ad iskkuɛ uyaziḍ, ad iyi-tnekṛeḍ tlata iberdan.  </p> 
 <p><a name="an_26:35">26:35</a> Buṭrus yerra-yas : Ɣas lukan d lmut ara mmteɣ yid-ek, d lmuḥal a k-nekkṛeɣ ! UUla d inelmaden nniḍen, nnan-as am nețța.  </p> 
 <p><a name="an_26:36">26:36</a> Imiren Sidna Ɛisa iṛuḥ yid-sen ɣer yiwen wemkan ițțusemman Jitsimani, yenna i inelmaden-is :  Qqimet dagi, nekk ad ṛuḥeɣ ad dɛuɣ ɣer Sidi Ṛebbi.  </p> 
 <p><a name="an_26:37">26:37</a> Yewwi yid-es Buṭrus akk-d sin warraw-nni n Zabadi ; yebda ikcem-it leḥzen d leɣben d ameqqran. </p> 
 <p><a name="an_26:38">26:38</a> Dɣa yenna-yasen : Ḥusseɣ s leḥzen n lmut, qqimet dagi, ɛiwzet yid-i !  </p> 
 <p><a name="an_26:39">26:39</a> Iṛuḥ yeɣli ɣef wudem, ibda yețẓalla yeqqaṛ : A Baba Ṛebbi, ma yella wamek , ssebɛed fell-i leɛtab-agi ! Meɛna akken tebɣiḍ keččini, mačči akken bɣiɣ nekkini.  </p> 
 <p><a name="an_26:40">26:40</a> Yuɣal-ed ɣer inelmaden-is, yufa-ten-id ṭṭsen, yenna i Buṭrus : Ur tezmirem ara aț-țɛiwzem yid-i ula d yiwet n ssaɛa ? </p> 
 <p><a name="an_26:41">26:41</a> Ɛiwzet, ẓallet, iwakken ur tɣellim ara deg ujeṛṛeb ! Axaṭer ṛṛuḥ n wemdan yeǧhed ma d lǧețța ur tezmir ara !  </p> 
 <p><a name="an_26:42">26:42</a> Ibɛed akkin tikkelt tis snat iwakken ad iẓẓal, yenna : A Baba, ma yella ulamek ara tesbeɛdeḍ fell-i leɛtab-agi , ma ilaq ad iyi-d-yaweḍ ! Ad yedṛu lebɣi-k !  </p> 
 <p><a name="an_26:43">26:43</a> Mi d-yuɣal, yufa-ten-id daɣen ṭṭsen, iɣleb-iten nuddam. </p> 
 <p><a name="an_26:44">26:44</a> Yeǧǧa-ten, issebɛed iman-is daɣen fell-asen, yedɛa tikkelt tis tlata am tikkelt-nni tamezwarut.  </p> 
 <p><a name="an_26:45">26:45</a> Mi d-yuɣal ɣer inelmaden-is, yufa-ten-id mazal-iten ṭṭsen ; yenna yasen : Lweqt i deg ara yețwasellem Mmi-s n bunadem ɣer ifassen n yemcumen iwweḍ-ed, kunwi mazal ikkun teṭṭsem ?  </p> 
 <p><a name="an_26:46">26:46</a> Dɣa yenna-yasen : Ekkret fell-awen, ataya iteddu-d win i yi-izzenzen !  </p> 
 <p><a name="an_26:47">26:47</a> Mazal-it ițmeslay mi d-iwweḍ Yudas yellan d yiwen si tnac-nni inelmaden. Ddan-d yid-es aṭas n lɣaci s iɛekkzan d ijenwiyen, ceggɛen ten-id imeqqranen n lmuqedmin akk-d imeqqranen n wegdud. </p> 
 <p><a name="an_26:48">26:48</a> Win akken i t-izzenzen, imsefham yid-sen, yenna-yasen : Win i ɣef ara selmeɣ, d win ara teṭṭfem !  </p> 
 <p><a name="an_26:49">26:49</a> Imiren, iqeṛṛeb ɣer Sidna Ɛisa yenna-yas : Sslam fell-ak a Sidi ! DDɣa isellem fell-as. </p> 
 <p><a name="an_26:50">26:50</a> Sidna Ɛisa yenna-yas : Ay amdakkel , ayen ɣef i d-tusiḍ a t-txedmeḍ, xdem-it ! DDɣa lɣaci-nni qeṛṛben ɣer Sidna Ɛisa ṭṭfen-t. </p> 
 <p><a name="an_26:51">26:51</a> Yiwen seg wid yeddan d Sidna Ɛisa ijbed-ed ajenwi-ines, yewwet aqeddac n lmuqeddem ameqqran, igzem-as-d ameẓẓuɣ-is. </p> 
 <p><a name="an_26:52">26:52</a> Sidna Ɛisa yenṭeq yenna-yas : Err ajenwi-inek ɣer wemkan-is, axaṭer kra win ara irefden ajenwi, ad immet s ujenwi ! </p> 
 <p><a name="an_26:53">26:53</a> Eɛni tɣileḍ ur zmireɣ ara ad ssutreɣ i Baba ad iyi-d-iceggeɛ luluf n lmalayekkat ? </p> 
 <p><a name="an_26:54">26:54</a> Amek ihi ara țwakemlent tira iqedsen i d-ixebbṛen belli ilaq a d-yedṛu wayagi meṛṛa ?  </p> 
 <p><a name="an_26:55">26:55</a> Imiren Sidna Ɛisa yenna i lɣaci : Kull ass țțiliɣ yid-wen di lǧameɛ sselmadeɣ, ur iyi-teṭṭifem ara, ass-agi tusam-d ɣuṛ-i s iɛekkzan d ijenwiyen am akken d bu tmegṛaḍ i lliɣ ! </p> 
 <p><a name="an_26:56">26:56</a> Meɛna annect-agi meṛṛa yedṛa-d iwakken ad țwakemlent tira n lenbiya. IImiren, inelmaden meṛṛa ǧǧan-t, yal yiwen ɣer wanda yerwel.  </p> 
 <p><a name="an_26:57">26:57</a> Wid yeṭṭfen Sidna Ɛisa wwin-t ɣer Kayef, ameqqran n lmuqedmin, anda i nnejmaɛen lɛulama n ccariɛa d imeqqranen n wegdud.  </p> 
 <p><a name="an_26:58">26:58</a> Buṭrus iḍfeṛ-it, ikcem ɣer wefrag n lmuqeddem ameqqran, yeqqim akk-d iɛessasen n lǧameɛ iwakken ad iẓer amek ara tefru temsalt. </p> 
 <p><a name="an_26:59">26:59</a> Imiren lmuqedmin imeqqranen akk-d usqamu n imeqqranen n ccariɛa, țnadin inigan ara icehden fell-as s ẓẓuṛ, iwakken ad ḥekmen fell-as s lmut, </p> 
 <p><a name="an_26:60">26:60</a> meɛna ur s-d-ufin ara sebba, ɣas akken aṭas i d-icehden fell-as.   </p> 
 <p><a name="an_26:61">26:61</a> A ten-aya sin yergazen qeṛṛben d nnan : D wagi i d-yennan : « zemreɣ ad huddeɣ lǧameɛ iqedsen n Sidi Ṛebbi, a s-ɛiwdeɣ lebni di tlata wussan.»   </p> 
 <p><a name="an_26:62">26:62</a> Ameqqran n lmuqedmin ikker, yenna-yas : Ulac ayen ara d-terreḍ ɣef wayen akka i d-ccetkan fell-ak ?  </p> 
 <p><a name="an_26:63">26:63</a> Sidna Ɛisa yessusem. Dɣa ameqqran n lmuqedmin yenna-yas :  Wekkleɣ-ak Sidi Ṛebbi yeddren, m'ur aɣ-d-tenniḍ ma d kečč i d Lmasiḥ, Mmi-s n Sidi Ṛebbi !  </p> 
 <p><a name="an_26:64">26:64</a> Sidna Ɛisa yerra-yas :  Tenniḍ-t-id ! Yerna a wen-iniɣ, ssya d asawen : Mmi-s n bunadem aa d iḥkem ɣɣ er uyeffus n Bab n tezmert, a t-twalim iteddu-d ɣef wusigna seg yigenni.   </p> 
 <p><a name="an_26:65">26:65</a> Imiren ameqqran n lmuqedmin icerreg aqenduṛ-is yenna : Ikfeṛ ! Iwumi neḥwaǧ inigan tura ? Teslam akk d acu i d-yenna ! D acu tennam ?  </p> 
 <p><a name="an_26:66">26:66</a> Rran-as-d : Yuklal lmut !  </p> 
 <p><a name="an_26:67">26:67</a> Dɣa bdan ssusufen-t, țțaken-as tiyitiwin d ibeqqayen, </p> 
 <p><a name="an_26:68">26:68</a> qqaṛen-as : Eɛqel a Lmasiḥ, ini-yaɣ-d anwa i k-iwwten !  </p> 
 <p><a name="an_26:69">26:69</a> Buṭrus yella yeqqim deg ufrag, tqeṛṛeb-ed ɣuṛ-es yiwet n tqeddact tenna-yas : Ula d kečč telliḍ d Ɛisa ajlili !  </p> 
 <p><a name="an_26:70">26:70</a> Meɛna yenkeṛ zdat-sen meṛṛa yenna-yas : Ur ẓriɣ ara d acu tebɣiḍ a d tiniḍ !  </p> 
 <p><a name="an_26:71">26:71</a> Akken ițeddu ɣer tewwurt, twala-t tqeddact nniḍen, tenna i wid akk yellan dinna : Wagi yella akk-d Ɛisa anaṣari !  </p> 
 <p><a name="an_26:72">26:72</a> Inkeṛ daɣen s limin : Qqaṛeɣ-awen ur ssineɣ ara argaz-agi !  </p> 
 <p><a name="an_26:73">26:73</a> Taswiɛt kan, wid yellan dinna qeṛṛben daɣen ɣer Buṭrus nnan-as : Mbla ccekk, kečč telliḍ seg-sen, tameslayt-ik tbeggen-ed belli d ajlili i telliḍ !  </p> 
 <p><a name="an_26:74">26:74</a> Ibda yețgalla irennu : Di leɛmeṛ ur ssineɣ argaz-agi ! IImiren kan yeskkuɛ uyaziḍ. </p> 
 <p><a name="an_26:75">26:75</a> Dɣa Buṭrus immekta-d imeslayen-nni i s-d-yenna Sidna Ɛisa : uqbel ad iskkuɛ uyaziḍ, ad iyi-tnekṛeḍ tlata n tikkal . Yeffeɣ ɣer beṛṛa s nndama tameqqrant, iṭṭerḍeq d imeṭṭawen.  </p> 
 <p><a name="an_27:1">27:1</a> Taṣebḥit, nnejmaɛen-d akk imeqqranen n lmuqedmin akk-d imeqqranen n wegdud, mcawaṛen iwakken ad nɣen Sidna Ɛisa. </p> 
 <p><a name="an_27:2">27:2</a> Dɣa urzen-t, wwin-t i Bilaṭus, yellan d lḥakem n Ṛuman yesteɛemṛen tamurt.  </p> 
 <p><a name="an_27:3">27:3</a> Mi gwala ḥekmen ɣef Sidna Ɛisa, Yudas-nni i t-ixedɛen yendem. DDi nndama-nni-ines tameqqrant, iṛuḥ ɣer imeqqranen n lmuqedmin d imeqqranen n wegdud, yerra-yasen tlatin-nni twiztin n lfeṭṭa, dɣa </p> 
 <p><a name="an_27:4">27:4</a> yenna yasen : Denbeɣ, nekk i wen-d-izzenzen amdan aḥeqqi. RRran-as-d : Ur aɣ-d-tewqiɛ ara lmeɛna, wagi d ccɣel-ik.  </p> 
 <p><a name="an_27:5">27:5</a> Yudas iḍeggeṛ tiwiztin-nni di lǧameɛ iqedsen, imiren iṛuḥ iɛelleq iman-is. </p> 
 <p><a name="an_27:6">27:6</a> Imeqqranen n lmuqedmin leqḍen-d tiwiztin-nni n lfeṭṭa, nnan : Ur yeḥlil ara a tent-nerr ɣer usenduq n lǧameɛ iqedsen, imi d lexlaṣ n temgeṛt.  </p> 
 <p><a name="an_27:7">27:7</a> Mi mcawaṛen d acu ara xedmen s twiztin-nni, msefhamen iwakken ad aɣen iger n yiwen wergaz ixeddmen afexxaṛ, iwakken a t-erren ț-țimeqbeṛt i ibeṛṛaniyen.  </p> 
 <p><a name="an_27:8">27:8</a> Daymi iger-agi ițțusemma ar ass-a : « Iger n idammen ». </p> 
 <p><a name="an_27:9">27:9</a> S wakka i gedṛa wayen i d-yenna nnbi Irmiya : Ddmen-d tlatin n twiztin n lfeṭṭa, d wagi i d azal i s-fkan warraw n wat Isṛail ;   </p> 
 <p><a name="an_27:10">27:10</a> uɣen-d yis iger ɣer win ixeddmen afexxaṛ, akken i yi-d-yumeṛ Sidi Ṛebbi.  </p> 
 <p><a name="an_27:11">27:11</a> Sidna Ɛisa ibedd zdat lḥakem Bilaṭus. Yesteqsa-t lḥakem-nni yenna-yas : D kečč i d agellid n wat Isṛail ? SSidna Ɛisa yerra-yas : Tenniḍ-t-id !   </p> 
 <p><a name="an_27:12">27:12</a> Lmuqedmin imeqqranen akk-d imeqqranen n wegdud sseḍlamen deg-s, ma d nețța ur sen-d-yerri ula d acemma. </p> 
 <p><a name="an_27:13">27:13</a> Bilaṭus yenna i Sidna Ɛisa : Eɛni ur tesliḍ ara ayen akk ɣef i d-țcetkin fell-ak ?  </p> 
 <p><a name="an_27:14">27:14</a> Sidna Ɛisa yessusem, ur as-d yerri ula d yiwen wawal, ayagi d ayen iswehmen lḥakem. </p> 
 <p><a name="an_27:15">27:15</a> Lḥakem yennum di mkul lɛid n Tuffɣa ițserriḥ-ed i yiwen umeḥbus i gextaṛ lɣaci. </p> 
 <p><a name="an_27:16">27:16</a> Yella yakan yiwen umeḥbus mechuṛen, ițțusemman Barabas. </p> 
 <p><a name="an_27:17">27:17</a> Mi d-nnejmaɛen akk lɣaci, Bilaṭus isteqsa-ten : Anwa i tebɣam a s-d-serrḥeɣ : i Barabas neɣ i Ɛisa ițțusemman Lmasiḥ ?  </p> 
 <p><a name="an_27:18">27:18</a> Axaṭer yeẓra s tismin i s-d-wwin Ɛisa. </p> 
 <p><a name="an_27:19">27:19</a> Mi gella Bilaṭus ițcaṛaɛ, tceggeɛ-as-d tmeṭṭut-is, tenna-yas-d : Ur ssekcam ara iman-ik di temsalt n uḥeqqi-agi, axaṭer ass-agi targit urgaɣ fell-as terwi-yi.  </p> 
 <p><a name="an_27:20">27:20</a> Imeqqranen n lmuqedmin akk-d imeqqranen n wegdud sḥeṛcen-d lɣaci iwakken ad ssutren a d-serrḥen i Barabas, ad nɣen Sidna Ɛisa. </p> 
 <p><a name="an_27:21">27:21</a> Lḥakem yenna-yasen : Anwa ger sin-agi iwumi tebɣam ad serrḥeɣ ? RRran-as : I Barabas ! Serreḥ-ed i Barabas !  </p> 
 <p><a name="an_27:22">27:22</a> Bilaṭus yenna-yasen : D acu ara xedmeɣ ihi i Ɛisa ițțusemman Lmasiḥ ? RRran-as-d akk : Ad ițțusemmeṛ ɣef wumidag !  </p> 
 <p><a name="an_27:23">27:23</a> Lḥakem yenna : D acu i gexdem n diri ? RRnan țɛeggiḍen : Semmeṛ-it ɣef wumidag !  </p> 
 <p><a name="an_27:24">27:24</a> Mi gwala lɣaci ugin a s-d ḥessen, yerna ccwal simmal yețțimɣuṛ, Bilaṭus yeddem-ed aman, issared ifassen-is zdat lɣaci, yenna : Nekk ur țekkiɣ ara di tazzla n idammen n uḥeqqi-agi, wagi d ccɣel-nwen.  </p> 
 <p><a name="an_27:25">27:25</a> Lɣaci meṛṛa nnan-as : Tazzla n idammen-is a d-tuɣal ɣef yirawen-nneɣ d yirawen n warraw-nneɣ.  </p> 
 <p><a name="an_27:26">27:26</a> Imiren Bilaṭus iserreḥ-ed i Barabas, jelkḍen Sidna Ɛisa, dɣa yefka lameṛ a t-semmṛen ɣef wumidag.  </p> 
 <p><a name="an_27:27">27:27</a> Iɛsekṛiyen n lḥakem ssufɣen Sidna Ɛisa ɣer ufrag, snejmaɛen-d akk iɛsekṛiyen nniḍen, zzin-as.  </p> 
 <p><a name="an_27:28">27:28</a> Imiren kksen-as llebsa-s, sselsen as abeṛnus azeggaɣ. </p> 
 <p><a name="an_27:29">27:29</a> Zḍan taɛeṣṣabt s isennanen, ssersen-as-ț ɣef wuqeṛṛuy is, fkan-as aɣanim ɣer ufus-is ayeffus. Bdan kennun zdat-es, stehzayen fell-as, qqaṛen-as : Sslam fell-ak ay agellid n wat Isṛail !  </p> 
 <p><a name="an_27:30">27:30</a> Ssusufen-t, kkaten-t s uɣanim ɣer uqeṛṛuy. </p> 
 <p><a name="an_27:31">27:31</a> Mi ɛyan deg ustehzi fell-as, kksen-as abeṛnus-nni, rran-as llebsa-s, wwin-t a t-semmṛen ɣef wumidag.  </p> 
 <p><a name="an_27:32">27:32</a> Mi d-ffɣen, mlalen-d yiwen wergaz n tmurt n Qiṛwan, isem-is semɛun, ḥettmen fell-as ad ibbib amidag n Sidna Ɛisa.  </p> 
 <p><a name="an_27:33">27:33</a> Mi wwḍen ɣer yiwen wemkan ițțusemman Gulguṭa, yeɛni « amkan n uxecxuc », </p> 
 <p><a name="an_27:34">27:34</a> fkan-as ad isew ccṛab i deg yexleḍ yiwen uɛeqqaṛ aṛzagan. Iɛreḍ-it, yugi a t-isew.  </p> 
 <p><a name="an_27:35">27:35</a> Mi t-semmṛen ɣef wumidag, gren tasɣaṛt iwakken ad feṛqen llebsa-ines, s wakka, yedṛa-d wayen yuran di tira iqedsen : feṛqen llebsa-w, gren tasɣart ɣef wuqenduṛ-iw.  </p> 
 <p><a name="an_27:36">27:36</a> Imiren qqimen a t-ɛassen, </p> 
 <p><a name="an_27:37">27:37</a> ɛelqen sennig uqeṛṛuy-is talwiḥt i ɣef uran sebba n lmut-is : Wagi d Ɛisa, agellid n wat Isṛail  </p> 
 <p><a name="an_27:38">27:38</a> Țwasemmṛen yid-es sin imcumen, yiwen ɣer uyeffus-is wayeḍ ɣer uẓelmaḍ-is.  </p> 
 <p><a name="an_27:39">27:39</a> Imsebriden țhuzzun iqeṛṛay-nsen reggmen-t, </p> 
 <p><a name="an_27:40">27:40</a> qqaṛen-as : A win ițhuddun lǧameɛ iqedsen ibennu-t di tlata wussan, sellek iman-ik ma d Mmi-s n Ṛebbi i telliḍ, ers-ed seg umidag-nni.  </p> 
 <p><a name="an_27:41">27:41</a> Imeqqranen n lmuqedmin, lɛulama n ccariɛa akk-d imeqqranen n wegdud, stehzayen fell-as ula d nutni, qqaṛen :   </p> 
 <p><a name="an_27:42">27:42</a> Isellek wiyaḍ ur yezmir ad isellek iman-is ! Ma d agellid n wat Isṛail i gella, a d-yers ihi seg umidag iwakken a namen yis.  </p> 
 <p><a name="an_27:43">27:43</a> Yețkel ɣef Ṛebbi, a t-isellek ihi Ṛebbi ma iḥemmel-it aaxaṭer yenna-d : « Nekk d Mmi-s n Ṛebbi.»  </p> 
 <p><a name="an_27:44">27:44</a> Ula d imcumen-nni ițțusemmṛen yid-es, regmen-t am wiyaḍ.  </p> 
 <p><a name="an_27:45">27:45</a> Deg uzzal si tnac armi d țlata n tmeddit, yeɣli-d ṭṭlam ɣef ddunit meṛṛa. </p> 
 <p><a name="an_27:46">27:46</a> Ɣef leǧwahi n țlata, Sidna Ɛisa iɛeggeḍ s tɛibṛanit, yenna :  Ilya, Ilya, lama sabaqtani ? Yeɛni : Illu-iw, Illu-iw, acuɣeṛ i yi-teǧǧiḍ ?  </p> 
 <p><a name="an_27:47">27:47</a> Kra seg wid iḥedṛen dinna slan d acu i genna, qqaṛen : Issawal i nnbi Ilyas !  </p> 
 <p><a name="an_27:48">27:48</a> Imiren kan yiwen seg-sen yuzzel yeddem-ed ameččim n taḍuṭ isselxes-it di lxell, icudd-it ɣef wuɣanim yefka-yas ad isew.  </p> 
 <p><a name="an_27:49">27:49</a> Wiyaḍ qqaṛen : Anfet-as, a nwali ma yella a d-yas nnbi Ilyas a t-isellek.  </p> 
 <p><a name="an_27:50">27:50</a> Sidna Ɛisa iɛeggeḍ tikkelt nniḍen, dɣa iffeɣ-it Ṛṛuḥ. </p> 
 <p><a name="an_27:51">27:51</a> Imiren kan abeḥnuq n leḥjab yellan di lǧameɛ iqedsen icerreg ɣef sin, seg ixef ufella armi d akessar. Lqaɛa tergagi, idurar ceqqeqen, </p> 
 <p><a name="an_27:52">27:52</a> iẓekwan ldin, aṭas seg iḥeqqiyen yemmuten, ḥyan-d </p> 
 <p><a name="an_27:53">27:53</a> ffɣen-d seg iẓekwan-nsen. Mi d-iḥya Sidna Ɛisa, kecmen ɣer temdint iqedsen, aṭas n lɣaci iwumi i d-ḍehṛen.  </p> 
 <p><a name="an_27:54">27:54</a> Lḥakem yellan d aqeṛṛuy ɣef meyya iɛsekṛiyen akk-d iɛsekṛiyen iɛussen Sidna Ɛisa, tekcem-iten tugdi mi walan tergagi lqaɛa d wayen akk yedṛan, nnan : Ț-țideț, argaz-agi d Mmi-s n Ṛebbi !  </p> 
 <p><a name="an_27:55">27:55</a> Aṭas n tilawin i gellan dinna, țmuqulent-ed si lebɛid. Ț-țiggad-nni i d-yeddan d Sidna Ɛisa si tmurt n Jlili iwakken a s-qedcent. </p> 
 <p><a name="an_27:56">27:56</a> Tella gar-asent Meryem tamagdalit, Meryem yemma-s n Yeɛqub d Yusef akk-d d yemma-s n warraw n Zabadi.  </p> 
 <p><a name="an_27:57">27:57</a> Mi d-tewweḍ tmeddit, yusa-d yiwen umeṛkanti n temdint n Arimati isem-is Yusef, ula d nețța d anelmad n Sidna Ɛisa. </p> 
 <p><a name="an_27:58">27:58</a> Iṛuḥ ɣer Bilaṭus yessuter-as lǧețța n Sidna Ɛisa. Bilaṭus yumeṛ ad a s-ț-id-fken.  </p> 
 <p><a name="an_27:59">27:59</a> Yusef yeddem-ed lǧețța n Sidna Ɛisa, ițțel-iț di lekfen zeddigen, </p> 
 <p><a name="an_27:60">27:60</a> issers-iț deg yiwen uẓekka ajdid, i geɣza deg uzṛu. Yeɣleq tawwurt n uẓekka-nni s weblaḍ ameqqran, dɣa iṛuḥ.  </p> 
 <p><a name="an_27:61">27:61</a> Meryem tamagdalit akk-d Meryem-nni nniḍen, llant dinna qqiment, qublent-ed aẓekka. </p> 
 <p><a name="an_27:62">27:62</a> Azekka nni yellan d ass n uheggi n wass n westeɛfu, imeqqranen n lmuqedmin d ifariziyen ṛuḥen ɣer Bilaṭus </p> 
 <p><a name="an_27:63">27:63</a> nnan-as : A Sidi, necfa ɣef wayen i d-yenna ukeddab-agi asmi yedder, yenna-d : Ad ɛeddin tlata wussan, a d-ḥyuɣ si ger lmegtin .  </p> 
 <p><a name="an_27:64">27:64</a> Ceggeɛ iɛessasen ad ɛassen aẓekka-s alamma d ass wis tlata, iwakken inelmaden-is ur d-țasen ara ad akren lǧețța-s, ad inin i wegdud : « iḥya-d si ger lmegtin. » Neɣ m'ulac lekdeb-agi aneggaru ad yaggar amezwaru.  </p> 
 <p><a name="an_27:65">27:65</a> Bilaṭus yenna-yasen-d : Awit iɛessasen, ṛuḥet ɛasset-ț akken tebɣam.  </p> 
 <p><a name="an_27:66">27:66</a> Ṛuḥen, zemmemen aken ilaq tawwurt n uẓekka-nni s weblaḍ, imiren sbedden taɛessast zdat-es.  </p> 
 <p><a name="an_28:1">28:1</a> Mi gɛedda wass n westeɛfu, taṣebḥit n wass amezwaru n dduṛt, Meryem tamagdalit akk-d meryem-nni nniḍen, ṛuḥent a d-ẓuṛent aẓekka n Sidna Ɛisa.  </p> 
 <p><a name="an_28:2">28:2</a> Deg yiwet n teswiɛt, tergagi lqaɛa ; ataya yiwen n lmelk n Sidi Ṛebbi, iṣubb-ed seg igenni, iqeṛṛeb ɣer uẓekka, iswexxeṛ ablaḍ-nni, yeqqim fell-as. </p> 
 <p><a name="an_28:3">28:3</a> Ițfeǧǧiǧ am lebṛaq, llebsa-s ț-țamellalt am udfel.  </p> 
 <p><a name="an_28:4">28:4</a> Iɛessasen yellan dinna, ikcem-iten lxuf d ameqqran, qquṛen amzun mmuten.  </p> 
 <p><a name="an_28:5">28:5</a> Yenṭeq lmelk ɣer tilawin-nni, yenna-yasent : Kunemti ur țțaggademt ara, axaṭer ẓriɣ ɣef Ɛisa i yețțusemmṛen ɣef wumidag i tețnadimt. </p> 
 <p><a name="an_28:6">28:6</a> Ulac-it dagi, iḥya-d si ger lmegtin akken i t-id-yenna. Asemt-ed aț-țwalimt amkan anda akken yeḍleq !  </p> 
 <p><a name="an_28:7">28:7</a> Azlemt aț-ținimt i inelmaden-is belli yeḥya-d si ger lmegtin. Atan a kkun-yezwir ɣer tmurt n Jlili ; dinna ara t-teẓrem. AAtan nniɣ-akunt-ț-id !  </p> 
 <p><a name="an_28:8">28:8</a> Imiren kan, ṛuḥent seg uẓekka-nni s lfeṛḥ d ameqqran ɣas akken uggadent. Uzzlent ad awint lexbaṛ i inelmaden n Sidna Ɛisa. </p> 
 <p><a name="an_28:9">28:9</a> Ataya Sidna Ɛisa imuger-itent-id, yenna-yasent : Sslam fell-akunt ! QQeṛṛbent, seǧǧdent zdat-es, ssudnent iḍaṛṛen-is. </p> 
 <p><a name="an_28:10">28:10</a> Imiren Sidna Ɛisa yenna-yasent : Ur țțaggademt ! Ṛuḥemt inimt i watmaten-iw ad ṛuḥen ɣer tmurt n Jlili, dinna ara yi-ẓren.  </p> 
 <p><a name="an_28:11">28:11</a> Mi țeddunt deg ubrid, kra seg iɛessasen-nni n uẓekka kecmen ɣer temdint, xebbṛen imeqqranen n lmuqedmin ɣef wayen akk yedṛan.  </p> 
 <p><a name="an_28:12">28:12</a> Mi nnejmaɛen imeqqranen n wegdud akk-d imeqqranen n lmuqedmin, msefhamen ad fken i iɛessasen-nni ssuma tameqqrant n yedrimen, </p> 
 <p><a name="an_28:13">28:13</a> nnan-asen : Init belli d inelmaden-is i d-yusan deg yiḍ, ukren lǧețța-s mi nella neṭṭes. </p> 
 <p><a name="an_28:14">28:14</a> Ma yella yewweḍ lexbaṛ-agi ɣer lḥakem, d nukkni ara t-iqablen iwakken ur awen-ixeddem ula d acemma.  </p> 
 <p><a name="an_28:15">28:15</a> Iɛsekṛiyen-nni ṭṭfen idrimen-nni, xedmen ayen akk i sen-d-nnan lmuqedmin. Tamsalt-agi țɛawaden-ț-id wat Isṛail ar ass-a.  </p> 
 <p><a name="an_28:16">28:16</a> Ḥdac inelmaden-nni ṛuḥen ɣer tmurt n Jlili, ɣer wedrar-nni anda i sen-yenna Sidna Ɛisa ad mlilen. </p> 
 <p><a name="an_28:17">28:17</a> Mi t-walan seǧǧden zdat-es, meɛna kra seg-sen mazal țcukkuten.  </p> 
 <p><a name="an_28:18">28:18</a> Sidna Ɛisa iqeṛṛeb-ed ɣuṛ-sen, yenna-yasen : Tețțunefk-iyi-d tezmert ama di lqaɛa ama deg igenwan.  </p> 
 <p><a name="an_28:19">28:19</a> Ṛuḥet ɣer tmura meṛṛa n ddunit, slemdet imdanen iwakken ad uɣalen d inelmaden-iw, ad țwaɣedṣen s yisem n Baba Ṛebbi, s yisem n Mmi-s akk-d yisem n Ṛṛuḥ iqedsen.  </p> 
 <p><a name="an_28:20">28:20</a> Slemdet-asen ad tebɛen ayen akk i wen-slemdeɣ. MMa d nekk ad iliɣ yid-wen yal ass alamma ț-țaggara n ddunit.  </p>
