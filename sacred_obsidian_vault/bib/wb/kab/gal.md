  <h3 align="center">Kabyle Bible: Galatians</h3> 
 <p><a name="an_1:1">1:1</a> Nekk Bulus, yellan d ṛṛasul, mačči s ufus neɣ s lebɣi n yemdanen, lameɛna s lebɣi n Ɛisa Lmasiḥ akk-d Baba Ṛebbi i t-id isseḥyan si ger lmegtin. </p> 
 <p><a name="an_1:2">1:2</a> Uriɣ tabṛaț-agi nekk d watmaten meṛṛa yellan yid-i, i tejmuyaɛ n watmaten n tmurt n Galasya : </p> 
 <p><a name="an_1:3">1:3</a> ṛṛeḥma d lehna a wen-d-țțunefkent s ɣuṛ Baba-tneɣ Ṛebbi akk-d Sidna Ɛisa Lmasiḥ.  </p> 
 <p><a name="an_1:4">1:4</a> Lmasiḥ isebbel iman-is ɣef ddnubat-nneɣ, iwakken a ɣ-isellek si yir lǧil-agi, s lebɣi n Baba-tneɣ Ṛebbi. </p> 
 <p><a name="an_1:5">1:5</a> I nețța tamanegt si lǧil ɣer lǧil ! Amin !  </p> 
 <p><a name="an_1:6">1:6</a> Wehmeɣ imi twexxṛem s lemɣawla ɣef win i wen-d-issawlen s ṛṛeḥma n Lmasiḥ, iwakken aț-țḍefṛem lexbaṛ n lxiṛ  nniḍen. </p> 
 <p><a name="an_1:7">1:7</a> Ur wen-d-nniɣ ara akka am akken yella lexbaṛ n lxiṛ  nniḍen , lameɛna qqaṛeɣ-awen-d akka, axaṭer llan kra lɣaci i kkun-icewwlen yerna bɣan ad sɛewjen lexbaṛ n lxiṛ  yeɛnan Lmasiḥ. </p> 
 <p><a name="an_1:8">1:8</a> ?as a lukan d nukni, neɣ d lmelk ara d-yasen seg igenni a wen-d ibbecceṛ lexbaṛ n lxiṛ yemxalafen ɣef win i wen-d-nbecceṛ si tazwara, atan ad yețwanɛel !  </p> 
 <p><a name="an_1:9">1:9</a> Nenna-yawen-t-id si zik, a s-d ɛiwdeɣ tura : yețwanɛel win ara wen-d ibeccṛen lexbaṛ n lxiṛ  yemxalafen ɣef win i tqeblem !  </p> 
 <p><a name="an_1:10">1:10</a> Ihi tura, eɛni d lemziya n yemdanen i țnadiɣ ? Xaṭi ! D lemziya n Sidi Ṛebbi i țnadiɣ. Neɣ eɛni țnadiɣ ad ɛeǧbeɣ i yemdanen ? Lemmer i mazal țnadiɣ ad ɛeǧbeɣ i yemdanen, tili ur țțilliɣ ara d aqeddac n Lmasiḥ.  </p> 
 <p><a name="an_1:11">1:11</a> A wen-d-iniɣ ay atmaten belli lexbaṛ-agi n lxiṛ  i wen-d-beccṛeɣ mačči s ɣuṛ wemdan i d-yekka ; </p> 
 <p><a name="an_1:12">1:12</a> axaṭer, nekk s yiman-iw mačči d amdan i yi-t-id-issawḍen neɣ i yi t-islemden, lameɛna d Sidna Ɛisa Lmasiḥ i yi-t-id-iweḥḥan.  </p> 
 <p><a name="an_1:13">1:13</a> Teslam ɣef tikli-inu n zik asmi lliɣ ttabaɛeɣ ddin n wat Isṛail ; teslam amek i qehṛeɣ tajmaɛt n imasiḥiyen d wamek bɣiɣ a ten-snegreɣ. </p> 
 <p><a name="an_1:14">1:14</a> Asmi lliɣ ḍefṛeɣ ddin d leɛwayed n wat Isṛail, ifeɣ akk tizyiwin-iw akk-d wid n laṣel-iw, axaṭer ṭṭfeɣ di leɛwayed n lejdud-iw akken ilaq.  </p> 
 <p><a name="an_1:15">1:15</a> Lameɛna s ṛṛeḥma-s, Sidi Ṛebbi ixtaṛ-iyi-d uqbel a d-laleɣ yerna yessawel-iyi-d iwakken ad qedceɣ fell-as.  </p> 
 <p><a name="an_1:16">1:16</a> Asmi i yi-d-ibeggen Sidi Ṛebbi Mmi-s iwakken ad beccṛeɣ lexbaṛ n lxiṛ  i t-iɛnan ger wid ur nelli ara n wat Isṛail, ur cawṛeɣ ula d yiwen ; </p> 
 <p><a name="an_1:17">1:17</a> yerna ur uliɣ ara ɣer temdint Lquds ɣer wid yellan d ṛṛusul uqbel-iw, lameɛna ṛuḥeɣ ɣer tmurt n waɛṛaben ; syenna uɣaleɣ-ed daɣen ɣer temdint n Dimacq.  </p> 
 <p><a name="an_1:18">1:18</a> Mi ɛeddan tlata iseggasen, uliɣ ɣer temdint Lquds iwakken ad issineɣ Buṭrus, qqimeɣ ɣuṛ-es xemseṭṭac wussan. </p> 
 <p><a name="an_1:19">1:19</a> Lameɛna ur ẓriɣ ula d yiwen nniḍen si ṛṛusul, anagar Yeɛqub, gma-s n Ssid-nneɣ.  </p> 
 <p><a name="an_1:20">1:20</a> Ayen i wen-d-uriɣ akka ț-țideț, zdat Ṛebbi ur skiddibeɣ ara. </p> 
 <p><a name="an_1:21">1:21</a> Syenna, ṛuḥeɣ ɣer tmura n Surya d Silisya ; </p> 
 <p><a name="an_1:22">1:22</a> kra wayen kkiɣ dinna, imasiḥiyen n tejmuyaɛ n tmurt n Yahuda leɛmeṛ ẓrin udem-iw. </p> 
 <p><a name="an_1:23">1:23</a> Slan kan qqaṛen fell-i : win akken yellan ițqehhiṛ deg-nneɣ, atan tura ițbecciṛ liman i gebɣa zik-nni a t-issenger. </p> 
 <p><a name="an_1:24">1:24</a> Dɣa țḥemmiden Ṛebbi ɣef ddemma-w.  </p> 
 <p><a name="an_2:1">2:1</a> Mi ɛeddan ṛbeɛṭac iseggasen, ɛawdeɣ uliɣ ɣer temdint n Lquds akk-d Barnabas, wwiɣ daɣen yid-i gma-tneɣ Tit. </p> 
 <p><a name="an_2:2">2:2</a> Uliɣ, axaṭer d Sidi Ṛebbi i yi-t-id yumṛen deg uweḥḥi. Sfehmeɣ-asen lexbaṛ n lxiṛ  i țbecciṛeɣ ger wid ur nelli ara n wat Isṛail ; mliɣ-asen-t weḥḥed-sen i yimeqqranen n tejmaɛt iwakken ayen xedmeɣ armi d ass-a ur ițṛuḥ ara baṭel.  </p> 
 <p><a name="an_2:3">2:3</a> Tit, aṛfiq-iw yellan d ayunani, yiwen ur iḥettem fell-as ad iḍheṛ, </p> 
 <p><a name="an_2:4">2:4</a> ɣas akken llan yir watmaten i d-ikecmen gar-aneɣ s tuffra, bɣan a t-ḥeṛsen ad iḍheṛ. Kecmen-d gar-aneɣ iwakken ad walin tilelli i nesɛa di tikli-nneɣ akk-d Ɛisa Lmasiḥ, bɣan a ɣ-rren d aklan-nsen.  </p> 
 <p><a name="an_2:5">2:5</a> Ula d yiwet n tikkelt ur nekni zdat-sen, iwakken tideț yellan di lexbaṛ n lxiṛ  aț-țeqqim gar-awen.  </p> 
 <p><a name="an_2:6">2:6</a> Ma d wid yesɛan lqima gar-awen, [ ur iyi-tewqiɛ ara lmeɛna d acu i llan zik, axaṭer Sidi Ṛebbi ur ixeddem ara lxilaf ger yemdanen ] ; wid-nni yesɛan lqima ur ḥețțmen ara fell-i ad xedmeɣ ayen nniḍen.  </p> 
 <p><a name="an_2:7">2:7</a> Lameɛna walan belli Sidi Ṛebbi iwekkel-iyi iwakken ad beccṛeɣ lexbaṛ n lxiṛ  i wid ur nelli ara n wat Isṛail, am akken i gceggeɛ Buṭrus ad ibecceṛ lexbaṛ n lxiṛ  i wat Isṛail.  </p> 
 <p><a name="an_2:8">2:8</a> Axaṭer, Sidi Ṛebbi i gerran Buṭrus d ṛṛasul n wid yellan n wat Isṛail, yerra-yi ula d nekk d ṛṛasul n wid ur nelli ara n wat Isṛail.  </p> 
 <p><a name="an_2:9">2:9</a> Yeɛqub, Buṭrus akk-d Yuḥenna yețwaḥesben d imeqqranen n tejmaɛt, setɛeṛfen belli d Sidi Ṛebbi i yi-d ifkan ccɣel-agi, dɣa selmen fell-aneɣ afus deg ufus iwakken a ɣ-d-beggnen belli qeblen ayagi. Imiren nemsefham : nutni ad ṛuḥen ad beccṛen i wat Isṛail, ma d nekk akk-d Barnabas, a nṛuḥ a nbecceṛ i wid ur nelli ara n wat Isṛail ; </p> 
 <p><a name="an_2:10">2:10</a> acu kan, weṣṣan-aɣ iwakken ur ntețțu ara igellilen n tejmuyaɛ-nsen yellan di temdint n Lquds, ayagi ur stehzaɣ ara deg-s, xedmeɣ-t seg-wul.  </p> 
 <p><a name="an_2:11">2:11</a> Asmi i d-yusa Buṭrus ɣer temdint n Antyuc, qubleɣ-t ɣer wudem-is zdat lɣaci, axaṭer yeɣleḍ.  </p> 
 <p><a name="an_2:12">2:12</a> Uqbel a d-asen kra n yemdanen i d-iceggeɛ Yeɛqub, yella itețț akk-d watmaten ur nelli ara n wat Isṛail, lameɛna akken kan i d-wwḍen, iḥbes lmakla yid-sen iwexxeṛ fell-asen, axaṭer yuggad wid iṭṭfen di lɛaddat n wat Isṛail.  </p> 
 <p><a name="an_2:13">2:13</a> Atmaten nniḍen n wat Isṛail xedmen am nețța, wexxṛen ula d nutni, yuɣal armi ula d Barnabas iɛuned-iten.  </p> 
 <p><a name="an_2:14">2:14</a> Mi walaɣ ur teddun ara akken ilaq am akken i d-yeqqaṛ lexbaṛ n lxiṛ , nniɣ i Buṭrus zdat lɣaci meṛṛa : ma yella kečč yellan seg wat Isṛail tețɛiceḍ am wid ur nelli ara n wat Isṛail, amek ara tḥeṛṣeḍ wid ur nelli ara n wat Isṛail ad ɛicen am wat Isṛail ?  </p> 
 <p><a name="an_2:15">2:15</a> Nukni ur nelli ara d imednuben am leǧnas nniḍen, laṣel-nneɣ seg wat Isṛail.  </p> 
 <p><a name="an_2:16">2:16</a> Lameɛna neẓra belli amdan yețțuɣal d aḥeqqi zdat Sidi Ṛebbi s liman-ines di Ɛisa Lmasiḥ, mačči imi gxeddem ayen i d-teqqaṛ ccariɛa ; daymi ula d nukni numen s Ɛisa Lmasiḥ, iwakken a nuɣal d iḥeqqiyen zdat Sidi Ṛebbi s liman di Ɛisa Lmasiḥ, mačči imi nxeddem ayen i d-teqqaṛ ccariɛa n Musa. Axaṭer ula d yiwen ur izmir ad yuɣal d aḥeqqi zdat Ṛebbi imi i gettabaɛ ccariɛa n Musa.  </p> 
 <p><a name="an_2:17">2:17</a> Nukni nețnadi a nuɣal d iḥeqqiyen s liman di Lmasiḥ ; ma nufa iman-nneɣ d imednuben ula d nukni am leǧnas nniḍen, eɛni d Lmasiḥ i ɣ-ițțawin ɣer ddnub ? Xaṭi !  </p> 
 <p><a name="an_2:18">2:18</a> Axaṭer ma uɣaleɣ ɣer ccariɛa nni i ǧǧiɣ, ihi rriɣ iman-iw d amɛaṣi ;  </p> 
 <p><a name="an_2:19">2:19</a> ɣef wayen yeɛnan ccariɛa aql-iyi mmuteɣ, yerna d nețțat i yi-wwin ɣer lmut ; ayagi yedṛa-d iwakken ad sɛuɣ tudert di Sidi Ṛebbi ; aql-iyi țwasemmṛeɣ ɣef wumidag akk-d Lmasiḥ.  </p> 
 <p><a name="an_2:20">2:20</a> Tura mačči d nekk i gețțidiren, d Lmasiḥ i gețțidiren deg-i ; axaṭer tudert-iw di ddunit, țɛiceɣ-ț s liman di Mmi-s n Sidi Ṛebbi i yi-ḥemmlen yerna isebbel iman-is fell-i.  </p> 
 <p><a name="an_2:21">2:21</a> Ur sseɣlayeɣ ara lqima n ṛṛeḥma n Sidi Ṛebbi, axaṭer ma nezmer a nuɣal d iḥeqqiyen zdat Ṛebbi s ccariɛa n Musa, ihi Lmasiḥ yemmut ɣef wulac !  </p> 
 <p><a name="an_3:1">3:1</a> Ay at Galasya ur nessin leṣlaḥ nsen ! Anwa i kkun-iseḥḥren kunwi iwumi i d-nessefhem akken ilaq lmut n Ɛisa Lmasiḥ ɣef wumidag ?  </p> 
 <p><a name="an_3:2">3:2</a> ?ef wayagi kan i bɣiɣ a kkun steqsiɣ : amek i wen-d-ițțunefk Ṛṛuḥ iqedsen, imi i txeddmem ayen i d-teqqaṛ ccariɛa neɣ imi teslam yerna tumnem s lexbaṛ n lxiṛ  n Lmasiḥ ?  </p> 
 <p><a name="an_3:3">3:3</a> Eɛni ixuṣṣ-ikkun leɛqel ? Amek, di tazwara tețțeklem ɣef Ṛṛuḥ iqedsen, tura tebɣam aț-țețțeklem ɣef yiman-nwen ? </p> 
 <p><a name="an_3:4">3:4</a> Eɛni ayen akk i tenɛețțabem iṛuḥ baṭel ? Ur ḥsiɣ ara iṛuḥ baṭel !  </p> 
 <p><a name="an_3:5">3:5</a> Win i wen-d-ifkan Ṛṛuḥ iqedsen, ixeddmen lbeṛhanat gar-awen, eɛni ixeddem-iten imi i tettabaɛem ccariɛa neɣ imi tumnem s lexbaṛ n lxiṛ  iwumi teslam ?  </p> 
 <p><a name="an_3:6">3:6</a> Akken yura :  Ibṛahim yumen s Ṛebbi, yețkel fell-as, daymi i t-iḥseb Sidi Ṛebbi d aḥeqqi.  </p> 
 <p><a name="an_3:7">3:7</a> Ilaq aț-țfehmem ihi, wid iteddun s liman, d nutni i d arraw n tideț n Sidna Ibṛahim.  </p> 
 <p><a name="an_3:8">3:8</a> Si zik i d-ixebbeṛ Sidi Ṛebbi di tira iqedsen belli ad yerr leǧnas ur nelli ara n wat Isṛail d iḥeqqiyen ɣef ddemma n liman nsen, mi genna i Sidna Ibṛahim lexbaṛ-agi n lxiṛ  : Leǧnas meṛṛa n ddunit ad țțubarken yis-ek ;  </p> 
 <p><a name="an_3:9">3:9</a> ihi kra n win yumnen s Ṛebbi akken yumen yis Ibṛahim, ad ițțubarek am nețța.  </p> 
 <p><a name="an_3:10">3:10</a> Axaṭer wid yettabaɛen ccariɛa iwakken ad țwaqeblen ɣer Ṛebbi, deɛwessu tezga fell-asen imi yura : ad ițwanɛel kra n win yettabaɛen ccariɛa, ur nxeddem ara ayen akk yuran deg-s.  </p> 
 <p><a name="an_3:11">3:11</a> Ula d yiwen ur izmir ad yuɣal d aḥeqqi zdat Ṛebbi s ccariɛa, ayagi d ayen ibanen imi yura : Aḥeqqi ad yidir s liman.   </p> 
 <p><a name="an_3:12">3:12</a> Abrid n ccariɛa d webrid n liman mxalafen, imi yura : d win ara ixedmen ayen akk i d-teqqaṛ ccariɛa ara yesɛun tudert n dayem.  </p> 
 <p><a name="an_3:13">3:13</a> Lmasiḥ icufeɛ-aɣ-d si deɛwessu n ccariɛa, d nețța i gețwaneɛlen deg umkan-nneɣ ; axaṭer yura :  ițwanɛel kra n win ițwaɛelqen ɣer ttejṛa.  </p> 
 <p><a name="an_3:14">3:14</a> Ayagi yedṛa-d iwakken s Ɛisa Lmasiḥ, leǧnas ur nelli ara n wat Isṛail ad weṛten lbaṛaka i gefka Sidi Ṛebbi i Ibṛahim ; s liman ara ɣ-d ițțunefk Ṛṛuḥ iqedsen i ɣ-iwɛed Sidi Ṛebbi.  </p> 
 <p><a name="an_3:15">3:15</a> Ay atmaten, a wen-d-fkeɣ lemtel yellan di lɛaddat-nneɣ : m'ara yexdem yiwen leɛqed akken ilaq, yiwen ur izmir a t-inkeṛ neɣ a s-yernu kra.  </p> 
 <p><a name="an_3:16">3:16</a> Ihi, Sidi Ṛebbi yefka lemɛahda i Ibṛahim akk-d win ara d-ilalen seg-s, ( ur d-yenni ara : i wid ara d-ilalen seg-s am akken aṭas i gellan ). Di tira iqedsen yura :  i win ara d-ilalen seg-k ; win i ɣef yura wayagi, d Lmasiḥ.  </p> 
 <p><a name="an_3:17">3:17</a> Atah wayen bɣiɣ a d-iniɣ : di tazwara Sidi Ṛebbi yesbedd leɛqed akken ilaq, ccariɛa i d-yusan ṛebɛa meyya utlatin iseggasen deffir-es, ur tezmir ara aț-țemḥu leɛqed-agi, neɣ m'ulac aț-țeɣli lqima n lemɛahda n Sidi Ṛebbi.  </p> 
 <p><a name="an_3:18">3:18</a> Lemmer si ccariɛa ara newṛet leslak, acuɣeṛ lemɛahda ? Axaṭer s lemɛahda-agi i d-ibeggen Sidi Ṛebbi ṛṛeḥma-s i Ibṛahim.  </p> 
 <p><a name="an_3:19">3:19</a> Acuɣeṛ ihi i d-tețțunefk ccariɛa ? Tețțunefk-ed iwakken a d-tesbeggen belli nɛuṣa lumuṛ n Sidi Ṛebbi, alamma yusa-d win akken ara d ilalen si dderya n Ibṛahim iwumi d tețțunefk lemɛahda. D lmalayekkat i d-yewwin ccariɛa ɣer ufus n Sidna Musa iwakken a ț-id-issiweḍ i yemdanen. </p> 
 <p><a name="an_3:20">3:20</a> Sidi Ṛebbi iceggeɛ-ed amdan iwakken a d-yessiweḍ ccariɛa i wiyaḍ, meɛna ur iḥwaǧ ara amdan iwakken ad isbedd leɛqed yid-sen. </p> 
 <p><a name="an_3:21">3:21</a> Eɛni ccariɛa txulef lemɛahdat n Sidi Ṛebbi ? Xaṭi ! Lemmer ccariɛa tezmer a d-tefk tudert i wemdan, tili yezmer wemdan ad yuɣal d aḥeqqi zdat Ṛebbi s ccariɛa.  </p> 
 <p><a name="an_3:22">3:22</a> Lameɛna tira iqedsen nnant-ed : ddunit meṛṛa tella seddaw tezmert n ddnub ; iwacu ? Iwakken lemɛahda n Sidi Ṛebbi aț-țețțunefk i yemdanen meṛṛa ɣef ddemma n liman-nsen di Ɛisa Lmasiḥ.  </p> 
 <p><a name="an_3:23">3:23</a> Uqbel a d-yas lweqt i deg ara namen s Lmasiḥ, nella seddaw leḥkum n ccariɛa, armi d ass i deg i d-yedheṛ webrid n liman.  </p> 
 <p><a name="an_3:24">3:24</a> S wakka, nețțuṛebba-d s ccariɛa armi d asmi d-yusa Lmasiḥ iwakken a ɣ-yerr d iḥeqqiyen zdat Ṛebbi s liman.  </p> 
 <p><a name="an_3:25">3:25</a> Tura yusa-d lweqt i deg ara namen s Lmasiḥ, ifuk fell-aneɣ leḥkum n ccariɛa. </p> 
 <p><a name="an_3:26">3:26</a> Axaṭer tuɣalem akk d arraw n Sidi Ṛebbi s liman di Ɛisa Lmasiḥ. </p> 
 <p><a name="an_3:27">3:27</a> Kunwi akk yețwaɣeḍsen s yisem n Lmasiḥ, tețɛicim tudert tajḍiṭ yellan di Lmasiḥ.  </p> 
 <p><a name="an_3:28">3:28</a> Tura ulac lxilaf ger wat Isṛail d iyunaniyen, ger waklan d iḥeṛṛiyen, ger yergazen ț-țilawin, axaṭer kunwi akk tuɣalem d yiwen di Ɛisa Lmasiḥ. </p> 
 <p><a name="an_3:29">3:29</a> Yerna ma tellam d ayla n Lmasiḥ, aql-ikkun si dderya n Ibṛahim, wid ara iweṛten akken i t-id-iwɛed Sidi Ṛebbi.  </p> 
 <p><a name="an_4:1">4:1</a> Atah wayen i bɣiɣ a t-id-iniɣ : qqaṛeɣ-awen skud win ara iweṛten mazal-it meẓẓi, ur yemxalaf ara nețța d win yellan d akli, ɣas akken d nețța ara yuɣalen d bab n kullec ; </p> 
 <p><a name="an_4:2">4:2</a> lameɛna ilaq-as ad yaɣ awal i wid ițelhayen yid-es, iseddayen lecɣal-is, alamma yessaweḍ leɛmeṛ akken i d-iweṣṣa baba-s.  </p> 
 <p><a name="an_4:3">4:3</a> Akken daɣen ula d nukni asmi nella meẓẓiyit, nɛac seddaw leḥkum n leqwanen n ddunit, </p> 
 <p><a name="an_4:4">4:4</a> meɛna asmi i d-yewweḍ lweqt i ɣ-iweɛɛed Sidi Ṛebbi, iceggeɛ-ed Mmi-s i d-ilulen si tmeṭṭut, iɛac seddaw leḥkum n ccariɛa, </p> 
 <p><a name="an_4:5">4:5</a> iwakken a d-ifdu wid yellan seddaw leḥkum n ccariɛa, iwakken a nuɣal d arraw-is.  </p> 
 <p><a name="an_4:6">4:6</a> Tura imi nuɣal d arraw n Sidi Ṛebbi, ifka-yaɣ-d ɣer wulawen-nneɣ Ṛṛuḥ n Mmi-s s wayes i nessawal i Ṛebbi : « Abba » ! Yeɛni : a Baba !  </p> 
 <p><a name="an_4:7">4:7</a> S wakka, ikkes fell-awen uzaglu n tkelwa, tuɣalem seg warraw-is ; ma tellam seg warraw-is, d kunwi ara iweṛten ayen i wen-iwɛed Sidi Ṛebbi s ṛṛeḥma-ines.  </p> 
 <p><a name="an_4:8">4:8</a> Zik-nni ur tessinem ara Ṛebbi yerna tețɛebbidem lemqamat d ssadaț deg umkan n Ṛebbi. </p> 
 <p><a name="an_4:9">4:9</a> Lameɛna tura mi tessnem Ṛebbi, yerna nețța iḥseb ikkun d arraw-is, amek armi tebɣam aț-țuɣalem ɣer lɛaddat n zik ur nesɛi lmeɛna iwakken aț-țilim seddaw uzaglu-nsent ?  </p> 
 <p><a name="an_4:10">4:10</a> Tețțakem lqima i leɛyudat n kra wussan, kra wagguren, kra lewqat d kra yiseggasen ! </p> 
 <p><a name="an_4:11">4:11</a> Uggadeɣ ayen i nɛețțabeɣ akk fell-awen iṛuḥ baṭel. </p> 
 <p><a name="an_4:12">4:12</a> Ay atmaten di leɛnaya-nwen, uɣalet am nekkini, imi ula d nekk lliɣ am kunwi. Ur iyi-texdimem acemma n diri.  </p> 
 <p><a name="an_4:13">4:13</a> Ur tecfim ara belli d lehlak i yi-ṭṭfen ɣuṛ-wen mi wen-beccṛeɣ lexbaṛ n lxiṛ  ass amezwaru. </p> 
 <p><a name="an_4:14">4:14</a> ?as akken lehlak-iw yeɛfen, ur teɛyim deg-i, ur iyi-teḥqiṛem, lameɛna telham yis-i am akken d lmelk n Sidi Ṛebbi neɣ d Ɛisa Lmasiḥ s yiman-is i lliɣ.  </p> 
 <p><a name="an_4:15">4:15</a> Anda-t lfeṛḥ-nni-nwen n zik ? Acu i gedṛan yid-wen ? Zemreɣ ad cehdeɣ fell-awen belli, lemmer d ayen iwumi tzemrem a t-txedmem, tili tekksem-d allen-nwen iwakken a yi-tent-id-tefkem.  </p> 
 <p><a name="an_4:16">4:16</a> Eɛni uɣaleɣ awen tura d aɛdaw imi i wen-d qqaṛeɣ tideț ?  </p> 
 <p><a name="an_4:17">4:17</a> Imdanen-agi țbegginen-awen-d belli ḥemmlen-kkun, lameɛna nutni ččuṛen ț-țiḥila ; bɣan kan a kkun sbeɛden fell-i iwakken a ten-tettebɛem nutni. </p> 
 <p><a name="an_4:18">4:18</a> Yelha ma ixeddem yiwen lxiṛ, mačči kan m'ara iliɣ gar-awen lameɛna di mkul lweqt.  </p> 
 <p><a name="an_4:19">4:19</a> Ay arraw-iw, aql-i nɛețțabeɣ tura daɣen ɣef ddemma-nwen am tmeṭṭut yeṭṭfen addud, alamma iban-ed Lmasiḥ deg-wen.  </p> 
 <p><a name="an_4:20">4:20</a> Acḥal mennaɣ ad iliɣ gar-awen di teswiɛt-agi i wakken a wen-d mmeslayeɣ akken nniḍen, axaṭer aql-iyi tḥeyyṛeɣ ɣef ddemma-nwen.  </p> 
 <p><a name="an_4:21">4:21</a> Init-iyi-d, kunwi yebɣan aț-țilim seddaw leḥkum n ccariɛa, eɛni ur teslim ara i wayen i d-teqqaṛ ccariɛa ?  </p> 
 <p><a name="an_4:22">4:22</a> Atah wayen i d-teqqaṛ : Sidna Ibṛahim yesɛa sin warraw-is, yiwen seg Haǧiṛa yellan ț-țaklit, wayeḍ seg Saṛa yellan ț-țaḥeṛṛit. </p> 
 <p><a name="an_4:23">4:23</a> Lameɛna mmi-s n taklit-nni, ilul-ed s lebɣi n wemdan, ma d mmi-s n tḥeṛṛit-nni, ilul-ed ɣef ddemma n lemɛahda n Sidi Ṛebbi.  </p> 
 <p><a name="an_4:24">4:24</a> Taqsiṭ-agi tesɛa lmeɛna nniḍen : snat n tilawin-agi țusemmant d sin leɛqud : yiwen n leɛqed d win n wedrar n Sinay i d-ițaǧǧan aklan, yeɛni : d Haǧiṛa.  </p> 
 <p><a name="an_4:25">4:25</a> Haǧiṛa, d adrar n Sinay yellan di tmurt n waɛṛaben, tețmettil-ed tamdint n Lquds n tura, i gellan ț-țaklit s nețțat s warraw-is. </p> 
 <p><a name="an_4:26">4:26</a> Lameɛna tamdint n Lquds yellan deg igenwan ț-țaḥeṛṛit, d nețțat i d yemma-tneɣ.  </p> 
 <p><a name="an_4:27">4:27</a> Axaṭer yura :  Efṛeḥ kemm a tiɛiqeṛt  ur nețțarew ara !    Ɛeggeḍ s lfeṛḥ kemm  ur nessin lewjaɛ n tarrawt !   Axaṭer tameṭṭut yețwaḥeqṛen, aț-țes ɛu  dderya  akteṛ n tin yesɛan argaz-is .  </p> 
 <p><a name="an_4:28">4:28</a> Ma d kunwi ay atmaten, d arraw n lemɛahda i tellam am Isḥaq.  </p> 
 <p><a name="an_4:29">4:29</a> Zik-nni, win i d-ilulen s lebɣi n wemdan ițqehhiṛ win i d-ilulen s lebɣi n Ṛṛuḥ n Ṛebbi, akka i gella lḥal ar ass-a.  </p> 
 <p><a name="an_4:30">4:30</a> Lameɛna d acu yuran di tira iqedsen ?  Enfu taklit d mmi-s ;axaṭer mmi-s n taklit ur ilaq ara ad iwṛet akk-d mmi-s n tmeṭṭut taḥeṛṛit .  </p> 
 <p><a name="an_4:31">4:31</a> Akka ay atmaten, ur nelli ara d arraw n taklit lameɛna d arraw n tmeṭṭut taḥeṛṛit.  </p> 
 <p><a name="an_5:1">5:1</a> Ɛisa Lmasiḥ isellek-aɣ iwakken a nesɛu tilleli n tideț. Ihi ḥerzet tilelli-agi, ḥadret aț-țuɣalem d aklan ! </p> 
 <p><a name="an_5:2">5:2</a> ?esset-iyi-d ! Nekk Bulus, nniɣ awen-d : ma yella tḍehṛem, Lmasiḥ ur kkun-infiɛ ara.  </p> 
 <p><a name="an_5:3">5:3</a> Yerna ad ɛawdeɣ a d-iniɣ i yal argaz i gḍehṛen, belli ilzem ad ixdem ayen akk i d-tenna ccariɛa.  </p> 
 <p><a name="an_5:4">5:4</a> Kunwi yețnadin aț-țuɣalem d iḥeqqiyen zdat Ṛebbi s ccariɛa, ur d-yeqqim wacemma gar-awen d Lmasiḥ, tețwaḥeṛmem si ṛṛeḥma n Sidi Ṛebbi. </p> 
 <p><a name="an_5:5">5:5</a> Ma d nukni, nessaram a ɣ-yerr Sidi Ṛebbi d iḥeqqiyen zdat-es ; ayagi nețṛaǧu-t s Ṛṛuḥ iqedsen akk-d liman.  </p> 
 <p><a name="an_5:6">5:6</a> Axaṭer, di tikli-nneɣ akk-d Ɛisa Lmasiḥ, neḍheṛ neɣ ur neḍhiṛ ara, ayagi ur yesɛi ara azal ; ayen yesɛan azal, d liman i d-țbegginen lecɣal i nxeddem s leḥmala.  </p> 
 <p><a name="an_5:7">5:7</a> Di tazwara tewwim abrid ilaqen, anwa i kkun-issufɣen i webrid n tideț ? </p> 
 <p><a name="an_5:8">5:8</a> Ṛṛay-agi ur d-yekki ara s ɣuṛ Sidi Ṛebbi i wen-d-issawlen. </p> 
 <p><a name="an_5:9">5:9</a> Akken qqaṛen : « Ciṭṭuḥ n temtunt ( iɣes n temtunt ) yessalay akk arukti.»  </p> 
 <p><a name="an_5:10">5:10</a> Lameɛna Sidi Ṛebbi ihedden ul-iw fell-awen, țekleɣ belli ur tețxemmimem ara akken nniḍen. Lameɛna win i kkun-icewwlen, akken ibɣu yili, a t-iɛaqeb Sidi Ṛebbi.  </p> 
 <p><a name="an_5:11">5:11</a> Lemmer mazal nehhuɣ ar tura ɣef ṭṭhaṛa, iwacu i yi-mazal țwaqehṛeɣ ? Tili lexbaṛ i nețbecciṛ ɣef Lmasiḥ ițwasemmṛen ɣef wumidag, ur yețțili d ugur ula i yiwen. </p> 
 <p><a name="an_5:12">5:12</a> Wid-nni i kkun-icewwlen ɣef ṭṭhaṛa ad ṛuḥen ihi ad ɛegben iman-nsen.  </p> 
 <p><a name="an_5:13">5:13</a> Ay atmaten, Sidi Ṛebbi ifka yawen tilelli, ur d-țafet ara sebba s wayes ara txedmem s tlelli-agi lebɣi n tnefsit-nwen ; lameɛna beggnet-ed lmaḥibba yellan gar-awen.  </p> 
 <p><a name="an_5:14">5:14</a> Axaṭer lameṛ i gesdukklen meṛṛa ccariɛa d wagi :  ḥemmel lɣiṛ-ik am yiman-ik .  </p> 
 <p><a name="an_5:15">5:15</a> Ma yella tețțemkerracem am lewḥuc yerna wa itețț wa, ḥadret iman-nwen neɣ m'ulac yiwen ad issenger wayeḍ.  </p> 
 <p><a name="an_5:16">5:16</a> Ddut ihi s lebɣi n Ṛṛuḥ iqedsen, iwakken ur txeddmem ara lebɣi n tnefsit-nwen. </p> 
 <p><a name="an_5:17">5:17</a> Axaṭer lebɣi n tnefsit ixulef lebɣi n Ṛṛuḥ iqedsen, akken daɣen lebɣi n Ṛṛuḥ iqedsen ixulef lebɣi n tnefsit ; d iɛdawen wway gar-asen, daymi ur tezmirem ara aț-țxedmem akken tebɣam.  </p> 
 <p><a name="an_5:18">5:18</a> Lameɛna ma yella tleḥḥum s lebɣi n Ṛṛuḥ iqedsen, atan ihi ur tellim ara seddaw leḥkum n ccariɛa.  </p> 
 <p><a name="an_5:19">5:19</a> Axaṭer, lefɛayel n tnefsit țbanen : zzna, leḥṛam, lefsad, </p> 
 <p><a name="an_5:20">5:20</a> zzyaṛa n lemqamat, ssḥur, lkeṛh, ccwal, tismin, zzɛaf, wid yebɣan a d-kken sennig wiyaḍ, assexṛeb, wid ifeṛqen ț-țirebbaɛ, </p> 
 <p><a name="an_5:21">5:21</a> ṭṭmeɛ, ssekṛan, at iɛebbaḍ ur nesɛi lqaɛ d wayen akk ițemcabin ɣer wayagi meṛṛa. Ɛeggneɣ-awen-d am akken i wen-d-nniɣ yakan : wid i gxeddmen lecɣal-agi ur seɛɛun ara amkan di tgelda n Sidi Ṛebbi.  </p> 
 <p><a name="an_5:22">5:22</a> Lameɛna atan wayen i d-itekken si Ṛṛuḥ iqedsen : leḥmala, lfeṛḥ, lehna, ṣṣbeṛ, lehhu, leḥnana, liman, </p> 
 <p><a name="an_5:23">5:23</a> ul yeṣfan, aɣlab n tnefsit ; eɛni ccariɛa tugi annect-agi ?  </p> 
 <p><a name="an_5:24">5:24</a> Wid yellan n Ɛisa Lmasiḥ, semmṛen ɣef wumidag lebɣi n tnefsit-nsen, s ccehwat-is d umenni ines. </p> 
 <p><a name="an_5:25">5:25</a> Imi d Ṛṛuḥ iqedsen i ɣ-d ifkan tudert, ilaq-aɣ ihi a neddu di lebɣi ines.  </p> 
 <p><a name="an_5:26">5:26</a> Ur ilaq a nessimɣuṛ iman nneɣ, ur ilaq a nețnaɣ neɣ a nețțemyasam wway-gar-aneɣ.  </p> 
 <p><a name="an_6:1">6:1</a> Ay atmaten, kunwi ileḥḥun s Ṛṛuḥ iqedsen, ma teṭṭfem yiwen yeɣleḍ, erret-ț-id ɣer webrid s leḥnana ; ḥadret ɣef yiman-nwen, axaṭer tzemrem aț-țeɣlim ula d kunwi deg ujeṛṛeb. </p> 
 <p><a name="an_6:2">6:2</a> ?emɛawanet deg unerfud n tɛekkmin nwen, s wakka ara tḍuɛem ayen i d-tenna ccariɛa n Lmasiḥ. </p> 
 <p><a name="an_6:3">6:3</a> Ma yeḥseb yiwen iman-is yesɛa lqima nețța ur yeswi kra, ɣef yiman-is i gețkellix.  </p> 
 <p><a name="an_6:4">6:4</a> Ilaq mkul yiwen ad imeyyez tikli-ines, imiren ma yufa ayen s wayes ara izux, ad izux i yiman-is kan mačči alamma imettel iman-is ɣer wiyaḍ. </p> 
 <p><a name="an_6:5">6:5</a> Axaṭer mkul yiwen ad yerfed taɛkumt-is weḥd-es.  </p> 
 <p><a name="an_6:6">6:6</a> Win iwumi sselmaden awal n Ṛebbi ilaq ad yeǧǧ amur deg wayen akk yesɛa i win i t-isselmaden.  </p> 
 <p><a name="an_6:7">6:7</a> ?adret aț-țɣelṭem : yiwen ur yezmir ad istehzi ɣef Ṛebbi ; ayen yezreɛ wemdan, a t-id-imger.  </p> 
 <p><a name="an_6:8">6:8</a> Win yettabaɛen lebɣi n tnefsit-is, a t-tessiweḍ ɣer lmut ; ma d win iteddun s Ṛṛuḥ iqedsen ad isɛu tudert n dayem.  </p> 
 <p><a name="an_6:9">6:9</a> Ilaq a nxeddem lxiṛ mbla ɛeyyu ; axaṭer m'ur nefcil ara, a d-nemger m'ara d-yaweḍ lweqt. </p> 
 <p><a name="an_6:10">6:10</a> Ihi, skud yella wamek, a nexdem lxiṛ i yemdanen meṛṛa, abeɛda atmaten di liman !  </p> 
 <p><a name="an_6:11">6:11</a> Walit tira-agi : s ufus-iw i wen tent-id-uriɣ. </p> 
 <p><a name="an_6:12">6:12</a> Wid meṛṛa i gebɣan ad ḥețțmen fell-awen ṭṭhaṛa, bɣan a d-sbeggnen iman-nsen zdat imdanen belli ṭṭfen di leɛwayed, iwakken ur țwaqehhaṛen ara ɣef ddemma n umidag n Lmasiḥ.  </p> 
 <p><a name="an_6:13">6:13</a> Widak-agi iḍehṛen, nutni s yiman nsen ur ttabaɛen ara ccariɛa ; lameɛna bɣan aț-țḍehṛem iwakken ad zuxxen imi ḥettmen fell-awen ṭṭhaṛa-agi.  </p> 
 <p><a name="an_6:14">6:14</a> Ma d nekk ur bɣiɣ ad zuxxeɣ s wacemma, anagar s lmut n Ɛisa Lmasiḥ ɣef wumidag ; axaṭer s lmut n Lmasiḥ ɣef wumidag uɣaleɣ ur ḥsibeɣ ara ddunit-agi, akken daɣen ddunit ur iyi-teḥsib ara ula d nekk. </p> 
 <p><a name="an_6:15">6:15</a> Axaṭer ayen yesɛan azal, mačči d ṭṭhaṛa, lameɛna ț-țalalit tajḍiṭ i d-itekken s ɣuṛ Sidi Ṛebbi.  </p> 
 <p><a name="an_6:16">6:16</a> I wid akk itebɛen abrid-agi , a sen-iniɣ: lehna d ṛṛeḥma n Sidi Ṛebbi a wen-d țțunefkent i kunwi d wegdud n Sidi Ṛebbi meṛṛa !  </p> 
 <p><a name="an_6:17">6:17</a> Sya ɣer zdat, ur ḥwaǧeɣ yiwen a yi-isseḥzen ; axaṭer ccwami yellan di lǧețța-w țbegginent-ed belli d aqeddac n Sidna Ɛisa i lliɣ.  </p> 
 <p><a name="an_6:18">6:18</a> Ay atmaten, ṛṛeḥma n Ssid-nneɣ Ɛisa Lmasiḥ aț-țili yid-wen meṛṛa ! Amin !  </p>
