  <h3 align="center">Kabyle Bible: Colossians</h3> 
 <p><a name="an_1:1">1:1</a> Nekk Bulus, amceggeɛ n Ɛisa Lmasiḥ s lebɣi n Sidi Ṛebbi akk-d gma-tneɣ Timuti, </p> 
 <p><a name="an_1:2">1:2</a> i watmaten yellan di temdint n Kulus, wid iteddun s ṣṣfa deg webrid n Lmasiḥ. Ṛṛeḥma ț-țalwit a wen-d-țțunefkent s ɣuṛ Ṛebbi Baba-tneɣ !  </p> 
 <p><a name="an_1:3">1:3</a> Nețḥemmid Sidi Ṛebbi Baba-s n Ssid-nneɣ Ɛisa Lmasiḥ yerna ndeɛɛu fell-awen ; </p> 
 <p><a name="an_1:4">1:4</a> axaṭer nesla s liman i tesɛam di Ɛisa Lmasiḥ d wamek i tḥemmlem agdud n Sidi Ṛebbi, </p> 
 <p><a name="an_1:5">1:5</a> ɣef ddemma n usirem i wen-ițțuheggan deg igenwan, asirem nni i wen-d-ițțuxebbṛen s wawal n tideț n lexbaṛ n lxiṛ.  </p> 
 <p><a name="an_1:6">1:6</a> Lexbeṛ-agi n lxiṛ  yewḍen armi d ɣuṛ-wen, yețnerni di ddunit meṛṛa ; akken daɣen i gețnerni deg-wen ula d kunwi seg wasmi i teslam yerna tessnem tideț ɣef ṛṛeḥma n Sidi Ṛebbi,  </p> 
 <p><a name="an_1:7">1:7</a> akken i wen-ț-isselmed Ibafras, aṛfiq-nneɣ eɛzizen di leqdic ; nețța iqeddcen ɣef Lmasiḥ s wul yeṣfan, </p> 
 <p><a name="an_1:8">1:8</a> ixebbeṛ-aɣ-ed ɣef leḥmala s wayes i kkun-yeččuṛ Ṛṛuḥ iqedsen.  </p> 
 <p><a name="an_1:9">1:9</a> Daymi nezga ndeɛɛu fell-awen ɣer Sidi Ṛebbi seg wasmi nesla yis-wen, nessutur-as a wen-d-yefk amek ara tissinem akken ilaq lebɣi ines, s lɛeqliya d lefhama i d-ițțak Ṛṛuḥ iqedsen. </p> 
 <p><a name="an_1:10">1:10</a> S wakka, aț-țleḥḥum s tikli yebɣa Sidi Ṛebbi iwakken a s tɛeǧbem di kullec, a d-tefkem lfakya yelhan, yerna aț-țnernim di tmusni n Sidi Ṛebbi,  </p> 
 <p><a name="an_1:11">1:11</a> Ndeɛɛu ɣer Ṛebbi akken a kkun-isseǧhed s tezmert-is tameqqrant iwakken aț-tṣebṛem i kullec alamma ț-țaggara.  </p> 
 <p><a name="an_1:12">1:12</a> Ḥemmdet, cekkṛet s lfeṛḥ Sidi Ṛebbi Baba-tneɣ i wen-d-yefkan amur di lweṛt i d-ihegga i warraw n tafat.  </p> 
 <p><a name="an_1:13">1:13</a> Ikkes-aɣ-d si tezmert n ṭṭlam, iwakken a ɣ-yessiweḍ ɣer tgelda n Mmi-s eɛzizen, </p> 
 <p><a name="an_1:14">1:14</a> s wayes nețwasellek yerna nesɛa leɛfu n ddnubat-nneɣ.  </p> 
 <p><a name="an_1:15">1:15</a> Lmasiḥ, d ṣṣifa n Sidi Ṛebbi ur nezmir a nwali, d nețța i d amezwaru n txelqit meṛṛa. </p> 
 <p><a name="an_1:16">1:16</a> Yis i gețwaxleq wayen nețwali d wayen ur nețwali ara ama deg igenwan ama di lqaɛa, tigeldiwin d igelliden, lḥekmat akk-d tezmar yellan deg igenwan ; kullec yețwaxleq yis, kullec yețwaxleq i nețța.  </p> 
 <p><a name="an_1:17">1:17</a> Yella-d uqbel a d-yețwaxleq kra yellan, d nețța i d lsas n txelqit, </p> 
 <p><a name="an_1:18">1:18</a> d aqeṛṛuy n tejmaɛt n imasiḥiyen yellan d lǧețța-s. Yella si tazwara, d nețța i d amenzu i d-iḥyan si ger lmegtin iwakken ad yili d amezwaru di kullec.  </p> 
 <p><a name="an_1:19">1:19</a> Axaṭer Sidi Ṛebbi yufa lfeṛḥ-is s lekmal di Mmi-s, </p> 
 <p><a name="an_1:20">1:20</a> s Lmasiḥ i gemṣalaḥ akk-d txelqit, yessers-ed lehna ama di lqaɛa ama deg igenwan s idammen n Mmi-s yuzzlen ɣef wumidag. </p> 
 <p><a name="an_1:21">1:21</a> Kunwi yellan zik-nni d ibeṛṛaniyen, d iɛdawen n Sidi Ṛebbi s ixemmimen-nwen d yir lecɣal nwen  </p> 
 <p><a name="an_1:22">1:22</a> Tura issemṣaleḥ-ikkun yid-es s lmut n Mmi-s iwakken a d-tbanem zdat-es teṣfam, mbla lɛib.  </p> 
 <p><a name="an_1:23">1:23</a> Ma yella teṭṭfem di liman-nwen s tideț, ma yella tbeddem, tṛeṣṣam. Ɣuṛ-wat aț-țțwexxṛem ɣef wusirem i wen-d-yewwi lexbaṛ n lxiṛ  iwumi teslam, lexbaṛ-agi n lxiṛ i gețțubeccṛen i yemdanen n ddunit meṛṛa, i ɣef uɣaleɣ nekk d aqeddac.  </p> 
 <p><a name="an_1:24">1:24</a> Tura feṛḥeɣ aṭas imi nɛețțabeɣ fell-awen, axaṭer țkemmileɣ ayen ixuṣṣen i leɛtab n Lmasiḥ ɣef ddemma n imasiḥiyen meṛṛa, yellan d lǧețța-s, i ɣef uɣaleɣ d aqeddac.  </p> 
 <p><a name="an_1:25">1:25</a> Iceggeɛ-iyi-d Sidi Ṛebbi iwakken a wen-d-beccṛeɣ awal-is s lekmal.  </p> 
 <p><a name="an_1:26">1:26</a> Lbaḍna yeffren si zik, si lǧil ɣer lǧil, ibeggen-iț-id tura i wid yellan d ayla-s.  </p> 
 <p><a name="an_1:27">1:27</a> Yebɣa a d-yesken ccan n lbaḍna agi ifazen i leǧnas nniḍen ur nelli ara n wat Isṛail, iwakken ad walin Lmasiḥ deg-wen, nețța yellan d lsas n lɛaḍima i ɣer nessaram.  </p> 
 <p><a name="an_1:28">1:28</a> D Lmasiḥ agi i nețbecciṛ i yemdanen, nețweṣṣi-ten, nesselmad iten s tmusni d leɛqel, iwakken yal amdan ad innekmal di Lmasiḥ zdat Sidi Ṛebbi. </p> 
 <p><a name="an_1:29">1:29</a> Ɣef wayagi i qeddceɣ s tezmert n Lmasiḥ yellan deg-i.  </p> 
 <p><a name="an_2:1">2:1</a> Bɣiɣ aț-țeẓrem acḥal meqqeṛ imenɣi i deg lliɣ ɣef ddemma nwen, ɣef watmaten n temdint n Ludikus akk-d wid meṛṛa i geslan yis-i meɛna leɛmeṛ ur ẓrin udem-iw.  </p> 
 <p><a name="an_2:2">2:2</a> Qebleɣ imenɣi-agi iwakken ad theddnen wulawen-nsen, ad ddukklen s leḥmala, ad ččaṛen d lefhama iwakken ad issinen lbaḍna n Sidi Ṛebbi Ṛebbi yellan di Lmasiḥ ; </p> 
 <p><a name="an_2:3">2:3</a> deg-s i ffren igerrujen n lefhama ț-țmusni.  </p> 
 <p><a name="an_2:4">2:4</a> Nniɣ-d akka iwakken ur kkun ițkellix yiwen s imeslayen yețɣuṛṛun.  </p> 
 <p><a name="an_2:5">2:5</a> Ɣas akken ur lliɣ ara yid-wen, leɛqel-iw atan ɣuṛ-wen ; feṛḥeɣ aṭas mi ẓriɣ acḥal i teṭṭfem di liman-nwen di Lmasiḥ yerna tsedduyem kullec akken ilaq.  </p> 
 <p><a name="an_2:6">2:6</a> Ihi imi tqeblem Ɛisa Lmasiḥ d Ssid-nwen, ddut di lebɣi-ines. </p> 
 <p><a name="an_2:7">2:7</a> Sbeddet lsas n tudert-nwen fell-as, sǧehdet liman-nwen akken i kkun nesselmed, țḥemmidet daymen Sidi Ṛebbi. </p> 
 <p><a name="an_2:8">2:8</a> Ḥadret a kkun-kellxen wid ițfelsifen, wid yessexdamen tiḥila ț-țiḥeṛci ; ayagi d lekdubat i gebnan ɣef leɛwayed n yemdanen akk ț-țizemmar n ddunit, mačči ɣef uselmed n Lmasiḥ.  </p> 
 <p><a name="an_2:9">2:9</a> Axaṭer ayen akk yellan di Sidi Ṛebbi yezdeɣ s lekmal di Lmasiḥ, </p> 
 <p><a name="an_2:10">2:10</a> kunwi tesɛam kullec deg-s, nețța yellan sennig lḥekmat ț-țzemmar.  </p> 
 <p><a name="an_2:11">2:11</a> Sṭhaṛa-nwen tekka-d si tikli nwen akk-d Lmasiḥ, mačči am tin yețwaxedmen s ufus n wemdan, lameɛna s ɣuṛ Lmasiḥ i d-tekka iwakken a kkun-isellek si tezmert n lǧețța tamednubt.  </p> 
 <p><a name="an_2:12">2:12</a> S weɣḍas-nwen tețwameḍlem akk-d Lmasiḥ, lameɛna teḥyam-d daɣen yid-es si lmut imi tumnem s tezmert n Sidi Ṛebbi i t-id-isseḥyan si ger lmegtin.  </p> 
 <p><a name="an_2:13">2:13</a> Kunwi ur neḍhiṛ ara, tellam temmutem ɣef ddemma n ddnubat nwen ; tura Sidi Ṛebbi yerra-kkun-id ɣer tudert s Lmasiḥ. Sidi Ṛebbi yeɛfa-yaɣ ddnubat-nneɣ meṛṛa. </p> 
 <p><a name="an_2:14">2:14</a> Yemḥa leɛqed-nni i ɣ-isseḍlamen, imi ur nezmir ara a nexdem ayen d-teqqaṛ ccariɛa ; yemḥa-t mi t-isemmeṛ ɣef wumidag.  </p> 
 <p><a name="an_2:15">2:15</a> Sidi Ṛebbi yekkes tazmert i lḥekmat ț-țzemmar n igenwan, ikcef itent-id zdat txelqit meṛṛa, iɣleb-itent s lmut n Mmi-s ɣef wumidag.  </p> 
 <p><a name="an_2:16">2:16</a> Ihi ur țțaǧat ara win ara kkun-iḥasben ɣef wayen ara teččem d wayen ara teswem neɣ imi ur tettabaɛem ara ayen yeɛnan leɛyudat, talalit n waggur d wussan n westeɛfu ; </p> 
 <p><a name="an_2:17">2:17</a> ayagi akk yezwar-ed d lemtel kan n wayen i d-iteddun, axaṭer tideț di Lmasiḥ i tella. </p> 
 <p><a name="an_2:18">2:18</a> Ɣuṛ-wat a wen-kellxen yemdanen ițțarran iman-nsen d wid yunzen, țɛebbiden lmalayekkat, a kkun-id-awin ɣer webrid-nsen iwakken ur tețțawḍem ara ɣer lmeqṣud-nwen ; imdanen am wigi ttabaɛen kan ixemmimen-nsen yerna xeddmen ccan i yiman-nsen.  </p> 
 <p><a name="an_2:19">2:19</a> Ur qbilen ara ad ṭṭfen di Lmasiḥ yellan d aqeṛṛuy, i gesdukklen lǧețța s lemfaṣel meṛṛa d iẓuran i d-icudden ɣuṛ-es ; yis i tețnerni lǧețța, i tețțimɣuṛ s wakken yella di lebɣi n Sidi Ṛebbi.  </p> 
 <p><a name="an_2:20">2:20</a> Ma yella temmutem akk-d Lmasiḥ, teṭṭaxṛem ɣef wayen akk yeɛnan tudert n ddunit, acimi ihi am akken di ddunit i tellam, tqebblem ad ḥețțmen fell-awen leɛwayed n ddunit m'ara wen-qqaṛen :  </p> 
 <p><a name="an_2:21">2:21</a> « Ur teddmet ara neɣ ur ɛeṛṛdet ara neɣ ur țnalet ara !»  </p> 
 <p><a name="an_2:22">2:22</a> Lumuṛat-agi meṛṛa i nxeddem d ayen ara yefnun, d leqwanen d uselmed n yemdanen ;  </p> 
 <p><a name="an_2:23">2:23</a> ț-țideț, lumuṛat-agi sselmaden leɛbada, annuz akk-d uɛețțeb n lǧețțat nneɣ, țbanen-d amakken d ayen yelhan, lameɛna d ayen ur nezmir ad iɛawen bunadem ad iṭṭef iman-is.  </p> 
 <p><a name="an_3:1">3:1</a> Kunwi ihi, imi i d-teḥyam akk-d Lmasiḥ, nadit ɣef leḥwayeǧ n igenwan anda yella Lmasiḥ i gḥekmen ɣer tama tayeffust n Sidi Ṛebbi.  </p> 
 <p><a name="an_3:2">3:2</a> Țḥebbiṛet ɣef wayen yellan deg igenwan mačči ɣef wayen yellan di ddunit-agi, </p> 
 <p><a name="an_3:3">3:3</a> axaṭer ɣer ddunit tețțusemmam temmutem, meɛna ɣer Sidi Ṛebbi tudert-nwen teffer akk-d Lmasiḥ.  </p> 
 <p><a name="an_3:4">3:4</a> M'ara d-iban Lmasiḥ yellan ț-țudert-nwen, kunwi daɣen a d tbanem yid-es. </p> 
 <p><a name="an_3:5">3:5</a> Ɣef ddemma n wayagi, xḍut i yir tikli n ddunit-agi s wayes tețțeddum zik : leḥṛam, ayen iɛefnen, lebɣi n tnefsit, ccehwat n diri akk-d ṭṭmeɛ, ayagi meṛṛa am win ițɛebbiden ayen ixulfen Sidi Ṛebbi.  </p> 
 <p><a name="an_3:6">3:6</a> D annect-agi i d-isseɣlayen urrif n Sidi Ṛebbi ɣef wid i t-ițɛaṣin. </p> 
 <p><a name="an_3:7">3:7</a> Ula d kunwi akka i tellam zik-nni mi tețɛicim di ddnub. </p> 
 <p><a name="an_3:8">3:8</a> Meɛna tura sbeɛdet iman-nwen ɣef wannect-agi meṛṛa : urrif, zzɛaf, ddɣel ; ḥadret a d-ffɣen seg imawen-nwen rregmat neɣ yir lehduṛ.  </p> 
 <p><a name="an_3:9">3:9</a> Ur skiddibet ara wway gar-awen imi tura teǧǧam yir tikli-nwen n zik.  </p> 
 <p><a name="an_3:10">3:10</a> Tuɣalem d ijdiden, tețnernim di tmusni n Sidi Ṛebbi iwakken aț țemcabim ɣuṛ-es, nețța i kkun-d-ixelqen.  </p> 
 <p><a name="an_3:11">3:11</a> Di tudert-agi tajdiṭ ulac lxilaf ger wat Isṛail d leǧnas nniḍen, ger win iḍehṛen d win ur neḍhiṛ, ger wid itqeddmen d wid ur netqeddem, neɣ ger wakli d uḥeṛṛi, imi d Lmasiḥ i gellan d kullec di mkul yiwen.  </p> 
 <p><a name="an_3:12">3:12</a> Sidi Ṛebbi iḥemmel-ikkun, yextaṛ-ikkun aț-țilim d agdud-is. Sɛut ihi ṛṛeḥma deg wulawen-nwen, leḥnana, lxiṛ, ṣṣbeṛ d wannuz.  </p> 
 <p><a name="an_3:13">3:13</a> Țemyawit wway gar-awen, ma yella win yesɛan accetki ɣef wayeḍ msamaḥet axiṛ wway gar-awen, akken i wen-isemmeḥ Lmasiḥ, msamaḥet ula d kunwi.  </p> 
 <p><a name="an_3:14">3:14</a> Nnig n wayagi meṛṛa sɛut leḥmala, ț-țin i gesdukkulen ulawen. </p> 
 <p><a name="an_3:15">3:15</a> Sɛut lehna n Lmasiḥ deg ulawen nwen iwakken aț-țilim d yiwen, ḥemmdet Sidi Ṛebbi imi ɣer lehna-agi i wen-d-yessawel.  </p> 
 <p><a name="an_3:16">3:16</a> Awal n Lmasiḥ ad izdeɣ gar-awen s lbaṛakat-is meṛṛa iwakken wa ad isselmad wa, wa ad inehhu wayeḍ s ṣṣwab. Ccnut s wulawen-nwen, ḥemmdet Sidi Ṛebbi s isefra n Zabuṛ akk-d ccnawi i d-ițțak Ṛṛuḥ iqedsen. </p> 
 <p><a name="an_3:17">3:17</a> Ama deg imeslayen ama di lecɣal ara txedmem, xedmet kullec s yisem n Ssid-nneɣ Ɛisa, s yisem-is țḥemmidet Sidi Ṛebbi baba-tneɣ.  </p> 
 <p><a name="an_3:18">3:18</a> A tilawin, ḍuɛemt irgazen-nkunt, ayagi d ayen ilaqen i tid i gețḍuɛun Sidi Ṛebbi. </p> 
 <p><a name="an_3:19">3:19</a> Ma d kunwi ay irgazen, ilaq aț-țḥemmlem tilawin-nwen, ur seɛɛut ara ulawen iqesḥen ɣuṛ-sent.  </p> 
 <p><a name="an_3:20">3:20</a> Ay arrac țțaɣet awal i imawlan nwen di kullec, axaṭer akka ara tɛeǧbem i Sidi Ṛebbi. </p> 
 <p><a name="an_3:21">3:21</a> Lameɛna kunwi ay imawlan ur sserfuyet ara dderya-nwen neɣ m'ulac ad feclen.  </p> 
 <p><a name="an_3:22">3:22</a> Ma d kunwi ay iqeddacen, țțaɣet awal i imɛellmen-nwen di ddunit-agi mačči kan zdat-nsen akken a sen tɛeǧbem meɛna ɣas ulac-iten xedmet ccɣel-nwen seg wul, am akken i Sidi Ṛebbi. </p> 
 <p><a name="an_3:23">3:23</a> Kra n wayen ara txedmem, xedmet-eț s wul yeṣfan am akken i Sidi Ṛebbi i t-txeddmem mačči i yemdanen, </p> 
 <p><a name="an_3:24">3:24</a> mmektit-ed lweṛt i kkun ițṛaǧun s ɣuṛ Sidi Ṛebbi akken i t-id-yewɛed ; axaṭer amɛellem n tideț i ɣef tqeddcem, d Lmasiḥ. </p> 
 <p><a name="an_3:25">3:25</a> Win ixeddmen cceṛ a d-iɣellet ayen yexdem ; axaṭer ɣer Sidi Ṛebbi ulac lxilaf.  </p> 
 <p><a name="an_4:1">4:1</a> Ay imɛellmen ddut s lḥeqq, ur xeddmet ara lxilaf ger iqeddacen nwen, axaṭer teẓram belli ula d kunwi tesɛam amɛellem deg igenwan. </p> 
 <p><a name="an_4:2">4:2</a> Ur ɣefflet ara, meɛna deɛɛut țḥemmidet daymen Sidi Ṛebbi.  </p> 
 <p><a name="an_4:3">4:3</a> Deɛɛut daɣen fell-aɣ ɣer Sidi Ṛebbi iwakken a ɣ-d-ildi tiwwura a nbecceṛ awal-is, a nxebbeṛ ɣef lbaḍna n Lmasiḥ i ɣef lliɣ di lḥebs, </p> 
 <p><a name="an_4:4">4:4</a> dɛut iwakken ad sfehmeɣ akken ilaq lbaḍna-agi i yemdanen.  </p> 
 <p><a name="an_4:5">4:5</a> Ddut s ṣṣwab zdat wid ur numin ara s Ɛisa Lmasiḥ ; faṛset tagniț, țbecciṛet awal n Ɛisa Lmasiḥ.  </p> 
 <p><a name="an_4:6">4:6</a> Sɛut iles aẓidan, imeslayen yesɛan lfayda, iwakken aț-țețțarram awal i yal yiwen s ṣṣwab.  </p> 
 <p><a name="an_4:7">4:7</a> Tucik, gma-tneɣ eɛzizen i ɣef țekleɣ, aṛfiq-nneɣ di leqdic ɣef Ssid-nneɣ, a wen-n-yawi lexbaṛ-iw. </p> 
 <p><a name="an_4:8">4:8</a> Ɣef wannect-agi i wen-t-in ceggɛeɣ iwakken a wen-n-yawi lexbaṛ-nneɣ yerna ad ihedden ulawen-nwen.  </p> 
 <p><a name="an_4:9">4:9</a> A d-yas nețța d gma-tneɣ eɛzizen Unizim i ɣef nețkel, yellan d yiwen seg-wen ; d nutni ara wen-n-yawin lexbaṛ ɣef wayen meṛṛa ideṛṛun dagi.  </p> 
 <p><a name="an_4:10">4:10</a> Ițsellim-ed fell-awen Arisṭark aṛfiq-iw yellan yid-i di lḥebs akk-d Maṛqus yețțilin i Barnabas, win i ɣef kkun-weṣṣaɣ aț-țesterḥbem yis ma yusa-d ɣuṛ-wen.  </p> 
 <p><a name="an_4:11">4:11</a> Ițsellim-ed daɣen fell-awen Ɛisa iwumi qqaṛen Yustus. Ger wat Isṛail anagar tlata watmaten-agi i gxeddmen yid-i i tgeldit n Sidi Ṛebbi, d nutni kan i yufiɣ ɣer tama-w.  </p> 
 <p><a name="an_4:12">4:12</a> Gma-tneɣ Ibafras n tmurt-nwen, yellan d aqeddac n Ɛisa Lmasiḥ ițsellim-ed fell-awen ; yezga ideɛɛu ɣer Ṛebbi fell-awen iwakken aț-țeṭṭfem, aț-țennernim di liman yerna aț-țilim twejdem aț-țxedmem lebɣi n Sidi Ṛebbi.  </p> 
 <p><a name="an_4:13">4:13</a> A d-cehdeɣ fell-as acḥal inɛețțab fell-awen, ɣef watmaten n tejmaɛt n Ludikus akk-d wid n Yirabulis.  </p> 
 <p><a name="an_4:14">4:14</a> Luqa aḥbib-nneɣ eɛzizen yellan d ṭṭbib akk-d Dimas, țsellimen-d fell-awen. </p> 
 <p><a name="an_4:15">4:15</a> Sellmet ɣef watmaten yellan di Ludikus, ɣef Nimfa akk d wid ițnejmaɛen deg wexxam-is.  </p> 
 <p><a name="an_4:16">4:16</a> M'ara teɣṛem tabṛaț-agi, fket-eț i watmaten n tejmaɛt n Ludikus a ț-ɣṛen, kunwi daɣen ɣṛet tabṛaț ara kkun-id-yawḍen s ɣuṛ-sen.  </p> 
 <p><a name="an_4:17">4:17</a> Init i Arcibus ad ibedd i ccɣel nni i ɣef t-iwekkel Ssid-nneɣ, a t-ixdem akken ilaq.  </p> 
 <p><a name="an_4:18">4:18</a> Nekk Bulus, s ufus-iw i wen-n uriɣ. Sslam fell-awen ! Ur tețțut ara belli d ameḥbus i lliɣ. Ṛṛeḥma n Sidi Ṛebbi aț-țili yid-wen.  </p>
