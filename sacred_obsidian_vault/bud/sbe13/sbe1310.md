 <h2 align="center">SEKHIYÂ DHAMMÂ.</h2>
 
 <p align="center">RULES REGARDING MATTERS CONNECTED WITH DISCIPLINE.</p>
 
 <p>   Here, venerable Sirs, the rules regarding matters connected with discipline come into recitation.</p>
 
 <p>   <a name="1"></a><font color="GREEN">1</font>. 'I will put on my under garment all around me<a name="page_59_fr_1"></a><a href="#page_59_note_1"><sup><small>1</small></sup></a>.' This is a discipline which ought to be observed.</p>
 
 <p>   <a name="2"></a><font color="GREEN">2</font>. 'I will put on my robe all around me.' This is a discipline which ought to be observed.</p>
 
 <p>   <a name="3"></a><font color="GREEN">3</font>. 'Properly clad will I go amidst the houses<a name="page_59_fr_2"></a><a href="#page_59_note_2"><sup><small>2</small></sup></a>. This is a discipline which ought to be observed.</p>
 
 <p>   <a name="4"></a><font color="GREEN">4</font>. 'Properly clad will I take my seat amidst the bouses.' This is a discipline which ought to be observed.</p>
 
 <p>   <a name="5"></a><font color="GREEN">5</font>. '(With my body) under proper control<a name="page_59_fr_3"></a><a href="#page_59_note_3"><sup><small>3</small></sup></a> will I go amidst the houses.' This is a discipline which ought to be observed.</p>
 
 <p>   <a name="6"></a><font color="GREEN">6</font>. '(With my body) under proper control will I take my seat amidst the houses.' This is a discipline which ought to be observed.</p>
 
 <p>   <a name="7"></a><font color="GREEN">7</font>. 'With downcast eye<a name="page_59_fr_4"></a><a href="#page_59_note_4"><sup><small>4</small></sup></a> will I go amidst the

 <a name="page_60"><font size="1" color="GREEN">p. 60</font></a>
 
 houses.' This is a discipline which ought to be observed.</p>
 
 <p>   <a name="8"></a><font color="GREEN">8</font>. 'With downcast eye will I take my seat amidst the houses.' This is a discipline which ought to be observed.</p>
 
 <p>   <a name="9"></a><font color="GREEN">9</font>. 'With robes not pulled up<a name="page_60_fr_1"></a><a href="#page_60_note_1"><sup><small>1</small></sup></a> will I go amidst the houses.' This is a discipline which ought to be observed.</p>
 
 <p>   <a name="10"></a><font color="GREEN">10</font>. 'With robes not pulled up will I take my seat amidst the houses.' This is a discipline which ought to be observed.</p>

 <p>   <a name="36"></a><font color="GREEN">36</font>. 'Neither the curry nor the condiment will I cover up with the rice, desiring to make it nicer<a name="page_63_fr_3"></a><a href="#page_63_note_3"><sup><small>3</small></sup></a>.' This is a discipline which ought to be observed.</p>
 
 <p>   <a name="37"></a><font color="GREEN">37</font>. 'Neither curry nor rice will I ask for, for my own particular use, unless I am sick.' This is discipline which ought to be observed.</p>

 <p>   <a name="39"></a><font color="GREEN">39</font>. 'Not into too large balls will I make (up my food).' This is a discipline which ought to be observed.</p>
 
 <p>   <a name="40"></a><font color="GREEN">40</font>. 'Into round mouthfuls will I make up my food.' This is a discipline which ought to be observed.</p>

 <hr width="33%" align="center">
 
 <p>   Venerable Sirs, the rules regarding matters of discipline have been recited.</p>
 
 <p>   ln respect of them I ask the venerable ones, 'Are you pure in this matter?'</p>
 
 <p>   A second time I ask the venerable ones, 'Are you pure in this matter?'</p>
 
 <p>   A third time I ask the venerable ones, 'Are you pure in this matter?'</p>
 
 <p>   The venerable ones are pure herein. Therefore do they keep silence. Thus I understand.</p>
