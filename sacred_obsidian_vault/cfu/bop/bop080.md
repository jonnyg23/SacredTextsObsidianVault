 <p><a name="page_88"><font size="1" color="green">p. 88</font></a></p>  <h4 align="center">III</h4>  
 <p></p><div style="font-size: 90%">The <i>Shu Yü T‘ien;</i> narrative. The admiration with which <i>Shu Tuan</i> was regarded.</div>  
 <p></p><div style="margin-left: 20%; margin-right: 20%; font-size: 110%;"><span class="margnote"><font color="GREEN" size="-1"><a name="an_1">1</a></font></span>To the hunt Shu has gone,<br> And people there are none<br>   Remaining in the street.<br> Perhaps a few you'll find;<br> But none like Shu so kind,<br>   So graceful, will you meet.  
 <p><span class="margnote"><font color="GREEN" size="-1"><a name="an_2">2</a></font></span>To the chase Shu has gone,<br> And people there are none<br>   Left feasting in the street.<br> If find a few you could,<br> Yet none like Shu so good,<br>   So graceful, would you meet.</p>  
 <p><span class="margnote"><font color="GREEN" size="-1"><a name="an_3">3</a></font></span>To the fields Shu has gone,<br> And people there are none<br>   Careering through the street.<br> Some riders though you told,<br> Yet none like Shu so bold,<br>   So graceful, would you meet.</p></div>
