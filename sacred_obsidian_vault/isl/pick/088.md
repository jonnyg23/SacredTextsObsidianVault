 <h1 align="CENTER">88. al-Ghashiya: The Overwhelming Calamity</h1>
 <a name="an_088_001">1</a> Hath there come unto thee tidings of the Overwhelming ? 
 <a name="an_088_002">2</a> On that day (many) faces will be downcast, 
 <a name="an_088_003">3</a> Toiling, weary, 
 <a name="an_088_004">4</a> Scorched by burning fire, 
 <a name="an_088_005">5</a> Drinking from a boiling spring, 
 <a name="an_088_006">6</a> No food for them save bitter thorn-fruit 
 <a name="an_088_007">7</a> Which doth not nourish nor release from hunger. 
 <a name="an_088_008">8</a> In that day other faces will be calm, 
 <a name="an_088_009">9</a> Glad for their effort past, 
 <a name="an_088_010">10</a> In a high Garden 
 <a name="an_088_011">11</a> Where they hear no idle speech, 
 <a name="an_088_012">12</a> Wherein is a gushing spring, 
 <a name="an_088_013">13</a> Wherein are couches raised 
 <a name="an_088_014">14</a> And goblets set at hand 
 <a name="an_088_015">15</a> And cushions ranged 
 <a name="an_088_016">16</a> And silken carpets spread. 
 <a name="an_088_017">17</a> Will they not regard the camels, how they are created ? 
 <a name="an_088_018">18</a> And the heaven, how it is raised ? 
 <a name="an_088_019">19</a> And the hills, how they are set up ? 
 <a name="an_088_020">20</a> And the earth, how it is spread ? 
 <a name="an_088_021">21</a> Remind them, for thou art but a remembrancer, 
 <a name="an_088_022">22</a> Thou art not at all a warder over them. 
 <a name="an_088_023">23</a> But whoso is averse and disbelieveth, 
 <a name="an_088_024">24</a> Allah will punish him with direst punishment. 
 <a name="an_088_025">25</a> Lo! unto Us is their return 
 <a name="an_088_026">26</a> And Ours their reckoning.
